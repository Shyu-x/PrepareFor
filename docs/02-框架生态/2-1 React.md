## 一、Hooks 专项

### 1.1 useEffect 与 useLayoutEffect 的区别

**参考答案：**

| 特性 | useEffect | useLayoutEffect |
| :--- | :--- | :--- |
| **执行时机** | 浏览器渲染并涂屏 **之后** | 浏览器计算完样式但 **还没涂屏** 之前 |
| **阻塞性** | 异步、不阻塞渲染 | **同步**、阻塞渲染 |
| **应用场景** | 大部分副作用（请求、日志） | 需要操作 DOM 且 **不希望用户看到闪烁** 的场景（如测量尺寸、自动聚焦） |
