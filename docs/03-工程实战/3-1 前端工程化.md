## 一、打包优化

### 1.1 Webpack 分包策略 (SplitChunks) 如何配置？

**参考答案：**
通过 `optimization.splitChunks` 实现资源复用与按需加载。

```javascript
optimization: {
  splitChunks: {
    chunks: 'all', // 初始、异步 chunk 均参与分包
    cacheGroups: {
      vendors: { // 第三方库
        test: /[\\/]node_modules[\\/]/,
        priority: -10,
        name: 'vendors'
      },
      common: { // 业务公共代码
        minChunks: 2,
        priority: -20,
        reuseExistingChunk: true
      }
    }
  }
}
```

## 二、微前端方案落地难点

### 2.1 微前端落地的坑与解决方案 (qiankun 为例)

1.  **样式污染**：
    *   解决方案：开启 `strictStyleIsolation` (Shadow DOM) 或 `experimentalStyleIsolation` (运行时选择器前缀)。
2.  **全局变量冲突**：
    *   解决方案：`ProxySandbox` 代理沙箱。
3.  **子应用通信**：
    *   解决方案：使用 `initGlobalState` 通信或 EventBus。
