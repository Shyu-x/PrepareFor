## 四、项目经验深度优化

### 4.1 WebEnv-OS 项目优化

**当前描述**

```
WebEnv-OS (基于Web的前端桌面与交互环境)
技术栈：HTML5 / CSS3、JavaScript、React Hooks、Zustand

为深入理解浏览器原生能力与React状态驱动机制，尝试在浏览器中从零构建了一个类桌面操作系统的交互界面。底层抛弃沉重的UI库，使用HTML5与CSS3搭建视窗结构，深入处理了复杂的多窗口层级（z-index）计算与绝对定位布局。

在业务交互上，结合React状态与原生JavaScript事件监听，实现了基于鼠标坐标差值的窗口拖拽、点击置顶及动态缩放等强DOM交互逻辑。了解并引入了Zustand进行系统级状态（如任务栏应用列表）管理。同时独立阅读官方API，将Monaco Editor成功嵌入自定义的沙箱窗口，锻炼了对复杂第三方库的集成排错能力。
```

**问题分析**

1. 项目亮点不够突出：多窗口管理、拖拽缩放、Monaco集成都是常见功能，需要突出差异化
2. 缺少成果数据：如代码行数、性能指标、用户反馈等
3. 缺少业务背景：为什么做这个项目？解决了什么需求？

**优化建议**

```
WebEnv-OS - 基于Web的前端桌面与交互环境
技术栈：React Hooks + Zustand + Monaco Editor + CSS3

项目背景：在学习React过程中，我发现市面上大多数教程只停留在“组件堆砌”层面，
缺乏对浏览器原生能力的深入理解。因此决定从零构建一个类桌面操作系统的交互界面，
作为深入学习React状态驱动机制的练手项目。

核心挑战与解决方案：
• 多窗口层级管理：使用Zustand维护全局z-index状态栈，实现窗口的点击置顶、层级计算
• 高性能拖拽：使用CSS transform替代top/left触发GPU加速，结合requestAnimationFrame
  优化拖拽帧率，将重排转化为重绘
• Monaco Editor集成：独立阅读官方文档，解决体积过大、首屏加载慢的问题，
  通过动态导入和automaticLayout配置实现流畅体验

成果数据：
• 实现了完整的桌面环境交互：窗口拖拽、缩放、最小化、最大化、置顶
• 成功嵌入Monaco代码编辑器，支持JavaScript语法高亮
• 项目获得首届海螺AI创意大赛二等奖
```

### 4.2 FastDocument 项目优化

**当前描述**

```
FastDocument (在线实时协作知识库)
技术栈：HTML5、JavaScript、React、Socket.io、LiveKit

一个仿照语雀、飞书、Notion等主流文档管理平台的项目，搭建了一个支持富文本录入和基础数据双向同步的知识库雏形。结合HTML5语义化标签构建了标准的UI与导航布局。在核心编辑区，了解到原生contenteditable的局限性，故引入并学习了基于原生JavaScript封装的开源编辑器TipTap，通过研读文档为其扩展了Markdown快捷输入与浮动气泡菜单。

实践了WebSocket全双工通信协议。在前端尝试引入socket.io-client建立长连接，初步探索并实现了多端文档内容与基础光标的广播同步机制。

此外，项目中还引入了LiveKit开源实时音视频引擎，探索并集成了基础的多人会议功能。
```

**问题分析**

1. 技术深度可以进一步挖掘：TipTap、Socket.io、LiveKit都是很好的技术点
2. 缺少实时协作的核心算法理解：虽然实现了基本同步，但可以补充对CRDT/OT的了解
3. 项目成果可以量化

**优化建议**

```
FastDocument - 在线实时协作知识库
技术栈：React + TipTap + Socket.io + LiveKit

项目背景：日常使用Notion、语雀等文档协作工具，对“多人同时编辑同一文档”
的技术实现产生了浓厚兴趣。通过这个项目系统学习WebSocket实时通信、富文本编辑器
原理及第三方库集成。

核心技术实现：
• 编辑器选型：深入对比原生contenteditable的局限性（光标控制困难、HTML结构不可预测），
  选择基于ProseMirror的TipTap，扩展Markdown快捷输入（#、**、``等自动转换）
• 实时同步：使用Socket.io实现WebSocket全双工通信，构建操作日志系统支持断线重连；
  通过监听selectionChange事件实现多端光标位置广播
• 视频会议：集成LiveKit开源引擎，探索性地实现了基础的多人视频会议功能

成果数据：
• 实现了完整的文档管理功能：创建、编辑、删除、实时同步
• 探索性地集成了实时音视频会议（LiveKit）
• 项目获得校内创新创业大赛银奖
```
