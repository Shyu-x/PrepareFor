## 一、JavaScript 核心高频考点

JavaScript是前端开发的基础，也是面试中考察最多的模块。根据简历描述，需要重点掌握以下内容：

### 1.1 事件循环（Event Loop）—— 最高频考点

**考察形式**：代码执行顺序题，给定一段包含Promise、setTimeout、async/await的代码，问最终输出顺序。

**核心知识点**：

JavaScript是单线程语言，通过事件循环（Event Loop）实现异步执行。事件循环中分为宏任务队列（Task Queue）和微任务队列（MicroTask Queue）。

```javascript
// 经典例题
async function async1() {
    console.log('async1 start');
    await async2();
    console.log('async1 end');
}

async function async2() {
    console.log('async2');
}

console.log('script start');

setTimeout(() => {
    console.log('setTimeout');
}, 0);

async1();

new Promise((resolve) => {
    console.log('promise1');
    resolve();
}).then(() => {
    console.log('promise2');
});

console.log('script end');
```

**执行顺序分析**：

```
1. 同步代码执行：
   - console.log('script start')
   - console.log('async1 start')
   - console.log('async2')
   - console.log('promise1')
   - console.log('script end')

2. 微任务队列执行（await 后的代码相当于 Promise 的 then）：
   - console.log('async1 end')
   - console.log('promise2')

3. 宏任务队列执行：
   - console.log('setTimeout')

最终输出：
script start → async1 start → async2 → promise1 → script end → async1 end → promise2 → setTimeout
```

**关键理解点**：

- async函数中，await后面的代码相当于Promise.then，会加入微任务队列
- Promise的resolve/reject回调也是微任务
- setTimeout、setInterval是宏任务
- 微任务队列在当前宏任务执行完毕后、下一个宏任务开始前清空

### 1.2 原型与原型链

**考察形式**：手写继承代码、解释原型链、this指向问题。

**核心知识点**：

```javascript

function Animal(name) {
    this.name = name;
}

Animal.prototype.speak = function() {
    console.log(`${this.name} makes a noise.`);
};

function Dog(name) {
    Animal.call(this, name);
}

// 原型链继承的关键：子类的原型指向父类的实例
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;

// 测试

dog.speak(); // "Rex makes a noise."
dog instanceof Animal; // true
```

**原型链图示**：

```
dog → Dog.prototype → Animal.prototype → Object.prototype → null
     ↑                   ↑
     constructor         constructor
```

**常见面试题**：instanceOf的原理实现

```javascript
function myInstanceOf(left, right) {
    let proto = Object.getPrototypeOf(left);
    const prototype = right.prototype;
    
    while (proto !== null) {
        if (proto === prototype) return true;
        proto = Object.getPrototypeOf(proto);
    }
    return false;
}
```

### 1.3 闭包

**考察形式**：手写闭包应用（防抖、节流、节流）、分析代码输出、内存泄漏分析。

**核心知识点**：

闭包是指函数能够访问其外部作用域中变量的能力。经典应用场景包括：

```javascript
// 计数器闭包
function createCounter() {
    let count = 0;
    return {
        increment() { return ++count; },
        decrement() { return --count; },
        getCount() { return count; }
    };
}

const counter = createCounter();
counter.increment(); // 1
counter.increment(); // 2
counter.getCount(); // 2

// 闭包陷阱：循环中的闭包
for (var i = 0; i < 3; i++) {
    setTimeout(() => console.log(i), 100);
}
// 输出：3, 3, 3（var没有块级作用域）

// 解决方案1：使用let
for (let i = 0; i < 3; i++) {
    setTimeout(() => console.log(i), 100);
}
// 输出：0, 1, 2

// 解决方案2：使用闭包
for (var i = 0; i < 3; i++) {
    ((j) => setTimeout(() => console.log(j), 100))(i);
}
```

```javascript
// 手写Promise.all
function promiseAll(promises) {
    return new Promise((resolve, reject) => {
        if (!Array.isArray(promises)) {
            return reject(new TypeError('参数数组'));
        }
        
        const results必须是 = [];
        let completed = 0;
        
        if (promises.length === 0) {
            return resolve(results);
        }
        
        promises.forEach((promise, index) => {
            Promise.resolve(promise).then(
                value => {
                    results[index] = value;
                    completed++;
                    if (completed === promises.length) {
                        resolve(results);
                    }
                },
                reason => reject(reason)
            );
        });
    });
}
```

### 1.5 深拷贝

**考察形式**：手写深拷贝函数，处理循环引用、Date、RegExp等特殊对象。

**核心知识点**：

```javascript
function deepClone(target, cache = new WeakMap()) {
    // 处理原始类型
    if (target === null || typeof target !== 'object') {
        return target;
    }
    
    // 处理Date
    if (target instanceof Date) {
        return new Date(target);
    }
    
    // 处理RegExp
    if (target instanceof RegExp) {
        return new RegExp(target);
    }
    
    // 处理循环引用
    if (cache.has(target)) {
        return cache.get(target);
    }
    
    // 处理数组和对象
    const cloneTarget = Array.isArray(target) ? [] : {};
    cache.set(target, cloneTarget);
    
    // 遍历所有属性
    for (const key in target) {
        if (target.hasOwnProperty(key)) {
            cloneTarget[key] = deepClone(target[key], cache);
        }
    }
    
    return cloneTarget;
}
```

```javascript
// Diff过程
function diff(oldTree, newTree) {
    // 1. 类型不同：直接替换整个子树
    if (oldTree.type !== newTree.type) {
        return newTree;
    }

    if (oldTree.type === newTree.type) {

    return diffChildren(oldTree.children, newTree.children);
```

### 4.4 TCP 三次握手与四次挥手

**考察形式**：握手/挥手过程、为什么需要这样设计。

**核心知识点**：

三次握手建立连接：

```
客户端              服务器              目的
  |                   |              
  |--- SYN=1, seq=x →|  客户端发送连接请求
  |←-- SYN=1,ACK=1 --|  服务器确认并发送连接请求
  |    seq=y,ack=x+1  |
  |                   |
  |--- ACK=1, seq=x+1→|  客户端确认
  |    ack=y+1        |
  |                   |
  连接建立
```

四次挥手关闭连接：

```
客户端              服务器              目的
  |                   |              
  |--- FIN=1, seq=u →|  客户端请求关闭
  |←-- ACK=1, ack=u+1|  服务器确认（可能还有数据要发）
  |                   |
  |←-- FIN=1, ACK=1 --|  服务器也请求关闭
  |    seq=w, ack=u+1  |
  |                   |
  |--- ACK=1, seq=u+1→|  客户端确认
  |    ack=w+1        |
  |                   |
  连接关闭
```

## 六、算法与数据结构高频考点

### 6.1 排序算法

**考察形式**：手写排序算法、时间复杂度分析。

**核心知识点**：

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|-----------|--------|
| 冒泡排序 | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(1) | 稳定 |
| 归并排序 | O(nlogn) | O(n) | 稳定 |
| 快速排序 | O(nlogn) | O(logn) | 不稳定 |
| 堆排序 | O(nlogn) | O(1) | 不稳定 |

```javascript
// 快速排序
function quickSort(arr) {
    if (arr.length <= 1) return arr;
    
    const pivot = arr[0];
    const left = arr.slice(1).filter(x => x < pivot);
    const right = arr.slice(1).filter(x => x >= pivot);
    
    return [...quickSort(left), pivot, ...quickSort(right)];
}

// 归并排序
function mergeSort(arr) {
    if (arr.length <= 1) return arr;
    
    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));
    
    return merge(left, right);
}

function merge(left, right) {
    const result = [];
    let i = 0, j = 0;
    
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            result.push(left[i++]);
        } else {
            result.push(right[j++]);
        }
    }
    
    return result.concat(left.slice(i)).concat(right.slice(j));
}
```

### 6.2 二分查找

```javascript
// 标准二分查找
function binarySearch(arr, target) {
    let left = 0;
    let right = arr.length - 1;
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        
        if (arr[mid] === target) {
            return mid;
        } else if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return -1;
}

// 查找左边界
function leftBound(arr, target) {
    let left = 0;
    let right = arr.length;
    
    while (left < right) {
        const mid = Math.floor((left + right) / 2);
        
        if (arr[mid] < target) {
            left = mid + 1;
        } else {
            right = mid;
        }
    }
    
    return left;
}
```
