## 一、从输入 URL 到页面展示的完整过程

**参考答案：**

1.  **URL 解析**：浏览器判断是搜索关键词还是合法 URL。
2.  **DNS 解析**：
    *   递归查询：浏览器缓存 -> 系统缓存 -> 路由器缓存 -> 本地 DNS 服务器。
    *   迭代查询：根域名服务器 -> 顶级域名服务器 -> 权威域名服务器。
3.  **TCP 连接 (三次握手)**：
    *   客户端 SYN -> 服务端 SYN+ACK -> 客户端 ACK。
4.  **TLS 握手 (HTTPS 必经)**：
    *   Client Hello -> Server Hello (带证书) -> 客户端验签并交换对称密钥。
5.  **发送 HTTP 请求**：构造请求行、请求头。
6.  **服务器响应**：返回响应状态码（如 304、200）、响应体。
7.  **浏览器渲染渲染流水线**：
    *   **Parsing**: 解析 HTML 生成 **DOM**，解析 CSS 生成 **CSSOM**。
    *   **Style**: 合并为 **Render Tree**。
    *   **Layout**: 计算几何位置（回流/Reflow）。
    *   **Layer**: 分层（Layer Tree）。
    *   **Paint**: 生成绘制指令。
    *   **Tiling & Rasterize**: 分块并利用 GPU 进行光栅化。
    *   **Composite**: 合成显示。
