# UnoThree 项目面试专题

> UnoThree 是一个 UNO 牌游戏的多人在线对战平台，采用现代化的 Web 技术栈构建，支持 2D 和 3D 两种渲染模式。项目使用了 Next.js, React, TypeScript, Zustand, React Three Fiber (Three.js), Framer Motion, Socket.io, Web Audio API 等技术。

## 1. 3D 渲染相关面试问题

### 问题 1：React Three Fiber 的原理是什么？它和原生 Three.js 有什么区别？

**考察点：** R3F 架构，React 和 Three.js 的结合

**参考答案：**

#### React Three Fiber 的核心原理

React Three Fiber (R3F) 是 Three.js 的 React 渲染器，它的本质是将 Three.js 的场景图 (Scene Graph) 与 React 的虚拟 DOM (Virtual DOM) 进行了深度整合：

```
┌─────────────────────────────────────────────────────────────────┐
│                    React Three Fiber 架构                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ┌──────────────┐      ┌──────────────┐      ┌─────────────┐ │
│   │   React      │      │     R3F      │      │   Three.js  │ │
│   │   Components │─────▶│   Reconciler │─────▶│   Scene     │ │
│   └──────────────┘      └──────────────┘      └─────────────┘ │
│         │                      │                      │         │
│         │               ┌──────▼──────┐               │         │
│         │               │   Fiber     │               │         │
│         │               │   Tree      │               │         │
│         │               └─────────────┘               │         │
│         │                                             │         │
│         └─────────────────────────────────────────────┘         │
│                        React DOM                                │
└─────────────────────────────────────────────────────────────────┘
```

**核心实现机制：**

1. **Fiber 架构**：R3F 使用 React Fiber (协程) 来调度 Three.js 的渲染任务
   - 每个 3D 组件对应一个 Fiber 节点
   - Fiber 节点负责创建、更新、销毁 Three.js 对象

2. **声明式 API**：将 Three.js 的命令式操作转化为声明式组件

```typescript
// 原生 Three.js (命令式)
const geometry = new THREE.BoxGeometry(1, 1, 1);
const material = new THREE.MeshStandardMaterial({ color: 'red' });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

// React Three Fiber (声明式)
function Cube() {
  return (
    <mesh>
      <boxGeometry args={[1, 1, 1]} />
      <meshStandardMaterial color="red" />
    </mesh>
  );
}
```

3. **生命周期集成**：R3F 提供了与 React 生命周期同步的 Hooks
   - `useFrame`：每帧执行（替代 requestAnimationFrame）
   - `useThree`：获取 Three.js 上下文
   - `useLoader`：异步加载资源

4. **事件系统**：R3F 将 DOM 事件映射到 3D 对象
   - 支持 onClick、onPointerOver 等事件
   - 自动处理射线检测 (Raycasting)

#### R3F 与原生 Three.js 的对比

| 特性 | 原生 Three.js | React Three Fiber |
|------|---------------|-------------------|
| **编程范式** | 命令式 | 声明式 |
| **状态管理** | 手动管理 | 借助 React 状态 |
| **组件复用** | 工厂函数/类 | React 组件 |
| **性能控制** | 精细粒度 | 需要优化 |
| **学习曲线** | 陡峭 | 较平滑 |
| **与 React 生态集成** | 困难 | 自然融合 |

**R3F 的优势：**
- 代码更简洁，声明式写法易于维护
- 自动内存管理（Fiber 节点生命周期）
- 与 React DevTools 集成，便于调试
- 社区生态丰富（@react-three/drei, @react-three/fiber）

**R3F 的局限：**
- 首屏性能开销较大（Fiber 协调器）
- 极端性能场景需要绕过 React 更新
- 需要理解 React Fiber 的更新机制

**可能的追问：**

1. **R3F 的性能瓶颈在哪里？如何优化？**
   - 答案要点：Fiber 协调开销、频繁的状态更新导致的重渲染、Three.js 对象的创建/销毁开销。优化方案包括：使用 `useMemo` 缓存对象、使用 `useFrame` 直接操作 ref 避免 React 更新、使用 `React.memo` 包装子组件、启用 `frameloop="demand"` 按需渲染。

2. **R3F 如何处理内存泄漏？**
   - 答案要点：Three.js 对象需要在组件卸载时手动 dispose，R3F 的 `<Canvas>` 组件会自动清理其创建的对象，但自定义的几何体和材质需要手动调用 `.dispose()`。可以使用 `@react-three/drei` 的 `useGLTF` 等加载器，它们会自动管理内存。

3. **R3F 和原生 Three.js 混用时需要注意什么？**
   - 答案要点：确保在同一个 render loop 中操作、避免跨框架修改共享对象、正确处理坐标系转换。需要理解 R3F 的坐标系与原生 Three.js 是一致的，主要差异在于更新方式。

#### 针对 100 张卡牌同时动画的优化方案

**1. InstancedMesh（实例化网格）- 核心方案**

当有多个月体对象使用相同几何体和材质时，使用 InstancedMesh 可以将 Draw Call 从 100 次降低到 1 次：

```typescript
import { useRef, useMemo } from 'react';
import * as THREE from 'three';

function CardInstances({ cards }: { cards: CardData[] }) {
  const meshRef = useRef<THREE.InstancedMesh>(null);

  // 创建实例化网格
  const { geometry, material } = useMemo(() => {
    return {
      geometry: new THREE.BoxGeometry(2.5, 3.5, 0.18),
      material: new THREE.MeshStandardMaterial({ color: 'white' })
    };
  }, []);

  // 使用 useFrame 直接操作矩阵，避免 React 更新
  useFrame(() => {
    if (!meshRef.current) return;

    const dummy = new THREE.Object3D();

    cards.forEach((card, index) => {
      dummy.position.set(card.x, card.y, card.z);
      dummy.rotation.set(card.rx, card.ry, card.rz);
      dummy.scale.setScalar(card.scale);
      dummy.updateMatrix();

      meshRef.current!.setMatrixAt(index, dummy.matrix);
    });

    meshRef.current.instanceMatrix.needsUpdate = true;
  });

  return (
    <instancedMesh
      ref={meshRef}
      args={[geometry, material, cards.length]}
    />
  );
}
```

**2. 对象池（Object Pool）- 动画对象复用**

```typescript
class CardAnimationPool {
  private pool: THREE.Mesh[] = [];
  private activeCards: Set<number> = new Set();

  constructor(private scene: THREE.Scene) {
    // 预创建 150 个卡牌对象（应对最大需求 + 缓冲）
    for (let i = 0; i < 150; i++) {
      const card = this.createCard();
      card.visible = false;
      this.pool.push(card);
      this.scene.add(card);
    }
  }

  private createCard(): THREE.Mesh {
    const geometry = new THREE.BoxGeometry(2.5, 3.5, 0.18);
    const material = new THREE.MeshStandardMaterial({ color: 'white' });
    const mesh = new THREE.Mesh(geometry, material);
    return mesh;
  }

  // 获取可用对象
  acquire(): THREE.Mesh | null {
    for (let i = 0; i < this.pool.length; i++) {
      if (!this.activeCards.has(i)) {
        this.activeCards.add(i);
        this.pool[i].visible = true;
        return this.pool[i];
      }
    }
    return null; // 池已满
  }

  // 归还对象
  release(mesh: THREE.Mesh) {
    const index = this.pool.indexOf(mesh);
    if (index !== -1) {
      this.activeCards.delete(index);
      this.pool[index].visible = false;
    }
  }
}
```

**3. 使用 useFrame 直接操作，避免 React 渲染周期**

```typescript
// 错误做法：每帧触发 React 重渲染
useFrame(() => {
  setCards(prev => prev.map(card => updatePosition(card)));
});

// 正确做法：直接操作 ref，不触发 React 更新
function CardAnimator({ cards }: { cards: CardData[] }) {
  const groupRef = useRef<THREE.Group>(null);
  const cardRefs = useRef<THREE.Mesh[]>([]);

  useFrame((state, delta) => {
    // 直接修改 Three.js 对象属性
    cardRefs.current.forEach((card, i) => {
      if (card && cards[i]) {
        // 物理模拟：Lerp 平滑移动
        card.position.x += (cards[i].targetX - card.position.x) * 0.1;
        card.position.y += (cards[i].targetY - card.position.y) * 0.1;
        card.rotation.z += cards[i].rotationSpeed * delta;
      }
    });
  });

  return (
    <group ref={groupRef}>
      {cards.map((card, i) => (
        <mesh
          key={card.id}
          ref={el => cardRefs.current[i] = el!}
        >
          <boxGeometry args={[2.5, 3.5, 0.18]} />
          <meshStandardMaterial color={card.color} />
        </mesh>
      ))}
    </group>
  );
}
```

**4. 性能监控与动态降级**

```typescript
function PerformanceMonitor() {
  const { gl, scene, camera } = useThree();
  const statsRef = useRef({ fps: 60, frameTime: 16 });

  useFrame((state, delta) => {
    const fps = 1 / delta;
    statsRef.current.fps = fps;
    statsRef.current.frameTime = delta * 1000;

    // 动态降级策略
    if (fps < 30 && delta > 0.033) {
      // 降低动画精度
      enableLowQualityMode();
    }
    if (fps < 20) {
      // 减少同时动画的卡牌数量
      reduceActiveAnimations();
    }
  });

  return null;
}
```

**5. 其他关键优化手段**

| 优化手段 | 说明 | 效果 |
|---------|------|------|
| **视锥体剔除 (Frustum Culling)** | 不渲染相机外的对象 | 减少 20-50% Draw Calls |
| **LOD (Level of Detail)** | 远距离使用低模 | 降低顶点处理 |
| **纹理压缩** | 使用 Basis/ETC2 格式 | 减少显存占用 |
| **延迟加载** | 按需加载卡牌纹理 | 减少首屏时间 |
| **合并几何体** | 静态对象合并 | 减少 Draw Calls |
| **Shader 优化** | 简化复杂效果 | 提升 GPU 性能 |

**可能的追问：**

1. **如何检测 3D 场景的性能瓶颈？**
   - 答案要点：使用 Chrome DevTools 的 Performance 面板、Three.js 内置的 Stats.js、R3F 的 `<PerformanceMonitor>` 组件、GPU 时序分析。通过分析帧时间组成（JavaScript 执行、样式计算、渲染）来定位瓶颈。

2. **100 张卡牌的动画如何实现平滑的抛物线轨迹？**
   - 答案要点：使用贝塞尔曲线或二次方程计算轨迹，在 useFrame 中逐帧更新位置。可以使用 tween.js 或自定义插值函数实现。关键是要控制动画的起止点、中点高度和时间参数。

3. **移动端和桌面端的性能策略有什么不同？**
   - 答案要点：移动端需要更保守的 Draw Call 限制、更低的纹理分辨率、更简单的 Shader。需要检测设备能力（GPU 型号、内存）并动态调整渲染策略。可以使用 `<drei>` 的 `PerformanceMonitor` 组件自动降级。

**4. 纹理缓存与复用**

```typescript
// 全局纹理缓存
const textureCache = new Map<string, THREE.Texture>();

function useCardTexture(cardId: string) {
  return useMemo(() => {
    if (textureCache.has(cardId)) {
      return textureCache.get(cardId)!;
    }

    const texture = loadTexture(`/textures/cards/${cardId}.png`);
    textureCache.set(cardId, texture);
    return texture;
  }, [cardId]);
}

// 清理未使用的纹理
function cleanupTextureCache() {
  const maxSize = 100 * 1024 * 1024; // 100MB 限制
  let currentSize = 0;

  textureCache.forEach((texture) => {
    currentSize += texture.image?.width * texture.image?.height * 4;
  });

  if (currentSize > maxSize) {
    // LRU 清理策略
    const entries = Array.from(textureCache.entries());
    entries.sort((a, b) => a[1].lastUsed - b[1].lastUsed);

    const toRemove = entries.slice(0, entries.length * 0.3);
    toRemove.forEach(([key, texture]) => {
      texture.dispose();
      textureCache.delete(key);
    });
  }
}
```

**可能的追问：**

1. **如何实现卡牌背面的动态翻转效果？**
   - 答案要点：使用 TWEEN 或手动在 useFrame 中插值旋转角度。当旋转到 90 度时切换材质或纹理。可以通过改变 mesh 的 scale.x 从 1 到 -1 来实现翻转视觉效果。

2. **加载大量纹理时如何显示加载进度？**
   - 答案要点：使用 R3F 的 `useProgress` Hook 配合 `@react-three/drei` 的 `<Progress />` 组件，或者自定义加载界面监听 `useProgress()` 返回的 progress 值。

3. **如何在 WebGL 内存受限的情况下优化纹理？**
   - 答案要点：使用纹理压缩格式（KTX2、Basis）、动态调整纹理分辨率、使用低分辨率占位图 + 渐进式加载、实施 LRU 缓存策略限制最大显存占用。

```typescript
// 状态机类实现
class GameStateMachine {
  private state: GameState;
  private history: GameState[] = [];

  // 状态转换表
  private readonly transitions: Record<string, TransitionRule[]> = {
    [GameStatus.WAITING]: [
      { event: 'startGame', from: GameStatus.WAITING, to: GameStatus.PLAYING, guard: this.canStartGame.bind(this) }
    ],
    [GameStatus.PLAYING]: [
      { event: 'roundEnd', from: GameStatus.PLAYING, to: GameStatus.ROUND_END, guard: this.isRoundEnd.bind(this) },
      { event: 'gameOver', from: GameStatus.PLAYING, to: GameStatus.GAME_OVER, guard: this.isGameOver.bind(this) }
    ],
    [GameStatus.ROUND_END]: [
      { event: 'startNextRound', from: GameStatus.ROUND_END, to: GameStatus.PLAYING, guard: this.canStartNextRound.bind(this) },
      { event: 'endGame', from: GameStatus.ROUND_END, to: GameStatus.GAME_OVER }
    ],
    [GameStatus.GAME_OVER]: [
      { event: 'reset', from: GameStatus.GAME_OVER, to: GameStatus.WAITING }
    ]
  };

  // 执行状态转换
  transition(event: string, payload?: any): Result<GameState, GameError> {
    const rules = this.transitions[this.state.status] || [];
    const matchedRule = rules.find(rule =>
      rule.event === event &&
      (!rule.guard || rule.guard(this.state, payload))
    );

    if (!matchedRule) {
      return Err(new GameError(`Invalid transition: ${event} from ${this.state.status}`));
    }

    // 保存历史状态（用于撤销）
    this.history.push(cloneDeep(this.state));

    // 执行转换
    const newState = this.applyTransition(this.state, matchedRule, payload);
    this.state = newState;

    return Ok(newState);
  }

  private applyTransition(state: GameState, rule: TransitionRule, payload: any): GameState {
    switch (rule.to) {
      case GameStatus.PLAYING:
        return this.initializeNewGame(state, payload);
      case GameStatus.ROUND_END:
        return this.calculateRoundResults(state);
      case GameStatus.GAME_OVER:
        return this.calculateFinalResults(state);
      default:
        return state;
    }
  }
}
```

```typescript
    },
    [TurnStatus.NEXT_PLAYER]: {
      onComplete: TurnStatus.WAITING
    }

    const nextState = transitions[event];

    if (!nextState) {

    const previousState = this.state;
    this.state = nextState;

    this.executeSideEffects(previousState, event);

    return this.state;
```

```typescript
class MediumAIStrategy implements AIStrategy {
  getThinkTime(): number {
    return 1500 + Math.random() * 1000; // 1.5-2.5秒
  }

  async makeDecision(gameState: GameState, aiPlayerId: string): Promise<AIDecision> {
    const hand = this.getPlayerHand(gameState, aiPlayerId);
    const playableCards = this.getPlayableCards(hand, gameState);

    if (playableCards.length === 0) {
      return { action: 'drawCard', confidence: 1, reasoning: 'No playable cards' };
    }

    // 启发式评分函数
    const scoredCards = playableCards.map(card => ({
      card,
      score: this.evaluateCard(card, hand, gameState)
    }));

    // 按评分排序，选择最高分
    scoredCards.sort((a, b) => b.score - a.score);
    const selectedCard = scoredCards[0].card;

    // 检查是否应该喊 UNO
    const shouldShoutUno = hand.length === 2;

    return {
      action: 'playCard',
      cardId: selectedCard.id,
      confidence: 0.7,
      reasoning: `Selected card with score ${scoredCards[0].score}`
    };
  }

  private evaluateCard(card: Card, hand: Card[], gameState: GameState): number {
    let score = 0;

    // 优先出功能牌（攻击性）
    if (card.type === CardType.DRAW_TWO) score += 30;
    if (card.type === CardType.REVERSE) score += 25;
    if (card.type === CardType.SKIP) score += 25;
    if (card.type === CardType.WILD_DRAW_FOUR) score += 40;

    // 保留万能牌到最后
    if (card.type === CardType.WILD) score -= 20;
    if (card.type === CardType.WILD_DRAW_FOUR) score -= 30;

    // 优先出同色牌中数量最多的颜色
    const colorCount = hand.filter(c => c.color === card.color).length;
    if (colorCount > 3) score += colorCount * 5;

    // 攻击下家（手牌少时）
    const nextPlayerHand = this.getNextPlayerHand(gameState);
    if (nextPlayerHand.length <= 3 &&
        [CardType.SKIP, CardType.REVERSE, CardType.DRAW_TWO].includes(card.type)) {
      score += 20;
    }

    return score;
  }

  chooseColor(hand: Card[]): CardColor {
    // 选择手牌中数量最多的颜色
    const colorCounts = this.countColors(hand);
    return Object.entries(colorCounts)
      .sort(([,a], [,b]) => b - a)[0][0] as CardColor;
  }
}

// 困难难度：基于博弈论的最优策略
class HardAIStrategy implements AIStrategy {
  getThinkTime(): number {
    return 2500 + Math.random() * 1000; // 2.5-3.5秒
  }

  async makeDecision(gameState: GameState, aiPlayerId: string): Promise<AIDecision> {
    const hand = this.getPlayerHand(gameState, aiPlayerId);
    const playableCards = this.getPlayableCards(hand, gameState);

    if (playableCards.length === 0) {
      return { action: 'drawCard', confidence: 1, reasoning: 'No playable cards' };
    }

    // 使用 Minimax 算法寻找最优解
    const bestMove = this.minimax(
      gameState,
      aiPlayerId,
      hand,
      3  // 搜索深度3层
    );

    return {
      action: 'playCard',
      cardId: bestMove.card.id,
      confidence: 0.95,
      reasoning: `Minimax selected: ${bestMove.reasoning}`
    };
  }

  private minimax(
    gameState: GameState,
    playerId: string,
    hand: Card[],
    depth: number
  ): { card: Card; score: number; reasoning: string } {

    if (depth === 0 || hand.length === 0) {
      return {
        card: hand[0],
        score: this.evaluateGameState(gameState, playerId),
        reasoning: 'Leaf node'
      };
    }

    const playableCards = this.getPlayableCards(hand, gameState);
    let bestScore = -Infinity;
    let bestCard = playableCards[0];
    let reasoning = '';

    for (const card of playableCards) {
      // 模拟出这张牌后的状态
      const simulatedState = this.simulatePlayCard(gameState, card);

      // 递归评估（假设对手也会做最优决策）
      const score = this.minimaxValue(
        simulatedState,
        playerId,
        depth - 1,
        false  // opponent's turn
      );

      if (score > bestScore) {
        bestScore = score;
        bestCard = card;
        reasoning = `Selected ${card.type} for score ${score}`;
      }
    }

    return { card: bestCard, score: bestScore, reasoning };
  }

  // 评估游戏状态的启发式函数
  private evaluateGameState(gameState: GameState, playerId: string): number {
    let score = 0;

    const myHand = this.getPlayerHand(gameState, playerId);
    score -= myHand.length * 100;  // 手牌越少越好

    // 检查是否即将获胜
    if (myHand.length === 0) score += 10000;

    // 考虑对手手牌数量
    const opponents = gameState.players.filter(p => p.id !== playerId);
    const opponentCards = opponents.map(p => p.hand.length);
    score += Math.min(...opponentCards) * 50;  // 对手牌少对我有利

    // 当前玩家是我
    const currentPlayer = gameState.players[gameState.currentPlayerIndex];
    if (currentPlayer.id === playerId) {
      score += 20;  // 轮到我出牌是有利的
    }

    return score;
  }

  chooseColor(hand: Card[]): CardColor {
    // 选择能最快出完手牌的颜色
    // 需要结合后续状态评估
    return this.optimalColorChoice(hand);
  }
}
```

```typescript
// 出牌验证服务
@Injectable()
class CardPlayValidator {

  // 验证是否可以出这张牌
  validatePlay(
    card: Card,
    playerHand: Card[],
    gameState: GameState
  ): Result<void, ValidationError> {

    // 1. 验证玩家拥有这张牌
    if (!playerHand.some(c => c.id === card.id)) {
      return Err(new ValidationError('Player does not have this card'));
    }

    // 2. 验证是否是万能牌
    if (card.color === CardColor.WILD) {
      return Ok();  // 万能牌总是可以出
    }

    // 3. 验证颜色匹配
    const topCard = gameState.discardPile[gameState.discardPile.length - 1];
    if (!this.isColorMatch(card, topCard, gameState.currentColor)) {
      return Err(new ValidationError('Color does not match'));
    }

    // 4. 验证类型/数值匹配（对于非万能牌）
    if (!this.isTypeOrValueMatch(card, topCard)) {
      return Err(new ValidationError('Type or value does not match'));
    }

    return Ok();
  }

  // 颜色匹配判断
  private isColorMatch(
    card: Card,
    topCard: Card,
    currentColor: CardColor
  ): boolean {
    // 万能牌可以匹配任何颜色
    if (card.color === CardColor.WILD) return true;

    // 顶牌是万能牌时，使用当前生效颜色
    const effectiveTopColor = topCard.color === CardColor.WILD
      ? currentColor
      : topCard.color;

    return card.color === effectiveTopColor;
  }

  // 类型/数值匹配判断
  private isTypeOrValueMatch(card: Card, topCard: Card): boolean {
    // 如果顶牌是数字牌，检查数值
    if (topCard.type === CardType.NUMBER && card.type === CardType.NUMBER) {
      return card.value === topCard.value;
    }

    // 功能牌类型匹配
    return card.type === topCard.type;
  }

  // 获取玩家所有可出的牌
  getPlayableCards(playerHand: Card[], gameState: GameState): Card[] {
    return playerHand.filter(card =>
      this.validatePlay(card, playerHand, gameState).isOk()
    );
  }
}
```

```typescript
class CardEffectProcessor {

  async processCardEffect(
    card: Card,
    gameState: GameState,
    chosenColor?: CardColor
  ): Promise<CardEffectResult> {
    const effects: GameEvent[] = [];
    let nextPlayerIndex = gameState.currentPlayerIndex;

    switch (card.type) {
      case CardType.NUMBER:

        break;

      case CardType.SKIP:
        // 跳过下家
        nextPlayerIndex = this.getNextPlayerIndex(gameState, 2);
        effects.push({
          type: 'playerSkipped',
          playerIndex: this.getNextPlayerIndex(gameState, 1)
        });
        break;

      case CardType.REVERSE:
        // 反转方向
        gameState.direction *= -1;
        effects.push({ type: 'directionReversed' });

        if (gameState.players.length === 2) {
          // 双人模式相当于跳过
          nextPlayerIndex = this.getNextPlayerIndex(gameState, 2);
          effects.push({
            type: 'playerSkipped',
            playerIndex: this.getNextPlayerIndex(gameState, 1)
          });
        }
        break;

      case CardType.DRAW_TWO:
        // +2 牌
        gameState.pendingDrawCard += 2;
        nextPlayerIndex = this.getNextPlayerIndex(gameState, 1);
        effects.push({
          type: 'drawTwo',
          amount: 2,
          targetPlayerIndex: nextPlayerIndex
        });
        break;

      case CardType.WILD:
        // 万能牌
        if (!chosenColor) {
          throw new Error('Color must be chosen for wild card');
        }
        gameState.currentColor = chosenColor;
        effects.push({ type: 'colorChanged', color: chosenColor });
        break;

      case CardType.WILD_DRAW_FOUR:
        // +4 牌
        if (!chosenColor) {
          throw new Error('Color must be chosen for wild draw four');
        }
        gameState.pendingDrawCard += 4;
        gameState.currentColor = chosenColor;
        gameState.challengeData = {
          cardId: card.id,
          playerId: gameState.players[gameState.currentPlayerIndex].id,
          canChallenge: true
        };
        effects.push({
          type: 'wildDrawFour',
          amount: 4,
          color: chosenColor
        });
        break;
    }

    return { nextPlayerIndex, effects };
  }

  private getNextPlayerIndex(
    gameState: GameState,
    steps: number
  ): number {
    const playerCount = gameState.players.length;
    let index = gameState.currentPlayerIndex;

    for (let i = 0; i < steps; i++) {
      index = (index + gameState.direction + playerCount) % playerCount;
    }

    return index;
  }
}
```

```typescript
@Injectable()
class RoomManager {
  private rooms: Map<string, Room> = new Map();
  private playerRooms: Map<string, string> = new Map();  // playerId -> roomId

  // 创建房间
  createRoom(options: CreateRoomOptions): Room {
    const room: Room = {
      id: this.generateRoomId(),
      name: options.name || `Room ${Date.now()}`,
      maxPlayers: options.maxPlayers || 10,
      players: [],
      gameState: null,
      status: 'waiting',
      createdAt: new Date(),
      settings: options.settings || {}
    };

    this.rooms.set(room.id, room);
    return room;
  }

  // 加入房间
  joinRoom(roomId: string, player: Player): Result<void, RoomError> {
    const room = this.rooms.get(roomId);

    if (!room) {
      return Err(new RoomError('Room not found'));
    }

    if (room.status !== 'waiting') {
      return Err(new RoomError('Game already started'));
    }

    if (room.players.length >= room.maxPlayers) {
      return Err(new RoomError('Room is full'));
    }

    // 检查玩家是否已在房间中
    if (room.players.some(p => p.id === player.id)) {
      return Err(new RoomError('Player already in room'));
    }

    room.players.push(player);
    this.playerRooms.set(player.id, roomId);

    return Ok(undefined);
  }

  // 离开房间
  leaveRoom(roomId: string, playerId: string): Result<void, RoomError> {
    const room = this.rooms.get(roomId);
    if (!room) return Err(new RoomError('Room not found'));

    room.players = room.players.filter(p => p.id !== playerId);
    this.playerRooms.delete(playerId);

    // 房间为空，清理房间
    if (room.players.length === 0) {
      this.closeRoom(roomId);
    }

    return Ok(undefined);
  }

  // 关闭房间
  private closeRoom(roomId: string): void {
    const room = this.rooms.get(roomId);
    if (room) {
      room.status = 'closed';
      // 延迟清理，允许重连
      setTimeout(() => {
        if (this.rooms.get(roomId)?.status === 'closed') {
          this.rooms.delete(roomId);
        }
      }, 5 * 60 * 1000); // 5分钟后彻底清理
    }
  }
}
```

```typescript
  handleConnection(client: Socket) {

    client.conn.on('heartbeat', () => {

  handleDisconnect(client: Socket) {
    const playerId = this.getPlayerId(client);
    if (!playerId) return;

    const roomId = this.roomManager.getPlayerRoom(playerId);
    if (!roomId) return;

    // 记录断线信息

      disconnectedAt: Date.now(),
      attemptCount: 0
    });

    // 通知房间内其他玩家
    this.server.to(roomId).emit('playerDisconnected', {
      playerId,
      reason: 'connection_lost'
    });

    // 启动重连超时计时器
    this.startReconnectTimer(playerId);
  }

  // 启动重连计时器
  private startReconnectTimer(playerId: string) {
    setTimeout(() => {
      const disconnectInfo = this.disconnectedPlayers.get(playerId);

      if (disconnectInfo) {
        // 重连超时，视为真正离开
        this.handlePlayerLeave(playerId, disconnectInfo.roomId);
        this.disconnectedPlayers.delete(playerId);
      }
    }, this.RECONNECT_TIMEOUT);
  }

  // 处理重连
  @SubscribeMessage('reconnect')
  handleReconnect(
    client: Socket,
    payload: ReconnectPayload
  ): Result<ReconnectResult, Error> {
    const { playerId, roomId, token } = payload;

    // 验证重连凭据
    const disconnectInfo = this.disconnectedPlayers.get(playerId);
    if (!disconnectInfo) {
      return Err(new Error('No disconnection record found'));
    }

    if (disconnectInfo.roomId !== roomId) {
      return Err(new Error('Room mismatch'));
    }

    if (disconnectInfo.attemptCount >= this.MAX_RECONNECT_ATTEMPTS) {
      return Err(new Error('Max reconnect attempts reached'));
    }

    // 验证 token（防冒用）
    const tokenValid = this.validateReconnectToken(playerId, token);
    if (!tokenValid) {
      disconnectInfo.attemptCount++;
      return Err(new Error('Invalid reconnect token'));
    }

    // 恢复连接
    const room = this.roomManager.getRoom(roomId);
    const player = room?.players.find(p => p.id === playerId);

    if (player) {
      // 更新玩家的 socketId
      player.socketId = client.id;
      client.join(roomId);

      // 删除断线记录
      this.disconnectedPlayers.delete(playerId);

      // 发送完整游戏状态
      return Ok({
        success: true,
        gameState: room?.gameState,
        playerId
      });
    }

    return Err(new Error('Player not found in room'));
  }

  // 处理玩家彻底离开
  private async handlePlayerLeave(playerId: string, roomId: string) {
    const room = this.roomManager.getRoom(roomId);
    if (!room) return;

    // 如果游戏进行中，玩家的牌作废
    if (room.status === 'playing') {
      await this.gameService.handlePlayerAbandoned(roomId, playerId);
    }

    // 移除玩家
    this.roomManager.leaveRoom(roomId, playerId);

    // 广播
    this.server.to(roomId).emit('playerLeft', {
      playerId,
      reason: 'abandoned'
    });
  }
}
```

```typescript
// 乐观更新管理器
class OptimisticUpdateManager {
  private pendingOperations: Map<string, Operation> = new Map();
  private operationId = 0;

  // 发起操作（乐观更新）
  async performAction<T>(
    actionType: string,
    payload: any,
    applyOptimistic: (state: GameState) => GameState,
    executeServer: (payload: any) => Promise<T>,
    rollback: () => void
  ): Promise<T> {
    const operationId = `op_${++this.operationId}`;
    const timestamp = Date.now();

    // 1. 应用乐观更新
    const optimisticState = applyOptimistic(getGlobalState());
    updateLocalState(optimisticState);

    // 记录操作（用于回滚）
    this.pendingOperations.set(operationId, {
      id: operationId,
      type: actionType,
      timestamp,
      payload,
      optimisticState: cloneDeep(optimisticState),
      rollback
    });

    try {
      // 2. 发送到服务器
      const result = await executeServer(payload);

      // 3. 成功，移除待确认操作
      this.pendingOperations.delete(operationId);

      return result;

    } catch (error) {
      // 4. 失败，回滚状态
      this.rollbackOperation(operationId);
      throw error;
    }
  }

  // 处理服务器状态推送（版本比对）
  handleServerState(serverState: GameState, serverVersion: StateVersion) {
    const localVersion = getCurrentVersion();

    if (serverVersion.version > localVersion.version) {
      // 服务器版本更新，检查是否有未确认的操作冲突

      const conflictingOps = Array.from(this.pendingOperations.values())
        .filter(op => op.timestamp > serverVersion.timestamp);

      if (conflictingOps.length > 0) {
        // 有冲突，需要回滚这些操作
        conflictingOps.forEach(op => {
          this.rollbackOperation(op.id);
        });
      }

      // 应用服务器状态
      updateLocalState(serverState);
    } else if (serverVersion.version === localVersion.version) {
      // 版本一致，确认本地操作
      // 可选：精细化差异更新
    }
    // 版本低于本地，说明是旧状态推送，忽略
  }

  private rollbackOperation(operationId: string) {
    const op = this.pendingOperations.get(operationId);
    if (op) {
      op.rollback();
      this.pendingOperations.delete(operationId);
    }
  }
}
```

```typescript
async function playCard(cardId: string, chosenColor?: CardColor) {

  // 获取当前状态
  const currentState = gameStore.getState();

  // 1. 乐观更新：移除手牌，添加到弃牌堆

    draft.version++;
  });

  // 应用乐观状态
  gameStore.setState(optimisticState);

  // 启动动画（独立于状态）
  animationController.playCardAnimation(cardId);

  try {

    const response = await socket.emitWithAck('playCard', {

      color: chosenColor,
      clientVersion: optimisticState.version,
      operationId
    });

      gameStore.setState(currentState);
      showError(response.error);
      return;

    if (response.state) {

      gameStore.setState(response.state);
```

```typescript
  private readonly LATENCY_SAMPLE_SIZE = 10;

  // 估算网络延迟
  estimateLatency(): number {
    if (this.latencyHistory.length === 0) return 0;

    // 使用移动平均
    const sum = this.latencyHistory.reduce((a, b) => a + b, 0);
    return sum / this.latencyHistory.length;
  }

  getAdaptiveTimeout(): number {
    const latency = this.estimateLatency();
    const baseTimeout = 5000; // 基础超时 5 秒

    // 超时时间 = 基础 + 延迟 * 3（p99 预估）
    return baseTimeout + latency * 3;
  }

  adjustAnimation(animation: AnimationConfig): AnimationConfig {
    const latency = this.estimateLatency();

    if (latency > 500) {
      // 高延迟：减少动画细节，加快速度
      return {
        ...animation,
        duration: animation.duration * 0.8,
        frameskip: true
      };
    }

    return animation;
  }

  // 延迟补偿（客户端预测）
  compensateLatency(playerAction: PlayerAction): CompensatedAction {
    const latency = this.estimateLatency();

    const predictedServerTime = Date.now() + latency;

    return {
      ...playerAction,
      serverTimestamp: predictedServerTime,
      // 预测执行结果
      predictedResult: this.predictResult(playerAction)
    };
```

```typescript
  type: 'playCard' | 'drawCard' | 'shoutUno' | 'turnChange';
  timestamp: number;
  duration: number;  // 预计持续时间
  sync: boolean;    // 是否需要 2D/3D 同步

// 动画协调器
class AnimationCoordinator {
  private eventQueue: AnimationEvent[] = [];
  private isAnimating = false;
  private animationController: AbortController | null = null;

  // 触发同步动画
  async triggerSyncAnimation(event: AnimationEvent) {
    // 1. 更新队列
    this.eventQueue.push(event);

    if (this.isAnimating) {
      return; // 等待当前动画完成
    }

    // 2. 开始动画序列
    await this.processEventQueue();
  }

  private async processEventQueue() {
    while (this.eventQueue.length > 0) {
      const event = this.eventQueue.shift()!;
      this.isAnimating = true;

      try {
        // 并行执行 2D 和 3D 动画
        await Promise.all([
          this.play2DAnimation(event),
          this.play3DAnimation(event)
        ]);
      } catch (error) {
        console.error('Animation error:', error);
      }

      this.isAnimating = false;
    }
  }

  // 2D 动画（Framer Motion）
  private play2DAnimation(event: AnimationEvent): Promise<void> {
    return new Promise((resolve) => {
      // 使用 Framer Motion 的 API 触发动画
      const animationId = `2d_${event.id}`;

      // 发送全局动画事件
      window.dispatchEvent(new CustomEvent('animation:start', {
        detail: { animationId, event }
      }));

      // 模拟动画完成
      setTimeout(() => {
        window.dispatchEvent(new CustomEvent('animation:complete', {
          detail: { animationId, event }
        }));
        resolve();
      }, event.duration);
    });
  }

  // 3D 动画（R3F）
  private play3DAnimation(event: AnimationEvent): Promise<void> {
    return new Promise((resolve) => {
      // 通过 Context 或 Zustand 触发动画
      animationStore.getState().trigger3DAnimation(event);

      // 等待动画完成信号
      const handleComplete = (e: Event) => {
        const detail = (e as CustomEvent).detail;
        if (detail.animationId === `3d_${event.id}`) {
          window.removeEventListener('3dAnimation:complete', handleComplete);
          resolve();
        }
      };

      window.addEventListener('3dAnimation:complete', handleComplete);
    });
  }
}
```

```typescript
function cubicBezier(
  p0: Vector2,
  p1: Vector2,
  p2: Vector2,
  p3: Vector2,
  t: number
): Vector2 {
  const u = 1 - t;
  const tt = t * t;
  const uu = u * u;
  const uuu = uu * u;
  const ttt = tt * t;

  return {
    x: uuu * p0.x + 3 * uu * t * p1.x + 3 * u * tt * p2.x + ttt * p3.x,
    y: uuu * p0.y + 3 * uu * t * p1.y + 3 * u * tt * p2.y + ttt * p3.y
  };
}
```

```typescript
  private play2D(cardId: string, start: Position, end: Position): Promise<void> {
    return new Promise((resolve) => {

      const container = document.getElementById('game-container');
      const animElement = createCardFlyElement(cardId, start, end);
      container?.appendChild(animElement);

      // 动画完成后移除
      setTimeout(() => {
        animElement.remove();
        resolve();
      }, 500);
    });
  }

  private play3D(cardId: string, start: Position3D, end: Position3D): Promise<void> {
    return new Promise((resolve) => {
      // 通过 Zustand 或 Context 触发 3D 动画
      animationStore.getState().play3DFly({

        startPos: start,
        endPos: end,
        onComplete: resolve
```

```typescript
      const height = window.innerHeight;

      // 检测触控能力
      const isTouch = 'ontouchstart' in window ||
                      navigator.maxTouchPoints > 0;

      let deviceType: DeviceType;
      if (height < 600) {
        deviceType = 'mobile';
      } else if (width < 768) {
        deviceType = 'mobile';
      } else if (width < 1024) {
        deviceType = 'tablet';
      } else {

      const responsiveConfig = generateConfig(deviceType, orientation, isTouch);

      setConfig(responsiveConfig);
```

```typescript
export const useCardInteraction = (
  cardId: string,
  onTap: () => void,
  onLongPress?: () => void
) => {
  const [isPressed, setIsPressed] = useState(false);
  const longPressTimer = useRef<NodeJS.Timeout>();
  const LONG_PRESS_DURATION = 500; // 长按阈值

  const handlePointerDown = useCallback((e: React.PointerEvent) => {
    e.preventDefault();
    setIsPressed(true);

    // 开始长按计时
    if (onLongPress) {
      longPressTimer.current = setTimeout(() => {
        onLongPress();
        setIsPressed(false);
      }, LONG_PRESS_DURATION);
    }
  }, [onLongPress]);

  const handlePointerUp = useCallback((e: React.PointerEvent) => {
    e.preventDefault();
    setIsPressed(false);

    // 清除长按计时器
    if (longPressTimer.current) {
      clearTimeout(longPressTimer.current);
      longPressTimer.current = undefined;
    }

    // 触发点击（如果没有触发长按）
    if (!longPressTimer.current) {
      onTap();
    }
  }, [onTap]);

  const handlePointerCancel = useCallback(() => {
    setIsPressed(false);
    if (longPressTimer.current) {
      clearTimeout(longPressTimer.current);
    }
  }, []);

  return {
    // 支持鼠标和触控
    onPointerDown: handlePointerDown,
    onPointerUp: handlePointerUp,
    onPointerCancel: handlePointerCancel,
    onPointerLeave: handlePointerCancel,

    // 状态
    isPressed,

    // 触摸参数（优化触摸体验）
    style: {
      touchAction: 'none',  // 防止浏览器默认手势
```

```typescript
// 手势识别 Hook
export const useGesture = (
  onSwipe?: (direction: 'left' | 'right' | 'up' | 'down') => void,
  onPinch?: (scale: number) => void
) => {
  const touchStart = useRef<{ x: number; y: number }>();
  const initialDistance = useRef<number>();

  const handleTouchStart = useCallback((e: React.TouchEvent) => {
    if (e.touches.length === 1) {
      touchStart.current = {
        x: e.touches[0].clientX,
        y: e.touches[0].clientY
      };
    } else if (e.touches.length === 2) {
      initialDistance.current = getDistance(
        e.touches[0],
        e.touches[1]
      );
    }
  }, []);

  const handleTouchEnd = useCallback((e: React.TouchEvent) => {
    if (e.changedTouches.length === 1 && touchStart.current) {
      const touch = e.changedTouches[0];
      const dx = touch.clientX - touchStart.current.x;
      const dy = touch.clientY - touchStart.current.y;

      // 判断滑动方向
      if (Math.abs(dx) > 50 || Math.abs(dy) > 50) {
        if (Math.abs(dx) > Math.abs(dy)) {
          onSwipe?.(dx > 0 ? 'right' : 'left');
        } else {
          onSwipe?.(dy > 0 ? 'down' : 'up');
        }
      }
    }
  }, [onSwipe]);

  return {
    onTouchStart: handleTouchStart,
    onTouchEnd: handleTouchEnd
  };
};

function getDistance(t1: Touch, t2: Touch): number {
  const dx = t1.clientX - t2.clientX;
  const dy = t1.clientY - t2.clientY;
  return Math.sqrt(dx * dx + dy * dy);
}
```

1. **如何处理移动端触控的误触问题？**
   - 答案要点：使用 Pointer Events 统一处理。添加点击区域 padding 增大触控目标。实现防抖防止重复触发。区分单击和长按，使用 300ms 延迟判断是点击还是长按。在游戏区域内禁用页面滚动和缩放。

   - 答案要点：使用媒体查询检测平板（768px-1024px）。为平板设计独立的布局模板。可以参考 iPad 的响应式断点。考虑支持分屏显示和 Apple Pencil。触控区域要适当调整，避免过小或过大。

```typescript
// 音频管理器单例
class AudioManager {
  private static instance: AudioManager;
  private audioContext: AudioContext | null = null;
  private masterGain: GainNode | null = null;
  private soundCache: Map<string, AudioBuffer> = new Map();
  private activeSources: Map<string, AudioBufferSourceNode> = new Map();

  private constructor() {}

  static getInstance(): AudioManager {
    if (!AudioManager.instance) {
      AudioManager.instance = new AudioManager();
    }
    return AudioManager.instance;
  }

  // 初始化（需要用户交互触发）
  async init(): Promise<void> {
    if (this.audioContext) return;

    this.audioContext = new AudioContext();

    // 创建主音量控制
    this.masterGain = this.audioContext.createGain();
    this.masterGain.connect(this.audioContext.destination);

    // 恢复音频上下文（iOS 需要）
    if (this.audioContext.state === 'suspended') {
      await this.audioContext.resume();
    }

    // 预加载常用音效
    await this.preloadSounds();
  }

  // 预加载音效
  private async preloadSounds(): Promise<void> {
    const loadPromises = Object.values(SoundEffect).map(async (url) => {
      try {
        const buffer = await this.loadSound(url);
        this.soundCache.set(url, buffer);
      } catch (error) {
        console.warn(`Failed to load sound: ${url}`, error);
      }
    });

    await Promise.all(loadPromises);
  }

  // 加载单个音效
  private async loadSound(url: string): Promise<AudioBuffer> {
    const response = await fetch(url);
    const arrayBuffer = await response.arrayBuffer();
    return this.audioContext!.decodeAudioData(arrayBuffer);
  }

  // 播放音效
  play(
    soundEffect: SoundEffect,
    options: PlayOptions = {}
  ): void {
    if (!this.audioContext || !this.masterGain) {
      console.warn('AudioContext not initialized');
      return;
    }

    const buffer = this.soundCache.get(soundEffect);
    if (!buffer) {
      console.warn(`Sound not loaded: ${soundEffect}`);
      return;
    }

    // 创建音源
    const source = this.audioContext.createBufferSource();
    source.buffer = buffer;

    // 创建增益节点（控制音量和淡入淡出）
    const gainNode = this.audioContext.createGain();

    // 设置音量
    gainNode.gain.value = options.volume || 1;

    // 淡入效果
    if (options.fadeIn) {
      gainNode.gain.setValueAtTime(0, this.audioContext.currentTime);
      gainNode.gain.linearRampToValueAtTime(
        options.volume || 1,
        this.audioContext.currentTime + options.fadeIn
      );
    }

    // 连接节点
    source.connect(gainNode);
    gainNode.connect(this.masterGain);

    // 播放
    source.start(0, options.startTime || 0);

    // 保存活跃音源（用于停止）
    const sourceId = `${soundEffect}_${Date.now()}`;
    this.activeSources.set(sourceId, source);

    // 播放完成自动清理
    source.onended = () => {
      this.activeSources.delete(sourceId);
    };

    // 如果需要淡出
    if (options.fadeOut && options.fadeOutDuration) {
      source.loop = true;

      // 计划淡出
      const fadeOutTime = this.audioContext!.currentTime +
        (options.duration || buffer.duration) - options.fadeOutDuration;

      gainNode.gain.setValueAtTime(
        options.volume || 1,
        fadeOutTime
      );
      gainNode.gain.linearRampToValueAtTime(
        0,
        fadeOutTime + options.fadeOutDuration
      );
    }
  }

  // 停止所有音效
  stopAll(): void {
    this.activeSources.forEach(source => {
      try {
        source.stop();
      } catch (e) {
        // 忽略已停止的错误
      }
    });
    this.activeSources.clear();
  }

  // 设置主音量
  setMasterVolume(volume: number): void {
    if (this.masterGain) {
      this.masterGain.gain.value = Math.max(0, Math.min(1, volume));
    }
  }

  // 清理资源
  dispose(): void {
    this.stopAll();
    this.audioContext?.close();
    this.audioContext = null;
    this.masterGain = null;
  }
}
```

```typescript
// 监听游戏状态自动播放音效
export const useGameSoundEffects = () => {
  const { playCardSound, playDrawSound, playUnoSound, playWinSound, playLoseSound } = useSoundEffects();
  const prevHandSize = useRef<number>(0);
  const prevGameState = useRef<GameState | null>(null);

  const playerId = useGameStore(state => state.playerId);

  useEffect(() => {
    if (!gameState || !prevGameState.current) return;

    const prev = prevGameState.current;
    const current = gameState;

    // 检测游戏开始
    if (prev.status !== GameStatus.PLAYING && current.status === GameStatus.PLAYING) {
      audioManager.play(SoundEffect.GAME_START);
    }

    // 检测回合变化
    if (prev.currentPlayerIndex !== current.currentPlayerIndex) {
      // 如果轮到当前玩家，震动提示
      const currentPlayer = current.players[current.currentPlayerIndex];
      if (currentPlayer?.id === playerId && 'vibrate' in navigator) {
        navigator.vibrate(100);
      }
    }

    // 检测手牌变化（出牌）
    const myPrevHand = prev.players.find(p => p.id === playerId)?.hand.length || 0;
    const myCurrentHand = current.players.find(p => p.id === playerId)?.hand.length || 0;

    if (myCurrentHand < myPrevHand) {
      // 出一张牌
      const topCard = current.discardPile[current.discardPile.length - 1];
      playCardSound(topCard.type);

      // 检测 UNO
      if (myCurrentHand === 1) {
        playUnoSound();
      }
    }

    // 检测摸牌
    if (myCurrentHand > myPrevHand) {
      playDrawSound();
    }

    // 检测游戏结束
    if (current.status === GameStatus.GAME_END) {
      const winner = current.players.find(p => p.hand.length === 0);
      if (winner?.id === playerId) {
        playWinSound();
      } else {
        playLoseSound();
      }
    }

    prevGameState.current = current;
  }, [gameState]);
};
```
