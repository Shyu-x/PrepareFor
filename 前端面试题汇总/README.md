## 面试准备建议

### 阶段一：夯实基础（1-2周）

1. **HTML/CSS（3天）**
   - 盒子模型、BFC、Flexbox、Grid
   - 响应式布局、CSS3新特性
   - 浏览器渲染原理（重排、重绘、合成）

2. **JavaScript（5天）**
   - 数据类型、执行上下文、作用域链
   - 原型链、继承方式
   - 闭包、事件循环
   - 异步编程（Promise、async/await）
   - DOM操作、BOM

3. **手写代码（3天）**
   - 防抖、节流
   - 深拷贝
   - Promise.all
   - 数组方法实现

### 阶段二：框架进阶（1-2周）

1. **React（4天）**
   - 组件化、虚拟DOM、Diff算法
   - Hooks原理、useEffect
   - 状态管理（Zustand/Redux）
   - 性能优化

2. **Vue（3天）**
   - 响应式原理
   - Composition API
   - 组件通信

3. **TypeScript（2天）**
   - 类型系统
   - 泛型
   - 工具类型

### 阶段三：工程与实战（1周）

1. **前端工程化（2天）**
   - Webpack/Vite原理
   - 构建优化

2. **计算机网络（2天）**
   - HTTP/HTTPS
   - TCP三次握手/四次挥手
   - DNS、缓存、跨域

3. **项目经验（2天）**
   - 梳理项目亮点
   - 准备技术方案

- 计算机网络（5分钟）
