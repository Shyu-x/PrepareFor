# 27 单调队列

单调队列用于滑动窗口最大/最小值

---

## 27.1 滑动窗口最大值（LeetCode 239）

```javascript
function maxSlidingWindow(nums, k) {
    const result = [];
    const deque = [];  // 存储索引，保持递减

    for (let i = 0; i < nums.length; i++) {
        // 移除超出窗口的索引
        while (deque.length && deque[0] <= i - k) {
            deque.shift();
        }

        // 移除比当前元素小的索引（保持递减）
        while (deque.length && nums[deque[deque.length - 1]] <= nums[i]) {
            deque.pop();
        }

        deque.push(i);

        // 记录窗口最大值
        if (i >= k - 1) {
            result.push(nums[deque[0]]);
        }
    }
    return result;
}
```

## 27.2 滑动窗口最小值

```javascript
function minSlidingWindow(nums, k) {
    const result = [];
    const deque = [];  // 存储索引，保持递增

    for (let i = 0; i < nums.length; i++) {
        while (deque.length && deque[0] <= i - k) {
            deque.shift();
        }

        while (deque.length && nums[deque[deque.length - 1]] >= nums[i]) {
            deque.pop();
        }

        deque.push(i);

        if (i >= k - 1) {
            result.push(nums[deque[0]]);
        }
    }
    return result;
}
```

---

## 27.3 核心思想

- 维护一个单调递减/递增的队列
- 队列存储索引
- 时间复杂度 O(n)，每个元素最多入队出队一次
