# 25 监控二叉树

## 25.1 监控二叉树（LeetCode 968）

```javascript
/**
 * 状态定义：
 * 0: 无覆盖（需要父节点监控）
 * 1: 有摄像头（当前节点已监控）
 * 2: 有覆盖（被子节点或自己监控）
 */
function minCameraCover(root) {
    let result = 0;

    function dfs(node) {
        if (!node) return 2;  // 空节点视为有覆盖

        const left = dfs(node.left);
        const right = dfs(node.right);

        // 子节点有未覆盖的，需要放摄像头
        if (left === 0 || right === 0) {
            result++;
            return 1;
        }

        // 子节点有摄像头，当前节点被覆盖
        if (left === 1 || right === 1) {
            return 2;
        }

        // 子节点都被覆盖，返回无覆盖
        return 0;
    }

    const status = dfs(root);
    // 根节点未覆盖，需要额外摄像头
    if (status === 0) result++;

    return result;
}
```

---

## 25.2 核心思想

- 后序遍历（自底向上）
- 优先在父节点放置摄像头
- 状态转移：子节点有未覆盖 -> 父节点放摄像头
