### 图片上传与存储

#### 前端上传流程

**文件位置**: `frontend/src/lib/upload.ts`

```typescript
// 上传文件到服务器
export async function uploadFile(
  file: File,
  options: UploadOptions = {}
): Promise<UploadResult> {
  // 验证文件
  const error = validateFile(file, options);
  if (error) {
    throw new Error(error);
  }

  // 创建 FormData
  const formData = new FormData();
  formData.append('file', file);

  // 使用 XMLHttpRequest 以支持上传进度
  return new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest();

    // 上传进度回调
    if (options.onProgress) {
      xhr.upload.addEventListener('progress', (event) => {
        if (event.lengthComputable) {
          const percent = Math.round((event.loaded / event.total) * 100);
          options.onProgress?.(percent);
        }
      });
    }

    // 发送请求到后端
    xhr.open('POST', `${process.env.NEXT_PUBLIC_BACKEND_URL}/uploads`);
    xhr.send(formData);
  });
}
```

**关键点**：
1. 使用 `FormData` + `XMLHttpRequest` 实现带进度上传
2. 文件验证：类型（MIME）、大小（默认 50MB）
3. 上传端点：`POST /uploads`

### 后端文件管理

#### 上传服务层

**文件位置**: `backend/src/uploads/uploads.service.ts`

```typescript
@Injectable()
export class UploadsService {
  private config: UploadConfig;

  constructor(
    @InjectRepository(UploadEntity)
    private uploadRepository: Repository<UploadEntity>,
  ) {
    // 配置上传目录
    this.config = DEFAULT_UPLOAD_CONFIG;
    this.ensureUploadDir();
  }

  // 默认上传配置
  const DEFAULT_UPLOAD_CONFIG: UploadConfig = {
    // Docker 容器中使用 /app/uploads，本地开发使用项目根目录的 uploads 文件夹
    uploadDir: process.env.NODE_ENV === 'production'
      ? '/app/uploads'
      : join(process.cwd(), 'uploads'),
    maxFileSize: 50 * 1024 * 1024, // 50MB
    allowedMimeTypes: [
      'image/jpeg',
      'image/png',
      'image/gif',
      'image/webp',
      'image/svg+xml',
      // ... 其他文件类型
    ],
  };

  // 处理文件上传
  async uploadFile(
    file: Express.Multer.File,
    userId?: string,
  ): Promise<UploadResult> {
    // 验证文件类型和大小
    if (!this.validateFileType(file.mimetype, file.originalname)) {
      throw new ForbiddenException(`不支持的文件类型: ${file.mimetype}`);
    }
    if (!this.validateFileSize(file.size)) {
      throw new ForbiddenException(`文件大小超过限制 (最大 50MB)`);
    }

    // 生成唯一文件名：UUID + 原始扩展名
    const ext = file.originalname.substring(file.originalname.lastIndexOf('.'));
    const storedFilename = `${uuidv4()}${ext}`;
    const filePath = join(this.config.uploadDir, storedFilename);

    // 写入文件到磁盘
    await fs.writeFile(filePath, file.buffer);

    // 保存元数据到数据库
    const uploadEntity = this.uploadRepository.create({
      filename: file.originalname,
      storedFilename,
      mimetype: file.mimetype,
      size: file.size,
      path: filePath,
      userId,
    });

    const savedEntity = await this.uploadRepository.save(uploadEntity);

    return {
      id: savedEntity.id,
      filename: savedEntity.filename,
      mimetype: savedEntity.mimetype,
      size: savedEntity.size,
      url: `/uploads/${savedEntity.id}/file`,
    };
  }
}
```

**文件存储位置**：
- **生产环境（Docker）**: `/app/uploads/{uuid}.{ext}`
- **开发环境**: `{项目根目录}/uploads/{uuid}.{ext}`

```typescript
@Controller('uploads')
export class UploadsController {
  /**
   * 上传文件
   * POST /uploads
   * 使用 Multer 中间件处理 multipart/form-data
   */
  @Post()
  @HttpCode(HttpStatus.CREATED)
  @UseInterceptors(
    FileInterceptor('file', {
      limits: {
        fileSize: 50 * 1024 * 1024, // 50MB
      },
      fileFilter: (req, file, callback) => {
        const allowedTypes = [
          'image/jpeg', 'image/png', 'image/gif',
          'image/webp', 'image/svg+xml',
          // ... video, audio, document, archive
        ];
        if (allowedTypes.includes(file.mimetype)) {
          callback(null, true);
        } else {
          callback(new Error('不支持的文件类型'), false);
        }
      },
    }),
  )
  async uploadFile(
    @UploadedFile() file: Express.Multer.File,
    @Body() dto: UploadDto,
  ): Promise<UploadResult> {
    return this.uploadsService.uploadFile(file, dto.userId);
  }

  /**
   * 获取文件内容
   * GET /uploads/:id/file
   */
  @Get(':id/file')
  async getFile(
    @Param('id') id: string,
    @Res() res: Response,
  ): Promise<void> {
    const { file, buffer } = await this.uploadsService.getFile(id);

    // 设置响应头
    res.set({
      'Content-Type': file.mimetype,
      'Content-Length': file.size,
      'Content-Disposition': `inline; filename="${encodeURIComponent(file.filename)}"`,
      'Cache-Control': 'public, max-age=31536000', // 缓存 1 年
    });

    res.send(buffer);
  }

  /**
   * 删除文件
   * DELETE /uploads/:id
   */
  @Delete(':id')
  async deleteFile(@Param('id') id: string): Promise<void> {
    await this.uploadsService.deleteFile(id);
  }
}
```

```typescript
   * 文件大小 (字节)

   * 文件存储路径 (完整路径)

   * 上传用户 ID (可选，用于权限控制)
   */

   * 文件创建时间

   * 文件更新时间
```

**PostgreSQL 表结构**：

| 字段 | 类型 | 描述 |
|------|------|------|
| id | uuid | 主键，文件唯一标识 |
| filename | varchar(255) | 原始文件名 |
| storedFilename | varchar(255) | 存储文件名 |
| mimetype | varchar(100) | MIME 类型 |
| size | bigint | 文件大小（字节） |
| path | text | 文件系统完整路径 |

#### 音频管理器

**文件位置**: `frontend/src/utils/audioManager.ts`

```typescript
export enum SoundEffect {
  LOGIN = '/sounds/login.mp3',
  MATCH_SUCCESS = '/sounds/match-success.mp3',
  GAME_START = '/sounds/game-start.mp3',
  PLAY_CARD = '/sounds/play-card.mp3',
  SHOUT_UNO = '/sounds/shout-uno.mp3',
  CHALLENGE = '/sounds/challenge.mp3',
  DRAW_CARD = '/sounds/draw-card.mp3',
  WIN = '/sounds/win.mp3',
  LOSE = '/sounds/lose.mp3',
  GAME_OVER = '/sounds/game-over.mp3',
}

class AudioManager {
  private static instance: AudioManager;
  private audioContext: AudioContext | null = null;
  private soundBuffers: Map<string, AudioBuffer> = new Map();
  private volume: number = 0.8;

  constructor() {
    // 兼容 WebKit
    const AudioContextClass = window.AudioContext || window.webkitAudioContext;
    if (AudioContextClass) {
      this.audioContext = new AudioContextClass();
    }
  }

  /**
   * 预加载所有音效资源
   */
  public async preloadAll(): Promise<void> {
    if (!this.audioContext) return;
    const loadPromises = Object.values(SoundEffect).map(
      url => this.loadSound(url)
    );
    await Promise.all(loadPromises);
    console.log('[AudioManager] 所有音效预加载完成');
  }

  /**
   * 播放指定的音效
   */
  public async play(effect: SoundEffect): Promise<void> {
    if (!this.audioContext) return;

    // 某些浏览器要求必须在交互后才能启动 AudioContext
    if (this.audioContext.state === 'suspended') {
      await this.audioContext.resume();
    }

    const buffer = this.soundBuffers.get(effect);
    if (!buffer) {
      // 如果没预加载好，尝试实时加载
      const newBuffer = await this.loadSound(effect);
      if (newBuffer) this.playBuffer(newBuffer);
      return;
    }

    this.playBuffer(buffer);
  }

  /**
   * 加载音频文件
   */
  private async loadSound(url: string): Promise<AudioBuffer | null> {
    if (!this.audioContext) return null;
    if (this.soundBuffers.has(url)) return this.soundBuffers.get(url)!;

    // Next.js 配置了 basePath: '/uno'，所以始终使用 /uno 前缀
    const basePath = '/uno';
    const fullUrl = `${basePath}${url}`;

    try {
      const response = await fetch(fullUrl);
      if (!response.ok) {
        console.error(`[AudioManager] HTTP ${response.status}: ${fullUrl}`);
        return null;
      }
      const arrayBuffer = await response.arrayBuffer();
      const audioBuffer = await this.audioContext.decodeAudioData(arrayBuffer);
      this.soundBuffers.set(url, audioBuffer);
      return audioBuffer;
    } catch (error) {
      console.error(`[AudioManager] 加载音效失败: ${url}`, error);
      return null;
    }
  }

  /**
   * 播放音频缓冲
   */
  private playBuffer(buffer: AudioBuffer): void {
    if (!this.audioContext) return;
    try {
      const source = this.audioContext.createBufferSource();
      const gainNode = this.audioContext.createGain();

      source.buffer = buffer;
      gainNode.gain.value = this.volume;

      source.connect(gainNode);
      gainNode.connect(this.audioContext.destination);
      source.start(0);
    } catch (e) {
      console.error('[AudioManager] 播放音频失败', e);
    }
  }
}
```

**音频管理总结**：
- **存储格式**: MP3 压缩音频
- **存储位置**: `public/sounds/`（静态资源）
- **加载方式**: Web Audio API (`AudioContext`)
- **预加载策略**: 游戏启动时预加载所有音效到 `AudioBuffer`
- **播放机制**: `createBufferSource()` → `createGain()` → `destination`
- **音量控制**: 通过 `GainNode` 控制音量

```tsx
        if (isHoverable) setHovered(true);
      }}
      onPointerOut={() => setHovered(false)}
    >
      {/* 卡牌正面 */}
      <mesh castShadow receiveShadow>
        <boxGeometry args={[2.5, 3.5, 0.18]} />
        <meshStandardMaterial color="#ffffff" roughness={0.3} />
      </mesh>

      {/* 卡牌内容区域 */}
      <group position={[0, 0, 0.1]}>
        <mesh receiveShadow>
          <planeGeometry args={[2.3, 3.3]} />
          <meshStandardMaterial color={cardColor} roughness={0.4} />
        </mesh>

        {/* 卡牌符号/数字 */}
        {!isHidden && (
          <Text color="white" fontSize={1.8} fontWeight="black"
                fontStyle="italic" anchorX="center" anchorY="middle">
            {card.value}
          </Text>
        )}
      </group>

      {/* 卡牌背面 */}
      <group position={[0, 0, -0.1]} rotation={[0, Math.PI, 0]}>
        <mesh receiveShadow>
          <planeGeometry args={[2.3, 3.3]} />
          <meshStandardMaterial color="#0f172a" />
        </mesh>
        <group rotation={[0, 0, Math.PI / 4.5]}>
          <mesh>
            <circleGeometry args={[1.2, 32]} />
            <meshStandardMaterial color="#d63031" />
          </mesh>
          <Text color="white" fontSize={0.6} position={[0, 0, 0.02]}
                fontWeight="black" fontStyle="italic">UNO</Text>
        </group>
      </group>
    </animated.group>
  );
};
```

### 虚拟文件系统

#### ZenFS 架构

**文件位置**: `webenv-os/src/lib/filesystem/zenfs.ts`

```typescript
import { configure, FS, configureFs } from '@zenfs/core';
import IndexedDB from 'indexedfs';
import { MemoryFS } from 'memoryfs';

/**
 * 虚拟文件系统配置
 * 支持两种模式：
 * 1. MemoryFS - 纯内存，速度快但数据不持久化
 * 2. IndexedDB - 持久化到浏览器 IndexedDB
 */
class VirtualFileSystem {
  private fs: FS | null = null;
  private isInitialized = false;

  async initialize(useIndexedDB = true) {
    if (this.isInitialized) return;

    if (useIndexedDB) {
      // 使用 IndexedDB 后端
      const idbfs = new IndexedDB('webenv-fs', {
        storeName: 'files',
      });

      // 配置 ZenFS
      await configure({
        mounts: {
          '/': idbfs,
        },
      });
    } else {
      // 使用内存后端（临时使用）
      await configure({
        mounts: {
          '/': new MemoryFS(),
        },
      });
    }

    // 获取文件系统实例
    this.fs = await configureFs();
    this.isInitialized = true;
  }

  /**
   * 写入文件
   */
  async writeFile(path: string, content: string): Promise<void> {
    if (!this.fs) await this.initialize();

    // 确保目录存在
    const dir = path.dirname(path);
    await this.fs.promises.mkdir(dir, { recursive: true });

    // 写入文件
    await this.fs.promises.writeFile(path, content);
  }

  /**
   * 读取文件
   */
  async readFile(path: string): Promise<string> {
    if (!this.fs) await this.initialize();

    const content = await this.fs.promises.readFile(path, 'utf8');
    return content as string;
  }

  /**
   * 创建目录
   */
  async mkdir(path: string): Promise<void> {
    if (!this.fs) await this.initialize();
    await this.fs.promises.mkdir(path, { recursive: true });
  }

  /**
   * 列出目录内容
   */
  async readdir(path: string): Promise<string[]> {
    if (!this.fs) await this.initialize();
    const entries = await this.fs.promises.readdir(path);
    return entries as string[];
  }

  /**
   * 删除文件
   */
  async unlink(path: string): Promise<void> {
    if (!this.fs) await this.initialize();
    await this.fs.promises.unlink(path);
  }
}
```

**虚拟文件系统总结**：
- **技术栈**: ZenFS (`@zenfs/core`) + IndexedDB
- **存储位置**: 浏览器 IndexedDB（数据库: `webenv-fs`，store: `files`）
- **持久化**: 是（IndexedDB 持久化到本地）
- **文件 API**: 标准 Node.js `fs` 风格 API
- **离线优先**: 支持（不依赖网络）

### Q2: UnoThree 为什么选择 SVG 而不是图片格式？

**A**: SVG 优势：
1. **矢量缩放**: 无损缩放，适合不同屏幕尺寸
2. **文件大小**: SVG 卡牌约 1-2KB，比 PNG (10-20KB) 小很多
3. **易于修改**: 可程序化生成（见 `generate-cards.js`）
4. **样式控制**: CSS 可直接修改 SVG 颜色
5. **3D 纹理**: Three.js 可直接将 SVG 作为纹理使用

### Q3: WebEnv-OS 的虚拟文件系统如何持久化？

**A**: 使用 ZenFS + IndexedDB：

```typescript
// 配置 IndexedDB 后端
const idbfs = new IndexedDB('webenv-fs', {
  storeName: 'files',
});

await configure({
  mounts: {
    '/': idbfs,  // 根目录挂载到 IndexedDB
  },
});

// 之后使用标准 fs API
await fs.promises.writeFile('/home/user/code.tsx', content);
```

**IndexedDB 优势**：
- 容量大（通常 50MB+）
- 持久化（页面关闭后数据保留）
- 异步操作（不阻塞主线程）
- 离线可用

### Q4: 如何优化音频加载性能？

**A**: UnoThree 使用预加载策略：

```typescript
// 游戏启动时预加载所有音效
await audioManager.preloadAll();

// 播放时直接从缓存获取
const buffer = this.soundBuffers.get(effect);
if (buffer) {
  this.playBuffer(buffer); // 立即播放
}
```

**优化点**：
1. **预加载**: 避免播放时再加载
2. **AudioBuffer 解码**: 一次性解码，播放时直接使用
3. **复用**: 同一音效可同时播放多次
4. **音量控制**: 使用 GainNode 控制全局音量

### Q5: 3D 渲染中的纹理如何优化？

**A**: 使用 `@react-three/drei` 的 `useTexture` hook：

```typescript
import { useTexture } from '@react-three/drei';

// 自动处理纹理加载和缓存
const cardTexture = useTexture('/assets/uno-cards/red-7.svg');

<mesh>
  <planeGeometry args={[2.5, 3.5]} />
  <meshStandardMaterial map={cardTexture} />
</mesh>
```

**优化点**：
1. **纹理缓存**: `useTexture` 自动缓存已加载纹理
2. **预加载**: 游戏开始时预加载常用纹理
3. **纹理压缩**: 考虑使用压缩格式（WebGL 支持）
4. **LOD 细节层次**: 远处可使用低分辨率纹理
