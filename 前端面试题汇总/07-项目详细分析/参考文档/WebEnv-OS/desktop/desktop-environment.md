# 桌面环境文档

## 概述
webEnvOS 桌面环境以 macOS 视觉与交互为基准，提供窗口管理、Dock、任务栏与应用运行能力。当前仅保留 macOS 主题风格，主题切换已移除。

## 访问路径
- 桌面入口：`/desktop`
- IDE 入口：`/ide`

## 核心组件
- **IndependentDesktopEnvironment**：桌面根组件，负责应用清单、壁纸、Dock、任务栏与窗口管理。
- **Desktop**：桌面背景与图标渲染。
- **WindowManager**：窗口创建/关闭/最小化/最大化/聚焦。
- **Window**：窗口容器与 macOS 风格控制按钮。
- **Dock**：启动栏与运行状态指示（计数、发光、右键关闭）。
- **Taskbar/TaskbarTop**：顶部任务栏与系统托盘。

## 窗口与 Dock 行为
- **单实例应用**：Docker、时钟、计算器、文本编辑器只允许一个窗口存在；重复点击会聚焦已存在窗口。
- **多实例应用**：终端、文件管理器、图片查看器等允许多窗口。
- **Dock 状态**：
  - 打开状态：底部白色指示点 + 发光
  - 多窗口：右上角计数徽标
  - 右键菜单：一键关闭该应用的所有窗口

## 壁纸系统
支持预设渐变、URL 图片与视频动态壁纸：
- **预设**：内置多款 mac 风格渐变壁纸。
- **自定义 URL**：支持图片或视频 URL。
- **本地上传**：支持上传图片/视频作为壁纸。
- **视频壁纸**：自动静音循环播放，建议 MP4/H.264 编码。

## SVG 图标规范
- 统一放置于 `webenv-os/public/icons`。
- 通过 `SVGIcon` 组件引用，路径自动补全为 `/icons/<name>.svg`。

## 已移除功能
- 主题切换与主题预览（现固定 macOS）。
- 旧的 ThemeSwitcher/ThemePreview 相关 UI。

## 手动验证清单
1. 进入 `/desktop` 后可看到 Dock 与顶部任务栏。
2. 点击 Dock 图标，窗口正常打开。
3. 单实例应用重复点击不会创建新窗口。
4. Dock 右键菜单可以关闭所有窗口。
5. 壁纸页可切换预设与视频壁纸。
