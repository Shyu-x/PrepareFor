# 桌面应用清单

## 概述
桌面应用运行在窗口系统之上，支持 Dock 快速启动与任务栏聚焦。

## 应用列表
- **IDE**：跳转至 `/ide` 进入完整 IDE。
- **Docker**（单实例）：容器管理与运行状态。
- **Kubernetes**：基础 UI 占位，后续接入集群能力。
- **文件管理器**：浏览与管理文件树。
- **应用管理器**：应用列表与安装入口。
- **任务管理器**：进程与资源概览。
- **剪贴板**：历史剪贴板管理。
- **终端 (webEnvOS)**：面向后端容器的交互式终端。
- **终端 (Debian)**：底层 Debian 终端会话。
- **3D 可视化**：Three.js 视图示例。
- **文本编辑器**（单实例）：多语言文本编辑。
- **图片查看器**：图片预览与操作。
- **计算器**（单实例）：基础与科学计算。
- **时钟**（单实例）：时钟/秒表/倒计时。
- **设置**：壁纸与系统信息。

## 单实例应用说明
以下应用仅允许一个窗口：Docker、时钟、计算器、文本编辑器。再次点击将聚焦已有窗口。

## Dock 行为
- 打开状态显示发光 + 指示点。
- 多窗口显示计数徽标。
- 右键 Dock 图标可关闭该应用全部窗口。
