# 语雀（Yuque）交互设计与底层 UI 规范

本文档详细描述语雀系统中最底层的用户交互设计（Interaction Design）与 UI 行为规范，旨在还原真实的用户操作体验。

---

## 1. 编辑器交互核心：块级操作 (Block-based Interaction)

语雀的 Lake 编辑器将文档内容视为一系列“块”（Blocks）。

### 1.1 块悬浮与拖拽
*   **悬浮状态**: 当鼠标悬浮在某个段落或卡片左侧时，会出现一个“拖拽句柄”（通常是 6 个点的图标）。
*   **拖拽行为**: 用户可以点击句柄并上下拖动以重新排列内容块。拖动时，目标位置会出现一条蓝色的“指示线”。
*   **块菜单**: 点击句柄通常会弹出针对该块的快捷菜单（如：删除、复制链接、转为卡片等）。

### 1.2 选区行为
*   **多块选中**: 支持跨块拖拽鼠标进行多块选中。选中的块会有一个浅蓝色的背景色。
*   **行内卡片选中**: 行内卡片（如公式、提及）被选中时，会有一个蓝色的外框。

---

## 2. 快捷输入与斜杠命令 (Slash Commands)

### 2.1 斜杠菜单 (`/`)
在空行输入 `/` 会唤起功能面板，支持通过拼音或英文搜索功能。菜单项包括：
*   **基础**: 表格、图片、附件、水平分割线。
*   **多媒体**: 视频、音频、嵌入本地文件（Word/PDF/PPT）。
*   **绘图与工程**: 流程图、思维导图、代码块、数学公式、PlantUML/Mermaid。
*   **第三方嵌入**: Bilibili, CodePen, Figma, 高德地图等。
*   **语雀服务**: 语雀内容卡片、打卡、日历。

### 2.2 Markdown 实时转换 (Markdown Triggers)
语雀支持在输入 Markdown 语法后按 `空格` 或 `回车` 立即转换：
*   `# ` -> 一级标题 (H1)
*   `- ` 或 `* ` -> 无序列表
*   `1. ` -> 有序列表
*   `[] ` -> 任务列表
*   `> ` -> 引用
*   `---` -> 分割线
*   `` `code` `` -> 行内代码 (输入第二个 ` 并继续输入时自动包裹)

## 3. 上下文工具栏 (Contextual Toolbars)

### 3.1 气泡工具栏 (Floating/Bubble Toolbar)
*   **文本选中**: 选中一段文字后，在选区上方弹出一个黑底白字的微型工具栏，包含加粗、斜体、链接、高亮、划词评论等最常用的格式化按钮。
*   **卡片选中**: 选中卡片（如图片、代码块）后，卡片上方出现专属工具栏。
    *   **图片工具栏**: 对齐方式、大小缩放、裁剪、替换图片。
    *   **代码块工具栏**: 语言切换（带搜索）、一键复制、切换主题。

---

## 4. 细节组件交互

### 4.1 图片处理 (Image Processing)
*   **缩放**: 选中图片后，四角出现蓝色控制点，支持等比例拉伸。
*   **裁剪**: 提供裁剪模式，支持矩形选框调整，实时预览。
*   **上传状态**: 正在上传的图片会显示半透明蒙层和进度条，上传失败显示带有重试按钮的异常卡片。

### 4.2 表格交互 (Table UX)
*   **行列快速增删**: 鼠标悬浮在表头或行首边缘时，会出现带有 `+` 号的蓝色触碰区域，点击即可插入行列。
*   **粘性表头**: 页面滚动时，如果表格较长，表头会自动吸顶。
*   **单元格操作**: 支持点击单元格左上角进行全选，支持拖拽表头边缘调整列宽。

### 4.3 实时协作 (Collaboration UI)
*   **协作者光标**: 不同协作者的光标以不同颜色的竖线显示，光标上方带有协作者的名称标签。
*   **冲突处理**: 采用毫秒级同步，当发生极小概率冲突时，通常保留最后输入或通过协同引擎自动合并。

## 5. 键盘快捷键 (Key Bindings)

| 快捷键 | 功能 |
| :--- | :--- |
| `Ctrl/Cmd + B` | 加粗 |
| `Ctrl/Cmd + K` | 插入链接 |
| `Ctrl/Cmd + E` | 快速插入/转为代码块 |
| `Ctrl/Cmd + /` | 显示/隐藏快捷键帮助 |
| `Ctrl/Cmd + Option + 1-6` | 切换标题等级 1-6 |
| `Enter` | 换行 (在列表末尾连按两次回车可跳出列表) |
| `Shift + Enter` | 软换行 (不产生新段落) |
| `Tab` | 列表缩进 / 表格移动至下一格 |

---

## 6. 状态响应 UI

### 6.1 自动保存提示
*   在页面顶部或底部状态栏显示“正在保存...”或“已保存到云端”。
*   断网时，显示“网络已断开，内容将暂存本地”的黄色警告条。

### 6.2 异常占位 (Exception Cards)
当某个卡片（如嵌入的网页或插件）加载失败时，显示统一的占位 UI：
*   **主标题**: “该卡片暂时无法显示”
*   **副标题**: “请刷新页面后再试”
*   **操作**: 提供一个“刷新”或“联系客服”的链接按钮。
