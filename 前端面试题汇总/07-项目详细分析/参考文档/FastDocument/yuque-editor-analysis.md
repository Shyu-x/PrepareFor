# 语雀编辑器功能分析报告

> 本文档基于对语雀官方帮助文档的研究，总结了语雀编辑器的核心功能特性，为 FastDocument 编辑器升级提供参考。

## 一、编辑器核心功能

### 1. 基础编辑能力

| 功能 | 说明 | FastDocument 当前状态 |
|------|------|---------------------|
| 文本编辑 | 支持富文本编辑 | ✅ 已实现 |
| 标题 H1-H3 | 多级标题支持 | ✅ 已实现 |
| 待办事项 | 带复选框的任务列表 | ✅ 已实现 |
| 提示框 (Callout) | 突出显示的提示信息 | ✅ 已实现 |
| 分隔线 | 视觉分隔内容 | ✅ 已实现 |
| 代码块 | 多语言语法高亮 | ⚠️ 基础实现 |
| 表格 | 插入表格 | ⚠️ 基础实现 |
| 图片 | 插入图片 | ⚠️ 基础实现 |

### 2. 高级块类型

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 思维导图 | 多种样式支持 | 高 |
| 流程图 | 多种样式支持 | 高 |
| 文本绘图 | PlantUML, Mermaid, Graphviz | 中 |
| 甘特图 | 项目管理甘特图 | 中 |
| 数学公式 | LaTeX 公式渲染 | 中 |
| 投票 | 收集用户意见 | 低 |
| 日历 | 日期选择器 | 低 |

### 3. 多媒体与嵌入

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 本地视频 | 支持嵌入本地视频 | 中 |
| 文件附件 | PDF, Word, Keynote, XMind | 中 |
| 第三方嵌入 | ProcessOn, Canva, Figma, CodePen | 低 |
| 语雀内容嵌入 | 嵌入其他语雀文档 | 低 |

### 4. 文档设置功能

| 功能 | 说明 |
|------|------|
| 文档宽度 | 窄/标准/宽 |
| 文档间距 | 紧凑/标准/宽松 |
| 分栏 | 多栏布局 |
| 锚点 | 文档内跳转 |
| 文档互链 | 链接到其他文档 |

### 5. 特色功能

| 功能 | 说明 |
|------|------|
| 翻译功能 | 全文翻译/智能翻译 |
| 代码运行 | 支持 Python, JavaScript 等在线运行 |
| 加密文本 | 敏感信息保护 |
| 自动目录 | 根据标题生成文档目录 |

## 二、代码块支持的语言

语雀支持的编程语言（基于 CodeMirror 集成）：

```
javascript, bash, c, cpp, csharp, go, java, php, rust,
shell, sql, typescript, vbnet, python, html,
systemverilog, verilog
```

## 三、思维导图特性

### 快捷键操作
- **Enter**: 插入子节点
- **Tab**: 插入兄弟节点
- **Delete**: 删除节点
- **双击**: 重命名节点

### 支持样式
- 思维导图
- 树状图
- 组织结构图
- 鱼骨图

## 七、技术实现建议

### 思维导图
- 推荐使用 `react-mindmap` 或 `xmind`
- 或使用 Mermaid 渲染

### 流程图
- 使用 Mermaid.js 库
- 支持实时预览

### 数学公式
- 使用 KaTeX 库
- 支持 LaTeX 语法

### 代码块
- 使用 Prism.js 或 highlight.js
- 支持代码运行（需要后端支持）
