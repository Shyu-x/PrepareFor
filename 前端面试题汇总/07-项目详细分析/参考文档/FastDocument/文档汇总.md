| 评论状态机 | 3状态 | open → resolved → archived |

| 会议状态机 | 4状态 | scheduled → active → ended → cancelled |
| 知识库状态 | 4级 | space → base → folder → document |
| 权限状态 | 4级 | owner → admin → editor → reader |

### 1. 文档协作流程 (01-document-collaboration.md)

**STAR 法则概述：**

| 维度 | 描述 |
|------|------|
| Situation | 多用户同时编辑同一文档，需要实时同步 |
| Task | 实现毫秒级文档协作，支持版本历史 |
| Action | 使用WebSocket实时推送、操作转换、块级锁 |
| Result | 完整的实时协作体验，版本回滚能力 |

**包含内容：**
- 文档创建和编辑流程

## 技术规范

### 端口配置标准

| 服务 | 端口 | 说明 |
|------|------|------|
| 前端 | 13000 | Next.js 开发服务器 |
| 后端 API | 5555 | NestJS API 服务 |
| PostgreSQL | 15432 | Docker 映射端口 |
| Redis | 16379 | Docker 映射端口 |
| LiveKit WS | 17880 | Docker 映射端口 |
| LiveKit HTTP | 17882 | Docker 映射端口 |

### 技术栈

**前端：**
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- Ant Design 6
- Zustand (状态管理)
- Socket.io Client
- LiveKit Client
- React Flow (知识图谱)

**后端：**
- NestJS
- TypeORM
- Socket.io
- PostgreSQL
- Redis

**基础设施：**
- Docker
- PM2
- Nginx
