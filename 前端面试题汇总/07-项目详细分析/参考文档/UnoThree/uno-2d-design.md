# UNO 2D 界面设计文档

作者：Shyu  | 更新日期：2026-02-18

> **实现状态**：2D 模式已完成并可正常运行，通过 `page.tsx` 中的 `viewMode` 状态切换。

## 设计目标

创建一个精美、高度纹理化的 2D 棋牌游戏界面，完全使用 SVG 绘制，支持响应式布局。

## 设计原则

1. **拟物化风格**：模拟真实纸牌质感，带有微妙的阴影、光泽和纹理
2. **丰富细节**：卡牌边缘、桌面纹理、玩家区域都有精细的SVG图案
3. **流畅动画**：卡牌移动、打出、抽牌都有流畅的动画效果
4. **响应式适配**：支持桌面端和移动端的横竖屏布局

### 卡牌设计

- **尺寸比例**：2.5:3.5（标准扑克牌比例）
- **圆角**：12px
- **边框**：3px 带渐变光泽
- **背面**：经典UNO图案，带UV光泽效果
- **正面**：
  - 数字/符号居中
  - 角落带有小标记（便于识别）
  - 带有细微的纸张纹理
  - 边缘有高光反射效果

### 桌面设计

- **主背景**：深绿色毛毡纹理 + 木质边框
- **纹理**：SVG pattern 实现毛毡编织效果
- **光影**：中央略亮，四周渐暗的晕影效果
- **玩家区域**：半透明弧形区域，带有微小纹理

### 玩家区域

- **布局**：椭圆围绕桌面中央
- **信息显示**：玩家名、血量/手牌数、UNO状态
- **当前回合**：发光高亮指示
- **手牌展示**：扇形展开，带有3D堆叠效果

## 动画效果

1. **卡牌打出**：从手牌位置飞向桌面，带旋转和缩放
2. **卡牌抽动画**：从牌堆飞向手牌，带轻微弹跳
3. **卡牌翻转**：背面翻转为正面，带3D翻转效果
4. **手牌展开**：进入游戏时手牌依次展开
5. **玩家高亮**：当前玩家区域发光脉冲
6. **颜色转换**：切换颜色时的渐变过渡

## 交互设计

1. **卡牌选择**：悬停时轻微上浮，点击选中
2. **可出牌提示**：可出的牌有发光边框
3. **拖拽出牌**：支持拖拽卡牌到桌面
4. **快速出牌**：双击卡牌直接打出（如果合法）
5. **颜色选择**：出Wild或+4时弹出颜色选择器

## 响应式适配

### 桌面端（>1024px）
- 4-8名玩家环绕布局
- 手牌水平展开在底部
- 大尺寸卡牌（120px宽）

### 平板端（768-1024px）
- 4名玩家布局
- 中等尺寸卡牌（90px宽）

### 移动端竖屏
- 紧凑布局
- 小尺寸卡牌（60px宽）
- 简化纹理

### 移动端横屏
- 横向拉伸布局
- 玩家分布在上下两侧
