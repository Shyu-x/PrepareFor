# 数据流向

## 一、数据流向概述

### 1.1 完整数据流

```
┌─────────────────────────────────────────────────────────────────┐
│                       完整数据流向                               │
│                                                                  │
│  用户点击卡牌                                                   │
│       │                                                         │
│       ▼                                                         │
│  ┌──────────────┐                                               │
│  │  Scene3D     │  前端处理点击事件                              │
│  └──────┬───────┘                                               │
│         │ onPlayCard                                            │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │GameSocketCtx │  emit('playCard')                             │
│  └──────┬───────┘                                               │
│         │ Socket.IO                                              │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │GameGateway   │  接收事件                                      │
│  └──────┬───────┘                                               │
│         │                                                        │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │GameService   │  验证规则、执行逻辑                            │
│  └──────┬───────┘                                               │
│         │ broadcastState                                         │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │GameGateway   │  脱敏广播                                      │
│  └──────┬───────┘                                               │
│         │ Socket.IO                                              │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │GameSocketCtx │  setGameState()                               │
│  └──────┬───────┘                                               │
│         │                                                        │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │ useGameStore │  状态更新                                      │
│  └──────┬───────┘                                               │
│         │                                                        │
│         ▼                                                         │
│  ┌──────────────┐                                               │
│  │ 组件重新渲染 │  Scene3D / HUD / Hand                         │
│  └──────────────┘                                               │
└─────────────────────────────────────────────────────────────────┘
```
