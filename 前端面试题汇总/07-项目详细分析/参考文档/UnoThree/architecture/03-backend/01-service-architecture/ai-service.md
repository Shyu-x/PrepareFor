# AI 服务

## 一、AI 概述

### 1.1 AI 难度

| 难度 | 策略 |
|------|------|
| EASY | 随机出牌 |
| MEDIUM | 颜色优化 |
| HARD | 攻击性策略 |

### 1.2 决策流程

```
AI 回合开始
     │
     ▼
筛选可出牌
     │
     ▼
根据难度选择策略
     │
     ├─▶ EASY ──▶ 随机选择
     │
     ├─▶ MEDIUM ──▶ 颜色最多优先
     │
     └─▶ HARD ──▶ 攻击下家优先
```
