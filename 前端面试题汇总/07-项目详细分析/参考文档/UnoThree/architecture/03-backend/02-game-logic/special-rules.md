# 特殊规则

## 一、功能牌效果

### 1.1 SKIP（跳过）

下一个玩家被跳过。

### 1.2 REVERSE（反转）

方向反转，二人游戏时等同于跳过。

### 1.3 DRAW_TWO（+2）

下家罚摸 2 张，无可质疑。

### 1.4 WILD（万能）

任意颜色。

### 1.5 WILD_DRAW_FOUR（+4）

任意颜色，下家罚摸 4 张，可质疑。
