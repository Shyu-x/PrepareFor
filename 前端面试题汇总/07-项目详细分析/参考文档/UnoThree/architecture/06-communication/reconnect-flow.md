# 重连流程

## 一、重连概述

### 1.1 重连机制

```
┌─────────────────────────────────────────────────────────────────┐
│                       重连流程                                   │
│                                                                  │
│  1. 首次加入                                                     │
│     ┌──────────┐    ┌──────────┐    ┌──────────┐            │
│     │ joinRoom │───▶│ 后端验证  │───▶│ 返回凭据  │            │
│     └──────────┘    └──────────┘    └────┬─────┘            │
│                                          │                    │
│                                          ▼                    │
│                                    ┌────────────┐             │
│                                    │ 保存凭据   │             │
│                                    │ (localSto) │             │
│                                    └────────────┘             │
│                                                                  │
│  2. 断线重连                                                     │
│     ┌──────────┐    ┌──────────┐    ┌──────────┐            │
│     │ 自动重连 │───▶│ 读取凭据  │───▶│ 恢复游戏 │            │
│     └──────────┘    └──────────┘    └──────────┘            │
│                                                                  │
│  3. 房间保留窗口                                                 │
│     所有人类玩家离线 ──▶ 60 秒倒计时 ──▶ 有人回来 ──▶ 取消    │
│                                       │                       │
│                                       ▼                       │
│                                   超时 ──▶ 解散房间            │
└─────────────────────────────────────────────────────────────────┘
```

## 二、凭据管理

### 2.1 凭据结构

```typescript
interface RoomReconnectCredential {
  sessionId: string;        // 会话 ID
  reconnectToken: string;    // 重连凭据
}
```

### 2.2 匹配顺序

1. reconnectToken（最准确）
2. sessionId（浏览器会话）
3. playerName（兜底匹配）

## 三、房间保留窗口

### 3.1 实现

```typescript
// 所有人类玩家离线后，设置 60 秒倒计时
if (allHumansDisconnected) {
  game.pendingDissolveAt = Date.now() + 60000;
}

// 有人恢复连接，清除倒计时
if (humanReconnected) {
  game.pendingDissolveAt = undefined;
}

// 定时检查
if (game.pendingDissolveAt && Date.now() > game.pendingDissolveAt) {
  this.closeRoom(roomId, '超时解散');
}
```
