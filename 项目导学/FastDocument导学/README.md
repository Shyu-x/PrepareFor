## 学习目标

完成本教学系统后，你将能够：

1. **完全理解** FastDocument 项目的技术实现
2. **掌握** 项目所涉及的所有技术栈
3. **具备** 独立开发和维护类似项目的能力
4. **建立** 完整的现代Web开发知识体系

## 核心技术栈

| 类别 | 技术 | 版本 |
|------|------|------|
| 前端框架 | Next.js | 16.1.6 |
| UI库 | React | 19.2.3 |
| 样式 | Tailwind CSS | v4 |
| 组件库 | Ant Design | 6.3.0 |
| 状态管理 | Zustand | 5.0.11 |
| 后端框架 | NestJS | 11.x |
| 数据库 | PostgreSQL | - |
| 缓存 | Redis | - |
| WebSocket | Socket.io | 4.8.3 |
| 视频会议 | LiveKit | 2.17.2 |
| 测试 | Playwright/Vitest | - |

## 项目功能模块

1. **文档编辑** - 原子化块编辑器，支持多种块类型
2. **实时协作** - WebSocket 实时同步
3. **知识库** - 空间、文档树、双向链接
4. **项目管理** - 看板、甘特图、日历视图
5. **视频会议** - LiveKit 集成
6. **评论批注** - 文档评论、文本标注
7. **分享系统** - 链接分享、权限控制
8. **通知系统** - 多类型通知

## 开始学习

建议按照以下顺序学习：

1. 先阅读 `00_项目概述与技术栈分析/项目技术栈完全指南.md`
2. 根据自身基础选择 `01_基础语法教程` 中的内容
3. 深入学习 `02_React核心教程`
4. 掌握 `03_项目技术栈详解` 中的具体技术
5. 通过 `04_项目代码解析` 理解项目实现
6. 完成 `05_实战练习` 巩固知识
7. 学习 `06_调试与问题解决` 提升技能
8. 最后参考 `07_项目扩展` 进行开发
