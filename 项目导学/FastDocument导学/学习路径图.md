## 详细学习路线

### 阶段1：基础准备（1-2周）

| 序号 | 技术点 | 预计时间 | 难度 | 重点 |
|------|--------|----------|------|------|
| 1.1 | HTML5 语义化标签 | 1天 | ⭐ | 标签语义、SEO |
| 1.2 | CSS3 布局（Flex/Grid） | 2天 | ⭐⭐ | 响应式布局 |
| 1.3 | JavaScript 基础 | 3天 | ⭐⭐ | 变量、函数、DOM |
| 1.4 | ES6+ 特性 | 3天 | ⭐⭐ | 箭头函数、Promise |
| 1.5 | TypeScript 基础 | 3天 | ⭐⭐⭐ | 类型系统、接口 |

**推荐资源**：
- MDN Web Docs
- JavaScript.info
- TypeScript 官方文档

---

### 阶段2：React 入门（2-3周）

| 序号 | 技术点 | 预计时间 | 难度 | 重点 |
|------|--------|----------|------|------|
| 2.1 | JSX 语法 | 1天 | ⭐ | 语法转换 |
| 2.2 | 组件定义与使用 | 2天 | ⭐⭐ | 函数组件 |
| 2.3 | Props 与 State | 2天 | ⭐⭐ | 数据流 |
| 2.4 | useState 状态 | 2天 | ⭐⭐ | 状态管理 |
| 2.5 | useEffect 副作用 | 2天 | ⭐⭐⭐ | 生命周期 |
| 2.6 | useContext 上下文 | 1天 | ⭐⭐ | 状态共享 |
| 2.7 | useReducer 复杂状态 | 1天 | ⭐⭐⭐ | 状态逻辑 |
| 2.8 | useMemo/useCallback | 1天 | ⭐⭐⭐ | 性能优化 |
| 2.9 | 自定义 Hooks | 2天 | ⭐⭐⭐ | 逻辑复用 |
| 2.10 | 条件渲染与列表 | 1天 | ⭐ | 渲染逻辑 |

**练习目标**：
- 能够独立开发 React 组件
- 理解 React 数据流
- 掌握 Hooks 用法

---

### 阶段3：项目技术栈（1-2周）

| 序号 | 技术点 | 预计时间 | 难度 | 重点 |

### 阶段4：项目实战（2-3周）

#### 4.1 文档编辑器模块

**目标**：实现原子化块编辑器

**关键学习点**：
- 块的数据结构设计
- 块操作的 CRUD
- 块类型渲染
- 块拖拽排序

**实践任务**：
1. 实现文本块
2. 实现标题块
3. 实现待办块
4. 实现代码块
5. 实现图片块

#### 4.2 知识库模块

**目标**：实现知识库系统

**关键学习点**：
- 空间/知识库/文档层级
- 树形数据结构
- 双向链接实现

#### 4.3 项目管理模块

**目标**：实现看板和甘特图

**关键学习点**：
- 看板拖拽
- 任务状态管理
- 甘特图渲染

#### 4.4 视频会议模块

**目标**：实现实时音视频

**关键学习点**：
- LiveKit 集成
- 屏幕共享
- 参会者管理

---

## 快速学习建议

### 如果你已经有经验

如果你已经熟悉 React 和现代 Web 开发，可以：

1. **跳过阶段1**：快速浏览
2. **重点阶段3**：学习项目特定技术栈
3. **专注阶段4**：通过项目实战深入

### 如果你是初学者

建议按顺序学习：

1. **先动手**：不要只看，要动手写代码
2. **做项目**：通过项目巩固知识
3. **多调试**：学会使用 React DevTools
4. **读源码**：学习优秀项目实现
