# CLAUDE.md

此文件为 Claude Code 提供在当前代码库中工作的指导方针。

## 项目概述

这是一个**教学文档系统**，用于教授 FastDocument 项目的开发技术。FastDocument 是一个功能完整的现代化文档协作平台，采用原子化设计理念，支持文档编辑、知识库管理、项目管理、视频会议等功能。

## 实际项目代码位置

- **FastDocument 源码**: `D:\Develeping\FastDocument`
  - 前端: `frontend/` (Next.js 16.1.6 + React 19.2.3)
  - 后端: `backend/` (NestJS 11.x)
  - 移动端: `mobile/`

## 核心功能模块

| 模块 | 功能描述 |
|------|----------|
| 文档编辑 | 原子化块编辑器，支持多种块类型 |
| 实时协作 | WebSocket 实时同步 |
| 知识库 | 空间、文档树、双向链接 |
| 项目管理 | 看板、甘特图、日历视图 |
| 视频会议 | LiveKit 集成 |
| 评论批注 | 文档评论、文本标注 |

## 核心技术栈

- **前端框架**: Next.js 16.1.6, React 19.2.3
- **状态管理**: Zustand 5.0.11
- **样式方案**: Tailwind CSS v4
- **组件库**: Ant Design 6.3.0
- **后端框架**: NestJS 11.x (11.1.14)
- **实时通信**: Socket.io 4.8.3, LiveKit 2.17.2
- **数据库**: PostgreSQL (pg 8.18.0) + TypeORM 0.3.28
- **缓存**: Redis (ioredis 5.9.3)
- **协作服务器**: Hocuspocus 3.4.4

## 教学系统规范

### 分阶段深度教学

本教学系统采用**分阶段、模块化、渐进式**的方法，基于实际项目代码构建。

#### 第一阶段：项目深度分析与基础架构理解
- 1.1 全面项目扫描与技术栈分析
- 1.2 基础语法层教学（逐文件精读）

#### 第二阶段：React核心概念逐组件解析
- 2.1 组件架构与JSX深度分析
- 2.2 React Hooks实战详解

#### 第三阶段：项目技术栈专项精读
- 3.1 状态管理方案深度解析
- 3.2 路由系统逐文件解析

#### 第四阶段：项目实战逐行精读
- 4.1 核心组件逐行注释
- 4.2 工具函数与Hooks深度解析

#### 第五阶段：实战练习与扩展开发
- 5.1 分层练习系统（Level 1-5）
- 5.2 调试与问题解决指南

### 执行策略：四步法

1. **深度扫描**（1-2小时）
   - 完全扫描项目所有文件
   - 建立完整的文件清单
   - 识别所有技术栈
   - 生成基础架构图

2. **分层教学**
   - 每个阶段独立完成
   - 每个知识点独立成篇
   - 每个代码片段独立分析
   - 每个组件单独注释

3. **交叉验证**
   - 技术概念与实际代码对应
   - 理论教学与实践练习对应
   - 基础语法与项目应用对应

4. **迭代完善**
   - 每阶段完成后自我检查
   - 确保无上下文丢失
   - 补充遗漏的细节
   - 优化教学逻辑
