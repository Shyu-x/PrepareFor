# 常见错误

本章节将详细介绍 WebEnv 项目开发过程中可能遇到的各类错误，以及对应的解决方案。

## 1. 开发环境常见错误

### 1.1 Node.js 版本不兼容

**错误表现：**
```
Error: The engine "node" is incompatible with this module
```

**解决方案：**

```bash
# 查看项目所需的 Node.js 版本
cat package.json | grep -A 5 "engines"

# 使用 nvm 切换 Node.js 版本
nvm install 20
nvm use 20
```

**项目要求：** Node.js 18.x 或更高版本

### 1.2 依赖安装失败

**错误表现：**
```
npm ERR! code ETARGET
npm ERR! Unable to resolve dependency
```

**解决方案：**

```bash
# 清理 npm 缓存
npm cache clean --force

# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装依赖
npm install
```

### 1.3 TypeScript 配置错误

**错误表现：**
```
error TS2307: Cannot find module '@/components/xxx'
```

**解决方案：** 检查 `tsconfig.json` 中的路径配置：

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

### 1.4 Next.js 配置问题

**错误表现：**
```
Error: No such file or directory: .next/cache/...
```

**解决方案：**

```bash
# 清理 Next.js 缓存
rm -rf .next
npm run build
```

## 2. 运行时错误

### 2.1 Hydration 错误

**错误表现：**
```
Warning: Expected server HTML to contain a matching <div> in <div>
```

**原因分析：** 服务端渲染（SSR）与客户端渲染内容不一致

**解决方案：**

```tsx
// 方案一：使用 useEffect 在客户端渲染后再显示动态内容
'use client';
import { useState, useEffect } from 'react';

export function TimeDisplay() {
  const [time, setTime] = useState<string>('');

  useEffect(() => {
    setTime(new Date().toLocaleTimeString());
  }, []);

  return <div>{time || 'Loading...'}</div>;
}

// 方案二：使用 suppressHydrationWarning
export function TimeDisplay() {
  return (
    <div suppressHydrationWarning>
      {typeof window !== 'undefined' ? new Date().toLocaleTimeString() : ''}
    </div>
  );
}
```

### 2.2 状态更新错误

**错误表现：**
```
Error: Maximum update depth exceeded
```

**原因分析：** 在 useEffect 中无限更新状态

**解决方案：**

```tsx
// 错误的写法 - 会导致无限循环
useEffect(() => {
  setCount(count + 1); // 每次渲染都会触发
}, [count]);

// 正确的写法 - 使用函数式更新
useEffect(() => {
  const timer = setInterval(() => {
    setCount(prev => prev + 1); // 使用函数式更新
  }, 1000);
  return () => clearInterval(timer);
}, []);

// 或者使用 useRef
const countRef = useRef(0);
useEffect(() => {
  countRef.current += 1;
}, []);
```

### 2.3 内存泄漏

**错误表现：**
```
Warning: Can't perform a React state update on an unmounted component
```

**解决方案：**

```tsx
// 使用 useRef 标记组件是否已挂载
export function DataFetcher() {
  const isMounted = useRef(true);

  useEffect(() => {
    fetchData().then(data => {
      if (isMounted.current) {
        setData(data);
      }
    });

    return () => {
      isMounted.current = false;
    };
  }, []);

  return <div>{data}</div>;
}

// 使用 AbortController 取消请求
useEffect(() => {
  const controller = new AbortController();

  fetch('/api/data', { signal: controller.signal })
    .then(res => res.json())
    .then(data => setData(data))
    .catch(err => {
      if (err.name !== 'AbortError') {
        console.error(err);
      }
    });

  return () => controller.abort();
}, []);
```

## 3. 构建错误

### 3.1 ESBuild 错误

**错误表现：**
```
Error: Could not resolve "xxx" from "xxx.tsx"
```

**解决方案：**

```bash
# 确保依赖已安装
npm install xxx

# 检查导入路径是否正确
import { something } from '@/components/xxx';
```

### 3.2 CSS 导入错误

**错误表现：**
```
Error: CSS import is not supported
```

**解决方案：** 检查 Tailwind CSS 配置和导入方式

```tsx
// 确保在全局 CSS 中正确导入 Tailwind
// app/globals.css
@tailwind base;
@tailwind components;
@tailwind utilities;

// 在组件中正确使用
import styles from './Component.module.css';
// 或使用 Tailwind 类名
<div className="bg-blue-500">Content</div>
```

### 3.3 图片导入错误

**错误表现：**
```
Error: Cannot find module './image.png'
```

**解决方案：** 确保图片文件存在，或使用正确的路径

```tsx
// 方案一：使用 import
import imageUrl from './assets/image.png';

// 方案二：使用 public 目录
// 将图片放在 public/images/ 目录
<img src="/images/image.png" alt="description" />

// 方案三：使用 URL
<img src="https://example.com/image.png" alt="description" />
```

## 4. 类型错误

### 4.1 类型推断错误

**错误表现：**
```
Type 'string' is not assignable to type 'number'
```

**解决方案：**

```tsx
// 明确指定类型
const count: number = 0;
const name: string = 'John';

// 使用类型断言
const value = someUnknown as string;

// 使用类型守卫
function handleValue(value: string | number) {
  if (typeof value === 'string') {
    console.log(value.toUpperCase());
  } else {
    console.log(value.toFixed(2));
  }
}
```

### 4.2 泛型类型错误

**错误表现：**
```
Type 'T' does not satisfy the constraint 'xxx'
```

**解决方案：**

```tsx
// 正确使用泛型约束
interface HasId {
  id: string | number;
}

function findItem<T extends HasId>(items: T[], id: string | number): T | undefined {
  return items.find(item => item.id === id);
}

// 使用默认类型参数
function createStore<T = string>(initialValue: T) {
  // ...
}
```

### 4.3 React 类型错误

**错误表现：**
```
Type '(props: Props) => JSX.Element' is not assignable to 'xxx'
```

**解决方案：**

```tsx
// 确保组件正确导入和导出
import { FC } from 'react';

interface Props {
  title: string;
}

const MyComponent: FC<Props> = ({ title }) => {
  return <div>{title}</div>;
};

// 或者使用函数组件
function MyComponent({ title }: Props) {
  return <div>{title}</div>;
}

export default MyComponent;
```

## 5. 状态管理错误

### 5.1 Zustand 状态错误

**错误表现：**
```
Error: State does not exist in the store
```

**解决方案：**

```tsx
// 正确创建 store
import { create } from 'zustand';

interface StoreState {
  count: number;
  increment: () => void;
}

const useStore = create<StoreState>((set) => ({
  count: 0,
  increment: () => set((state) => ({ count: state.count + 1 })),
}));

// 正确使用 store
function Counter() {
  const { count, increment } = useStore();
  return <button onClick={increment}>{count}</button>;
}
```

### 5.2 状态初始化错误

**错误表现：**
```
Error: Cannot read properties of undefined
```

**解决方案：**

```tsx
// 使用可选链和默认值
const { user } = useStore();
const name = user?.name ?? 'Guest';

// 提供默认状态
const useStore = create<StoreState>((set) => ({
  user: null,
  // ...
}));

// 初始化检查
if (!user) {
  return <div>Loading...</div>;
}
```

### 5.3 派生状态错误

**错误表现：**
```
Warning: Derived state is deprecated
```

**解决方案：** 使用 useMemo 计算派生状态

```tsx
// 避免在 store 中存储派生状态
const useStore = create<StoreState>((set, get) => ({
  items: [],
  // 错误：不应在这里计算派生状态
  // total: items.reduce((sum, item) => sum + item.price, 0),

  // 正确：在组件中使用 useMemo
  getTotal: () => get().items.reduce((sum, item) => sum + item.price, 0),
}));

// 在组件中计算
function Total() {
  const items = useStore(state => state.items);
  const total = useMemo(() =>
    items.reduce((sum, item) => sum + item.price, 0),
    [items]
  );

  return <div>Total: ${total}</div>;
}
```

## 6. 样式问题

### 6.1 Tailwind CSS 不生效

**问题原因：**
1. 配置文件路径错误
2. PostCSS 配置缺失
3. 类名拼写错误

**解决方案：**

```bash
# 确保 tailwind.config.js 存在
npx tailwindcss init -p

# 配置 content 路径
module.exports = {
  content: [
    './app/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
}
```

### 6.2 样式冲突

**问题表现：**
样式互相覆盖，布局错乱

**解决方案：**

```tsx
// 使用 CSS Modules
import styles from './Button.module.css';

export function Button() {
  return <button className={styles.button}>Click</button>;
}

// 使用 Tailwind 的 !important
<div className="!absolute !top-0">

// 使用 style 属性覆盖
<div style={{ zIndex: 100 }}>
```

### 6.3 响应式样式问题

**问题表现：**
移动端和桌面端样式不一致

**解决方案：**

```tsx
// 使用响应式前缀
<div className="text-sm md:text-lg lg:text-xl">
  响应式文本
</div>

// 使用自定义断点
// tailwind.config.js
module.exports = {
  theme: {
    screens: {
      'xs': '375px',
      'sm': '640px',
      'md': '768px',
      'lg': '1024px',
      'xl': '1280px',
      '2xl': '1536px',
    },
  },
}
```

## 8. 常见错误速查表

| 错误类型 | 错误信息 | 解决方案 |
|---------|---------|---------|
| 模块解析 | Cannot find module | 检查导入路径和依赖安装 |
| 类型错误 | Type 'X' is not assignable to 'Y' | 添加类型注解或类型断言 |
| Hydration | Hydration failed | 使用 useEffect 或 suppressHydrationWarning |

## 练习题

1. 修复以下代码中的类型错误：
```tsx
function add(a: number, b: number) {
  return a + b;
}
console.log(add("1", "2")); // 如何修复？
```

2. 解决 Hydration 错误：
```tsx
// 当前代码会导致 Hydration 错误，如何修复？
export function Clock() {
  return <div>{new Date().toLocaleTimeString()}</div>;
}
```

3. 修复状态更新导致的无限循环：
```tsx
function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    setCount(count + 1); // 这会导致什么问题？
  }, []);

  return <div>{count}</div>;
}
```

## 答案解析

### 练习题 1 答案
```tsx
// 方案一：类型检查
console.log(add(Number("1"), Number("2")));

// 方案二：类型守卫
function add(a: number | string, b: number | string) {
  const numA = typeof a === 'string' ? Number(a) : a;
  const numB = typeof b === 'string' ? Number(b) : b;
  return numA + numB;
}
console.log(add("1", "2"));
```

### 练习题 2 答案
```tsx
'use client';
import { useState, useEffect } from 'react';

export function Clock() {
  const [time, setTime] = useState('');

  useEffect(() => {
    setTime(new Date().toLocaleTimeString());
  }, []);

  return <div>{time || 'Loading...'}</div>;
}
```

### 练习题 3 答案
```tsx
function Counter() {
  const [count, setCount] = useState(0);

  useEffect(() => {
    // 方案一：使用函数式更新
    setCount(prev => prev + 1);

    // 方案二：移除依赖项，只运行一次
  }, []); // 空依赖数组表示只在挂载时运行一次

  return <div>{count}</div>;
}
```
