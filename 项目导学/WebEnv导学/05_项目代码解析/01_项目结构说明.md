# 项目结构说明

本文档详细解析 WebEnv-OS 项目的目录结构和核心文件，帮助开发者快速理解项目架构。

## 1. 项目根目录结构

```
webenv-os/
├── src/                      # 源代码目录
│   ├── app/                  # Next.js App Router 页面
│   ├── apps/                 # 独立应用（终端等）
│   ├── components/           # React 组件库
│   ├── constants/            # 常量定义
│   ├── kernel/               # 内核模块（进程、系统调用）
│   ├── lib/                  # 工具库和业务逻辑
│   ├── store/                # Zustand 状态管理
│   ├── styles/               # 样式和主题
│   ├── test/                 # 测试文件
│   └── types/                # TypeScript 类型定义
├── public/                   # 静态资源
├── node_modules/             # 依赖包
├── package.json              # 项目配置
├── next.config.mjs           # Next.js 配置
├── postcss.config.mjs        # PostCSS 配置
└── tsconfig.json             # TypeScript 配置
```

## 3. 核心文件清单

### 3.1 布局类文件

| 文件路径 | 功能描述 |
|---------|---------|
| `src/components/IDELayout.tsx` | IDE 主布局，包含菜单栏、活动栏、侧边栏、编辑器面板、底部面板、状态栏 |
| `src/components/desktop/Desktop.tsx` | 桌面环境，包含壁纸、桌面图标、右键菜单 |
| `src/components/desktop/WindowManager.tsx` | 窗口管理系统，处理窗口创建、关闭、最小化、最大化 |

### 3.2 IDE 核心组件

| 文件路径 | 功能描述 |
|---------|---------|
| `src/components/ide/ActivityBar.tsx` | 左侧活动栏，提供视图切换快捷入口 |
| `src/components/ide/Sidebar.tsx` | 侧边栏容器，显示各种面板内容 |
| `src/components/ide/PanelArea.tsx` | 底部面板，包含终端、问题、输出、调试等 |
| `src/components/ide/MenuBar.tsx` | 顶部菜单栏 |

### 3.3 编辑器组件

| 文件路径 | 功能描述 |
|---------|---------|
| `src/components/editor/CodeEditor.tsx` | Monaco 编辑器包装，支持 LSP 智能提示 |
| `src/components/editor/EditorPane.tsx` | 编辑器窗格，管理多个标签页 |
| `src/components/editor/IntelliSensePanel.tsx` | 智能提示面板 |

### 3.4 状态管理

| 文件路径 | 功能描述 |
|---------|---------|
| `src/store/useIDEStore.ts` | IDE 主状态管理（600+ 行），管理文件、编辑器、面板、Git、运行配置等 |
| `src/store/useThemeStore.ts` | 主题状态（明暗模式） |
| `src/store/useWorkspaceStore.ts` | 工作区状态 |

## 4. 文件命名规范

### 4.1 组件命名

- **页面组件**: `PageName.tsx` (如 `IDELayout.tsx`)
- **子组件**: `ComponentName.tsx` (如 `ActivityBar.tsx`)
- **面板组件**: `*Panel.tsx` (如 `SearchPanel.tsx`)
- **工具组件**: `*Manager.tsx` (如 `WindowManager.tsx`)

### 4.2 状态管理文件

- **Store 命名**: `use*Store.ts` (如 `useIDEStore.ts`)
- **类型定义**: 与 store 同名或 `types/index.ts`

### 4.3 目录命名

- **组件目录**: 复数形式 (如 `components/desktop/`)
- **工具目录**: 复数形式 (如 `lib/hooks/`)
- **类型目录**: 复数形式 (如 `types/`)

## 7. 技术栈概览

- **框架**: Next.js 14 (App Router)
- **UI 库**: Ant Design 5
- **状态管理**: Zustand
- **编辑器**: Monaco Editor
- **动画**: Framer Motion
- **样式**: Tailwind CSS + CSS Variables
- **语言**: TypeScript
