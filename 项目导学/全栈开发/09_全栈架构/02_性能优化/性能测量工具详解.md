### 1.1 Performance面板

Performance面板是分析页面性能的核心工具：

```javascript
// Performance面板分析流程

/*
┌─────────────────────────────────────────────────────────────────────┐
│                       Performance面板使用指南                          │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. 打开方式                                                        │
│     - F12打开DevTools                                               │
│     - Ctrl+Shift+P打开命令菜单                                      │
│     - 选择"Performance"                                             │
│                                                                     │
│  2. 录制控制                                                        │
│     - 点击录制按钮或按Ctrl+E开始录制                                  │
│     - 刷新页面自动录制                                               │
│     - 点击停止或按Ctrl+E停止                                         │
│                                                                     │
│  3. 关键指标区域                                                    │
│     - FPS：绿色越高越好，红色表示掉帧                                  │
│     - CPU：显示主线程活动                                            │
│     - NET：网络请求瀑布流                                           │
│                                                                     │
│  4. 火焰图（Main）                                                 │
│     - 显示主线程执行情况                                             │
│     - 顶部是脚本执行                                                │
│     - 下方是任务队列                                                │
│                                                                     │
│  5. 关键事件标记                                                    │
│     - FP：首次绘制                                                  │
│     - FCP：首次内容绘制                                              │
│     - LCP：最大内容绘制                                              │
│     - DCL：DOM内容加载完成                                           │
│     - L：页面加载完成                                                │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
*/

// 使用JavaScript模拟Performance面板分析
function analyzePerformanceTrace() {
  // 获取所有性能条目
  const entries = performance.getEntries();

  // 分类统计
  const byType = entries.reduce((acc, entry) => {
    const type = entry.entryType;
    if (!acc[type]) acc[type] = [];
    acc[type].push(entry);
    return acc;
  }, {});

  // 渲染分析
  if (byType.paint) {
    byType.paint.forEach(entry => {
      console.log(`绘制: ${entry.name}`, entry.startTime.toFixed(2) + 'ms');
    });
  }

  // 资源分析
  if (byType.resource) {
    const slowResources = byType.resource
      .filter(r => r.duration > 1000)
      .sort((a, b) => b.duration - a.duration);

    console.log('慢资源:', slowResources.map(r => ({
      name: r.name.split('/').pop(),
      duration: r.duration.toFixed(2) + 'ms',
    })));
  }

  // 长任务分析
  if (byType.longtask) {
    byType.longtask.forEach(task => {
      console.warn('长任务:', task.duration.toFixed(2) + 'ms');
    });
  }
}
```

Network面板用于分析网络请求：

```javascript
// Network面板分析

/*
┌─────────────────────────────────────────────────────────────────────┐
│                       Network面板功能                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. 请求列表列                                                      │
│     - Name：资源名称                                                │
│     - Status：HTTP状态码                                            │

│     - DNS Lookup：DNS查询                                           │
│     - Initial Connection：连接建立                                    │
│     - SSL：SSL握手                                                  │
│     - Request Sent：请求发送                                        │
│     - Waiting (TTFB)：等待首字节                                     │
│     - Content Download：内容下载                                      │
│                                                                     │
│  3. 筛选器                                                          │
│     - 文本类型：HTML/CSS/JS/Images/Fonts/XHR/Fetch                   │
│     - Domain：按域名筛选                                           │
│     - Larger than：按大小筛选                                       │
│     - 反向筛选                                                      │
│                                                                     │
│  4. 功能选项                                                        │
│     - Preserve log：保留日志                                         │
│     - Disable cache：禁用缓存                                       │
│     - Slow 3G：模拟慢速网络                                        │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
*/

// JavaScript分析网络请求
function analyzeNetworkRequests() {
  const resources = performance.getEntriesByType('resource');

  // 按类型分组
  const byType = resources.reduce((acc, r) => {
    const type = r.initiatorType;
    if (!acc[type]) acc[type] = [];
    acc[type].push(r);
    return acc;
  }, {});

  // 计算各类资源统计
  Object.entries(byType).forEach(([type, items]) => {
    const totalSize = items.reduce((sum, r) => sum + (r.transferSize || 0), 0);
    const totalTime = items.reduce((sum, r) => sum + r.duration, 0);

    console.log(`${type}:`, {
      请求数: items.length,
      总大小: (totalSize / 1024).toFixed(2) + 'KB',
      平均时间: (totalTime / items.length).toFixed(2) + 'ms',
    });
  });

  // 慢请求分析
  const slowRequests = resources
    .filter(r => r.duration > 2000)
    .map(r => ({
      name: r.name.split('/').pop(),
      duration: r.duration.toFixed(2) + 'ms',
      size: ((r.transferSize || 0) / 1024).toFixed(2) + 'KB',
    }));

  console.log('慢请求:', slowRequests);

  // 大文件分析
  const largeFiles = resources
    .filter(r => (r.transferSize || 0) > 500 * 1024)
    .map(r => ({
      name: r.name.split('/').pop(),
      size: ((r.transferSize || 0) / 1024).toFixed(2) + 'KB',
    }));

  console.log('大文件:', largeFiles);
}
```

```javascript
│                                                                     │
│  4. 内存泄漏检测                                                    │
│     - 不断增长的堆快照                                              │

│     - 闭包引用                                                      │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘

// JavaScript内存监控
function monitorMemory() {
  // 获取内存信息（Chrome特定）
  const memory = (performance as any).memory;

  if (memory) {
    console.log('内存使用:', {
      已用堆: ((memory.usedJSHeapSize || 0) / 1024 / 1024).toFixed(2) + 'MB',
      总堆: ((memory.totalJSHeapSize || 0) / 1024 / 1024).toFixed(2) + 'MB',
      堆限制: ((memory.jsHeapSizeLimit || 0) / 1024 / 1024).toFixed(2) + 'MB',
      使用率: ((memory.usedJSHeapSize / memory.jsHeapSizeLimit) * 100).toFixed(2) + '%',
    });
  }

  // 记录内存时间线
  const memoryTimeline = [];
  const interval = setInterval(() => {
    if (memory) {
      memoryTimeline.push({
        time: Date.now(),
        used: memory.usedJSHeapSize,
      });
    }
  }, 1000);

  // 5分钟后停止
  setTimeout(() => clearInterval(interval), 5 * 60 * 1000);
}

// 检测内存泄漏
function detectMemoryLeaks() {
  const snapshots = [];
  let intervalId: number;

  // 定期获取堆快照引用
  const takeSnapshot = () => {
    if ((performance as any).memory) {
      snapshots.push({
        time: Date.now(),
        usedJSHeapSize: (performance as any).memory.usedJSHeapSize,
      });
    }
  };

  // 开始监控
  intervalId = setInterval(takeSnapshot, 5000);

  // 分析趋势
  const analyzeTrend = () => {
    if (snapshots.length < 10) return;

    const recent = snapshots.slice(-10);
    const growth = recent[9].usedJSHeapSize - recent[0].usedJSHeapSize;
    const growthRate = (growth / recent[0].usedJSHeapSize) * 100;

    if (growthRate > 10) {
      console.warn(`内存增长警告: ${growthRate.toFixed(2)}%`);
    }
  };

  return () => {
    clearInterval(intervalId);
    analyzeTrend();
  };
}
```

```javascript
// 检查元素是否在合成层中
function checkCompositing(element) {
  const computed = window.getComputedStyle(element);

  // 检查是否触发合成层
  const compositingReasons = [];

  if (computed.transform !== 'none') {
    compositingReasons.push('transform');
  }

  if (computed.willChange !== 'auto') {
    compositingReasons.push(`will-change: ${computed.willChange}`);
  }

  if (computed.filter !== 'none') {
    compositingReasons.push('filter');
  }

  if (computed.backdropFilter !== 'none') {
    compositingReasons.push('backdrop-filter');
  }

  if (computed.opacity < 1) {
    compositingReasons.push(`opacity: ${computed.opacity}`);
  }

  console.log('合成原因:', compositingReasons);

  return compositingReasons.length > 0;
}
```

```javascript
// Performance API基础

// 1. 导航计时
function getNavigationTiming() {
  const timing = performance.getEntriesByType('navigation')[0];

  return {
    // 连接阶段
    dns: timing.domainLookupEnd - timing.domainLookupStart,
    tcp: timing.connectEnd - timing.connectStart,
    ssl: timing.secureConnectionStart > 0
      ? timing.connectEnd - timing.secureConnectionStart
      : 0,

    // 请求阶段
    ttfb: timing.responseStart - timing.requestStart,
    request: timing.requestStart - timing.fetchStart,

    domComplete: timing.domComplete - timing.fetchStart,

    // 加载阶段
    load: timing.loadEventEnd - timing.fetchStart,
  };
}

function getResourceTiming() {
```

```javascript
function observePaintEvents() {

}

// 3. 观察布局偏移
function observeLayoutShifts() {
  let totalCLS = 0;

  const observer = new PerformanceObserver((list) => {
    list.getEntries().forEach(entry => {
      if (!(entry as any).hadRecentInput) {
        totalCLS += entry.value;
        console.log('布局偏移:', {
          值: entry.value.toFixed(4),
          累积: totalCLS.toFixed(4),
          来源: entry.sources?.map(s => s.node?.tagName),
        });
      }
    });
  });

  observer.observe({ entryTypes: ['layout-shift'] });
}

// 4. 观察资源加载
function observeResourceTiming() {
  const observer = new PerformanceObserver((list) => {
    list.getEntries().forEach(entry => {
      if (entry.duration > 1000) {
        console.warn('慢资源:', {
          名称: entry.name.split('/').pop(),
          持续时间: entry.duration.toFixed(2) + 'ms',
          类型: entry.initiatorType,
        });
      }
    });
  });

  observer.observe({ entryTypes: ['resource'] });
}

// 5. 观察最大内容绘制
function observeLCP() {
  let lcpEntry = null;

  const observer = new PerformanceObserver((list) => {
    const entries = list.getEntries();

    // LCP是最新的条目
    lcpEntry = entries[entries.length - 1];
  });

  observer.observe({ type: 'largest-contentful-paint', buffered: true });

  // 页面卸载时上报
  window.addEventListener('visibilitychange', () => {
    if (document.visibilityState === 'hidden' && lcpEntry) {
      console.log('LCP:', lcpEntry.startTime.toFixed(2) + 'ms');
    }
  });
}
```

### 4.3 自定义性能测量

```javascript
// 自定义性能测量

// 1. 组件渲染时间测量
function measureComponentRender() {
  class PerformanceComponent {
    private marks: Map<string, number> = new Map();

    startMark(name: string) {
      this.marks.set(name, performance.now());
    }

    endMark(name: string): number {
      const start = this.marks.get(name);
      if (start === undefined) {
        console.warn(`未找到标记: ${name}`);
        return 0;
      }

      const duration = performance.now() - start;
      performance.measure(name, `${name}-start`, `${name}-end`);
      this.marks.delete(name);

      return duration;
    }

    clear() {
      this.marks.clear();
      performance.clearMarks();
      performance.clearMeasures();
    }
  }

  return new PerformanceComponent();
}

// 2. API调用时间测量
async function measureAPI(endpoint: string, options?: RequestInit) {
  const start = performance.now();

  const response = await fetch(endpoint, options);

  const duration = performance.now() - start;

  return {
    response,
    duration,
    // TTFB
    ttfb: response.headers.get('date')
      ? new Date(response.headers.get('date')!).getTime() - start
      : 0,
  };
}

// 3. 帧率测量
function measureFPS() {
  let frames = 0;
  let lastTime = performance.now();

  function checkFrame() {
    frames++;
    const currentTime = performance.now();

    if (currentTime >= lastTime + 1000) {
      const fps = frames;
      console.log(`FPS: ${fps}`);

      if (fps < 30) {
        console.warn('帧率过低!');
      }

      frames = 0;
      lastTime = currentTime;
    }

    requestAnimationFrame(checkFrame);
  }

  requestAnimationFrame(checkFrame);
}
```

### 5.2 性能监控SDK

```javascript
// 性能监控SDK实现

interface PerformanceConfig {
  appId: string;
  endpoint: string;
  sampleRate: number;
  maxQueueSize: number;
}

class PerformanceMonitor {
  private config: PerformanceConfig;
  private queue: any[] = [];
  private isInitialized = false;

  constructor(config: Partial<PerformanceConfig>) {
    this.config = {
      appId: 'app',
      endpoint: '/api/performance',
      sampleRate: 1,
      maxQueueSize: 10,
      ...config,
    };
  }

  init() {
    if (this.isInitialized) return;

    this.collectWebVitals();
    this.collectResourceTiming();
    this.collectErrors();
    this.startReportTimer();
    this.setupVisibilityHandler();

    this.isInitialized = true;
  }

  private collectWebVitals() {
    import('web-vitals').then(({ onLCP, onINP, onCLS, onFCP, onTTFB }) => {
      onLCP((m) => this.addMetric('LCP', m));
      onINP((m) => this.addMetric('INP', m));
      onCLS((m) => this.addMetric('CLS', m));
      onFCP((m) => this.addMetric('FCP', m));
      onTTFB((m) => this.addMetric('TTFB', m));
    });
  }

  private collectResourceTiming() {
    const observer = new PerformanceObserver((list) => {
      list.getEntries().forEach((entry) => {
        if (entry.entryType === 'resource') {
          this.addMetric('resource', {
            name: entry.name,
            type: (entry as any).initiatorType,
            duration: entry.duration,
            size: (entry as any).transferSize,
          });
        }
      });
    });

    observer.observe({ entryTypes: ['resource'] });
  }

  private collectErrors() {
    window.addEventListener('error', (event) => {
      this.addMetric('error', {
        type: 'js',
        message: event.message,
        filename: event.filename,
        lineno: event.lineno,
        colno: event.colno,
      });
    });

    window.addEventListener('unhandledrejection', (event) => {
      this.addMetric('error', {
        type: 'promise',
        message: String(event.reason),
      });
    });
  }

  private addMetric(name: string, data: any) {
    // 采样
    if (Math.random() > this.config.sampleRate) return;

    this.queue.push({
      name,
      data,
      timestamp: Date.now(),
      url: window.location.href,
    });

    // 达到队列上限立即上报
    if (this.queue.length >= this.config.maxQueueSize) {
      this.report();
    }
  }

  private report() {
    if (this.queue.length === 0) return;

    const data = [...this.queue];
    this.queue = [];

    const body = JSON.stringify({
      appId: this.config.appId,
      metrics: data,
    });

    if (navigator.sendBeacon) {
      navigator.sendBeacon(this.config.endpoint, body);
    } else {
      fetch(this.config.endpoint, {
        method: 'POST',
        body,
        keepalive: true,
      });
    }
  }

  private startReportTimer() {
    setInterval(() => this.report(), 5000);
  }

  private setupVisibilityHandler() {
    document.addEventListener('visibilitychange', () => {
      if (document.visibilityState === 'hidden') {
        this.report();
      }
    });
  }
}

// 使用
const monitor = new PerformanceMonitor({
  appId: 'my-app',
  endpoint: '/api/performance',
  sampleRate: 0.1, // 10%采样
});

monitor.init();
```

### 问题1：有哪些常用的性能测量工具？

**答案：**

1. **Chrome DevTools**
   - Performance面板：分析页面加载和运行时性能
   - Network面板：分析网络请求
   - Memory面板：分析内存使用

   - Google开发的性能审计工具
   - 可在Chrome DevTools中运行
   - 支持命令行和CI集成

3. **WebPageTest**
   - 专业的在线性能测试工具
   - 支持多地点、多浏览器测试
   - 提供详细的瀑布流分析

4. **Performance API**
   - 浏览器原生性能测量API
   - 编程方式获取性能数据
   - 支持自定义测量

5. **web-vitals库**
   - Google的标准Web Vitals测量库
   - 提供标准化的指标收集
   - 支持自定义上报
