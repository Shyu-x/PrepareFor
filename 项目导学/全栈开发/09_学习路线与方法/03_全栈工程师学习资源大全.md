## 九、学习路线总结

### 9.1 免费学习路径推荐

```
第一阶段：Web基础（2-4周）
├── MDN Web Docs + W3Schools
├── freeCodeCamp HTML/CSS
└── Bilibili 前端入门视频

第二阶段：JavaScript核心（4-6周）
├── 《JavaScript高级程序设计》
├── 《你不知道的JavaScript》
├── LeetCode算法刷题
└── freeCodeCamp JavaScript

第三阶段：前端框架（4-8周）
├── React官方文档 + Frontend Masters
├── Next.js官方教程
└── 实战项目练习

第四阶段：后端开发（4-8周）
├── Node.js + Express
├── Full Stack Open课程
├── NestJS官方文档
└── 项目实战

第五阶段：全栈实战（8-12周）
├── 项目驱动学习
├── 构建完整全栈应用
└── 准备面试
```

### 9.2 付费学习路径推荐

```
极客时间/拉勾教育体系课 + Coursera专项课程 + Udemy项目课程
```
