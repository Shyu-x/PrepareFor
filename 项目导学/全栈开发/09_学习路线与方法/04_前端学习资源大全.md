## 资源使用建议

### 按学习阶段选择资源

**初学者阶段（0-6个月）**
1. 先从 freeCodeCamp 或 The Odin Project 开始系统学习
2. 配合 MDN Web Docs 阅读官方文档
3. 在 CodePen 和 Codewars 练习基础代码
4. 关注 YouTube 频道如 Traversy Media、Web Dev Simplified

**中级阶段（6-12个月）**
1. 深入学习一个主流框架（React/Vue）
2. 阅读框架官方文档和博客
3. 在 Frontend Mentor 做实战项目
4. 开始阅读源码，如 React、Vue 核心代码

**高级阶段（1年+）**
1. 订阅前端技术周刊，保持技术敏感度
2. 阅读高级博客如 Overreacted、Josh Comeau
3. 参与开源项目贡献
4. 系统学习计算机科学基础知识

### 按学习目标选择资源

| 目标 | 推荐资源 |
|------|----------|
| 快速找到工作 | freeCodeCamp + Frontend Mentor + 前端面试手册 |
| 深入理解框架原理 | 阅读源码 + 高级博客 + 技术周刊 |
| 提升设计能力 | Dribbble + CSS-Tricks + Kevin Powell |
| 保持技术更新 | JavaScript Weekly + Dev.to + 掘金 |
| 系统夯实基础 | MDN Web Docs + CS自学指南 |
