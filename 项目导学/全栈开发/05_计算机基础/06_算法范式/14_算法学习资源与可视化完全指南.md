**代码示例**：

```javascript
// Algorithm Visualizer 中的快速排序可视化代码
function quickSort(arr, left, right) {
  if (left < right) {
    const pivotIndex = partition(arr, left, right);
    quickSort(arr, left, pivotIndex - 1);
    quickSort(arr, pivotIndex + 1, right);
  }
  return arr;
}

function partition(arr, left, right) {
  const pivot = arr[right];
  let i = left - 1;
  for (let j = left; j < right; j++) {
    if (arr[j] <= pivot) {
      i++;
      [arr[i], arr[j]] = [arr[j], arr[i]];
    }
  }
  [arr[i + 1], arr[right]] = [arr[right], arr[i + 1]];
  return i + 1;
}
```

### 5.3 代码模板汇总

#### 5.3.1 二分查找模板

```python
def binary_search(nums, target):
    """
    二分查找标准模板
    时间复杂度：O(log n)
    空间复杂度：O(1)
    """
    left, right = 0, len(nums) - 1

    while left <= right:
        mid = left + (right - left) // 2

        if nums[mid] == target:
            return mid
        elif nums[mid] < target:
            left = mid + 1
        else:
            right = mid - 1

    return -1
```

#### 5.3.2 DFS模板

```python
def dfs(node, visited=None):
    """
    深度优先搜索标准模板
    时间复杂度：O(V + E)
    空间复杂度：O(V)
    """
    if visited is None:
        visited = set()

    # terminator
    if node in visited:
        return

    # process current node
    visited.add(node)

    # drill down
    for next_node in node.neighbors:
        if next_node not in visited:
            dfs(next_node, visited)

    # reverse state (if needed)
    visited.remove(node)
```

#### 5.3.3 BFS模板

```python
from collections import deque

def bfs(start, end):
    """
    广度优先搜索标准模板
    时间复杂度：O(V + E)
    空间复杂度：O(V)
    """
    visited = set()
    queue = deque([start])
    visited.add(start)
    step = 0

    while queue:
        size = len(queue)

        for _ in range(size):
            node = queue.popleft()

            if node == end:
                return step

            for next_node in node.neighbors:
                if next_node not in visited:
                    visited.add(next_node)
                    queue.append(next_node)

        step += 1

    return -1
```

#### 5.3.4 动态规划模板

```python
def dynamic_programming(nums):
    """
    动态规划标准模板
    时间复杂度：O(n)
    空间复杂度：O(n) 或 O(1)

    动态规划解题步骤：
    1. 定义dp数组及其含义
    2. 确定状态转移方程
    3. 初始化dp数组
    4. 确定遍历顺序
    5. 举例验证
    """
    n = len(nums)

    # 步骤1：定义dp数组
    # dp[i] 表示第i个位置的状态
    dp = [0] * n

    # 步骤3：初始化
    dp[0] = nums[0]

    # 步骤4：遍历顺序
    for i in range(1, n):
        # 步骤2：状态转移方程
        dp[i] = max(dp[i-1] + nums[i], nums[i])

    # 返回结果
    return max(dp)
```

#### 5.3.5 滑动窗口模板

```python
def sliding_window(s, k):
    """
    滑动窗口标准模板
    时间复杂度：O(n)
    空间复杂度：O(1)

    适用场景：
    - 连续子数组/子字符串问题
    - 最大/最小/最长等问题
    """
    left, right = 0, 0
    window = {}
    result = float('-inf')  # 或 float('inf')

    while right < len(s):
        # 扩大窗口
        c = s[right]
        right += 1
        window[c] = window.get(c, 0) + 1

        # 缩小窗口
        while left < right and 窗口需要收缩:
            d = s[left]
            left += 1
            window[d] -= 1

        # 更新结果
        result = max(result, right - left)

    return result
```

## 十、总结与建议

### 10.1 学习建议

1. **重视基础**：不要急于刷题，先打好数据结构和算法基础
2. **理解原理**：每个算法都要理解为什么这样设计、解决了什么问题
3. **勤于动手**：只看不动手等于没学，必须亲自实现和练习
4. **善于总结**：每做完一道题都要总结解题思路和关键点
5. **持续练习**：算法能力需要长期保持，建议每天保持一定量的练习

### 10.2 资源选择建议

1. **入门阶段**：优先使用可视化工具理解原理，配合简单题目练习
2. **基础阶段**：系统学习一本教材，配合分类刷题巩固
3. **面试阶段**：以LeetCode为主，专注高频题型和面试真题
4. **竞赛阶段**：参加竞赛平台比赛，学习高级算法和数据结构

### 10.3 学习心态建议

1. **保持耐心**：算法学习是一个长期过程，不要急于求成
2. **接受挫折**：遇到难题是正常的，不要轻易放弃
3. **循序渐进**：按照自己的节奏逐步提升难度
4. **享受过程**：把算法学习当作提升自己的机会，享受解决问题的乐趣
