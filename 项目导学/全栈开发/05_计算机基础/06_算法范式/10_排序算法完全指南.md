# 排序算法完全指南

## 一、排序算法概述

### 1.1 排序的重要性

**排序（Sorting）** 是计算机科学中最基本的问题之一：将一组数据按照指定的顺序（升序或降序）重新排列。

**类比理解——图书馆整理书籍：**

想象你在整理图书馆的书。你可以按照书名的字母顺序、作者的名字顺序、出版年份等来排列。排序就是让杂乱无章的数据变得有序，方便后续的查找和处理。

### 1.2 排序算法的分类

```
排序算法分类

比较排序（Comparison Sort）
├── O(n²) 级别
│   ├── 冒泡排序（Bubble Sort）
│   ├── 选择排序（Selection Sort）
│   └── 插入排序（Insertion Sort）
│
├── O(n log n) 级别
│   ├── 归并排序（Merge Sort）
│   ├── 快速排序（Quick Sort）
│   └── 堆排序（Heap Sort）
│
└── O(n) 级别（线性排序）
    ├── 计数排序（Counting Sort）
    ├── 桶排序（Bucket Sort）
    └── 基数排序（Radix Sort）

非比较排序
└── 不通过元素比较进行排序
```

## 二、基础排序算法

### 2.1 冒泡排序

**冒泡排序（Bubble Sort）** 是最简单的排序算法之一。

**核心思想：** 重复地遍历数组，比较相邻元素，如果顺序错误就交换，就像水中的气泡一样，大的元素会慢慢"浮"到数组末端。

```typescript
/**
 * 冒泡排序
 *
 * 算法思想：
 * - 从数组第一个元素开始，比较相邻元素
 * - 如果前者大于后者，交换
 * - 每轮遍历后，最大的元素会"冒泡"到最后
 * - 重复n轮
 *
 * 时间复杂度：O(n²)
 * 空间复杂度：O(1)
 * 稳定性：稳定
 */

function bubbleSort(arr: number[]): number[] {
  const n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    // 标记是否发生交换
    let swapped = false;

    // 每轮遍历把最大的元素放到最后
    for (let j = 0; j < n - 1 - i; j++) {
      if (arr[j] > arr[j + 1]) {
        // 交换
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
        swapped = true;
      }
    }

    // 如果没有发生交换，说明已经有序，可以提前结束
    if (!swapped) break;
  }

  return arr;
}

/**
 * 冒泡排序（优化：记录最后交换位置）
 */
function bubbleSortOptimized(arr: number[]): number[] {
  const n = arr.length;
  let lastSwap = n - 1;

  while (lastSwap > 0) {
    let lastSwapPos = 0;

    for (let j = 0; j < lastSwap; j++) {
      if (arr[j] > arr[j + 1]) {
        [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
        lastSwapPos = j;
      }
    }

    lastSwap = lastSwapPos;
  }

  return arr;
}
```

### 2.2 选择排序

**选择排序（Selection Sort）** 是一种简单直观的排序算法。

**核心思想：** 在未排序的序列中找到最小（或最大）元素，放到排序序列的起始位置，然后从剩余未排序元素中继续寻找。

```typescript
/**
 * 选择排序
 *
 * 算法思想：
 * - 在未排序序列中找到最小元素
 * - 将其放到排序序列的起始位置
 * - 从剩余未排序元素中继续寻找最小元素
 * - 重复直到所有元素排序完成
 *
 * 时间复杂度：O(n²)
 * 空间复杂度：O(1)
 * 稳定性：不稳定
 */

function selectionSort(arr: number[]): number[] {
  const n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    // 找到未排序部分的最小元素
    let minIdx = i;

    for (let j = i + 1; j < n; j++) {
      if (arr[j] < arr[minIdx]) {
        minIdx = j;
      }
    }

    // 将最小元素与未排序部分的第一个元素交换
    if (minIdx !== i) {
      [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
    }
  }

  return arr;
}

/**
 * 选择排序（同时找最大和最小）
 */
function selectionSortDouble(arr: number[]): number[] {
  const n = arr.length;
  let left = 0;
  let right = n - 1;

  while (left < right) {
    let minIdx = left;
    let maxIdx = left;

    // 在[left, right]范围内找最小和最大
    for (let j = left; j <= right; j++) {
      if (arr[j] < arr[minIdx]) minIdx = j;
      if (arr[j] > arr[maxIdx]) maxIdx = j;
    }

    // 将最小放到left，最大放到right
    [arr[left], arr[minIdx]] = [arr[minIdx], arr[left]];

    // 防止最大元素就是left位置（已交换）
    if (maxIdx === left) {
      maxIdx = minIdx;
    }

    [arr[right], arr[maxIdx]] = [arr[maxIdx], arr[right]];

    left++;
    right--;
  }

  return arr;
}
```

### 2.3 插入排序

**插入排序（Insertion Sort）** 像整理扑克牌一样。

**核心思想：** 将数组分为已排序和未排序两部分，逐个将未排序元素插入到已排序部分的正确位置。

```typescript
/**
 * 插入排序
 *
 * 算法思想：
 * - 从第二个元素开始（假设第一个已排序）
 * - 将当前元素与已排序部分从后向前比较
 * - 如果当前元素小于已排序元素，已排序元素后移
 * - 找到正确位置，插入当前元素
 *
 * 时间复杂度：O(n²)
 * 空间复杂度：O(1)
 * 稳定性：稳定
 * 特点：对于近乎有序的数组效率很高
 */

function insertionSort(arr: number[]): number[] {
  const n = arr.length;

  for (let i = 1; i < n; i++) {
    const current = arr[i];
    let j = i - 1;

    // 将比current大的元素后移
    while (j >= 0 && arr[j] > current) {
      arr[j + 1] = arr[j];
      j--;
    }

    // 插入current到正确位置
    arr[j + 1] = current;
  }

  return arr;
}

/**
 * 插入排序（带二分查找优化）
 */
function insertionSortBinary(arr: number[]): number[] {
  const n = arr.length;

  for (let i = 1; i < n; i++) {
    const current = arr[i];

    // 使用二分查找找到插入位置
    let left = 0;
    let right = i;

    while (left < right) {
      const mid = Math.floor((left + right) / 2);
      if (arr[mid] <= current) {
        left = mid + 1;
      } else {
        right = mid;
      }
    }

    // 将元素后移
    for (let j = i; j > left; j--) {
      arr[j] = arr[j - 1];
    }

    arr[left] = current;
  }

  return arr;
}

/**
 * 希尔排序（插入排序的改进版）
 *
 * 算法思想：
 * - 设置一个递减的间隔序列（gap）
 * - 对每个gap，使用插入排序
 * - 最后gap=1时就是完整的插入排序
 *
 * 时间复杂度：O(n^1.3)（平均）
 * 空间复杂度：O(1)
 * 稳定性：不稳定
 */

function shellSort(arr: number[]): number[] {
  const n = arr.length;

  // 初始间隔为 n/2，之后每次减半
  for (let gap = Math.floor(n / 2); gap > 0; gap = Math.floor(gap / 2)) {
    // 对间隔为gap的元素进行插入排序
    for (let i = gap; i < n; i++) {
      const current = arr[i];
      let j = i;

      while (j >= gap && arr[j - gap] > current) {
        arr[j] = arr[j - gap];
        j -= gap;
      }

      arr[j] = current;
    }
  }

  return arr;
}
```

## 三、高效排序算法

### 3.1 归并排序

**归并排序（Merge Sort）** 是分治法的典型应用。

**核心思想：**
1. **分：** 将数组分成两半
2. **治：** 递归地对左右两半进行排序
3. **合：** 合并两个有序数组

```typescript
/**
 * 归并排序
 *
 * 时间复杂度：O(n log n)
 * 空间复杂度：O(n)
 * 稳定性：稳定
 */

function mergeSort(arr: number[]): number[] {
  const n = arr.length;

  // 基本情况：数组长度小于等于1
  if (n <= 1) return arr;

  const mid = Math.floor(n / 2);

  // 分成两半
  const left = arr.slice(0, mid);
  const right = arr.slice(mid);

  // 递归排序并合并
  return merge(mergeSort(left), mergeSort(right));
}

/**
 * 合并两个有序数组
 */
function merge(left: number[], right: number[]): number[] {
  const result: number[] = [];
  let i = 0, j = 0;

  // 比较两个数组的元素，按顺序放入结果数组
  while (i < left.length && j < right.length) {
    if (left[i] <= right[j]) {
      result.push(left[i++]);
    } else {
      result.push(right[j++]);
    }
  }

  // 将剩余元素加入结果数组
  while (i < left.length) {
    result.push(left[i++]);
  }

  while (j < right.length) {
    result.push(right[j++]);
  }

  return result;
}

/**
 * 归并排序（原地版本）
 */
function mergeSortInPlace(arr: number[]): number[] {
  const n = arr.length;

  function sort(left: number, right: number): void {
    if (left >= right) return;

    const mid = Math.floor((left + right) / 2);
    sort(left, mid);
    sort(mid + 1, right);
    mergeInPlace(left, mid, right);
  }

  function mergeInPlace(left: number, mid: number, right: number): void {
    const temp: number[] = [];
    let i = left, j = mid + 1;

    while (i <= mid && j <= right) {
      if (arr[i] <= arr[j]) {
        temp.push(arr[i++]);
      } else {
        temp.push(arr[j++]);
      }
    }

    while (i <= mid) temp.push(arr[i++]);
    while (j <= right) temp.push(arr[j++]);

    for (let k = 0; k < temp.length; k++) {
      arr[left + k] = temp[k];
    }
  }

  sort(0, n - 1);
  return arr;
}
```

### 3.2 快速排序

**快速排序（Quick Sort）** 是最常用的排序算法。

**核心思想：**
1. 选择一个基准元素（pivot）
2. 将数组分成两部分：小于pivot的元素和大于pivot的元素
3. 递归地对两部分进行快速排序

```typescript
/**
 * 快速排序
 *
 * 时间复杂度：O(n log n)（平均），O(n²)（最坏）
 * 空间复杂度：O(log n)（递归栈）
 * 稳定性：不稳定
 */

function quickSort(arr: number[]): number[] {
  const n = arr.length;

  function partition(left: number, right: number): number {
    // 选择最右边的元素作为基准
    const pivot = arr[right];
    let i = left - 1;

    for (let j = left; j < right; j++) {
      if (arr[j] < pivot) {
        i++;
        [arr[i], arr[j]] = [arr[j], arr[i]];
      }
    }

    // 将基准放到正确位置
    [arr[i + 1], arr[right]] = [arr[right], arr[i + 1]];
    return i + 1;
  }

  function sort(left: number, right: number): void {
    if (left >= right) return;

    const pivotIndex = partition(left, right);
    sort(left, pivotIndex - 1);
    sort(pivotIndex + 1, right);
  }

  sort(0, n - 1);
  return arr;
}

/**
 * 快速排序（随机基准优化）
 * 解决数组近乎有序时退化为O(n²)的问题
 */
function quickSortRandom(arr: number[]): number[] {
  const n = arr.length;

  function partition(left: number, right: number): number {
    // 随机选择基准
    const randomIndex = left + Math.floor(Math.random() * (right - left + 1));
    [arr[randomIndex], arr[right]] = [arr[right], arr[randomIndex]];

    const pivot = arr[right];
    let i = left - 1;

    for (let j = left; j < right; j++) {
      if (arr[j] < pivot) {
        i++;
        [arr[i], arr[j]] = [arr[j], arr[i]];
      }
    }

    [arr[i + 1], arr[right]] = [arr[right], arr[i + 1]];
    return i + 1;
  }

  function sort(left: number, right: number): void {
    if (left >= right) return;

    const pivotIndex = partition(left, right);
    sort(left, pivotIndex - 1);
    sort(pivotIndex + 1, right);
  }

  sort(0, n - 1);
  return arr;
}

/**
 * 快速排序（三数取中优化）
 */
function quickSortMedian(arr: number[]): number[] {
  const n = arr.length;

  function medianOfThree(a: number, b: number, c: number): number {
    if ((arr[a] - arr[b]) * (arr[a] - arr[c]) <= 0) return a;
    if ((arr[b] - arr[a]) * (arr[b] - arr[c]) <= 0) return b;
    return c;
  }

  function partition(left: number, right: number): number {
    const mid = Math.floor((left + right) / 2);
    const pivotIndex = medianOfThree(left, mid, right);
    [arr[pivotIndex], arr[right]] = [arr[right], arr[pivotIndex]];

    const pivot = arr[right];
    let i = left - 1;

    for (let j = left; j < right; j++) {
      if (arr[j] < pivot) {
        i++;
        [arr[i], arr[j]] = [arr[j], arr[i]];
      }
    }

    [arr[i + 1], arr[right]] = [arr[right], arr[i + 1]];
    return i + 1;
  }

  function sort(left: number, right: number): void {
    if (left >= right) return;

    const pivotIndex = partition(left, right);
    sort(left, pivotIndex - 1);
    sort(pivotIndex + 1, right);
  }

  sort(0, n - 1);
  return arr;
}
```

### 3.3 堆排序

**堆排序（Heap Sort）** 利用堆这种数据结构进行排序。

**核心思想：**
1. 构建最大堆
2. 重复提取最大元素并调整堆

```typescript
/**
 * 堆排序
 *
 * 时间复杂度：O(n log n)
 * 空间复杂度：O(1)
 * 稳定性：不稳定
 */

function heapSort(arr: number[]): number[] {
  const n = arr.length;

  /**
   * 构建最大堆
   */
  function buildMaxHeap(): void {
    // 从最后一个非叶子节点开始下沉
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
      heapify(i, n);
    }
  }

  /**
   * 堆化：从节点i开始下沉
   */
  function heapify(i: number, size: number): void {
    let largest = i;
    const left = 2 * i + 1;
    const right = 2 * i + 2;

    if (left < size && arr[left] > arr[largest]) {
      largest = left;
    }

    if (right < size && arr[right] > arr[largest]) {
      largest = right;
    }

    if (largest !== i) {
      [arr[i], arr[largest]] = [arr[largest], arr[i]];
      heapify(largest, size);
    }
  }

  // 构建最大堆
  buildMaxHeap();

  // 逐个提取最大元素
  for (let i = n - 1; i > 0; i--) {
    // 将最大元素移到数组末尾
    [arr[0], arr[i]] = [arr[i], arr[0]];
    // 对剩余元素重新堆化
    heapify(0, i);
  }

  return arr;
}

/**
 * 堆（最小堆）实现
 */
class MinHeap {
  private heap: number[] = [];

  /**
   * 获取父节点索引
   */
  private parent(i: number): number {
    return Math.floor((i - 1) / 2);
  }

  /**
   * 获取左子节点索引
   */
  private leftChild(i: number): number {
    return 2 * i + 1;
  }

  /**
   * 获取右子节点索引
   */
  private rightChild(i: number): number {
    return 2 * i + 2;
  }

  /**
   * 上浮操作
   */
  private swim(i: number): void {
    while (i > 0 && this.heap[this.parent(i)] > this.heap[i]) {
      [this.heap[this.parent(i)], this.heap[i]] =
        [this.heap[i], this.heap[this.parent(i)]];
      i = this.parent(i);
    }
  }

  /**
   * 下沉操作
   */
  private sink(i: number): void {
    const n = this.heap.length;

    while (this.leftChild(i) < n) {
      let smallest = this.leftChild(i);

      if (this.rightChild(i) < n &&
          this.heap[this.rightChild(i)] < this.heap[smallest]) {
        smallest = this.rightChild(i);
      }

      if (this.heap[i] <= this.heap[smallest]) break;

      [this.heap[i], this.heap[smallest]] =
        [this.heap[smallest], this.heap[i]];

      i = smallest;
    }
  }

  /**
   * 插入
   */
  insert(val: number): void {
    this.heap.push(val);
    this.swim(this.heap.length - 1);
  }

  /**
   * 提取最小元素
   */
  extractMin(): number | null {
    if (this.heap.length === 0) return null;

    const min = this.heap[0];
    const last = this.heap.pop()!;

    if (this.heap.length > 0) {
      this.heap[0] = last;
      this.sink(0);
    }

    return min;
  }

  /**
   * 获取最小元素（不删除）
   */
  peek(): number | null {
    return this.heap.length > 0 ? this.heap[0] : null;
  }

  /**
   * 获取堆大小
   */
  size(): number {
    return this.heap.length;
  }
}
```

## 四、线性排序算法

### 4.1 计数排序

**计数排序（Counting Sort）** 是一种非比较排序，适用于范围较小的整数排序。

**核心思想：** 统计每个元素出现的次数，然后根据计数将元素放到正确位置。

```typescript
/**
 * 计数排序
 *
 * 时间复杂度：O(n + k)，k为数据范围
 * 空间复杂度：O(k)
 * 稳定性：稳定
 * 限制：只适用于整数，且范围不能太大
 */

function countingSort(arr: number[]): number[] {
  if (arr.length === 0) return arr;

  // 找到数据范围
  let min = arr[0], max = arr[0];
  for (const num of arr) {
    if (num < min) min = num;
    if (num > max) max = num;
  }

  // 创建计数数组
  const range = max - min + 1;
  const count = new Array(range).fill(0);

  // 统计每个元素出现的次数
  for (const num of arr) {
    count[num - min]++;
  }

  // 将计数转换为累积计数（用于稳定性）
  for (let i = 1; i < range; i++) {
    count[i] += count[i - 1];
  }

  // 从后向前遍历，保证稳定性
  const result = new Array(arr.length);
  for (let i = arr.length - 1; i >= 0; i--) {
    const num = arr[i];
    result[count[num - min] - 1] = num;
    count[num - min]--;
  }

  return result;
}

/**
 * 计数排序（非稳定版本）
 */
function countingSortUnstable(arr: number[]): number[] {
  if (arr.length === 0) return arr;

  let min = arr[0], max = arr[0];
  for (const num of arr) {
    if (num < min) min = num;
    if (num > max) max = num;
  }

  const range = max - min + 1;
  const count = new Array(range).fill(0);

  for (const num of arr) {
    count[num - min]++;
  }

  let index = 0;
  for (let i = 0; i < range; i++) {
    while (count[i] > 0) {
      arr[index++] = i + min;
      count[i]--;
    }
  }

  return arr;
}
```

### 4.2 桶排序

**桶排序（Bucket Sort）** 是计数排序的扩展，将数据分到有限数量的桶中，每个桶内分别排序。

```typescript
/**
 * 桶排序
 *
 * 时间复杂度：O(n + k)（平均），O(n²)（最坏）
 * 空间复杂度：O(n + k)
 * 稳定性：取决于桶内排序算法
 * 适用：数据分布均匀的情况
 */

function bucketSort(arr: number[], bucketSize: number = 5): number[] {
  if (arr.length === 0) return arr;

  // 找到数据范围
  let min = arr[0], max = arr[0];
  for (const num of arr) {
    if (num < min) min = num;
    if (num > max) max = num;
  }

  // 计算桶的数量
  const bucketCount = Math.floor((max - min) / bucketSize) + 1;
  const buckets: number[][] = Array.from({length: bucketCount}, () => []);

  // 将元素分配到桶中
  for (const num of arr) {
    const bucketIndex = Math.floor((num - min) / bucketSize);
    buckets[bucketIndex].push(num);
  }

  // 对每个桶进行排序
  for (const bucket of buckets) {
    bucket.sort((a, b) => a - b);
  }

  // 合并所有桶
  const result: number[] = [];
  for (const bucket of buckets) {
    result.push(...bucket);
  }

  return result;
}

/**
 * 桶排序（使用链表优化插入）
 */
function bucketSortWithLinkedList(
  arr: number[],
  bucketSize: number = 5
): number[] {
  if (arr.length === 0) return arr;

  let min = arr[0], max = arr[0];
  for (const num of arr) {
    if (num < min) min = num;
    if (num > max) max = num;
  }

  const bucketCount = Math.floor((max - min) / bucketSize) + 1;
  const buckets: number[][] = Array.from({length: bucketCount}, () => []);

  for (const num of arr) {
    const bucketIndex = Math.floor((num - min) / bucketSize);
    buckets[bucketIndex].push(num);
  }

  const result: number[] = [];
  for (const bucket of buckets) {
    if (bucket.length === 0) continue;
    // 对于接近有序的数据，插入排序更快
    insertionSortInPlace(bucket);
    result.push(...bucket);
  }

  return result;
}

function insertionSortInPlace(arr: number[]): void {
  for (let i = 1; i < arr.length; i++) {
    const current = arr[i];
    let j = i - 1;
    while (j >= 0 && arr[j] > current) {
      arr[j + 1] = arr[j];
      j--;
    }
    arr[j + 1] = current;
  }
}
```

### 4.3 基数排序

**基数排序（Radix Sort）** 按位数进行排序，从低位到高位（或从高位到低位）。

```typescript
/**
 * 基数排序（LSD - Least Significant Digit）
 *
 * 时间复杂度：O(nk)，k为数字位数
 * 空间复杂度：O(n + k)
 * 稳定性：稳定
 * 适用：整数或字符串排序
 */

function radixSort(arr: number[]): number[] {
  if (arr.length === 0) return arr;

  // 找到最大值
  let max = Math.abs(arr[0]);
  for (const num of arr) {
    if (Math.abs(num) > max) max = Math.abs(num);
  }

  // 处理负数：分成负数和正数两部分
  const negatives = arr.filter(x => x < 0).map(x => -x);
  const positives = arr.filter(x => x >= 0);

  // 对正数部分进行基数排序
  const sortedPositives = radixSortPositive(positives);
  const sortedNegatives = radixSortPositive(negatives).reverse().map(x => -x);

  return [...sortedNegatives, ...sortedPositives];
}

function radixSortPositive(arr: number[]): number[] {
  if (arr.length === 0) return arr;

  // 找到最大值，确定位数
  let max = arr[0];
  for (const num of arr) {
    if (num > max) max = num;
  }

  // 从个位开始，逐位排序
  let exp = 1;
  while (Math.floor(max / exp) > 0) {
    arr = countingSortByDigit(arr, exp);
    exp *= 10;
  }

  return arr;
}

function countingSortByDigit(arr: number[], exp: number): number[] {
  const n = arr.length;
  const result = new Array(n);
  const count = new Array(10).fill(0);

  // 统计每个数字出现的次数
  for (let i = 0; i < n; i++) {
    const digit = Math.floor(arr[i] / exp) % 10;
    count[digit]++;
  }

  // 累积计数
  for (let i = 1; i < 10; i++) {
    count[i] += count[i - 1];
  }

  // 从后向前遍历，保证稳定性
  for (let i = n - 1; i >= 0; i--) {
    const digit = Math.floor(arr[i] / exp) % 10;
    result[count[digit] - 1] = arr[i];
    count[digit]--;
  }

  return result;
}

/**
 * 基数排序（字符串版本）
 */
function radixSortStrings(arr: string[]): string[] {
  if (arr.length === 0) return arr;

  // 找到最长的字符串
  let maxLen = 0;
  for (const s of arr) {
    if (s.length > maxLen) maxLen = s.length;
  }

  // 从最后一个字符开始排序
  for (let i = maxLen - 1; i >= 0; i--) {
    arr = countingSortByChar(arr, i);
  }

  return arr;
}

function countingSortByChar(arr: string[], index: number): string[] {
  const n = arr.length;
  const result = new Array(n);
  const count = new Array(256).fill(0); // ASCII字符

  // 统计每个字符出现的次数
  for (let i = 0; i < n; i++) {
    const charCode = index < arr[i].length
      ? arr[i].charCodeAt(index)
      : 0;
    count[charCode]++;
  }

  // 累积计数
  for (let i = 1; i < 256; i++) {
    count[i] += count[i - 1];
  }

  // 从后向前遍历
  for (let i = n - 1; i >= 0; i--) {
    const charCode = index < arr[i].length
      ? arr[i].charCodeAt(index)
      : 0;
    result[count[charCode] - 1] = arr[i];
    count[charCode]--;
  }

  return result;
}
```

### 5.1 合并区间

```typescript
/**
 * 排序应用：合并区间
 */
function mergeIntervals(intervals: number[][]): number[][] {
  if (intervals.length <= 1) return intervals;

  // 按起点排序
  intervals.sort((a, b) => a[0] - b[0]);

  const result: number[][] = [intervals[0]];

  for (let i = 1; i < intervals.length; i++) {
    const current = intervals[i];
    const last = result[result.length - 1];

    if (current[0] <= last[1]) {
      // 有重叠，合并
      last[1] = Math.max(last[1], current[1]);
    } else {
      // 无重叠，直接加入
      result.push(current);
    }
  }

  return result;
}
```

### 5.2 排序搜索

```typescript
/**
 * 排序搜索：在排序数组中查找元素位置
 */
function binarySearch(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = left + Math.floor((right - left) / 2);

    if (arr[mid] === target) return mid;
    if (arr[mid] < target) {
      left = mid + 1;
    } else {
      right = mid - 1;
    }
  }

  return -1;
}

/**
 * 查找元素第一次出现的位置
 */
function lowerBound(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length;

  while (left < right) {
    const mid = left + Math.floor((right - left) / 2);

    if (arr[mid] < target) {
      left = mid + 1;
    } else {
      right = mid;
    }
  }

  return left;
}

/**
 * 查找元素最后一次出现的位置
 */
function upperBound(arr: number[], target: number): number {
  let left = 0;
  let right = arr.length;

  while (left < right) {
    const mid = left + Math.floor((right - left) / 2);

    if (arr[mid] <= target) {
      left = mid + 1;
    } else {
      right = mid;
    }
  }

  return left - 1;
}
```

## 七、总结

### 7.1 排序算法选择指南

```
排序算法选择

数据规模小（n < 50）：
- 插入排序：简单高效
- 冒泡排序：不推荐

数据规模中等（n < 1000）：
- 快速排序：平均最快
- 归并排序：稳定

数据规模大（n > 10000）：
- 快速排序：平均最快
- 堆排序：最坏情况保证O(n log n)

整数排序，范围小：
- 计数排序：O(n)

浮点数或均匀分布：
- 桶排序：O(n)

稳定排序：
- 归并排序
- 插入排序
- 计数排序
- 桶排序

原地排序：
- 快速排序
- 堆排序
```

### 7.2 排序算法核心要点

```
排序算法要点

O(n²) 算法：
- 冒泡：稳定，每轮冒泡最大元素
- 选择：不稳定，每轮选最小
- 插入：稳定，对近乎有序数据效率高

O(n log n) 算法：
- 归并：稳定，分治思想，需要O(n)额外空间
- 快排：不稳定，平均最快，递归栈O(log n)
- 堆排：不稳定，原地排序

O(n) 算法：
- 计数：适用于整数，范围不能太大
- 桶排：适用于均匀分布
- 基数：按位排序，适用于整数或字符串

实际应用：
- JavaScript: Array.sort() 使用 Timsort
- Python: sorted() 使用 Timsort
- Chrome V8: 快速排序 + 插入排序混合
```
