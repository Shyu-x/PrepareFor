# 位运算与比特算法完全指南

## 一、位运算基础

### 1.1 位运算概述

**位运算（Bitwise Operations）** 是直接对整数的二进制位进行操作。

**类比理解——开关灯：**

想象你有一排开关，每个开关只有开（1）和关（0）两种状态。位运算就像同时操作多个开关，比如：
- 打开第3个开关（设置某位为1）
- 关闭第5个开关（设置某位为0）
- 检查第2个开关是开还是关（检查某位）

### 1.2 六种基本位运算

```typescript
/**
 * 六种基本位运算
 */

// 1. AND（按位与）&
console.log(5 & 3);  // 101 & 011 = 001 = 1
// 特点：两位都为1时结果为1
// 应用：判断奇偶 (n & 1)

// 2. OR（按位或）|
console.log(5 | 3);  // 101 | 011 = 111 = 7
// 特点：任一位为1时结果为1
// 应用：设置某位为1

// 3. XOR（按位异或）^
console.log(5 ^ 3);  // 101 ^ 011 = 110 = 6
// 特点：两位相异时结果为1，相同为0
// 应用：交换两数、找出唯一数

// 4. NOT（按位取反）~
console.log(~5);      // ~00000101 = 11111010 = -6
// 特点：0变1,1变0（包括符号位）
// 注意：JavaScript中整数是32位

// 5. 左移 <<
console.log(5 << 1);  // 101 << 1 = 1010 = 10
// 特点：二进制左移，右边补0
// 应用：乘以2^n

// 6. 右移 >>
console.log(5 >> 1);  // 101 >> 1 = 10 = 2
// 特点：二进制右移，左边补符号位
// 应用：除以2^n（有符号）

// 无符号右移 >>>
console.log(-5 >>> 1); // 一个很大的正数
// 特点：左边始终补0
```

### 1.3 位运算速查表

| 运算 | 符号 | 示例 | 结果 |
|------|------|------|------|
| AND | & | 5 & 3 | 1 |
| OR | \| | 5 \| 3 | 7 |
| XOR | ^ | 5 ^ 3 | 6 |
| NOT | ~ | ~5 | -6 |

## 二、位运算技巧

### 2.1 常用位运算技巧

```typescript
/**
 * 常用位运算技巧
 */

// 技巧1：判断奇偶
function isOdd(n: number): boolean {
  return (n & 1) === 1;
}
// 等价于：n % 2 === 1
// 原理：奇数的二进制最后一位是1

// 技巧2：交换两个数（不使用临时变量）
function swap(a: number, b: number): [number, number] {
  a = a ^ b;
  b = a ^ b;  // b = (a ^ b) ^ b = a
  a = a ^ b;  // a = (a ^ b) ^ a = b
  return [a, b];
}

// 技巧3：取绝对值
function abs(n: number): number {
  const mask = n >> 31;  // 获取符号位：负数为-1，正数为0
  return (n ^ mask) - mask;
  // 原理：正数 ^ 0 = 正数，负数 ^ -1 = ~负数 + 1 = 正数
}

// 技巧4：获取最高位1的位置
function highestBit(n: number): number {
  let pos = 0;
  if (n > 0xFFFF) { n >>= 16; pos += 16; }
  if (n > 0xFF) { n >>= 8; pos += 8; }
  if (n > 0xF) { n >>= 4; pos += 4; }
  if (n > 0x3) { n >>= 2; pos += 2; }
  if (n > 0x1) { pos += 1; }
  return pos;
}

// 技巧5：判断是否是2的幂
function isPowerOfTwo(n: number): boolean {
  return n > 0 && (n & (n - 1)) === 0;
}
// 原理：2的幂只有一位是1，如 1000
// n-1 会把最低位的1变成0，如 0111
// 所以 n & (n-1) = 0

// 技巧6：统计二进制中1的个数
function countBits(n: number): number {
  let count = 0;
  while (n > 0) {
    n &= (n - 1);  // 消除最低位的1
    count++;
  }
  return count;
}

// 技巧7：获取第i位的值（0或1）
function getBit(n: number, i: number): number {
  return (n >> i) & 1;
}

// 技巧8：设置第i位为1
function setBit(n: number, i: number): number {
  return n | (1 << i);
}

// 技巧9：设置第i位为0
function clearBit(n: number, i: number): number {
  return n & ~(1 << i);
}

// 技巧10：更新第i位（设置为v）
function updateBit(n: number, i: number, v: number): number {
  const mask = ~(1 << i);
  return (n & mask) | (v << i);
}
```

### 2.2 高级位运算技巧

```typescript
/**
 * 高级位运算技巧
 */

// 技巧11：获取最低位的1（保留最低位的1）
function lowestBit(n: number): number {
  return n & (-n);
}
// 原理：负数的二进制是补码
// 如 n = 12 = 1100，-n = -12 = ...0100
// n & (-n) = 0100 = 4

// 技巧12：消除最低位的1
function eliminateLowestBit(n: number): number {
  return n & (n - 1);
}
// 原理：n-1会从最低位的1开始借位
// 如 n = 12 = 1100，n-1 = 11 = 1011
// n & (n-1) = 1000 = 8

// 技巧13：判断两位是否不同
function bitsDiffer(a: number, b: number, i: number, j: number): boolean {
  return ((a >> i) & 1) !== ((b >> j) & 1);
}
// 或者：((a ^ b) >> i) & 1

// 技巧14：生成掩码
function generateMask(start: number, end: number): number {
  return ((1 << (end - start + 1)) - 1) << start;
}
// 生成从start到end位都是1的掩码

// 技巧15：循环左移
function rotateLeft(n: number, d: number, bits: number = 32): number {
  return ((n << d) | (n >>> (bits - d))) & ((1 << bits) - 1);
}

// 技巧16：循环右移
function rotateRight(n: number, d: number, bits: number = 32): number {
  return ((n >>> d) | (n << (bits - d))) & ((1 << bits) - 1);
}

// 技巧17：加法（不使用+号）
function add(a: number, b: number): number {
  while (b !== 0) {
    const carry = a & b;  // 进位
    a = a ^ b;            // 无进位加法
    b = carry << 1;       // 进位左移
  }
  return a;
}

// 技巧18：减法（使用加法和取反）
function subtract(a: number, b: number): number {
  return add(a, add(~b, 1));
}

// 技巧19：乘法（位运算版）
function multiply(a: number, b: number): number {
  let result = 0;
  while (b > 0) {
    if (b & 1) {
      result = add(result, a);
    }
    a <<= 1;
    b >>= 1;
  }
  return result;
}

// 技巧20：除法（位运算版）
function divide(dividend: number, divisor: number): number {
  if (divisor === 0) return Infinity;
  const isNegative = (dividend < 0) !== (divisor < 0);
  let a = Math.abs(dividend);
  const b = Math.abs(divisor);

  let quotient = 0;
  for (let i = 31; i >= 0; i--) {
    if ((a >>> i) >= b) {
      quotient |= (1 << i);
      a -= (b << i);
    }
  }

  return isNegative ? -quotient : quotient;
}
```

## 三、位运算应用

### 3.1 状态压缩DP

**状态压缩DP** 是利用二进制位来表示状态进行动态规划。

```typescript
/**
 * 状态压缩DP
 *
 * 适用场景：
 * - 问题的状态可以用二进制位表示
 * - 状态数不超过 20-30 个
 *
 * 示例：旅行商问题的状态压缩DP
 */

function tspStateCompression(dist: number[][]): number {
  const n = dist.length;
  if (n <= 1) return 0;

  const INF = Infinity;
  const fullMask = (1 << n) - 1;

  // dp[mask][i] = 访问集合mask中的城市，以i结尾的最短路径
  const dp = Array.from({length: 1 << n}, () => new Array(n).fill(INF));
  dp[1][0] = 0;

  for (let mask = 1; mask <= fullMask; mask++) {
    for (let i = 0; i < n; i++) {
      if (!(mask & (1 << i))) continue;

      const prevMask = mask ^ (1 << i);
      if (prevMask === 0) continue;

      for (let j = 0; j < n; j++) {
        if (prevMask & (1 << j)) {
          dp[mask][i] = Math.min(dp[mask][i], dp[prevMask][j] + dist[j][i]);
        }
      }
    }
  }

  let ans = INF;
  for (let i = 1; i < n; i++) {
    ans = Math.min(ans, dp[fullMask][i] + dist[i][0]);
  }

  return ans;
}

/**
 * 蒙德里安的梦想（多米诺骨牌铺砖问题）
 */
function dominosCount(n: number, m: number): number {
  if (m > n) [n, m] = [m, n];

  const fullMask = (1 << m) - 1;

  // 检查状态是否有效（没有连续奇数个0）
  const valid = new Array(1 << m).fill(true);
  for (let mask = 0; mask <= fullMask; mask++) {
    let cnt = 0;
    for (let i = 0; i < m; i++) {
      if (mask & (1 << i)) {
        if (cnt % 2 === 1) {
          valid[mask] = false;
          break;
        }
        cnt = 0;
      } else {
        cnt++;
      }
    }
    if (cnt % 2 === 1) valid[mask] = false;
  }

  // 预计算状态兼容性
  const compatible: boolean[][] = [];
  for (let state1 = 0; state1 <= fullMask; state1++) {
    compatible[state1] = [];
    for (let state2 = 0; state2 <= fullMask; state2++) {
      if (valid[state1] && valid[state2] && (state1 & state2) === 0) {
        const merged = state1 | state2;
        compatible[state1][state2] = valid[merged];
      }
    }
  }

  // dp[i][state] = 第i行状态为state的方案数
  const dp = Array.from({length: n + 1}, () => new Array(1 << m).fill(0));
  dp[0][fullMask] = 1;

  for (let i = 1; i <= n; i++) {
    for (let state = 0; state <= fullMask; state++) {
      if (!valid[state]) continue;

      let sum = 0;
      for (let prevState = 0; prevState <= fullMask; prevState++) {
        if (compatible[prevState][state]) {
          sum += dp[i - 1][prevState];
        }
      }
      dp[i][state] = sum;
    }
  }

  return dp[n][fullMask];
}
```

### 3.2 布隆过滤器

**布隆过滤器（Bloom Filter）** 使用位数组和多个哈希函数来快速判断元素是否存在。

```typescript
/**
 * 布隆过滤器
 *
 * 特点：
 * - 空间效率高
 * - 可能存在假阳性（说存在可能不存在）
 * - 不存在的一定不存在（假阴性率为0）
 *
 * 应用：
 * - 缓存穿透防护
 * - 黑名单过滤
 * - 网页去重
 */

class BloomFilter {
  private bits: number[];  // 位数组
  private size: number;     // 位数组大小
  private hashCount: number; // 哈希函数数量

  /**
   * @param expectedItems 期望存储的元素数量
   * @param falsePositiveRate 期望的假阳性率
   */
  constructor(expectedItems: number, falsePositiveRate: number = 0.01) {
    // 计算位数组大小
    // m = -n * ln(p) / (ln(2)^2)
    this.size = Math.ceil(
      -expectedItems * Math.log(falsePositiveRate) / (Math.log(2) ** 2)
    );

    // 计算哈希函数数量
    // k = (m/n) * ln(2)
    this.hashCount = Math.ceil((this.size / expectedItems) * Math.log(2));

    // 初始化位数组
    this.bits = new Array(Math.ceil(this.size / 32)).fill(0);
  }

  /**
   * 添加元素
   */
  add(item: string): void {
    for (let i = 0; i < this.hashCount; i++) {
      const index = this.getHash(item, i);
      this.bits[Math.floor(index / 32)] |= (1 << (index % 32));
    }
  }

  /**
   * 检查元素是否存在
   */
  contains(item: string): boolean {
    for (let i = 0; i < this.hashCount; i++) {
      const index = this.getHash(item, i);
      if ((this.bits[Math.floor(index / 32)] & (1 << (index % 32))) === 0) {
        return false;
      }
    }
    return true;
  }

  /**
   * 生成哈希值
   */
  private getHash(item: string, seed: number): number {
    let hash1 = this.hash1(item);
    let hash2 = this.hash2(item);
    return (hash1 + seed * hash2) % this.size;
  }

  private hash1(s: string): number {
    let hash = 0;
    for (let i = 0; i < s.length; i++) {
      hash = ((hash << 5) - hash + s.charCodeAt(i)) | 0;
    }
    return Math.abs(hash);
  }

  private hash2(s: string): number {
    let hash = 5381;
    for (let i = 0; i < s.length; i++) {
      hash = ((hash << 5) + hash) ^ s.charCodeAt(i);
      hash = hash & hash;  // 强制转换为32位整数
    }
    return Math.abs(hash);
  }
}

/**
 * 布隆过滤器使用示例
 */
function bloomFilterExample(): void {
  const bloom = new BloomFilter(1000, 0.01);

  // 添加元素
  bloom.add("hello");
  bloom.add("world");
  bloom.add("bit");

  // 检查元素
  console.log("hello exists:", bloom.contains("hello")); // true
  console.log("world exists:", bloom.contains("world")); // true
  console.log("foo exists:", bloom.contains("foo")); // false (可能)

  // 注意：布隆过滤器可能出现假阳性
  // 即它说"存在"但实际上不存在
}
```

### 3.3 位图（Bitmap）

**位图（Bitmap）** 使用位数组来表示大量布尔值。

```typescript
/**
 * 位图（Bitmap）
 *
 * 用于在有限集合中快速查找、添加、删除元素
 * 常用于数据库索引、缓存淘汰算法等
 */

class Bitmap {
  private bits: number[];
  private size: number;

  constructor(size: number) {
    this.size = size;
    // 每个number有32位
    this.bits = new Array(Math.ceil(size / 32)).fill(0);
  }

  /**
   * 设置第i位为1
   */
  set(i: number): void {
    if (i < 0 || i >= this.size) return;
    this.bits[Math.floor(i / 32)] |= (1 << (i % 32));
  }

  /**
   * 设置第i位为0
   */
  clear(i: number): void {
    if (i < 0 || i >= this.size) return;
    this.bits[Math.floor(i / 32)] &= ~(1 << (i % 32));
  }

  /**
   * 获取第i位的值
   */
  get(i: number): boolean {
    if (i < 0 || i >= this.size) return false;
    return (this.bits[Math.floor(i / 32)] & (1 << (i % 32))) !== 0;
  }

  /**
   * 将另一个位图OR到当前位图
   */
  or(other: Bitmap): void {
    if (this.size !== other.size) return;
    for (let i = 0; i < this.bits.length; i++) {
      this.bits[i] |= other.bits[i];
    }
  }

  /**
   * 将另一个位图AND到当前位图
   */
  and(other: Bitmap): void {
    if (this.size !== other.size) return;
    for (let i = 0; i < this.bits.length; i++) {
      this.bits[i] &= other.bits[i];
    }
  }
}

/**
 * 位图排序（对一定范围内的整数排序）
 */
function bitmapSort(arr: number[]): number[] {
  if (arr.length === 0) return [];

  // 找到范围
  let min = arr[0], max = arr[0];
  for (const num of arr) {
    if (num < min) min = num;
    if (num > max) max = num;
  }

  // 创建位图
  const bitmap = new Bitmap(max - min + 1);

  // 设置位
  for (const num of arr) {
    bitmap.set(num - min);
  }

  // 读取排序结果
  const result: number[] = [];
  for (let i = 0; i <= max - min; i++) {
    if (bitmap.get(i)) {
      result.push(i + min);
    }
  }

  return result;
}
```

## 四、常见问题

### 4.1 只出现一次的数字

```typescript
/**
 * 问题1：数组中除一个元素外，其他元素都出现两次，找出只出现一次的元素
 * 异或运算：相同为0，不同为1，且满足交换律和结合律
 */
function singleNumber(nums: number[]): number {
  let result = 0;
  for (const num of nums) {
    result ^= num;
  }
  return result;
}

/**
 * 问题2：数组中除一个元素外，其他元素都出现两次，找出只出现一次的元素（进阶版）
 * 需要同时找出两个只出现一次的元素
 */
function singleNumberII(nums: number[]): number[] {
  // 第一步：所有元素异或，得到 a ^ b
  let xor = 0;
  for (const num of nums) {
    xor ^= num;
  }

  // 第二步：找到xor中最低位的1
  let lowestBit = xor & (-xor);

  // 第三步：按照lowestBit分组，分别异或
  let a = 0, b = 0;
  for (const num of nums) {
    if ((num & lowestBit) === 0) {
      a ^= num;
    } else {
      b ^= num;
    }
  }

  return [a, b];
}

/**
 * 问题3：数组中除一个元素外，其他元素都出现三次，找出只出现一次的元素
 */
function singleNumberIII(nums: number[]): number {
  // 使用位运算统计每一位上1出现的次数
  // 如果出现3次，那么每一位上1的总数应该是3的倍数
  // 剩下的余数就是只出现一次的元素

  let result = 0;

  // 遍历每一位（0-31）
  for (let bit = 0; bit < 32; bit++) {
    let count = 0;
    for (const num of nums) {
      if ((num >> bit) & 1) {
        count++;
      }
    }

    // 如果count % 3 !== 0，说明只出现一次的元素在这一位是1
    if (count % 3 !== 0) {
      result |= (1 << bit);
    }
  }

  return result;
}
```

/**
 * 问题4：计算两个整数的和（不能使用+号）
 */
function getSum(a: number, b: number): number {
  while (b !== 0) {
    const carry = (a & b) << 1;
    a = a ^ b;
    b = carry;
  }
  return a;
}

/**
 * 问题5：判断一个数是否是2的幂

/**
 * 问题6：判断一个数是否是4的幂
 */
function isPowerOfFour(n: number): boolean {
  if (n <= 0) return false;
  // 4的幂特点：二进制只有一位为1，且在奇数位
  // 如 4=100, 16=10000, 64=1000000
  if ((n & (n - 1)) !== 0) return false;
  return (n & 0x55555555) !== 0;  // 0x55555555 = 010101010101...
}

/**
 * 问题7：颠倒整数的二进制位
 */
function reverseBits(n: number): number {
  let result = 0;
  for (let i = 0; i < 32; i++) {
    result = (result << 1) | (n & 1);
    n >>= 1;
  }
  return result >>> 0;  // 转为无符号整数
}

/**
 * 问题8：字符的所有组合（幂集）
 */
function subsets(chars: string[]): string[][] {
  const n = chars.length;
  const result: string[][] = [];

  // 2^n种组合，用位掩码表示
  for (let mask = 0; mask < (1 << n); mask++) {
    const subset: string[] = [];
    for (let i = 0; i < n; i++) {
      if (mask & (1 << i)) {
        subset.push(chars[i]);
      }
    }
    result.push(subset);
  }

  return result;
}

/**
 * 问题10：比特计数（汉明权重）
 * 计算n的二进制表示中有多少个1
 */
function hammingWeight(n: number): number {
  let count = 0;
  while (n !== 0) {
    n &= (n - 1);  // 消除最低位的1
    count++;
  }
  return count;
}

/**
 * 比特计数（查表法，更快）
 */
function hammingWeightFast(n: number): number {
  const table = [
    0, 1, 1, 2, 1, 2, 2, 3, 1, 2, 2, 3, 2, 3, 3, 4,
    1, 2, 2, 3, 2, 3, 3, 4, 2, 3, 3, 4, 3, 4, 4, 5
  ];

  let count = 0;
  while (n !== 0) {
    count += table[n & 0xF];  // 每次处理4位
    n >>= 4;
  }
  return count;
}

### 4.3 权限管理

```typescript
/**
 * 位运算在权限管理中的应用
 */

// 定义权限常量
const Permission = {
  READ: 1 << 0,    // 1 = 0001
  WRITE: 1 << 1,   // 2 = 0010
  EXECUTE: 1 << 2, // 4 = 0100
  DELETE: 1 << 3   // 8 = 1000
} as const;

/**
 * 权限系统
 */
class PermissionManager {
  private permissions: number = 0;

  /**
   * 添加权限
   */
  grant(permission: number): void {
    this.permissions |= permission;
  }

  /**
   * 移除权限
   */
  revoke(permission: number): void {
    this.permissions &= ~permission;
  }

  /**
   * 检查是否有权限
   */
  has(permission: number): boolean {
    return (this.permissions & permission) === permission;
  }

  /**
   * 检查是否有所有权限
   */
  hasAll(permissions: number): boolean {
    return (this.permissions & permissions) === permissions;
  }

  /**
   * 检查是否有任一权限
   */
  hasAny(permissions: number): boolean {
    return (this.permissions & permissions) !== 0;
  }

  /**
   * 获取当前权限
   */
  getPermissions(): number {
    return this.permissions;
  }
}

/**
 * 使用示例
 */
function permissionExample(): void {
  const user = new PermissionManager();

  // 添加读和写权限
  user.grant(Permission.READ);
  user.grant(Permission.WRITE);

  console.log("有读权限:", user.has(Permission.READ)); // true
  console.log("有执行权限:", user.has(Permission.EXECUTE)); // false

  // 添加执行权限
  user.grant(Permission.EXECUTE);

  console.log("有读和执行:", user.hasAll(Permission.READ | Permission.EXECUTE)); // true
}
```

## 六、总结

### 6.1 位运算核心要点

```
位运算要点

六种基本运算：
- AND (&)：两位都为1结果为1
- OR (|)：任一位为1结果为1
- XOR (^)：两位相异结果为1
- NOT (~)：0变1,1变0
- << 左移：右边补0
- >> 右移：左边补符号位

常用技巧：
- 判断奇偶：n & 1
- 消除最低位的1：n & (n-1)
- 获取最低位的1：n & (-n)
- 判断2的幂：n > 0 && (n & (n-1)) === 0
- 交换两数：a ^= b; b ^= a; a ^= b;

位运算应用：
- 状态压缩DP
- 布隆过滤器
- 位图排序
- 权限管理
```

### 6.2 位运算优先级

```
位运算优先级（从高到低）
1. ~
2. << >>
3. &
4. ^
5. |

建议：使用括号明确优先级
```

### 6.3 位运算与其它算法的关系

```
位运算与其它算法

动态规划：
- 状态压缩DP利用位运算表示状态

字符串算法：
- Trie树用位运算优化

排序算法：
- 位图排序

图算法：
- 邻接矩阵用位运算优化存储

缓存：
- LRU/LFU可以用位运算优化
```
