# 分支限界法与随机算法完全指南

## 一、分支限界法基础

### 1.1 什么是分支限界法？

**分支限界法（Branch and Bound）** 是一种用于求解组合优化问题的算法设计技术。

**类比理解——迷宫寻宝：**

想象你在迷宫中寻找宝藏，迷宫很大，岔路很多。盲目搜索会花费很长时间。但你可以这样思考：

1. **分支：** 把迷宫分成几个区域
2. **限界：** 估算每个区域最远需要多少步能找到宝藏
3. **剪枝：** 如果某个区域的最小步数已经超过已知的最短步数，就不搜索这个区域

这就是分支限界法的核心思想：**通过估算下界来剪枝，避免搜索不可能产生最优解的分支。**

### 1.2 分支限界法 vs 回溯法

| 特性 | 回溯法 | 分支限界法 |
|------|--------|------------|
| **搜索策略** | 深度优先 | 广度优先或最佳优先 |
| **目标** | 找所有解或一个解 | 找最优解 |
| **剪枝依据** | 约束条件 | 界限函数（成本下界/上界） |
| **存储结构** | 栈（递归） | 队列或优先队列 |
| **终止条件** | 找到解或遍历完 | 找到最优解或证明无解 |

```
回溯法搜索树：
        A
      / | \
     B  C  D    ← 深度优先，一条路走到黑
    / \  |
   E   F G      ← 不行就回溯
```

```
分支限界法搜索树：
        A
      / | \
     B  C  D    ← 同时探索所有分支
    |  |  |     ← 但根据限界函数剪枝
    E  X  F     ← X被剪枝（不可能产生最优解）
```

### 1.3 分支限界法的基本框架

```typescript
/**
 * 分支限界法基本框架
 *
 * @param problem 问题实例
 * @returns 最优解
 */

interface Node {
  // 节点状态描述
  state: any;
  // 从根到当前节点的成本
  cost: number;
  // 成本下界（乐观估计）
  lowerBound: number;
  // 父节点
  parent: Node | null;
}

// 优先队列按 lowerBound 从小到大排序（最佳优先）
class BranchAndBound {
  private bestSolution: any = null;
  private bestCost: number = Infinity;
  private pq: Node[] = [];  // 优先队列

  solve(initialState: any): any {
    // 初始化根节点
    const root = this.createNode(initialState, null, 0);
    this.pq.push(root);

    while (this.pq.length > 0) {
      // 取出最有希望的节点
      const node = this.pq.shift()!;

      // 剪枝：如果当前节点的下界 >= 已知的最佳成本，跳过
      if (node.lowerBound >= this.bestCost) {
        continue;
      }

      // 分支：生成所有子节点
      for (const child of this.branch(node)) {
        // 计算子节点的下界
        child.lowerBound = this.calculateLowerBound(child);

        // 如果是解且成本更优，更新最佳解
        if (this.isSolution(child) && child.cost < this.bestCost) {
          this.bestCost = child.cost;
          this.bestSolution = child.state;
        }

        // 限界：只有下界小于最佳成本的节点才加入队列
        if (child.lowerBound < this.bestCost) {
          this.pq.push(child);
        }
      }
    }

    return this.bestSolution;
  }

  private createNode(state: any, parent: Node | null, cost: number): Node {
    return {state, cost, lowerBound: cost, parent};
  }

  // 分支：生成子节点
  private branch(node: Node): Node[] {
    // 子类实现
    return [];
  }

  // 计算下界
  private calculateLowerBound(node: Node): number {
    // 子类实现
    return node.cost;
  }

  // 判断是否是解
  private isSolution(node: Node): boolean {
    // 子类实现
    return false;
  }
}
```

## 二、0-1背包问题的分支限界

### 2.1 问题分析

**0-1背包问题** 是分支限界法的经典应用场景。

```typescript
/**
 * 0-1背包问题的分支限界法解法
 *
 * 上界计算（贪心）：
 * 如果剩余物品可以全带走，计算它们的价值上界
 * 上界 = 当前价值 + 剩余容量 × 单位重量价值最高的物品
 */

interface Item {
  weight: number;
  value: number;
}

class KnapsackBranchAndBound {
  private bestValue = 0;
  private bestItems: boolean[] = [];
  private pq: Array<{
    index: number;       // 当前处理的物品索引
    weight: number;      // 当前总重量
    value: number;       // 当前总价值
    upperBound: number;   // 上界
    items: boolean[];    // 已选择的物品
  }> = [];

  /**
   * 计算上界（乐观估计）
   */
  private calculateUpperBound(
    items: Item[],
    index: number,
    weight: number,
    capacity: number,
    value: number
  ): number {
    // 如果已经超过容量，上界为负无穷
    if (weight > capacity) return -Infinity;

    // 计算剩余容量
    const remainingCapacity = capacity - weight;

    // 贪心：按单位重量价值排序，依次装入直到装满
    let upperBound = value;
    let remainingWeight = remainingCapacity;

    // 创建剩余物品的副本并按单位价值排序
    const remainingItems: Array<{item: Item, unitValue: number}> = [];
    for (let i = index; i < items.length; i++) {
      remainingItems.push({
        item: items[i],
        unitValue: items[i].value / items[i].weight
      });
    }
    remainingItems.sort((a, b) => b.unitValue - a.unitValue);

    for (const {item, unitValue} of remainingItems) {
      if (remainingWeight >= item.weight) {
        upperBound += item.value;
        remainingWeight -= item.weight;
      } else {
        upperBound += remainingWeight * unitValue;
        break;
      }
    }

    return upperBound;
  }

  /**
   * 分支限界求解0-1背包问题
   */
  solve(items: Item[], capacity: number): {value: number, items: boolean[]} {
    // 按单位重量价值降序排序
    const sortedItems = items.map((item, index) => ({item, index}))
      .sort((a, b) =>
        b.item.value / b.item.weight - a.item.value / a.item.weight
      );

    // 重排后的物品
    const orderedItems = sortedItems.map(({item}) => item);

    // 初始化：从第一个物品开始，左分支（选）和右分支（不选）
    const root = {
      index: 0,
      weight: 0,
      value: 0,
      upperBound: this.calculateUpperBound(orderedItems, 0, 0, capacity, 0),
      items: new Array(items.length).fill(false)
    };

    this.pq.push(root);
    this.bestValue = 0;
    this.bestItems = new Array(items.length).fill(false);

    while (this.pq.length > 0) {
      // 按上界降序排序（优先探索更有可能产生最优解的分支）
      this.pq.sort((a, b) => b.upperBound - a.upperBound);

      const node = this.pq.shift()!;

      // 剪枝：如果上界 <= 当前最佳价值，跳过
      if (node.upperBound <= this.bestValue) {
        continue;
      }

      // 分支：尝试选择或不选择当前物品
      const item = orderedItems[node.index];

      // 左分支：选择当前物品
      if (node.weight + item.weight <= capacity) {
        const leftUpperBound = this.calculateUpperBound(
          orderedItems,
          node.index + 1,
          node.weight + item.weight,
          capacity,
          node.value + item.value
        );

        if (leftUpperBound > this.bestValue) {
          const leftItems = [...node.items];
          leftItems[sortedItems[node.index].index] = true;

          this.pq.push({
            index: node.index + 1,
            weight: node.weight + item.weight,
            value: node.value + item.value,
            upperBound: leftUpperBound,
            items: leftItems
          });

          // 更新最佳值
          if (node.value + item.value > this.bestValue) {
            this.bestValue = node.value + item.value;
            this.bestItems = leftItems;
          }
        }
      }

      // 右分支：不选择当前物品
      const rightUpperBound = this.calculateUpperBound(
        orderedItems,
        node.index + 1,
        node.weight,
        capacity,
        node.value
      );

      if (rightUpperBound > this.bestValue) {
        this.pq.push({
          index: node.index + 1,
          weight: node.weight,
          value: node.value,
          upperBound: rightUpperBound,
          items: [...node.items]
        });
      }
    }

    return {value: this.bestValue, items: this.bestItems};
  }
}
```

## 三、旅行商问题的分支限界

### 3.1 问题分析

**旅行商问题（TSP）** 是另一个分支限界法的经典应用。

```typescript
/**
 * 旅行商问题（TSP）的分支限界法解法
 *
 * 状态表示：(path, cost)
 * - path: 当前路径
 * - cost: 当前成本
 *
 * 下界计算：
 * - 每个未访问城市的最小出边之和
 * - 加上当前路径的最小回边（如果所有城市都访问过）
 */

class TSPBranchAndBound {
  private bestCost = Infinity;
  private bestPath: number[] = [];

  /**
   * 计算下界（简化版本）
   */
  private calculateLowerBound(
    currentPath: number[],
    currentCost: number,
    adjMatrix: number[][]
  ): number {
    const n = adjMatrix.length;
    const visited = new Set(currentPath);
    let lowerBound = currentCost;

    // 计算每个未访问城市的最小出边之和
    for (const city of currentPath) {
      if (!visited.has(city)) continue;

      let minEdge = Infinity;
      for (let j = 0; j < n; j++) {
        if (!visited.has(j) && adjMatrix[city][j] < minEdge) {
          minEdge = adjMatrix[city][j];
        }
      }
      if (minEdge !== Infinity) {
        lowerBound += minEdge;
      }
    }

    return lowerBound;
  }

  /**
   * TSP分支限界
   */
  solve(dist: number[][]): {cost: number, path: number[]} | null {
    const n = dist.length;
    if (n === 0) return null;

    this.bestCost = Infinity;
    this.bestPath = [];

    // 优先队列：[lowerBound, currentPath, currentCost]
    const pq: Array<{lb: number, path: number[], cost: number}> = [];

    // 从城市0开始
    pq.push({lb: 0, path: [0], cost: 0});

    while (pq.length > 0) {
      // 按下界排序
      pq.sort((a, b) => a.lb - b.lb);

      const node = pq.shift()!;

      // 剪枝
      if (node.cost >= this.bestCost) continue;

      const {path, cost} = node;

      // 所有城市都访问过了，检查是否需要回到起点
      if (path.length === n) {
        const returnCost = dist[path[n - 1]][path[0]];
        if (returnCost !== Infinity) {
          const totalCost = cost + returnCost;
          if (totalCost < this.bestCost) {
            this.bestCost = totalCost;
            this.bestPath = path;
          }
        }
        continue;
      }

      // 分支：选择下一个要访问的城市
      const currentCity = path[path.length - 1];

      for (let nextCity = 0; nextCity < n; nextCity++) {
        // 跳过已访问的城市和不可达的城市
        if (path.includes(nextCity) || dist[currentCity][nextCity] === Infinity) {
          continue;
        }

        const newCost = cost + dist[currentCity][nextCity];
        const newPath = [...path, nextCity];
        const lowerBound = this.calculateLowerBound(newPath, newCost, dist);

        if (lowerBound < this.bestCost) {
          pq.push({lb: lowerBound, path: newPath, cost: newCost});
        }
      }
    }

    return this.bestCost === Infinity
      ? null
      : {cost: this.bestCost, path: this.bestPath};
  }
}
```

## 四、随机算法基础

### 4.1 随机算法的定义与分类

**随机算法（Randomized Algorithm）** 是使用随机数来影响算法行为的算法。

**类比理解——抛硬币决定下一步：**

想象你在迷宫里遇到岔路口，正常的算法会分析每条路哪个更好。但随机算法呢？

**Monte Carlo（蒙特卡洛）：**
- "随便选一条路试试"
- 结果不一定对，但大概率对
- 特点：**结果正确但时间不确定**

**Las Vegas（拉斯维加斯）：**
- "一直试，直到找到正确答案"
- 结果一定对，但不知道要试多久
- 特点：**时间不确定但结果正确**

### 4.2 两类随机算法对比

| 特性 | Monte Carlo | Las Vegas |
|------|-------------|-----------|
| **结果正确性** | 可能错误（但概率可控） | 一定正确 |
| **运行时间** | 确定（有界） | 不确定 |
| **应用** | 近似计算、采样 | 搜索、排序 |
| **容错** | 可以控制错误概率 | 自然保证 |

```typescript
/**
 * Monte Carlo vs Las Vegas 示例
 */

// Monte Carlo：随机快速排序
// 结果可能不是完全有序的，但概率很低
function randomQuickSortMC(arr: number[]): number[] {
  if (arr.length <= 1) return arr;

  // 随机选择基准
  const pivotIndex = Math.floor(Math.random() * arr.length);
  const pivot = arr[pivotIndex];

  const left = arr.filter((x, i) => i !== pivotIndex && x < pivot);
  const middle = arr.filter((x, i) => i !== pivotIndex && x === pivot);
  const right = arr.filter((x, i) => i !== pivotIndex && x > pivot);

  // 注意：这里有概率导致排序不完全正确（如果随机选择导致划分极度不均）
  return [...randomQuickSortMC(left), ...middle, ...randomQuickSortMC(right)];
}

// Las Vegas：随机化快速排序
// 结果一定正确，但时间不确定
function randomQuickSortLV(arr: number[]): number[] {
  if (arr.length <= 1) return arr;

  const pivotIndex = Math.floor(Math.random() * arr.length);
  const pivot = arr[pivotIndex];

  const left: number[] = [];
  const middle: number[] = [];
  const right: number[] = [];

  for (const x of arr) {
    if (x < pivot) left.push(x);
    else if (x === pivot) middle.push(x);
    else right.push(x);
  }

  // LV版本：每次都正确划分，只是基准是随机的
  // 结果一定正确，但划分质量不确定（影响时间复杂度）
  return [...randomQuickSortLV(left), ...middle, ...randomQuickSortLV(right)];
}
```

## 五、蒙特卡洛算法

### 5.1 蒙特卡洛方法原理

**蒙特卡洛方法（Monte Carlo Method）** 通过随机采样来近似计算难以精确计算的问题。

**经典应用：**
- 计算圆周率 π
- 数值积分
- 粒子传输模拟
- 机器学习中的随机优化

### 5.2 计算圆周率

```typescript
/**
 * 蒙特卡洛方法计算圆周率
 *
 * 原理：
 * - 在边长为2的正方形内随机投点
 * - 圆的半径为1，圆心在正方形中心
 * - 点落在圆内的概率 = 圆的面积 / 正方形面积 = π/4
 * - π ≈ 4 × (圆内点数 / 总点数)
 */

function estimatePi(numSamples: number): number {
  let insideCircle = 0;

  for (let i = 0; i < numSamples; i++) {
    // 随机生成[-1, 1]范围内的x, y坐标
    const x = Math.random() * 2 - 1;
    const y = Math.random() * 2 - 1;

    // 检查点是否在圆内（x² + y² <= 1）
    if (x * x + y * y <= 1) {
      insideCircle++;
    }
  }

  // π ≈ 4 × (圆内点数 / 总点数)
  return 4 * insideCircle / numSamples;
}

/**
 * 蒙特卡洛方法计算定积分
 *
 * 计算 ∫₀¹ f(x) dx
 *
 * 原理：
 * - 在 [0,1] × [0,1] 的矩形内随机投点
 * - 落在曲线y=f(x)下方的概率 ≈ 积分值
 */

function monteCarloIntegrate(
  f: (x: number) => number,
  numSamples: number,
  maxY: number = 1
): number {
  let count = 0;

  for (let i = 0; i < numSamples; i++) {
    const x = Math.random();
    const y = Math.random() * maxY;

    if (y <= f(x)) {
      count++;
    }
  }

  // 矩形面积为 maxY × 1 = maxY
  // 积分值 ≈ (count / numSamples) × maxY
  return (count / numSamples) * maxY;
}

/**
 * 示例：计算 ∫₀¹ x² dx = 1/3
 */
function exampleIntegral(): void {
  const f = (x: number) => x * x;
  const result = monteCarloIntegrate(f, 100000, 1);
  console.log(`∫₀¹ x² dx ≈ ${result.toFixed(6)}`);  // 约等于 0.333333
  console.log(`实际值 = ${1/3}`);  // 0.333333...
}
```

### 5.3 蒙特卡洛搜索树（MCTS）

**蒙特卡洛树搜索（MCTS）** 是用于博弈问题的随机搜索算法。

**算法步骤：**
1. **选择（Selection）：** 从根开始，选择最优子节点直到叶子节点
2. **扩展（Expansion）：** 添加一个或多个子节点
3. **模拟（Simulation）：** 随机选择走法直到游戏结束
4. **反向传播（Backpropagation）：** 更新路径上所有节点的统计数据

```typescript
/**
 * 蒙特卡洛树搜索（MCTS）
 *
 * 用于：围棋、AI对战、博弈问题
 */

/**
 * MCTS节点
 */
interface MCTSNode {
  // 节点状态
  state: any;
  // 父节点
  parent: MCTSNode | null;
  // 子节点
  children: MCTSNode[];
  // 访问次数
  visitCount: number;
  // 胜利次数
  winCount: number;
  // 可用的动作
  untriedActions: any[];
}

class MCTS {
  private explorationConstant = 1.414;  // √2

  /**
   * UCB1公式：选择子节点
   * UCB1 = average_reward + exploration_constant × √(ln(parent_visit) / child_visit)
   */
  private ucb1(node: MCTSNode): number {
    if (node.visitCount === 0) return Infinity;

    const averageReward = node.winCount / node.visitCount;
    const exploration = this.explorationConstant *
      Math.sqrt(Math.log(node.parent!.visitCount) / node.visitCount);

    return averageReward + exploration;
  }

  /**
   * 选择阶段：从根节点选择最佳子节点直到叶子节点
   */
  private select(node: MCTSNode): MCTSNode {
    while (node.children.length > 0) {
      // 选择UCB1值最大的子节点
      node = node.children.reduce((best, child) =>
        this.ucb1(child) > this.ucb1(best) ? child : best
      );
    }
    return node;
  }

  /**
   * 扩展阶段：添加一个或多个子节点
   */
  private expand(node: MCTSNode): MCTSNode {
    if (node.untriedActions.length === 0) {
      return node;
    }

    // 随机选择一个未尝试的动作
    const action = node.untriedActions.splice(
      Math.floor(Math.random() * node.untriedActions.length),
      1
    )[0];

    const newState = this.applyAction(node.state, action);
    const childNode: MCTSNode = {
      state: newState,
      parent: node,
      children: [],
      visitCount: 0,
      winCount: 0,
      untriedActions: this.getLegalActions(newState)
    };

    node.children.push(childNode);
    return childNode;
  }

  /**
   * 模拟阶段：随机选择走法直到游戏结束
   */
  private simulate(state: any): number {
    // 返回 1 表示胜利，0 表示失败
    let currentState = state;

    while (!this.isTerminal(currentState)) {
      const actions = this.getLegalActions(currentState);
      if (actions.length === 0) return 0;

      const randomAction = actions[Math.floor(Math.random() * actions.length)];
      currentState = this.applyAction(currentState, randomAction);
    }

    return this.getReward(currentState);
  }

  /**
   * 反向传播阶段：更新路径上所有节点的统计
   */
  private backpropagate(node: MCTSNode, result: number): void {
    while (node !== null) {
      node.visitCount++;
      node.winCount += result;
      node = node.parent!;
    }
  }

  /**
   * 单次MCTS迭代
   */
  private iterate(root: MCTSNode): void {
    // 1. 选择
    const node = this.select(root);

    // 2. 扩展或模拟
    let expandNode = node;
    if (node.untriedActions.length > 0) {
      expandNode = this.expand(node);
    }

    // 3. 模拟
    const result = this.simulate(expandNode.state);

    // 4. 反向传播
    this.backpropagate(expandNode, result);
  }

  /**
   * 运行MCTS搜索
   */
  search(initialState: any, iterations: number): any {
    const root: MCTSNode = {
      state: initialState,
      parent: null,
      children: [],
      visitCount: 0,
      winCount: 0,
      untriedActions: this.getLegalActions(initialState)
    };

    for (let i = 0; i < iterations; i++) {
      this.iterate(root);
    }

    // 返回访问次数最多的子节点对应的动作
    return root.children.reduce((best, child) =>
      child.visitCount > best.visitCount ? child : best
    ).state;
  }

  // 以下方法需要子类实现
  private applyAction(state: any, action: any): any {
    throw new Error('子类实现');
  }

  private getLegalActions(state: any): any[] {
```

## 六、拉斯维加斯算法

### 6.1 拉斯维加斯算法原理

**拉斯维加斯算法（Las Vegas Algorithm）** 保证结果正确，但运行时间不确定。

**典型应用：**
- 随机化快速排序
- 随机化跳跃表
- 随机化选择算法
- 随机化字符串匹配

### 6.2 快速选择算法（随机化）

**快速选择算法** 用于在未排序的数组中找到第k小的元素。

```typescript
/**
 * 快速选择算法（Las Vegas版本）
 *
 * 平均时间复杂度：O(n)
 * 最坏时间复杂度：O(n²)（但概率极低）
 * 结果：一定正确
 */

function quickSelect(arr: number[], k: number): number {
  // k是0-indexed，即第k小的元素
  return quickSelectHelper(arr, 0, arr.length - 1, k);
}

function quickSelectHelper(
  arr: number[],
  left: number,
  right: number,
  k: number
): number {
  // 如果子数组只有一个元素，直接返回
  if (left === right) {
    return arr[left];
  }

  // 随机选择基准元素（拉斯维加斯特点）
  const pivotIndex = randomPartition(arr, left, right);

  // 计算pivot的位置
  const pivotRank = pivotIndex - left;

  if (k === pivotRank) {
    return arr[pivotIndex];
  } else if (k < pivotRank) {
    return quickSelectHelper(arr, left, pivotIndex - 1, k);
  } else {
    return quickSelectHelper(arr, pivotIndex + 1, right, k - pivotRank - 1);
  }
}

/**
 * 随机分区（随机化快速排序的核心）
 */
function randomPartition(arr: number[], left: number, right: number): number {
  // 随机选择一个元素作为基准
  const randomIndex = left + Math.floor(Math.random() * (right - left + 1));

  // 将随机选择的元素交换到right位置
  [arr[randomIndex], arr[right]] = [arr[right], arr[randomIndex]];

  const pivot = arr[right];
  let i = left - 1;

  for (let j = left; j < right; j++) {
    if (arr[j] <= pivot) {
      i++;
      [arr[i], arr[j]] = [arr[j], arr[i]];
    }
  }

  [arr[i + 1], arr[right]] = [arr[right], arr[i + 1]];

  return i + 1;
}

/**
 * 快速选择（非递归版本）
 */
function quickSelectIterative(arr: number[], k: number): number {
  let left = 0;
  let right = arr.length - 1;

  while (left < right) {
    const pivotIndex = randomPartition(arr, left, right);
    const pivotRank = pivotIndex - left;

    if (k === pivotRank) {
      return arr[pivotIndex];
    } else if (k < pivotRank) {
      right = pivotIndex - 1;
    } else {
      left = pivotIndex + 1;
      k = k - pivotRank - 1;
    }
  }

  return arr[left];
}

/**
 * 找到第k小的元素并排序左边
 */
function findKthSmallest(arr: number[], k: number): {kth: number, leftSorted: number[]} {
  const result = quickSelect(arr.slice(), k);

  // 对左边k个元素排序
  const left = arr.slice(0, k + 1).sort((a, b) => a - b);

  return {kth: result, leftSorted: left};
}
```

### 6.3 跳跃表（Skip List）

**跳跃表** 是一种随机化的数据结构，可以实现O(log n)的查找、插入、删除。

```typescript
/**
 * 跳跃表（Skip List）
 *
 * 拉斯维加斯特点：
 * - 插入时随机决定节点的层数
 * - 结果一定正确（正确的有序性）
 * - 性能取决于随机性（期望O(log n)）
 */

interface SkipListNode {
  key: number;
  value: any;
  // forward[i] 表示第i层的下一个节点
  forward: SkipListNode[];
}

class SkipList {
  private head: SkipListNode;
  private maxLevel: number;
  private probability = 0.5;  // 升层概率

  constructor(maxLevel: number = 16) {
    this.maxLevel = maxLevel;
    this.head = {
      key: -Infinity,
      value: null,
      forward: new Array(maxLevel).fill(null)
    };
  }

  /**
   * 随机生成层数
   * 使用几何分布，层数越高概率越低
   */
  private randomLevel(): number {
    let level = 0;
    while (Math.random() < this.probability && level < this.maxLevel - 1) {
      level++;
    }
    return level;
  }

  /**
   * 搜索
   */
  search(key: number): any | null {
    let current = this.head;

    // 从最高层开始搜索
    for (let i = this.maxLevel - 1; i >= 0; i--) {
      while (current.forward[i] !== null && current.forward[i].key < key) {
        current = current.forward[i];
      }
    }

    current = current.forward[0];

    if (current !== null && current.key === key) {
      return current.value;
    }

    return null;
  }

  /**
   * 插入
   */
  insert(key: number, value: any): void {
    const level = this.randomLevel();
    const newNode: SkipListNode = {
      key,
      value,
      forward: new Array(level + 1).fill(null)
    };

    const update: SkipListNode[] = new Array(this.maxLevel).fill(this.head);

    let current = this.head;
    for (let i = this.maxLevel - 1; i >= 0; i--) {
      while (current.forward[i] !== null && current.forward[i].key < key) {
        current = current.forward[i];
      }
      update[i] = current;
    }

    for (let i = 0; i <= level; i++) {
      newNode.forward[i] = update[i].forward[i];
      update[i].forward[i] = newNode;
    }
  }

  /**
   * 删除
   */
  delete(key: number): boolean {
    const update: SkipListNode[] = new Array(this.maxLevel).fill(null);

    let current = this.head;
    for (let i = this.maxLevel - 1; i >= 0; i--) {
      while (current.forward[i] !== null && current.forward[i].key < key) {
        current = current.forward[i];
      }
      update[i] = current;
    }

    current = current.forward[0];

    if (current !== null && current.key === key) {
      for (let i = 0; i < this.maxLevel; i++) {
        if (update[i].forward[i] !== current) break;
        update[i].forward[i] = current.forward[i];
      }
      return true;
    }

    return false;
  }
}
```

### 7.1 随机洗牌算法

```typescript
/**
 * Fisher-Yates 洗牌算法
 *
 * 正确性：产生均匀随机排列
 * 时间复杂度：O(n)
 */

function fisherYatesShuffle<T>(arr: T[]): T[] {
  const result = [...arr];

  for (let i = result.length - 1; i > 0; i--) {
    // 随机选择[0, i]范围内的元素
    const j = Math.floor(Math.random() * (i + 1));

    // 交换
    [result[i], result[j]] = [result[j], result[i]];
  }

  return result;
}

/**
 * 错误洗牌示例（不要用）
 * 这种方法不是均匀分布的
 */
function badShuffle<T>(arr: T[]): T[] {
  const result = [...arr];

  for (let i = 0; i < result.length; i++) {
    const j = Math.floor(Math.random() * result.length);
    [result[i], result[j]] = [result[j], result[i]];
  }

  return result;
}
```

### 7.2 随机化字符串匹配

```typescript
/**
 * Rabin-Karp 字符串匹配算法
 *
 * 使用滚动哈希来加速匹配
 * 蒙特卡洛版本：快速但有小概率错误
 * 拉斯维加斯版本：验证哈希冲突，保证正确
 */

function rabinKarp(text: string, pattern: string): number[] {
  const results: number[] = [];
  const n = text.length;
  const m = pattern.length;

  if (m === 0) return results;

  // 基数（一个质数）
  const BASE = 31;
  // 模数（一个大质数，防止整数溢出）
  const MOD = 10 ** 9 + 7;

  // 计算pattern的哈希值
  let patternHash = 0;
  for (let i = 0; i < m; i++) {
    patternHash = (patternHash * BASE + pattern.charCodeAt(i)) % MOD;
  }

  // 计算text第一个窗口的哈希值
  let textHash = 0;
  for (let i = 0; i < m; i++) {
    textHash = (textHash * BASE + text.charCodeAt(i)) % MOD;
  }

  // 计算BASE^(m-1) % MOD
  let highestBase = 1;
  for (let i = 0; i < m - 1; i++) {
    highestBase = (highestBase * BASE) % MOD;
  }

  // 滚动哈希
  for (let i = 0; i <= n - m; i++) {
    // 如果哈希值匹配，进一步验证
    if (textHash === patternHash) {
      // 验证子字符串（拉斯维加斯验证步骤）
      let match = true;
      for (let j = 0; j < m; j++) {
        if (text[i + j] !== pattern[j]) {
          match = false;
          break;
        }
      }
      if (match) {
        results.push(i);
      }
    }

    // 计算下一个窗口的哈希值
    if (i < n - m) {
      textHash = ((textHash - text.charCodeAt(i) * highestBase) * BASE +
                  text.charCodeAt(i + m)) % MOD;

      // 处理负数情况
      if (textHash < 0) {
        textHash += MOD;
      }
    }
  }

  return results;
}

/**
 * Monte Carlo版本（更快但可能错误）
 */
function rabinKarpMonteCarlo(
  text: string,
  pattern: string,
  numHashes: number = 3
): number | null {
  const n = text.length;
  const m = pattern.length;

  if (m === 0 || m > n) return null;

  // 使用多个哈希函数来降低冲突概率
  const hashes = [
    {base: 31, mod: 10 ** 9 + 7},
    {base: 37, mod: 10 ** 9 + 9},
    {base: 41, mod: 10 ** 9 + 21}
  ];

  // 计算pattern的多个哈希值
  const patternHashes = hashes.map(({base, mod}) => {
    let hash = 0;
    for (let i = 0; i < m; i++) {
      hash = (hash * base + pattern.charCodeAt(i)) % mod;
    }
    return hash;
  });

  // 计算最高位权重
  const highestPowers = hashes.map(({base, mod}) => {
    let power = 1;
    for (let i = 0; i < m - 1; i++) {
      power = (power * base) % mod;
    }
    return power;
  });

  // 计算text第一个窗口的哈希值
  const textHashes = hashes.map(({base, mod}, idx) => {
    let hash = 0;
    for (let i = 0; i < m; i++) {
      hash = (hash * base + text.charCodeAt(i)) % mod;
    }
    return hash;
  });

  // 滚动检查
  for (let i = 0; i <= n - m; i++) {
    let match = true;

    for (let h = 0; h < numHashes; h++) {
      if (textHashes[h] !== patternHashes[h]) {
        match = false;
        break;
      }
    }

    if (match) {
      return i;  // Monte Carlo：返回第一次匹配的位置（有小概率错误）
    }

    // 更新哈希值
    if (i < n - m) {
      for (let h = 0; h < numHashes; h++) {
        const {base, mod} = hashes[h];
        textHashes[h] = ((textHashes[h] - text.charCodeAt(i) * highestPowers[h]) *
                          base + text.charCodeAt(i + m)) % mod;
        if (textHashes[h] < 0) textHashes[h] += mod;
      }
    }
  }

  return null;
}
```

## 九、总结

### 9.1 分支限界法核心要点

```
分支限界法要点

核心思想：通过界限函数剪枝，避免搜索不可能产生最优解的分支

与回溯法的区别：
- 回溯法：深度优先搜索所有可能
- 分支限界：广度/最佳优先，只搜索有希望的分支

关键步骤：
1. 分支：生成问题的子问题
2. 限界：计算子问题的成本下界
3. 剪枝：跳过下界不优于当前最优的分支
4. 更新：发现更优解时更新

经典应用：
- 0-1背包问题
- 旅行商问题（TSP）
- 任务分配问题
```

### 9.2 随机算法核心要点

```
随机算法要点

Monte Carlo：
- 结果正确但时间确定
- 适合近似计算、采样
- 可以控制错误概率

Las Vegas：
- 时间不确定但结果一定正确
- 适合搜索、排序
- 性能取决于随机性

经典应用：
- 蒙特卡洛积分
- MCTS（蒙特卡洛树搜索）
- 快速排序（随机基准）
- 快速选择
- 跳跃表
```

### 9.3 算法选择指南

```
分支限界 vs 随机算法选择

组合优化问题：
- 可以计算下界 → 分支限界
- 需要近似解 → Monte Carlo

需要确定结果：
- 允许随机基准 → Las Vegas
- 需要完美结果 → 确定算法

搜索问题：
- 博弈/游戏 → MCTS
- 大状态空间 → 随机化搜索
```
