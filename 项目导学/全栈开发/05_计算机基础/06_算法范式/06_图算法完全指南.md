# 图算法完全指南

## 一、图的基础概念

### 1.1 图的定义与分类

**图（Graph）** 是由顶点（Vertex）和边（Edge）组成的数据结构。

**类比理解：** 想象一个城市交通网络：
- 顶点 = 城市中的重要地点（火车站、商场、公园）
- 边 = 连接这些地点的道路
- 边的权重 = 道路的长度或行驶时间

### 1.2 图的分类

```
图的分类

按方向分：
┌─────────────┐     ┌─────────────┐
│   无向图    │     │   有向图    │
│  边无方向   │     │   边有方向  │
│  A ─── B    │     │   A ───→ B  │
└─────────────┘     └─────────────┘

按权重分：
┌─────────────┐     ┌─────────────┐
│   加权图    │     │   无权图    │
│  边有权重   │     │   边无权重  │
│  A ─3── B   │     │   A ─── B   │
└─────────────┘     └─────────────┘

按连通性分：
┌─────────────┐     ┌─────────────┐
│ 连通图      │     │ 非连通图    │
│ 所有顶点互通│     │ 部分顶点孤立│
└─────────────┘     └─────────────┘
```

### 1.3 图的表示方法

```typescript
/**
 * 图的表示方法
 */

// 方法1：邻接矩阵
// 适合稠密图，空间O(V²)
class AdjacencyMatrix {
  private matrix: number[][];
  private n: number;

  constructor(n: number) {
    this.n = n;
    this.matrix = Array.from({length: n}, () => new Array(n).fill(0));
  }

  addEdge(i: number, j: number, weight: number = 1): void {
    this.matrix[i][j] = weight;
    // 如果是无向图，还需要 this.matrix[j][i] = weight;
  }

  getNeighbors(v: number): number[] {
    const neighbors: number[] = [];
    for (let i = 0; i < this.n; i++) {
      if (this.matrix[v][i] !== 0) {
        neighbors.push(i);
      }
    }
    return neighbors;
  }
}

// 方法2：邻接表
// 适合稀疏图，空间O(V+E)
class AdjacencyList {
  private adj: Map<number, Array<{to: number, weight: number}>>;
  private n: number;

  constructor(n: number) {
    this.n = n;
    this.adj = new Map();
    for (let i = 0; i < n; i++) {
      this.adj.set(i, []);
    }
  }

  addEdge(i: number, j: number, weight: number = 1): void {
    this.adj.get(i)!.push({to: j, weight});
    // 如果是无向图，还需要 this.adj.get(j)!.push({to: i, weight});
  }

  getNeighbors(v: number): Array<{to: number, weight: number}> {
    return this.adj.get(v) || [];
  }
}

// 方法3：边列表
// 适合边的排序和批量操作
interface Edge {
  from: number;
  to: number;
  weight?: number;
}

class EdgeList {
  private edges: Edge[] = [];

  addEdge(from: number, to: number, weight: number = 1): void {
    this.edges.push({from, to, weight});
  }

  getEdges(): Edge[] {
    return this.edges;
  }
}
```

## 二、图的遍历

### 2.1 BFS广度优先搜索

**BFS（Breadth-First Search）** 是按层次遍历图的算法。

**核心思想：** 先访问所有邻居，再访问邻居的邻居。

**类比理解：** 就像往湖里扔一块石头，涟漪一圈一圈向外扩散。BFS就是这种"扩散式"搜索。

```typescript
/**
 * BFS广度优先搜索
 *
 * 时间复杂度：O(V + E)
 * 空间复杂度：O(V)
 *
 * BFS的特点：
 * - 找到最短路径（无权图）
 * - 按层次访问
 * - 使用队列实现
 */

/**
 * BFS遍历图（返回访问顺序）
 */
function bfs(graph: AdjacencyList, start: number): number[] {
  const visited = new Set<number>();
  const queue: number[] = [start];
  const order: number[] = [];

  visited.add(start);

  while (queue.length > 0) {
    const v = queue.shift()!;
    order.push(v);

    for (const {to} of graph.getNeighbors(v)) {
      if (!visited.has(to)) {
        visited.add(to);
        queue.push(to);
      }
    }
  }

  return order;
}

/**
 * BFS找最短路径（无权图）
 */
function bfsShortestPath(
  graph: AdjacencyList,
  start: number,
  end: number
): number[] | null {
  if (start === end) return [start];

  const visited = new Set<number>();
  const queue: number[] = [start];
  // 记录每个节点的父节点
  const parent = new Map<number, number>();

  visited.add(start);

  while (queue.length > 0) {
    const v = queue.shift()!;

    if (v === end) {
      // 重建路径
      const path: number[] = [];
      let curr = end;
      while (curr !== start) {
        path.push(curr);
        curr = parent.get(curr)!;
      }
      path.push(start);
      return path.reverse();
    }

    for (const {to} of graph.getNeighbors(v)) {
      if (!visited.has(to)) {
        visited.add(to);
        parent.set(to, v);
        queue.push(to);
      }
    }
  }

  return null;  // 无法到达
}

/**
 * BFS检测图的连通分量
 */
function countConnectedComponents(graph: AdjacencyList): number {
  const visited = new Set<number>();
  let count = 0;

  for (let v = 0; v < graph.getNeighbors(v).length; v++) {
    if (!visited.has(v)) {
      bfsComponent(graph, v, visited);
      count++;
    }
  }

  return count;
}

function bfsComponent(
  graph: AdjacencyList,
  start: number,
  visited: Set<number>
): void {
  const queue: number[] = [start];
  visited.add(start);

  while (queue.length > 0) {
    const v = queue.shift()!;
    for (const {to} of graph.getNeighbors(v)) {
      if (!visited.has(to)) {
        visited.add(to);
        queue.push(to);
      }
    }
  }
}

/**
 * BFS判断二分图
 * 二分图：图的顶点可以分为两个不相交的集合，使得每条边连接的两个顶点属于不同集合
 */
function isBipartite(graph: AdjacencyList): boolean {
  const n = graph.getNeighbors(0).length;
  const color = new Array(n).fill(-1);  // -1表示未染色，0和1表示两种颜色

  for (let i = 0; i < n; i++) {
    if (color[i] === -1) {
      // BFS染色
      const queue: number[] = [i];
      color[i] = 0;

      while (queue.length > 0) {
        const v = queue.shift()!;
        for (const {to} of graph.getNeighbors(v)) {
          if (color[to] === -1) {
            color[to] = 1 - color[v];
            queue.push(to);
          } else if (color[to] === color[v]) {
            return false;  // 相邻节点颜色相同，不是二分图
          }
        }
      }
    }
  }

  return true;
}
```

### 2.2 DFS深度优先搜索

**DFS（Depth-First Search）** 是沿着一条路径走到底，然后回溯的算法。

**核心思想：** 先深入探索，再回溯。

**类比理解：** 就像走迷宫时，你选择一个方向一直往前走，直到碰壁才往回走，换一个方向继续探索。

```typescript
/**
 * DFS深度优先搜索
 *
 * 时间复杂度：O(V + E)
 * 空间复杂度：O(V)（递归栈）
 *
 * DFS的特点：
 * - 递归实现简单
 * - 可以用于拓扑排序、连通分量等
 * - 适合递归处理的问题
 */

/**
 * DFS递归版本
 */
function dfsRecursive(graph: AdjacencyList, start: number, visited: Set<number>): void {
  visited.add(start);

  for (const {to} of graph.getNeighbors(start)) {
    if (!visited.has(to)) {
      dfsRecursive(graph, to, visited);
    }
  }
}

/**
 * DFS迭代版本（使用栈）
 */
function dfsIterative(graph: AdjacencyList, start: number): number[] {
  const visited = new Set<number>();
  const stack: number[] = [start];
  const order: number[] = [];

  while (stack.length > 0) {
    const v = stack.pop()!;

    if (!visited.has(v)) {
      visited.add(v);
      order.push(v);

      // 注意：这里是逆序入栈，保证访问顺序
      const neighbors = graph.getNeighbors(v);
      for (let i = neighbors.length - 1; i >= 0; i--) {
        if (!visited.has(neighbors[i].to)) {
          stack.push(neighbors[i].to);
        }
      }
    }
  }

  return order;
}

/**
 * 岛屿数量（LeetCode 200）
 * 使用DFS flood fill
 */
function numIslands(grid: string[][]): number {
  if (grid.length === 0) return 0;

  const m = grid.length;
  const n = grid[0].length;
  let count = 0;

  function dfs(i: number, j: number): void {
    // 越界或已是水
    if (i < 0 || i >= m || j < 0 || j >= n || grid[i][j] === '0') {
      return;
    }

    // 标记为已访问（水）
    grid[i][j] = '0';

    // 四个方向深度优先
    dfs(i + 1, j);
    dfs(i - 1, j);
    dfs(i, j + 1);
    dfs(i, j - 1);
  }

  for (let i = 0; i < m; i++) {
    for (let j = 0; j < n; j++) {
      if (grid[i][j] === '1') {
        count++;
        dfs(i, j);
      }
    }
  }

  return count;
}

/**
 * 太平洋大西洋水流问题（LeetCode 417）
 * 找出所有能同时流向太平洋和大西洋的单元格
 */
function pacificAtlantic(heights: number[][]): number[][] {
  const m = heights.length;
  const n = heights[0].length;

  const pacific = new Set<string>();
  const atlantic = new Set<string>();

  // 从四条边开始DFS
  // 上边和左边 -> 太平洋
  // 下边和右边 -> 大西洋

  function dfs(i: number, j: number, visited: Set<string>, prevHeight: number): void {
    const key = `${i},${j}`;
    if (visited.has(key)) return;
    if (i < 0 || i >= m || j < 0 || j >= n) return;
    if (heights[i][j] < prevHeight) return;

    visited.add(key);
    dfs(i + 1, j, visited, heights[i][j]);
    dfs(i - 1, j, visited, heights[i][j]);
    dfs(i, j + 1, visited, heights[i][j]);
    dfs(i, j - 1, visited, heights[i][j]);
  }

  // 太平洋
  for (let j = 0; j < n; j++) dfs(0, j, pacific, -1);
  for (let i = 0; i < m; i++) dfs(i, 0, pacific, -1);

  // 大西洋
  for (let j = 0; j < n; j++) dfs(m - 1, j, atlantic, -1);
  for (let i = 0; i < m; i++) dfs(i, n - 1, atlantic, -1);

  // 找出交集
  const result: number[][] = [];
  for (let i = 0; i < m; i++) {
    for (let j = 0; j < n; j++) {
      if (pacific.has(`${i},${j}`) && atlantic.has(`${i},${j}`)) {
        result.push([i, j]);
      }
    }
  }

  return result;
}
```

### 2.3 BFS与DFS对比

| 特性 | BFS | DFS |
|------|-----|-----|
| **数据结构** | 队列 | 栈（递归） |
| **搜索顺序** | 层次优先 | 深度优先 |
| **最短路径** | 可以（无权图） | 不可以 |
| **内存** | 较多（队列） | 较少（栈） |
| **适合问题** | 最短路径、层级遍历 | 连通分量、拓扑排序 |

## 三、最短路径算法

### 3.1 Dijkstra算法

**Dijkstra算法** 用于计算单源最短路径（所有边权重非负）。

**核心思想：** 贪心 + 优先队列，每次选择当前距离源点最近的未访问顶点。

```typescript
/**
 * Dijkstra算法
 *
 * 时间复杂度：O((V + E) log V)  使用优先队列
 * 空间复杂度：O(V)
 *
 * 适用条件：所有边权重非负
 *
 * 算法步骤：
 * 1. 初始化距离数组，源点距离为0，其他为无穷大
 * 2. 使用优先队列（最小堆）选择当前最短距离的顶点
 * 3. 对该顶点的所有边进行松弛操作
 * 4. 重复直到所有顶点都被处理
 */

/**
 * Dijkstra算法（邻接表 + 优先队列）
 */
function dijkstra(
  graph: AdjacencyList,
  source: number,
  destination: number = -1
): Map<number, number> {
  const n = graph.getNeighbors(0).length;  // 顶点数
  const dist = new Map<number, number>();   // 距离
  const visited = new Set<number>();         // 已访问

  // 初始化距离为无穷大
  for (let i = 0; i < n; i++) {
    dist.set(i, Infinity);
  }
  dist.set(source, 0);

  // 优先队列：[distance, vertex]
  const pq: Array<[number, number]> = [[0, source]];

  while (pq.length > 0) {
    // 取出最短距离的顶点
    pq.sort((a, b) => a[0] - b[0]);
    const [d, u] = pq.shift()!;

    // 如果已访问，跳过
    if (visited.has(u)) continue;
    visited.add(u);

    // 如果到达目标，可以提前返回（如果只求单源到单目标）
    if (u === destination) break;

    // 松弛操作
    for (const {to, weight} of graph.getNeighbors(u)) {
      if (!visited.has(to)) {
        const newDist = dist.get(u)! + weight;
        if (newDist < dist.get(to)!) {
          dist.set(to, newDist);
          pq.push([newDist, to]);
        }
      }
    }
  }

  return dist;
}

/**
 * Dijkstra算法（返回最短路径）
 */
function dijkstraWithPath(
  graph: AdjacencyList,
  source: number,
  target: number
): {distance: number, path: number[]} | null {
  const n = graph.getNeighbors(0).length;
  const dist = new Array(n).fill(Infinity);
  const parent = new Array(n).fill(-1);
  const visited = new Array(n).fill(false);

  dist[source] = 0;
  const pq: Array<[number, number]> = [[0, source]];

  while (pq.length > 0) {
    pq.sort((a, b) => a[0] - b[0]);
    const [d, u] = pq.shift()!;

    if (visited[u]) continue;
    visited[u] = true;

    if (u === target) break;

    for (const {to, weight} of graph.getNeighbors(u)) {
      if (!visited[to]) {
        const newDist = dist[u] + weight;
        if (newDist < dist[to]) {
          dist[to] = newDist;
          parent[to] = u;
          pq.push([newDist, to]);
        }
      }
    }
  }

  if (dist[target] === Infinity) return null;

  // 重建路径
  const path: number[] = [];
  let curr = target;
  while (curr !== -1) {
    path.push(curr);
    curr = parent[curr];
  }

  return {distance: dist[target], path: path.reverse()};
}

/**
 * 使用真实优先队列（最小堆）的Dijkstra
 */
class MinHeap<T> {
  private heap: T[] = [];
  private compare: (a: T, b: T) => number;

  constructor(compare: (a: T, b: T) => number) {
    this.compare = compare;
  }

  push(item: T): void {
    this.heap.push(item);
    this.bubbleUp(this.heap.length - 1);
  }

  pop(): T | undefined {
    if (this.heap.length === 0) return undefined;
    const top = this.heap[0];
    const last = this.heap.pop()!;
    if (this.heap.length > 0) {
      this.heap[0] = last;
      this.bubbleDown(0);
    }
    return top;
  }

  private bubbleUp(i: number): void {
    while (i > 0) {
      const parent = Math.floor((i - 1) / 2);
      if (this.compare(this.heap[i], this.heap[parent]) >= 0) break;
      [this.heap[i], this.heap[parent]] = [this.heap[parent], this.heap[i]];
      i = parent;
    }
  }

  private bubbleDown(i: number): void {
    while (true) {
      const left = 2 * i + 1;
      const right = 2 * i + 2;
      let smallest = i;

      if (left < this.heap.length && this.compare(this.heap[left], this.heap[smallest]) < 0) {
        smallest = left;
      }
      if (right < this.heap.length && this.compare(this.heap[right], this.heap[smallest]) < 0) {
        smallest = right;
      }

      if (smallest === i) break;
      [this.heap[i], this.heap[smallest]] = [this.heap[smallest], this.heap[i]];
      i = smallest;
    }
  }

  isEmpty(): boolean {
    return this.heap.length === 0;
  }
}

function dijkstraWithMinHeap(
  graph: AdjacencyList,
  source: number
): Map<number, number> {
  const n = graph.getNeighbors(0).length;
  const dist = new Map<number, number>();
  const visited = new Set<number>();

  for (let i = 0; i < n; i++) dist.set(i, Infinity);
  dist.set(source, 0);

  const pq = new MinHeap<[number, number]>((a, b) => a[0] - b[0]);
  pq.push([0, source]);

  while (!pq.isEmpty()) {
    const [d, u] = pq.pop()!;

    if (visited.has(u)) continue;
    visited.add(u);

    for (const {to, weight} of graph.getNeighbors(u)) {
      if (!visited.has(to)) {
        const newDist = d + weight;
        if (newDist < dist.get(to)!) {
          dist.set(to, newDist);
          pq.push([newDist, to]);
        }
      }
    }
  }

  return dist;
}
```

### 3.2 Bellman-Ford算法

**Bellman-Ford算法** 可以处理带有负权边的图，也能检测负权环。

```typescript
/**
 * Bellman-Ford算法
 *
 * 时间复杂度：O(V × E)
 * 空间复杂度：O(V)
 *
 * 算法步骤：
 * 1. 初始化距离数组
 * 2. 对所有边进行V-1次松弛操作
 * 3. 检查是否有负权环（如果第V次还有更新，则存在负权环）
 */

function bellmanFord(
  edges: Edge[],
  n: number,
  source: number
): {dist: number[], hasNegativeCycle: boolean} | null {
  // 初始化距离
  const dist = new Array(n).fill(Infinity);
  dist[source] = 0;

  // 进行V-1次松弛
  for (let i = 0; i < n - 1; i++) {
    let updated = false;
    for (const {from, to, weight} of edges) {
      if (dist[from] !== Infinity && dist[from] + weight < dist[to]) {
        dist[to] = dist[from] + weight;
        updated = true;
      }
    }
    // 如果没有更新，提前结束
    if (!updated) break;
  }

  // 检查负权环
  let hasNegativeCycle = false;
  for (const {from, to, weight} of edges) {
    if (dist[from] !== Infinity && dist[from] + weight < dist[to]) {
      hasNegativeCycle = true;
      break;
    }
  }

  return {dist, hasNegativeCycle};
}

/**
 * Bellman-Ford算法（检测负权环并输出路径）
 */
function bellmanFordWithPath(
  edges: Edge[],
  n: number,
  source: number,
  target: number
): {distance: number, path: number[], hasNegativeCycle: boolean} | null {
  const dist = new Array(n).fill(Infinity);
  const parent = new Array(n).fill(-1);
  dist[source] = 0;

  // 松弛
  for (let i = 0; i < n - 1; i++) {
    let updated = false;
    for (const {from, to, weight} of edges) {
      if (dist[from] !== Infinity && dist[from] + weight < dist[to]) {
        dist[to] = dist[from] + weight;
        parent[to] = from;
        updated = true;
      }
    }
    if (!updated) break;
  }

  // 检查负权环
  for (const {from, to, weight} of edges) {
    if (dist[from] !== Infinity && dist[from] + weight < dist[to]) {
      return null;  // 存在负权环，无法得到正确最短路
    }
  }

  if (dist[target] === Infinity) return null;

  // 重建路径
  const path: number[] = [];
  let curr = target;
  while (curr !== -1) {
    path.push(curr);
    curr = parent[curr];
  }

  return {
    distance: dist[target],
    path: path.reverse(),
    hasNegativeCycle: false
  };
}
```

### 3.3 Floyd-Warshall算法

**Floyd-Warshall算法** 用于计算所有顶点对之间的最短路径。

```typescript
/**
 * Floyd-Warshall算法
 *
 * 时间复杂度：O(V³)
 * 空间复杂度：O(V²)
 *
 * 算法思想：动态规划
 * dp[k][i][j] = 使用顶点0..k作为中间节点，i到j的最短路径
 * dp[k][i][j] = min(dp[k-1][i][j], dp[k-1][i][k] + dp[k-1][k][j])
 *
 * 空间优化：只用2维数组，因为dp[k]只依赖dp[k-1]
 */

function floydWarshall(n: number, edges: Edge[]): number[][] {
  // 初始化距离矩阵
  const dist = Array.from({length: n}, () => new Array(n).fill(Infinity));

  // 对角线为0
  for (let i = 0; i < n; i++) {
    dist[i][i] = 0;
  }

  // 添加边
  for (const {from, to, weight} of edges) {
    dist[from][to] = weight;
    // 如果是无向图，还需要 dist[to][from] = weight;
  }

  // Floyd-Warshall
  for (let k = 0; k < n; k++) {
    for (let i = 0; i < n; i++) {
      for (let j = 0; j < n; j++) {
        if (dist[i][k] + dist[k][j] < dist[i][j]) {
          dist[i][j] = dist[i][k] + dist[k][j];
        }
      }
    }
  }

  return dist;
}

/**
 * Floyd-Warshall（输出路径）
 */
function floydWarshallWithPath(n: number, edges: Edge[]): {
  dist: number[][],
  next: number[][]
} {
  const dist = Array.from({length: n}, () => new Array(n).fill(Infinity));
  const next = Array.from({length: n}, () => new Array(n).fill(-1));

  for (let i = 0; i < n; i++) {
    dist[i][i] = 0;
  }

  for (const {from, to, weight} of edges) {
    dist[from][to] = weight;
    next[from][to] = to;
  }

  for (let k = 0; k < n; k++) {
    for (let i = 0; i < n; i++) {
      for (let j = 0; j < n; j++) {
        if (dist[i][k] + dist[k][j] < dist[i][j]) {
          dist[i][j] = dist[i][k] + dist[k][j];
          next[i][j] = next[i][k];
        }
      }
    }
  }

  return {dist, next};
}

/**
 * 重建路径
 */
function reconstructPath(next: number[][], source: number, target: number): number[] | null {
  if (next[source][target] === -1) return null;

  const path = [source];
  let curr = source;

  while (curr !== target) {
    curr = next[curr][target];
    if (curr === -1) return null;  // 无法到达
    path.push(curr);
  }

  return path;
}
```

### 3.4 最短路径算法对比

| 算法 | 时间复杂度 | 空间复杂度 | 适用场景 |
|------|------------|------------|----------|
| **BFS** | O(V+E) | O(V) | 无权图最短路径 |
| **Dijkstra** | O((V+E)logV) | O(V) | 非负权单源最短路 |

## 四、最小生成树

### 4.1 Prim算法

**Prim算法** 是构造最小生成树的贪心算法。

**核心思想：** 从一个顶点开始，每次选择连接已访问顶点和未访问顶点的最小权边。

```typescript
/**
 * Prim算法
 *
 * 时间复杂度：O((V + E) log V)  使用优先队列
 * 空间复杂度：O(V)
 *
 * 贪心策略：每次选择连接已选集合和未选集合的最小权边
 */

function prim(n: number, edges: Edge[]): {weight: number, mst: Edge[]} {
  // 构建邻接表
  const adj = Array.from({length: n}, (): Array<[number, number]> => []);
  for (const {from, to, weight} of edges) {
    adj[from].push([to, weight]);
    adj[to].push([from, weight]);
  }

  const visited = new Array(n).fill(false);
  const pq: Array<[number, number, number]> = [];  // [weight, from, to]
  let totalWeight = 0;
  const mst: Edge[] = [];

  // 从顶点0开始
  visited[0] = true;
  for (const [to, weight] of adj[0]) {
    pq.push([weight, 0, to]);
  }

  while (pq.length > 0 && mst.length < n - 1) {
    pq.sort((a, b) => a[0] - b[0]);
    const [weight, from, to] = pq.shift()!;

    if (visited[to]) continue;

    visited[to] = true;
    totalWeight += weight;
    mst.push({from, to, weight});

    for (const [next, w] of adj[to]) {
      if (!visited[next]) {
        pq.push([w, to, next]);
      }
    }
  }

  return {weight: totalWeight, mst};
}
```

### 4.2 Kruskal算法

**Kruskal算法** 也是构造最小生成树的贪心算法，但采用不同的策略。

**核心思想：** 按边的权值从小到大排序，每次选择不形成环的最小权边。

```typescript
/**
 * Kruskal算法
 *
 * 时间复杂度：O(E log E)
 * 空间复杂度：O(V + E)
 *
 * 贪心策略：按边权排序，依次选择不形成环的最小边
 * 使用并查集判断是否形成环
 */

/**
 * 并查集
 */
class UnionFind {
  parent: number[];
  rank: number[];

  constructor(n: number) {
    this.parent = new Array(n).fill(0).map((_, i) => i);
    this.rank = new Array(n).fill(0);
  }

  find(x: number): number {
    if (this.parent[x] !== x) {
      this.parent[x] = this.find(this.parent[x]);  // 路径压缩
    }
    return this.parent[x];
  }

  union(x: number, y: number): boolean {
    const px = this.find(x);
    const py = this.find(y);

    if (px === py) return false;  // 已在同一集合

    // 按秩合并
    if (this.rank[px] < this.rank[py]) {
      this.parent[px] = py;
    } else if (this.rank[px] > this.rank[py]) {
      this.parent[py] = px;
    } else {
      this.parent[py] = px;
      this.rank[px]++;
    }

    return true;
  }

  connected(x: number, y: number): boolean {
    return this.find(x) === this.find(y);
  }
}

function kruskal(n: number, edges: Edge[]): {weight: number, mst: Edge[]} {
  // 按权值排序
  edges.sort((a, b) => (a.weight || 0) - (b.weight || 0));

  const uf = new UnionFind(n);
  let totalWeight = 0;
  const mst: Edge[] = [];

  for (const edge of edges) {
    if (uf.union(edge.from, edge.to)) {
      totalWeight += edge.weight || 0;
      mst.push(edge);

      if (mst.length === n - 1) break;  // 已有n-1条边
    }
  }

  return {weight: totalWeight, mst};
}
```

### 4.3 Prim vs Kruskal

| 特性 | Prim | Kruskal |
|------|------|---------|
| **策略** | 选择顶点 | 选择边 |
| **适用图** | 稠密图 | 稀疏图 |
| **时间复杂度** | O(V²) 或 O(E log V) | O(E log E) |
| **实现方式** | 邻接矩阵/优先队列 | 边排序 + 并查集 |

## 五、拓扑排序

### 5.1 拓扑排序原理

**拓扑排序** 是对有向无环图（DAG）的顶点进行排序，使得对于每条有向边 (u, v)，u 都在 v 之前。

**应用场景：** 任务调度、课程安排、依赖解析。

```typescript
/**
 * 拓扑排序
 *
 * 适用条件：有向无环图（DAG）
 *
 * 方法1：Kahn算法（BFS）
 * 方法2：DFS后序遍历（栈）
 */

/**
 * Kahn算法（BFS + 度）
 */
function topologicalSortKahn(n: number, edges: Edge[]): number[] | null {
  // 计算入度
  const inDegree = new Array(n).fill(0);
  const adj = Array.from({length: n}, () => number[] = []);

  for (const {from, to} of edges) {
    adj[from].push(to);
    inDegree[to]++;
  }

  // 将入度为0的顶点加入队列
  const queue: number[] = [];
  for (let i = 0; i < n; i++) {
    if (inDegree[i] === 0) queue.push(i);
  }

  const result: number[] = [];

  while (queue.length > 0) {
    const u = queue.shift()!;
    result.push(u);

    for (const v of adj[u]) {
      inDegree[v]--;
      if (inDegree[v] === 0) {
        queue.push(v);
      }
    }
  }

  // 如果结果不包含所有顶点，说明存在环
  if (result.length !== n) return null;

  return result;
}

/**
 * DFS后序遍历 + 栈
 */
function topologicalSortDFS(n: number, edges: Edge[]): number[] | null {
  // 构建邻接表
  const adj = Array.from({length: n}, () => number[] = []);
  for (const {from, to} of edges) {
    adj[from].push(to);
  }

  const visited = new Array(n).fill(0);  // 0=未访问，1=访问中，2=已完成
  const stack: number[] = [];

  function dfs(u: number): boolean {
    visited[u] = 1;  // 标记为访问中

    for (const v of adj[u]) {
      if (visited[v] === 1) return false;  // 遇到"访问中"的节点，说明有环
      if (visited[v] === 0 && !dfs(v)) return false;
    }

    visited[u] = 2;  // 标记为已完成
    stack.push(u);
    return true;
  }

  // 对所有顶点进行DFS
  for (let i = 0; i < n; i++) {
    if (visited[i] === 0) {
      if (!dfs(i)) return null;  // 发现环
    }
  }

  // 栈中的顺序就是拓扑序（逆序）
  return stack.reverse();
}
```

```typescript
 * 课程表（LeetCode 207）
```

### 7.1 图算法核心要点

```
图算法核心要点

图的表示：
- 邻接矩阵：适合稠密图，O(V²)空间
- 邻接表：适合稀疏图，O(V+E)空间
- 边列表：适合边的批量操作

图的遍历：
- BFS：层次遍历，队列实现，找最短路径
- DFS：深度遍历，栈/递归实现，连通分量

最短路径：
- BFS：无权图，O(V+E)
- Dijkstra：非负权图，O((V+E)logV)
- Bellman-Ford：含负权，O(VE)，可检测负环
- Floyd-Warshall：全源最短路，O(V³)

最小生成树：
- Prim：选顶点，适合稠密图
- Kruskal：选边，适合稀疏图

拓扑排序：
- Kahn算法：BFS + 入度
- DFS后序：利用dfs的特性
```
