# React 状态管理方案对比

## 一、状态管理概述

### 1.1 为什么需要状态管理

在 React 应用中，状态管理是核心问题之一：

| 问题 | 说明 |
|------|------|
| **状态共享** | 多个组件需要访问同一状态 |
| **状态同步** | 状态变化需要及时反映到所有使用处 |
| **状态持久化** | 页面刷新后状态需要保留 |
| **状态调试** | 状态变化需要可追踪、可调试 |
| **性能优化** | 避免不必要的重渲染 |

### 1.2 主流状态管理方案

| 方案 | 特点 | 适用场景 |
|------|------|----------|
| **Zustand** | 极简、轻量、无样板代码 | 中小型项目、快速开发 |
| **Redux Toolkit** | 成熟、生态丰富、时间旅行 | 大型项目、团队协作 |
| **Jotai** | 原子化、细粒度更新 | 需要精确控制的场景 |
| **Recoil** | Facebook 出品、原子化 | 复杂状态依赖关系 |
| **MobX** | 响应式、自动追踪 | 面向对象风格开发 |
| **Context + useReducer** | React 内置、无依赖 | 简单状态共享 |

## 二、Zustand

### 2.1 简介

Zustand 是一个极简的状态管理库，特点：

- **体积极小**：约 1KB gzipped
- **无样板代码**：直接定义状态和方法
- **TypeScript 友好**：完整的类型推断
- **React 外可用**：可在 React 组件外使用

### 2.2 基础用法

```typescript
// stores/userStore.ts
import { create } from 'zustand';
import { persist, devtools } from 'zustand/middleware';

// 定义状态类型
interface UserState {
  // 状态
  user: User | null;
  isAuthenticated: boolean;
  
  // 操作
  login: (user: User) => void;
  logout: () => void;
  updateUser: (data: Partial<User>) => void;
}

// 创建 store
export const useUserStore = create<UserState>()(
  // 开发工具中间件
  devtools(
    // 持久化中间件
    persist(
      (set) => ({
        // 初始状态
        user: null,
        isAuthenticated: false,
        
        // 登录
        login: (user) => set({ user, isAuthenticated: true }),
        
        // 登出
        logout: () => set({ user: null, isAuthenticated: false }),
        
        // 更新用户
        updateUser: (data) => set((state) => ({
          user: state.user ? { ...state.user, ...data } : null,
        })),
      }),
      {
        name: 'user-storage', // localStorage 键名
      }
    )
  )
);
```

### 2.4 异步操作

```typescript
// stores/productStore.ts
import { create } from 'zustand';

interface ProductState {
  products: Product[];
  loading: boolean;
  error: string | null;
  
  fetchProducts: () => Promise<void>;
  addProduct: (product: Omit<Product, 'id'>) => Promise<void>;
}

export const useProductStore = create<ProductState>((set) => ({
  products: [],
  loading: false,
  error: null,
  
  // 获取产品列表
  fetchProducts: async () => {
    set({ loading: true, error: null });
    
    try {
      const response = await fetch('/api/products');
      const products = await response.json();
      set({ products, loading: false });
    } catch (error) {
      set({ error: '获取产品失败', loading: false });
    }
  },
  
  // 添加产品
  addProduct: async (product) => {
    try {
      const response = await fetch('/api/products', {
        method: 'POST',
        body: JSON.stringify(product),
      });
      const newProduct = await response.json();
      
      set((state) => ({
        products: [...state.products, newProduct],
      }));
    } catch (error) {
      set({ error: '添加产品失败' });
    }
  },
}));
```

## 三、Redux Toolkit

### 3.1 简介

Redux Toolkit 是 Redux 官方推荐的写法，特点：

- **简化配置**：自动配置 Redux DevTools
- **内置 Immer**：可以直接修改状态
- **内置 Thunk**：支持异步操作
- **类型安全**：完整的 TypeScript 支持

### 3.2 基础用法

```typescript
// store/slices/userSlice.ts
import { createSlice, createAsyncThunk, PayloadAction } from '@reduxjs/toolkit';

// 异步操作
export const fetchUser = createAsyncThunk(
  'user/fetchUser',
  async (userId: string) => {
    const response = await fetch(`/api/users/${userId}`);
    return response.json();
  }
);

// 初始状态
interface UserState {
  user: User | null;
  loading: boolean;
  error: string | null;
}

const initialState: UserState = {
  user: null,
  loading: false,
  error: null,
};

// 创建 slice
const userSlice = createSlice({
  name: 'user',
  initialState,
  reducers: {
    // 同步操作
    setUser: (state, action: PayloadAction<User>) => {
      state.user = action.payload;
    },
    clearUser: (state) => {
      state.user = null;
    },
    updateUser: (state, action: PayloadAction<Partial<User>>) => {
      if (state.user) {
        // 可以直接修改（Immer 自动处理）
        state.user = { ...state.user, ...action.payload };
      }
    },
  },
  // 异步操作
  extraReducers: (builder) => {
    builder
      .addCase(fetchUser.pending, (state) => {
        state.loading = true;
        state.error = null;
      })
      .addCase(fetchUser.fulfilled, (state, action) => {
        state.loading = false;
        state.user = action.payload;
      })
      .addCase(fetchUser.rejected, (state, action) => {
        state.loading = false;
        state.error = action.error.message || '获取用户失败';
      });
  },
});

export const { setUser, clearUser, updateUser } = userSlice.actions;
export default userSlice.reducer;
```

```typescript
      </button>
```

## 四、Jotai

### 4.1 简介

Jotai 是一个原子化的状态管理库，特点：

- **原子化**：状态拆分为最小单元
- **细粒度更新**：只更新变化的组件
- **派生状态**：自动计算依赖
- **极简 API**：学习成本低

### 4.2 基础用法

```typescript
// atoms/userAtom.ts
import { atom } from 'jotai';

// 基础原子
export const userAtom = atom<User | null>(null);
export const isAuthenticatedAtom = atom(false);

// 只读派生原子
export const userNameAtom = atom((get) => {
  const user = get(userAtom);
  return user?.name || '游客';
});

// 可写派生原子
export const loginAtom = atom(
  null, // 读取函数为 null
  (get, set, user: User) => {
    set(userAtom, user);
    set(isAuthenticatedAtom, true);
  }
);

export const logoutAtom = atom(null, (get, set) => {
  set(userAtom, null);
  set(isAuthenticatedAtom, false);
});
```

### 4.3 组件中使用

```tsx
import { useAtom, useAtomValue, useSetAtom } from 'jotai';
import { userAtom, userNameAtom, loginAtom, logoutAtom } from '@/atoms/userAtom';

// 读取和写入
function UserCard() {
  const [user, setUser] = useAtom(userAtom);
  
  return (
    <div>
      <h1>{user?.name}</h1>
      <button onClick={() => setUser({ ...user, name: '新名字' })}>
        更新
      </button>
    </div>
  );
}

// 只读取
function UserName() {
  const userName = useAtomValue(userNameAtom);
  return <span>{userName}</span>;
}

// 只写入
function LoginButton() {
  const login = useSetAtom(loginAtom);
  const logout = useSetAtom(logoutAtom);
  
  return (
    <div>
      <button onClick={() => login({ id: 1, name: 'Alice' })}>
        登录
      </button>
      <button onClick={logout}>登出</button>
    </div>
  );
}
```

| **性能** | 优秀 | 良好 | 优秀 |

### 5.3 选型建议

| **小型项目** | Zustand | 简单、快速、无样板代码 |

## 七、实战项目示例

### 7.1 Zustand 实战：电商购物车

```typescript
// ==================== 购物车 Store ====================
// stores/cartStore.ts
import { create } from 'zustand';
import { persist, createJSONStorage } from 'zustand/middleware';

// 购物车商品类型
interface CartItem {
  id: string;
  productId: string;
  name: string;
  price: number;
  quantity: number;
  image: string;
  selected: boolean;
}

// 购物车状态类型
interface CartState {
  items: CartItem[];
  totalItems: number;
  totalPrice: number;

  // 操作方法
  addItem: (item: Omit<CartItem, 'selected'>) => void;
  removeItem: (id: string) => void;
  updateQuantity: (id: string, quantity: number) => void;
  toggleSelect: (id: string) => void;
  selectAll: (selected: boolean) => void;
  clearCart: () => void;

  // 计算方法
  getSelectedItems: () => CartItem[];
  getSelectedTotal: () => number;
}

// 创建购物车 Store
export const useCartStore = create<CartState>()(
  persist(
    (set, get) => ({
      items: [],
      totalItems: 0,
      totalPrice: 0,

      // 添加商品
      addItem: (item) => {
        set((state) => {
          const existingIndex = state.items.findIndex(
            (i) => i.productId === item.productId
          );

          let newItems: CartItem[];

          if (existingIndex > -1) {
            // 已存在，增加数量
            newItems = state.items.map((i, index) =>
              index === existingIndex
                ? { ...i, quantity: i.quantity + item.quantity }
                : i
            );
          } else {
            // 不存在，添加新商品
            newItems = [...state.items, { ...item, selected: true }];
          }

          return {
            items: newItems,
            totalItems: newItems.reduce((sum, i) => sum + i.quantity, 0),
            totalPrice: newItems.reduce((sum, i) => sum + i.price * i.quantity, 0),
          };
        });
      },

      // 移除商品
      removeItem: (id) => {
        set((state) => {
          const newItems = state.items.filter((i) => i.id !== id);
          return {
            items: newItems,
            totalItems: newItems.reduce((sum, i) => sum + i.quantity, 0),
            totalPrice: newItems.reduce((sum, i) => sum + i.price * i.quantity, 0),
          };
        });
      },

      // 更新数量
      updateQuantity: (id, quantity) => {
        set((state) => {
          if (quantity <= 0) {
            return get().removeItem(id);
          }

          const newItems = state.items.map((i) =>
            i.id === id ? { ...i, quantity } : i
          );

          return {
            items: newItems,
            totalItems: newItems.reduce((sum, i) => sum + i.quantity, 0),
            totalPrice: newItems.reduce((sum, i) => sum + i.price * i.quantity, 0),
          };
        });
      },

      // 切换选中状态
      toggleSelect: (id) => {
        set((state) => ({
          items: state.items.map((i) =>
            i.id === id ? { ...i, selected: !i.selected } : i
          ),
        }));
      },

      // 全选/取消全选
      selectAll: (selected) => {
        set((state) => ({
          items: state.items.map((i) => ({ ...i, selected })),
        }));
      },

      // 清空购物车
      clearCart: () => {
        set({
          items: [],
          totalItems: 0,
          totalPrice: 0,
        });
      },

      // 获取选中商品
      getSelectedItems: () => {
        return get().items.filter((i) => i.selected);
      },

      // 获取选中商品总价
      getSelectedTotal: () => {
        return get()
          .items.filter((i) => i.selected)
          .reduce((sum, i) => sum + i.price * i.quantity, 0);
      },
    }),
    {
      name: 'cart-storage',
      storage: createJSONStorage(() => localStorage),
      partialize: (state) => ({
        items: state.items,
        totalItems: state.totalItems,
        totalPrice: state.totalPrice,
      }),
    }
  )
);

// ==================== 购物车组件 ====================
// components/Cart/CartItem.tsx
import { useCartStore } from '@/stores/cartStore';

interface CartItemProps {
  item: CartItem;
}

export function CartItemCard({ item }: CartItemProps) {
  const { updateQuantity, removeItem, toggleSelect } = useCartStore();

  return (
    <div className="flex items-center gap-4 p-4 border-b">
      {/* 选择框 */}
      <input
        type="checkbox"
        checked={item.selected}
        onChange={() => toggleSelect(item.id)}
        className="w-5 h-5"
      />

      {/* 商品图片 */}
      <img src={item.image} alt={item.name} className="w-20 h-20 object-cover" />

      {/* 商品信息 */}
      <div className="flex-1">
        <h3 className="font-medium">{item.name}</h3>
        <p className="text-gray-500">¥{item.price.toFixed(2)}</p>
      </div>

      {/* 数量控制 */}
      <div className="flex items-center gap-2">
        <button
          onClick={() => updateQuantity(item.id, item.quantity - 1)}
          className="w-8 h-8 border rounded"
        >
          -
        </button>
        <span className="w-12 text-center">{item.quantity}</span>
        <button
          onClick={() => updateQuantity(item.id, item.quantity + 1)}
          className="w-8 h-8 border rounded"
        >
          +
        </button>
      </div>

      {/* 小计 */}
      <div className="w-24 text-right">
        <p className="font-bold">¥{(item.price * item.quantity).toFixed(2)}</p>
      </div>

      {/* 删除按钮 */}
```

```typescript
  return theme;

  set(themeAtom, next);
});

export function ThemeProvider({ children }: { children: React.ReactNode }) {
  const [theme] = useAtom(themeAtom);
  const [resolvedTheme] = useAtom(resolvedThemeAtom);

  // 应用主题到 DOM
  useEffect(() => {
    const root = document.documentElement;

    // 移除旧主题类
    root.classList.remove('light', 'dark');

    // 添加新主题类
    root.classList.add(resolvedTheme);

    // 更新 color-scheme
    root.style.colorScheme = resolvedTheme;
  }, [resolvedTheme]);

  // 监听系统主题变化
  useEffect(() => {
    if (theme !== 'system') return;

    const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');

    const handleChange = () => {
      // 触发重新计算
      const newTheme = mediaQuery.matches ? 'dark' : 'light';
      document.documentElement.classList.remove('light', 'dark');
      document.documentElement.classList.add(newTheme);
    };

    mediaQuery.addEventListener('change', handleChange);
    return () => mediaQuery.removeEventListener('change', handleChange);
  }, [theme]);

  return <>{children}</>;
}
```

#### Q3：Jotai 的原子化设计有什么优势？

**答案要点：**

1. **细粒度更新**：每个原子独立更新，不影响其他原子
2. **自动依赖追踪**：派生原子自动追踪依赖
3. **最小化重渲染**：只有使用变化原子的组件才会重渲染
4. **组合性强**：原子可以自由组合和派生

**代码示例：**

```typescript

// Redux DevTools 会自动支持时间旅行

// ==================== 自定义历史记录 ====================

interface HistoryState<T> {
  past: T[];
  present: T;
  future: T[];
}

interface HistoryActions<T> {
  set: (value: T) => void;
  undo: () => void;
  redo: () => void;
  canUndo: boolean;
  canRedo: boolean;
}

function useHistory<T>(initialValue: T): [T, HistoryActions<T>] {
  const [state, setState] = useState<HistoryState<T>>({
    past: [],
    present: initialValue,
    future: [],
  });

  const set = useCallback((value: T) => {
    setState((prev) => ({
      past: [...prev.past, prev.present],
      present: value,
      future: [],
    }));
  }, []);

  const undo = useCallback(() => {
    setState((prev) => {
      if (prev.past.length === 0) return prev;
      const previous = prev.past[prev.past.length - 1];
      return {
        past: prev.past.slice(0, -1),
        present: previous,
        future: [prev.present, ...prev.future],
      };
    });
  }, []);

  const redo = useCallback(() => {
    setState((prev) => {
      if (prev.future.length === 0) return prev;
      const next = prev.future[0];
      return {
        past: [...prev.past, prev.present],
        present: next,
        future: prev.future.slice(1),
      };
    });
  }, []);

  return [
    state.present,
    {
      set,
      undo,
      redo,
      canUndo: state.past.length > 0,
      canRedo: state.future.length > 0,
    },
  ];
}
```

#### Q6：如何实现一个支持离线优先的数据同步系统？

```typescript
// ==================== 离线数据同步 Store ====================
// stores/offlineStore.ts
import { create } from 'zustand';
import { persist, createJSONStorage } from 'zustand/middleware';

interface SyncItem {
  id: string;
  type: 'create' | 'update' | 'delete';
  entity: string;
  data: any;
  timestamp: number;
  synced: boolean;
}

interface OfflineState {
  // 离线队列
  queue: SyncItem[];
  // 本地数据缓存
  cache: Record<string, any>;
  // 同步状态
  isOnline: boolean;
  isSyncing: boolean;
  lastSyncTime: number | null;

  // 操作方法
  addToQueue: (item: Omit<SyncItem, 'id' | 'timestamp' | 'synced'>) => void;
  removeFromQueue: (id: string) => void;
  updateCache: (key: string, data: any) => void;
  setOnline: (online: boolean) => void;
  sync: () => Promise<void>;
}

export const useOfflineStore = create<OfflineState>()(
  persist(
    (set, get) => ({
      queue: [],
      cache: {},
      isOnline: typeof navigator !== 'undefined' ? navigator.onLine : true,
      isSyncing: false,
      lastSyncTime: null,

      addToQueue: (item) => {
        const syncItem: SyncItem = {
          ...item,
          id: crypto.randomUUID(),
          timestamp: Date.now(),
          synced: false,
        };

        set((state) => ({
          queue: [...state.queue, syncItem],
        }));

        // 如果在线，立即尝试同步
        if (get().isOnline) {
          get().sync();
        }
      },

      removeFromQueue: (id) => {
        set((state) => ({
          queue: state.queue.filter((item) => item.id !== id),
        }));
      },

      updateCache: (key, data) => {
        set((state) => ({
          cache: { ...state.cache, [key]: data },
        }));
      },

      setOnline: (online) => {
        set({ isOnline: online });
        if (online) {
          get().sync();
        }
      },

      sync: async () => {
        const { queue, isSyncing } = get();
        if (isSyncing || queue.length === 0) return;

        set({ isSyncing: true });

        try {
          // 按时间顺序同步
          const sortedQueue = [...queue].sort((a, b) => a.timestamp - b.timestamp);

          for (const item of sortedQueue) {
            try {
              const response = await fetch(`/api/${item.entity}`, {
                method: item.type === 'create' ? 'POST' : item.type === 'update' ? 'PUT' : 'DELETE',
                headers: { 'Content-Type': 'application/json' },
                body: JSON.stringify(item.data),
              });

              if (response.ok) {
                get().removeFromQueue(item.id);
              }
            } catch (error) {
              console.error(`同步失败: ${item.id}`, error);
            }
          }

          set({ lastSyncTime: Date.now() });
        } finally {
          set({ isSyncing: false });
        }
      },
    }),
    {
      name: 'offline-store',
      storage: createJSONStorage(() => localStorage),
    }
  )
);

// ==================== 离线感知组件 ====================
// components/OfflineIndicator.tsx
import { useEffect } from 'react';
import { useOfflineStore } from '@/stores/offlineStore';

export function OfflineIndicator() {
  const { isOnline, isSyncing, queue, setOnline, sync } = useOfflineStore();

  useEffect(() => {
    const handleOnline = () => setOnline(true);
    const handleOffline = () => setOnline(false);

    window.addEventListener('online', handleOnline);
    window.addEventListener('offline', handleOffline);

    return () => {
      window.removeEventListener('online', handleOnline);
      window.removeEventListener('offline', handleOffline);
    };
  }, [setOnline]);

  if (isOnline && queue.length === 0) return null;

  return (
    <div className="fixed bottom-4 right-4 p-4 bg-yellow-100 rounded-lg shadow">
      {!isOnline && (
        <p className="text-yellow-800">您当前处于离线状态</p>
      )}
      {isOnline && queue.length > 0 && (
        <div>
          <p className="text-yellow-800">
            正在同步 {queue.length} 个待处理操作...
          </p>
          <button
            onClick={sync}
            disabled={isSyncing}
            className="mt-2 px-4 py-1 bg-yellow-500 text-white rounded"
          >
            {isSyncing ? '同步中...' : '立即同步'}
          </button>
        </div>
      )}
    </div>
  );
}
```

```tsx
// stores/cartStore.ts - 购物车状态
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

interface CartItem {
  id: string;
  name: string;
  price: number;
  quantity: number;
  image: string;
}

interface CartState {
  items: CartItem[];
  isOpen: boolean;

  // Actions
  addItem: (item: Omit<CartItem, 'quantity'>, quantity?: number) => void;
  removeItem: (id: string) => void;
  updateQuantity: (id: string, quantity: number) => void;
  clearCart: () => void;
  toggleCart: () => void;

  // Computed
  getTotalItems: () => number;
  getTotalPrice: () => number;
}

export const useCartStore = create<CartState>()(
  persist(
    (set, get) => ({
      items: [],
      isOpen: false,

      addItem: (item, quantity = 1) => {
        set((state) => {
          const existingItem = state.items.find((i) => i.id === item.id);

          if (existingItem) {
            return {
              items: state.items.map((i) =>
                i.id === item.id
                  ? { ...i, quantity: i.quantity + quantity }
                  : i
              ),
            };
          }

          return {
            items: [...state.items, { ...item, quantity }],
          };
        });
      },

      removeItem: (id) => {
        set((state) => ({
          items: state.items.filter((i) => i.id !== id),
        }));
      },

      updateQuantity: (id, quantity) => {
        if (quantity <= 0) {
          get().removeItem(id);
          return;
        }

        set((state) => ({
          items: state.items.map((i) =>
            i.id === id ? { ...i, quantity } : i
          ),
        }));
      },

      clearCart: () => set({ items: [] }),
      toggleCart: () => set((state) => ({ isOpen: !state.isOpen })),

      getTotalItems: () => get().items.reduce((sum, i) => sum + i.quantity, 0),
      getTotalPrice: () =>
        get().items.reduce((sum, i) => sum + i.price * i.quantity, 0),
    }),
    {
      name: 'cart-storage',
    }
  )
);

// stores/userStore.ts - 用户状态

          throw error;
        }
      },

      logout: () => {
        set({ user: null, isAuthenticated: false });
      },

      updateProfile: (data) => {
        set((state) => ({
          user: state.user ? { ...state.user, ...data } : null,
        }));
      },
    }),
    { name: 'user-store' }
  )
);
```

## 十四、常见面试题

### 14.1 基础概念面试题

**问题1：React 为什么需要状态管理？**

答案：
- **Props 钻探问题**：多层组件传递 props 时，中间组件不需要这些 props 但必须转发
- **状态共享**：多个不相关组件需要访问和修改同一份数据
- **全局状态**：用户登录状态、主题设置等应用级状态
- **时间旅行调试**：需要记录状态变化历史，方便调试

**问题2：localStorage 和状态管理库有什么区别？**

答案：
- **localStorage**：浏览器持久化 API，只能存储字符串，需要手动序列化/反序列化
- **状态管理库**：内存中的状态管理，自动处理序列化，支持订阅和更新通知
- **最佳实践**：状态管理库管理运行时状态，配合 localStorage 进行持久化

**问题3：什么时候应该使用状态管理库？**

答案：
- 应用有多个页面需要共享状态
- 组件层级较深（超过 3 层）
- 状态变化需要触发多个组件更新
- 需要时间旅行调试功能
- 状态逻辑复杂（多个 reducer、异步操作）

### 14.2 Zustand 相关面试题

**问题4：Zustand 相比 Redux 有什么优势？**

答案：
- **代码量少**：Zustand 没有 boilerplate，不需要 action types、reducers 等
- **学习曲线低**：API 简单直观
- **性能好**：默认浅比较，不需要 memoize 选择器
- **TypeScript 支持好**：类型推断自然
- **DevTools 是可选的**：生产环境可以移除

**问题5：Zustand 的 persist 中间件如何使用？**

答案：
```tsx
import { create } from 'zustand';
import { persist } from 'zustand/middleware';

const useStore = create(
  persist(
    (set) => ({
      // 状态
      name: '',
      // actions
      setName: (name) => set({ name }),
    }),
    {
      name: 'my-storage', // localStorage key
      storage: createJSONStorage(() => localStorage),
      partialize: (state) => ({ name: state.name }), // 只持久化部分
    }
  )
);
```

### 14.3 Redux 相关面试题

**问题6：Redux Toolkit 相比传统 Redux 有什么改进？**

答案：
- **简化样板代码**：createSlice 自动生成 action creators 和 reducers
- **内置 Immer**：可以直接修改 state
- **内置 Redux Thunk**：异步操作开箱即用
- **configureStore**：简化 store 配置
- **RTK Query**：内置数据获取和缓存

**问题7：Redux 的单向数据流是什么？**

答案：
1. 调用 `store.dispatch(action)`
2. Redux 调用 reducer，传入当前 state 和 action
3. reducer 返回新的 state
4. Store 调用所有订阅的监听器
5. 监听器调用 `getState()` 获取新状态，更新组件

### 14.4 性能优化面试题

**问题8：如何避免状态更新导致不必要的重渲染？**

答案：
1. **使用选择器**：只选择需要的状态片段
```tsx
// ❌ 不好：每次 store 变化都重渲染
const { name, age } = useStore();

// ✅ 好：只关心 name
const name = useStore((state) => state.name);
```

2. **使用 shallow 比较**
```tsx
const { name, age } = useStore(
  (state) => ({ name: state.name, age: state.age }),
  shallow
);
```

3. **拆分状态**：将不相关状态分离到不同 store
```tsx
// 不好：所有状态在一个 store
const store = create((set) => ({
  user: null,
  cart: [],
  theme: 'light',
  // ...
}));

// 好：按功能拆分
const useUserStore = create(/* ... */);
const useCartStore = create(/* ... */);
const useUIStore = create(/* ... */);
```

**问题9：如何处理异步状态更新？**

答案：
```tsx
// Zustand
const useStore = create((set) => ({
  data: null,
  loading: false,

  fetchData: async () => {
    set({ loading: true });
    try {
      const data = await api.fetchData();
      set({ data, loading: false });
    } catch (error) {
      set({ loading: false, error });
    }
  },
}));

// Redux Toolkit
const slice = createSlice({
  name: 'data',
  initialState: { data: null, loading: false },
  reducers: {},
  extraReducers: (builder) => {
    builder
      .addCase(fetchData.pending, (state) => {
        state.loading = true;
      })
      .addCase(fetchData.fulfilled, (state, action) => {
        state.data = action.payload;
        state.loading = false;
      })
      .addCase(fetchData.rejected, (state) => {
        state.loading = false;
      });
  },
});
```

### 14.5 综合设计面试题

**问题10：如何设计一个聊天应用的状态管理？**

答案：
```tsx
// stores/chatStore.ts
import { create } from 'zustand';

interface Message {
  id: string;
  content: string;
  senderId: string;
  timestamp: number;
  status: 'sending' | 'sent' | 'failed';
}

interface ChatState {
  conversations: Map<string, Message[]>;
  activeConversationId: string | null;
  onlineUsers: Set<string>;

  // Actions
  sendMessage: (conversationId: string, content: string) => void;
  receiveMessage: (conversationId: string, message: Message) => void;
  setActiveConversation: (id: string) => void;
  addOnlineUser: (userId: string) => void;
  removeOnlineUser: (userId: string) => void;

  // Computed
  getActiveConversationMessages: () => Message[];
  getUnreadCount: () => number;
}

export const useChatStore = create<ChatState>()((set, get) => ({
  conversations: new Map(),
  activeConversationId: null,
  onlineUsers: new Set(),

  sendMessage: (conversationId, content) => {
    const message: Message = {
      id: crypto.randomUUID(),
      content,
      senderId: 'me',
      timestamp: Date.now(),
      status: 'sending',
    };

    set((state) => {
      const newConversations = new Map(state.conversations);
      const messages = newConversations.get(conversationId) || [];
      newConversations.set(conversationId, [...messages, message]);
      return { conversations: newConversations };
    });

    // 模拟发送
    setTimeout(() => {
      set((state) => {
        const newConversations = new Map(state.conversations);
        const messages = newConversations.get(conversationId) || [];
        const updatedMessages = messages.map((m) =>
          m.id === message.id ? { ...m, status: 'sent' as const } : m
        );
        newConversations.set(conversationId, updatedMessages);
        return { conversations: newConversations };
      });
    }, 1000);
  },

  receiveMessage: (conversationId, message) => {
    set((state) => {
      const newConversations = new Map(state.conversations);
      const messages = newConversations.get(conversationId) || [];
      newConversations.set(conversationId, [...messages, message]);
      return { conversations: newConversations };
    });
  },

  setActiveConversation: (id) => set({ activeConversationId: id }),

  addOnlineUser: (userId) => {
    set((state) => ({
      onlineUsers: new Set([...state.onlineUsers, userId]),
    }));
  },

  removeOnlineUser: (userId) => {
    set((state) => {
      const newOnlineUsers = new Set(state.onlineUsers);
      newOnlineUsers.delete(userId);
      return { onlineUsers: newOnlineUsers };
    });
  },

  getActiveConversationMessages: () => {
    const { conversations, activeConversationId } = get();
    return activeConversationId
      ? conversations.get(activeConversationId) || []
      : [];
  },

  getUnreadCount: () => {
    // 计算未读消息数
    return 0;
  },
}));
```
