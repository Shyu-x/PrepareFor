## 1. React Native基础

### 1.1 React Native vs React Web

```
架构对比：

React Web：
┌─────────────────────────────────────┐
│  浏览器（Chrome/Safari）      │
│  ┌───────────────────────────┐  │
│  │  React运行时               │  │
│  │  DOM渲染                  │  │
│  │  Web API                   │  │
│  └───────────────────────────┘  │
└─────────────────────────────────────┘

React Native (iOS）：
┌─────────────────────────────────────┐
│  JavaScriptCore（JSC/V8）     │
│  ┌───────────────────────────┐  │
│  │  React Native运行时        │  │
│  │  Native Modules桥接       │  │
│  │  UIKit/Swift                │  │
│  └───────────────────────────┘  │
└─────────────────────────────────────┘

React Native (Android）：
┌─────────────────────────────────────┐
│  JavaScriptCore（Hermes/V8）  │
│  ┌───────────────────────────┐  │
│  │  React Native运行时        │  │
│  │  Native Modules桥接       │  │
│  │  Android原生组件           │  │
│  └───────────────────────────┘  │
└─────────────────────────────────────┘
```

| 特性 | React Web | React Native |
|------|----------|-------------|
| **运行环境** | 浏览器 | JavaScriptCore + 原生桥接 |
| **UI渲染** | DOM | 原生组件 |
| **网络请求** | fetch/XHR | fetch + 原生桥接 |
| **存储** | localStorage/IndexedDB | AsyncStorage |
| **导航** | History API | 原生导航 |
| **性能** | JavaScript引擎 | 原生优化 |

### 1.2 React Native版本对比

### 5.1 Fetch API基础

```javascript
import React, { useState, useEffect } from 'react';
import { Text, StyleSheet } from 'react-native';

function FetchExample() {
  const [data, setData] = useState<any>(null);
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    fetchData();
  }, []);

  const fetchData = async () => {
    setLoading(true);
    setError(null);

    try {
      const response = await fetch('https://api.example.com/data');

      if (!response.ok) {
        throw new Error(`HTTP错误: ${response.status}`);
      }

      const json = await response.json();
      setData(json);
    } catch (err) {
      setError(err.message);
    } finally {
      setLoading(false);
    }
  };

  if (loading) {
    return <Text style={styles.text}>加载中...</Text>;
  }

  if (error) {
    return <Text style={[styles.text, styles.error]}>错误：{error}</Text>;
  }

  return (
    <View style={styles.container}>
      <Text style={styles.text}>数据：{JSON.stringify(data)}</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
  text: {
    fontSize: 16,
  },
  error: {
    color: 'red',
  },
});
```

import AsyncStorage from '@react-native-async-storage/async-storage';

// 存储数据
const saveData = async (key: string, value: any) => {
  try {
    await AsyncStorage.setItem(key, JSON.stringify(value));
    console.log('数据保存成功');
  } catch (error) {
    console.error('数据保存失败', error);
  }

// 读取数据
const loadData = async (key: string) => {
  try {
    const json = await AsyncStorage.getItem(key);
    if (json !== null) {
      return JSON.parse(json);
    }
    return null;
  } catch (error) {
    console.error('数据读取失败', error);
    return null;
  }
};

// 删除数据
const removeData = async (key: string) => {
  try {
    await AsyncStorage.removeItem(key);
    console.log('数据删除成功');
  } catch (error) {

  }
};

// 清空所有数据
const clearAllData = async () => {

  } catch (error) {

  const data = [...Array(1000).keys()].map(key => ({ id: key, name: `Item ${key}` }));

  return (
    <FlatList
      data={data}
      renderItem={({ item }) => (
        <View key={item.id}>
          <Text>{item.name}</Text>
        </View>
      )}
      keyExtractor={(item) => item.id}
      getItemLayout={(data, index) => ({
        length: 50,
        offset: 50 * index,
      })}
      windowSize={50}
      initialNumToRender={10}
      maxToRenderPerBatch={5}
      removeClippedSubviews={true}
    />
  );
}

// 2. 避免内联函数
// ❌ 不推荐
<FlatList
  renderItem={({ item }) => (
    <View>
      <ExpensiveComponent data={item} />
    </View>
  )}
/>

// ✅ 推荐
const renderItem = React.memo(({ item }: { item: any }) => (
  <View>
    <ExpensiveComponent data={item} />
  </View>
));

// 3. 使用memo和useCallback
const ListItem = React.memo(({ item, onPress }: any) => {
  console.log('渲染item:', item.id);
  return (
    <TouchableOpacity onPress={onPress}>
      <Text>{item.name}</Text>
    </TouchableOpacity>
  );
});

## 8. 打包与发布

### 8.1 生成签名密钥（iOS）

```bash
# 1. 生成签名密钥（不需要上传到App Store）
keytool -M
# 保存生成的Key.p8文件

# 2. 生成推送通知密钥（上传到App Store）
keytool -M
# 记录显示的Key ID，上传到Apple Developer Console
```

###`8.2 生成密钥库（Android）

```bash
# 1. 生成密钥库
keytool -J -keystore my-app.keystore -alias my-alias -keypass PASSWORD -storepass PASSWORD

# 2. 查看密钥
keytool -J -list my-app.keystore

# 3. 删除密钥
keytool -J -delete my-app.keystore alias my-alias
```
