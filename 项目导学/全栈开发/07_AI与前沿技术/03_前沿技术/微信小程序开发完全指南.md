## 1. 小程序基础

### 1.1 小程序 vs H5 vs 原生App对比

微信小程序是一种运行在微信内部的轻量级应用，与传统的Web页面（H5）和原生App相比，有各自的优势和适用场景。

| 对比维度 | 微信小程序 | H5页面 | 原生App |
|----------|------------|--------|---------|
| **运行环境** | 微信客户端 | 浏览器 | iOS/Android系统 |
| **开发成本** | 中等 | 低 | 高 |
| **用户体验** | 流畅，接近原生 | 一般 | 最佳 |
| **获取用户** | 通过微信扫一扫/分享 | 通过URL传播 | 应用商店下载 |
| **性能** | 较好 | 一般 | 最佳 |
| **系统权限** | 受限于微信 | 受限于浏览器 | 完全开放 |
| **更新方式** | 热更新（无需审核） | 热更新 | 需审核上架 |
| **包体积限制** | 单包不超过2MB | 无限制 | 无限制 |

**小程序的核心优势：**

1. **流量入口丰富**：微信扫一扫、分享链接、公众号关联、附近的小程序等
2. **用户获取成本低**：无需下载，扫码即用
3. **开发效率高**：使用前端技术栈，学习成本低
4. **生态完善**：微信支付、登录、广告等能力一应俱全

**app.js - 小程序入口：**

```javascript
// app.js
// 小程序启动时执行的全局逻辑
App({
  // 小程序初始化时执行（全局只触发一次）
  onLaunch: function(options) {
    // 获取小程序启动参数
    console.log('启动参数:', options);
    // 初始化云开发环境
    wx.cloud.init({
      env: 'cloud1-xxxxx',
      traceUser: true
    });
  },

  // 小程序显示时执行（每次显示都会触发）
  onShow: function(options) {
    // 检查用户是否已登录
    const userInfo = wx.getStorageSync('userInfo');
    if (!userInfo) {
      // 跳转登录页
    }
  },

  // 小程序隐藏时执行
  onHide: function() {
    // 保存用户状态
    console.log('小程序进入后台');
  },

  // 全局数据
  globalData: {
    userInfo: null,
    apiBase: 'https://api.example.com',
    version: '1.0.0'
  },

  // 全局方法
  getUserInfo: function(callback) {
    // 获取用户信息的全局方法
    const that = this;
    if (this.globalData.userInfo) {
      callback(this.globalData.userInfo);
    } else {
      wx.getUserProfile({
        desc: '获取用户信息用于完善会员资料',
        success: function(res) {
          that.globalData.userInfo = res.userInfo;
          callback(res.userInfo);
        }
      });
    }
  }
});
```

## 2. WXML模板

WXML（WeiXin Markup Language）是微信小程序的模板语言，语法与HTML类似，但有一些特有的语法和组件。

### 2.1 数据绑定

小程序使用 `{{}}` 语法进行数据绑定，将逻辑层的数据渲染到视图层。

**基本数据绑定：**

```html
<!-- wxml 文件 -->
<view class="container">
  <!-- 字符串绑定 -->
  <text class="title">{{title}}</text>

  <!-- 数字绑定 -->
  <text class="count">数量: {{count}}</text>

  <!-- 布尔值绑定 -->
  <text class="status">{{isActive ? '激活' : '未激活'}}</text>

  <!-- 对象绑定 -->
  <view class="user-info">
    <text>姓名: {{user.name}}</text>
    <text>年龄: {{user.age}}</text>
  </view>

  <!-- 数组绑定 -->
  <text class="first-item">第一项: {{items[0]}}</text>
</view>
```

```javascript
// js 文件
Page({
  data: {
    title: '欢迎使用小程序',
    count: 10,
    isActive: true,
    user: {
      name: '张三',
      age: 25
    },
    items: ['苹果', '香蕉', '橙子']
  }
})
```

**运算表达式：**

```html
<!-- 算术运算 -->
<text>{{a + b}}</text>  <!-- 输出: 30 -->

<!-- 三元运算 -->
<text>{{flag ? '是' : '否'}}</text>

<!-- 逻辑判断 -->
<text>{{type === 'vip' ? 'VIP用户' : '普通用户'}}</text>

<!-- 字符串拼接 -->
<text>{{'您好,' + name}}</text>

<!-- 方法调用 -->
<text>{{handler.toUpperCase()}}</text>

<!-- 过滤器（需使用 wxs） -->
<text>{{msg}}</text>
```

### 2.2 列表渲染

使用 `wx:for` 指令可以渲染数组，生成重复的视图结构。

**基础列表渲染：**

```html
<!-- 基础列表渲染 -->
<view class="product-list">
  <view
    wx:for="{{products}}"
    wx:for-item="product"
    wx:for-index="index"
    wx:key="id"
    class="product-item"
  >
    <image src="{{product.image}}" class="product-image" />
    <view class="product-info">
      <text class="product-name">{{product.name}}</text>
      <text class="product-price">¥{{product.price}}</text>
      <text class="product-stock">库存: {{product.stock}}</text>
    </view>
    <button
      bindtap="addToCart"
      data-id="{{product.id}}"
      data-name="{{product.name}}"
      class="add-btn"
    >
      加入购物车
    </button>
  </view>
</view>

<!-- 简写形式（使用默认变量名 item，索引 index） -->
<view wx:for="{{array}}">
  {{index}}: {{item.message}}
</view>

<!-- 多层嵌套循环 -->
<view wx:for="{{categories}}" wx:key="id">
  <view class="category-title">{{item.name}}</view>
  <view
    wx:for="{{item.products}}"
    wx:for-item="product"
    wx:key="id"
    class="product"
  >
    {{product.name}}
  </view>
</view>
```

```javascript
Page({
  data: {
    products: [
      { id: 1, name: 'iPhone 15', price: 5999, stock: 100, image: '/images/iphone.png' },
      { id: 2, name: 'MacBook Pro', price: 9999, stock: 50, image: '/images/mac.png' },
      { id: 3, name: 'AirPods Pro', price: 1999, stock: 200, image: '/images/airpods.png' }
    ]
  },

  // 添加到购物车
  addToCart: function(e) {
    const dataset = e.currentTarget.dataset;
    const productId = dataset.id;
    const productName = dataset.name;

    wx.showToast({
      title: `已将 ${productName} 加入购物车`,
      icon: 'success',
      duration: 1500
    });

    // 获取当前购物车数据
    const cart = wx.getStorageSync('cart') || [];

    // 检查商品是否已在购物车
    const existIndex = cart.findIndex(item => item.id === productId);

    if (existIndex > -1) {
      // 已存在，数量+1
      cart[existIndex].quantity += 1;
    } else {
      // 不存在，添加新商品
      cart.push({
        id: productId,
        name: productName,
        quantity: 1,
        selected: true
      });
    }

    // 保存到本地存储
    wx.setStorageSync('cart', cart);
  }
})
```

**wx:key 的作用：**

wx:key 用于指定列表中项目的唯一标识符，帮助小程序高效地更新列表。当数组元素变化时，小程序可以根据 key 进行高效的 diff 比较，只更新变化的项目，而不是重新渲染整个列表。

```html
<!-- 使用商品 ID 作为唯一标识 -->
<view wx:for="{{products}}" wx:key="id">
  {{item.name}}
</view>

<!-- 使用数组索引作为 key（当列表是静态的或没有唯一ID时） -->
<view wx:for="{{products}}" wx:key="*this">
  {{item}}
</view>

<!-- 复杂对象作为 key -->
<view wx:for="{{objectArray}}" wx:key="id">
  {{item.name}}
</view>
```

### 2.3 条件渲染

使用条件渲染可以根据数据状态决定是否渲染某些元素。

**wx:if 条件渲染：**

```html
<!-- 基本条件渲染 -->
<view wx:if="{{isLogin}}">
  <text>欢迎回来，{{userName}}</text>
</view>
<view wx:else>
  <button bindtap="goToLogin">请先登录</button>
</view>

<!-- 多重条件判断 -->
<view wx:if="{{type === 'success'}}">
  <image src="/images/success.png" />
  <text>操作成功</text>
</view>
<view wx:elif="{{type === 'error'}}">
  <image src="/images/error.png" />
  <text>操作失败</text>
</view>
<view wx:elif="{{type === 'warning'}}">
  <image src="/images/warning.png" />
  <text>警告信息</text>
</view>
<view wx:else>
  <image src="/images/default.png" />
  <text>未知状态</text>
</view>

<!-- wx:if vs hidden 对比 -->
<!-- wx:if：条件为 false 时完全不渲染 -->

  <text>使用 wx:if</text>

<!-- hidden：条件为 true 时添加 hidden 属性（仍然渲染） -->

```

**hidden vs wx:if 的选择：**

```javascript

// 因为 hidden 只是切换 CSS 的 display 属性，不会重新创建/销毁组件

// wx:if 适用于条件很少变化的场景
// 因为 wx:if 会真正地创建/销毁组件，初始渲染成本更低

Page({
  data: {
    // 频繁切换的场景
    showModal: false,
    showTabs: true,

    // 很少变化的场景
    isVipUser: true,
    hasPermission: false
  },

  toggleModal: function() {
    // 频繁切换，使用 hidden 更合适
    this.setData({ showModal: !this.data.showModal });
  },

  checkPermission: function() {
    // 检查权限（很少变化），使用 wx:if 更合适
    const hasPermission = checkUserPermission();
    this.setData({ hasPermission });
  }
})
```

### 2.5 事件处理

小程序的事件系统与Web开发类似，但有一些区别。

**基础事件绑定：**

```html
<!-- bindtap：冒泡事件（事件会向上传递） -->

</view>

</view>

</view>

<!-- 事件传参：通过 data-* 属性 -->

  bindtap="handleClick"
  data-id="123"
  data-name="商品名称"
  data-price="99.9"
>
  点击按钮
</button>

<!-- 触摸事件 -->
<view
  bindtouchstart="handleTouchStart"
  bindtouchmove="handleTouchMove"
  bindtouchend="handleTouchEnd"
  bindtouchcancel="handleTouchCancel"

  触摸区域
</view>

<!-- 通用事件属性 -->

  bindtap="handleTap"
  data-info="自定义数据"
>
  <text>事件对象包含：type, timeStamp, target, currentTarget</text>
</view>
```

```javascript
Page({
  // 事件处理函数
  handleTap: function(e) {
    // 事件对象详解
    console.log('事件类型:', e.type);           // tap
    console.log('时间戳:', e.timeStamp);        // 页面加载到事件发生的毫秒数
    console.log('触发元素:', e.target);         // 触发事件的元素
    console.log('绑定元素:', e.currentTarget);  // 绑定事件的元素

    // 获取 data-* 传递的参数
    const dataset = e.currentTarget.dataset;
    console.log('ID:', dataset.id);
    console.log('名称:', dataset.name);
    console.log('价格:', dataset.price);
  },

  // 触摸事件处理
  handleTouchStart: function(e) {
    // 触摸起始点坐标
    console.log('起始X:', e.touches[0].clientX);
    console.log('起始Y:', e.touches[0].clientY);
    this.startX = e.touches[0].clientX;
    this.startY = e.touches[0].clientY;
  },

  handleTouchMove: function(e) {
    // 触摸移动坐标
    const currentX = e.touches[0].clientX;
    const currentY = e.touches[0].clientY;
    const moveX = currentX - this.startX;
    const moveY = currentY - this.startY;

    // 实现滑动删除等功能
    if (Math.abs(moveX) > Math.abs(moveY)) {
      // 水平滑动
      console.log('水平滑动:', moveX);
    } else {
      // 垂直滑动
      console.log('垂直滑动:', moveY);
    }
  },

  handleTouchEnd: function(e) {
    // 触摸结束
    console.log('触摸结束');
  }
})
```

| longpress | 长按 | 是 |
| longtap | 长按（已废弃，推荐使用 longpress） | 是 |
| touchstart | 触摸开始 | 否 |
| touchmove | 触摸移动 | 否 |
| touchend | 触摸结束 | 否 |
| touchcancel | 触摸被打断（来电等） | 否 |
| input | 输入 | 否 |
| change | 变更 | 否 |
| submit | 表单提交 | 否 |

### 2.6 冒泡事件与非冒泡事件

小程序中的事件分为冒泡事件和非冒泡事件。

**冒泡事件：**

```html
<!-- 冒泡事件示例：点击内部文字会依次触发 text、view1、view2 的事件 -->
<view id="view2" bindtap="onView2Tap">

```

```html

    <text catchtap="onTextTap">点击文字</text>
  </view>
</view>
<!-- 触发顺序：只触发 onTextTap -->
```

## 3. WXSS样式

WXSS（WeiXin Style Sheets）是小程序的样式语言，语法与CSS基本一致，但有一些扩展和限制。

### 3.1 rpx单位

rpx（responsive pixel）是小程序特有的响应式像素单位，可以根据屏幕宽度进行自动计算。

**rpx vs px 对比：**

| 单位 | 说明 | 适用场景 |
|------|------|----------|
| px | 固定像素单位 | 元素大小不变的场景 |
| rpx | 响应式像素，根据屏幕宽度自动缩放 | 元素需要适配不同屏幕 |

**rpx 计算规则：**

```
以 iPhone 6 为标准（屏幕宽度 375px）：
- 1rpx = 0.5px = 1物理像素

换算公式：
- 屏幕宽度为 750rpx（固定值）
- rpx = 750 / 屏幕宽度 px
```

**各机型适配参考：**

| 机型 | 屏幕宽度 | 1rpx = ?px | 750rpx 占屏幕比例 |
|------|----------|------------|-------------------|
| iPhone 5 | 320px | 0.42px | 93.75% |
| iPhone 6 | 375px | 0.5px | 100% |
| iPhone 6 Plus | 414px | 0.55px | 90.6% |
| iPhone X | 375px | 0.5px | 100% |
| iPad | 768px | 1.024px | 97.6% |

**rpx 使用示例：**

```css
/* 固定宽度容器（在不同设备上表现一致） */
.fixed-width {
  width: 375px;  /* 不推荐使用 px 做宽度适配 */
}

/* 响应式宽度（自动适配屏幕） */
.responsive-width {
  width: 700rpx;  /* 推荐：约占屏幕的 93% */
  margin: 0 auto;
}

/* 响应式字体大小 */
.title {
  font-size: 32rpx;  /* iPhone 6 上为 16px */
}

.subtitle {
  font-size: 24rpx;  /* iPhone 6 上为 12px */
}

/* 响应式间距 */
.card {
  padding: 24rpx;
  margin: 20rpx;
  border-radius: 16rpx;
}
```

```javascript
Page({
  // 页面初始数据（类似 React 的 state）
  data: {

      this.setData({ currentId: id });

    this.loadBanners();

    this.loadCategories();

    // 加载商品
    this.loadProducts();

   * 加载轮播图
   */
  loadBanners: function() {
    // 模拟请求
    wx.request({
      url: 'https://api.example.com/banners',
      success: (res) => {
        if (res.data.code === 0) {
          this.setData({ banners: res.data.data });
        }
      }
    });
  },

  /**
   * 加载分类
   */
  loadCategories: function() {
    // 模拟请求
    wx.request({
      url: 'https://api.example.com/categories',
      success: (res) => {
        if (res.data.code === 0) {
          this.setData({ categories: res.data.data });
        }
      }
    });
  },

  /**
   * 加载商品列表
   */
  loadProducts: function() {
    if (this.data.loading || !this.data.hasMore) {
      return Promise.resolve();
    }

    this.setData({ loading: true });

    return new Promise((resolve) => {
      wx.request({
        url: 'https://api.example.com/products',
        data: {
          page: this.data.page,
          pageSize: this.data.pageSize,
          category: this.data.currentTab,
          keyword: this.data.searchKeyword
        },
        success: (res) => {
          if (res.data.code === 0) {
            const { list, hasMore } = res.data.data;
            this.setData({
              products: [...this.data.products, ...list],
              hasMore,
              loading: false
            });
          }
        },
        fail: () => {
          wx.showToast({
            title: '加载失败',
            icon: 'none'
          });
          this.setData({ loading: false });
        },
        complete: () => {
          resolve();
        }
      });
    });
  },

  /**
   * 检查登录状态
   */
  checkLoginStatus: function() {
    const token = wx.getStorageSync('token');
    const userInfo = wx.getStorageSync('userInfo');

    if (token && userInfo) {
      this.setData({
        isLogin: true,
        userInfo
      });
    } else {
      this.setData({
        isLogin: false,
        userInfo: null
      });
    }
  },

  /**
   * 刷新购物车数量
   */
  refreshCartCount: function() {
    const cart = wx.getStorageSync('cart') || [];
    const count = cart.reduce((sum, item) => sum + item.quantity, 0);

    // 更新 tabBar 徽章
    if (count > 0) {
      wx.setTabBarBadge({
        index: 2, // 购物车 tabBar 索引
        text: count.toString()
      });
    } else {
      wx.removeTabBarBadge({ index: 2 });
    }
  },

  /**
   * 保存表单数据到本地
   */
  saveFormData: function() {
    // 保存搜索历史
    if (this.data.searchKeyword) {
      const history = wx.getStorageSync('searchHistory') || [];
      if (!history.includes(this.data.searchKeyword)) {
        history.unshift(this.data.searchKeyword);
        wx.setStorageSync('searchHistory', history.slice(0, 20));
      }
    }
  }
});
```

```javascript
 * 商品卡片组件

Component({
  // 组件选项
  options: {
    // 启用插槽
    multipleSlots: true,
    // 组件样式隔离
    styleIsolation: 'isolated',
    // 允许页面的样式影响组件
    addGlobalClass: true
  },

  // 组件的属性列表（从父组件接收）
  properties: {
    // 简单属性
    title: {
      type: String,
      value: ''
    },
    price: {
      type: Number,
      value: 0
    },
    image: {
      type: String,
      value: ''
    },

    // 带 Observer 的属性
    count: {
      type: Number,
      value: 0,
      observer: function(newVal, oldVal, changedPath) {
        // 属性变化时触发
        console.log('数量变化:', oldVal, '->', newVal);
        this.updateTotalPrice();
      }
    },

    // 对象属性
    product: {
      type: Object,
      value: {}
    },

    // 布尔属性
    disabled: {
      type: Boolean,
      value: false
    }
  },

  // 组件的初始数据
  data: {
    // 计算后的总价
    totalPrice: 0,
    // 加载状态
    loading: false
  },

  // 组件生命周期函数
  lifetimes: {
    created: function() {
      // 组件实例化，但尚未挂载
      console.log('组件创建');
    },
    attached: function() {
      // 组件进入页面节点树
      console.log('组件挂载');
      this.initComponent();
    },
    ready: function() {
      // 组件布局完成
      console.log('组件渲染完成');
    },
    detached: function() {
      // 组件离开页面节点树
      console.log('组件卸载');
      this.cleanup();
    }
  },

  // 所在页面的生命周期（可选）
  pageLifetimes: {
    show: function() {
      // 页面显示
      console.log('组件所在页面显示');
    },
    hide: function() {
      // 页面隐藏
      console.log('组件所在页面隐藏');
    }
  },

  // 方法列表（暴露给视图层）
  methods: {
    /**
     * 初始化组件
     */
    initComponent: function() {
      this.updateTotalPrice();
    },

    /**
     * 更新总价
     */
    updateTotalPrice: function() {
      const price = this.data.price;
      const count = this.data.count;
      this.setData({
        totalPrice: (price * count).toFixed(2)
      });
    },

    /**
     * 处理点击事件
     */
    onTap: function(e) {
      // 触发自定义事件，通知父组件
      this.triggerEvent('tap', {
        id: this.data.product.id,
        name: this.data.title
      });
    },

    /**
     * 处理加号点击
     */
    onAdd: function() {
      const count = this.data.count + 1;
      this.setData({ count });

      // 触发事件通知父组件
      this.triggerEvent('change', {
        count,
        type: 'add'
      });
    },

    /**
     * 处理减号点击
     */
    onReduce: function() {
      if (this.data.count <= 0) {
        return;
      }

      const count = this.data.count - 1;
      this.setData({ count });

      // 触发事件通知父组件
      this.triggerEvent('change', {
        count,
        type: 'reduce'
      });
    },

    /**
     * 清理工作
     */
    cleanup: function() {
      // 清除定时器等资源
      console.log('组件清理');
    }
  }
});
```

### 5.2 自定义组件开发

自定义组件是小程序模块化的核心。

**定义组件：**

```javascript
// components/stepper/stepper.js
Component({
  // 组件命名空间
  component: true,

  // 组件属性
  properties: {
    // 最小值
    min: {
      type: Number,
      value: 0
    },
    // 最大值
    max: {
      type: Number,
      value: 100
    },
    // 步进值
    step: {
      type: Number,
      value: 1
    },
    // 当前值
    value: {
      type: Number,
      value: 0,
      observer: 'onValueChange'
    },
    // 是否禁用
    disabled: {
      type: Boolean,
      value: false
    }
  },

  // 组件数据
  data: {
    inputValue: 0
  },

  // 组件方法
  methods: {
    /**
     * 值变化监听
     */
    onValueChange: function(newVal) {
      this.setData({
        inputValue: newVal
      });
    },

    /**
     * 点击减号
     */
    onMinus: function() {
      if (this.data.disabled) return;

      let newVal = this.data.value - this.data.step;
      if (newVal < this.data.min) {
        newVal = this.data.min;
      }

      this.triggerEvent('change', {
        value: newVal,
        type: 'minus'
      });
    },

    /**
     * 点击加号
     */
    onPlus: function() {
      if (this.data.disabled) return;

      let newVal = this.data.value + this.data.step;
      if (newVal > this.data.max) {
        newVal = this.data.max;
      }

      this.triggerEvent('change', {
        value: newVal,
        type: 'plus'
      });
    },

    /**
     * 输入框失焦
     */
    onBlur: function(e) {
      let newVal = parseInt(e.detail.value, 10);

      if (isNaN(newVal)) {
        newVal = this.data.min;
      }

      if (newVal < this.data.min) {
        newVal = this.data.min;
      }

      if (newVal > this.data.max) {
        newVal = this.data.max;
      }

      this.triggerEvent('change', {
        value: newVal,
        type: 'input'
      });
    }
```

### 7.1 setData优化

setData是小程序中频繁使用的操作，但不当使用会影响性能。

**优化策略：**

```javascript
// 避免频繁调用 setData
Page({
  data: { count: 0 },

  // 错误示例：每次输入都调用 setData
  onInput: function(e) {
    this.setData({ inputValue: e.detail.value });  // 高频调用
  },

  // 正确示例：使用防抖
  onInput: function(e) {
    clearTimeout(this.inputTimer);
    this.inputTimer = setTimeout(() => {
      this.setData({ inputValue: e.detail.value });
    }, 300);
  },

  // 正确示例：合并多次 setData
  updateMultipleTimes: function() {
    // 错误：多次调用
    // this.setData({ a: 1 });
    // this.setData({ b: 2 });
    // this.setData({ c: 3 });

    // 正确：合并为一次
    this.setData({
      a: 1,
      b: 2,
      c: 3
    });
  },

  // 只更新变化的数据（路径写法）
  updatePartialData: function() {
    const list = this.data.list;
    list[0].name = '新名称';

    // 正确：只更新第一个元素的 name
    this.setData({
      'list[0].name': '新名称'
    });
  },

  // 避免在 setData 中传输大量数据
  badExample: function() {
    // 错误：每次都传输完整列表
    this.setData({ list: this.data.list });
  },

  goodExample: function() {
    // 正确：只更新变化的项
    const changedItem = {};
    changedItem['list[' + index + ']'] = newValue;
    this.setData(changedItem);
  }
})
```

```javascript
// pages/todo/todo.js - Todo页面逻辑
Page({
  data: {
    // 输入框的值
    inputValue: '',

    // 待办事项列表
    todoList: [],

    // 筛选状态：all | active | completed
    filter: 'all',

    // 当前编辑的索引
    editingIndex: -1,

    // 编辑输入框的值
    editValue: '',

    // 统计数据
    stats: {
      total: 0,
      active: 0,
      completed: 0
    }
  },

  /**
   * 页面加载时初始化数据
   */
  onLoad: function() {
    this.loadTodos();
  },

  /**
   * 从本地存储加载待办事项
   */
  loadTodos: function() {
    const todos = wx.getStorageSync('todos') || [];
    this.setData({ todoList: todos });
    this.updateStats();
  },

  /**
   * 保存待办事项到本地存储
   */
  saveTodos: function() {
    wx.setStorageSync('todos', this.data.todoList);
    this.updateStats();
  },

  /**
   * 更新统计数据
   */
  updateStats: function() {
    const todoList = this.data.todoList;
    const total = todoList.length;
    const active = todoList.filter(item => !item.completed).length;
    const completed = total - active;

    this.setData({
      stats: { total, active, completed }
    });
  },

  /**
   * 添加新的待办事项
   */
  onAddTodo: function() {
    const value = this.data.inputValue.trim();

    if (!value) {
      wx.showToast({
        title: '请输入待办事项',
        icon: 'none'
      });
      return;
    }

    const newTodo = {
      id: Date.now(),                    // 使用时间戳作为唯一ID
      content: value,                    // 待办内容
      completed: false,                  // 是否完成
      createTime: new Date().toLocaleString(),  // 创建时间
      completedTime: null                // 完成时间
    };

    const todoList = [newTodo, ...this.data.todoList];

    this.setData({
      todoList,
      inputValue: ''
    });

    this.saveTodos();

    wx.showToast({
      title: '添加成功',
      icon: 'success',
      duration: 1000
    });
  },

  /**
   * 输入框内容变化
   */
  onInputChange: function(e) {
    this.setData({
      inputValue: e.detail.value
    });
  },

  /**
   * 输入框回车事件
   */
  onInputConfirm: function() {
    this.onAddTodo();
  },

  /**
   * 切换待办事项的完成状态
   */
  onToggleComplete: function(e) {
    const index = e.currentTarget.dataset.index;
    const todoList = this.data.todoList;

    todoList[index].completed = !todoList[index].completed;

    if (todoList[index].completed) {
      todoList[index].completedTime = new Date().toLocaleString();
    } else {
      todoList[index].completedTime = null;
    }

    this.setData({ todoList });
    this.saveTodos();
  },

  /**
   * 删除待办事项
   */
  onDeleteTodo: function(e) {
    const index = e.currentTarget.dataset.index;

    wx.showModal({
      title: '确认删除',
      content: '确定要删除这条待办事项吗？',
      success: (res) => {
        if (res.confirm) {
          const todoList = this.data.todoList;
          todoList.splice(index, 1);

          this.setData({ todoList });
          this.saveTodos();

          wx.showToast({
            title: '已删除',
            icon: 'success',
            duration: 1000
          });
        }
      }
    });
  },

  /**
   * 开始编辑待办事项
   */
  onStartEdit: function(e) {
    const index = e.currentTarget.dataset.index;
    const todo = this.data.todoList[index];

    this.setData({
      editingIndex: index,
      editValue: todo.content
    });
  },

  /**
   * 编辑输入框内容变化
   */
  onEditChange: function(e) {
    this.setData({
      editValue: e.detail.value
    });
  },

  /**
   * 完成编辑
   */
  onFinishEdit: function() {
    const index = this.data.editingIndex;
    const newContent = this.data.editValue.trim();

    if (!newContent) {
      wx.showToast({
        title: '内容不能为空',
        icon: 'none'
      });
      return;
    }

    const todoList = this.data.todoList;
    todoList[index].content = newContent;

    this.setData({
      todoList,
      editingIndex: -1,
      editValue: ''
    });

    this.saveTodos();
  },

  /**
   * 取消编辑
   */
  onCancelEdit: function() {
    this.setData({
      editingIndex: -1,
      editValue: ''
    });
  },

  /**
   * 切换筛选状态
   */
  onFilterChange: function(e) {
    const filter = e.currentTarget.dataset.filter;
    this.setData({ filter });
  },

  /**
   * 清除所有已完成的待办事项
   */
  onClearCompleted: function() {
    const completedTodos = this.data.todoList.filter(item => item.completed);

    if (completedTodos.length === 0) {
      wx.showToast({
        title: '没有已完成的项',
        icon: 'none'
      });
      return;
    }

    wx.showModal({
      title: '确认清除',
      content: `确定要清除 ${completedTodos.length} 条已完成的待办事项吗？`,
      success: (res) => {
        if (res.confirm) {
          const todoList = this.data.todoList.filter(item => !item.completed);

          this.setData({ todoList });
          this.saveTodos();

          wx.showToast({
            title: '已清除',
            icon: 'success',
            duration: 1000
          });
        }
      }
    });
  },

  /**
   * 全部标记为完成
   */
  onCompleteAll: function() {
    const todoList = this.data.todoList.map(item => ({
      ...item,
      completed: true,
      completedTime: item.completedTime || new Date().toLocaleString()
    }));

    this.setData({ todoList });
    this.saveTodos();
  },

  /**
   * 全部标记为未完成
   */
  onUncompleteAll: function() {
    const todoList = this.data.todoList.map(item => ({
      ...item,
      completed: false,
      completedTime: null
    }));

    this.setData({ todoList });
    this.saveTodos();
  },

  /**
   * 获取筛选后的待办列表
   */
  getFilteredList: function() {
    const { todoList, filter } = this.data;

    switch (filter) {
      case 'active':
        return todoList.filter(item => !item.completed);
      case 'completed':
        return todoList.filter(item => item.completed);
      default:
        return todoList;
    }
```

### 8.2 商城购物车小程序

完整的购物车功能小程序，包含商品选择、数量调整、价格计算、订单确认等功能。

```javascript
// pages/cart/cart.js - 购物车页面逻辑
Page({
  data: {
    // 购物车商品列表
    cartList: [],

    // 是否全选
    isAllSelected: false,

    // 总数量
    totalCount: 0,

    // 总价格
    totalPrice: 0,

    // 是否编辑模式
    isEditMode: false,

    // 选中的商品ID列表
    selectedIds: [],

    // 用户信息
    userInfo: null,

    // 地址信息
    address: null
  },

  /**
   * 页面加载
   */
  onLoad: function() {
    // 获取用户信息
    const userInfo = wx.getStorageSync('userInfo');
    this.setData({ userInfo });

    // 加载购物车数据
    this.loadCart();
  },

  /**
   * 页面显示时刷新数据
   */
  onShow: function() {
    // 如果需要刷新
    if (this.data.needRefresh) {
      this.loadCart();
      this.setData({ needRefresh: false });
    }

    // 刷新tabBar徽章
    this.updateTabBarBadge();
  },

  /**
   * 加载购物车数据
   */
  loadCart: function() {
    const cartList = wx.getStorageSync('cart') || [];

    // 计算总价和总数量
    this.calculateTotal(cartList);

    // 检查是否全选
    this.checkAllSelected(cartList);

    this.setData({ cartList });
  },

  /**
   * 计算总价和总数量
   */
  calculateTotal: function(cartList) {
    let totalCount = 0;
    let totalPrice = 0;

    cartList.forEach(item => {
      if (item.selected) {
        totalCount += item.quantity;
        totalPrice += item.price * item.quantity;
      }
    });

    // 保留两位小数
    totalPrice = Math.round(totalPrice * 100) / 100;

    this.setData({
      totalCount,
      totalPrice
    });
  },

  /**
   * 检查是否全选
   */
  checkAllSelected: function(cartList) {
    if (cartList.length === 0) {
      this.setData({ isAllSelected: false });
      return;
    }

    const isAllSelected = cartList.every(item => item.selected);
    this.setData({ isAllSelected });
  },

  /**
   * 更新tabBar徽章
   */
  updateTabBarBadge: function() {
    const cartList = wx.getStorageSync('cart') || [];
    const count = cartList.reduce((sum, item) => sum + item.quantity, 0);

    if (count > 0) {
      wx.setTabBarBadge({
        index: 2,
        text: count.toString()
      });
    } else {
      wx.removeTabBarBadge({ index: 2 });
    }
  },

  /**
   * 切换商品选中状态
   */
  onToggleSelect: function(e) {
    const index = e.currentTarget.dataset.index;
    const cartList = this.data.cartList;

    cartList[index].selected = !cartList[index].selected;

    this.setData({ cartList });
    this.calculateTotal(cartList);
    this.checkAllSelected(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);
  },

  /**
   * 切换全选状态
   */
  onToggleAllSelect: function() {
    const isAllSelected = !this.data.isAllSelected;
    const cartList = this.data.cartList.map(item => ({
      ...item,
      selected: isAllSelected
    }));

    this.setData({
      cartList,
      isAllSelected
    });

    this.calculateTotal(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);
  },

  /**
   * 增加数量
   */
  onIncrease: function(e) {
    const index = e.currentTarget.dataset.index;
    const cartList = this.data.cartList;

    // 检查库存
    if (cartList[index].quantity >= cartList[index].stock) {
      wx.showToast({
        title: '库存不足',
        icon: 'none'
      });
      return;
    }

    cartList[index].quantity += 1;

    this.setData({ cartList });
    this.calculateTotal(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);

    // 更新tabBar徽章
    this.updateTabBarBadge();
  },

  /**
   * 减少数量
   */
  onDecrease: function(e) {
    const index = e.currentTarget.dataset.index;
    const cartList = this.data.cartList;

    if (cartList[index].quantity <= 1) {
      // 提示是否删除
      wx.showModal({
        title: '确认删除',
        content: '确定要删除这件商品吗？',
        success: (res) => {
          if (res.confirm) {
            this.removeItem(index);
          }
        }
      });
      return;
    }

    cartList[index].quantity -= 1;

    this.setData({ cartList });
    this.calculateTotal(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);

    // 更新tabBar徽章
    this.updateTabBarBadge();
  },

  /**
   * 直接设置数量
   */
  onQuantityChange: function(e) {
    const index = e.currentTarget.dataset.index;
    const quantity = parseInt(e.detail.value, 10);
    const cartList = this.data.cartList;

    if (quantity < 1) {
      wx.showToast({
        title: '数量不能小于1',
        icon: 'none'
      });
      return;
    }

    if (quantity > cartList[index].stock) {
      wx.showToast({
        title: '库存不足',
        icon: 'none'
      });
      return;
    }

    cartList[index].quantity = quantity;

    this.setData({ cartList });
    this.calculateTotal(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);
  },

  /**
   * 删除单个商品
   */
  onDeleteItem: function(e) {
    const index = e.currentTarget.dataset.index;

    wx.showModal({
      title: '确认删除',
      content: '确定要删除这件商品吗？',
      success: (res) => {
        if (res.confirm) {
          this.removeItem(index);
        }
      }
    });
  },

  /**
   * 移除商品
   */
  removeItem: function(index) {
    const cartList = this.data.cartList;
    cartList.splice(index, 1);

    this.setData({ cartList });
    this.calculateTotal(cartList);
    this.checkAllSelected(cartList);

    // 保存到本地
    wx.setStorageSync('cart', cartList);

    // 更新tabBar徽章
    this.updateTabBarBadge();

    wx.showToast({
      title: '已删除',
      icon: 'success',
      duration: 1000
    });
  },

  /**
   * 清空已选商品
   */
  onClearSelected: function() {
    const selectedItems = this.data.cartList.filter(item => item.selected);

    if (selectedItems.length === 0) {
      wx.showToast({
        title: '请选择要删除的商品',
        icon: 'none'
      });
      return;
    }

    wx.showModal({
      title: '确认清空',
      content: `确定要清空 ${selectedItems.length} 件已选商品吗？`,
      success: (res) => {
        if (res.confirm) {
          const cartList = this.data.cartList.filter(item => !item.selected);

          this.setData({ cartList });
          this.calculateTotal(cartList);
          this.checkAllSelected(cartList);

          // 保存到本地
          wx.setStorageSync('cart', cartList);

          // 更新tabBar徽章
          this.updateTabBarBadge();

          wx.showToast({
            title: '已清空',
            icon: 'success',
            duration: 1000
          });
        }
      }
    });
  },

  /**
   * 切换编辑模式
   */
  onToggleEditMode: function() {
    this.setData({
      isEditMode: !this.data.isEditMode
    });
  },

  /**
   * 跳转到商品详情
   */
  onGoToDetail: function(e) {
    const goodsId = e.currentTarget.dataset.id;
    wx.navigateTo({
      url: '/pages/goods/detail?id=' + goodsId
    });
  },

  /**
   * 跳转到结算页面
   */
  onCheckout: function() {
    const selectedItems = this.data.cartList.filter(item => item.selected);

    if (selectedItems.length === 0) {
      wx.showToast({
        title: '请选择要结算的商品',
        icon: 'none'
      });
      return;
    }

    // 计算订单信息
    const orderInfo = {
      goods: selectedItems,
      totalCount: this.data.totalCount,
      totalPrice: this.data.totalPrice,
      freight: this.data.totalPrice >= 99 ? 0 : 10,  // 满99免运费
      finalPrice: this.data.totalPrice + (this.data.totalPrice >= 99 ? 0 : 10)
    };

    // 存储订单信息
    wx.setStorageSync('orderInfo', orderInfo);

    // 跳转到确认订单页面
    wx.navigateTo({
      url: '/pages/order/confirm'
    });
  },

  /**
   * 跳转购物车
   */
  onGoShopping: function() {
    wx.switchTab({
      url: '/pages/index/index'
    });
  },

  /**
   * 获取选中的商品列表
   */
  getSelectedItems: function() {
    return this.data.cartList.filter(item => item.selected);
  }
});
```
