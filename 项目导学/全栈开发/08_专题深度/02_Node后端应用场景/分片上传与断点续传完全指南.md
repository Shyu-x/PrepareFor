# 分片上传与断点续传完全指南

## 概述

在现代Web应用中，大文件上传是一个常见且具有挑战性的需求。传统的单文件上传方式在处理大文件时存在诸多问题：上传时间长、失败率高、无法续传、占用内存大等。分片上传与断点续传技术应运而生，成为了解决大文件上传问题的标准方案。

本指南将深入讲解分片上传与断点续传的完整技术体系，涵盖前端实现、后端架构、MD5秒传、断点续传、并发控制、分布式存储等多个维度，并提供基于NestJS和React的完整实战代码。通过本指南的学习，你将掌握构建企业级大文件上传系统的全部核心技能。

---

## 一、分片上传原理

### 1.1 为什么要分片

传统的文件上传方式将整个文件作为一个请求发送到服务器，这种方式在处理大文件时存在以下严重问题：

**第一，服务器内存压力巨大。** 当用户上传一个10GB的视频文件时，服务器需要将整个文件加载到内存中进行处理。对于并发上传的场景，服务器内存很快就会被耗尽，导致服务崩溃。

**第二，网络中断导致前功尽弃。** 在上传大文件时，网络波动是不可避免的。一旦网络断开，整个上传过程就需要从头开始。用户可能需要等待数小时才能完成的上传，因为一次短暂的网络抖动而全部重来。这种体验对于用户来说是极其糟糕的。

**第三，超时问题难以解决。** 大文件上传需要较长的传输时间，而HTTP请求有超时限制。即使将超时时间设置得很长，在高并发场景下，服务器和客户端的性能都会受到严重影响。

**第四，无法利用并行带宽。** 单一请求只能使用单一连接，而实际上用户的网络带宽可能是100Mbps甚至更高，但没有得到充分利用。

**第五，无法实现增量更新。** 如果文件只是修改了一小部分，也需要重新上传整个文件，浪费带宽和时间。

分片上传通过将大文件分割成多个小块（称为"分片"或"Chunk"），每个分片独立上传，彻底解决了上述问题。每个分片较小，内存压力可控；网络中断时只需重传失败的单个分片；可以充分利用网络带宽实现并行上传。

### 1.2 分片大小选择

分片大小的选择是一个需要权衡多个因素的决策过程。常见的分片大小有5MB和10MB两种选择，但实际项目中需要根据具体场景进行调整。

**5MB分片的优势在于** ：首先，5MB是大多数云存储服务的最小分片要求，如阿里云OSS、AWS S3等都要求分片不小于5MB；其次，5MB在网络超时处理时较为合适，单个分片上传失败后重传的开销可控；再次，5MB分片在内存中处理时占用较小，适合在移动设备上使用；最后，5MB的分片大小在断点续传时能够提供较好的粒度。

**10MB分片的优势在于** ：首先，10MB分片能够减少HTTP请求的数量，降低服务器处理开销；其次，在高带宽环境下，10MB分片能够更好地利用网络吞吐量；再次，10MB分片减少了分片数量，简化了合并逻辑和状态管理；最后，对于内部网络或高带宽低延迟环境，10MB是更好的选择。

**实际项目中的分片大小策略** 应该是动态的：小文件（小于20MB）使用单一分片，即整个文件作为一个分片上传；中等文件（20MB-100MB）使用5MB分片；大文件（100MB以上）使用10MB或更大的分片；同时需要考虑用户网络环境，在移动网络下适当减小分片大小，在高带宽环境下可以使用更大的分片。

### 1.3 并发上传原理

分片上传的核心优势之一就是能够实现并行上传，充分利用用户的网络带宽。并发上传的基本原理是同时发起多个分片的上传请求，每个请求独立传输一个分片，当所有分片都上传完成后，在服务器端合并这些分片。

假设一个500MB的文件使用10MB分片上传，如果使用串行上传，每个分片需要依次上传完成，总时间是所有分片时间之和。但如果使用5个并发连接同时上传，分片1、分片2、分片3、分片4、分片5同时上传，然后分片6-10同时上传，总时间约为串行上传的五分之一，效率提升显著。

并发上传需要解决的关键问题包括：分片顺序管理，虽然分片可以并行上传，但合并时必须按照正确顺序；进度计算，需要汇总所有分片的上传进度来计算总体进度；冲突处理，多个分片同时上传时，服务器需要正确处理这些并发请求；失败重试，单个分片失败不应该影响其他分片的进度。

## 二、前端分片实现

### 2.1 Blob.slice切割原理

在JavaScript中，File对象继承自Blob对象，提供了slice()方法来切割文件。Blob.slice(start, end, contentType)方法返回一个新的Blob对象，包含原文件中从start到end（不包含end）的字节数据。

```javascript
// Blob.slice() 方法签名
Blob.slice(start: number, end?: number, contentType?: string): Blob

// 示例：切割文件的前10MB
const file = document.querySelector('input[type="file"]').files[0];
const chunk = file.slice(0, 10 * 1024 * 1024); // 前10MB
console.log(chunk.size); // 输出: 10485760 (10MB字节数)
```

需要特别注意的是，不同浏览器对Blob.slice()方法的支持和命名有所不同。在较老版本的浏览器中，方法名可能是webkitSlice()或mozSlice()。为了兼容性，实际项目中通常需要做一层封装：

```javascript
/**
 * 跨浏览器兼容的文件切片方法
 * @param {File|Blob} file - 要切片的文件对象
 * @param {number} start - 起始字节位置
 * @param {number} end - 结束字节位置
 * @returns {Blob} 切片后的Blob对象
 */
function sliceFile(file, start, end) {
  // 现代浏览器
  if (file.slice) {
    return file.slice(start, end);
  }
  // 老版Firefox
  if (file.mozSlice) {
    return file.mozSlice(start, end);
  }
  // 老版Chrome/Opera
  if (file.webkitSlice) {
    return file.webkitSlice(start, end);
  }
  throw new Error('当前浏览器不支持文件切片');
}
```

### 2.2 File对象处理

File对象是Blob的子类，除了包含文件内容外，还提供了文件的元数据信息，包括文件名、大小、MIME类型、最后修改时间等。这些信息在分片上传时都需要用到。

```javascript
/**
 * 文件信息提取工具
 * @param {File} file - File对象
 * @returns {Object} 包含文件元信息的对象
 */
function getFileInfo(file) {
  return {
    name: file.name,           // 文件名
    size: file.size,           // 文件大小（字节）
    type: file.type,           // MIME类型
    lastModified: file.lastModified,     // 最后修改时间戳
    lastModifiedDate: file.lastModifiedDate, // 最后修改时间（Date对象）
    // 计算可读的文件大小
    sizeReadable: formatFileSize(file.size)
  };
}

/**
 * 格式化文件大小为可读字符串
 * @param {number} bytes - 字节数
 * @returns {string} 格式化后的大小字符串
 */
function formatFileSize(bytes) {
  if (bytes === 0) return '0 B';
  const k = 1024;
  const sizes = ['B', 'KB', 'MB', 'GB', 'TB'];
  const i = Math.floor(Math.log(bytes) / Math.log(k));
  return parseFloat((bytes / Math.pow(k, i)).toFixed(2)) + ' ' + sizes[i];
}

// 使用示例
const input = document.querySelector('input[type="file"]');
input.addEventListener('change', (e) => {
  const file = e.target.files[0];
  if (file) {
    const info = getFileInfo(file);
    console.log('文件信息:', info);
    // 输出: { name: "video.mp4", size: 524288000, type: "video/mp4",
    //         lastModified: 1704067200000, sizeReadable: "500 MB" }
  }
});
```

### 2.3 分片切割实现

将一个大文件切分成多个分片是前端分片上传的核心逻辑。我们需要创建一个函数，能够根据分片大小将文件切割成多个Blob对象，并计算每个分片的唯一标识符。

```javascript
/**
 * 分片切割器类
 * 用于将大文件切分成多个分片，并管理分片状态
 */
class FileChunker {
  constructor(file, chunkSize = 5 * 1024 * 1024) {
    this.file = file;                    // 原始File对象
    this.chunkSize = chunkSize;          // 分片大小（默认5MB）
    this.totalChunks = Math.ceil(file.size / chunkSize); // 总分片数
    this.chunks = [];                    // 分片列表
  }

  /**
   * 获取所有分片的信息
   * @returns {Array} 分片信息数组
   */
  getChunksInfo() {
    const chunks = [];
    let start = 0;
    let index = 0;

    while (start < this.file.size) {
      const end = Math.min(start + this.chunkSize, this.file.size);
      const chunk = this.file.slice(start, end);

      chunks.push({
        index: index,                    // 分片索引（0-based）
        hash: this.generateChunkHash(index), // 分片唯一标识
        start: start,                    // 起始字节
        end: end,                       // 结束字节
        size: chunk.size,               // 分片大小
        blob: chunk,                    // 分片Blob对象
        loaded: false,                   // 是否已上传
        xhr: null                        // XMLHttpRequest对象
      });

      start = end;
      index++;
    }

    this.chunks = chunks;
    return chunks;
  }

  /**
   * 生成分片唯一哈希值
   * 基于文件标识 + 分片索引，确保分片在整个文件范围内唯一
   * @param {number} index - 分片索引
   * @returns {string} 分片哈希值
   */
  generateChunkHash(index) {
    // 使用简单的索引+时间戳生成，实际项目中应使用更复杂的方法
    const fileIdentifier = `${this.file.name}_${this.file.size}_${this.file.lastModified}`;
    return `${this.simpleHash(fileIdentifier)}_${index}`;
  }

  /**
   * 简单的哈希函数
   * @param {string} str - 输入字符串
   * @returns {string} 哈希值
   */
  simpleHash(str) {
    let hash = 0;
    for (let i = 0; i < str.length; i++) {
      const char = str.charCodeAt(i);
      hash = ((hash << 5) - hash) + char;
      hash = hash & hash; // Convert to 32bit integer
    }
    return Math.abs(hash).toString(16);
  }

  /**
   * 获取已上传的分片索引列表
   * @param {string} fileHash - 文件哈希
   * @returns {Promise<Array>} 已上传的分片索引数组
   */
  async getUploadedChunks(fileHash) {
    try {
      const response = await fetch(`/api/upload/chunk-status/${fileHash}`);
      const data = await response.json();
      return data.uploadedChunks || [];
    } catch (error) {
      console.error('获取已上传分片失败:', error);
      return [];
    }
  }

  /**
   * 过滤出未上传的分片
   * @param {Array} uploadedIndices - 已上传的分片索引数组
   * @returns {Array} 未上传的分片数组
   */
  filterUnuploadedChunks(uploadedIndices) {
    const uploadedSet = new Set(uploadedIndices);
    return this.chunks.filter(chunk => !uploadedSet.has(chunk.index));
  }
}

// 使用示例
const chunker = new FileChunker(file, 5 * 1024 * 1024); // 5MB分片
const chunksInfo = chunker.getChunksInfo();
console.log(`文件将被切分成 ${chunksInfo.length} 个分片`);
chunksInfo.forEach(chunk => {
  console.log(`分片${chunk.index}: ${chunk.start}-${chunk.end} (${chunk.size} bytes)`);
});
```

### 2.4 进度计算

在分片上传过程中，准确计算和展示上传进度是用户体验的关键。进度计算需要考虑所有分片的状态，并能够实时更新。

```javascript
/**
 * 上传进度管理器
 * 负责计算整体进度、管理分片状态、处理并发上传
 */
class UploadProgressManager {
  constructor(totalChunks) {
    this.totalChunks = totalChunks;          // 总分片数
    this.uploadedChunks = new Map();        // 已上传的分片 Map<index, chunkInfo>
    this.failedChunks = [];                 // 失败的分组
    this.totalBytes = 0;                    // 文件总字节数
    this.uploadedBytes = 0;                  // 已上传字节数
    this.speed = 0;                          // 上传速度（字节/秒）
    this.startTime = 0;                      // 开始时间
    this.speedSamples = [];                  // 速度采样数据
  }

  /**
   * 初始化进度管理器
   * @param {number} totalBytes - 文件总大小
   */
  init(totalBytes) {
    this.totalBytes = totalBytes;
    this.uploadedBytes = 0;
    this.startTime = Date.now();
    this.speedSamples = [];
  }

  /**
   * 更新单个分片的上传进度
   * @param {number} index - 分片索引
   * @param {number} loaded - 已加载的字节数
   * @param {number} chunkSize - 分片大小
   */
  updateChunkProgress(index, loaded, chunkSize) {
    // 计算这个分片的已上传字节数
    const chunkUploadedBytes = Math.min(loaded, chunkSize);

    // 如果这个分片之前没有记录，则增加总已上传字节数
    if (!this.uploadedChunks.has(index)) {
      this.uploadedChunks.set(index, {
        loaded: chunkUploadedBytes,
        size: chunkSize
      });
      this.uploadedBytes += chunkUploadedBytes;
    } else {
      // 更新已上传字节数
      const prev = this.uploadedChunks.get(index);
      const diff = chunkUploadedBytes - prev.loaded;
      this.uploadedBytes += diff;
      prev.loaded = chunkUploadedBytes;
    }

    // 更新速度采样
    this.updateSpeed(chunkUploadedBytes);
  }

  /**
   * 标记分片上传完成
   * @param {number} index - 分片索引
   * @param {number} chunkSize - 分片大小
   */
  markChunkComplete(index, chunkSize) {
    if (!this.uploadedChunks.has(index)) {
      this.uploadedChunks.set(index, {
        loaded: chunkSize,
        size: chunkSize
      });
      this.uploadedBytes += chunkSize;
    }
  }

  /**
   * 标记分片上传失败
   * @param {number} index - 分片索引
   */
  markChunkFailed(index) {
    if (!this.failedChunks.includes(index)) {
      this.failedChunks.push(index);
    }
  }

  /**
   * 更新上传速度
   * @param {number} bytes - 本次上传的字节数
   */
  updateSpeed(bytes) {
    const now = Date.now();
    const elapsed = (now - this.startTime) / 1000; // 转换为秒

    if (elapsed > 0) {
      const currentSpeed = this.uploadedBytes / elapsed;
      // 保留最近5个采样点用于平滑计算
      this.speedSamples.push(currentSpeed);
      if (this.speedSamples.length > 5) {
        this.speedSamples.shift();
      }
      // 计算平均速度
      this.speed = this.speedSamples.reduce((a, b) => a + b, 0) / this.speedSamples.length;
    }
  }

  /**
   * 获取当前上传进度百分比
   * @returns {number} 进度百分比（0-100）
   */
  getProgress() {
    if (this.totalBytes === 0) return 0;
    return Math.round((this.uploadedBytes / this.totalBytes) * 100 * 100) / 100;
  }

  /**
   * 获取预计剩余时间（秒）
   * @returns {number} 剩余秒数
   */
  getRemainingTime() {
    if (this.speed === 0) return 0;
    const remainingBytes = this.totalBytes - this.uploadedBytes;
    return Math.ceil(remainingBytes / this.speed);
  }

  /**
   * 获取格式化的时间字符串
   * @param {number} seconds - 秒数
   * @returns {string} 格式化的时间字符串
   */
  formatTime(seconds) {
    if (seconds === 0) return '--';
    const hours = Math.floor(seconds / 3600);
    const minutes = Math.floor((seconds % 3600) / 60);
    const secs = seconds % 60;

    if (hours > 0) {
      return `${hours}小时${minutes}分钟`;
    } else if (minutes > 0) {
      return `${minutes}分钟${secs}秒`;
    } else {
      return `${secs}秒`;
    }
  }

  /**
   * 获取完整的进度信息
   * @returns {Object} 进度信息对象
   */
  getStatus() {
    return {
      progress: this.getProgress(),                    // 进度百分比
      uploadedBytes: this.uploadedBytes,               // 已上传字节数
      totalBytes: this.totalBytes,                      // 总字节数
      uploadedChunks: this.uploadedChunks.size,        // 已上传分片数
      totalChunks: this.totalChunks,                   // 总分片数
      failedChunks: this.failedChunks.length,           // 失败分片数
      speed: this.speed,                                // 速度（字节/秒）
      speedReadable: this.formatSpeed(this.speed),      // 格式化速度
      remainingTime: this.getRemainingTime(),           // 剩余时间（秒）
      remainingTimeReadable: this.formatTime(this.getRemainingTime()), // 格式化剩余时间
      isComplete: this.uploadedChunks.size === this.totalChunks, // 是否完成
      hasFailed: this.failedChunks.length > 0           // 是否有失败
    };
  }

  /**
   * 格式化速度
   * @param {number} bytesPerSecond - 字节/秒
   * @returns {string} 格式化速度字符串
   */
  formatSpeed(bytesPerSecond) {
    if (bytesPerSecond === 0) return '0 KB/s';
    const kb = bytesPerSecond / 1024;
    const mb = kb / 1024;

    if (mb >= 1) {
      return `${mb.toFixed(2)} MB/s`;
    } else {
      return `${kb.toFixed(2)} KB/s`;
    }
  }
}
```

### 2.5 完整前端实现

以下是一个完整的分片上传前端实现，集成了分片切割、并发上传、进度计算、断点续传等功能：

```javascript
/**
 * 分片上传管理器
 * 完整的分片上传解决方案，支持并发上传、进度追踪、断点续传
 */
class ChunkedUploader {
  constructor(options = {}) {
    // 配置参数
    this.chunkSize = options.chunkSize || 5 * 1024 * 1024;  // 默认5MB
    this.concurrentUploads = options.concurrentUploads || 3; // 默认3个并发
    this.maxRetries = options.maxRetries || 3;               // 默认重试次数
    this.retryDelay = options.retryDelay || 1000;            // 重试延迟（毫秒）
    this.uploadUrl = options.uploadUrl || '/api/upload/chunk';
    this.mergeUrl = options.mergeUrl || '/api/upload/merge';
    this.statusUrl = options.statusUrl || '/api/upload/status';

    // 状态变量
    this.file = null;
    this.fileHash = '';
    this.chunks = [];
    this.uploadTasks = [];
    this.progressManager = null;
    this.isPaused = false;
    this.isCancelled = false;
    this.abortControllers = new Map();

    // 回调函数
    this.onProgress = options.onProgress || (() => {});
    this.onComplete = options.onComplete || (() => {});
    this.onError = options.onError || (() => {});
    this.onPause = options.onPause || (() => {});
    this.onResume = options.onResume || (() => {});
  }

  /**
   * 初始化上传
   * @param {File} file - 要上传的文件
   * @returns {Promise<string>} 文件哈希
   */
  async init(file) {
    this.file = file;

    // 计算文件MD5哈希
    this.fileHash = await this.calculateFileHash(file);
    console.log(`文件哈希: ${this.fileHash}`);

    // 创建分片
    const chunker = new FileChunker(file, this.chunkSize);
    this.chunks = chunker.getChunksInfo();

    // 初始化进度管理器
    this.progressManager = new UploadProgressManager(this.chunks.length);
    this.progressManager.init(file.size);

    // 查询已上传的分片（断点续传）
    const uploadedChunks = await this.queryUploadedChunks();
    console.log(`已上传分片: ${uploadedChunks.join(', ')}`);

    // 过滤未上传的分片
    this.pendingChunks = this.chunks.filter(
      chunk => !uploadedChunks.includes(chunk.index)
    );

    return this.fileHash;
  }

  /**
   * 计算文件MD5哈希
   * 使用Web Workers避免阻塞主线程
   * @param {File} file - 文件对象
   * @returns {Promise<string>} MD5哈希值
   */
  async calculateFileHash(file) {
    return new Promise((resolve, reject) => {
      // 尝试使用Web Worker
      if (typeof Worker !== 'undefined') {
        this.calculateHashWithWorker(file, resolve, reject);
      } else {
        // 回退到主线程计算
        this.calculateHashMainThread(file, resolve, reject);
      }
    });
  }

  /**
   * 使用Web Worker计算哈希
   */
  calculateHashWithWorker(file, resolve, reject) {
    const worker = new Worker(URL.createObjectURL(new Blob([`
      importScripts('https://cdn.jsdelivr.net/npm/spark-md5@3.0.2/spark-md5.min.js');

      self.onmessage = function(e) {
        const file = e.data.file;
        const chunkSize = e.data.chunkSize;
        const chunks = Math.ceil(file.size / chunkSize);
        const spark = new SparkMD5.ArrayBuffer();
        const reader = new FileReaderSync();

        let cursor = 0;
        for (let i = 0; i < chunks; i++) {
          const chunk = reader.readAsArrayBuffer(file.slice(cursor, cursor + chunkSize));
          spark.append(chunk);
          cursor += chunkSize;

          // 发送进度
          self.postMessage({ type: 'progress', percent: Math.round((i / chunks) * 100) });
        }

        const hash = spark.end();
        self.postMessage({ type: 'complete', hash: hash });
      };
    `])));

    worker.onmessage = (e) => {
      if (e.data.type === 'complete') {
        resolve(e.data.hash);
        worker.terminate();
      } else if (e.data.type === 'progress') {
        console.log(`MD5计算进度: ${e.data.percent}%`);
      }
    };

    worker.onerror = (error) => {
      reject(error);
      worker.terminate();
    };

    worker.postMessage({ file: file, chunkSize: 2 * 1024 * 1024 });
  }

  /**
   * 在主线程计算哈希（不推荐用于大文件）
   */
  async calculateHashMainThread(file, resolve, reject) {
    // 动态加载spark-md5库
    const SparkMD5 = await import('https://cdn.jsdelivr.net/npm/spark-md5@3.0.2/spark-md5.min.js');

    const chunkSize = 2 * 1024 * 1024; // 2MB块
    const chunks = Math.ceil(file.size / chunkSize);
    const spark = new SparkMD5.default.ArrayBuffer();

    let cursor = 0;
    for (let i = 0; i < chunks; i++) {
      const chunk = file.slice(cursor, cursor + chunkSize);
      const arrayBuffer = await chunk.arrayBuffer();
      spark.append(arrayBuffer);
      cursor += chunkSize;

      // 报告进度
      this.onProgress({
        type: 'hash',
        percent: Math.round((i / chunks) * 100)
      });
    }

    resolve(spark.end());
  }

  /**
   * 查询已上传的分片（用于断点续传）
   * @returns {Promise<Array>} 已上传的分片索引数组
   */
  async queryUploadedChunks() {
    try {
      const response = await fetch(`${this.statusUrl}/${this.fileHash}`);
      if (!response.ok) {
        throw new Error('查询上传状态失败');
      }
      const data = await response.json();
      return data.uploadedChunks || [];
    } catch (error) {
      console.warn('断点续传查询失败，将从头开始上传:', error);
      return [];
    }
  }

  /**
   * 开始上传
   */
  async start() {
    if (!this.file) {
      throw new Error('请先调用init()方法');
    }

    this.isPaused = false;
    this.isCancelled = false;
    this.uploadTasks = [];

    console.log(`开始上传，共${this.pendingChunks.length}个分片待上传`);

    // 创建并发控制
    const concurrentLimiter = new ConcurrentLimiter(this.concurrentUploads);

    // 并发上传所有待上传的分片
    const uploadPromises = this.pendingChunks.map(chunk =>
      concurrentLimiter.run(() => this.uploadChunk(chunk))
    );

    try {
      await Promise.all(uploadPromises);

      // 所有分片上传完成，触发合并
      if (!this.isPaused && !this.isCancelled) {
        await this.mergeChunks();
        this.onComplete({
          fileHash: this.fileHash,
          fileName: this.file.name,
          fileSize: this.file.size
        });
      }
    } catch (error) {
      if (!this.isPaused) {
        this.onError(error);
      }
    }
  }

  /**
   * 上传单个分片
   * @param {Object} chunk - 分片信息
   * @returns {Promise<void>}
   */
  async uploadChunk(chunk) {
    const { index, blob, hash } = chunk;
    let retries = 0;

    while (retries <= this.maxRetries) {
      if (this.isCancelled) {
        throw new Error('上传已取消');
      }

      if (this.isPaused) {
        throw new Error('上传已暂停');
      }

      try {
        const formData = new FormData();
        formData.append('chunk', blob);
        formData.append('index', index.toString());
        formData.append('hash', hash);
        formData.append('fileHash', this.fileHash);
        formData.append('fileName', this.file.name);
        formData.append('totalChunks', this.chunks.length.toString());

        const result = await this.sendChunkRequest(formData, chunk);

        // 标记分片完成
        this.progressManager.markChunkComplete(index, blob.size);

        // 更新进度
        this.reportProgress();

        console.log(`分片 ${index} 上传完成`);
        return result;

      } catch (error) {
        retries++;
        console.error(`分片 ${index} 上传失败 (${retries}/${this.maxRetries}):`, error);

        if (retries <= this.maxRetries) {
          // 等待后重试
          await this.delay(this.retryDelay * retries);
        } else {
          // 标记失败
          this.progressManager.markChunkFailed(index);
          throw error;
        }
      }
    }
  }

  /**
   * 发送分片上传请求
   * @param {FormData} formData - 表单数据
   * @param {Object} chunk - 分片信息
   * @returns {Promise<Object>} 响应结果
   */
  sendChunkRequest(formData, chunk) {
    return new Promise((resolve, reject) => {
      const xhr = new XMLHttpRequest();
      const controller = new AbortController();

      // 保存控制器以便暂停/取消
      this.abortControllers.set(chunk.index, controller);

      xhr.upload.onprogress = (e) => {
        if (e.lengthComputable) {
          this.progressManager.updateChunkProgress(chunk.index, e.loaded, chunk.blob.size);
          this.reportProgress();
        }
      };

      xhr.onload = () => {
        if (xhr.status >= 200 && xhr.status < 300) {
          try {
            const response = JSON.parse(xhr.responseText);
            resolve(response);
          } catch {
            resolve({ success: true });
          }
        } else {
          reject(new Error(`HTTP ${xhr.status}: ${xhr.statusText}`));
        }
      };

      xhr.onerror = () => reject(new Error('网络错误'));
      xhr.onabort = () => reject(new Error('请求已取消'));

      xhr.open('POST', this.uploadUrl);
      xhr.setRequestHeader('X-File-Hash', this.fileHash);
      xhr.send(formData);
    });
  }

  /**
   * 报告进度
   */
  reportProgress() {
    const status = this.progressManager.getStatus();
    this.onProgress({
      type: 'upload',
      ...status
    });
  }

  /**
   * 合并分片
   * @returns {Promise<Object>} 合并结果
   */
  async mergeChunks() {
    console.log('开始合并分片...');

    const response = await fetch(this.mergeUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        fileHash: this.fileHash,
        fileName: this.file.name,
        fileSize: this.file.size,
        totalChunks: this.chunks.length
      })
    });

    if (!response.ok) {
      throw new Error('分片合并失败');
    }

    return await response.json();
  }

  /**
   * 暂停上传
   */
  pause() {
    this.isPaused = true;
    console.log('上传已暂停');

    // 中止所有进行中的请求
    this.abortControllers.forEach((controller) => {
      controller.abort();
    });
    this.abortControllers.clear();

    this.onPause(this.progressManager.getStatus());
  }

  /**
   * 恢复上传
   */
  async resume() {
    if (!this.isPaused) return;

    console.log('恢复上传...');
    this.isPaused = false;

    // 重新查询已上传的分片
    const uploadedChunks = await this.queryUploadedChunks();
    this.pendingChunks = this.chunks.filter(
      chunk => !uploadedChunks.includes(chunk.index)
    );

    this.onResume();
    await this.start();
  }

  /**
   * 取消上传
   */
  cancel() {
    this.isCancelled = true;
    this.isPaused = false;

    // 中止所有请求
    this.abortControllers.forEach((controller) => {
      controller.abort();
    });
    this.abortControllers.clear();

    console.log('上传已取消');
  }

  /**
   * 延迟函数
   * @param {number} ms - 毫秒数
   * @returns {Promise<void>}
   */
  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

/**
 * 并发控制器
 * 限制同时进行的任务数量
 */
class ConcurrentLimiter {
  constructor(maxConcurrent) {
    this.maxConcurrent = maxConcurrent;
    this.running = 0;
    this.queue = [];
  }

  /**
   * 添加任务
   * @param {Function} task - 异步任务函数
   * @returns {Promise}
   */
  async run(task) {
    return new Promise((resolve, reject) => {
      this.queue.push({ task, resolve, reject });
      this.process();
    });
  }

  /**
   * 处理队列
   */
  async process() {
    while (this.running < this.maxConcurrent && this.queue.length > 0) {
      const { task, resolve, reject } = this.queue.shift();
      this.running++;

      task()
        .then(resolve)
        .catch(reject)
        .finally(() => {
          this.running--;
          this.process();
        });
    }
  }
}
```

### 3.2 分片存储管理

分片存储是分片上传系统中的核心模块，需要管理分片的接收、存储、查询和清理等操作。

```typescript
// src/upload/upload.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { existsSync, mkdirSync, readdirSync, unlinkSync, createReadStream, createWriteStream, statSync } from 'fs';
import { join } from 'path';
import { createHash } from 'crypto';

@Injectable()
export class UploadService {
  private readonly logger = new Logger(UploadService.name);
  private readonly tempDir = './uploads/temp';
  private readonly finalDir = './uploads/files';

  constructor() {
    // 确保目录存在
    [this.tempDir, this.finalDir].forEach(dir => {
      if (!existsSync(dir)) {
        mkdirSync(dir, { recursive: true });
      }
    });
  }

  /**
   * 保存分片
   * @param file - 分片文件
   * @param fileHash - 文件哈希
   * @param chunkIndex - 分片索引
   * @param totalChunks - 总分片数
   */
  async saveChunk(file: Express.Multer.File, fileHash: string, chunkIndex: number, totalChunks: number): Promise<{
    uploadedChunks: number[];
    isComplete: boolean;
    progress: number;
  }> {
    const uploadDir = join(this.tempDir, fileHash);

    // 确保上传目录存在
    if (!existsSync(uploadDir)) {
      mkdirSync(uploadDir, { recursive: true });
    }

    // 记录已上传的分片信息到文件
    const statusFile = join(uploadDir, 'status.json');
    let status = this.loadChunkStatus(statusFile);

    // 更新状态
    status.chunks[chunkIndex] = {
      size: file.size,
      uploadedAt: new Date().toISOString()
    };

    // 保存状态
    this.saveChunkStatus(statusFile, status);

    // 获取已上传的分片列表
    const uploadedChunks = Object.keys(status.chunks).map(Number).sort((a, b) => a - b);
    const isComplete = uploadedChunks.length === totalChunks;
    const progress = Math.round((uploadedChunks.length / totalChunks) * 100);

    this.logger.log(`分片 ${chunkIndex} 已保存, 进度: ${progress}%, 已上传: ${uploadedChunks.length}/${totalChunks}`);

    return {
      uploadedChunks,
      isComplete,
      progress
    };
  }

  /**
   * 加载分片状态
   */
  private loadChunkStatus(statusFile: string): {
    chunks: Record<number, { size: number; uploadedAt: string }>
  } {
    try {
      if (existsSync(statusFile)) {
        const fs = require('fs');
        const data = fs.readFileSync(statusFile, 'utf-8');
        return JSON.parse(data);
      }
    } catch (error) {
      this.logger.error('加载分片状态失败:', error);
    }
    return { chunks: {} };
  }

  /**
   * 保存分片状态
   */
  private saveChunkStatus(statusFile: string, status: any): void {
    const fs = require('fs');
    fs.writeFileSync(statusFile, JSON.stringify(status, null, 2));
  }

  /**
   * 获取文件上传状态
   */
  async getUploadStatus(fileHash: string): Promise<{
    uploadedChunks: number[];
    totalChunks: number;
    isComplete: boolean;
    fileSize: number;
  }> {
    const uploadDir = join(this.tempDir, fileHash);
    const statusFile = join(uploadDir, 'status.json');

    if (!existsSync(statusFile)) {
      return {
        uploadedChunks: [],
        totalChunks: 0,
        isComplete: false,
        fileSize: 0
      };
    }

    const status = this.loadChunkStatus(statusFile);
    const uploadedChunks = Object.keys(status.chunks).map(Number).sort((a, b) => a - b);
    const totalChunks = status.totalChunks || 0;
    const fileSize = status.fileSize || 0;

    return {
      uploadedChunks,
      totalChunks,
      isComplete: uploadedChunks.length === totalChunks && totalChunks > 0,
      fileSize
    };
  }

  /**
   * 设置文件总大小
   */
  async setFileSize(fileHash: string, totalChunks: number, fileSize: number): Promise<void> {
    const uploadDir = join(this.tempDir, fileHash);
    const statusFile = join(uploadDir, 'status.json');

    let status = this.loadChunkStatus(statusFile);
    status.totalChunks = totalChunks;
    status.fileSize = fileSize;
    status.createdAt = new Date().toISOString();

    this.saveChunkStatus(statusFile, status);
  }

  /**
   * 合并分片
   */
  async mergeChunks(fileHash: string, fileName: string, totalChunks: number): Promise<{
    success: boolean;
    filePath: string;
    fileSize: number;
    md5: string;
  }> {
    const uploadDir = join(this.tempDir, fileHash);
    const statusFile = join(uploadDir, 'status.json');

    // 验证所有分片都已上传
    const status = this.loadChunkStatus(statusFile);
    const uploadedChunks = Object.keys(status.chunks).map(Number).sort((a, b) => a - b);

    if (uploadedChunks.length !== totalChunks) {
      throw new Error(`分片不完整: 已上传 ${uploadedChunks.length}/${totalChunks}`);
    }

    // 验证每个分片的大小
    for (const index of uploadedChunks) {
      const chunkFile = join(uploadDir, `chunk_${index.toString().padStart(6, '0')}`);
      if (!existsSync(chunkFile)) {
        throw new Error(`分片文件丢失: ${index}`);
      }
    }

    // 创建目标文件
    const ext = fileName.substring(fileName.lastIndexOf('.'));
    const finalFileName = `${fileHash}${ext}`;
    const finalFilePath = join(this.finalDir, finalFileName);

    // 使用流式合并，避免内存溢出
    const writeStream = createWriteStream(finalFilePath);
    const md5Hash = createHash('md5');

    for (const index of uploadedChunks) {
      const chunkFile = join(uploadDir, `chunk_${index.toString().padStart(6, '0')}`);

      // 读取分片并写入目标文件
      const chunkBuffer = await this.readFilePromise(chunkFile);
      writeStream.write(chunkBuffer);
      md5Hash.update(chunkBuffer);
    }

    writeStream.end();
    await new Promise<void>((resolve, reject) => {
      writeStream.on('finish', resolve);
      writeStream.on('error', reject);
    });

    const finalMd5 = md5Hash.digest('hex');
    const finalSize = statSync(finalFilePath).size;

    // 清理临时文件
    await this.cleanupTempFiles(fileHash);

    this.logger.log(`文件合并完成: ${finalFilePath}, MD5: ${finalMd5}, 大小: ${finalSize}`);

    return {
      success: true,
      filePath: finalFilePath,
      fileSize: finalSize,
      md5: finalMd5
    };
  }

  /**
   * Promise版本的读取文件
   */
  private readFilePromise(filePath: string): Promise<Buffer> {
    return new Promise((resolve, reject) => {
      const fs = require('fs');
      fs.readFile(filePath, (err, data) => {
        if (err) reject(err);
        else resolve(data);
      });
    });
  }

  /**
   * 清理临时文件
   */
  async cleanupTempFiles(fileHash: string): Promise<void> {
    const uploadDir = join(this.tempDir, fileHash);

    try {
      const files = readdirSync(uploadDir);
      for (const file of files) {
        const filePath = join(uploadDir, file);
        unlinkSync(filePath);
      }
      // 删除空目录
      const fs = require('fs');
      fs.rmdirSync(uploadDir);
      this.logger.log(`临时文件已清理: ${fileHash}`);
    } catch (error) {
      this.logger.error(`清理临时文件失败: ${error}`);
    }
  }

  /**
   * 删除上传记录（取消上传）
   */
  async deleteUpload(fileHash: string): Promise<void> {
    await this.cleanupTempFiles(fileHash);
    this.logger.log(`上传记录已删除: ${fileHash}`);
  }
}
```

### 3.4 临时目录管理

临时目录管理是分片上传系统中容易被忽视但非常重要的部分。合理的临时文件管理可以避免磁盘空间耗尽，提高系统稳定性。

```typescript
// src/upload/temp-file.manager.ts
import { Injectable, Logger, OnModuleInit, OnModuleDestroy } from '@nestjs/common';
import { existsSync, mkdirSync, readdirSync, statSync, unlinkSync, rmdirSync } from 'fs';
import { join } from 'path';
import { promisify } from 'util';

@Injectable()
export class TempFileManager implements OnModuleInit, OnModuleDestroy {
  private readonly logger = new Logger(TempFileManager.name);
  private readonly tempDir = './uploads/temp';
  private readonly maxAge = 24 * 60 * 60 * 1000; // 24小时
  private cleanupInterval: NodeJS.Timeout;

  onModuleInit() {
    // 确保临时目录存在
    if (!existsSync(this.tempDir)) {
      mkdirSync(this.tempDir, { recursive: true });
    }

    // 启动定时清理任务
    this.cleanupInterval = setInterval(() => {
      this.cleanupOldFiles();
    }, 60 * 60 * 1000); // 每小时执行一次

    this.logger.log('临时文件管理器已启动');
  }

  onModuleDestroy() {
    if (this.cleanupInterval) {
      clearInterval(this.cleanupInterval);
    }
    this.logger.log('临时文件管理器已停止');
  }

  /**
   * 清理过期文件
   */
  async cleanupOldFiles(): Promise<{
    cleanedFiles: number;
    cleanedSize: number;
  }> {
    this.logger.log('开始清理过期临时文件...');

    let cleanedFiles = 0;
    let cleanedSize = 0;
    const now = Date.now();

    try {
      const dirs = readdirSync(this.tempDir);

      for (const dir of dirs) {
        const dirPath = join(this.tempDir, dir);
        const stat = statSync(dirPath);

        // 检查目录年龄
        const age = now - stat.mtimeMs;

        if (age > this.maxAge) {
          // 清理目录
          const files = readdirSync(dirPath);
          for (const file of files) {
            const filePath = join(dirPath, file);
            const fileStat = statSync(filePath);
            cleanedSize += fileStat.size;
            unlinkSync(filePath);
            cleanedFiles++;
          }
          rmdirSync(dirPath);
          this.logger.log(`已清理过期目录: ${dir}`);
        }
      }

      this.logger.log(`清理完成: 清理了 ${cleanedFiles} 个文件，释放 ${this.formatSize(cleanedSize)}`);
      return { cleanedFiles, cleanedSize };

    } catch (error) {
      this.logger.error('清理过期文件失败:', error);
      return { cleanedFiles: 0, cleanedSize: 0 };
    }
  }

  /**
   * 获取临时目录使用情况
   */
  async getTempDirUsage(): Promise<{
    totalDirs: number;
    totalFiles: number;
    totalSize: number;
    oldestFile: Date | null;
    newestFile: Date | null;
  }> {
    let totalFiles = 0;
    let totalSize = 0;
    let oldestTime = Infinity;
    let newestTime = 0;

    try {
      const dirs = readdirSync(this.tempDir);

      for (const dir of dirs) {
        const dirPath = join(this.tempDir, dir);
        const files = readdirSync(dirPath);

        for (const file of files) {
          if (file === 'status.json') continue; // 跳过状态文件

          const filePath = join(dirPath, file);
          const stat = statSync(filePath);

          totalFiles++;
          totalSize += stat.size;
          oldestTime = Math.min(oldestTime, stat.mtimeMs);
          newestTime = Math.max(newestTime, stat.mtimeMs);
        }
      }

      return {
        totalDirs: dirs.length,
        totalFiles,
        totalSize,
        oldestFile: oldestTime === Infinity ? null : new Date(oldestTime),
        newestFile: newestTime === 0 ? null : new Date(newestTime)
      };

    } catch (error) {
      this.logger.error('获取临时目录使用情况失败:', error);
      return {
        totalDirs: 0,
        totalFiles: 0,
        totalSize: 0,
        oldestFile: null,
        newestFile: null
      };
    }
  }

  /**
   * 格式化文件大小
   */
  private formatSize(bytes: number): string {
    if (bytes === 0) return '0 B';
    const k = 1024;
    const sizes = ['B', 'KB', 'MB', 'GB'];
    const i = Math.floor(Math.log(bytes) / Math.log(k));
    return parseFloat((bytes / Math.pow(k, i)).toFixed(2)) + ' ' + sizes[i];
  }
}
```

## 四、MD5秒传

### 4.1 文件MD5计算

MD5（Message-Digest Algorithm 5）是一种广泛使用的哈希函数，可以将任意长度的数据转换为128位的哈希值。在文件上传场景中，MD5主要用于唯一标识文件、实现秒传功能和校验文件完整性。

MD5哈希具有以下特性，使其非常适合用于文件标识：确定性，即相同文件必然产生相同的MD5值；不可逆性，即无法从MD5值反推出原始数据；唯一性（理论），即不同文件产生相同MD5值的概率极低（1/2^128）；雪崩效应，即文件微小变化会导致MD5值完全不同。

在前端计算大文件的MD5，如果使用传统方式将整个文件读入内存，会导致内存占用过高和界面卡顿。解决方案是分块读取文件，只在内存中保留当前块的数据，通过增量计算得到最终MD5。

```javascript
/**
 * 大文件MD5计算器
 * 使用分块读取方式，避免内存溢出
 * 兼容浏览器和Web Worker环境
 */
class FileMD5Calculator {
  /**
   * 计算文件的MD5值
   * @param {File} file - 文件对象
   * @param {Function} onProgress - 进度回调函数
   * @returns {Promise<string>} MD5哈希值（32位十六进制字符串）
   */
  static async calculate(file, onProgress = () => {}) {
    return new Promise((resolve, reject) => {
      // 动态导入SparkMD5库（也可用其他MD5库）
      this.loadSparkMD5().then(SparkMD5 => {
        const chunkSize = 2 * 1024 * 1024; // 每次读取2MB
        const chunks = Math.ceil(file.size / chunkSize);
        const spark = new SparkMD5.ArrayBuffer();
        const reader = new FileReader();

        let currentChunk = 0;

        // 读取下一块
        const readNextChunk = () => {
          const start = currentChunk * chunkSize;
          const end = Math.min(start + chunkSize, file.size);

          if (start >= file.size) {
            // 读取完成
            const md5 = spark.end();
            resolve(md5);
            return;
          }

          reader.onload = (e) => {
            // 将数据追加到MD5计算器
            spark.append(e.target.result);
            currentChunk++;

            // 报告进度
            const progress = Math.round((currentChunk / chunks) * 100);
            onProgress(progress);

            // 读取下一块
            readNextChunk();
          };

          reader.onerror = () => {
            reject(new Error('文件读取失败'));
          };

          // 读取指定范围的二进制数据
          reader.readAsArrayBuffer(file.slice(start, end));
        };

        // 开始读取第一块
        readNextChunk();
      }).catch(reject);
    });
  }

  /**
   * 加载SparkMD5库
   */
  static async loadSparkMD5() {
    if (window.SparkMD5) {
      return window.SparkMD5;
    }

    return new Promise((resolve, reject) => {
      const script = document.createElement('script');
      script.src = 'https://cdn.jsdelivr.net/npm/spark-md5@3.0.2/spark-md5.min.js';
      script.onload = () => resolve(window.SparkMD5);
      script.onerror = reject;
      document.head.appendChild(script);
    });
  }

  /**
   * 计算文件的MD5（同步版本，适用于小文件）
   * @param {ArrayBuffer} arrayBuffer - 文件数据
   * @returns {string} MD5哈希值
   */
  static calculateSync(arrayBuffer) {
    // 同步计算需要先加载库
    const spark = new window.SparkMD5.ArrayBuffer();
    spark.append(arrayBuffer);
    return spark.end();
  }

  /**
   * 验证文件MD5
   * @param {File} file - 文件对象
   * @param {string} expectedMD5 - 期望的MD5值
   * @returns {Promise<boolean>} 是否匹配
   */
  static async verify(file, expectedMD5) {
    const actualMD5 = await this.calculate(file);
    return actualMD5.toLowerCase() === expectedMD5.toLowerCase();
  }
}

// 使用示例
const input = document.createElement('input');
input.type = 'file';
input.onchange = async (e) => {

  console.log('开始计算MD5...');

    const md5 = await FileMD5Calculator.calculate(file, (progress) => {
      console.log(`计算进度: ${progress}%`);
      document.getElementById('progress').textContent = `MD5计算中: ${progress}%`;
    });

    console.log(`文件MD5: ${md5}`);
    console.log(`文件大小: ${file.size} bytes`);
    console.log(`文件名: ${file.name}`);

  } catch (error) {
    console.error('MD5计算失败:', error);
  }
};
```

### 4.2 秒传原理

秒传（Instant Upload）是一种利用文件MD5值直接判断文件是否已存在于服务器的技术。当用户选择要上传的文件时，前端首先计算文件的MD5值，然后查询服务器是否已存在相同MD5的文件。如果存在，服务器直接返回已存在文件的访问地址，无需真正上传数据，实现"秒传"效果。

秒传的技术原理基于MD5哈希的确定性特性：相同内容的文件必然产生相同的MD5值。因此，只要服务器存储了每个上传文件的MD5值和存储地址，就可以在O(1)时间内完成文件存在性查询。

秒传的实现流程如下：用户选择文件后，前端立即计算文件MD5（不实际上传）；前端发送MD5查询请求到服务器；服务器在文件索引表中查找该MD5；如果找到，返回已存在文件的URL，前端显示"秒传成功"；如果未找到，继续正常的分片上传流程。

秒传的适用场景包括：重复文件上传，用户上传相同的文件到不同目录；分享文件，同事之间分享大文件（如视频、安装包）；备份恢复，备份软件识别已备份过的文件。

```typescript
// 秒传检测API
interface InstantUploadRequest {
  fileHash: string;      // 文件MD5
  fileName: string;       // 文件名
  fileSize: number;       // 文件大小
}

interface InstantUploadResponse {
  success: boolean;
  instant: boolean;       // 是否秒传成功
  fileUrl?: string;       // 已存在文件的URL
  fileId?: string;        // 文件ID
  needUpload: boolean;    // 是否需要上传
  uploadedChunks?: number[]; // 已上传的分片（断点续传用）
}

// 秒传检测实现
async checkInstantUpload(request: InstantUploadRequest): Promise<InstantUploadResponse> {
  const { fileHash, fileName, fileSize } = request;

  // 1. 在文件索引表中查找MD5
  const existingFile = await this.fileRepository.findByHash(fileHash);

  if (existingFile) {
    // 2. 秒传成功 - 文件已存在
    return {
      success: true,
      instant: true,
      fileUrl: existingFile.url,
      fileId: existingFile.id,
      needUpload: false
    };
  }

  // 3. 检查是否有未完成的上传（断点续传）
  const pendingUpload = await this.uploadRepository.findPendingByHash(fileHash);

  if (pendingUpload) {
    // 存在未完成的上传，返回已上传的分片列表
    const uploadedChunks = await this.uploadService.getUploadedChunks(fileHash);

    return {
      success: true,
      instant: false,
      needUpload: true,
      uploadedChunks
    };
  }

  // 4. 需要完整上传
  return {
    success: true,
    instant: false,
    needUpload: true
  };
}
```

### 4.3 秒传API集成

将秒传检测集成到前端上传流程中，形成完整的上传体验。

```javascript
/**
 * 智能上传管理器
 * 集成秒传检测、分片上传、断点续传
 */
class SmartUploader {
  constructor(options = {}) {
    this.uploadUrl = options.uploadUrl || '/api/upload/chunk';
    this.instantCheckUrl = options.instantCheckUrl || '/api/upload/check';
    this.mergeUrl = options.mergeUrl || '/api/upload/merge';
    this.chunkSize = options.chunkSize || 5 * 1024 * 1024;
    this.concurrentUploads = options.concurrentUploads || 3;

    // 回调函数
    this.onInstantSuccess = options.onInstantSuccess || (() => {});  // 秒传成功回调
    this.onUploadProgress = options.onUploadProgress || (() => {}); // 上传进度回调
    this.onComplete = options.onComplete || (() => {});
    this.onError = options.onError || (() => {});
  }

  /**
   * 上传文件（智能判断秒传或分片上传）
   * @param {File} file - 文件对象
   */
  async upload(file) {
    try {
      // 第一步：计算文件MD5
      this.onUploadProgress({ stage: 'calculating_hash', progress: 0 });

      const fileHash = await FileMD5Calculator.calculate(file, (progress) => {
        this.onUploadProgress({ stage: 'calculating_hash', progress });
      });

      this.onUploadProgress({ stage: 'hash_computed', hash: fileHash });
      console.log(`文件MD5: ${fileHash}`);

      // 第二步：检查是否可以秒传
      this.onUploadProgress({ stage: 'checking_instant', progress: 0 });

      const checkResult = await this.checkInstantUpload({
        fileHash,
        fileName: file.name,
        fileSize: file.size
      });

      if (checkResult.instant) {
        // 秒传成功！
        console.log('秒传成功！');
        this.onInstantSuccess({
          fileHash,
          fileName: file.name,
          fileSize: file.size,
          fileUrl: checkResult.fileUrl
        });
        return;
      }

      // 第三步：分片上传
      if (checkResult.uploadedChunks && checkResult.uploadedChunks.length > 0) {
        // 断点续传 - 跳过已上传的分片
        console.log(`检测到未完成的上传，已上传分片: ${checkResult.uploadedChunks.length}`);
        await this.resumableUpload(file, fileHash, checkResult.uploadedChunks);
      } else {
        // 全新上传
        console.log('开始分片上传...');
        await this.chunkedUpload(file, fileHash);
      }

    } catch (error) {
      this.onError(error);
    }
  }

  /**
   * 检查秒传
   */
  async checkInstantUpload(request) {
    const response = await fetch(this.instantCheckUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify(request)
    });

    if (!response.ok) {
      throw new Error('秒传检测请求失败');
    }

    return await response.json();
  }

  /**
   * 完整的分片上传
   */
  async chunkedUpload(file, fileHash) {
    const chunks = this.createChunks(file);
    const totalChunks = chunks.length;
    let uploadedCount = 0;

    // 并发上传控制
    const limiter = new ConcurrentLimiter(this.concurrentUploads);
    const uploadPromises = chunks.map((chunk, index) =>
      limiter.run(() => this.uploadChunk(chunk, index, fileHash, totalChunks))
    );

    await Promise.all(uploadPromises.map(p => p.catch(e => ({ error: e }))));

    // 合并分片
    await this.mergeChunks(fileHash, file.name, totalChunks);

    this.onComplete({ fileHash, success: true });
  }

  /**
   * 断点续传
   */
  async resumableUpload(file, fileHash, uploadedChunks) {
    const chunks = this.createChunks(file);
    const uploadedSet = new Set(uploadedChunks);
    const pendingChunks = chunks.filter((_, index) => !uploadedSet.has(index));

    console.log(`待上传分片: ${pendingChunks.length}/${chunks.length}`);

    // 并发上传待上传的分片
    const limiter = new ConcurrentLimiter(this.concurrentUploads);
    const uploadPromises = pendingChunks.map(chunk =>
      limiter.run(() => this.uploadChunk(chunk, chunk.index, fileHash, chunks.length))
    );

    await Promise.all(uploadPromises.map(p => p.catch(e => ({ error: e }))));

    // 合并分片
    await this.mergeChunks(fileHash, file.name, chunks.length);

    this.onComplete({ fileHash, success: true, resumed: true });
  }

  /**
   * 创建文件分片
   */
  createChunks(file) {
    const chunks = [];
    const totalChunks = Math.ceil(file.size / this.chunkSize);
    let start = 0;

    for (let i = 0; i < totalChunks; i++) {
      const end = Math.min(start + this.chunkSize, file.size);
      chunks.push({
        index: i,
        blob: file.slice(start, end),
        start,
        end,
        size: end - start
      });
      start = end;
    }

    return chunks;
  }

  /**
   * 上传单个分片
   */
  async uploadChunk(chunk, index, fileHash, totalChunks) {
    const formData = new FormData();
    formData.append('chunk', chunk.blob);
    formData.append('index', index.toString());
    formData.append('fileHash', fileHash);
    formData.append('fileName', chunk.fileName || 'unknown');
    formData.append('totalChunks', totalChunks.toString());

    const response = await fetch(this.uploadUrl, {
      method: 'POST',
      body: formData
    });

    if (!response.ok) {
      throw new Error(`分片 ${index} 上传失败`);
    }

    const result = await response.json();
    this.onUploadProgress({
      stage: 'uploading',
      progress: Math.round((result.uploadedChunks.length / totalChunks) * 100),
      currentChunk: index
    });

    return result;
  }

  /**
   * 合并分片
   */
  async mergeChunks(fileHash, fileName, totalChunks) {
    this.onUploadProgress({ stage: 'merging' });

    const response = await fetch(this.mergeUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        fileHash,
        fileName,
        totalChunks
      })
    });

    if (!response.ok) {
      throw new Error('分片合并失败');
    }

    return await response.json();
  }
}
```

## 五、断点续传

### 5.1 已上传记录

断点续传（Resume Upload）是分片上传的重要扩展功能，允许在上传中断后从已上传的分片位置继续传输，而不是从头开始。这对于大文件上传和网络不稳定的场景至关重要。

已上传记录是断点续传的基础。服务器需要持久化存储每个文件的分片上传状态，包括哪些分片已上传、每个分片的大小、文件元信息等。

### 5.2 分片状态管理

分片状态管理是断点续传的核心，需要准确跟踪每个分片的当前状态，并根据状态决定是否需要上传。

```typescript
// src/upload/upload-state.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { UploadRecord } from './entities/upload-record.entity';
import { UploadService } from './upload.service';

@Injectable()
export class UploadStateService {
  private readonly logger = new Logger(UploadStateService.name);

  constructor(
    @InjectRepository(UploadRecord)
    private uploadRecordRepository: Repository<UploadRecord>,
    private uploadService: UploadService
  ) {}

  /**
   * 获取或创建上传记录
   */
  async getOrCreateRecord(params: {
    fileHash: string;
    fileName: string;
    fileSize: number;
    totalChunks: number;
    userId?: string;
  }): Promise<UploadRecord> {
    let record = await this.uploadRecordRepository.findOne({
      where: { fileHash: params.fileHash }
    });

    if (!record) {
      record = await this.uploadRecordRepository.createRecord(params);
      this.logger.log(`创建上传记录: ${params.fileHash}`);
    }

    return record;
  }

  /**
   * 获取当前上传状态
   */
  async getUploadStatus(fileHash: string): Promise<{
    uploadedChunks: number[];
    totalChunks: number;
    progress: number;
    status: string;
  }> {
    const record = await this.uploadRecordRepository.getRecord(fileHash);

    if (!record) {
      return {
        uploadedChunks: [],
        totalChunks: 0,
        progress: 0,
        status: 'not_found'
      };
    }

    const progress = record.totalChunks > 0
      ? Math.round((record.uploadedChunks.length / record.totalChunks) * 100)
      : 0;

    return {
      uploadedChunks: record.uploadedChunks,
      totalChunks: record.totalChunks,
      progress,
      status: record.status
    };
  }

  /**
   * 标记分片已上传
   */
  async markChunkUploaded(
    fileHash: string,
    chunkIndex: number,
    chunkSize: number
  ): Promise<void> {
    await this.uploadRecordRepository.updateChunkStatus(fileHash, chunkIndex, {
      size: chunkSize
    });
  }

  /**
   * 获取需要续传的缺失分片
   */
  async getMissingChunks(fileHash: string): Promise<number[]> {
    const record = await this.uploadRecordRepository.getRecord(fileHash);

    if (!record || record.uploadedChunks.length === 0) {
      return []; // 没有记录，需要全部上传
    }

    // 生成完整的分片索引列表
    const allChunks = Array.from(
      { length: record.totalChunks },
      (_, i) => i
    );

    // 找出缺失的分片
    const uploadedSet = new Set(record.uploadedChunks);
    return allChunks.filter(index => !uploadedSet.has(index));
  }

  /**
   * 完成上传并合并文件
   */
  async completeUpload(fileHash: string): Promise<{
    success: boolean;
    filePath: string;
    fileSize: number;
    md5: string;
  }> {
    const record = await this.uploadRecordRepository.getRecord(fileHash);

    if (!record) {
      throw new Error('上传记录不存在');
    }

    if (record.uploadedChunks.length !== record.totalChunks) {
      throw new Error(`分片不完整: ${record.uploadedChunks.length}/${record.totalChunks}`);
    }

    // 合并分片
    const result = await this.uploadService.mergeChunks(
      fileHash,
      record.fileName,
      record.totalChunks
    );

    // 更新记录状态
    record.status = 'completed';
    record.filePath = result.filePath;
    record.finalMd5 = result.md5;
    await this.uploadRecordRepository.save(record);

    // 清理临时文件
    await this.uploadService.cleanupTempFiles(fileHash);

    return result;
  }

  /**
   * 取消上传
   */
  async cancelUpload(fileHash: string): Promise<void> {
    const record = await this.uploadRecordRepository.getRecord(fileHash);

    if (record) {
      // 清理临时文件
      await this.uploadService.cleanupTempFiles(fileHash);

      // 删除记录
      await this.uploadRecordRepository.deleteRecord(fileHash);

      this.logger.log(`已取消上传: ${fileHash}`);
    }
  }

  /**
   * 清理过期上传
   */
  async cleanupExpiredUploads(): Promise<number> {
    const expiredRecords = await this.uploadRecordRepository.find({
      where: {
        status: 'uploading'
      }
    });

    let cleanedCount = 0;

    for (const record of expiredRecords) {
      if (record.expiresAt && new Date() > record.expiresAt) {
        await this.cancelUpload(record.fileHash);
        cleanedCount++;
      }
    }

    this.logger.log(`清理了 ${cleanedCount} 个过期上传记录`);
    return cleanedCount;
  }
}
```

### 5.3 续传请求处理

前端在恢复上传时，需要正确处理续传逻辑，从服务器获取已上传的分片信息，然后只上传缺失的分片。

```javascript
/**
 * 续传管理器
 * 处理上传中断后的恢复逻辑
 */
class ResumableUploader {
  constructor(options = {}) {
    this.chunkSize = options.chunkSize || 5 * 1024 * 1024;
    this.maxRetries = options.maxRetries || 3;
    this.concurrentUploads = options.concurrentUploads || 3;
    this.onProgress = options.onProgress || (() => {});
    this.onComplete = options.onComplete || (() => {});
    this.onError = options.onError || (() => {});

    // 内部状态
    this.file = null;
    this.fileHash = '';
    this.chunks = [];
    this.uploadedChunks = new Set();
    this.failedChunks = [];
    this.isPaused = false;
    this.isCancelled = false;
  }

  /**
   * 初始化上传
   */
  async init(file) {
    this.file = file;

    // 计算文件哈希
    this.fileHash = await FileMD5Calculator.calculate(file);

    // 创建分片
    this.chunks = this.createChunks(file);

    // 查询已上传的分片
    const status = await this.queryUploadStatus();

    if (status.uploadedChunks) {
      this.uploadedChunks = new Set(status.uploadedChunks);
    }

    console.log(`初始化完成: 总分片${this.chunks.length}, 已上传${this.uploadedChunks.size}`);
  }

  /**
   * 创建分片
   */
  createChunks(file) {
    const chunks = [];
    const totalChunks = Math.ceil(file.size / this.chunkSize);
    let start = 0;

    for (let i = 0; i < totalChunks; i++) {
      const end = Math.min(start + this.chunkSize, file.size);
      chunks.push({
        index: i,
        blob: file.slice(start, end),
        start,
        end,
        size: end - start,
        uploaded: false
      });
      start = end;
    }

    return chunks;
  }

  /**
   * 查询上传状态
   */
  async queryUploadStatus() {
    try {
      const response = await fetch(`/api/upload/status/${this.fileHash}`);
      if (response.ok) {
        return await response.json();
      }
    } catch (error) {
      console.warn('查询上传状态失败:', error);
    }
    return { uploadedChunks: [] };
  }

  /**
   * 开始/恢复上传
   */
  async start() {
    if (!this.file) {
      throw new Error('请先调用init()');
    }

    this.isPaused = false;
    this.isCancelled = false;

    // 找出需要上传的分片
    const pendingChunks = this.chunks.filter(
      chunk => !this.uploadedChunks.has(chunk.index)
    );

    console.log(`开始上传，待上传分片: ${pendingChunks.length}`);

    // 使用并发控制器
    const limiter = new ConcurrentLimiter(this.concurrentUploads);
    const promises = pendingChunks.map(chunk =>
      limiter.run(() => this.uploadChunkWithRetry(chunk))
    );

    try {
      const results = await Promise.allSettled(promises);

      // 检查是否有失败
      const failures = results.filter(r => r.status === 'rejected');
      if (failures.length > 0 && !this.isPaused) {
        throw new Error(`${failures.length}个分片上传失败`);
      }

      // 所有分片上传完成，触发合并
      if (!this.isPaused && !this.isCancelled) {
        await this.mergeChunks();
        this.onComplete({
          fileHash: this.fileHash,
          fileName: this.file.name,
          success: true
        });
      }

    } catch (error) {
      if (!this.isPaused) {
        this.onError(error);
      }
    }
  }

  /**
   * 带重试的分片上传
   */
  async uploadChunkWithRetry(chunk, retries = 0) {
    try {
      await this.uploadChunk(chunk);
      this.uploadedChunks.add(chunk.index);
    } catch (error) {
      if (retries < this.maxRetries) {
        console.log(`分片${chunk.index}失败，${retries + 1}次重试...`);
        await this.delay(1000 * (retries + 1));
        return this.uploadChunkWithRetry(chunk, retries + 1);
      }
      this.failedChunks.push(chunk.index);
      throw error;
    }
  }

  /**
   * 上传单个分片
   */
  async uploadChunk(chunk) {
    const formData = new FormData();
    formData.append('chunk', chunk.blob);
    formData.append('index', chunk.index.toString());
    formData.append('fileHash', this.fileHash);
    formData.append('totalChunks', this.chunks.length.toString());

    const response = await fetch('/api/upload/chunk', {
      method: 'POST',
      body: formData
    });

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}`);
    }

    const result = await response.json();

    // 报告进度
    const progress = Math.round(
      ((this.uploadedChunks.size + 1) / this.chunks.length) * 100
    );
    this.onProgress({
      uploadedChunks: this.uploadedChunks.size + 1,
      totalChunks: this.chunks.length,
      progress,
      currentChunk: chunk.index
    });

    return result;
  }

  /**
   * 合并分片
   */
  async mergeChunks() {
    const response = await fetch('/api/upload/merge', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        fileHash: this.fileHash,
        fileName: this.file.name,
        totalChunks: this.chunks.length
      })
    });

    if (!response.ok) {
      throw new Error('合并失败');
    }

    return await response.json();
  }

  /**
   * 暂停上传
   */
  pause() {
    this.isPaused = true;
    console.log('上传已暂停');
  }

  /**
   * 恢复上传
   */
  async resume() {
    if (!this.isPaused) return;

    console.log('恢复上传...');
    this.isPaused = false;

    // 重新查询状态
    const status = await this.queryUploadStatus();
    if (status.uploadedChunks) {
      this.uploadedChunks = new Set(status.uploadedChunks);
    }

    await this.start();
  }

  /**
   * 取消上传
   */
  async cancel() {
    this.isCancelled = true;
    this.isPaused = false;

    // 请求服务器删除上传记录
    try {
      await fetch(`/api/upload/${this.fileHash}`, {
        method: 'DELETE'
      });
    } catch (error) {
      console.error('取消上传请求失败:', error);
    }

    console.log('上传已取消');
  }

  /**
   * 延迟函数
   */
  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

// 导出给外部使用
window.ResumableUploader = ResumableUploader;
```

## 六、并发控制

### 6.1 并发数量策略

并发上传的分片数量直接影响上传速度和系统资源占用。合理的并发策略需要在上传速度和服务器压力之间取得平衡。

并发数量策略需要考虑以下因素：网络带宽，用户带宽是决定并发数量的首要因素；服务器承受能力，高并发会增加服务器CPU、内存、IO压力；浏览器限制，浏览器对同一域名的并发连接数有限制（通常为6个）；文件大小，对于小文件，过多并发反而会增加开销。

以下是一个自适应的并发控制器，可以根据实际情况动态调整并发数量：

```javascript
/**
 * 自适应并发控制器
 * 根据上传速度和服务器响应时间动态调整并发数量
 */
class AdaptiveConcurrencyController {
  constructor(options = {}) {
    this.minConcurrent = options.minConcurrent || 1;      // 最小并发数
    this.maxConcurrent = options.maxConcurrent || 6;       // 最大并发数
    this.targetTime = options.targetTime || 30000;        // 目标单分片上传时间(ms)
    this.adjustInterval = options.adjustInterval || 10000; // 调整间隔(ms)

    // 状态
    this.currentConcurrent = options.initialConcurrent || 3;
    this.runningTasks = 0;
    this.queue = [];
    this.sampleTimes = [];
    this.lastAdjustTime = Date.now();

    // 性能指标
    this.metrics = {
      totalChunks: 0,
      successChunks: 0,
      failedChunks: 0,
      avgTime: 0,
      avgSpeed: 0
    };
  }

  /**
   * 添加任务
   */
  run(task) {
    return new Promise((resolve, reject) => {
      this.queue.push({ task, resolve, reject });
      this.schedule();
    });
  }

  /**
   * 调度任务
   */
  schedule() {
    // 调整并发
    this.adjustConcurrency();

    // 如果有空闲槽位且队列不为空，则启动任务
    while (this.runningTasks < this.currentConcurrent && this.queue.length > 0) {
      const { task, resolve, reject } = this.queue.shift();
      this.runningTasks++;

      const startTime = Date.now();

      task()
        .then(result => {
          this.runningTasks--;
          this.recordSuccess(Date.now() - startTime);
          resolve(result);
          this.schedule(); // 调度下一个任务
        })
        .catch(error => {
          this.runningTasks--;
          this.recordFailure(Date.now() - startTime);
          reject(error);
          this.schedule();
        });
    }
  }

  /**
   * 记录成功
   */
  recordSuccess(time) {
    this.sampleTimes.push(time);
    if (this.sampleTimes.length > 10) {
      this.sampleTimes.shift();
    }
    this.metrics.successChunks++;
    this.metrics.totalChunks++;
    this.metrics.avgTime = this.sampleTimes.reduce((a, b) => a + b, 0) / this.sampleTimes.length;
  }

  /**
   * 记录失败
   */
  recordFailure(time) {
    this.sampleTimes.push(time);
    if (this.sampleTimes.length > 10) {
      this.sampleTimes.shift();
    }
    this.metrics.failedChunks++;
    this.metrics.totalChunks++;
    this.metrics.avgTime = this.sampleTimes.reduce((a, b) => a + b, 0) / this.sampleTimes.length;
  }

  /**
   * 调整并发数量
   */
  adjustConcurrency() {
    const now = Date.now();

    // 避免频繁调整
    if (now - this.lastAdjustTime < this.adjustInterval) {
      return;
    }

    this.lastAdjustTime = now;

    const avgTime = this.metrics.avgTime;

    // 根据平均上传时间调整并发
    if (avgTime > this.targetTime * 1.5 && this.currentConcurrent > this.minConcurrent) {
      // 上传时间过长，减少并发
      this.currentConcurrent = Math.max(this.minConcurrent, this.currentConcurrent - 1);
      console.log(`📉 降低并发到 ${this.currentConcurrent} (平均时间: ${avgTime}ms)`);

    } else if (avgTime < this.targetTime * 0.5 && this.currentConcurrent < this.maxConcurrent) {
      // 上传时间过短，增加并发
      this.currentConcurrent = Math.min(this.maxConcurrent, this.currentConcurrent + 1);
      console.log(`📈 增加并发到 ${this.currentConcurrent} (平均时间: ${avgTime}ms)`);
    }
  }

  /**
   * 获取当前状态
   */
  getStatus() {
    return {
      currentConcurrent: this.currentConcurrent,
      runningTasks: this.runningTasks,
      queueLength: this.queue.length,
      ...this.metrics
    };
  }

  /**
   * 重置
   */
  reset() {
    this.currentConcurrent = 3;
    this.runningTasks = 0;
    this.queue = [];
    this.sampleTimes = [];
    this.metrics = {
      totalChunks: 0,
      successChunks: 0,
      failedChunks: 0,
      avgTime: 0,
      avgSpeed: 0
    };
  }
}

/**
 * 简单的并发限制器
 */
class ConcurrentLimiter {
  constructor(maxConcurrent) {
    this.maxConcurrent = maxConcurrent;
    this.running = 0;
    this.queue = [];
  }

  /**
   * 执行任务
   */
  async run(task) {
    return new Promise((resolve, reject) => {
      this.queue.push({ task, resolve, reject });
      this.process();
    });
  }

  /**
   * 处理队列
   */
  async process() {
    while (this.running < this.maxConcurrent && this.queue.length > 0) {
      const { task, resolve, reject } = this.queue.shift();
      this.running++;

      task()
        .then(resolve)
        .catch(reject)
        .finally(() => {
          this.running--;
          this.process();
        });
    }
  }
```

### 6.2 速度控制

速度控制用于限制上传速度，避免占用过多带宽影响其他网络活动。

```javascript
/**
 * 限速上传器
 * 通过控制发送的数据量实现速度限制
 */
class RateLimitedUploader {
  constructor(options = {}) {
    this.maxSpeed = options.maxSpeed || 1024 * 1024; // 默认1MB/s
    this.chunkSize = options.chunkSize || 64 * 1024; // 限速块大小64KB
    this.currentSpeed = 0;
    this.uploadedBytes = 0;
    this.startTime = 0;
  }

  /**
   * 限速上传
   * @param {Blob} blob - 要上传的数据
   * @param {Function} uploadFunc - 实际上传函数
   */
  async uploadWithLimit(blob, uploadFunc) {
    if (!this.startTime) {
      this.startTime = Date.now();
    }

    // 将大块数据分成小块，模拟限速
    const chunks = [];
    let offset = 0;

    while (offset < blob.size) {
      const chunk = blob.slice(offset, offset + this.chunkSize);
      chunks.push(chunk);
      offset += this.chunkSize;
    }

    // 分块上传，每块之间检查是否需要限速
    for (const chunk of chunks) {
      // 检查当前速度
      const elapsed = (Date.now() - this.startTime) / 1000; // 秒
      const currentSpeed = this.uploadedBytes / elapsed;

      if (currentSpeed > this.maxSpeed) {
        // 超过限速，等待
        const waitTime = (this.uploadedBytes / this.maxSpeed) * 1000 - elapsed * 1000;
        if (waitTime > 0) {
          await this.delay(waitTime);
        }
      }

      // 上传当前块
      await uploadFunc(chunk);
      this.uploadedBytes += chunk.size;
    }
  }

  /**
   * 延迟函数
   */
  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }

  /**
   * 获取当前速度
   */
  getCurrentSpeed() {
    if (!this.startTime) return 0;
    const elapsed = (Date.now() - this.startTime) / 1000;
    return this.uploadedBytes / elapsed;
  }

  /**
   * 重置
   */
  reset() {
    this.currentSpeed = 0;
    this.uploadedBytes = 0;
    this.startTime = 0;
  }
}

class TokenBucketRateLimiter {
  constructor(options = {}) {
    this.capacity = options.capacity || 1024 * 1024; // 桶容量（字节）
    this.refillRate = options.refillRate || 500 * 1024; // 每秒补充速度（字节）
    this.tokens = this.capacity;
    this.lastRefill = Date.now();
  }

  /**
   * 获取令牌
   */
  async getToken(bytes) {
    // 先补充令牌
    this.refill();

    // 等待足够的令牌
    while (this.tokens < bytes) {
      const waitTime = ((bytes - this.tokens) / this.refillRate) * 1000;
      await this.delay(waitTime);
      this.refill();
    }

    // 消耗令牌
    this.tokens -= bytes;
  }

  /**
   * 补充令牌
   */
  refill() {
    const now = Date.now();
    const elapsed = (now - this.lastRefill) / 1000;
    const tokensToAdd = elapsed * this.refillRate;

    this.tokens = Math.min(this.capacity, this.tokens + tokensToAdd);
    this.lastRefill = now;
  }

  /**
   * 延迟函数
   */
  delay(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
```

```javascript
/**
 * 上传队列管理器

class UploadQueueManager {
  constructor(options = {}) {
    this.maxConcurrent = options.maxConcurrent || 2; // 同时上传的文件数
    this.maxRetries = options.maxRetries || 3;
    this.runningUploads = new Map();  // 正在上传的文件
    this.pendingQueue = [];           // 等待上传的文件
    this.completedUploads = [];      // 已完成的上传
    this.failedUploads = [];         // 失败的上传

    // 回调函数
    this.onQueueChange = options.onQueueChange || (() => {});
    this.onUploadProgress = options.onUploadProgress || (() => {});
    this.onUploadComplete = options.onUploadComplete || (() => {});
    this.onUploadError = options.onUploadError || (() => {});
  }

  /**
   * 添加文件到上传队列
   */
  addFile(file, options = {}) {
    const uploadItem = {
      id: this.generateId(),
      file,
      fileHash: null,         // 稍后计算
      status: 'pending',      // pending, calculating_hash, checking, uploading, completed, failed, cancelled
      progress: 0,
      error: null,
      createdAt: new Date(),
      options
    };

    this.pendingQueue.push(uploadItem);
    this.onQueueChange(this.getQueueInfo());

    // 自动开始处理队列
    this.processQueue();

    return uploadItem.id;
  }

  /**
   * 处理队列
   */
  async processQueue() {
    // 如果正在上传的文件数已达到上限，等待
    if (this.runningUploads.size >= this.maxConcurrent) {
      return;
    }

    // 从等待队列中取出下一个
    const nextItem = this.pendingQueue.shift();
    if (!nextItem) {
      return;
    }

    // 开始上传
    this.runningUploads.set(nextItem.id, nextItem);
    nextItem.status = 'calculating_hash';

    try {
      await this.processUpload(nextItem);
    } catch (error) {
      nextItem.status = 'failed';
      nextItem.error = error.message;
      this.failedUploads.push(nextItem);
      this.onUploadError(nextItem.id, error);
    } finally {
      this.runningUploads.delete(nextItem.id);
      this.onQueueChange(this.getQueueInfo());

      // 处理队列中的下一个
      this.processQueue();
    }
  }

  /**
   * 处理单个文件的上传

  async processUpload(item) {
    // 计算文件哈希

      item.status = 'completed';
      this.completedUploads.push(item);
      this.onUploadComplete(item.id, { instant: true, fileUrl: instantResult.fileUrl });
      return;
    }

    // 分片上传
    item.status = 'uploading';
    const uploader = new ChunkedUploader({
      chunkSize: item.options.chunkSize || 5 * 1024 * 1024,
      concurrentUploads: item.options.concurrentUploads || 3,
      maxRetries: this.maxRetries

      item.progress = progress.progress;
      this.onUploadProgress(item.id, { stage: 'upload', ...progress });
    };

    uploader.onComplete = () => {
      item.status = 'completed';
      this.completedUploads.push(item);
      this.onUploadComplete(item.id, { instant: false });
```

## 七、分片合并

### 7.1 流式合并

流式合并是处理大文件分片合并的最佳实践，它使用流的方式逐步读取分片并写入目标文件，避免将所有分片一次性加载到内存中。

```typescript
// src/upload/merge.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { createReadStream, createWriteStream, statSync, unlinkSync, readdirSync } from 'fs';
import { join, dirname } from 'path';
import { createHash } from 'crypto';
import { pipeline } from 'stream/promises';

@Injectable()
export class FileMergeService {
  private readonly logger = new Logger(FileMergeService.name);

  /**
   * 流式合并分片
   * 使用pipeline确保数据完整性
   */
  async mergeChunksStream(
    uploadDir: string,
    outputPath: string,
    totalChunks: number
  ): Promise<{
    success: boolean;
    outputPath: string;
    outputSize: number;
    md5: string;
  }> {
    this.logger.log(`开始流式合并: ${totalChunks} 个分片`);

    // 确保输出目录存在
    const outputDir = dirname(outputPath);
    const fs = require('fs');
    if (!fs.existsSync(outputDir)) {
      fs.mkdirSync(outputDir, { recursive: true });
    }

    // 创建写入流
    const writeStream = createWriteStream(outputPath);
    const md5Hash = createHash('md5');

    // 按顺序合并分片
    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);

      // 验证分片存在
      if (!fs.existsSync(chunkPath)) {
        writeStream.close();
        unlinkSync(outputPath); // 删除不完整的文件
        throw new Error(`分片 ${i} 不存在`);
      }

      // 读取分片内容
      const chunkBuffer = await this.readFile(chunkPath);

      // 更新MD5
      md5Hash.update(chunkBuffer);

      // 写入目标文件
      writeStream.write(chunkBuffer);

      this.logger.debug(`分片 ${i} 已合并`);
    }

    // 结束写入
    writeStream.end();

    // 等待写入完成
    await new Promise<void>((resolve, reject) => {
      writeStream.on('finish', resolve);
      writeStream.on('error', reject);
    });

    // 验证输出文件
    const outputStats = statSync(outputPath);
    const finalMd5 = md5Hash.digest('hex');

    this.logger.log(`合并完成: ${outputPath}, 大小: ${outputStats.size}, MD5: ${finalMd5}`);

    return {
      success: true,
      outputPath,
      outputSize: outputStats.size,
      md5: finalMd5
    };
  }

  /**
   * Promise版本的读取文件
   */
  private readFile(filePath: string): Promise<Buffer> {
    return new Promise((resolve, reject) => {
      const fs = require('fs');
      fs.readFile(filePath, (err, data) => {
        if (err) reject(err);
        else resolve(data);
      });
    });
  }

  /**
   * 验证分片完整性
   */
  async verifyChunks(uploadDir: string, totalChunks: number): Promise<{
    valid: boolean;
    missingChunks: number[];
    invalidChunks: number[];
  }> {
    const missingChunks: number[] = [];
    const invalidChunks: number[] = [];

    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);

      if (!require('fs').existsSync(chunkPath)) {
        missingChunks.push(i);
      }
    }

    return {
      valid: missingChunks.length === 0,
      missingChunks,
      invalidChunks
    };
  }

  /**
   * 获取合并进度（用于大文件监控）
   */
  async getMergeProgress(
    uploadDir: string,
    outputPath: string,
    totalChunks: number
  ): Promise<{
    currentChunk: number;
    totalChunks: number;
    progress: number;
    outputSize: number;
  }> {
    const fs = require('fs');

    let currentChunk = 0;

    // 检查已合并的分片数量
    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);
      if (!fs.existsSync(chunkPath)) {
        break;
      }
      currentChunk++;
    }

    let outputSize = 0;
    if (fs.existsSync(outputPath)) {
      outputSize = statSync(outputPath).size;
    }

    return {
      currentChunk,
      totalChunks,
      progress: Math.round((currentChunk / totalChunks) * 100),
      outputSize
    };
  }
}
```

### 7.2 内存优化

在处理超大文件时，内存优化至关重要。以下策略可以显著减少内存占用：

```typescript
// src/upload/optimized-merge.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { createReadStream, createWriteStream, statSync } from 'fs';
import { join } from 'path';
import { createHash } from 'crypto';
import { Readable } from 'stream';

@Injectable()
export class OptimizedMergeService {
  private readonly logger = new Logger(OptimizedMergeService.name);
  private readonly BUFFER_SIZE = 64 * 1024; // 64KB缓冲区

  /**
   * 优化的分片合并 - 使用流式管道
   * 内存占用恒定，不随文件大小增加
   */
  async mergeWithPipeline(
    uploadDir: string,
    outputPath: string,
    totalChunks: number
  ): Promise<{ md5: string; outputSize: number }> {
    this.logger.log(`使用管道合并 ${totalChunks} 个分片`);

    const fs = require('fs');

    // 创建写入流
    const writeStream = createWriteStream(outputPath, {
      highWaterMark: this.BUFFER_SIZE
    });

    const md5Hash = createHash('md5');
    let totalBytes = 0;

    // 创建写入 Promise
    const writePromises: Promise<void>[] = [];

    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);

      if (!fs.existsSync(chunkPath)) {
        throw new Error(`分片 ${i} 不存在`);
      }

      // 创建读取流
      const readStream = createReadStream(chunkPath, {
        highWaterMark: this.BUFFER_SIZE
      });

      // 使用管道将读取流直接导向写入流
      const pipePromise = new Promise<void>((resolve, reject) => {
        readStream.on('data', (chunk: Buffer) => {
          md5Hash.update(chunk);
          totalBytes += chunk.length;
        });

        readStream.on('end', resolve);
        readStream.on('error', reject);
      });

      // 将读取流接入写入流
      readStream.pipe(writeStream, { end: false });

      writePromises.push(pipePromise);
    }

    // 等待所有读取完成
    await Promise.all(writePromises);

    // 结束写入流
    writeStream.end();

    // 等待写入完成
    await new Promise<void>((resolve, reject) => {
      writeStream.on('finish', resolve);
      writeStream.on('error', reject);
    });

    const finalMd5 = md5Hash.digest('hex');
    this.logger.log(`合并完成，总大小: ${totalBytes} bytes, MD5: ${finalMd5}`);

    return {
      md5: finalMd5,
      outputSize: totalBytes
    };
  }

  /**
   * 并行流式合并 - 多线程加速
   */
  async parallelMerge(
    uploadDir: string,
    outputPath: string,
    totalChunks: number,
    parallelCount = 4
  ): Promise<{ md5: string; outputSize: number }> {
    this.logger.log(`并行合并 ${totalChunks} 个分片（并发度: ${parallelCount}）`);

    const fs = require('fs');
    const md5Hash = createHash('md5');
    let totalBytes = 0;

    // 创建写入流
    const writeStream = createWriteStream(outputPath);

    // 将分片分成多组
    const chunkGroups: number[][] = [];
    for (let i = 0; i < totalChunks; i += parallelCount) {
      const group: number[] = [];
      for (let j = i; j < i + parallelCount && j < totalChunks; j++) {
        group.push(j);
      }
      chunkGroups.push(group);
    }

    // 逐组合并
    for (const group of chunkGroups) {
      // 并行读取组内所有分片
      const groupPromises = group.map(chunkIndex => {
        const chunkPath = join(uploadDir, `chunk_${chunkIndex.toString().padStart(6, '0')}`);
        return this.readChunk(chunkPath);
      });

      const chunks = await Promise.all(groupPromises);

      // 顺序写入
      for (const chunk of chunks) {
        md5Hash.update(chunk);
        totalBytes += chunk.length;
        writeStream.write(chunk);
      }
    }

    writeStream.end();

    await new Promise<void>((resolve, reject) => {
      writeStream.on('finish', resolve);
      writeStream.on('error', reject);
    });

    return {
      md5: md5Hash.digest('hex'),
      outputSize: totalBytes
    };
  }

  /**
   * 读取分片（返回Promise<Buffer>）
   */
  private readChunk(chunkPath: string): Promise<Buffer> {
    return new Promise((resolve, reject) => {
      const fs = require('fs');
      fs.readFile(chunkPath, (err, data) => {
        if (err) reject(err);
        else resolve(data);
      });
    });
  }

  /**
   * 零拷贝合并（使用硬链接，需要同文件系统）
   */
  async zeroCopyMerge(
    uploadDir: string,
    outputPath: string,
    totalChunks: number
  ): Promise<{ md5: string; outputSize: number }> {
    this.logger.log(`尝试零拷贝合并 ${totalChunks} 个分片`);

    const fs = require('fs');
    const path = require('path');

    // 首先尝试创建硬链接
    const links: string[] = [];
    let canZeroCopy = true;

    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);
      const linkPath = join(uploadDir, `link_${i.toString().padStart(6, '0')}`);

      try {
        // 尝试创建硬链接
        fs.linkSync(chunkPath, linkPath);
        links.push(linkPath);
      } catch (error) {
        // 不支持硬链接，降级到普通合并
        canZeroCopy = false;
        // 清理已创建的链接
        for (const link of links) {
          try { fs.unlinkSync(link); } catch {}
        }
        break;
      }
    }

    if (!canZeroCopy) {
      // 降级到普通合并
      return this.mergeWithPipeline(uploadDir, outputPath, totalChunks);
    }

    // 零拷贝方案：将链接的文件合并
    // 实际应用中可能需要使用专门的工具如concat-files
    this.logger.log('零拷贝不可用，降级到流式合并');

    // 清理链接
    for (const link of links) {
      try { fs.unlinkSync(link); } catch {}
    }

    return this.mergeWithPipeline(uploadDir, outputPath, totalChunks);
  }
}
```

### 7.3 完整性校验

合并完成后，必须对文件进行完整性校验，确保合并过程没有数据丢失或损坏。

```typescript
// src/upload/integrity.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { createReadStream } from 'fs';
import { createHash } from 'crypto';

@Injectable()
export class IntegrityService {
  private readonly logger = new Logger(IntegrityService.name);

  /**
   * 验证文件完整性
   * @param filePath - 文件路径
   * @param expectedMd5 - 期望的MD5值
   * @returns 是否匹配
   */
  async verifyIntegrity(filePath: string, expectedMd5: string): Promise<{
    valid: boolean;
    actualMd5: string;
  }> {
    const actualMd5 = await this.calculateFileMd5(filePath);

    return {
      valid: actualMd5.toLowerCase() === expectedMd5.toLowerCase(),
      actualMd5
    };
  }

  /**
   * 计算文件的MD5
   * 使用流式计算，避免内存溢出
   */
  async calculateFileMd5(filePath: string): Promise<string> {
    return new Promise((resolve, reject) => {
      const hash = createHash('md5');
      const stream = createReadStream(filePath);

      stream.on('data', (chunk) => {
        hash.update(chunk);
      });

      stream.on('end', () => {
        resolve(hash.digest('hex'));
      });

      stream.on('error', (error) => {
        reject(error);
      });
    });
  }

  /**
   * 验证分片列表的完整性
   */
  async verifyChunkList(
    uploadDir: string,
    expectedChunks: number[]
  ): Promise<{
    valid: boolean;
    missing: number[];
    size: number;
  }> {
    const fs = require('fs');
    const missing: number[] = [];
    let totalSize = 0;

    for (const index of expectedChunks) {
      const chunkPath = `${uploadDir}/chunk_${index.toString().padStart(6, '0')}`;

      if (!fs.existsSync(chunkPath)) {
        missing.push(index);
      } else {
        const stat = fs.statSync(chunkPath);
        totalSize += stat.size;
      }
    }

    return {
      valid: missing.length === 0,
      missing,
      size: totalSize
    };
  }

  /**
   * 快速校验（仅检查大小）
   */
  verifyBySize(filePath: string, expectedSize: number): boolean {
    const fs = require('fs');
    const stat = fs.statSync(filePath);
    return stat.size === expectedSize;
  }

  /**
   * CRC32校验（比MD5更快）
   */
  async calculateFileCrc32(filePath: string): Promise<string> {
    const crc32Table = this.getCrc32Table();
    let crc = 0xFFFFFFFF;

    return new Promise((resolve, reject) => {
      const stream = createReadStream(filePath);

      stream.on('data', (chunk: Buffer) => {
        for (let i = 0; i < chunk.length; i++) {
          crc = crc32Table[(crc ^ chunk[i]) & 0xFF] ^ (crc >>> 8);
        }
      });

      stream.on('end', () => {
        crc = (crc ^ 0xFFFFFFFF) >>> 0;
        resolve(crc.toString(16).padStart(8, '0'));
      });

      stream.on('error', reject);
    });
  }

  /**
   * 生成CRC32查找表
   */
  private getCrc32Table(): number[] {
    const table: number[] = [];
    for (let i = 0; i < 256; i++) {
      let c = i;
      for (let j = 0; j < 8; j++) {
        c = (c & 1) ? (0xEDB88320 ^ (c >>> 1)) : (c >>> 1);
      }
      table[i] = c;
    }
    return table;
  }
}
```

```typescript
   * 初始化存储节点

    // 实际应用中应该从数据库或配置中心获取
    const nodeConfigs = this.configService.get<StorageNode[]>('storage.nodes') || [];

   * 选择最佳存储节点
   * 使用加权随机算法，考虑节点负载和剩余空间
   */

    const activeNodes = this.nodes.filter(n => n.status === 'active');

    if (activeNodes.length === 0) {
      throw new Error('没有可用的存储节点');
    }

    // 计算可用空间
    const availableNodes = activeNodes.filter(n => (n.total - n.used) > fileSize);

    if (availableNodes.length === 0) {
      throw new Error('所有节点空间不足');
    }

    // 加权随机选择（可用空间越大，权重越高）
    const totalWeight = availableNodes.reduce((sum, n) => sum + (n.total - n.used), 0);
    let random = Math.random() * totalWeight;

    for (const node of availableNodes) {
      random -= (node.total - node.used);
      if (random <= 0) {
        return node;
      }
    }

    // 兜底返回第一个节点
    return availableNodes[0];
  }
```

### 8.2 分片路由策略

分片路由策略决定了如何将分片分配到不同的存储节点。

```typescript
// src/upload/sharding-router.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { DistributedStorageService } from './distributed-storage.service';

interface ShardingStrategy {
  name: string;
  selectNode: (fileHash: string, chunkIndex: number, totalChunks: number, nodes: any[]) => any;
}

/**
 * 分片路由服务
 * 实现多种分片策略
 */
@Injectable()
export class ShardingRouterService {
  private readonly logger = new Logger(ShardingRouterService.name);

  // 预定义的路由策略
  private strategies: Map<string, ShardingStrategy> = new Map();

  constructor(private storageService: DistributedStorageService) {
    this.registerDefaultStrategies();
  }

  /**
   * 注册默认策略
   */
  private registerDefaultStrategies() {
    // 轮询策略
    this.strategies.set('round-robin', {
      name: '轮询',
      selectNode: (fileHash, chunkIndex, totalChunks, nodes) => {
        const activeNodes = nodes.filter(n => n.status === 'active');
        const index = chunkIndex % activeNodes.length;
        return activeNodes[index];
      }
    });

    // 哈希策略
    this.strategies.set('hash', {
      name: '哈希',
      selectNode: (fileHash, chunkIndex, totalChunks, nodes) => {
        const activeNodes = nodes.filter(n => n.status === 'active');
        const hash = this.hashCode(`${fileHash}_${chunkIndex}`);
        const index = hash % activeNodes.length;
        return activeNodes[index];
      }
    });

    // 一致性哈希策略
    this.strategies.set('consistent-hash', {
      name: '一致性哈希',
      selectNode: (fileHash, chunkIndex, totalChunks, nodes) => {
        // 简化实现，实际应用应使用环形哈希
        const activeNodes = nodes.filter(n => n.status === 'active');
        const hash = this.hashCode(fileHash);
        const index = Math.abs(hash) % activeNodes.length;
        return activeNodes[index];
      }
    });

    // 多副本策略
    this.strategies.set('replication', {
      name: '多副本',
      selectNode: (fileHash, chunkIndex, totalChunks, nodes) => {
        // 返回多个节点用于副本存储
        return null; // 需要特殊处理
      }
    });

    // 容量优先策略
    this.strategies.set('capacity', {
      name: '容量优先',
      selectNode: (fileHash, chunkIndex, totalChunks, nodes) => {
        const activeNodes = nodes.filter(n => n.status === 'active' && (n.total - n.used) > 0);
        // 选择可用空间最多的节点
        return activeNodes.sort((a, b) => (b.total - b.used) - (a.total - a.used))[0];
      }
    });
  }

  /**
   * 选择分片存储节点
   */
  selectNodeForChunk(
    fileHash: string,
    chunkIndex: number,
    strategyName = 'hash'
  ): string {
    const strategy = this.strategies.get(strategyName);
    if (!strategy) {
      throw new Error(`未知策略: ${strategyName}`);
    }

    const nodes = this.storageService.getAllNodesStatus();
    const selectedNode = strategy.selectNode(fileHash, chunkIndex, 0, nodes);

    if (!selectedNode) {
      throw new Error('没有可用的存储节点');
    }

    this.logger.debug(`分片 ${chunkIndex} 使用策略 ${strategyName} 选择节点 ${selectedNode.id}`);

    return selectedNode.id;
  }

  /**
   * 获取文件所有分片的位置
   */
  async getChunkLocations(
    fileHash: string,
    totalChunks: number,
    strategyName = 'hash'
  ): Promise<Array<{ index: number; nodeId: string }>> {
    const locations: Array<{ index: number; nodeId: string }> = [];

    for (let i = 0; i < totalChunks; i++) {
      const nodeId = this.selectNodeForChunk(fileHash, i, strategyName);
      locations.push({ index: i, nodeId });
    }

    return locations;
  }

  /**
   * 简单的哈希函数
   */
  private hashCode(str: string): number {
    let hash = 0;
    for (let i = 0; i < str.length; i++) {
      const char = str.charCodeAt(i);
      hash = ((hash << 5) - hash) + char;
      hash = hash & hash;
    }
    return hash;
  }

  /**
   * 注册自定义策略
   */
  registerStrategy(name: string, strategy: ShardingStrategy) {
    this.strategies.set(name, strategy);
    this.logger.log(`已注册分片策略: ${name}`);
  }

  /**
   * 获取所有可用策略
   */
  getAvailableStrategies(): string[] {
    return Array.from(this.strategies.keys());
  }
}
```

## 九、上传优化

### 9.1 压缩优化

在分片上传前对数据进行压缩，可以减少传输量，但会增加CPU开销。以下是gzip压缩的实现：

```javascript
/**
 * 压缩上传管理器
 * 在上传前对分片进行gzip压缩
 */
class CompressedUploader {
  constructor(options = {}) {
    this.compressionLevel = options.compressionLevel || 6; // 1-9
    this.minSizeForCompression = options.minSizeForCompression || 1024; // 1KB以上才压缩
    this.onProgress = options.onProgress || (() => {});
  }

  /**
   * 压缩数据
   * @param {Blob} data - 要压缩的数据
   * @returns {Promise<Blob>} 压缩后的数据
   */
  async compress(data) {
    // 检查数据大小
    if (data.size < this.minSizeForCompression) {
      return data;
    }

    // 使用Compression Streams API（现代浏览器）
    if ('CompressionStream' in window) {
      return this.compressWithStreams(data);
    }

    // 回退到pako库（需要引入）
    return this.compressWithPako(data);
  }

  /**
   * 使用Compression Streams API压缩
   */
  async compressWithStreams(data) {
    const cs = new CompressionStream('gzip');
    const writer = cs.writable.getWriter();
    writer.write(data);
    writer.close();

    const reader = cs.readable.getReader();
    const chunks = [];

    while (true) {
      const { done, value } = await reader.read();
      if (done) break;
      chunks.push(value);
    }

    return new Blob(chunks, { type: 'application/gzip' });
  }

  /**
   * 使用pako库压缩
   */
  async compressWithPako(data) {
    const pako = await import('https://cdn.jsdelivr.net/npm/pako@2.1.0/dist/pako.min.js');

    const arrayBuffer = await data.arrayBuffer();
    const uint8Array = new Uint8Array(arrayBuffer);
    const compressed = pako.gzip(uint8Array, { level: this.compressionLevel });

    return new Blob([compressed], { type: 'application/gzip' });
  }

  /**
   * 压缩后上传
   */
  async uploadWithCompression(file, uploadConfig) {
    // 创建分片
    const chunkSize = 5 * 1024 * 1024; // 5MB
    const chunks = [];
    let start = 0;
    let index = 0;

    while (start < file.size) {
      const end = Math.min(start + chunkSize, file.size);
      const chunk = file.slice(start, end);
      chunks.push({ index, blob: chunk });
      start = end;
      index++;
    }

    // 逐个压缩并上传
    const totalChunks = chunks.length;

    for (let i = 0; i < chunks.length; i++) {
      this.onProgress({ stage: 'compressing', progress: Math.round((i / totalChunks) * 50) });

      const compressed = await this.compress(chunks[i].blob);
      console.log(`分片${i}压缩率: ${(compressed.size / chunks[i].blob.size * 100).toFixed(1)}%`);

      this.onProgress({ stage: 'uploading', progress: 50 + Math.round((i / totalChunks) * 50) });

      // 上传压缩后的分片
      await uploadConfig.uploadFunc(compressed, chunks[i].index, file.size);
    }
  }
}
```

### 9.2 CDN加速

CDN（内容分发网络）可以将上传流量分散到全球各地的边缘节点，减轻源站压力并提升上传速度。

```typescript
// CDN上传配置示例
interface CDNUploadConfig {
  provider: 'aliyun' | 'aws' | 'cloudflare' | 'custom';
  bucket: string;
  region: string;
  accessKey: string;

/**
 * CDN直传管理器

 */
class CDNUploadManager {
  constructor(private config: CDNUploadConfig) {}

  /**
   * 获取预签名上传URL
   * @param fileName - 文件名
   * @param fileHash - 文件哈希
   * @param contentType - 内容类型
   * @returns 上传URL和headers
   */
  async getPresignedUrl(
    fileName: string,
    fileHash: string,
    contentType: string
  ): Promise<{
    uploadUrl: string;
    headers: Record<string, string>;
  }> {
    // 向后端请求预签名URL
    const response = await fetch('/api/upload/presigned', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },

      })

   * 直传到CDN
   */

    const { uploadUrl, headers } = await this.getPresignedUrl(
      file.name,
      await FileMD5Calculator.calculate(file),
      file.type
    );

    // 2. 直接上传到CDN
    const response = await fetch(uploadUrl, {
      method: 'PUT',
      headers: {
        ...headers,
        'Content-Type': file.type
      },
      body: file
    });

    if (!response.ok) {
      throw new Error('CDN上传失败');
    }

    return {
      success: true,
      url: uploadUrl.split('?')[0] // 返回不带签名的URL
    };
  }

  /**
   * 分片直传到CDN
   */
  async chunkedDirectUpload(file) {
    const chunkSize = 5 * 1024 * 1024;
    const chunks = [];
    let start = 0;
    let index = 0;

    while (start < file.size) {
      const end = Math.min(start + chunkSize, file.size);
      chunks.push({ index, blob: file.slice(start, end) });
      start = end;
      index++;
    }

    // 并发上传分片到CDN
    const limiter = new ConcurrentLimiter(3);

    await Promise.all(
      chunks.map(chunk =>
        limiter.run(async () => {
          const { uploadUrl, headers } = await this.getPresignedUrl(
            `${file.name}.part${chunk.index}`,
            `${file.name}_${chunk.index}`,
            file.type
          );

          await fetch(uploadUrl, {
            method: 'PUT',
            headers,
            body: chunk.blob
          });
        })
      )
    );

    // 通知服务器合并分片
    await fetch('/api/upload/complete-multipart', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        fileName: file.name,
        totalChunks: chunks.length
      })
    });
  }
}
```

### 10.1 重试机制

健壮的重试机制是确保大文件上传可靠性的关键。

```javascript
/**
 * 智能重试器
 * 支持指数退避、熔断器模式
 */
class SmartRetry {
  constructor(options = {}) {
    this.maxRetries = options.maxRetries || 5;
    this.baseDelay = options.baseDelay || 1000;    // 基础延迟（毫秒）
    this.maxDelay = options.maxDelay || 30000;     // 最大延迟（毫秒）
    this.backoffMultiplier = options.backoffMultiplier || 2;
    this.jitter = options.jitter || true;          // 添加随机抖动

    // 熔断器状态
    this.failureCount = 0;
    this.failureThreshold = 5;
    this.circuitOpen = false;
    this.circuitOpenedAt = null;
    this.circuitResetTimeout = 60000; // 1分钟后尝试恢复
  }

  /**
   * 执行带重试的操作
   */
  async execute(task, context = '') {
    let lastError;

    for (let attempt = 0; attempt <= this.maxRetries; attempt++) {
      try {
        // 检查熔断器
        if (this.circuitOpen) {
          if (Date.now() - this.circuitOpenedAt > this.circuitResetTimeout) {
            this.tryResetCircuit();
          } else {
            throw new Error('Circuit breaker is open');
          }
        }

        // 执行任务
        return await task();

      } catch (error) {
        lastError = error;
        this.failureCount++;

        // 检查是否应该打开熔断器
        if (this.failureCount >= this.failureThreshold) {
          this.openCircuit();
        }

        // 如果还有重试次数，等待后重试
        if (attempt < this.maxRetries) {
          const delay = this.calculateDelay(attempt);
          console.log(`${context} 第${attempt + 1}次尝试失败，${delay}ms后重试...`);
          await this.sleep(delay);
        }
      }
    }

    throw lastError;
  }

  /**
   * 计算延迟时间
   */
  calculateDelay(attempt) {
    // 指数退避
    let delay = this.baseDelay * Math.pow(this.backoffMultiplier, attempt);
    delay = Math.min(delay, this.maxDelay);

    // 添加随机抖动（避免惊群效应）
    if (this.jitter) {
      delay = delay * (0.5 + Math.random() * 0.5);
    }

    return Math.round(delay);
  }

  /**
   * 打开熔断器
   */
  openCircuit() {
    this.circuitOpen = true;
    this.circuitOpenedAt = Date.now();
    console.log('🔴 熔断器已打开');
  }

  /**
   * 尝试重置熔断器
   */
  tryResetCircuit() {
    this.circuitOpen = false;
    this.failureCount = 0;
    console.log('🟢 熔断器已重置');
  }

  /**
   * 睡眠
   */
  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

 * 带重试的分片上传
 */
```

### 10.4 容错设计

完整的容错设计需要考虑各种异常场景。

```javascript
/**
 * 容错上传管理器
 * 完整的错误处理和恢复机制
 */
class FaultTolerantUploader {
  constructor(options = {}) {
    this.maxRetries = options.maxRetries || 3;
    this.chunkTimeout = options.chunkTimeout || 60000;
    this.enableRecovery = options.enableRecovery !== false;
    this.autoRecovery = options.autoRecovery || false;

    // 状态
    this.status = 'idle';
    this.chunkStates = new Map();  // chunkIndex -> { status, retries, error }
    this.networkErrors = 0;
    this.serverErrors = 0;
  }

  /**
   * 上传文件（带完整容错）
   */
  async upload(file) {
    this.status = 'uploading';
    this.resetCounters();

    try {
      // 初始化
      const { fileHash, chunks } = await this.initializeUpload(file);

      // 上传所有分片
      await this.uploadAllChunks(chunks, fileHash);

      // 合并
      await this.mergeChunks(fileHash);

      this.status = 'completed';
      return { success: true, fileHash };

    } catch (error) {
      this.status = 'failed';
      return this.handleFailure(error);
    }
  }

  /**
   * 初始化上传
   */
  async initializeUpload(file) {
    // 计算文件哈希
    const fileHash = await FileMD5Calculator.calculate(file);

    // 检查是否已存在（秒传）
    const instantResult = await this.checkInstantUpload(fileHash);
    if (instantResult.instant) {
      this.status = 'completed';
      return { instant: true, ...instantResult };
    }

    // 创建分片
    const chunks = this.createChunks(file);

    // 初始化分片状态
    for (const chunk of chunks) {
      this.chunkStates.set(chunk.index, {
        status: 'pending',
        retries: 0,
        error: null
      });
    }

    return { fileHash, chunks };
  }

  /**
   * 上传所有分片
   */
  async uploadAllChunks(chunks, fileHash) {
    const limiter = new ConcurrentLimiter(3);
    const results = await Promise.allSettled(
      chunks.map(chunk =>
        limiter.run(() => this.uploadChunkWithFaultTolerance(chunk, fileHash))
      )
    );

    // 检查是否有失败
    const failures = results.filter(r => r.status === 'rejected');
    if (failures.length > 0) {
      throw new Error(`${failures.length} 个分片上传失败`);
    }
  }

  /**
   * 带容错的分片上传
   */
  async uploadChunkWithFaultTolerance(chunk, fileHash) {
    const state = this.chunkStates.get(chunk.index);

    while (state.retries <= this.maxRetries) {
      try {
        // 带超时上传
        await this.uploadChunkWithTimeout(chunk, fileHash);
        state.status = 'completed';
        return;

      } catch (error) {
        state.retries++;
        state.error = error.message;

        if (this.isNetworkError(error)) {
          this.networkErrors++;
          // 网络错误，使用指数退避重试
          const delay = Math.min(1000 * Math.pow(2, state.retries), 10000);
          await this.sleep(delay);
        } else if (this.isServerError(error)) {
          this.serverErrors++;
          // 服务器错误，减少并发重试
          await this.sleep(2000);
        } else {
          // 客户端错误，不重试
          throw error;
        }
      }
    }

    throw new Error(`分片 ${chunk.index} 上传失败，已重试 ${this.maxRetries} 次`);
  }

  /**
   * 带超时的分片上传
   */
  async uploadChunkWithTimeout(chunk, fileHash) {
    return new Promise((resolve, reject) => {
      const timeoutId = setTimeout(() => {
        reject(new Error('上传超时'));
      }, this.chunkTimeout);

      this.uploadChunk(chunk, fileHash)
        .then(resolve)
        .catch(reject)
        .finally(() => clearTimeout(timeoutId));
    });
  }

  /**
   * 处理失败
   */
  handleFailure(error) {
    const failedChunks = [];
    this.chunkStates.forEach((state, index) => {
      if (state.status !== 'completed') {
        failedChunks.push({ index, error: state.error });
      }
    });

    return {
      success: false,
      error: error.message,
      failedChunks,
      networkErrors: this.networkErrors,
      serverErrors: this.serverErrors,
      recoverable: this.networkErrors > 0 && this.serverErrors < 10
    };
  }

  /**
   * 判断是否为网络错误
   */
  isNetworkError(error) {
    return error.message.includes('timeout') ||
           error.message.includes('network') ||
           error.name === 'AbortError';
  }

  /**
   * 判断是否为服务器错误
   */
  isServerError(error) {
    return error.message.includes('HTTP 5');
  }

  /**
   * 重置计数器
   */
  resetCounters() {
    this.networkErrors = 0;
    this.serverErrors = 0;
    this.chunkStates.clear();
  }

  /**
   * 睡眠
   */
  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

```javascript
          uploadId,

          fileId: instantResult.fileId,
          fileUrl: instantResult.fileUrl
        });
        return uploadId;
      }

      // 上传阶段
      task.status = 'uploading';
      await this.uploadChunks(task);

      // 合并阶段
      task.status = 'merging';
      const mergeResult = await this.mergeChunks(task);

      // 完成
      task.status = 'completed';
      task.progress = 100;
      this.onComplete({
        uploadId,
        fileId: mergeResult.fileId,
        fileUrl: mergeResult.fileUrl,
        fileHash: task.fileHash
      });

    } catch (error) {
      task.status = 'failed';
      task.error = error.message;
      this.onError({ uploadId, error: error.message });
    }

    this.onQueueUpdate(this.getQueueInfo());
    return uploadId;
  }

  /**
   * 准备上传
   */
  async prepareUpload(task) {
    // 计算文件MD5
    task.fileHash = await this.calculateFileHash(task.file, (progress) => {
      this.updateTaskProgress(task.id, { hashProgress: progress });
    });

    // 创建分片
    task.chunks = this.createChunks(task.file);

    console.log(`准备完成: ${task.file.name}, ${task.chunks.length} 个分片, MD5: ${task.fileHash}`);
  }

  /**
   * 计算文件MD5
   */
  async calculateFileHash(file, onProgress) {
    return FileMD5Calculator.calculate(file, onProgress);
  }

  /**
   * 创建分片
   */
  createChunks(file) {
    const chunks = [];
    const totalSize = file.size;
    let start = 0;
    let index = 0;

    while (start < totalSize) {
      const end = Math.min(start + this.chunkSize, totalSize);
      chunks.push({
        index,
        start,
        end,
        size: end - start,
        blob: file.slice(start, end),
        status: 'pending',
        retries: 0
      });
      start = end;
      index++;
    }

    return chunks;
  }

  /**
   * 秒传检测
   */
  async checkInstantUpload(task) {
    try {
      const response = await fetch(`${this.apiBase}/upload/check`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
          fileHash: task.fileHash,
          fileName: task.file.name,
          fileSize: task.file.size,
          folderId: task.folderId
        })
      });

      return await response.json();
    } catch (error) {
      console.error('秒传检测失败:', error);
      return { success: false, instant: false };
    }
  }

  /**
   * 上传分片
   */
  async uploadChunks(task) {
    // 查询已上传的分片
    const status = await this.queryUploadStatus(task.fileHash);
    const uploadedSet = new Set(status.uploadedChunks || []);

    // 过滤未上传的分片
    const pendingChunks = task.chunks.filter(c => !uploadedSet.has(c.index));

    console.log(`已上传 ${uploadedSet.size}/${task.chunks.length} 个分片`);

    // 并发上传
    const limiter = new ConcurrentLimiter(this.concurrentUploads);

    await Promise.all(
      pendingChunks.map(chunk =>
        limiter.run(() => this.uploadChunkWithRetry(task, chunk))
      )
    );
  }

  /**
   * 查询上传状态
   */
  async queryUploadStatus(fileHash) {
    try {
      const response = await fetch(`${this.apiBase}/upload/status/${fileHash}`);
      if (response.ok) {
        return await response.json();
      }
    } catch (error) {
      console.warn('查询上传状态失败:', error);
    }
    return { uploadedChunks: [] };
  }

  /**
   * 带重试的分片上传
   */
  async uploadChunkWithRetry(task, chunk, retries = 0) {
    try {
      await this.uploadSingleChunk(task, chunk);
      chunk.status = 'completed';
      this.updateTaskProgress(task.id, {});
    } catch (error) {
      if (retries < this.maxRetries) {
        chunk.retries++;
        console.log(`分片 ${chunk.index} 失败，${retries + 1}/${this.maxRetries} 次重试...`);
        await this.sleep(1000 * Math.pow(2, retries));
        return this.uploadChunkWithRetry(task, chunk, retries + 1);
      }
      chunk.status = 'failed';
      throw error;
    }
  }

  /**
   * 上传单个分片
   */
  async uploadSingleChunk(task, chunk) {
    const formData = new FormData();
    formData.append('chunk', chunk.blob);
    formData.append('index', chunk.index.toString());
    formData.append('fileHash', task.fileHash);
    formData.append('fileName', task.file.name);
    formData.append('totalChunks', task.chunks.length.toString());
    formData.append('folderId', task.folderId);

    const response = await fetch(`${this.apiBase}/upload/chunk`, {
      method: 'POST',
      body: formData
    });

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}`);
    }

    return await response.json();
  }

  /**
   * 更新任务进度
   */
  updateTaskProgress(uploadId, updates) {
    const task = this.uploads.get(uploadId);
    if (!task) return;

    const completedChunks = task.chunks.filter(c => c.status === 'completed').length;
    task.progress = Math.round((completedChunks / task.chunks.length) * 100);
    task.uploadedBytes = task.chunks.slice(0, completedChunks).reduce((sum, c) => sum + c.size, 0);

    const elapsed = (Date.now() - task.startTime) / 1000;
    task.speed = task.uploadedBytes / elapsed;

    this.onProgress({
      uploadId,
      progress: task.progress,
      uploadedBytes: task.uploadedBytes,
      totalBytes: task.file.size,
      speed: task.speed,
      speedReadable: this.formatSpeed(task.speed)
    });
  }

  /**
   * 合并分片
   */
  async mergeChunks(task) {
    const response = await fetch(`${this.apiBase}/upload/merge`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        fileHash: task.fileHash,
        fileName: task.file.name,
        fileSize: task.file.size,
        totalChunks: task.chunks.length,
        folderId: task.folderId
      })
    });

    if (!response.ok) {
      throw new Error('合并失败');
    }

    return await response.json();
  }

  /**
   * 获取队列信息
   */
  getQueueInfo() {
    const uploads = [];
    this.uploads.forEach((task, id) => {
      uploads.push({
        id,
        name: task.file.name,
        size: task.file.size,
        progress: task.progress,
        status: task.status,
        error: task.error
      });
    });

    return {
      total: uploads.length,
      active: this.activeUploads,
      uploads
    };
  }

  /**
   * 格式化速度
   */
  formatSpeed(bytesPerSecond) {
    if (bytesPerSecond < 1024) return `${bytesPerSecond.toFixed(0)} B/s`;
    if (bytesPerSecond < 1024 * 1024) return `${(bytesPerSecond / 1024).toFixed(1)} KB/s`;
    return `${(bytesPerSecond / 1024 / 1024).toFixed(1)} MB/s`;
  }

  /**
   * 生成唯一ID
   */
  generateId() {
    return `upload_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
  }

  /**
   * 睡眠
   */
  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```

```typescript
// src/cloud-disk/cloud-disk.service.ts
import { Injectable, Logger } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository } from 'typeorm';
import { FileRecord } from './entities/file-record.entity';
import { UploadRecord } from './entities/upload-record.entity';
import { existsSync, mkdirSync, readdirSync, unlinkSync, createWriteStream, statSync } from 'fs';
import { join } from 'path';
import { createHash } from 'crypto';

@Injectable()
export class CloudDiskService {
  private readonly logger = new Logger(CloudDiskService.name);
  private readonly tempDir = './uploads/temp';
  private readonly storageDir = './uploads/storage';

  constructor(
    @InjectRepository(FileRecord)
    private fileRepository: Repository<FileRecord>,
    @InjectRepository(UploadRecord)
    private uploadRepository: Repository<UploadRecord>
  ) {
    this.ensureDirectories();
  }

  private ensureDirectories() {
    [this.tempDir, this.storageDir].forEach(dir => {
      if (!existsSync(dir)) {
        mkdirSync(dir, { recursive: true });
      }
    });
  }

  /**
   * 检查秒传
   */
  async checkInstantUpload(fileHash: string, fileName: string, fileSize: number, folderId: string) {
    // 查找已存在的文件
    const existingFile = await this.fileRepository.findOne({
      where: { fileHash, isDeleted: false }
    });

    if (existingFile) {
      return {
        success: true,
        instant: true,
        fileId: existingFile.id,
        fileUrl: existingFile.url
      };
    }

    // 检查是否有未完成的上传
    const pendingUpload = await this.uploadRepository.findOne({
      where: { fileHash, status: 'uploading' }
    });

    if (pendingUpload) {
      return {
        success: true,
        instant: false,
        needUpload: true,
        uploadedChunks: pendingUpload.uploadedChunks
      };
    }

    return {
      success: true,
      instant: false,
      needUpload: true
    };
  }

  /**
   * 保存分片
   */
  async saveChunk(file: Express.Multer.File, fileHash: string, chunkIndex: number, totalChunks: number, fileName: string) {
    // 查找或创建上传记录
    let record = await this.uploadRepository.findOne({ where: { fileHash } });

    if (!record) {
      record = this.uploadRepository.create({
        fileHash,
        fileName,
        totalChunks,
        uploadedChunks: [],
        status: 'uploading'
      });
      await this.uploadRepository.save(record);
    }

    // 更新已上传分片
    if (!record.uploadedChunks.includes(chunkIndex)) {
      record.uploadedChunks.push(chunkIndex);
      record.uploadedChunks.sort((a, b) => a - b);
      await this.uploadRepository.save(record);
    }

    // 检查是否全部完成
    const isComplete = record.uploadedChunks.length === totalChunks;

    return {
      success: true,
      chunkIndex,
      uploadedChunks: record.uploadedChunks,
      isComplete,
      progress: Math.round((record.uploadedChunks.length / totalChunks) * 100)
    };
  }

  /**
   * 获取上传状态
   */
  async getUploadStatus(fileHash: string) {
    const record = await this.uploadRepository.findOne({
      where: { fileHash }
    });

    if (!record) {
      return {
        uploadedChunks: [],
        totalChunks: 0,
        isComplete: false
      };
    }

    return {
      uploadedChunks: record.uploadedChunks,
      totalChunks: record.totalChunks,
      isComplete: record.uploadedChunks.length === record.totalChunks
    };
  }

  /**
   * 合并分片
   */
  async mergeChunks(fileHash: string, fileName: string, totalChunks: number, folderId: string) {
    const uploadDir = join(this.tempDir, fileHash);
    const record = await this.uploadRepository.findOne({ where: { fileHash } });

    if (!record) {
      throw new Error('上传记录不存在');
    }

    // 验证分片完整性
    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);
      if (!existsSync(chunkPath)) {
        throw new Error(`分片 ${i} 不存在`);
      }
    }

    // 创建最终文件
    const ext = fileName.substring(fileName.lastIndexOf('.'));
    const finalFileName = `${fileHash}${ext}`;
    const finalPath = join(this.storageDir, finalFileName);

    // 流式合并
    const writeStream = createWriteStream(finalPath);
    const md5Hash = createHash('md5');

    for (let i = 0; i < totalChunks; i++) {
      const chunkPath = join(uploadDir, `chunk_${i.toString().padStart(6, '0')}`);
      const chunkData = await this.readFile(chunkPath);

      writeStream.write(chunkData);
      md5Hash.update(chunkData);
    }

    writeStream.end();
    await new Promise<void>((resolve, reject) => {
      writeStream.on('finish', resolve);
      writeStream.on('error', reject);
    });

    const finalMd5 = md5Hash.digest('hex');
    const fileSize = statSync(finalPath).size;

    // 保存文件记录
    const fileRecord = this.fileRepository.create({
      fileName,
      fileHash,
      fileMd5: finalMd5,
      fileSize,
      url: `/files/${finalFileName}`,
      folderId,
      mimeType: this.getMimeType(ext)
    });
    await this.fileRepository.save(fileRecord);

    // 更新上传记录状态
    record.status = 'completed';
    await this.uploadRepository.save(record);

    // 清理临时文件
    await this.cleanupTempFiles(fileHash);

    return {
      success: true,
      fileId: fileRecord.id,
      fileUrl: fileRecord.url,
      fileMd5: finalMd5
    };
  }

  private async readFile(path: string): Promise<Buffer> {
    return new Promise((resolve, reject) => {
      const fs = require('fs');
      fs.readFile(path, (err, data) => {
        if (err) reject(err);
        else resolve(data);
      });
    });
  }

  private async cleanupTempFiles(fileHash: string) {
    const uploadDir = join(this.tempDir, fileHash);
    try {
      const files = readdirSync(uploadDir);
      for (const file of files) {
        unlinkSync(join(uploadDir, file));
      }
      require('fs').rmdirSync(uploadDir);
    } catch (error) {
      this.logger.error('清理临时文件失败:', error);
    }
  }

  private getMimeType(ext: string): string {
    const mimeTypes: Record<string, string> = {
      '.jpg': 'image/jpeg',
      '.jpeg': 'image/jpeg',
      '.png': 'image/png',
      '.gif': 'image/gif',
      '.pdf': 'application/pdf',
      '.mp4': 'video/mp4',
      '.zip': 'application/zip'
    };
    return mimeTypes[ext.toLowerCase()] || 'application/octet-stream';
  }
}
```

### 11.5 我的思考：大文件上传的挑战

在设计和实现大文件上传系统的过程中，我深刻认识到了其中的技术挑战和权衡：

**第一个挑战是上传的可靠性。** 网络中断是不可避免的，特别是对于大文件来说，上传时间可能长达数小时。用户可能在地铁里、电梯中或者网络不稳定的区域。因此，断点续传不是锦上添花的功能，而是必须具备的核心能力。在实现中，我意识到状态管理是断点续传的关键——需要同时在客户端和服务器端维护上传状态，并且要处理各种边界情况，比如服务器端的临时文件被清理、客户端刷新页面等。

**第二个挑战是用户体验的平衡。** 分片越大，HTTP请求数量越少，服务器处理开销越小，但重试成本越高。反之，分片越小，重试成本低，但请求数量多，服务器压力大。我倾向于选择5MB作为默认值，这是一个在大多数场景下都比较合理的折中。同时，并发数量的选择也需要权衡——用户可能同时上传多个文件，每个文件都开满并发会导致带宽被占满，页面响应变慢。因此，我实现了自适应并发控制，让系统能够根据实际情况动态调整。

**第三个挑战是安全性。** 上传接口容易被滥用，特别是未授权的上传。我采用了预签名URL的方案，让客户端直接从云存储获取临时凭证再上传，这样即使有人抓包也只能获取临时的、有时效的凭证，无法直接访问存储服务。同时，文件类型验证、文件大小限制、MD5校验等都是必要的安全措施。

**第四个挑战是存储成本和性能的权衡。** 对于云盘这种海量存储场景，本地存储很快就会成为瓶颈。分布式存储是必然的选择，但如何设计分片路由策略是一个复杂的问题。一致性哈希可以保证相同的分片总是路由到相同的节点，便于管理，但节点故障时需要迁移大量数据。我倾向于在初期使用简单的哈希策略，在规模扩大后再引入更复杂的方案。

**第五个挑战是完整性校验。** 分片合并后，必须验证文件的完整性。MD5是最常用的方案，但计算MD5需要读取整个文件，对于超大文件来说这本身就是一个耗时操作。在实践中，我采用了流式MD5计算，并放在合并过程中一并完成，避免额外的IO开销。

```javascript
// upload-list.component.js
class UploadListUI {
  constructor(containerId, uploader) {
    this.container = document.getElementById(containerId);
    this.uploader = uploader;
    this.uploads = new Map();

    // 监听上传事件
    this.uploader.onProgress = this.handleProgress.bind(this);
    this.uploader.onComplete = this.handleComplete.bind(this);
    this.uploader.onError = this.handleError.bind(this);
    this.uploader.onQueueUpdate = this.handleQueueUpdate.bind(this);

    // 绑定按钮事件
    this.bindEvents();
  }

  /**
   * 绑定按钮事件
   */
  bindEvents() {
    // 监听取消上传事件
    window.addEventListener('cancelUpload', (e) => {
      const uploadId = e.detail.uploadId;
      this.cancelUpload(uploadId);
    });

    // 监听重试上传事件
    window.addEventListener('retryUpload', (e) => {
      const uploadId = e.detail.uploadId;
      this.retryUpload(uploadId);
    });
  }

  /**
   * 添加上传项
   */
  addUploadItem(uploadId, file) {
    const item = {
      id: uploadId,
      file,
      status: 'pending',
      progress: 0,
      error: null
    };

    this.uploads.set(uploadId, item);
    this.render();

    return item;
  }

  /**
   * 处理进度更新
   */
  handleProgress(data) {
    const item = this.uploads.get(data.uploadId);
    if (item) {
      item.progress = data.progress;
      item.speed = data.speed;
      item.remainingTime = data.remainingTime;
      item.status = 'uploading';
      this.render();
    }
  }

  /**
   * 处理上传完成
   */
  handleComplete(data) {
    const item = this.uploads.get(data.uploadId);
    if (item) {
      item.status = 'completed';
      item.progress = 100;
      this.render();
    }
  }

  /**
   * 处理上传错误
   */
  handleError(data) {
    const item = this.uploads.get(data.uploadId);
    if (item) {
      item.status = 'failed';
      item.error = data.error;
      this.render();
    }
  }

  /**
   * 处理队列更新
   */
  handleQueueUpdate(data) {
    this.updateStats(data);
  }

  /**
   * 取消上传
   */
  cancelUpload(uploadId) {
    const item = this.uploads.get(uploadId);
    if (item && item.status === 'uploading') {
      item.status = 'cancelled';
      this.render();
    }
  }

  /**
   * 重试上传
   */
  retryUpload(uploadId) {
    const item = this.uploads.get(uploadId);
    if (item && item.status === 'failed') {
      item.status = 'pending';
      item.error = null;
      item.progress = 0;
      this.uploader.uploadFile(item.file);
      this.render();
    }
  }

  /**
   * 更新统计信息
   */
  updateStats(data) {
    const pending = data.uploads.filter(u => u.status === 'pending').length;
    const uploading = data.uploads.filter(u => u.status === 'uploading').length;
    const completed = data.uploads.filter(u => u.status === 'completed').length;
    const failed = data.uploads.filter(u => u.status === 'failed').length;

    document.getElementById('pendingCount').textContent = pending;
    document.getElementById('uploadingCount').textContent = uploading;
    document.getElementById('completedCount').textContent = completed;
    document.getElementById('failedCount').textContent = failed;

    // 更新总进度
    const totalProgress = data.uploads.reduce((sum, u) => sum + u.progress, 0);
    const avgProgress = data.uploads.length > 0 ? Math.round(totalProgress / data.uploads.length) : 0;

    document.getElementById('totalProgressFill').style.width = `${avgProgress}%`;
    document.getElementById('totalProgressText').textContent = `${avgProgress}%`;
  }

  /**
   * 渲染上传列表
   */
  render() {
    const listEl = document.getElementById('uploadList');
    if (!listEl) return;

    listEl.innerHTML = Array.from(this.uploads.values())
      .map(item => this.renderItem(item))
      .join('');

    // 绑定新生成的事件
    listEl.querySelectorAll('.cancel-btn').forEach(btn => {
      btn.addEventListener('click', () => {
        window.dispatchEvent(new CustomEvent('cancelUpload', {
          detail: { uploadId: btn.dataset.uploadId }
        }));
      });
    });

    listEl.querySelectorAll('.retry-btn').forEach(btn => {
      btn.addEventListener('click', () => {
        window.dispatchEvent(new CustomEvent('retryUpload', {
          detail: { uploadId: btn.dataset.uploadId }
        }));
      });
    });
  }

  /**
   * 渲染单个上传项
   */
  renderItem(item) {
    const statusClass = `status-${item.status}`;
    const statusLabel = {
      pending: '等待中',
      uploading: '上传中',
      completed: '已完成',
      failed: '上传失败',
      cancelled: '已取消'
    };

    const icon = {
      pending: '⏳',
      uploading: '⬆️',
      completed: '✅',
      failed: '❌',
      cancelled: '🚫'
    };

    return `
      <div class="upload-item ${statusClass}">
        <div class="upload-info">
          <span class="upload-icon">${icon[item.status]}</span>
          <div class="upload-details">
            <div class="upload-name">${item.file.name}</div>
            <div class="upload-meta">
              ${this.formatSize(item.file.size)} · ${statusLabel[item.status]}
              ${item.speed ? ` · ${this.formatSpeed(item.speed)}` : ''}
            </div>
          </div>
          ${item.status === 'uploading' ? `
            <button class="cancel-btn" data-upload-id="${item.id}">取消</button>
          ` : ''}
          ${item.status === 'failed' ? `
            <button class="retry-btn" data-upload-id="${item.id}">重试</button>
          ` : ''}
        </div>
        <div class="upload-progress-bar">
          <div class="upload-progress-fill" style="width: ${item.progress}%"></div>
        </div>
        <div class="upload-stats">
          <span class="upload-percentage">${item.progress}%</span>
          ${item.remainingTime > 0 ? `
            <span class="upload-remaining">剩余 ${this.formatTime(item.remainingTime)}</span>
          ` : ''}
        </div>
        ${item.error ? `<div class="upload-error">${item.error}</div>` : ''}
      </div>
    `;
  }

  formatSize(bytes) {
    if (bytes < 1024) return `${bytes} B`;
    if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`;
    if (bytes < 1024 * 1024 * 1024) return `${(bytes / 1024 / 1024).toFixed(1)} MB`;
    return `${(bytes / 1024 / 1024 / 1024).toFixed(2)} GB`;
  }

  formatSpeed(bytesPerSecond) {
    if (bytesPerSecond < 1024) return `${bytesPerSecond.toFixed(0)} B/s`;
    if (bytesPerSecond < 1024 * 1024) return `${(bytesPerSecond / 1024).toFixed(1)} KB/s`;
    return `${(bytesPerSecond / 1024 / 1024).toFixed(1)} MB/s`;
  }

  formatTime(seconds) {
    if (!seconds || seconds < 0) return '--';
    const hours = Math.floor(seconds / 3600);
    const minutes = Math.floor((seconds % 3600) / 60);
    const secs = Math.floor(seconds % 60);
    if (hours > 0) return `${hours}小时${minutes}分`;
    if (minutes > 0) return `${minutes}分${secs}秒`;
    return `${secs}秒`;
  }
}
```

## 总结

本指南详细介绍了分片上传与断点续传的完整技术体系。从分片原理、前端实现、后端架构、MD5秒传、断点续传、并发控制、分片合并、分布式存储、上传优化到失败处理，覆盖了大文件上传系统的各个方面。

核心要点总结如下：

**分片上传的核心优势在于** 将大文件分割成小块独立上传，可以实现并发传输提高速度，网络中断时只需重传失败的单个分片而不是整个文件，分片较小内存压力可控，可以充分利用带宽。

**秒传机制通过MD5哈希** 在上传前检查文件是否已存在于服务器，对于重复文件的场景可以实现秒级上传，大幅提升用户体验。

**断点续传依赖持久化的上传状态记录** 服务器需要存储每个分片的上传状态，客户端需要保存当前进度，中断后可以从断点继续而不是从头开始。

**并发控制是性能优化的关键** 过多的并发会耗尽带宽和服务器资源，过少则无法充分利用带宽，自适应并发控制是最佳实践。

**分布式存储是生产环境的必备能力** 需要考虑数据分片、路由策略、一致性保证、故障恢复等多个方面。

**健壮的容错机制是可靠性的保障** 包括重试机制、超时处理、状态恢复、熔断器模式等，确保在各种异常情况下系统仍能正常工作。
