# 微服务与前端的协同模式

> 如果把整个系统比作一家大型餐厅，那么微服务就是后厨的不同工位——有做前菜的、有做主菜的、有做甜点的；而前端就是前厅的服务员，他需要协调后厨的各个工位，把顾客（用户）点的菜按照正确的顺序、合适的节奏端上来。BFF模式就像前厅的领班，专门负责和后厨沟通，避免前厅服务员（前端）直接和每个后厨工人（微服务）打交道。本文档将深入探讨前后端在微服务架构下的协同模式，让你的"餐厅"运转得更加流畅。

## 一、微服务架构下的前端挑战

### 1.1 传统前端与微服务的问题

**问题场景：**

```
┌─────────────────────────────────────────────────────────────────┐
│                 传统微服务前端调用模式问题                        │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  前端直接调用多个微服务                                          │
│                                                                 │
│        ┌──────────────────────────────────────────────────┐   │
│        │                   前端应用                         │   │
│        │   ┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐    │   │
│        │   │调用用户│ │调用订单│ │调用商品│ │调用支付│    │   │
│        │   │ 服务   │ │ 服务   │ │ 服务   │ │ 服务   │    │   │
│        │   └────┬───┘ └────┬───┘ └────┬───┘ └────┬───┘    │   │
│        └────────┼──────────┼──────────┼──────────┼─────────┘   │
│                 │          │          │          │             │
│      ┌──────────┴──────────┴──────────┴──────────┴─────────┐   │
│      │                                                        │   │
│      ▼              ▼              ▼              ▼            │   │
│  ┌────────┐    ┌────────┐    ┌────────┐    ┌────────┐         │   │
│  │用户服务│    │订单服务│    │商品服务│    │支付服务│         │   │
│  └────────┘    └────────┘    └────────┘    └────────┘         │
│                                                                 │
│  问题：                                                           │
│  1. 前端需要知道所有微服务的地址                                  │
│  2. 每个服务的接口协议、认证方式可能不同                          │
│  3. 需要处理跨域问题                                              │
│  4. 性能问题：多次往返，后端数据聚合在客户端                       │
│  5. 移动端带宽敏感                                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**具体问题分析：**

| 问题 | 描述 | 影响 |
|------|------|------|
| **接口碎片化** | 前端需要调用十几个接口才能加载一个页面 | 页面加载慢，用户体验差 |
| **协议不一致** | 有的用REST、有的用gRPC、有的用WebSocket | 维护成本高 |
| **认证复杂** | 每个接口可能需要不同的认证token | 安全风险 |
| **跨域问题** | 不同域名的微服务需要CORS配置 | 部署复杂 |
| **数据冗余** | 同样的数据可能需要在多个服务中获取 | 带宽浪费 |
| **版本同步** | 某服务API变更，前端需要同步更新 | 敏捷性降低 |

### 1.2 解决方案：BFF模式

**BFF（Backend For Frontend）模式：**

```

├─────────────────────────────────────────────────────────────────┤
```

## 二、BFF实现模式

### 2.1 NestJS BFF实现

**BFF服务架构：**

```typescript
// BFF网关服务 - NestJS实现
// BFF是前端与微服务之间的胶水层

import { Module } from '@nestjs/common';
import { ClientsModule, Transport } from '@nestjs/microservices';
import { BffController } from './bff.controller';
import { BffService } from './bff.service';
import { UserService } from './services/user.service';
import { OrderService } from './services/order.service';
import { ProductService } from './services/product.service';
import { CacheModule } from './cache.module';

@Module({
  imports: [
    // 注册微服务客户端
    // BFF连接后端的各个微服务
    ClientsModule.register([
      {
        name: 'USERS_SERVICE',
        transport: Transport.TCP,
        options: {
          host: process.env.USERS_SERVICE_HOST || 'localhost',
          port: +process.env.USERS_SERVICE_PORT || 3001,
        },
      },
      {
        name: 'ORDERS_SERVICE',
        transport: Transport.TCP,
        options: {
          host: process.env.ORDERS_SERVICE_HOST || 'localhost',
          port: +process.env.ORDERS_SERVICE_PORT || 3002,
        },
      },
      {
        name: 'PRODUCTS_SERVICE',
        transport: Transport.TCP,
        options: {
          host: process.env.PRODUCTS_SERVICE_HOST || 'localhost',
          port: +process.env.PRODUCTS_SERVICE_PORT || 3003,
        },
      },
    ]),

    // 缓存模块 - Redis缓存
    CacheModule,
  ],
  controllers: [BffController],
  providers: [BffService, UserService, OrderService, ProductService],
})
export class BffModule {}
```

**BFF控制器：**

```typescript
// BFF控制器 - 为不同前端提供定制化的API
// 一个BFF可以服务Web、iOS、Android等多个前端

import {
  Controller,
  Get,
  Post,
  Param,
  Query,
  Body,
  Headers,
  Logger,
  UseGuards,
} from '@nestjs/common';
import { BffService } from './bff.service';
import { AuthGuard } from '../guards/auth.guard';
import { SkipThrottle, Throttle } from '@nestjs/throttler';

// 页面路由控制器
@Controller('api/v1')
export class BffController {
  private readonly logger = new Logger('BffController');

  constructor(private readonly bffService: BffService) {}

  // ============================================================
  // 首页聚合接口 - 一次请求获取首页所需所有数据
  // ============================================================

  @Get('home')
  @UseGuards(AuthGuard)
  @SkipThrottle()  // 首页访问频繁，跳过限流
  async getHomePage(@Headers('x-user-id') userId: string) {
    this.logger.debug(`获取首页数据, userId=${userId}`);

    // 调用BffService获取聚合数据
    // BffService内部会并行调用多个微服务
    // 然后按需合并、裁剪数据返回给前端
    return this.bffService.getHomePageData(userId);
  }

  // ============================================================
  // 用户详情聚合接口
  // ============================================================

  @Get('users/:userId/profile')
  @UseGuards(AuthGuard)
  async getUserProfile(
    @Param('userId') userId: string,
    @Headers('x-user-id') currentUserId: string,
  ) {
    // 获取用户详情
    // 如果是查看自己的资料，返回完整信息
    // 如果是查看他人资料，返回脱敏信息
    const isSelf = userId === currentUserId;
    return this.bffService.getUserProfile(userId, isSelf);
  }

  // ============================================================
  // 订单列表聚合接口
  // ============================================================

  @Get('orders')
  @UseGuards(AuthGuard)
  @Throttle({ default: { limit: 100, ttl: 60000 } })  // 1分钟最多100次
  async getOrders(
    @Headers('x-user-id') userId: string,
    @Query('status') status: string,
    @Query('page') page: number,
    @Query('pageSize') pageSize: number,
  ) {
    // 获取订单列表
    // 包含订单基本信息
    // 同时返回订单关联的商品信息、店铺信息
    return this.bffService.getOrders(userId, { status, page, pageSize });
  }

  // ============================================================
  // 订单详情聚合接口
  // ============================================================

  @Get('orders/:orderId')
  @UseGuards(AuthGuard)
  async getOrderDetail(
    @Param('orderId') orderId: string,
    @Headers('x-user-id') userId: string,
  ) {
    // 获取订单详情
    // 包含：订单信息、商品信息、支付信息、物流信息、操作日志
    return this.bffService.getOrderDetail(orderId, userId);
  }

  // ============================================================
  // 商品详情聚合接口
  // ============================================================

  @Get('products/:productId')
  @UseGuards(AuthGuard)
  @SkipThrottle()
  async getProductDetail(@Param('productId') productId: string) {
    // 获取商品详情
    // 包含：商品基本信息、价格信息、库存信息、评价信息、推荐商品
    return this.bffService.getProductDetail(productId);
  }

  // ============================================================
  // 搜索聚合接口
  // ============================================================

  @Get('search')
  @UseGuards(AuthGuard)
  async search(
    @Query('keyword') keyword: string,
    @Query('category') category: string,
    @Query('sort') sort: string,
    @Query('page') page: number,
    @Query('pageSize') pageSize: number,
  ) {
    // 搜索商品
    // 并行查询商品服务、搜索服务、推荐服务
    // 聚合返回搜索结果
    return this.bffService.search({ keyword, category, sort, page, pageSize });
  }

  // ============================================================
  // 移动端专用接口（数据更精简）
  // ============================================================

  @Get('mobile/home')
  @UseGuards(AuthGuard)
  async getMobileHomePage(@Headers('x-user-id') userId: string) {
    // 移动端首页
    // 相比Web版，返回更少的数据，优化流量
    return this.bffService.getMobileHomePageData(userId);
  }

  // ============================================================
  // 批量接口
  // ============================================================

  @Post('batch')
  @UseGuards(AuthGuard)
  async batchRequest(
    @Body() requests: Array<{ type: string; id: string; params?: any }>,
  ) {
    // 批量请求
    // 前端可以一次请求多个资源
    // BFF并行获取后聚合返回
    return this.bffService.batchGet(requests);
  }
}
```

**BFF服务实现：**

```typescript
// BFF核心服务 - 负责数据聚合、裁剪、缓存

import { Injectable, Inject, Logger, CACHE_KEY } from '@nestjs/common';
import { ClientProxy } from '@nestjs/microservices';
import { Observable, forkJoin, of, from } from 'rxjs';
import { map, catchError, timeout, retry, switchMap } from 'rxjs/operators';
import { RedisService } from '../cache/redis.service';

@Injectable()
export class BffService {
  private readonly logger = new Logger('BffService');

  constructor(
    // 注入微服务客户端
    @Inject('USERS_SERVICE') private readonly usersClient: ClientProxy,
    @Inject('ORDERS_SERVICE') private readonly ordersClient: ClientProxy,
    @Inject('PRODUCTS_SERVICE') private readonly productsClient: ClientProxy,
    // 注入Redis缓存
    private readonly redisService: RedisService,
  ) {}

  // ============================================================
  // 首页数据聚合
  // ============================================================

  async getHomePageData(userId: string): Promise<HomePageData> {
    // 并行调用多个服务获取数据
    // 使用Promise.all并行执行，提高性能
    const [userInfo, recommendations, banners, categories] = await Promise.all([
      // 用户信息
      this.callService<User>(
        this.usersClient,
        { pattern: 'users.findOne', data: { id: userId } },
        3000, // 3秒超时
      ).catch(() => null),

      // 推荐商品
      this.callService<Product[]>(
        this.productsClient,
        { pattern: 'products.recommendations', data: { userId, limit: 20 } },
        5000,
      ).catch(() => []),

      // 首页Banner
      this.callService<Banner[]>(
        this.productsClient,
        { pattern: 'banners.home', data: {} },
        2000,
      ).catch(() => []),

      // 分类信息
      this.callService<Category[]>(
        this.productsClient,
        { pattern: 'categories.all', data: {} },
        2000,
      ).catch(() => []),
    ]);

    // 聚合数据并返回
    // 只返回前端需要的字段，减少数据传输
    return {
      user: userInfo ? this.shrinkUserInfo(userInfo) : null,
      recommendations: recommendations.map((p) => this.shrinkProduct(p)),
      banners: banners.map((b) => ({
        id: b.id,
        imageUrl: b.imageUrl,
        link: b.link,
      })),
      categories: categories.map((c) => ({
        id: c.id,
        name: c.name,
        icon: c.icon,
      })),
      // 附加信息
      timestamp: Date.now(),
    };
  }

  // ============================================================
  // 用户资料聚合
  // ============================================================

  async getUserProfile(userId: string, isSelf: boolean): Promise<UserProfile> {
    // 调用用户服务获取基本信息
    const user = await this.callService<User>(
      this.usersClient,
      { pattern: 'users.findOne', data: { id: userId } },
    );

    // 调用订单服务获取统计信息
    const stats = await this.callService<OrderStats>(
      this.ordersClient,
      { pattern: 'orders.stats', data: { userId } },
    );

    // 根据是否是自己返回不同级别的信息
    const baseInfo = {
      id: user.id,
      name: user.name,
      avatar: user.avatar,
      bio: user.bio,
      level: user.level,
    };

    if (isSelf) {
      // 返回完整信息
      return {
        ...baseInfo,
        email: user.email,         // 邮箱（仅自己可见）
        phone: user.phone,         // 电话（仅自己可见）
        stats: stats,
        settings: user.settings,
      };
    } else {
      // 返回脱敏信息
      return {
        ...baseInfo,
        stats: {
          totalOrders: stats.totalOrders,
          totalSpent: stats.totalSpent,
        },
      };
    }
  }

  // ============================================================
  // 订单列表聚合
  // ============================================================

  async getOrders(
    userId: string,
    params: { status?: string; page: number; pageSize: number },
  ): Promise<PaginatedOrders> {
    // 调用订单服务获取订单列表
    const ordersResponse = await this.callService<OrdersResponse>(
      this.ordersClient,
      {
        pattern: 'orders.findMany',
        data: { userId, ...params },
      },
    );

    // 获取订单关联的商品信息（批量获取）
    const productIds = ordersResponse.orders.flatMap((o) =>
      o.items.map((item) => item.productId),
    );

    // 调用商品服务批量获取商品信息
    const products = await this.callService<Product[]>(
      this.productsClient,
      {
        pattern: 'products.findMany',
        data: { ids: productIds },
      },
    ).catch(() => []);

    // 将商品信息映射到订单
    const productMap = new Map(products.map((p) => [p.id, p]));

    // 聚合订单和商品信息
    const enrichedOrders = ordersResponse.orders.map((order) => ({
      id: order.id,
      status: order.status,
      totalAmount: order.totalAmount,
      createdAt: order.createdAt,
      items: order.items.map((item) => {
        const product = productMap.get(item.productId);
        return {
          productId: item.productId,
          productName: product?.name || '商品已下架',
          productImage: product?.mainImage,
          price: item.price,
          quantity: item.quantity,
        };
      }),
    }));

    return {
      orders: enrichedOrders,
      total: ordersResponse.total,
      page: params.page,
      pageSize: params.pageSize,
    };
  }

  // ============================================================
  // 商品详情聚合
  // ============================================================

  async getProductDetail(productId: string): Promise<ProductDetail> {
    // 先检查缓存
    const cacheKey = `product:${productId}`;
    const cached = await this.redisService.get(cacheKey);
    if (cached) {
      return JSON.parse(cached);
    }

    // 并行调用多个服务
    const [product, inventory, reviews, similarProducts] = await Promise.all([
      // 商品基本信息
      this.callService<Product>(this.productsClient, {
        pattern: 'products.findOne',
        data: { id: productId },
      }),

      // 库存信息
      this.callService<Inventory>(this.productsClient, {
        pattern: 'products.inventory',
        data: { id: productId },
      }),

      // 评价信息
      this.callService<ReviewsResponse>(this.productsClient, {
        pattern: 'products.reviews',
        data: { id: productId, page: 1, pageSize: 5 },
      }),

      // 相似商品推荐
      this.callService<Product[]>(this.productsClient, {
        pattern: 'products.similar',
        data: { id: productId, limit: 10 },
      }),
    ]);

    const result: ProductDetail = {
      // 基本信息
      id: product.id,
      name: product.name,
      description: product.description,
      mainImage: product.mainImage,
      images: product.images,
      price: product.price,
      originalPrice: product.originalPrice,
      discount: product.discount,

      // 库存信息
      stock: inventory.stock,
      hasStock: inventory.stock > 0,

      // 评价摘要
      reviewSummary: {
        total: reviews.total,
        averageScore: reviews.averageScore,
        distribution: reviews.distribution,
      },

      // 推荐商品（只返回必要字段）
      similarProducts: similarProducts.map((p) => ({
        id: p.id,
        name: p.name,
        price: p.price,
        mainImage: p.mainImage,
      })),
    };

    // 写入缓存（5分钟）
    await this.redisService.setex(cacheKey, 300, JSON.stringify(result));

    return result;
  }

  // ============================================================
  // 移动端首页数据（精简版）
  // ============================================================

  async getMobileHomePageData(userId: string): Promise<MobileHomeData> {
    // 相比Web版，减少返回数据量，优化移动端体验
    const [userInfo, recommendations] = await Promise.all([
      this.callService<User>(this.usersClient, {
        pattern: 'users.findOne',
        data: { id: userId },
      }).catch(() => null),

      this.callService<Product[]>(this.productsClient, {
        pattern: 'products.recommendations',
        data: { userId, limit: 10 },  // 只返回10个推荐
      }).catch(() => []),
    ]);

    return {
      user: userInfo ? { id: userInfo.id, name: userInfo.name } : null,
      recommendations: recommendations.map((p) => ({
        id: p.id,
        name: p.name,
        price: p.price,
        image: p.mainImage,
      })),
    };
  }

  // ============================================================
  // 批量请求处理
  // ============================================================

  async batchGet(
    requests: Array<{ type: string; id: string; params?: any }>,
  ): Promise<BatchResult[]> {
    // 使用Promise.all并行处理所有请求
    const results = await Promise.all(
      requests.map(async (req) => {
        try {
          let data: any;

          switch (req.type) {
            case 'user':
              data = await this.callService(this.usersClient, {
                pattern: 'users.findOne',
                data: { id: req.id },
              });
              break;
            case 'order':
              data = await this.callService(this.ordersClient, {
                pattern: 'orders.findOne',
                data: { id: req.id },
              });
              break;
            case 'product':
              data = await this.callService(this.productsClient, {
                pattern: 'products.findOne',
                data: { id: req.id },
              });
              break;
            default:
              throw new Error(`Unknown type: ${req.type}`);
          }

          return { type: req.type, id: req.id, data, success: true };
        } catch (error) {
          return { type: req.type, id: req.id, error: error.message, success: false };
        }
      }),
    );

    return results;
  }

  // ============================================================
  // 私有辅助方法
  // ============================================================

  // 调用微服务（带超时和重试）
  private async callService<T>(
    client: ClientProxy,
    pattern: { pattern: string; data: any },
    timeoutMs = 5000,
  ): Promise<T> {
    const { firstValueFrom } = await import('rxjs');

    return firstValueFrom(
      client.send(pattern.pattern, pattern.data).pipe(
        timeout(timeoutMs),
        retry({ count: 2, delay: 500 }),
        map((data) => data as T),
        catchError((error) => {
          this.logger.error(`Service call failed: ${error.message}`);
          throw error;
        }),
      ),
    );
  }

  // 裁剪用户信息（只返回必要字段）
  private shrinkUserInfo(user: User): Partial<User> {
    return {
      id: user.id,
      name: user.name,
      avatar: user.avatar,
      level: user.level,
    };
  }

  // 裁剪商品信息（只返回必要字段）
  private shrinkProduct(product: Product): Partial<Product> {
    return {
      id: product.id,
      name: product.name,
      price: product.price,
      mainImage: product.mainImage,
    };
  }
}
```

### 4.2 WebSocket实现

```typescript
// WebSocket网关 - 实时数据推送
// NestJS使用socket.io实现WebSocket通信

import {
  Controller,
  Logger,
  UseGuards,
  WebSocketServer,
  WebSocketGateway,
  SubscribeMessage,
  OnGatewayConnection,
  OnGatewayDisconnect,
  MessageBody,
  ConnectedSocket,
} from '@nestjs/websockets';
import { Server, Socket } from 'socket.io';
import { WsAuthGuard } from '../guards/ws-auth.guard';
import { AuthService } from '../auth/auth.service';

@WebSocketGateway({
  // WebSocket命名空间
  // 不同业务可以使用不同命名空间
  namespace: '/ws',
  // CORS配置
  cors: {
    origin: '*',
    credentials: true,
  },
  // Ping-Pong配置
  pingTimeout: 60000,
  pingInterval: 25000,
})
export class WebSocketGateway
  implements OnGatewayConnection, OnGatewayDisconnect
{
  private readonly logger = new Logger('WebSocketGateway');

  @WebSocketServer()
  server: Server;

  constructor(
    private readonly authService: AuthService,
  ) {}

  // 客户端连接
  async handleConnection(client: Socket) {
    try {
      // 验证Token
      const token = client.handshake.auth.token || client.handshake.headers['authorization'];
      const user = await this.authService.verifyWsToken(token);

      // 将用户信息关联到socket
      client.data.user = user;
      client.data.userId = user.userId;

      // 加入用户房间（用于定向推送）
      client.join(`user:${user.userId}`);

      this.logger.log(`用户 ${user.userId} WebSocket连接成功`);

      // 发送连接成功消息
      client.emit('connected', {
        message: '连接成功',
        userId: user.userId,
        timestamp: Date.now(),
      });
    } catch (error) {
      this.logger.warn(`WebSocket连接认证失败: ${error.message}`);
      client.disconnect();
    }
  }

  // 客户端断开
  handleDisconnect(client: Socket) {
    if (client.data.userId) {
      this.logger.log(`用户 ${client.data.userId} WebSocket断开`);
    }
  }

  // ============================================================
  // 订阅消息
  // ============================================================

  // 订阅订单更新
  @SubscribeMessage('subscribe:orders')
  handleSubscribeOrders(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { orderId?: string },
  ) {
    // 订阅自己的订单更新
    const userId = client.data.userId;

    if (data.orderId) {
      // 订阅特定订单
      client.join(`order:${data.orderId}`);
      this.logger.debug(`用户 ${userId} 订阅订单 ${data.orderId}`);
    } else {
      // 订阅所有自己的订单
      client.join(`user:${userId}:orders`);
    }

    return { success: true, message: '订阅成功' };
  }

  // 取消订阅
  @SubscribeMessage('unsubscribe:orders')
  handleUnsubscribeOrders(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { orderId?: string },
  ) {
    const userId = client.data.userId;

    if (data.orderId) {
      client.leave(`order:${data.orderId}`);
    } else {
      client.leave(`user:${userId}:orders`);
    }

    return { success: true };
  }

  // 订阅商品库存
  @SubscribeMessage('subscribe:inventory')
  handleSubscribeInventory(
    @ConnectedSocket() client: Socket,
    @MessageBody() data: { productIds: string[] },
  ) {
    // 批量订阅商品库存
    // 每个商品一个房间
    data.productIds.forEach((productId) => {
      client.join(`inventory:${productId}`);
    });

    return { success: true, subscribed: data.productIds.length };
  }

  // ============================================================
  // 主动推送方法（由其他服务调用）
  // ============================================================

  // 推送订单状态更新
  pushOrderUpdate(userId: string, orderId: string, update: OrderUpdate) {
    // 推送到用户的订单房间
    this.server.to(`user:${userId}`).to(`order:${orderId}`).emit('order:updated', {
      orderId,
      ...update,
      timestamp: Date.now(),
    });
  }

  // 推送库存变更
  pushInventoryUpdate(productId: string, inventory: InventoryUpdate) {
    this.server.to(`inventory:${productId}`).emit('inventory:updated', {
      productId,
      ...inventory,
      timestamp: Date.now(),
    });
  }

  // 推送系统通知
  pushSystemNotification(userId: string, notification: SystemNotification) {
    this.server.to(`user:${userId}`).emit('notification:system', {
      ...notification,
      timestamp: Date.now(),
    });
  }

  // 推送全员广播
  broadcast(message: string, data: any) {
    this.server.emit('broadcast', { message, data, timestamp: Date.now() });
  }
}
```

### 4.3 前端订阅实现

```typescript
// 前端WebSocket服务 - TypeScript实现
// 使用socket.io-client库

import { io, Socket } from 'socket.io-client';
import { Observable, Subject } from 'rxjs';

interface ServerToClientEvents {
  connected: (data: { message: string; userId: string; timestamp: number }) => void;
  'order:updated': (data: OrderUpdate) => void;
  'inventory:updated': (data: InventoryUpdate) => void;
  'notification:system': (data: SystemNotification) => void;
  broadcast: (data: any) => void;
  error: (data: { message: string }) => void;
}

interface ClientToServerEvents {
  'subscribe:orders': (data: { orderId?: string }) => void;
  'unsubscribe:orders': (data: { orderId?: string }) => void;
  'subscribe:inventory': (data: { productIds: string[] }) => void;
}

class WebSocketService {
  private socket: Socket<ServerToClientEvents, ClientToServerEvents>;
  private connectionStatus: 'connecting' | 'connected' | 'disconnected' = 'disconnected';

  // 事件Subject，用于发布订阅
  private orderUpdate$ = new Subject<OrderUpdate>();
  private inventoryUpdate$ = new Subject<InventoryUpdate>();
  private notification$ = new Subject<SystemNotification>();

  constructor() {}

  // 连接服务器
  connect(token: string): Observable<boolean> {
    return new Observable((observer) => {
      this.connectionStatus = 'connecting';

      // 创建Socket连接
      this.socket = io('/ws', {
        auth: { token },
        transports: ['websocket'],  // 使用WebSocket传输
        reconnection: true,         // 自动重连
        reconnectionAttempts: 5,     // 重连次数
        reconnectionDelay: 1000,    // 重连间隔
      });

      // 连接成功
      this.socket.on('connect', () => {
        console.log('WebSocket连接成功');
        this.connectionStatus = 'connected';
        observer.next(true);
      });

      // 连接错误
      this.socket.on('connect_error', (error) => {
        console.error('WebSocket连接失败:', error);
        this.connectionStatus = 'disconnected';
        observer.error(error);
      });

      // 断开连接
      this.socket.on('disconnect', (reason) => {
        console.log('WebSocket断开:', reason);
        this.connectionStatus = 'disconnected';
      });

      // 订单更新事件
      this.socket.on('order:updated', (data) => {
        console.log('收到订单更新:', data);
        this.orderUpdate$.next(data);
      });

      // 库存更新事件
      this.socket.on('inventory:updated', (data) => {
        console.log('收到库存更新:', data);
        this.inventoryUpdate$.next(data);
      });

      // 系统通知事件
      this.socket.on('notification:system', (data) => {
        console.log('收到系统通知:', data);
        this.notification$.next(data);
      });

      // 返回取消订阅函数
      return () => {
        this.disconnect();
      };
    });
  }

  // 断开连接
  disconnect() {
    if (this.socket) {
      this.socket.disconnect();
      this.socket = null;
    }
  }

  // 订阅订单
  subscribeOrders(orderId?: string): void {
    if (this.socket) {
      this.socket.emit('subscribe:orders', { orderId });
    }
  }

  // 取消订阅订单
  unsubscribeOrders(orderId?: string): void {
    if (this.socket) {
      this.socket.emit('unsubscribe:orders', { orderId });
    }
  }

  // 订阅商品库存
  subscribeInventory(productIds: string[]): void {
    if (this.socket) {
      this.socket.emit('subscribe:inventory', { productIds });
    }
  }

  // 获取Observable
  get orderUpdates(): Observable<OrderUpdate> {
    return this.orderUpdate$.asObservable();
  }

  get inventoryUpdates(): Observable<InventoryUpdate> {
    return this.inventoryUpdate$.asObservable();
  }

  get notifications(): Observable<SystemNotification> {
    return this.notification$.asObservable();
  }
}

// 使用示例
const wsService = new WebSocketService();

wsService.connect('user-token').subscribe({
  next: (connected) => {
    if (connected) {
      // 订阅订单更新
      wsService.subscribeOrders();

      // 订阅商品库存
      wsService.subscribeInventory(['P001', 'P002', 'P003']);
    }
  },
  error: (err) => console.error(err),
});

// 监听订单更新
wsService.orderUpdates.subscribe((update) => {
  // 更新本地订单状态
  store.updateOrder(update.orderId, update);
});

// 监听库存更新

  // 更新本地库存显示
  store.updateInventory(update.productId, update.stock);
});
```

## 五、实战踩坑经验

### 5.1 BFF设计常见问题

**坑1：BFF成为性能瓶颈**

```
问题：BFF服务承载所有前端请求，成为单点瓶颈

原因：
- 所有前端请求都经过BFF
- BFF没有做好水平扩展
- 没有做好缓存

解决方案：
1. 合理拆分BFF（按前端类型、业务域）
2. 做好BFF的弹性扩展（HPA）
3. 多级缓存（本地+Redis）
4. 异步处理非关键请求
```

**坑2：数据聚合时部分服务失败**

```
问题：聚合接口中某个服务超时，导致整体超时

症状：用户看到白屏或加载失败

解决方案：
1. 使用Promise.allSettled代替Promise.all
2. 部分失败时返回部分数据
3. 实现降级逻辑（返回缓存数据或友好提示）

示例：
async getHomePageData(userId: string) {
  // 使用Promise.allSettled，不管成功失败都返回
  const results = await Promise.allSettled([
    this.getUserInfo(userId),
    this.getRecommendations(userId),
    this.getBanners(),
  ]);

  // 分别处理每个结果
  const [userResult, recResult, bannerResult] = results;

  return {
    user: userResult.status === 'fulfilled' ? userResult.value : null,
    recommendations: recResult.status === 'fulfilled' ? recResult.value : [],
    banners: bannerResult.status === 'fulfilled' ? bannerResult.value : [],
    partial: userResult.status === 'rejected',  // 标记是否有部分失败
  };
}
```

### 5.2 前后端接口协调问题

**坑3：接口版本不同步**

```
问题：前端更新了，但后端API还没更新，用户看到错误

解决方案：
1. 使用API版本管理
2. 保持接口向后兼容
3. 使用Feature Toggle控制
4. 灰度发布，逐步切换

# API版本策略
GET /api/v1/users/{id}  # 稳定版本
GET /api/v2/users/{id}  # 新版本
GET /api/beta/users/{id} # 灰度版本
```

**坑4：移动端和Web端数据不一致**

```
问题：同一接口返回数据量不同，移动端显示异常

原因：
- 移动端做了数据裁剪
- 但前端代码没有适配

解决方案：
1. 明确数据裁剪规则
2. 使用响应结构而非字段判断
3. 提供移动端专用接口

# 响应结构统一化

  },
  "meta": {
```

```typescript
  }
}
```

## 六、总结

微服务架构下的前后端协同需要解决接口碎片化、数据聚合、性能优化等问题。BFF模式通过在服务端聚合数据、裁剪字段、统一认证，为前端提供更加友好的API。同时，通过WebSocket等实时通信方案，可以实现前后端数据的实时同步。

| 场景 | 方案 |
|------|------|
| 聚合多个微服务数据 | BFF网关 |
