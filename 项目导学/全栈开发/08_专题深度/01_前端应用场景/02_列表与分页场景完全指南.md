## 1. 场景概述：什么是列表与分页？

### 1.1 生活中的类比

## 2. 列表渲染基础：map函数

### 2.1 map是什么？

**类比理解**：map就像是一个"加工流水线"。

想象一下面包工厂：
- 原材料（面粉、水、酵母）进来
- 经过流水线加工
- 产出面包

在代码中，map做的事情一模一样：
- 原始数组进来
- 经过处理函数
- 产出新数组

```
原材料数组          加工函数           产出的数组
    │                  │                  │
    ▼                  ▼                  ▼
[1, 2, 3]         →     x => x * 2    →   [2, 4, 6]
  面粉            →      揉面烘焙         面包

['a', 'b']        →    s => s.toUpperCase() → ['A', 'B']
  小麦            →      磨粉发酵           面粉
```

### 2.2 React中渲染列表

```tsx
// React中使用map渲染列表的基础写法

// 定义商品类型
interface Product {
  id: number;      // 商品唯一标识
  name: string;    // 商品名称
  price: number;   // 商品价格
}

function ProductList() {
  // 商品数据
  const products: Product[] = [
    { id: 1, name: '苹果', price: 5 },
    { id: 2, name: '香蕉', price: 3 },
    { id: 3, name: '橙子', price: 6 },
  ];

  // 使用map渲染列表
  // map会遍历products数组，对每个元素执行函数，返回新数组
  return (
    <ul>
      {products.map((product) => {
        // 返回一个JSX元素
        return (
          <li key={product.id}>
            {product.name} - ¥{product.price}
          </li>
        );
      })}
    </ul>
  );
}

/*
执行过程分解：

第1次循环：product = { id: 1, name: '苹果', price: 5 }
  → 返回 <li key={1}>苹果 - ¥5</li>

第2次循环：product = { id: 2, name: '香蕉', price: 3 }
  → 返回 <li key={2}>香蕉 - ¥3</li>

第3次循环：product = { id: 3, name: '橙子', price: 6 }
  → 返回 <li key={3}>橙子 - ¥6</li>

最终数组：
[<li>苹果</li>, <li>香蕉</li>, <li>橙子</li>]
*/
```

### 2.3 key的重要性

**key是React识别每个列表项的"身份证号"**。

类比理解：想象老师在教室里点名
- 没有key：老师只能靠猜 "最后一排那个同学"
- 有key：老师精准叫号 "3号座位张小明"

```tsx
// key的作用：帮助React高效更新列表

// ✅ 正确：使用稳定的唯一标识作为key
function GoodKeyExample() {
  const users = [
    { id: 1, name: '张三' },
    { id: 2, name: '李四' },
  ];

  return (
    <ul>
      {users.map((user) => (
        // 使用id作为key，id是唯一且稳定的
        <li key={user.id}>
          {user.name}
        </li>
      ))}
    </ul>
  );
}

// ❌ 错误：使用数组索引作为key
function BadKeyExample() {
  const users = [
    { id: 1, name: '张三' },
    { id: 2, name: '李四' },
  ];

  return (
    <ul>
      {users.map((user, index) => (
        // 使用index作为key，这是错误做法！
        // 如果数组顺序变化，React会混淆
        <li key={index}>
          {user.name}
        </li>
      ))}
    </ul>
  );
}

/*
为什么不能用index作为key？

场景：用户在列表中间插入一个新商品

原始数组：        [A, B, C]
使用index key：  [0:A, 1:B, 2:C]
                 ↓ 在B前面插入X
新数组：          [A, X, B, C]
使用index key：  [0:A, 1:X, 2:B, 3:C]

问题来了！
- React看到key=0还是A，没变化
- React看到key=1从B变成了X，以为B被删了，X是新的
- React看到key=2从C变成了B，以为C被删了
- React看到key=3是新的C

这会导致React错误地复用DOM节点，出现bug！
*/

// 某些情况下确实可以用index：
// 1. 列表是静态的，不会变化
// 2. 列表不会重新排序
// 3. 列表项不会被删除
function StaticList() {
  // 固定的帮助信息列表，可以用index
  const helps = ['使用帮助', '联系我们', '用户协议'];

  return (
    <ul>
      {helps.map((help, index) => (
        <li key={index}>{help}</li>
      ))}
    </ul>
  );
}
```

### 2.4 常见错误与纠正

```tsx
// 常见错误1：忘记加key
function Error1() {
  const items = ['a', 'b', 'c'];
  return (
    <ul>
      {items.map((item) => (
        <li>{item}</li>  // ❌ 缺少key
      ))}
    </ul>
  );
}

// 常见错误2：在map的返回值外面包了不必要的标签
function Error2() {
  const items = ['a', 'b', 'c'];
  return (
    <div>
      {items.map((item) => (
        <>
          <div>标题</div>     // ❌ 多个元素没有用Fragment包裹
          <li>{item}</li>
        </>
      ))}
    </div>
  );
}

// 正确写法
function Correct2() {
  const items = ['a', 'b', 'c'];
  return (
    <ul>
      {items.map((item) => (
        // 使用Fragment包裹多个元素
        <React.Fragment key={item}>
          <div>标题</div>
          <li>{item}</li>
        </React.Fragment>
      ))}
    </ul>
  );
}

// 常见错误3：在循环中返回null
function Error3() {
  const items = [1, 2, 3];
  return (
    <ul>
      {items.map((item) => {
        if (item === 2) {
          return null;  // ⚠️ 可以返回null，但要有理由
        }
        return <li key={item}>{item}</li>;
      })}
    </ul>
  );
}

// 常见错误4：key用错了位置
function Error4() {
  const items = [
    { id: 1, children: ['a', 'b'] },
    { id: 2, children: ['c', 'd'] },
  ];

  return (
    <div>
      {items.map((item) => (
        <div key={item.id}>
          {item.children.map((child) => (
            <span key={child}>{child}</span>  // ❌ 这里的key应该是唯一的
          ))}
        </div>
      ))}
    </div>
  );
}
```

## 3. 数组操作三剑客：map、filter、find

### 3.1 三个函数的对比理解

**类比：图书馆的管理员**

| 操作 | 管理员做什么 | 代码对应 |
|------|------------|----------|
| **map** | 复印一本书，内容可能有变化 | 转换每个元素 |
| **filter** | 根据条件筛选出要借的书 | 过滤出符合条件的元素 |
| **find** | 找到第一本符合条件的书 | 找到第一个符合条件的元素 |

```
原始数据（书架上的书）：
[《红楼梦》, 《西游记》, 《三国演义》, 《水浒传》]

map操作（全部复印，但换了封面）：
→ [📕《红楼梦》, 📗《西游记》, 📘《三国演义》, 📙《水浒传》]

filter操作（只找出四大的书）：
→ [《红楼梦》, 《西游记》, 《三国演义》, 《水浒传》]

find操作（找第一本）：
→ 《红楼梦》
```

### 3.2 map：转换操作

```typescript
// map的使用场景：把数据转换成另一种格式

// 示例1：数字数组转字符串数组
const numbers = [1, 2, 3];
const stringNumbers = numbers.map((num) => {
  return String(num);
});
// 结果：['1', '2', '3']

// 示例2：对象数组的字段提取
const products = [
  { id: 1, name: '苹果', price: 5 },
  { id: 2, name: '香蕉', price: 3 },
];

// 只提取商品名称
const names = products.map((product) => {
  return product.name;
});
// 结果：['苹果', '香蕉']

// 示例3：对象数组的字段转换
const users = [
  { firstName: '张', lastName: '三' },
  { firstName: '李', lastName: '四' },
];

// 合并成完整姓名
const fullNames = users.map((user) => {
  return user.firstName + user.lastName;
});
// 结果：['张三', '李四']

// 示例4：数字数组的数学运算
const prices = [10, 20, 30];

// 计算打折后的价格（8折）
const discountedPrices = prices.map((price) => {
  return price * 0.8;
});
// 结果：[8, 16, 24]

// 示例5：嵌套数据结构处理
const orders = [
  {
    id: 1,
    items: [
      { name: '苹果', quantity: 2 },
      { name: '香蕉', quantity: 1 },
    ],
  },
];

// 提取所有订单的商品数量
const quantities = orders.map((order) => {
  return order.items.reduce((sum, item) => sum + item.quantity, 0);
```

### 3.3 filter：过滤操作

```typescript
// filter的使用场景：筛选出符合条件的元素

// 示例1：过滤正数
const numbers = [-1, 2, -3, 4, 0, -5];

  return num > 0;

const words = ['apple', 'banana', 'apricot', 'blueberry'];

const products = [

  { id: 2, name: '白菜', category: '蔬菜', price: 3 },

// 过滤蔬菜类商品
const vegetables = products.filter((product) => {
  return product.category === '蔬菜';
});
// 结果：[白菜, 胡萝卜]

// 过滤价格大于10的商品
const expensiveProducts = products.filter((product) => {
  return product.price > 10;
});
// 结果：[牛肉]

// 示例4：多重条件过滤
const users = [
  { name: '张三', age: 25, city: '北京' },
  { name: '李四', age: 30, city: '上海' },
  { name: '王五', age: 25, city: '北京' },
  { name: '赵六', age: 30, city: '北京' },
];

// 过滤25岁且在北京的用户
const filteredUsers = users.filter((user) => {

// 示例5：排除某些元素
const colors = ['红色', '蓝色', '绿色', '黄色', '蓝色'];

```

### 3.4 find：查找操作

```typescript
// find的使用场景：找到第一个符合条件的元素

// 示例1：查找数字
const numbers = [1, 5, 10, 15, 20];
const firstEven = numbers.find((num) => {
  return num % 2 === 0;
});
// 结果：10（第一个偶数）

// 示例2：查找对象
const products = [
  { id: 1, name: '苹果', price: 5 },
  { id: 2, name: '香蕉', price: 3 },
  { id: 3, name: '橙子', price: 6 },
];

// 查找id为2的商品
const product2 = products.find((product) => {
  return product.id === 2;
});
// 结果：{ id: 2, name: '香蕉', price: 3 }

// 示例3：find和filter的区别
const numbers = [1, 2, 3, 4, 5];

// find返回第一个符合条件的元素
const found = numbers.find((num) => num > 3);
// 结果：4（是数字4，不是数组）

// filter返回所有符合条件的元素组成的数组
const filtered = numbers.filter((num) => num > 3);
// 结果：[4, 5]（是数组）

// 示例4：找不到的情况
const users = [
  { name: '张三', id: 1 },
  { name: '李四', id: 2 },
];

// 查找不存在的用户
const nonExistent = users.find((user) => {
  return user.id === 999;
});
// 结果：undefined

// 安全处理
const foundUser = users.find((user) => {
  return user.id === 999;
});
if (foundUser) {
  console.log(foundUser.name);
} else {
  console.log('用户不存在');
}

// 示例5：findIndex获取位置
const fruits = ['苹果', '香蕉', '橙子'];
const bananaIndex = fruits.findIndex((fruit) => {
  return fruit === '香蕉';
});
// 结果：1（索引位置）
```

### 3.5 组合使用

```typescript
// 实际开发中经常需要组合使用这些函数

const products = [
  { id: 1, name: '苹果', category: '水果', price: 5, stock: 10 },
  { id: 2, name: '白菜', category: '蔬菜', price: 3, stock: 0 },
  { id: 3, name: '牛肉', category: '肉类', price: 40, stock: 5 },
  { id: 4, name: '胡萝卜', category: '蔬菜', price: 2, stock: 20 },
  { id: 5, name: '猪肉', category: '肉类', price: 25, stock: 0 },
  { id: 6, name: '葡萄', category: '水果', price: 15, stock: 8 },
];

// 场景1：找出有库存的水果，显示名称
const availableFruits = products
  .filter((p) => p.category === '水果' && p.stock > 0)  // 第一步过滤
  .map((p) => p.name);  // 第二步转换
// 结果：['苹果', '葡萄']

// 场景2：计算有库存蔬菜的总价
const totalVegetablePrice = products
  .filter((p) => p.category === '蔬菜' && p.stock > 0)  // 过滤蔬菜且有库存
  .reduce((sum, p) => sum + p.price, 0);  // 累加价格
// 结果：7（5 + 2 = 7，但苹果不在蔬菜里，实际是3+2=5）
// 等等，纠正：蔬菜里有库存的是白菜3 + 胡萝卜2 = 5

// 场景3：找出最贵的肉类商品
const expensiveMeat = products
  .filter((p) => p.category === '肉类')  // 过滤肉类

// 结果：{ id: 3, name: '牛肉', price: 40 }

// 场景4：分页前的数据处理
const page = 1;      // 第1页
const pageSize = 3;  // 每页3条

const paginatedProducts = products
  .filter((p) => p.stock > 0)  // 只显示有库存的
  .slice((page - 1) * pageSize, page * pageSize);  // 取当前页
// 结果：前3个有库存的商品

// 场景5：搜索过滤
function searchProducts(keyword: string) {
  return products
    .filter((p) => {
      const searchText = keyword.toLowerCase();
      return (
        p.name.toLowerCase().includes(searchText) ||  // 名字匹配
        p.category.toLowerCase().includes(searchText)   // 分类匹配
      );
    })
    .map((p) => ({
      id: p.id,
      name: p.name,
      category: p.category,
    }));
}

// 使用
searchProducts('果');  // ['苹果', '葡萄']
searchProducts('蔬菜'); // [白菜, 胡萝卜]
```

## 4. 分页原理详解

### 4.1 分页的数学原理

**类比理解**：把一本1000页的书分成10页一章

```
整本书：1000页
每章：100页
章节数：10章

第3章的内容 = 第201页到第300页

计算公式：
- 开始位置 = (页码 - 1) * 每页数量
- 结束位置 = 页码 * 每页数量
```

**核心公式**：

```typescript
// 分页的核心数学公式

// 假设：
// - total: 总数据量（比如100条商品）
// - page: 当前页码（从1开始）
// - pageSize: 每页显示多少条（比如10条）

// 计算总页数
const totalPages = Math.ceil(total / pageSize);
// Math.ceil向上取整：100 / 10 = 10页，99 / 10 = 10页

// 计算当前页的数据起始索引
// 重要：数组索引是从0开始的！
const startIndex = (page - 1) * pageSize;
// page=1: (1-1)*10 = 0，从第0个元素开始
// page=2: (2-1)*10 = 10，从第10个元素开始
// page=3: (3-1)*10 = 20，从第20个元素开始

// 计算当前页的数据结束索引（不包含）
const endIndex = page * pageSize;
// page=1: 1*10 = 10，取到索引0-9，不包含10
// page=2: 2*10 = 20，取到索引10-19，不包含20

// 使用slice截取数据
const currentPageData = allData.slice(startIndex, endIndex);

/*
完整示例：

total = 100条商品
pageSize = 10条/页
page = 3

totalPages = Math.ceil(100/10) = 10
startIndex = (3-1)*10 = 20
endIndex = 3*10 = 30

当前页数据 = allData.slice(20, 30)
即取第21-30条商品（数组索引20-29）
*/
```

### 4.2 分页的类型


### 4.3 分页组件设计

```tsx
// 分页组件设计 - 展示页码、上一页、下一页

interface PaginationProps {
  currentPage: number;    // 当前页码
  totalPages: number;     // 总页数
  onPageChange: (page: number) => void;  // 页码变化回调
}

function Pagination({
  currentPage,
  totalPages,
  onPageChange,
}: PaginationProps) {
  // 生成显示的页码数组
  const getPageNumbers = () => {
    const pages: (number | string)[] = [];
    const showPages = 5;  // 最多显示5个页码

    if (totalPages <= showPages) {
      // 总页数小于等于5，全部显示
      for (let i = 1; i <= totalPages; i++) {
        pages.push(i);
      }
    } else {
      // 总页数大于5，需要省略
      if (currentPage <= 3) {
        // 当前页靠前：显示1,2,3,4,...,最后页
        pages.push(1, 2, 3, 4, '...', totalPages);
      } else if (currentPage >= totalPages - 2) {
        // 当前页靠后：显示1,...,最后4页
        pages.push(1, '...', totalPages - 3, totalPages - 2, totalPages - 1, totalPages);
      } else {
        // 当前页在中间：显示1,...,当前页前后1页,...,最后页
        pages.push(1, '...', currentPage - 1, currentPage, currentPage + 1, '...', totalPages);
      }
    }

    return pages;
  };

  return (
    <div className="pagination">
      {/* 上一页按钮 */}
      <button
        onClick={() => onPageChange(currentPage - 1)}
        disabled={currentPage === 1}
      >
        上一页
      </button>

      {/* 页码按钮 */}
      {getPageNumbers().map((page, index) => {
        if (page === '...') {
          // 省略号
          return <span key={`ellipsis-${index}`}>...</span>;
        }

        return (
          <button
            key={page}
            onClick={() => onPageChange(page as number)}
            className={currentPage === page ? 'active' : ''}
          >
            {page}
          </button>
        );
      })}

      {/* 下一页按钮 */}
      <button
        onClick={() => onPageChange(currentPage + 1)}
        disabled={currentPage === totalPages}
      >
        下一页
      </button>
    </div>
  );
}
```

## 6. 后端分页实现

### 6.1 什么时候用后端分页？

**类比**：就像在搜索引擎里搜资料

- **数据在服务器上**（几十亿条网页）
- 你告诉搜索引擎要找什么，它帮你找
- 只返回当前需要的那一页

**后端分页的适用场景**：

| 特点 | 说明 |
|------|------|
| 数据量大 | 通常 > 1000 条 |
| 数据需要实时 | 如订单、消息、用户列表 |
| 搜索筛选复杂 | 后端更擅长数据库查询 |
| 多条件排序 | 需要数据库索引支持 |

### 6.2 后端分页API设计

```typescript
// 后端分页API设计 - 基于Express示例

/*
请求格式：
GET /api/products?page=1&pageSize=10&category=水果&sort=price&order=asc

响应格式：
{
  "code": 200,
  "message": "success",
  "data": {
    "list": [...],        // 当前页数据
    "pagination": {
      "currentPage": 1,   // 当前页码
      "pageSize": 10,     // 每页条数
      "totalItems": 100,  // 总条数
      "totalPages": 10    // 总页数
    }
  }
}
*/

// 后端路由处理（Express）
app.get('/api/products', async (req, res) => {
  try {
    // 1. 解析分页参数
    const page = Math.max(1, parseInt(req.query.page as string) || 1);
    const pageSize = Math.min(100, Math.max(1, parseInt(req.query.pageSize as string) || 10));

    // 2. 解析筛选参数
    const { category, minPrice, maxPrice, keyword } = req.query;

    // 3. 构建查询条件
    const where: Record<string, any> = {};

    if (category) {
      where.category = category;
    }

    if (minPrice) {
      where.price = { ...where.price, $gte: parseFloat(minPrice as string) };
    }

    if (maxPrice) {
      where.price = { ...where.price, $lte: parseFloat(maxPrice as string) };
    }

    if (keyword) {
      // 模糊搜索名称
      where.name = { $regex: keyword };
    }

    // 4. 构建排序
    const sortField = (req.query.sort as string) || 'id';
    const sortOrder = req.query.order === 'desc' ? -1 : 1;
    const sort = { [sortField]: sortOrder };

    // 5. 计算分页偏移量
    const offset = (page - 1) * pageSize;

    // 6. 查询数据库（假设使用Mongoose）
    const [products, total] = await Promise.all([
      ProductModel
        .find(where)
        .sort(sort)
        .skip(offset)      // 跳过前N条
        .limit(pageSize)   // 取N条
        .lean(),           // 返回普通JS对象
      ProductModel.countDocuments(where),  // 计数
    ]);

    // 7. 返回响应
    res.json({
      code: 200,
      message: 'success',
      data: {
        list: products,
        pagination: {
          currentPage: page,
          pageSize: pageSize,
          totalItems: total,
          totalPages: Math.ceil(total / pageSize),
        },
      },
    });
  } catch (error) {
    res.status(500).json({
      code: 500,
      message: '服务器错误',
      data: null,
    });
  }
});
```

### 6.3 前端调用后端分页API

```tsx
// 前端调用后端分页API - 完整实现

import { useState, useEffect, useCallback } from 'react';

// 定义类型
interface Product {
  id: number;
  name: string;
  category: string;
  price: number;
  stock: number;
}

interface PaginationInfo {
  currentPage: number;
  pageSize: number;
  totalItems: number;
  totalPages: number;
}

interface ApiResponse<T> {
  code: number;
  message: string;
  data: {
    list: T[];
    pagination: PaginationInfo;
  };
}

function ProductListBackendPagination() {
  // 状态定义
  const [products, setProducts] = useState<Product[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  // 分页状态
  const [pagination, setPagination] = useState<PaginationInfo>({
    currentPage: 1,
    pageSize: 10,
    totalItems: 0,
    totalPages: 0,
  });

  // 筛选状态
  const [filters, setFilters] = useState({
    category: '',
    keyword: '',
    minPrice: '',
    maxPrice: '',
  });

  // 排序状态
  const [sort, setSort] = useState({
    field: 'id',
    order: 'asc',
  });

  // 获取数据函数
  const fetchProducts = useCallback(async () => {
    try {
      setLoading(true);
      setError(null);

      // 构建查询参数
      const params = new URLSearchParams({
        page: pagination.currentPage.toString(),
        pageSize: pagination.pageSize.toString(),
      });

      if (filters.category) {
        params.append('category', filters.category);
      }
      if (filters.keyword) {
        params.append('keyword', filters.keyword);
      }
      if (filters.minPrice) {
        params.append('minPrice', filters.minPrice);
      }
      if (filters.maxPrice) {
        params.append('maxPrice', filters.maxPrice);
      }
      if (sort.field) {
        params.append('sort', sort.field);
        params.append('order', sort.order);
      }

      // 调用API
      const response = await fetch(`/api/products?${params.toString()}`);
      const result: ApiResponse<Product> = await response.json();

      if (result.code === 200) {
        setProducts(result.data.list);
        setPagination(result.data.pagination);
      } else {
        throw new Error(result.message);
      }
    } catch (err) {
      setError(err instanceof Error ? err.message : '获取数据失败');
    } finally {
      setLoading(false);
    }
  }, [pagination.currentPage, pagination.pageSize, filters, sort]);

  // 监听分页、筛选、排序变化，自动获取数据
  useEffect(() => {
    fetchProducts();
  }, [fetchProducts]);

  // 处理页码变化
  const handlePageChange = (newPage: number) => {
    if (newPage >= 1 && newPage <= pagination.totalPages) {
      setPagination((prev) => ({ ...prev, currentPage: newPage }));
    }
  };

  // 处理每页条数变化
  const handlePageSizeChange = (newSize: number) => {
    setPagination((prev) => ({
      ...prev,
      pageSize: newSize,
      currentPage: 1,  // 重置到第一页
    }));
  };

  // 处理筛选变化
  const handleFilterChange = (key: string, value: string) => {
    setFilters((prev) => ({ ...prev, [key]: value }));
    setPagination((prev) => ({ ...prev, currentPage: 1 }));  // 重置到第一页
  };

  // 处理排序变化
  const handleSortChange = (field: string) => {
    setSort((prev) => ({
      field,
      order: prev.field === field && prev.order === 'asc' ? 'desc' : 'asc',
    }));
    setPagination((prev) => ({ ...prev, currentPage: 1 }));
  };

  return (
    <div className="product-list">
      <h2>商品列表（后端分页）</h2>

      {/* 加载状态 */}
      {loading && <div className="loading">加载中...</div>}

      {/* 错误提示 */}
      {error && <div className="error">{error}</div>}

      {/* 筛选器 */}
      <div className="filters">
        <input
          type="text"
          placeholder="搜索商品名称"
          value={filters.keyword}
          onChange={(e) => handleFilterChange('keyword', e.target.value)}
        />

        <select
          value={filters.category}
          onChange={(e) => handleFilterChange('category', e.target.value)}
        >
          <option value="">全部分类</option>
          <option value="水果">水果</option>
          <option value="蔬菜">蔬菜</option>
          <option value="肉类">肉类</option>
        </select>

        <input
          type="number"
          placeholder="最低价"
          value={filters.minPrice}
          onChange={(e) => handleFilterChange('minPrice', e.target.value)}
        />

        <input
          type="number"
          placeholder="最高价"
          value={filters.maxPrice}
          onChange={(e) => handleFilterChange('maxPrice', e.target.value)}
        />
      </div>

      {/* 商品列表 */}
      {!loading && !error && (
        <table>
          <thead>
            <tr>
              <th onClick={() => handleSortChange('id')}>
                ID {sort.field === 'id' && (sort.order === 'asc' ? '↑' : '↓')}
              </th>
              <th onClick={() => handleSortChange('name')}>
                名称 {sort.field === 'name' && (sort.order === 'asc' ? '↑' : '↓')}
              </th>
              <th onClick={() => handleSortChange('category')}>
                分类 {sort.field === 'category' && (sort.order === 'asc' ? '↑' : '↓')}
              </th>
              <th onClick={() => handleSortChange('price')}>
                价格 {sort.field === 'price' && (sort.order === 'asc' ? '↑' : '↓')}
              </th>
              <th onClick={() => handleSortChange('stock')}>
                库存 {sort.field === 'stock' && (sort.order === 'asc' ? '↑' : '↓')}
              </th>
            </tr>
          </thead>
          <tbody>
            {products.map((product) => (
              <tr key={product.id}>
                <td>{product.id}</td>
                <td>{product.name}</td>
                <td>{product.category}</td>
                <td>¥{product.price}</td>
                <td>{product.stock > 0 ? product.stock : '缺货'}</td>
              </tr>
            ))}
          </tbody>
        </table>
      )}

      {/* 分页控制 */}
      {!loading && !error && (
        <div className="pagination">
          <span>
            共 {pagination.totalItems} 条数据，当前第 {pagination.currentPage} /{' '}
            {pagination.totalPages} 页
          </span>

          <select
            value={pagination.pageSize}
            onChange={(e) => handlePageSizeChange(Number(e.target.value))}
          >
            <option value={5}>5条/页</option>
            <option value={10}>10条/页</option>
            <option value={20}>20条/页</option>
            <option value={50}>50条/页</option>
          </select>

          <button
            onClick={() => handlePageChange(1)}
            disabled={pagination.currentPage === 1}
          >
            首页
          </button>

          <button
            onClick={() => handlePageChange(pagination.currentPage - 1)}
            disabled={pagination.currentPage === 1}
          >
            上一页
          </button>

  );
}
```

| 自动加载 | 滚动到底部自动触发 |
| 沉浸体验 | 用户不停下来就一直加载 |
| 适合内容流 | 社交媒体、新闻推荐 |

### 7.2 无限滚动的实现原理

```tsx
// 无限滚动的实现原理

import { useState, useEffect, useRef, useCallback } from 'react';

function InfiniteScroll() {
  const [items, setItems] = useState<Item[]>([]);
  const [page, setPage] = useState(1);
  const [hasMore, setHasMore] = useState(true);
  const [loading, setLoading] = useState(false);

  // 用于触发加载的引用（放在列表底部）
  const observerTarget = useRef<HTMLDivElement>(null);

  // 加载数据的函数
  const loadMoreData = useCallback(async () => {
    if (loading || !hasMore) return;

    try {
      setLoading(true);
      const response = await fetch(`/api/items?page=${page}&pageSize=20`);
      const data = await response.json();

      if (data.items.length === 0) {
        setHasMore(false);
      } else {
        setItems((prev) => [...prev, ...data.items]);
        setPage((prev) => prev + 1);
      }
    } catch (error) {
      console.error('加载失败:', error);
    } finally {
      setLoading(false);
    }
  }, [page, loading, hasMore]);

  // 使用IntersectionObserver监听滚动
  useEffect(() => {
    const observer = new IntersectionObserver(
      (entries) => {
        // 当目标元素可见时，加载更多
        if (entries[0].isIntersecting) {
          loadMoreData();
        }
      },
      {
        threshold: 0.1,        // 目标元素10%可见时触发
        rootMargin: '100px',   // 提前100px开始加载
      }
    );

    if (observerTarget.current) {
      observer.observe(observerTarget.current);
    }

    return () => {
      if (observerTarget.current) {
        observer.unobserve(observerTarget.current);
      }
    };
  }, [loadMoreData]);

  return (
    <div className="infinite-scroll">
      {/* 内容列表 */}
      {items.map((item) => (
        <div key={item.id} className="item">
          {item.content}
        </div>
      ))}

      {/* 加载指示器 */}
      {loading && <div className="loading">加载中...</div>}

      {/* 没有更多数据 */}
      {!hasMore && <div className="no-more">没有更多了</div>}

      {/* 滚动监听目标 */}
      <div ref={observerTarget} className="load-trigger" />
    </div>
  );
}
```

```tsx
        matchMinPrice &&
        matchMaxPrice &&

          placeholder="最高价"
          value={maxPrice ?? ''}
          onChange={(e) =>
```

### 8.3 防抖搜索

```tsx
// 防抖搜索 - 避免每次输入都发送请求

import { useState, useEffect, useRef } from 'react';

function DebouncedSearch() {
  const [keyword, setKeyword] = useState('');
  const [results, setResults] = useState<string[]>([]);
  const [loading, setLoading] = useState(false);

  // 使用useRef保存防抖定时器
  const debounceRef = useRef<number | null>(null);

  // 搜索函数
  const search = async (query: string) => {
    if (!query) {
      setResults([]);
      return;
    }

    setLoading(true);
    try {
      const response = await fetch(`/api/search?q=${encodeURIComponent(query)}`);
      const data = await response.json();
      setResults(data.results);
    } catch (error) {
      console.error('搜索失败:', error);
    } finally {
      setLoading(false);
    }
  };

  // 处理输入变化 - 带防抖
  const handleInputChange = (value: string) => {
    setKeyword(value);

    // 清除之前的定时器
    if (debounceRef.current) {
      clearTimeout(debounceRef.current);
    }

    // 设置新的定时器，300ms后执行搜索
    debounceRef.current = window.setTimeout(() => {
      search(value);
    }, 300);
  };

  // 组件卸载时清除定时器
  useEffect(() => {
    return () => {
      if (debounceRef.current) {
        clearTimeout(debounceRef.current);
      }
    };
  }, []);

  return (
    <div>
      <input
        type="text"
        value={keyword}
        onChange={(e) => handleInputChange(e.target.value)}
        placeholder="输入搜索关键词..."
      />

      {loading && <span>搜索中...</span>}

      <ul>
        {results.map((result, index) => (
          <li key={index}>{result}</li>
        ))}
      </ul>
    </div>
  );
}

// 使用自定义Hook封装防抖逻辑（更好的实践）
function useDebounce<T>(value: T, delay: number): T {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const timer = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => {
      clearTimeout(timer);
    };
  }, [value, delay]);

  return debouncedValue;
}

function BetterDebouncedSearch() {
  const [keyword, setKeyword] = useState('');
  const [results, setResults] = useState<string[]>([]);
  const [loading, setLoading] = useState(false);

  // 使用防抖Hook
  const debouncedKeyword = useDebounce(keyword, 300);

  // 监听防抖后的关键词变化
  useEffect(() => {
    if (!debouncedKeyword) {
      setResults([]);
      return;
    }

    const search = async () => {
      setLoading(true);
      try {
        const response = await fetch(`/api/search?q=${encodeURIComponent(debouncedKeyword)}`);
        const data = await response.json();
        setResults(data.results);
      } catch (error) {
        console.error('搜索失败:', error);
      } finally {
        setLoading(false);
      }
    };

    search();
  }, [debouncedKeyword]);

  return (
    <div>
      <input
        type="text"
        value={keyword}
        onChange={(e) => setKeyword(e.target.value)}
        placeholder="输入搜索关键词..."
      />

      {loading && <span>搜索中...</span>}

      <ul>
        {results.map((result, index) => (
          <li key={index}>{result}</li>
        ))}
      </ul>
    </div>
  );
}

/*
防抖原理：

没有防抖：
输入 "a" → 立即搜索
输入 "ab" → 立即搜索
输入 "abc" → 立即搜索
输入 "abcd" → 立即搜索
→ 发送4次请求，浪费资源！

有防抖（300ms）：
输入 "a" → 开始计时
输入 "ab" → 重置计时
输入 "abc" → 重置计时
输入 "abcd" → 重置计时
→ 停止输入300ms后，才搜索
→ 只发送1次请求！
*/
```

## 9. 排序功能：让数据井然有序

### 9.1 排序的概念

**类比**：书架上的书可以按不同方式排列

| 排序方式 | 说明 |
|----------|------|
| 按书名首字母 | A-Z |
| 按出版时间 | 最新在前 |
| 按价格 | 从低到高 |
| 按评分 | 最高分在前 |

### 9.2 前端排序实现

```tsx
// 前端排序实现 - 单字段和多字段排序

import { useState, useMemo } from 'react';

interface Product {
  id: number;
  name: string;
  category: string;
  price: number;
  stock: number;
  rating: number;  // 评分
}

function Sorting() {
  // 模拟数据
  const products: Product[] = [
    { id: 1, name: '苹果', category: '水果', price: 5, stock: 10, rating: 4.5 },
    { id: 2, name: '白菜', category: '蔬菜', price: 3, stock: 0, rating: 4.0 },
    { id: 3, name: '牛肉', category: '肉类', price: 40, stock: 5, rating: 4.8 },
    { id: 4, name: '胡萝卜', category: '蔬菜', price: 2, stock: 20, rating: 4.2 },
    { id: 5, name: '猪肉', category: '肉类', price: 25, stock: 0, rating: 4.3 },
    { id: 6, name: '葡萄', category: '水果', price: 15, stock: 8, rating: 4.6 },
  ];

  // 排序状态
  const [sortField, setSortField] = useState<keyof Product | ''>('');
  const [sortOrder, setSortOrder] = useState<'asc' | 'desc'>('asc');

  // 排序后的数据
  const sortedProducts = useMemo(() => {
    if (!sortField) {
      return products;  // 没有排序字段，返回原数组
    }

    return [...products].sort((a, b) => {
      // 获取比较的值
      const aValue = a[sortField as keyof Product];
      const bValue = b[sortField as keyof Product];

      // 不同类型的比较逻辑
      if (typeof aValue === 'number' && typeof bValue === 'number') {
        // 数字比较
        return sortOrder === 'asc' ? aValue - bValue : bValue - aValue;
      }

      if (typeof aValue === 'string' && typeof bValue === 'string') {
        // 字符串比较
        const comparison = aValue.localeCompare(bValue);
        return sortOrder === 'asc' ? comparison : -comparison;
      }

      return 0;
    });
  }, [products, sortField, sortOrder]);

  // 处理排序变化
  const handleSort = (field: keyof Product) => {
    if (field === sortField) {
      // 点击同一字段，切换排序方向
      setSortOrder((prev) => (prev === 'asc' ? 'desc' : 'asc'));
    } else {
      // 点击新字段，默认升序
      setSortField(field);
      setSortOrder('asc');
    }
  };

  // 获取排序指示器
  const getSortIndicator = (field: keyof Product) => {
    if (field !== sortField) {
      return '';  // 没有排序
    }
    return sortOrder === 'asc' ? '↑' : '↓';
  };

  return (
    <div className="sorting">
      <h2>商品排序</h2>

      <table>
        <thead>
          <tr>
            <th onClick={() => handleSort('id')}>
              ID {getSortIndicator('id')}
            </th>
            <th onClick={() => handleSort('name')}>
              名称 {getSortIndicator('name')}
            </th>
            <th onClick={() => handleSort('category')}>
              分类 {getSortIndicator('category')}
            </th>
            <th onClick={() => handleSort('price')}>
              价格 {getSortIndicator('price')}
            </th>
            <th onClick={() => handleSort('stock')}>
              库存 {getSortIndicator('stock')}
            </th>
            <th onClick={() => handleSort('rating')}>
              评分 {getSortIndicator('rating')}
            </th>
          </tr>
        </thead>
        <tbody>
          {sortedProducts.map((product) => (
            <tr key={product.id}>
              <td>{product.id}</td>
              <td>{product.name}</td>
              <td>{product.category}</td>
              <td>¥{product.price}</td>
              <td>{product.stock}</td>
              <td>{product.rating}</td>
            </tr>
          ))}
        </tbody>
      </table>

      {/* 排序说明 */}
      <div className="sort-info">
        <p>当前排序：{sortField || '无'} ({sortOrder === 'asc' ? '升序' : '降序'})</p>
      </div>
    </div>
  );
}
```

```tsx
import { useState, useMemo } from 'react';

interface Product {

function MultiFieldSorting() {

    { id: 6, name: '猪肉', category: '肉类', price: 25 },
  ];

  // 多字段排序：先按分类，再按价格
  const sortedProducts = useMemo(() => {
    return [...products].sort((a, b) => {
      // 第一排序：按分类
      const categoryCompare = a.category.localeCompare(b.category);
      if (categoryCompare !== 0) {
        return categoryCompare;
      }

      // 第二排序：按价格
      return a.price - b.price;
    });
  }, [products]);

  // 更灵活的多字段排序实现
  type SortConfig = {
    field: keyof Product;
    order: 'asc' | 'desc';
  };

  const [sortConfigs, setSortConfigs] = useState<SortConfig[]>([
    { field: 'category', order: 'asc' },
    { field: 'price', order: 'asc' },
  ]);

  const flexibleSortedProducts = useMemo(() => {
    return [...products].sort((a, b) => {
      for (const config of sortConfigs) {
        const aValue = a[config.field];
        const bValue = b[config.field];

        let comparison = 0;
        if (typeof aValue === 'number' && typeof bValue === 'number') {
          comparison = aValue - bValue;
        } else if (typeof aValue === 'string' && typeof bValue === 'string') {
          comparison = aValue.localeCompare(bValue);
        }

        // 如果不相等，返回结果
        if (comparison !== 0) {
          return config.order === 'asc' ? comparison : -comparison;
        }
      }

      return 0;  // 所有字段都相等
    });
  }, [products, sortConfigs]);

  return (
    <div>
      <h3>多字段排序结果（先分类，再价格）</h3>
      <ul>
        {sortedProducts.map((p) => (
          <li key={p.id}>

      {/*

        水果 - 香蕉 - ¥3
        水果 - 苹果 - ¥5
        肉类 - 猪肉 - ¥25
        肉类 - 牛肉 - ¥40
      */}
    </div>
  );
}
```

### 11.3 分页性能优化

```tsx
// 分页性能优化

// 问题：切换页码时，白屏闪烁
// 解决1：保持上下页数据缓存

function CachedPagination() {
  const [cache, setCache] = useState<Record<number, any[]>>({});
  const [currentPage, setCurrentPage] = useState(1);

  const goToPage = useCallback(async (page: number) => {
    if (cache[page]) {
      // 使用缓存数据
      setCurrentPage(page);
      return;
    }

    // 加载新数据
    const data = await fetchPageData(page);
    setCache((prev) => ({ ...prev, [page]: data }));
    setCurrentPage(page);
  }, [cache]);

  // 解决2：预加载相邻页数据
  const prefetchAdjacentPages = useCallback((currentPage: number) => {
    // 预加载上一页
    if (currentPage > 1 && !cache[currentPage - 1]) {
      fetchPageData(currentPage - 1).then((data) => {
        setCache((prev) => ({ ...prev, [currentPage - 1]: data }));
      });
    }

    // 预加载下一页
    if (!cache[currentPage + 1]) {
      fetchPageData(currentPage + 1).then((data) => {
        setCache((prev) => ({ ...prev, [currentPage + 1]: data }));
      });
    }
  }, [cache]);

  return (
    <div>
      {/* 内容 */}
      <button onClick={() => {
        const newPage = currentPage - 1;
        goToPage(newPage);
        prefetchAdjacentPages(newPage);
      }}>
        上一页
      </button>
    </div>
  );
}
```

## 12. 常见问题与解决方案

### 12.1 列表相关问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **列表不显示** | 数据为空或undefined | 检查数据初始化 |
| **列表项错乱** | key用错或重复 | 确保key唯一且稳定 |
| **切换页面卡顿** | 数据量大 | 使用分页或虚拟列表 |
| **搜索结果闪烁** | 多次重复请求 | 使用防抖 |
| **排序后顺序错乱** | 原数组被修改 | 使用[...arr].sort() |

### 12.2 分页相关问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **总页数为0** | total计算错误 | Math.ceil(total / pageSize) |
| **页码越界** | 删除数据后未重置 | 切换页码时检查边界 |
| **数据重复** | 两次请求同时返回 | 使用版本号或abort |
| **白屏闪烁** | 重新加载数据 | 使用缓存或骨架屏 |

### 12.3 搜索相关问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| **输入卡顿** | 每次输入都搜索 | 使用防抖（debounce） |
| **搜索不到** | 大小写不匹配 | toLowerCase() 统一 |
| **特殊字符报错** | URL编码问题 | encodeURIComponent |
| **空结果不提示** | 未处理空数组 | 显示友好提示 |
