## 第一部分：文件上传

### 1.1 文件上传基础概念

#### 1.1.1 核心API概述

文件上传涉及多个Web API，它们之间的关系如下：

```
┌─────────────────────────────────────────────────────────────────┐
│                         用户交互层                               │
├─────────────────────────────────────────────────────────────────┤
│  <input type="file">  │  拖拽区域  │  粘贴板(clipboardData)      │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                         文件对象层                               │
├─────────────────────────────────────────────────────────────────┤
│  File 对象（继承自 Blob）                                       │
│  ├── name: 文件名                                                │
│  ├── size: 文件大小（字节）                                      │
│  ├── type: MIME类型                                              │
│  ├── lastModified: 最后修改时间                                  │
│  └── ArrayBuffer / Blob / DataURL 转换方法                      │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                         传输层                                   │
├─────────────────────────────────────────────────────────────────┤
│  FormData 对象                                                   │
│  ├── 包装文件数据                                                │
│  ├── 支持 multipart/form-data 编码                               │
│  └── 可添加其他字段                                               │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
┌─────────────────────────────────────────────────────────────────┐
│                         网络传输层                               │
├─────────────────────────────────────────────────────────────────┤
│  axios / fetch                                                  │
│  ├── 上传进度监控 (XMLHttpRequest.upload)                       │
│  ├── 断点续传 (Range 请求头)                                     │
│  └── 分片上传 (多个请求并发)                                     │
└─────────────────────────────────────────────────────────────────┘
```

#### 1.1.2 File API 详解

**File 接口继承自 Blob，扩展了以下属性：**

| 属性 | 类型 | 说明 |
|------|------|------|
| `name` | string | 文件名（不含路径） |
| `size` | number | 文件大小（字节） |
| `type` | string | MIME类型，如 `image/png` |
| `lastModified` | number | 最后修改时间（时间戳） |
| `lastModifiedDate` | Date | 最后修改日期对象（已废弃） |
| `webkitRelativePath` | string | 相对路径（部分浏览器支持） |

**File 继承自 Blob 的方法：**

| 方法 | 返回值 | 说明 |
|------|--------|------|
| `slice(start, end)` | Blob | 截取文件片段 |
| `stream()` | ReadableStream | 返回读取流 |
| `text()` | Promise<string> | 异步读取为文本 |
| `arrayBuffer()` | Promise<ArrayBuffer> | 异步读取为数组缓冲 |
| `blob()` | Promise<Blob> | 异步读取为 Blob |

#### 1.1.3 Blob 对象详解

**Blob（Binary Large Object）表示二进制大数据：**

```typescript
// 构造函数
const blob = new Blob(dataArray, options);

// dataArray: 数据数组，可以是 ArrayBuffer、TypedArray、Blob、string 等
// options: 配置对象
//   - type: MIME类型
//   - endings: '\n' 是转换为 'rn' 还是保持 '\n'

// 示例：创建文本 Blob
const textBlob = new Blob(['Hello, World!'], { type: 'text/plain' });

// 示例：创建 JSON Blob
const jsonBlob = new Blob(
  [JSON.stringify({ name: '张三', age: 25 })],
  { type: 'application/json' }
);

// 示例：创建二进制 Blob（模拟文件头）
const header = new Uint8Array([0x89, 0x50, 0x4E, 0x47]); // PNG 文件头
const imageBlob = new Blob([header], { type: 'image/png' });
```

**Blob URL（对象URL）的创建与释放：**

```typescript
// 创建 Blob URL
// URL.createObjectURL(blob) 返回一个字符串，格式为 blob:null/uuid
const blobUrl = URL.createObjectURL(file);

// 释放 Blob URL（重要：避免内存泄漏）
URL.revokeObjectURL(blobUrl);

// 使用场景：
// 1. 图片预览
// 2. 视频/音频播放
// 3. 文件下载
```

### 1.2 文件选择与预览

#### 1.2.1 单文件选择器实现

**基础文件输入组件：**

```typescript
// components/FileUploader/BasicFileInput.tsx
"use client";

import React, { useRef, useState } from 'react';

/**
 * 基础文件选择器组件
 *
 * 功能说明：
 * 1. 点击按钮打开文件选择对话框
 * 2. 支持限制文件类型和大小
 * 3. 选择后显示文件信息
 *
 * 核心概念：
 * - input[type="file"] 是原生文件选择元素
 * - ref 用于获取 DOM 元素引用
 * - accept 属性限制可选文件类型
 * - onChange 处理文件选择事件
 */
export default function BasicFileInput() {
  // 使用 ref 获取隐藏的 input 元素
  const inputRef = useRef<HTMLInputElement>(null);

  // 存储选中的文件
  const [selectedFile, setSelectedFile] = useState<File | null>(null);

  // 错误信息
  const [error, setError] = useState<string>('');

  // 文件大小格式化函数（字节转换为人类可读格式）
  const formatFileSize = (bytes: number): string => {
    if (bytes === 0) return '0 B';

    // 文件大小单位数组
    const units = ['B', 'KB', 'MB', 'GB', 'TB'];
    // 计算单位索引（每1024字节一个单位）
    const k = 1024;
    const i = Math.floor(Math.log(bytes) / Math.log(k));

    // 格式化输出，保留两位小数
    return `${parseFloat((bytes / Math.pow(k, i)).toFixed(2))} ${units[i]}`;
  };

  // 获取文件 MIME 类型图标（简单实现）
  const getFileIcon = (mimeType: string): string => {
    if (mimeType.startsWith('image/')) return '🖼️';
    if (mimeType.startsWith('video/')) return '🎬';
    if (mimeType.startsWith('audio/')) return '🎵';
    if (mimeType.startsWith('text/')) return '📄';
    if (mimeType.includes('pdf')) return '📕';
    if (mimeType.includes('zip') || mimeType.includes('rar')) return '📦';
    return '📁';
  };

  // 点击按钮触发文件选择
  const handleClick = () => {
    // 调用 input 的 click 方法打开文件对话框
    inputRef.current?.click();
  };

  // 文件选择处理
  const handleChange = (event: React.ChangeEvent<HTMLInputElement>) => {
    // 获取选择的文件（files 是 FileList 类数组对象）
    const file = event.target.files?.[0];

    // 重置错误
    setError('');

    if (!file) {
      // 用户取消选择
      return;
    }

    // 验证文件大小（限制 10MB）
    const maxSize = 10 * 1024 * 1024; // 10MB in bytes
    if (file.size > maxSize) {
      setError(`文件大小超过限制（最大 ${formatFileSize(maxSize)}）`);
      setSelectedFile(null);
      return;
    }

    // 验证文件类型（仅允许图片）
    const allowedTypes = ['image/jpeg', 'image/png', 'image/gif', 'image/webp'];
    if (!allowedTypes.includes(file.type)) {
      setError('不支持的文件类型，请选择图片文件（JPEG、PNG、GIF、WebP）');
      setSelectedFile(null);
      return;
    }

    // 文件验证通过，保存文件对象
    setSelectedFile(file);
  };

  return (
    <div className="p-6 border-2 border-dashed border-gray-300 rounded-lg">
      {/* 隐藏的 input 元素 */}
      <input
        ref={inputRef}
        type="file"
        accept="image/jpeg,image/png,image/gif,image/webp"
        onChange={handleChange}
        className="hidden"
      />

      {/* 上传按钮 */}
      <button
        onClick={handleClick}
        className="px-6 py-3 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors"
      >
        选择图片文件
      </button>

      {/* 错误提示 */}
      {error && (
        <div className="mt-4 p-3 bg-red-50 border border-red-200 rounded-lg text-red-600">
          {error}
        </div>
      )}

      {/* 文件信息展示 */}
      {selectedFile && (
        <div className="mt-4 p-4 bg-gray-50 rounded-lg">
          <div className="flex items-center gap-4">
            {/* 文件图标 */}
            <span className="text-4">{getFileIcon(selectedFile.type)}</span>

            {/* 文件信息 */}
            <div>
              <p className="font-medium text-gray-900">{selectedFile.name}</p>
              <p className="text-sm text-gray-500">
                {formatFileSize(selectedFile.size)} | {selectedFile.type}
              </p>
            </div>
          </div>
        </div>
      )}
    </div>
  );
}
```

#### 1.2.2 图片预览实现

**图片预览组件（使用 FileReader API）：**

```typescript
// components/FileUploader/ImagePreview.tsx
"use client";

import React, { useState, useRef, useCallback } from 'react';

/**
 * 图片预览组件
 *
 * 功能说明：
 * 1. 支持单图片选择和预览
 * 2. 使用 FileReader API 读取文件内容
 * 3. 将图片转为 DataURL 进行预览
 * 4. 支持拖拽上传
 *
 * 核心概念：
 * - FileReader API：异步读取文件内容
 * - readAsDataURL：将文件读取为 Base64 编码的 DataURL
 * - DataURL 格式：data:[<mediatype>][;base64],<data>
 */
interface ImagePreviewProps {
  // 可选：限制图片尺寸（宽度）
  maxWidth?: number;
  // 可选：限制图片尺寸（高度）
  maxHeight?: number;
  // 可选：接受的文件类型
  accept?: string;
  // 可选：最大文件大小（字节）
  maxSize?: number;
  // 可选：文件大小格式化函数
  formatSize?: (bytes: number) => string;
  // 可选：上传成功回调
  onUpload?: (file: File, dataUrl: string) => void;
}

/**
 * 拖拽状态枚举
 * 用于区分用户当前是否正在拖拽文件到上传区域
 */
type DragState = 'idle' | 'dragover' | 'dragleave';

export default function ImagePreview({
  maxWidth = 800,
  maxHeight = 800,
  accept = 'image/*',
  maxSize = 5 * 1024 * 1024, // 默认 5MB
  formatSize = (bytes) => `${(bytes / 1024).toFixed(1)} KB`,
  onUpload,
}: ImagePreviewProps) {
  // 存储预览图片的 DataURL
  const [previewUrl, setPreviewUrl] = useState<string | null>(null);

  // 存储当前选中的文件
  const [currentFile, setCurrentFile] = useState<File | null>(null);

  // 拖拽状态
  const [dragState, setDragState] = useState<DragState>('idle');

  // 错误信息
  const [error, setError] = useState<string>('');

  // 引用隐藏的文件输入框
  const fileInputRef = useRef<HTMLInputElement>(null);

  /**
   * 验证文件是否合法
   * @param file 要验证的文件
   * @returns 验证结果对象
   */
  const validateFile = useCallback((file: File): { valid: boolean; error?: string } => {
    // 检查文件类型（必须是图片）
    if (!file.type.startsWith('image/')) {
      return { valid: false, error: '请选择图片文件' };
    }

    // 检查文件大小
    if (file.size > maxSize) {
      return { valid: false, error: `文件大小不能超过 ${formatSize(maxSize)}` };
    }

    return { valid: true };
  }, [maxSize, formatSize]);

  /**
   * 读取文件并生成预览 URL
   * 使用 FileReader API 异步读取文件内容
   *
   * @param file 文件对象
   * @returns Promise<string> 预览 URL
   */
  const readFileAsDataURL = (file: File): Promise<string> => {
    return new Promise((resolve, reject) => {
      // 创建 FileReader 实例
      const reader = new FileReader();

      // 文件读取成功时的回调
      // onload 在文件完全读取后触发
      reader.onload = (e) => {
        // e.target?.result 包含文件的 DataURL 表示
        const result = e.target?.result as string;
        resolve(result);
      };

      // 文件读取失败时的回调
      reader.onerror = () => {
        reject(new Error('文件读取失败'));
      };

      // 开始读取文件为 DataURL
      // DataURL 格式: data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAE...
      reader.readAsDataURL(file);
    });
  };

  /**
   * 处理文件选择
   * 验证文件、读取内容、生成预览
   */
  const handleFileSelect = async (file: File) => {
    // 重置错误状态
    setError('');

    // 验证文件
    const validation = validateFile(file);
    if (!validation.valid) {
      setError(validation.error || '文件验证失败');
      return;
    }

    try {
      // 读取文件并生成预览 URL
      const dataUrl = await readFileAsDataURL(file);

      // 更新状态
      setCurrentFile(file);
      setPreviewUrl(dataUrl);

      // 触发上传回调
      onUpload?.(file, dataUrl);
    } catch (err) {
      setError('文件读取失败，请重试');
      console.error('文件读取错误:', err);
    }
  };

  /**
   * 处理 input 改变事件
   */
  const handleInputChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0];
    if (file) {
      handleFileSelect(file);
    }
  };

  /**
   * 处理点击上传区域
   */
  const handleClick = () => {
    fileInputRef.current?.click();
  };

  /**
   * 处理拖拽进入
   * 当用户拖拽文件进入上传区域时触发
   */
  const handleDragOver = (e: React.DragEvent) => {
    // 阻止默认行为，否则无法触发 drop 事件
    e.preventDefault();
    e.stopPropagation();

    // 设置拖拽状态为 dragover
    setDragState('dragover');
  };

  /**
   * 处理拖拽离开
   * 当用户拖拽文件离开上传区域时触发
   */
  const handleDragLeave = (e: React.DragEvent) => {
    // 阻止事件冒泡
    e.preventDefault();
    e.stopPropagation();

    // 设置拖拽状态为 idle
    setDragState('idle');
  };

  /**
   * 处理文件放下
   * 当用户释放鼠标完成拖拽时触发
   */
  const handleDrop = (e: React.DragEvent) => {
    // 阻止默认行为
    e.preventDefault();
    e.stopPropagation();

    // 重置拖拽状态
    setDragState('idle');

    // 获取拖拽的文件（可能多个）
    const files = e.dataTransfer.files;

    if (files.length > 0) {
      // 只处理第一个文件
      handleFileSelect(files[0]);
    }
  };

  /**
   * 删除当前图片
   */
  const handleDelete = () => {
    // 清除预览
    setPreviewUrl(null);
    setCurrentFile(null);
    setError('');

    // 清空 input 的值，允许重新选择相同文件
    if (fileInputRef.current) {
      fileInputRef.current.value = '';
    }
  };

  // 计算拖拽状态对应的样式
  const getDragStateStyles = () => {
    switch (dragState) {
      case 'dragover':
        return 'border-blue-500 bg-blue-50';
      default:
        return 'border-gray-300 hover:border-gray-400';
    }
  };

  return (
    <div className="space-y-4">
      {/* 隐藏的文件输入框 */}
      <input
        ref={fileInputRef}
        type="file"
        accept={accept}
        onChange={handleInputChange}
        className="hidden"
      />

      {/* 上传/预览区域 */}
      <div
        onClick={handleClick}
        onDragOver={handleDragOver}
        onDragLeave={handleDragLeave}
        onDrop={handleDrop}
        className={`
          relative cursor-pointer rounded-xl border-2 border-dashed
          transition-all duration-200 ease-out
          ${getDragStateStyles()}
          ${previewUrl ? 'aspect-video max-w-md' : 'min-h-[300px]'}
        `}
      >
        {/* 预览模式：显示图片 */}
        {previewUrl ? (
          <div className="relative w-full h-full">
            <img
              src={previewUrl}
              alt="预览"
              className="w-full h-full object-contain rounded-xl"
            />

            {/* 删除按钮 */}
            <button
              onClick={(e) => {
                e.stopPropagation();
                handleDelete();
              }}
              className="
                absolute top-2 right-2 w-8 h-8
                bg-red-500 text-white rounded-full
                flex items-center justify-center
                hover:bg-red-600 transition-colors
                shadow-lg
              "
              title="删除图片"
            >
              ✕
            </button>

            {/* 文件信息叠加层 */}
            <div className="absolute bottom-0 left-0 right-0 p-3 bg-gradient-to-t from-black/60 to-transparent rounded-b-xl">
              <p className="text-white text-sm truncate">{currentFile?.name}</p>
              <p className="text-white/80 text-xs">
                {currentFile && formatSize(currentFile.size)}
              </p>
            </div>
          </div>
        ) : (
          /* 空状态：显示上传提示 */
          <div className="flex flex-col items-center justify-center h-full p-8 text-center">
            <div className="w-16 h-16 mb-4 text-gray-400">
              {/* 上传图标 SVG */}
              <svg viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2">
                <path d="M21 15v4a2 2 0 0 1-2 2H5a2 2 0 0 1-2-2v-4" />
                <polyline points="17,8 12,3 7,8" />
                <line x1="12" y1="3" x2="12" y2="15" />
              </svg>
            </div>

            <p className="text-lg font-medium text-gray-700 mb-2">
              拖拽图片到此处，或点击选择
            </p>
            <p className="text-sm text-gray-500">
              支持 JPG、PNG、GIF、WebP 格式
            </p>
            <p className="text-xs text-gray-400 mt-2">
              最大文件大小：{formatSize(maxSize)}
            </p>
          </div>
        )}

        {/* 拖拽中覆盖层 */}
        {dragState === 'dragover' && (
          <div className="absolute inset-0 bg-blue-500/20 rounded-xl flex items-center justify-center">
            <span className="text-blue-600 font-medium">释放以上传</span>
          </div>
        )}
      </div>

      {/* 错误提示 */}
      {error && (
        <div className="p-3 bg-red-50 border border-red-200 rounded-lg text-red-600 text-sm">
          {error}
        </div>
      )}

      {/* 文件详细信息 */}
      {currentFile && (
        <div className="p-4 bg-gray-50 rounded-lg space-y-2">
          <h4 className="font-medium text-gray-900">文件详情</h4>
          <dl className="grid grid-cols-2 gap-2 text-sm">
            <div>
              <dt className="text-gray-500">文件名</dt>
              <dd className="text-gray-900 font-mono truncate">{currentFile.name}</dd>
            </div>
            <div>
              <dt className="text-gray-500">文件大小</dt>
              <dd className="text-gray-900">{formatSize(currentFile.size)}</dd>
            </div>
            <div>
              <dt className="text-gray-500">MIME 类型</dt>
              <dd className="text-gray-900">{currentFile.type}</dd>
            </div>
            <div>
              <dt className="text-gray-500">最后修改</dt>
              <dd className="text-gray-900">
                {new Date(currentFile.lastModified).toLocaleString()}
              </dd>
            </div>
          </dl>
        </div>
      )}
    </div>
  );
}
```

#### 1.2.3 多文件选择器实现

**多文件上传组件：**

```typescript
// components/FileUploader/MultiFileUploader.tsx
"use client";

import React, { useState, useRef, useCallback } from 'react';

/**
 * 多文件上传组件
 *
 * 功能说明：
 * 1. 支持同时选择多个文件
 * 2. 显示所有选中文件的缩略图/图标
 * 3. 支持删除单个文件
 * 4. 显示总体积和文件数量
 *
 * 核心概念：
 * - input[type="file"] 的 multiple 属性允许选择多个文件
 * - FileList 是类数组对象，不是真正的数组
 * - Array.from() 或展开运算符将其转为真正数组
 */
interface MultiFileUploaderProps {
  // 最大文件数量
  maxFiles?: number;
  // 单个文件最大大小（字节）
  maxSizePerFile?: number;
  // 最大总体积（字节）
  maxTotalSize?: number;
  // 接受的文件类型（accept 属性）
  accept?: string;
  // 自定义文件大小格式化
  formatSize?: (bytes: number) => string;
}

interface FileItem {
  // 文件对象
  file: File;
  // 预览 URL（如果是图片）
  previewUrl?: string;
  // 唯一标识符
  id: string;
  // 上传进度（0-100）
  progress: number;
  // 上传状态
  status: 'pending' | 'uploading' | 'success' | 'error';
  // 错误信息
  error?: string;
}

/**
 * 生成唯一 ID 的工具函数
 * 使用时间戳 + 随机数确保唯一性
 */
const generateId = (): string => {
  return `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
};

export default function MultiFileUploader({
  maxFiles = 10,
  maxSizePerFile = 5 * 1024 * 1024,
  maxTotalSize = 50 * 1024 * 1024,
  accept = 'image/*',
  formatSize = (bytes) => `${(bytes / 1024).toFixed(1)} KB`,
}: MultiFileUploaderProps) {
  // 文件列表状态
  const [files, setFiles] = useState<FileItem[]>([]);

  // 错误信息
  const [error, setError] = useState<string>('');

  // 引用文件输入框
  const fileInputRef = useRef<HTMLInputElement>(null);

  /**
   * 验证单个文件
   * @param file 要验证的文件
   * @param currentFiles 当前已选的文件（用于计算总体积）
   * @returns 验证结果
   */
  const validateFile = useCallback((
    file: File,
    currentFiles: File[]
  ): { valid: boolean; error?: string } => {
    // 验证文件类型
    if (accept !== '*' && accept !== 'image/*') {
      const acceptedTypes = accept.split(',').map(t => t.trim());
      if (!acceptedTypes.some(type =>
        file.type === type || // 精确匹配
        (type.endsWith('/*') && file.type.startsWith(type.replace('/*', '/'))) // 通配符匹配
      )) {
        return { valid: false, error: `不支持的文件类型: ${file.type}` };
      }
    }

    // 验证单个文件大小
    if (file.size > maxSizePerFile) {
      return { valid: false, error: `${file.name} 超过大小限制 (${formatSize(maxSizePerFile)})` };
    }

    // 计算当前总体积 + 新文件的体积
    const currentTotal = currentFiles.reduce((sum, f) => sum + f.size, 0);
    if (currentTotal + file.size > maxTotalSize) {
      return { valid: false, error: `总体积超过限制 (${formatSize(maxTotalSize)})` };
    }

    return { valid: true };
  }, [accept, maxSizePerFile, maxTotalSize, formatSize]);

  /**
   * 为文件生成预览 URL
   * 仅图片文件生成缩略图
   */
  const generatePreview = (file: File): string | undefined => {
    if (file.type.startsWith('image/')) {
      // createObjectURL 比 FileReader 更高效，适合预览
      return URL.createObjectURL(file);
    }
    return undefined;
  };

  /**
   * 处理文件选择
   * 验证并添加文件到列表
   */
  const handleFileSelect = (e: React.ChangeEvent<HTMLInputElement>) => {
    const selectedFiles = e.target.files;
    if (!selectedFiles) return;

    setError('');

    // 将 FileList 转为数组
    const fileArray = Array.from(selectedFiles);

    // 检查文件数量限制
    if (files.length + fileArray.length > maxFiles) {
      setError(`最多只能选择 ${maxFiles} 个文件`);
      return;
    }

    // 处理每个文件
    const newFileItems: FileItem[] = [];

    for (const file of fileArray) {
      // 验证文件
      const validation = validateFile(file, files.map(f => f.file));
      if (!validation.valid) {
        setError(prev => prev ? `${prev}\n${validation.error}` : validation.error || '');
        continue;
      }

      // 创建文件项
      const fileItem: FileItem = {
        file,
        id: generateId(),
        previewUrl: generatePreview(file),
        progress: 0,
        status: 'pending',
      };

      newFileItems.push(fileItem);
    }

    // 添加新文件到列表
    setFiles(prev => [...prev, ...newFileItems]);

    // 清空 input，允许重新选择相同文件
    if (fileInputRef.current) {
      fileInputRef.current.value = '';
    }
  };

  /**
   * 删除指定文件
   * 释放预览 URL 以避免内存泄漏
   */
  const handleRemove = (id: string) => {
    setFiles(prev => {
      const file = prev.find(f => f.id === id);
      // 释放预览 URL（如果是图片）
      if (file?.previewUrl) {
        URL.revokeObjectURL(file.previewUrl);
      }
      return prev.filter(f => f.id !== id);
    });
  };

  /**
   * 清空所有文件
   */
  const handleClearAll = () => {
    // 释放所有预览 URL
    files.forEach(f => {
      if (f.previewUrl) {
        URL.revokeObjectURL(f.previewUrl);
      }
    });
    setFiles([]);
    setError('');
  };

  /**
   * 获取文件类型图标
   */
  const getFileIcon = (mimeType: string): string => {
    if (mimeType.startsWith('image/')) return '🖼️';
    if (mimeType.startsWith('video/')) return '🎬';
    if (mimeType.startsWith('audio/')) return '🎵';
    if (mimeType.startsWith('text/')) return '📄';
    if (mimeType.includes('pdf')) return '📕';
    if (mimeType.includes('zip') || mimeType.includes('rar') || mimeType.includes('tar')) return '📦';
    if (mimeType.includes('word') || mimeType.includes('document')) return '📝';
    if (mimeType.includes('excel') || mimeType.includes('sheet')) return '📊';
    return '📁';
  };

  // 计算总体积
  const totalSize = files.reduce((sum, f) => sum + f.file.size, 0);

  return (
    <div className="space-y-4">
      {/* 隐藏的文件输入框 */}
      <input
        ref={fileInputRef}
        type="file"
        multiple
        accept={accept}
        onChange={handleFileSelect}
        className="hidden"
      />

      {/* 操作按钮区 */}
      <div className="flex gap-3">
        <button
          onClick={() => fileInputRef.current?.click()}
          className="px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors"
        >
          选择文件
        </button>

        {files.length > 0 && (
          <button
            onClick={handleClearAll}
            className="px-4 py-2 bg-gray-200 text-gray-700 rounded-lg hover:bg-gray-300 transition-colors"
          >
            清空全部
          </button>
        )}
      </div>

      {/* 统计信息 */}
      {files.length > 0 && (
        <div className="flex gap-4 text-sm text-gray-600">
          <span>文件数量：{files.length} / {maxFiles}</span>
          <span>总体积：{formatSize(totalSize)}</span>
        </div>
      )}

      {/* 错误提示 */}
      {error && (
        <div className="p-3 bg-red-50 border border-red-200 rounded-lg text-red-600 text-sm whitespace-pre-line">
          {error}
        </div>
      )}

      {/* 文件列表 */}
      {files.length > 0 && (
        <div className="grid grid-cols-2 md:grid-cols-3 lg:grid-cols-4 gap-4">
          {files.map((item) => (
            <div
              key={item.id}
              className="relative group bg-gray-50 rounded-lg overflow-hidden border border-gray-200"
            >
              {/* 预览区域 */}
              <div className="aspect-square flex items-center justify-center bg-gray-100">
                {item.previewUrl ? (
                  <img
                    src={item.previewUrl}
                    alt={item.file.name}
                    className="w-full h-full object-cover"
                  />
                ) : (
                  <span className="text-4xl">{getFileIcon(item.file.type)}</span>
                )}
              </div>

              {/* 删除按钮（悬浮显示） */}
              <button
                onClick={() => handleRemove(item.id)}
                className="
                  absolute top-1 right-1 w-6 h-6
                  bg-red-500 text-white rounded-full
                  flex items-center justify-center
                  opacity-0 group-hover:opacity-100
                  transition-opacity
                "
                title="删除"
              >
                ✕
              </button>

              {/* 文件信息 */}
              <div className="p-2">
                <p className="text-xs text-gray-700 truncate" title={item.file.name}>
                  {item.file.name}
                </p>
                <p className="text-xs text-gray-500">
                  {formatSize(item.file.size)}
                </p>
              </div>

              {/* 上传进度条（如果有进度） */}
              {item.status === 'uploading' && item.progress > 0 && (
                <div className="absolute bottom-0 left-0 right-0 h-1 bg-gray-200">
                  <div
                    className="h-full bg-blue-500 transition-all"
                    style={{ width: `${item.progress}%` }}
                  />
                </div>
              )}
            </div>
          ))}
        </div>
      )}

      {/* 空状态提示 */}
      {files.length === 0 && (
        <div className="flex flex-col items-center justify-center py-12 text-gray-400 border-2 border-dashed border-gray-300 rounded-lg">
          <span className="text-5xl mb-4">📂</span>
          <p className="text-gray-500">还没有选择文件</p>
          <p className="text-sm text-gray-400">点击上方按钮或拖拽文件到此区域</p>
        </div>
      )}
    </div>
  );
}

```typescript
// components/FileUploader/AdvancedDragUploader.tsx
"use client";

import React, { useState, useRef, useCallback, useEffect } from 'react';

/**
 * 高级拖拽上传组件
 *
 * 功能说明：
 * 1. 全区域拖拽上传（不仅是指定区域）
 * 2. 拖拽时显示视觉反馈（半透明遮罩）
 * 3. 支持粘贴截图上传（Ctrl+V / Cmd+V）
 * 4. 拖拽进入/离开整个文档时也能响应
 *
 * 核心概念：
 * - 文档级拖拽事件：在文档任意位置拖入/拖出时响应
 * - clipboardData：粘贴事件中访问系统剪贴板内容
 * - DataTransfer：拖拽事件中访问拖拽的数据
 */
interface AdvancedDragUploaderProps {
  // 最大文件数量
  maxFiles?: number;
  // 单个文件最大大小
  maxSize?: number;
  // 接受的文件类型
  accept?: string;
  // 上传回调
  onUpload?: (files: File[]) => void;
}

interface DragOverlayState {
  active: boolean;
  count: number;
}

export default function AdvancedDragUploader({
  maxFiles = 10,
  maxSize = 10 * 1024 * 1024,
  accept = '*',
  onUpload,
}: AdvancedDragUploaderProps) {
  // 文件列表
  const [files, setFiles] = useState<File[]>([]);

  // 拖拽覆盖层状态
  const [overlay, setOverlay] = useState<DragOverlayState>({ active: false, count: 0 });

  // 错误信息
  const [error, setError] = useState<string>('');

  // 是否禁用（用于防止重复上传）
  const [disabled, setDisabled] = useState(false);

  // 输入框引用
  const inputRef = useRef<HTMLInputElement>(null);

  /**
   * 格式化文件大小
   */
  const formatSize = (bytes: number): string => {
    if (bytes < 1024) return `${bytes} B`;
    if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`;
    return `${(bytes / (1024 * 1024)).toFixed(1)} MB`;
  };

  /**
   * 验证文件
   */
  const validateFile = useCallback((file: File): { valid: boolean; error?: string } => {
    // 检查文件大小
    if (file.size > maxSize) {
      return { valid: false, error: `${file.name} 超过大小限制 (${formatSize(maxSize)})` };
    }

    // 检查文件类型（如果指定了类型）
    if (accept && accept !== '*' && accept !== 'image/*') {
      const acceptedTypes = accept.split(',').map(t => t.trim());
      const isAccepted = acceptedTypes.some(type => {
        if (type.endsWith('/*')) {
          const category = type.replace('/*', '/');
          return file.type.startsWith(category);
        }
        return file.type === type;
      });

      if (!isAccepted) {
        return { valid: false, error: `不支持的文件类型: ${file.type}` };
      }
    }

    return { valid: true };
  }, [accept, maxSize]);

  /**
   * 处理文件
   */
  const processFiles = useCallback((fileList: FileList | File[]) => {
    const newFiles = Array.from(fileList);
    setError('');

    // 检查数量限制
    if (files.length + newFiles.length > maxFiles) {
      setError(`最多只能上传 ${maxFiles} 个文件`);
      return;
    }

    // 验证并添加文件
    const validFiles: File[] = [];
    for (const file of newFiles) {
      const validation = validateFile(file);
      if (validation.valid) {
        validFiles.push(file);
      } else {
        setError(prev => prev ? `${prev}\n${validation.error}` : validation.error || '');
      }
    }

    if (validFiles.length > 0) {
      setFiles(prev => [...prev, ...validFiles]);
      onUpload?.(validFiles);
    }
  }, [files.length, maxFiles, validateFile, onUpload]);

  /**
   * 处理粘贴事件
   * 用户在文档任意位置按 Ctrl+V 时触发
   */
  const handlePaste = useCallback((e: ClipboardEvent) => {
    // 获取剪贴板内容
    const clipboardData = e.clipboardData;
    if (!clipboardData) return;

    // 检查是否有文件（从文件管理器粘贴时）
    const items = clipboardData.items;
    const fileItems: File[] = [];

    for (const item of items) {
      // 检查 item 类型是否为文件
      if (item.kind === 'file') {
        // 获取文件对象
        const file = item.getAsFile();
        if (file) {
          fileItems.push(file);
        }
      }
    }

    // 如果有文件，处理它们
    if (fileItems.length > 0) {
      e.preventDefault(); // 阻止默认粘贴行为
      processFiles(fileItems);
    }
  }, [processFiles]);

  /**
   * 处理拖拽进入文档
   * 当用户拖拽文件进入浏览器窗口时触发
   */
  const handleDocumentDragEnter = useCallback((e: DragEvent) => {
    e.preventDefault();
    e.stopPropagation();

    // 获取拖入的文件数量（如果支持的话）
    let count = 0;
    if (e.dataTransfer?.items) {
      count = e.dataTransfer.items.length;
    }

    setOverlay({ active: true, count });
  }, []);

  /**
   * 处理拖拽离开文档
   * 当用户拖拽文件离开浏览器窗口时触发
   */
  const handleDocumentDragLeave = useCallback((e: DragEvent) => {
    e.preventDefault();
    e.stopPropagation();

    // 只有当真正离开文档时才隐藏遮罩
    // 检查 relatedTarget 来确定是否真的离开了
    if (!e.relatedTarget || !document.contains(e.relatedTarget as Node)) {
      setOverlay({ active: false, count: 0 });
    }
  }, []);

  /**
   * 处理拖拽悬停（必须阻止默认行为才能接收 drop 事件）
   */
  const handleDocumentDragOver = useCallback((e: DragEvent) => {
    e.preventDefault();
    e.stopPropagation();
  }, []);

  /**
   * 处理文件放下
   */
  const handleDocumentDrop = useCallback((e: DragEvent) => {
    e.preventDefault();
    e.stopPropagation();

    setOverlay({ active: false, count: 0 });

    // 获取放下的文件
    const droppedFiles = e.dataTransfer?.files;
    if (droppedFiles && droppedFiles.length > 0) {
      processFiles(droppedFiles);
    }
  }, [processFiles]);

  /**
   * 处理 input change
   */
  const handleInputChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    const selectedFiles = e.target.files;
    if (selectedFiles) {
      processFiles(selectedFiles);
    }
    // 清空 input
    e.target.value = '';
  };

  /**
   * 删除文件
   */
  const handleRemove = (index: number) => {
    setFiles(prev => prev.filter((_, i) => i !== index));
  };

  /**
   * 组件挂载时注册文档级事件监听
   */
  useEffect(() => {
    // 注册拖拽事件（监听整个文档）
    document.addEventListener('dragenter', handleDocumentDragEnter);
    document.addEventListener('dragleave', handleDocumentDragLeave);
    document.addEventListener('dragover', handleDocumentDragOver);
    document.addEventListener('drop', handleDocumentDrop);

    // 注册粘贴事件
    document.addEventListener('paste', handlePaste);

    // 组件卸载时移除事件监听
    return () => {
      document.removeEventListener('dragenter', handleDocumentDragEnter);
      document.removeEventListener('dragleave', handleDocumentDragLeave);
      document.removeEventListener('dragover', handleDocumentDragOver);
      document.removeEventListener('drop', handleDocumentDrop);
      document.removeEventListener('paste', handlePaste);
    };
  }, [
    handleDocumentDragEnter,
    handleDocumentDragLeave,
    handleDocumentDragOver,
    handleDocumentDrop,
    handlePaste
  ]);

  /**
   * 获取文件类型图标
   */
  const getFileIcon = (file: File): string => {
    if (file.type.startsWith('image/')) return '🖼️';
    if (file.type.startsWith('video/')) return '🎬';
    if (file.type.startsWith('audio/')) return '🎵';
    if (file.type.startsWith('text/')) return '📄';
    return '📁';
  };

  return (
    <div className="space-y-4">
      {/* 主内容区域 */}
      <div className="relative">
        {/* 中心上传区域 */}
        <div
          onClick={() => inputRef.current?.click()}
          className="
            cursor-pointer p-12 border-2 border-dashed border-gray-300 rounded-xl
            hover:border-blue-400 hover:bg-blue-50/50
            transition-all duration-200
          "
        >
          <div className="flex flex-col items-center text-center">
            <span className="text-5xl mb-4">📤</span>
            <p className="text-lg font-medium text-gray-700">
              点击选择文件，或将文件拖入此处
            </p>
            <p className="text-sm text-gray-500 mt-2">
              支持拖拽上传、粘贴截图上传（Ctrl+V）
            </p>
          </div>
        </div>

        {/* 隐藏的文件输入框 */}
        <input
          ref={inputRef}
          type="file"
          multiple
          accept={accept}
          onChange={handleInputChange}
          className="hidden"
        />

        {/* 拖拽覆盖层 */}
        {overlay.active && (
          <div className="
            absolute inset-0 z-50
            bg-blue-500/90 rounded-xl
            flex flex-col items-center justify-center
            backdrop-blur-sm
          ">
            <span className="text-6xl mb-4">📥</span>
            <p className="text-xl font-medium text-white">
              释放以上传 {overlay.count} 个文件
            </p>
            <p className="text-sm text-white/80 mt-2">
              松开鼠标即可完成上传
            </p>
          </div>
        )}
      </div>

      {/* 错误提示 */}
      {error && (
        <div className="p-3 bg-red-50 border border-red-200 rounded-lg text-red-600 text-sm whitespace-pre-line">
          {error}
        </div>
      )}

      {/* 文件列表 */}
      {files.length > 0 && (
        <div className="space-y-3">
          <h3 className="font-medium text-gray-900">
            已选择 {files.length} 个文件
          </h3>

          <div className="space-y-2">
            {files.map((file, index) => (
              <div
                key={`${file.name}-${index}`}
                className="flex items-center gap-3 p-3 bg-gray-50 rounded-lg"
              >
                {/* 预览/图标 */}
                {file.type.startsWith('image/') ? (
                  <img
                    src={URL.createObjectURL(file)}
                    alt={file.name}
                    className="w-12 h-12 object-cover rounded-lg"
                  />
                ) : (
                  <div className="w-12 h-12 flex items-center justify-center bg-gray-200 rounded-lg text-2xl">
                    {getFileIcon(file)}
                  </div>
                )}

                {/* 文件信息 */}
                <div className="flex-1 min-w-0">
                  <p className="text-sm font-medium text-gray-900 truncate">
                    {file.name}
                  </p>
                  <p className="text-xs text-gray-500">
                    {formatSize(file.size)} | {file.type || '未知类型'}
                  </p>
                </div>

                {/* 删除按钮 */}
                <button
                  onClick={() => handleRemove(index)}
                  className="
                    w-8 h-8 flex items-center justify-center
                    text-gray-400 hover:text-red-500
                    hover:bg-red-50 rounded-lg
                    transition-colors
                  "
                >
                  ✕
                </button>
              </div>
            ))}
          </div>
        </div>
      )}

      {/* 使用提示 */}
      <div className="text-xs text-gray-400 space-y-1">
        <p>💡 提示：</p>
        <ul className="list-disc list-inside space-y-0.5">
          <li>直接从文件管理器拖拽文件到浏览器窗口即可上传</li>
          <li>截图后按 <kbd className="px-1 py-0.5 bg-gray-100 rounded">Ctrl</kbd>+<kbd className="px-1 py-0.5 bg-gray-100 rounded">V</kbd> 粘贴截图</li>
          <li>从网页图片右键复制也可粘贴上传</li>
        </ul>
      </div>
    </div>
  );
}

### 1.4 上传进度与分片上传

#### 1.4.1 带进度的文件上传组件

```typescript
// components/FileUploader/ProgressUploader.tsx
"use client";

import React, { useState, useRef, useCallback, useEffect } from 'react';

/**
 * 带进度的文件上传组件
 *
 * 功能说明：
 * 1. 实时显示上传进度（百分比）
 * 2. 视觉进度条
 * 3. 支持取消上传
 * 4. 支持暂停/恢复上传
 * 5. 模拟断点续传（记录已上传大小）
 *
 * 核心概念：
 * - XMLHttpRequest.upload.onprogress：监听上传进度
 * - abort()：取消上传请求
 * - 模拟断点续传：记录已上传字节数，下次从该位置继续
 */

interface UploadTask {
  // 唯一标识
  id: string;
  // 文件对象
  file: File;
  // 预览 URL
  previewUrl?: string;
  // 上传进度（0-100）
  progress: number;
  // 已上传字节数
  uploadedBytes: number;
  // 上传状态
  status: 'pending' | 'uploading' | 'paused' | 'completed' | 'error' | 'cancelled';
  // 错误信息
  error?: string;
  // xhr 引用（用于取消）
  xhr?: XMLHttpRequest;
  // 模拟的断点（已确认上传的字节数）
  confirmedBytes: number;
}

interface ProgressUploaderProps {
  // 上传接口地址
  uploadUrl?: string;
  // 最大并发数
  maxConcurrency?: number;
  // 单个文件最大大小
  maxSize?: number;
  // 自定义上传函数
  customUpload?: (file: File, onProgress: (progress: number) => void) => Promise<string>;
}

const generateId = (): string => `${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;

export default function ProgressUploader({
  uploadUrl = '/api/upload',
  maxConcurrency = 2,
  maxSize = 100 * 1024 * 1024,
  customUpload,
}: ProgressUploaderProps) {
  // 上传任务列表
  const [tasks, setTasks] = useState<UploadTask[]>([]);

  // 正在进行的任务数
  const [activeCount, setActiveCount] = useState(0);

  // 上传完成回调
  const [completedUrls, setCompletedUrls] = useState<string[]>([]);

  /**
   * 格式化文件大小
   */
  const formatSize = (bytes: number): string => {
    if (bytes < 1024) return `${bytes} B`;
    if (bytes < 1024 * 1024) return `${(bytes / 1024).toFixed(1)} KB`;
    if (bytes < 1024 * 1024 * 1024) return `${(bytes / (1024 * 1024)).toFixed(1)} MB`;
    return `${(bytes / (1024 * 1024 * 1024)).toFixed(2)} GB`;
  };

  /**
   * 模拟上传过程
   * 实际项目中替换为真实的 axios/fetch 上传
   */
  const simulateUpload = useCallback((
    task: UploadTask,
    onProgress: (progress: number, uploadedBytes: number) => void,
    signal: AbortSignal
  ): Promise<string> => {
    return new Promise((resolve, reject) => {
      // 检查是否已取消
      if (signal.aborted) {
        reject(new Error('上传已取消'));
        return;
      }

      // 模拟分片上传：每 100ms 上传 5% 的数据
      const totalChunks = 20;
      const chunkSize = task.file.size / totalChunks;
      let currentChunk = Math.floor(task.confirmedBytes / chunkSize);

      // 随机上传成功后的文件 URL
      const resultUrl = `https://example.com/uploads/${task.id}/${task.file.name}`;

      const interval = setInterval(() => {
        // 检查是否已取消
        if (signal.aborted) {
          clearInterval(interval);
          reject(new Error('上传已取消'));
          return;
        }

        currentChunk++;

        // 计算进度
        const progress = Math.min((currentChunk / totalChunks) * 100, 100);
        const uploadedBytes = Math.min(currentChunk * chunkSize, task.file.size);

        onProgress(progress, uploadedBytes);

        // 上传完成
        if (currentChunk >= totalChunks) {
          clearInterval(interval);
          resolve(resultUrl);
        }
      }, 100); // 每 100ms 上传一个"块"
    });
  }, []);

  /**
   * 开始上传任务
   */
  const startUpload = useCallback(async (taskId: string) => {
    const task = tasks.find(t => t.id === taskId);
    if (!task) return;

    // 创建 AbortController 用于取消
    const abortController = new AbortController();
    const signal = abortController.signal;

    // 更新任务状态
    setTasks(prev => prev.map(t =>
      t.id === taskId
        ? { ...t, status: 'uploading', xhr: undefined }
        : t
    ));

    try {
      // 执行上传（这里使用模拟上传，实际用 axios）
      const resultUrl = await simulateUpload(
        { ...task, status: 'uploading' },
        (progress, uploadedBytes) => {
          // 更新进度
          setTasks(prev => prev.map(t =>
            t.id === taskId
              ? { ...t, progress, uploadedBytes, confirmedBytes: uploadedBytes }
              : t
          ));
        },
        signal
      );

      // 上传成功
      setTasks(prev => prev.map(t =>
        t.id === taskId
          ? { ...t, status: 'completed', progress: 100 }
          : t
      ));

      setCompletedUrls(prev => [...prev, resultUrl]);

      // 触发下一个任务
      setActiveCount(c => Math.max(0, c - 1));

    } catch (error: any) {
      if (error.message === '上传已取消') {
        setTasks(prev => prev.map(t =>
          t.id === taskId
            ? { ...t, status: 'cancelled' }
            : t
        ));
      } else {
        setTasks(prev => prev.map(t =>
          t.id === taskId
            ? { ...t, status: 'error', error: error.message }
            : t
        ));
      }

      setActiveCount(c => Math.max(0, c - 1));
    }
  }, [tasks, simulateUpload]);

  /**
   * 添加上传任务
   */
  const addTasks = useCallback((fileList: FileList | File[]) => {
    const newTasks: UploadTask[] = Array.from(fileList).map(file => ({
      id: generateId(),
      file,
      previewUrl: file.type.startsWith('image/') ? URL.createObjectURL(file) : undefined,
      progress: 0,
      uploadedBytes: 0,
      confirmedBytes: 0,
      status: 'pending',
    })).filter(task => task.file.size <= maxSize);

    setTasks(prev => [...prev, ...newTasks]);
  }, [maxSize]);

  /**
   * 暂停指定任务（模拟）
   */
  const pauseTask = useCallback((taskId: string) => {
    // 模拟暂停：实际应用中需要中止 xhr 请求并记录进度
    setTasks(prev => prev.map(t =>
      t.id === taskId && t.status === 'uploading'
        ? { ...t, status: 'paused' }
        : t
    ));
  }, []);

  /**
   * 恢复指定任务
   */
  const resumeTask = useCallback((taskId: string) => {
    startUpload(taskId);
  }, [startUpload]);

  /**
   * 取消指定任务
   */
  const cancelTask = useCallback((taskId: string) => {
    const task = tasks.find(t => t.id === taskId);
    if (task?.xhr) {
      task.xhr.abort(); // 取消 xhr 请求
    }

    setTasks(prev => prev.map(t =>
      t.id === taskId
        ? { ...t, status: 'cancelled' }
        : t
    ));
  }, [tasks]);

  /**
   * 删除指定任务
   */
  const removeTask = useCallback((taskId: string) => {
    cancelTask(taskId); // 先取消
    setTasks(prev => {
      const task = prev.find(t => t.id === taskId);
      // 释放预览 URL
      if (task?.previewUrl) {
        URL.revokeObjectURL(task.previewUrl);
      }
      return prev.filter(t => t.id !== taskId);
    });
  }, [cancelTask]);

  /**
   * 重试失败的任务
   */
  const retryTask = useCallback((taskId: string) => {
    setTasks(prev => prev.map(t =>
      t.id === taskId
        ? { ...t, status: 'pending', progress: 0, uploadedBytes: 0, error: undefined }
        : t
    ));
  }, []);

  /**
   * 清空已完成/已取消的任务
   */
  const clearCompleted = useCallback(() => {
    setTasks(prev => {
      // 释放所有预览 URL
      prev.forEach(t => {
        if (t.previewUrl) {
          URL.revokeObjectURL(t.previewUrl);
        }
      });
      return prev.filter(t => t.status !== 'completed' && t.status !== 'cancelled');
    });
  }, []);

  /**
   * 处理文件选择
   */
  const handleFileSelect = (e: React.ChangeEvent<HTMLInputElement>) => {
    const files = e.target.files;
    if (files) {
      addTasks(files);
    }
    e.target.value = '';
  };

  // 自动开始pending的任务
  useEffect(() => {
    const pendingTasks = tasks.filter(t => t.status === 'pending');
    if (pendingTasks.length > 0 && activeCount < maxConcurrency) {
      // 启动下一个任务
      startUpload(pendingTasks[0].id);
      setActiveCount(c => c + 1);
    }
  }, [tasks, activeCount, maxConcurrency, startUpload]);

  /**
   * 获取状态对应的颜色
   */
  const getStatusColor = (status: UploadTask['status']): string => {
    switch (status) {
      case 'pending': return 'bg-gray-400';
      case 'uploading': return 'bg-blue-500';
      case 'paused': return 'bg-yellow-500';
      case 'completed': return 'bg-green-500';
      case 'error': return 'bg-red-500';
      case 'cancelled': return 'bg-gray-400';
      default: return 'bg-gray-400';
    }
  };

  /**
   * 获取状态对应的文字
   */
  const getStatusText = (status: UploadTask['status']): string => {
    switch (status) {
      case 'pending': return '等待中';
      case 'uploading': return '上传中';
      case 'paused': return '已暂停';

      case 'error': return '上传失败';

  return (

      <div className="flex gap-3">

          cursor-pointer hover:bg-blue-600
          transition-colors
        ">
          <span>选择文件</span>
          <input
            type="file"
            multiple
            onChange={handleFileSelect}
            className="hidden"
          />
        </label>

        {tasks.length > 0 && (
          <button
            onClick={clearCompleted}
            className="px-4 py-2 bg-gray-200 text-gray-700 rounded-lg hover:bg-gray-300 transition-colors"
          >
            清空已完成
          </button>
        )}
      </div>

      {/* 统计信息 */}
      {tasks.length > 0 && (
        <div className="flex gap-4 text-sm">
          <span className="text-gray-600">

          <span className="text-gray-600">

            失败: {tasks.filter(t => t.status === 'error').length}

```typescript
// utils/ChunkUploader.ts

/**
 * 分片上传工具类
 *
 * 功能说明：
 * 1. 将大文件分割成多个小块
 * 2. 并发上传多个分片
 * 3. 支持断点续传（记录已上传分片）
 * 4. 所有分片上传完成后通知服务器合并
 *
 * 核心概念：
 * - Blob.slice()：分割文件
 * - Promise.all()：并发上传
 * - 唯一标识：使用 file.size + file.name + lastModified 生成
 */

interface ChunkInfo {
  // 分片索引
  index: number;
  // 分片数据
  blob: Blob;
  // 分片大小
  size: number;
  // 分片 hash（用于验证）
  hash: string;
}

interface UploadProgress {
  // 已上传分片数
  uploadedChunks: number;
  // 总分片数
  totalChunks: number;
  // 进度百分比
  percentage: number;
  // 已上传字节数
  uploadedBytes: number;
  // 总字节数
  totalBytes: number;
}

interface ChunkUploaderOptions {
  // 分片大小（默认 2MB）
  chunkSize?: number;
  // 并发上传数
  concurrency?: number;
  // 上传接口
  uploadUrl?: string;
  // 合并接口
  mergeUrl?: string;
  // 自定义上传函数
  uploadRequest?: (chunk: Blob, hash: string, index: number) => Promise<void>;
  // 自定义合并函数
  mergeRequest?: (hash: string, totalChunks: number, filename: string) => Promise<string>;
  // 获取已上传分片列表（用于断点续传）
  getUploadedChunks?: (fileHash: string) => Promise<number[]>;
  // 保存已上传分片记录
  saveUploadedChunks?: (fileHash: string, chunks: number[]) => Promise<void>;
}

/**
 * 生成分片 Hash 的工具函数
 * 用于唯一标识每个分片
 */
const generateChunkHash = (fileHash: string, chunkIndex: number): string => {
  return `${fileHash}-chunk-${chunkIndex}`;
};

/**
 * 生成文件 Hash 的工具函数
 * 基于文件大小、名称、最后修改时间
 */
const generateFileHash = async (file: File): Promise<string> => {
  // 使用 File 的属性生成简单 hash
  // 实际应用中应使用 crypto.subtle.digest 或 spark-md5
  const str = `${file.name}-${file.size}-${file.lastModified}`;
  const encoder = new TextEncoder();
  const data = encoder.encode(str);

  // 使用 SubtleCrypto API 计算 SHA-256
  const hashBuffer = await crypto.subtle.digest('SHA-256', data);
  const hashArray = Array.from(new Uint8Array(hashBuffer));
  const hashHex = hashArray.map(b => b.toString(16).padStart(2, '0')).join('');

  return hashHex;
};

/**
 * 分片上传类
 */
export class ChunkUploader {
  private file: File;
  private options: Required<ChunkUploaderOptions>;
  private fileHash: string;
  private chunks: ChunkInfo[] = [];

  constructor(file: File, options: ChunkUploaderOptions = {}) {
    this.file = file;

    // 设置默认选项
    this.options = {
      chunkSize: options.chunkSize || 2 * 1024 * 1024, // 2MB
      concurrency: options.concurrency || 3,
      uploadUrl: options.uploadUrl || '/api/upload-chunk',
      mergeUrl: options.mergeUrl || '/api/merge-chunks',
      uploadRequest: options.uploadRequest || this.defaultUploadRequest.bind(this),
      mergeRequest: options.mergeRequest || this.defaultMergeRequest.bind(this),
      getUploadedChunks: options.getUploadedChunks || (async () => []),
      saveUploadedChunks: options.saveUploadedChunks || (async () => {}),
    };
  }

  /**
   * 获取文件
   */
  getFile(): File {
    return this.file;
  }

  /**
   * 获取文件 Hash
   */
  async getFileHash(): Promise<string> {
    if (!this.fileHash) {
      this.fileHash = await generateFileHash(this.file);
    }
    return this.fileHash;
  }

  /**
   * 生成分片列表
   * 将文件分割成固定大小的块
   */
  async generateChunks(): Promise<ChunkInfo[]> {
    const chunks: ChunkInfo[] = [];
    const totalChunks = Math.ceil(this.file.size / this.options.chunkSize);

    for (let i = 0; i < totalChunks; i++) {
      // 计算分片的起始和结束位置
      const start = i * this.options.chunkSize;
      const end = Math.min(start + this.options.chunkSize, this.file.size);

      // 提取分片数据
      const blob = this.file.slice(start, end);

      // 生成分片 hash
      const chunkHash = generateChunkHash(await this.getFileHash(), i);

      chunks.push({
        index: i,
        blob,
        size: blob.size,
        hash: chunkHash,
      });
    }

    this.chunks = chunks;
    return chunks;
  }

  /**
   * 获取总分片数
   */
  getTotalChunks(): number {
    return Math.ceil(this.file.size / this.options.chunkSize);
  }

  /**
   * 默认的上传请求（可替换）
   */
  private async defaultUploadRequest(
    chunk: Blob,
    hash: string,
    index: number
  ): Promise<void> {
    const formData = new FormData();
    formData.append('chunk', chunk);
    formData.append('hash', hash);
    formData.append('index', index.toString());

    // 使用 fetch 上传
    const response = await fetch(this.options.uploadUrl, {
      method: 'POST',
      body: formData,
    });

    if (!response.ok) {
      throw new Error(`分片 ${index} 上传失败`);
    }
  }

  /**
   * 默认的合并请求（可替换）
   */
  private async defaultMergeRequest(
    hash: string,
    totalChunks: number,
    filename: string
  ): Promise<string> {
    const response = await fetch(this.options.mergeUrl, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ hash, totalChunks, filename }),
    });

    if (!response.ok) {
      throw new Error('合并请求失败');
    }

    const data = await response.json();
    return data.url; // 返回合并后的文件 URL
  }

  /**
   * 上传单个分片（带重试）
   */
  private async uploadChunkWithRetry(
    chunk: ChunkInfo,
    maxRetries: number = 3
  ): Promise<void> {
    let lastError: Error | undefined;

    for (let i = 0; i < maxRetries; i++) {
      try {
        await this.options.uploadRequest(chunk.blob, chunk.hash, chunk.index);
        return; // 上传成功
      } catch (error) {
        lastError = error as Error;

        // 等待后重试（指数退避）
        if (i < maxRetries - 1) {
          await new Promise(resolve =>
            setTimeout(resolve, Math.pow(2, i) * 1000)
          );
        }
      }
    }

    throw lastError || new Error(`分片 ${chunk.index} 上传失败`);
  }

  /**
   * 执行分片上传
   * @param onProgress 进度回调
   * @returns 合并后的文件 URL
   */
  async upload(
    onProgress?: (progress: UploadProgress) => void
  ): Promise<string> {
    // 如果还没有生成分片，先生成
    if (this.chunks.length === 0) {
      await this.generateChunks();
    }

    const fileHash = await this.getFileHash();
    const totalChunks = this.chunks.length;

    // 获取已上传的分片列表（断点续传）
    const uploadedChunks = await this.options.getUploadedChunks(fileHash);

    // 过滤出未上传的分片
    const pendingChunks = this.chunks.filter(c => !uploadedChunks.includes(c.index));

    // 记录已上传的分片
    const confirmedChunks = [...uploadedChunks];

    // 上传进度
    const uploadedCount = { value: uploadedChunks.length };

    /**
     * 上传单个分片并更新进度
     */
    const uploadAndTrack = async (chunk: ChunkInfo): Promise<void> => {
      await this.uploadChunkWithRetry(chunk);
      confirmedChunks.push(chunk.index);
      uploadedCount.value++;

      // 触发进度回调
      onProgress?.({
        uploadedChunks: uploadedCount.value,
        totalChunks,
        percentage: Math.round((uploadedCount.value / totalChunks) * 100),
        uploadedBytes: Math.min(
          uploadedCount.value * this.options.chunkSize,
          this.file.size
        ),
        totalBytes: this.file.size,
      });

      // 保存已上传分片记录
      await this.options.saveUploadedChunks(fileHash, confirmedChunks);
    };

    /**
     * 并发控制的上传函数
     * 使用 Promise.all + concurrency 限制并发数
     */
    const uploadWithConcurrency = async (): Promise<void> => {
      // 将分片按批次处理
      for (let i = 0; i < pendingChunks.length; i += this.options.concurrency) {
        const batch = pendingChunks.slice(i, i + this.options.concurrency);
        await Promise.all(batch.map(chunk => uploadAndTrack(chunk)));
      }
    };

    // 执行上传
    await uploadWithConcurrency();

    // 所有分片上传完成，通知服务器合并
    const resultUrl = await this.options.mergeRequest(
      fileHash,
      totalChunks,
      this.file.name
    );

    return resultUrl;
  }

  /**
   * 取消上传
   * 注意：这只是一个标记，实际的 xhr 请求需要在外部管理
   */

    setResult(null);

        ...(options || {}),

   * 恢复上传
   */

    if (uploaderRef.current && status === 'paused') {
      setStatus('uploading');
      // 实际应用中需要从保存的状态继续
    }
  };

  /**
   * 取消上传
   */

      uploaderRef.current.cancel();
      setStatus('idle');
      setProgress(null);
    }
  };

    status,
    result,
    error,

// utils/ImageCompressor.ts

/**
 * 图片压缩工具类
 *
 * 功能说明：
 * 1. 使用 Canvas 绘制图片
 * 2. 调整图片尺寸和质量
 * 3. 输出压缩后的 Blob
 *
 * 核心概念：
 * - Canvas.toBlob()：将 Canvas 内容转为 Blob
 * - createImageBitmap()：异步加载图片
 * - drawImage()：在 Canvas 上绘制图片（可缩放）
 */

interface CompressOptions {
  // 输出宽度（如果指定，会按比例调整高度）
  width?: number;
  // 输出高度（如果指定，会按比例调整宽度）
  height?: number;
  // 质量（0-1），仅对 JPEG 和 WebP 有效
  quality?: number;
  // 输出格式
  mimeType?: 'image/jpeg' | 'image/png' | 'image/webp';
  // 最大宽高限制（如果图片超过限制则缩小）
  maxWidth?: number;
  // 最大宽高限制（如果图片超过限制则缩小）
  maxHeight?: number;
  // 是否使用worker（避免阻塞主线程）
  useWorker?: boolean;
}

interface CompressResult {
  // 压缩后的 Blob
  blob: Blob;
  // 压缩后的宽度
  width: number;
  // 压缩后的高度
  height: number;
  // 原始文件大小
  originalSize: number;
  // 压缩后大小
  compressedSize: number;
  // 压缩比
  ratio: number;
}

/**
 * 加载图片为 ImageBitmap
 */
const loadImage = (src: string | Blob): Promise<ImageBitmap> => {
  return new Promise((resolve, reject) => {
    if (src instanceof Blob) {
      // 从 Blob 加载
      const url = URL.createObjectURL(src);
      createImageBitmap(new Image())
        .catch(() => {
          // 如果 Image 无法加载，尝试用 canvas
          const img = new Image();
          img.onload = () => URL.revokeObjectURL(url);
          img.onerror = reject;
          img.src = url;
        });

      // 使用 fetch + createImageBitmap
      fetch(url)
        .then(res => res.blob())
        .then(blob => createImageBitmap(blob))
        .then(resolve)
        .catch(reject)
        .finally(() => URL.revokeObjectURL(url));
    } else {
      // 从 URL 加载
      createImageBitmap(new Image())
        .catch(() => {
          const img = new Image();
          img.onerror = reject;
          img.src = src;
        });

      fetch(src)
        .then(res => res.blob())
        .then(blob => createImageBitmap(blob))
        .then(resolve)
        .catch(reject);
    }
  });
};

/**
 * 计算输出尺寸
 * 按比例缩放以适应最大尺寸限制
 */
const calculateOutputSize = (
  originalWidth: number,
  originalHeight: number,
  options: CompressOptions
): { width: number; height: number } => {
  let { width, height } = { width: originalWidth, height: originalHeight };

  // 如果指定了 maxWidth/maxHeight，进行缩放
  if (options.maxWidth || options.maxHeight) {
    const maxW = options.maxWidth || Infinity;
    const maxH = options.maxHeight || Infinity;

    // 计算缩放比例
    const ratioW = maxW / width;
    const ratioH = maxH / height;
    const ratio = Math.min(ratioW, ratioH, 1); // 不放大，只缩小

    if (ratio < 1) {
      width = Math.round(width * ratio);
      height = Math.round(height * ratio);
    }
  }

  // 如果指定了 width/height，按比例调整
  if (options.width || options.height) {
    if (options.width && !options.height) {
      // 只指定宽度，按比例计算高度
      const ratio = options.width / width;
      height = Math.round(height * ratio);
      width = options.width;
    } else if (options.height && !options.width) {
      // 只指定高度，按比例计算宽度
      const ratio = options.height / height;
      width = Math.round(width * ratio);
      height = options.height;
    } else if (options.width && options.height) {
      // 两者都指定，直接使用
      width = options.width;
      height = options.height;
    }
  }

  return { width, height };
};

/**
 * 压缩图片
 * @param input 图片源（File/Blob/URL/DataURL）
 * @param options 压缩选项
 * @returns Promise<CompressResult>
 */
export const compressImage = async (
  input: File | Blob | string,
  options: CompressOptions = {}
): Promise<CompressResult> => {
  // 设置默认值
  const {
    quality = 0.8,
    mimeType = 'image/jpeg',
    maxWidth = 4096,
    maxHeight = 4096,
  } = options;

  // 1. 加载图片
  let imageSrc: string;

  if (input instanceof File || input instanceof Blob) {
    imageSrc = URL.createObjectURL(input);
  } else {
    imageSrc = input;
  }

  // 创建图片元素
  const img = new Image();

  await new Promise<void>((resolve, reject) => {
    img.onload = () => resolve();
    img.onerror = () => reject(new Error('图片加载失败'));
    img.src = imageSrc;
  });

  // 2. 计算输出尺寸
  const { width, height } = calculateOutputSize(img.naturalWidth, img.naturalHeight, {
    ...options,
    maxWidth,
    maxHeight,
  });

  // 3. 创建 Canvas 并绘制
  const canvas = document.createElement('canvas');
  canvas.width = width;
  canvas.height = height;

  const ctx = canvas.getContext('2d');
  if (!ctx) {
    throw new Error('无法获取 Canvas 2D 上下文');
  }

  // 使用高质量模式绘制
  // imageSmoothingEnabled 和 imageSmoothingQuality 是关键
  ctx.imageSmoothingEnabled = true;
  ctx.imageSmoothingQuality = 'high';

  // 绘制图片（自动处理缩放）
  ctx.drawImage(img, 0, 0, width, height);

  // 4. 导出为 Blob
  const blob: Blob = await new Promise((resolve, reject) => {
    canvas.toBlob(
      (blob) => {
        if (blob) {
          resolve(blob);
        } else {
          reject(new Error('图片压缩失败'));
        }
      },
      mimeType,
      quality
    );
  });

  // 5. 计算压缩结果
  const originalSize = input instanceof File ? input.size :
    (input instanceof Blob ? input.size : 0);

  return {
    blob,
    width,
    height,
    originalSize,
    compressedSize: blob.size,
    ratio: originalSize > 0 ? (1 - blob.size / originalSize) : 0,
  };
};

/**
 * 压缩图片（简化版，用于小文件）
 * @param file 图片文件
 * @param maxWidth 最大宽度
 * @param maxHeight 最大高度
 * @param quality 质量（0-1）
 * @returns Promise<Blob>
 */
export const compressImageSimple = (
  file: File,
  maxWidth: number = 1920,
  maxHeight: number = 1080,
  quality: number = 0.8
): Promise<Blob> => {
  return new Promise((resolve, reject) => {
    // 创建 FileReader 读取文件
    const reader = new FileReader();

    reader.onload = (e) => {
      const img = new Image();

      img.onload = () => {
        // 计算缩放比例
        let width = img.width;
        let height = img.height;

        if (width > maxWidth) {
          height = (height * maxWidth) / width;
          width = maxWidth;
        }

        if (height > maxHeight) {
          width = (width * maxHeight) / height;
          height = maxHeight;
        }

        // 创建 Canvas
        const canvas = document.createElement('canvas');
        canvas.width = width;
        canvas.height = height;

        const ctx = canvas.getContext('2d');
        if (!ctx) {
          reject(new Error('无法获取 Canvas 上下文'));
          return;
        }

        // 绘制
        ctx.drawImage(img, 0, 0, width, height);

        // 导出
        canvas.toBlob(
          (blob) => {
            if (blob) {
              resolve(blob);
            } else {
              reject(new Error('压缩失败'));
            }
          },
          'image/jpeg',
          quality
        );
      };

      img.onerror = () => reject(new Error('图片加载失败'));
      img.src = e.target?.result as string;
    };

    reader.onerror = () => reject(new Error('文件读取失败'));
    reader.readAsDataURL(file);
  });
};

/**
 * React Hook for 图片压缩
 */
export function useImageCompressor() {
  const [compressing, setCompressing] = useState(false);
  const [result, setResult] = useState<CompressResult | null>(null);
  const [error, setError] = useState<string | null>(null);

  const compress = async (file: File, options?: CompressOptions) => {
    setCompressing(true);
    setError(null);

    try {
      const compressed = await compressImage(file, options);
      setResult(compressed);
      return compressed;
    } catch (err) {
      setError((err as Error).message);
      throw err;
    } finally {
      setCompressing(false);
    }
  };

  const reset = () => {
    setResult(null);
    setError(null);
  };

  return {
    compressing,
    result,
    error,
    compress,
    reset,
  };
}

#### 2.4.1 Markdown 解析器实现

```typescript
// utils/MarkdownParser.ts

/**
 * 简化的 Markdown 解析器
 *
 * 功能说明：
 * 1. 将 Markdown 文本解析为 HTML
 * 2. 支持常用语法：标题、列表、代码块、链接、图片、引用等
 *
 * 核心概念：
 * - 正则表达式匹配 Markdown 语法
 * - 递归解析嵌套结构
 * - XSS 防护：转义 HTML 特殊字符
 */

/**
 * 转义 HTML 特殊字符
 * 防止 XSS 攻击
 *
 * @param text 原始文本
 * @returns 转义后的文本
 */
export const escapeHtml = (text: string): string => {
  const escapeMap: Record<string, string> = {
    '&': '&amp;',
    '<': '&lt;',
    '>': '&gt;',
    '"': '&quot;',
    "'": '&#039;',
  };

  // 使用 replace 和正则表达式进行全局替换
  return text.replace(/[&<>"']/g, (char) => escapeMap[char]);
};

/**
 * Markdown 解析器类
 */
export class MarkdownParser {
  // 配置选项
  private options: {
    // 是否启用代码高亮
    highlightCode: boolean;
    // 是否启用 GFM（GitHub Flavored Markdown）
    gfm: boolean;
    // 自定义链接处理
    linkHandler?: (href: string, text: string) => string;
  };

  constructor(options: Partial<typeof MarkdownParser.prototype.options> = {}) {
    this.options = {
      highlightCode: true,
      gfm: true,
      ...options,
    };
  }

  /**
   * 解析 Markdown 文本为 HTML
   * @param markdown Markdown 文本
   * @returns HTML 字符串
   */
  parse(markdown: string): string {
    // 预处理：移除 BOM 和多余空白
    let text = markdown.replace(/^\uFEFF/, '');

    // 步骤1：处理代码块（先处理，因为代码块内不应解析其他语法）
    text = this.parseCodeBlocks(text);

    // 步骤2：处理行级代码（inline code）
    text = this.parseInlineCode(text);

    // 步骤3：处理图片（需要在链接之前，因为图片语法包含链接）
    text = this.parseImages(text);

    // 步骤4：处理链接
    text = this.parseLinks(text);

    // 步骤5：处理标题
    text = this.parseHeadings(text);

    // 步骤6：处理引用块
    text = this.parseBlockquotes(text);

    // 步骤7：处理无序列表
    text = this.parseUnorderedLists(text);

    // 步骤8：处理有序列表
    text = this.parseOrderedLists(text);

    // 步骤9：处理粗体和斜体
    text = this.parseEmphasis(text);

    // 步骤10：处理水平线
    text = this.parseHorizontalRules(text);

    // 步骤11：处理换行
    text = this.parseLineBreaks(text);

    return text;
  }

  /**
   * 处理代码块
   * 支持 ```language 或 ~~~language 语法
   */
  private parseCodeBlocks(text: string): string {
    // 匹配代码块：```language\ncode\n``` 或 ```\ncode\n```
    const codeBlockRegex = /```(\w*)\n([\s\S]*?)```/g;

    return text.replace(codeBlockRegex, (_, language, code) => {
      // 解码 HTML 实体（因为代码块内容应该保持原样）
      const decodedCode = code
        .replace(/&lt;/g, '<')
        .replace(/&gt;/g, '>')
        .replace(/&amp;/g, '&');

      // 转义 HTML
      const escapedCode = escapeHtml(decodedCode);

      // 简单语法高亮（关键词）
      const highlighted = this.highlightSyntax(escapedCode, language);

      return `<pre class="code-block" data-language="${language}"><code>${highlighted}</code></pre>`;
    });
  }

  /**
   * 简单语法高亮
   * 实际应用中应使用 highlight.js 或 Prism
   */
  private highlightSyntax(code: string, language: string): string {
    if (!this.options.highlightCode) {
      return code;
    }

    // 根据语言添加不同的类名
    // 实际项目中可以使用 highlight.js 进行真正的语法高亮
    const languageClass = language ? `language-${language}` : '';

    // 简单的高亮模式
    let highlighted = code;

    // 高亮字符串
    highlighted = highlighted.replace(
      /(["'`])(?:(?!\1)[^\\]|\\.)*\1/g,
      '<span class="token string">$&</span>'
    );

    // 高亮关键字
    const keywords = this.getKeywords(language);
    if (keywords.length > 0) {
      highlighted = highlighted.replace(
        new RegExp(`\\b(${keywords.join('|')})\\b`, 'g'),
        '<span class="token keyword">$1</span>'
      );
    }

    // 高亮数字
    highlighted = highlighted.replace(
      /\b(\d+\.?\d*)\b/g,
      '<span class="token number">$1</span>'
    );

    // 高亮注释
    if (language === 'html' || language === 'xml') {
      highlighted = highlighted.replace(
        /(&lt;!--[\s\S]*?--&gt;)/g,
        '<span class="token comment">$1</span>'
      );
    } else {
      highlighted = highlighted.replace(
        /(\/\/[^\n]*|\/\*[\s\S]*?\*\/)/g,
        '<span class="token comment">$1</span>'
      );
    }

    return highlighted;
  }

  /**
   * 获取编程语言的关键字
   */
  private getKeywords(language: string): string[] {
    const keywordMap: Record<string, string[]> = {
      javascript: ['const', 'let', 'var', 'function', 'return', 'if', 'else', 'for', 'while', 'class', 'extends', 'import', 'export', 'default', 'async', 'await', 'try', 'catch', 'throw', 'new', 'this', 'typeof', 'instanceof', 'true', 'false', 'null', 'undefined'],
      typescript: ['const', 'let', 'var', 'function', 'return', 'if', 'else', 'for', 'while', 'class', 'extends', 'import', 'export', 'default', 'async', 'await', 'try', 'catch', 'throw', 'new', 'this', 'typeof', 'instanceof', 'true', 'false', 'null', 'undefined', 'interface', 'type', 'enum', 'implements', 'public', 'private', 'protected', 'readonly'],
      python: ['def', 'class', 'if', 'elif', 'else', 'for', 'while', 'return', 'import', 'from', 'as', 'try', 'except', 'finally', 'with', 'lambda', 'True', 'False', 'None', 'and', 'or', 'not', 'in', 'is', 'pass', 'break', 'continue', 'raise', 'yield', 'async', 'await'],
      html: ['html', 'head', 'body', 'div', 'span', 'p', 'a', 'img', 'script', 'style', 'link', 'meta', 'title', 'header', 'footer', 'nav', 'main', 'section', 'article', 'aside'],
      css: ['color', 'background', 'font', 'margin', 'padding', 'border', 'width', 'height', 'display', 'position', 'top', 'left', 'right', 'bottom', 'flex', 'grid', 'transform', 'transition', 'animation'],
    };

    return keywordMap[language.toLowerCase()] || [];
  }

  /**
   * 处理行内代码
   */
  private parseInlineCode(text: string): string {
    // 匹配 `code`
    return text.replace(/`([^`]+)`/g, '<code class="inline-code">$1</code>');
  }

  /**
   * 处理图片
   * 格式：![alt](url)
   */
  private parseImages(text: string): string {
    // 匹配 ![alt](url) 或 ![alt](url "title")
    return text.replace(
      /!\[([^\]]*)\]\(([^)]+)\)/g,
      (_, alt, src) => {
        const escapedAlt = escapeHtml(alt);
        const escapedSrc = escapeHtml(src);
        return `<img src="${escapedSrc}" alt="${escapedAlt}" class="markdown-image" />`;
      }
    );
  }

  /**
   * 处理链接
   * 格式：[text](url)
   */
  private parseLinks(text: string): string {
    // 匹配 [text](url) 或 [text](url "title")
    return text.replace(
      /\[([^\]]+)\]\(([^)]+)\)/g,
      (_, text, href) => {
        const escapedText = this.parseInline(text); // 链接文本可以包含行内格式
        const escapedHref = escapeHtml(href);

        // 如果有自定义链接处理器，使用它
        if (this.options.linkHandler) {
          return this.options.linkHandler(escapedHref, escapedText);
        }

        // 默认：打开链接时检查是否为外部链接
        const isExternal = href.startsWith('http://') || href.startsWith('https://');
        const target = isExternal ? ' target="_blank" rel="noopener noreferrer"' : '';

        return `<a href="${escapedHref}"${target} class="markdown-link">${escapedText}</a>`;
      }
    );
  }

  /**
   * 处理标题
   */
  private parseHeadings(text: string): string {
    // 按行处理
    return text.replace(
      /^(#{1,6})\s+(.+)$/gm,
      (_, hashes, content) => {
        const level = hashes.length;
        const parsedContent = this.parseInline(content);
        return `<h${level} class="heading-${level}">${parsedContent}</h${level}>`;
      }
    );
  }

  /**
   * 处理引用块
   */
  private parseBlockquotes(text: string): string {
    // 匹配以 > 开头的行
    return text.replace(
      /^&gt;\s*(.+)$/gm,
      '<blockquote class="blockquote">$1</blockquote>'
    );
  }

  /**
   * 处理无序列表
   */
  private parseUnorderedLists(text: string): string {
    // 匹配 - item 或 * item
    return text.replace(
      /^[\-\*]\s+(.+)$/gm,
      '<li class="list-item">$1</li>'
    ).replace(
      /(<li class="list-item">.*<\/li>)(\n<li class="list-item">)*/g,
      (match) => `<ul class="unordered-list">${match}</ul>`
    );
  }

  /**
   * 处理有序列表
   */
  private parseOrderedLists(text: string): string {
    // 匹配 1. item
    return text.replace(
      /^\d+\.\s+(.+)$/gm,
      '<li class="list-item">$1</li>'
    );
  }

  /**
   * 处理粗体和斜体
   */
  private parseEmphasis(text: string): string {
    // 粗体：**text** 或 __text__
    text = text.replace(
      /\*\*([^*]+)\*\*/g,
      '<strong>$1</strong>'
    );
    text = text.replace(
      /__([^_]+)__/g,
      '<strong>$1</strong>'
    );

    // 斜体：*text* 或 _text_
    text = text.replace(
      /\*([^*]+)\*/g,
      '<em>$1</em>'
    );
    text = text.replace(
      /_([^_]+)_/g,
      '<em>$1</em>'
    );

    // 删除线：~~text~~
    text = text.replace(
      /~~([^~]+)~~/g,
      '<del>$1</del>'
    );

    return text;
  }

  /**
   * 处理水平线
   */
  private parseHorizontalRules(text: string): string {
    // 匹配 --- 或 *** 或 ___
    return text.replace(
      /^([\-\*_]){3,}$/gm,
      '<hr class="horizontal-rule" />'
    );
  }

  /**
   * 处理换行
   */
  private parseLineBreaks(text: string): string {
    // 两个空格 + 换行 = <br>
    text = text.replace(/  \n/g, '<br>\n');

    // 段落之间的空行
    text = text.replace(/\n\n+/g, '</p><p class="paragraph">');

    // 包裹在 p 标签中
    text = '<p class="paragraph">' + text + '</p>';

    // 清理空段落
    text = text.replace(/<p class="paragraph"><\/p>/g, '');

    return text;
  }

  /**
   * 解析行内元素（递归调用）
   * 用于链接文本等可以包含行内格式的地方
   */
  private parseInline(text: string): string {
    // 转义 HTML
    text = escapeHtml(text);

    // 粗体
    text = text.replace(/\*\*([^*]+)\*\*/g, '<strong>$1</strong>');

    // 斜体
    text = text.replace(/\*([^*]+)\*/g, '<em>$1</em>');

    // 行内代码
    text = text.replace(/`([^`]+)`/g, '<code>$1</code>');

    return text;
  }
}

/**

 */
interface Tab {

  label: string;
}

const TABS: Tab[] = [
  { id: 'write', label: '编辑' },
  { id: 'preview', label: '预览' },
  { id: 'split', label: '分屏' },
];

export default function MarkdownEditor({
  initialValue = '',
  placeholder = '在这里输入 Markdown...',
  livePreview = true,
  previewDelay = 300,
  maxLength = 100000,
  className = '',
  onChange,
}: MarkdownEditorProps) {
  // 编辑器内容
  const [content, setContent] = useState(initialValue);

  // 预览 HTML
  const [previewHtml, setPreviewHtml] = useState('');

  // 当前 Tab
  const [activeTab, setActiveTab] = useState<'write' | 'preview' | 'split'>('split');

  // 是否全屏
  const [isFullscreen, setIsFullscreen] = useState(false);

  // 编辑器引用
  const textareaRef = useRef<HTMLTextAreaElement>(null);

  // 防抖定时器引用
  const previewTimeoutRef = useRef<NodeJS.Timeout>();

  // 本地存储键名
  const STORAGE_KEY = 'markdown-editor-draft';

  /**
   * 保存草稿到本地存储
   */
  const saveDraft = useCallback((text: string) => {
    try {
      localStorage.setItem(STORAGE_KEY, text);
    } catch (error) {
      console.warn('保存草稿失败:', error);
    }
  }, []);

  /**
   * 从本地存储加载草稿
   */
  const loadDraft = useCallback((): string => {
    try {
      return localStorage.getItem(STORAGE_KEY) || '';
    } catch (error) {
      console.warn('加载草稿失败:', error);
      return '';
    }
  }, []);

  /**
   * 清除草稿
   */
  const clearDraft = useCallback(() => {
    try {
      localStorage.removeItem(STORAGE_KEY);
    } catch (error) {
      console.warn('清除草稿失败:', error);
    }
  }, []);

  /**
   * 更新预览 HTML（带防抖）
   */
  const updatePreview = useCallback((text: string) => {
    // 清除之前的定时器
    if (previewTimeoutRef.current) {
      clearTimeout(previewTimeoutRef.current);
    }

    // 设置新的定时器
    previewTimeoutRef.current = setTimeout(() => {
      const html = parseMarkdown(text);
      setPreviewHtml(html);
    }, previewDelay);
  }, [previewDelay]);

  /**
   * 处理内容变化
   */
  const handleChange = useCallback((e: React.ChangeEvent<HTMLTextAreaElement>) => {
    const newValue = e.target.value;

    // 检查最大长度
    if (newValue.length > maxLength) {
      return;
    }

    setContent(newValue);

    // 保存草稿
    saveDraft(newValue);

    // 更新预览
    if (livePreview) {
      updatePreview(newValue);
    }

    // 触发回调
    onChange?.(newValue);
  }, [maxLength, livePreview, saveDraft, updatePreview, onChange]);

  /**
   * 插入文本到光标位置
   * 用于工具栏快捷操作
   */
  const insertText = useCallback((before: string, after: string = '') => {
    const textarea = textareaRef.current;
    if (!textarea) return;

    // 获取选区
    const start = textarea.selectionStart;
    const end = textarea.selectionEnd;
    const selectedText = content.substring(start, end);

    // 构建新内容
    const newContent =
      content.substring(0, start) +
      before +
      selectedText +
      after +
      content.substring(end);

    // 更新状态
    setContent(newContent);
    saveDraft(newContent);

    // 更新预览
    if (livePreview) {
      updatePreview(newContent);
    }

    // 触发回调
    onChange?.(newContent);

    // 恢复焦点和选区
    setTimeout(() => {
      textarea.focus();
      // 设置选区到插入的内容
      const newStart = start + before.length;
      const newEnd = newStart + selectedText.length;
      textarea.setSelectionRange(newStart, newEnd);
    }, 0);
  }, [content, livePreview, saveDraft, updatePreview, onChange]);

  /**
   * 快捷工具栏配置
   */
  const toolbarActions = [
    // 标题
    { label: 'H1', title: '一级标题', action: () => insertText('# ') },
    { label: 'H2', title: '二级标题', action: () => insertText('## ') },
    { label: 'H3', title: '三级标题', action: () => insertText('### ') },
    // 分隔

// utils/XSSProtection.ts

/**
 * XSS（跨站脚本攻击）防护工具
 *
 * 攻击原理：
 * 攻击者在用户输入中注入恶意 JavaScript 代码
 * 当其他用户查看这些内容时，恶意代码会执行
 *
 * 防护策略：
 * 1. 输入过滤：移除或转义危险字符
 * 2. 输出编码：将特殊字符转义为 HTML 实体
 * 3. Content Security Policy (CSP)：限制脚本执行
 * 4. HttpOnly Cookie：防止 JavaScript 访问敏感 Cookie
 */

/**
 * HTML 实体编码映射
 */
const HTML_ESCAPE_MAP: Record<string, string> = {
  '&': '&amp;',
  '<': '&lt;',
  '>': '&gt;',
  '"': '&quot;',
  "'": '&#x27;',  // 使用 &#x27; 而不是 &#039;，更短
  '/': '&#x2F;',  // 斜线转义防止 </script> 绕过
  '`': '&#x60;',  // 反引号（模板字符串中可能造成问题）
  '=': '&#x3D;',  // 等号（某些场景下可能造成问题）
};

/**
 * 解码 HTML 实体
 */
const HTML_UNESCAPE_MAP: Record<string, string> = Object.fromEntries(
  Object.entries(HTML_ESCAPE_MAP).map(([k, v]) => [v, k])
);

/**
 * 转义 HTML 特殊字符
 * 将 < > & " ' 等字符转换为 HTML 实体
 */
export const escapeHtml = (text: string): string => {
  // 使用正则表达式全局替换所有需要转义的字符
  return text.replace(/[&<>"'`=/]/g, (char) => HTML_ESCAPE_MAP[char] || char);
};

/**
 * 解码 HTML 实体
 * 将 HTML 实体转换回原始字符
 */
export const unescapeHtml = (text: string): string => {
  // 匹配 &#xHHHH; 或 &#dddd; 格式的数字实体
  return text.replace(/&#(x[\da-f]+|\d+);/gi, (_, hex) => {
    // 处理十六进制格式 &#xHHHH;
    if (hex.startsWith('x')) {
      const codePoint = parseInt(hex.slice(1), 16);
      return String.fromCodePoint(codePoint);
    }
    // 处理十进制格式 &#dddd;
    return String.fromCodePoint(parseInt(hex, 10));
  }).replace(/&(amp|lt|gt|quot|#x27|#x2F|#x60|#x3D);/gi, (_, name) => {
    return HTML_UNESCAPE_MAP[`&${name};`] || `&${name};`;
  });
};

/**
 * 危险标签黑名单
 * 这些标签的内容和属性会被完全移除
 */
const DANGEROUS_TAGS = [
  'script',    // JavaScript 代码
  'style',     // CSS（可导致 XSS）
  'iframe',    // 嵌入外部内容
  'object',    // 嵌入对象（如 Flash）
  'embed',     // 嵌入对象
  'form',      // 表单（防止钓鱼）
  'input',     // 输入框（可在某些情况下被利用）
  'button',    // 按钮（可被点击）
  'select',    // 下拉框
  'textarea',  // 文本域
  'link',      // 链接（可导入恶意 CSS/JS）
  'meta',      // 元信息
  'base',      // 基础 URL（可被利用）
  'svg',       // SVG（可包含脚本）
  'math',      // MathML（可包含脚本）
];

/**
 * 危险属性黑名单
 * 包含这些属性的标签会被移除
 */
const DANGEROUS_ATTRIBUTES = [
  // JavaScript 相关
  'onload', 'onerror', 'onclick', 'onmouseover', 'onmouseout',
  'onfocus', 'onblur', 'onchange', 'onsubmit', 'onkeydown',
  'onkeyup', 'onkeypress', 'ondblclick', 'oncontextmenu',
  // URL 相关
  'href', 'src', 'action', 'data', 'poster',
  // 其他
  'style', 'class', 'id',
];

/**
 * XSS 防护配置
 */
interface XSSOptions {
  // 允许的标签白名单（空数组表示全部禁止）
  allowedTags?: string[];
  // 允许的属性白名单
  allowedAttributes?: string[];
  // 是否移除危险标签内容
  stripTagContent?: boolean;
  // 自定义过滤器
  customFilter?: (tag: string, html: string) => string;
}

/**
 * HTML 字符串净化器
 *
 * @param html 原始 HTML 字符串
 * @param options 配置选项
 * @returns 净化后的 HTML 字符串
 */
export const sanitizeHtml = (html: string, options: XSSOptions = {}): string => {
  const {
    allowedTags = [],      // 空数组 = 不允许任何标签
    allowedAttributes = [], // 空数组 = 不允许任何属性
    stripTagContent = true, // 默认移除标签内容
    customFilter,
  } = options;

  // 如果设置了白名单，只允许白名单中的标签
  const useWhitelist = allowedTags.length > 0;

  // 创建 DOM 解析器
  // 使用浏览器原生 API 进行解析
  const parser = new DOMParser();
  const doc = parser.parseFromString(html, 'text/html');

  /**
   * 递归遍历并净化节点
   */
  const sanitizeNode = (node: Node): Node | null => {
    // 文本节点：直接返回
    if (node.nodeType === Node.TEXT_NODE) {
      return node.cloneNode(true);
    }

    // 元素节点
    if (node.nodeType === Node.ELEMENT_NODE) {
      const element = node as Element;
      const tagName = element.tagName.toLowerCase();

      // 执行自定义过滤器
      if (customFilter) {
        const result = customFilter(tagName, element.innerHTML);
        if (result === null) {
          return null; // 移除
        }
        if (typeof result === 'string') {
          // 返回处理后的字符串（创建临时容器解析）
          const temp = document.createElement('div');
          temp.innerHTML = result;
          return temp.firstChild;
        }
      }

      // 黑名单检查
      if (DANGEROUS_TAGS.includes(tagName)) {
        // 如果需要移除内容，移除整个标签
        // 否则只移除标签，保留内容
        if (stripTagContent) {
          return null;
        } else {
          // 保留子节点，移除标签
          const fragment = document.createDocumentFragment();
          Array.from(element.childNodes).forEach(child => {
            const sanitized = sanitizeNode(child);
            if (sanitized) {
              fragment.appendChild(sanitized);
            }
          });
          return fragment;
        }
      }

      // 白名单检查（如果设置了白名单）
      if (useWhitelist && !allowedTags.includes(tagName)) {
        return null;
      }

      // 创建新元素
      const newElement = document.createElement(tagName);

      // 复制属性
      Array.from(element.attributes).forEach(attr => {
        const attrName = attr.name.toLowerCase();

        // 检查危险属性
        if (DANGEROUS_ATTRIBUTES.includes(attrName)) {
          return; // 跳过
        }

        // 检查 href/src 属性（可能包含 javascript: 协议）
        if ((attrName === 'href' || attrName === 'src') && isDangerousUrl(attr.value)) {
          return; // 跳过
        }

        // 检查 style 属性（可能包含 expression() 等危险 CSS 表达式）
        if (attrName === 'style' && isDangerousStyle(attr.value)) {
          return; // 跳过
        }

        // 白名单属性检查
        if (useWhitelist && allowedAttributes.length > 0) {
          if (!allowedAttributes.includes(attrName)) {
            return; // 跳过
          }
        }

        // 复制安全属性
        newElement.setAttribute(attr.name, attr.value);
      });

      // 递归处理子节点
      Array.from(element.childNodes).forEach(child => {
        const sanitized = sanitizeNode(child);
        if (sanitized) {
          newElement.appendChild(sanitized);
        }
      });

      return newElement;
    }

    // 其他节点类型（如注释）：移除
    return null;
  };

  // 处理 body 中的所有子节点
  const body = doc.body;
  const fragment = document.createDocumentFragment();

  Array.from(body.childNodes).forEach(child => {
    const sanitized = sanitizeNode(child);
    if (sanitized) {
      fragment.appendChild(sanitized);
    }
  });

  // 转换为 HTML 字符串
  const temp = document.createElement('div');
  temp.appendChild(fragment);
  return temp.innerHTML;
};

/**
 * 检查 URL 是否危险
 * 检测 javascript: 协议和其他危险模式
 */
const isDangerousUrl = (url: string): boolean => {
  // 标准化 URL
  const normalizedUrl = url.trim().toLowerCase();

  // 检查 javascript: 协议
  if (normalizedUrl.startsWith('javascript:')) {
    return true;
  }

  // 检查 data: 协议（可能用于内嵌脚本）
  if (normalizedUrl.startsWith('data:text/html') || normalizedUrl.startsWith('data:image/svg')) {
    // 注意：data:image/png 等是安全的
    return true;
  }

  // 检查 vbscript: 协议
  if (normalizedUrl.startsWith('vbscript:')) {
    return true;
  }

  // 检查伪协议
  if (normalizedUrl.startsWith('file:') || normalizedUrl.startsWith('blob:')) {
    // 这些协议可能危险，根据场景判断
    return true;
  }

  return false;
};

/**
 * 检查 CSS 样式是否危险
 */
const isDangerousStyle = (style: string): boolean => {
  // 检查 expression()（IE 的动态表达式）
  if (style.includes('expression(')) {
    return true;
  }

  // 检查 url()（可能导入恶意资源）
  // 注意：这里可以进一步验证 URL 的安全性
  if (/\burl\s*\(/i.test(style)) {
    // 检查 url 中的内容
    const urlMatch = style.match(/url\s*\(\s*['"]?([^'")]+)['"]?\s*\)/i);
    if (urlMatch && isDangerousUrl(urlMatch[1])) {
      return true;
    }
  }

  // 检查 @import
  if (/@import/i.test(style)) {
    return true;
  }

  return false;
};

    'a',
    'img',

/**
 * 压缩图片
 */
const compressImage = (
  file: File,
  quality: number = 0.8,
  maxWidth: number = 1920,
  maxHeight: number = 1080
): Promise<File> => {
  return new Promise((resolve, reject) => {
    // 如果不是图片，直接返回原文件
    if (!file.type.startsWith('image/')) {
      resolve(file);
      return;
    }

    // 创建 FileReader
    const reader = new FileReader();

    reader.onload = (e) => {
      const img = new Image();

      img.onload = () => {
        // 计算缩放比例
        let width = img.width;
        let height = img.height;

        // 如果图片超过限制，进行缩放
        if (width > maxWidth) {
          height = (height * maxWidth) / width;
          width = maxWidth;
        }

        if (height > maxHeight) {
          width = (width * maxHeight) / height;
          height = maxHeight;
        }

        // 创建 Canvas
        const canvas = document.createElement('canvas');
        canvas.width = width;
        canvas.height = height;

        const ctx = canvas.getContext('2d');
        if (!ctx) {
          reject(new Error('无法创建 Canvas 上下文'));
          return;
        }

        // 绘制
        ctx.drawImage(img, 0, 0, width, height);

        // 导出为 Blob
        canvas.toBlob(
          (blob) => {
            if (blob) {
              // 创建新的 File 对象
              const compressedFile = new File(
                [blob],
                file.name,
                { type: file.type }
              );
              resolve(compressedFile);
            } else {
              reject(new Error('图片压缩失败'));
            }
          },
          file.type,
          quality
        );
      };

      img.onerror = () => reject(new Error('图片加载失败'));
      img.src = e.target?.result as string;
    };

    reader.onerror = () => reject(new Error('文件读取失败'));
    reader.readAsDataURL(file);
  });
};

const uploadFile = (
  file: File,
  uploadUrl: string,
  onProgress: (progress: number) => void
): Promise<string> => {
  return new Promise((resolve, reject) => {
    // 使用 XMLHttpRequest 以支持进度监控
    const xhr = new XMLHttpRequest();

    // 监听进度
    xhr.upload.onprogress = (e) => {
      if (e.lengthComputable) {
        const progress = Math.round((e.loaded / e.total) * 100);
        onProgress(progress);
      }
    };

    // 监听完成
    xhr.onload = () => {
      if (xhr.status >= 200 && xhr.status < 300) {
        try {
          const response = JSON.parse(xhr.responseText);
          resolve(response.url || response.data?.url || '');
        } catch {
          resolve(xhr.responseText);
        }
      } else {
        reject(new Error(`上传失败: ${xhr.status}`));
      }
    };

    // 监听错误
    xhr.onerror = () => reject(new Error('网络错误'));

    // 创建 FormData
    const formData = new FormData();
    formData.append('file', file);

    // 发送请求
    xhr.open('POST', uploadUrl);
    xhr.send(formData);
  });
};

export default function CompleteImageUploader({
  uploadUrl = '/api/upload',
  maxFiles = 9,
  maxSize = 10 * 1024 * 1024,
  multiple = true,
  enableCompress = true,
  compressQuality = 0.8,
  compressMaxWidth = 1920,
  compressMaxHeight = 1080,
  onUpload,
  onError,
}: CompleteImageUploaderProps) {
  // 图片列表
  const [images, setImages] = useState<UploadedImage[]>([]);

  // 拖拽状态
  const [isDragging, setIsDragging] = useState(false);

  // 错误信息
  const [error, setError] = useState<string>('');

  // 文件输入引用
  const fileInputRef = useRef<HTMLInputElement>(null);

  // 编辑器引用（用于粘贴）
  const containerRef = useRef<HTMLDivElement>(null);

  /**
   * 验证文件
   */
  const validateFile = useCallback((file: File): { valid: boolean; error?: string } => {
    // 检查是否为图片
    if (!file.type.startsWith('image/')) {
      return { valid: false, error: `${file.name} 不是图片文件` };
    }

    // 检查文件大小
    if (file.size > maxSize) {
      return { valid: false, error: `${file.name} 超过大小限制 (${formatSize(maxSize)})` };
    }

    return { valid: true };
  }, [maxSize]);

  /**
   * 添加文件到列表
   */
  const addFiles = useCallback(async (files: FileList | File[]) => {
    setError('');

    // 检查数量限制
    if (images.length + files.length > maxFiles) {
      setError(`最多只能上传 ${maxFiles} 张图片`);
      return;
    }

    // 处理每个文件
    const newImages: UploadedImage[] = [];

    for (const file of Array.from(files)) {
      // 验证
      const validation = validateFile(file);
      if (!validation.valid) {
        setError(prev => prev ? `${prev}\n${validation.error}` : validation.error || '');
        continue;
      }

      // 压缩（如启用）
      let processedFile = file;
      if (enableCompress && file.type.startsWith('image/')) {
        try {
          processedFile = await compressImage(
            file,
            compressQuality,
            compressMaxWidth,
            compressMaxHeight
          );
        } catch (err) {
          console.warn('图片压缩失败，使用原图:', err);
        }
      }

      // 创建预览
      const previewUrl = URL.createObjectURL(processedFile);

      // 创建图片项
      const imageItem: UploadedImage = {
        id: generateId(),
        file: processedFile,
        previewUrl,
        progress: 0,
        status: 'pending',
      };

      newImages.push(imageItem);
    }

    // 添加到列表

  }, [
    images.length,
    maxFiles,
    validateFile,
    enableCompress,
    compressQuality,
    compressMaxWidth,
    compressMaxHeight
  ]);

   * 删除图片
   */
  const removeImage = useCallback((id: string) => {
    setImages(prev => {
      const image = prev.find(img => img.id === id);
      if (image) {
        // 释放预览 URL
        URL.revokeObjectURL(image.previewUrl);
      }
      return prev.filter(img => img.id !== id);
    });
  }, []);

   * 重新上传失败的图片
   */
  const retryUpload = useCallback((id: string) => {
    setImages(prev => prev.map(img =>
      img.id === id
        ? { ...img, status: 'pending', progress: 0, error: undefined }
        : img
    ));
  }, []);

  /**
   * 开始上传
   */
  const startUpload = useCallback(async () => {
    const pendingImages = images.filter(img => img.status === 'pending');

    for (const image of pendingImages) {
      // 更新状态为上传中
      setImages(prev => prev.map(img =>
        img.id === image.id
          ? { ...img, status: 'uploading' }
          : img
      ));

      try {
        // 上传到服务器
        const url = await uploadFile(
          image.file,
          uploadUrl,
          (progress) => {
            // 更新进度
            setImages(prev => prev.map(img =>
              img.id === image.id
                ? { ...img, progress }
                : img
            ));
          }
        );

        // 上传成功
        setImages(prev => prev.map(img =>
          img.id === image.id
            ? { ...img, status: 'completed', url, progress: 100 }
            : img
        ));
      } catch (err) {
        // 上传失败
        setImages(prev => prev.map(img =>
          img.id === image.id
            ? { ...img, status: 'error', error: (err as Error).message }
            : img
        ));

        setError(prev => prev ? `${prev}\n${image.file.name} 上传失败` : `${image.file.name} 上传失败`);
        onError?.((err as Error).message);
      }
    }
  }, [images, uploadUrl, onError]);

  const handleFileSelect = (e: React.ChangeEvent<HTMLInputElement>) => {

  };

  const handleDragLeave = (e: React.DragEvent) => {

    const rect = containerRef.current?.getBoundingClientRect();
    if (rect) {

        }
      }

const escapeHtml = (text: string): string => {
  const map: Record<string, string> = {
    '&': '&amp;',
    '<': '&lt;',
    '>': '&gt;',
    '"': '&quot;',
    "'": '&#039;',

/**
 * 解析 Markdown 为 HTML
 */
const parseMarkdown = (markdown: string): string => {
  let html = markdown;

  // 处理代码块
  html = html.replace(/```(\w*)\n([\s\S]*?)```/g, (_, lang, code) => {
    const escaped = escapeHtml(code.trim());
    return `<pre class="code-block" data-lang="${lang}"><code class="language-${lang}">${escaped}</code></pre>`;
  });

  // 处理行内代码
  html = html.replace(/`([^`]+)`/g, '<code class="inline-code">$1</code>');

  // 处理图片
  html = html.replace(/!\[([^\]]*)\]\(([^)]+)\)/g,
    '<img src="$2" alt="$1" class="md-image" />');

  // 处理链接
  html = html.replace(/\[([^\]]+)\]\(([^)]+)\)/g,
    '<a href="$2" class="md-link" target="_blank" rel="noopener">$1</a>');

  // 处理标题
  html = html.replace(/^### (.+)$/gm, '<h3 class="md-h3">$1</h3>');
  html = html.replace(/^## (.+)$/gm, '<h2 class="md-h2">$1</h2>');
  html = html.replace(/^# (.+)$/gm, '<h1 class="md-h1">$1</h1>');

  // 处理引用
  html = html.replace(/^> (.+)$/gm, '<blockquote class="md-blockquote">$1</blockquote>');

  // 处理无序列表
  html = html.replace(/^[\-\*] (.+)$/gm, '<li class="md-li">$1</li>');

  // 处理有序列表
  html = html.replace(/^\d+\. (.+)$/gm, '<li class="md-li md-ol">$1</li>');

  // 处理粗体
  html = html.replace(/\*\*([^*]+)\*\*/g, '<strong>$1</strong>');
  html = html.replace(/__([^_]+)__/g, '<strong>$1</strong>');

  // 处理斜体
  html = html.replace(/\*([^*]+)\*/g, '<em>$1</em>');
  html = html.replace(/_([^_]+)_/g, '<em>$1</em>');

  // 处理删除线
  html = html.replace(/~~([^~]+)~~/g, '<del>$1</del>');

  // 处理水平线
  html = html.replace(/^[\-\*_]{3,}$/gm, '<hr class="md-hr" />');

  // 处理换行（两个空格+换行）
  html = html.replace(/  \n/g, '<br>\n');

  // 处理段落（空行分隔）
  html = html.split(/\n\n+/).map(p => {
    // 跳过已经是块级元素的段落
    if (p.match(/^<(h[1-6]|pre|blockquote|ul|ol|li|hr)/)) {
      return p;
    }
    // 跳过空段落
    if (!p.trim()) {
      return '';
    }
    return `<p class="md-p">${p}</p>`;
  }).join('\n');

  // 处理列表包装
  html = html.replace(/(<li class="md-li"[^>]*>[\s\S]*?<\/li>)+/g, (match) => {
    const isOrdered = match.includes('md-ol');
    const tag = isOrdered ? 'ol' : 'ul';
    return `<${tag} class="md-list">${match}</${tag}>`;
  });

  return html;
};

    }, 0);
  }, [content]);

  /**
   * 在行首插入文本
   */
  const insertAtLineStart = useCallback((prefix: string) => {
    const textarea = textareaRef.current;
    if (!textarea) return;

    const start = textarea.selectionStart;
    // 找到当前行的起始位置
    const lineStart = content.lastIndexOf('\n', start - 1) + 1;

    const newContent =
      content.substring(0, lineStart) +
      prefix +
      content.substring(lineStart);

    setContent(newContent);
    saveToStorage(newContent);

    // 恢复焦点和选区
    setTimeout(() => {
      textarea.focus();
      textarea.setSelectionRange(
        start + prefix.length,
        start + prefix.length
      );
    }, 0);
  }, [content]);

  /**
   * 保存到本地存储
   */
  const saveToStorage = useCallback((text: string) => {
    if (autoSave) {
      try {
        localStorage.setItem(storageKey, text);
        setLastSaved(new Date());
      } catch (e) {
        console.warn('自动保存失败:', e);
      }
    }
  }, [autoSave, storageKey]);

  /**
   * 从本地存储加载
   */
  const loadFromStorage = useCallback((): string => {
    if (autoSave) {
      try {
        return localStorage.getItem(storageKey) || '';
      } catch (e) {
        console.warn('加载草稿失败:', e);
        return '';
      }
    }
    return '';
  }, [autoSave, storageKey]);

  /**
   * 处理内容变化
   */
  const handleChange = useCallback((e: React.ChangeEvent<HTMLTextAreaElement>) => {
    const newValue = e.target.value;

    if (newValue.length > maxLength) {
      return;
    }

    setContent(newValue);

    // 防抖更新预览
    if (autoSaveTimerRef.current) {
      clearTimeout(autoSaveTimerRef.current);
    }

    autoSaveTimerRef.current = setTimeout(() => {
      setPreviewHtml(parseMarkdown(newValue));
      saveToStorage(newValue);
    }, 200);

    onChange?.(newValue);
    getPlainText?.();
  }, [maxLength, onChange, getPlainText, saveToStorage]);

  /**
   * 处理键盘事件
   */
  const handleKeyDown = useCallback((e: React.KeyboardEvent<HTMLTextAreaElement>) => {
    // Tab：插入空格
    if (e.key === 'Tab') {
      e.preventDefault();
      insertAtLineStart('  ');
    }

    // Ctrl+B：粗体
    if ((e.ctrlKey || e.metaKey) && e.key === 'b') {
      e.preventDefault();
      wrapSelection('**', '**');
    }

    // Ctrl+I：斜体
    if ((e.ctrlKey || e.metaKey) && e.key === 'i') {
      e.preventDefault();
      wrapSelection('*', '*');
    }

    // Ctrl+K：链接
    if ((e.ctrlKey || e.metaKey) && e.key === 'k') {
      e.preventDefault();
      wrapSelection('[', '](url)');
    }
  }, [wrapSelection, insertAtLineStart]);

  /**
   * 导出为 HTML
   */
  const exportAsHtml = useCallback(() => {
    const html = `<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Markdown Export</title>
  <style>
    body { font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif; max-width: 800px; margin: 40px auto; padding: 20px; line-height: 1.6; }
    .md-h1 { font-size: 2em; margin: 1em 0; border-bottom: 2px solid #eee; padding-bottom: 0.3em; }
    .md-h2 { font-size: 1.5em; margin: 0.8em 0; }
    .md-h3 { font-size: 1.25em; margin: 0.6em 0; }
    .md-blockquote { border-left: 4px solid #ddd; padding-left: 1em; margin: 1em 0; color: #666; }
    .md-code { background: #f4f4f4; padding: 0.2em 0.4em; border-radius: 3px; font-family: monospace; }
    .md-pre { background: #f4f4f4; padding: 1em; border-radius: 6px; overflow-x: auto; }
    .md-link { color: #0366d6; }
    .md-image { max-width: 100%; }
  </style>
</head>
<body>
${previewHtml}
</body>
</html>`;

    // 下载文件
    const blob = new Blob([html], { type: 'text/html' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'export.html';
    a.click();
    URL.revokeObjectURL(url);
  }, [previewHtml]);

  /**
   * 导出为 Markdown
   */
  const exportAsMarkdown = useCallback(() => {
    const blob = new Blob([content], { type: 'text/markdown' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'document.md';
    a.click();
    URL.revokeObjectURL(url);
  }, [content]);

  /**
   * 复制为纯文本
   */
  const copyAsPlainText = useCallback(() => {
    // 简单处理：移除 markdown 语法
    const plain = content
      .replace(/^#{1,6} /gm, '') // 移除标题标记
      .replace(/\*\*([^*]+)\*\*/g, '$1') // 移除粗体
      .replace(/\*([^*]+)\*/g, '$1') // 移除斜体
      .replace(/~~([^~]+)~~/g, '$1') // 移除删除线
      .replace(/`([^`]+)`/g, '$1') // 移除行内代码
      .replace(/\[([^\]]+)\]\([^)]+\)/g, '$1') // 链接转为纯文本
      .replace(/!\[([^\]]*)\]\([^)]+\)/g, ''); // 图片移除

    navigator.clipboard.writeText(plain);
  }, [content]);

  // 初始化：加载草稿和生成预览
  useEffect(() => {
    const draft = loadFromStorage();
    if (draft) {
      setContent(draft);
      setPreviewHtml(parseMarkdown(draft));
    } else if (initialValue) {
      setContent(initialValue);
      setPreviewHtml(parseMarkdown(initialValue));
    }
  }, []);

  // 清理定时器
  useEffect(() => {
    return () => {
      if (autoSaveTimerRef.current) {
        clearTimeout(autoSaveTimerRef.current);
      }
    };
  }, []);

  // 全屏样式
  const fullscreenStyle: React.CSSProperties = isFullscreen ? {
    position: 'fixed',
    top: 0,
    left: 0,
    right: 0,
    bottom: 0,
    zIndex: 9999,
    borderRadius: 0,
  } : {};

  return (
    <div
      className={`markdown-editor border border-gray-200 rounded-lg overflow-hidden bg-white flex flex-col ${className}`}
      style={fullscreenStyle}
    >
      {/* 工具栏 */}
      <div className="flex items-center gap-1 p-2 border-b border-gray-200 bg-gray-50 flex-wrap shrink-0">
        {toolbarButtons.map((btn, index) => {
          if (btn.type === 'divider') {
            return <div key={index} className="w-px h-6 bg-gray-300 mx-1" />;
          }

          return (
            <button
              key={index}
              type="button"
              title={btn.title}
              onClick={btn.action}
              className="px-2 py-1 text-sm text-gray-600 hover:bg-gray-200 rounded transition-colors"
            >
              {btn.label}
            </button>
          );
        })}

        {/* 视图切换 */}
        <div className="ml-auto flex items-center gap-1">
          <button
            type="button"
            title="分屏"
            onClick={() => setViewMode('split')}
            className={`px-3 py-1 text-sm rounded ${viewMode === 'split' ? 'bg-blue-100 text-blue-600' : 'text-gray-500 hover:bg-gray-200'}`}
          >
            分屏
          </button>
          <button
            type="button"
            title="编辑"
            onClick={() => setViewMode('write')}
            className={`px-3 py-1 text-sm rounded ${viewMode === 'write' ? 'bg-blue-100 text-blue-600' : 'text-gray-500 hover:bg-gray-200'}`}
          >
            编辑
          </button>
          <button
            type="button"
            title="预览"
            onClick={() => setViewMode('preview')}
            className={`px-3 py-1 text-sm rounded ${viewMode === 'preview' ? 'bg-blue-100 text-blue-600' : 'text-gray-500 hover:bg-gray-200'}`}
          >
            预览
          </button>
        </div>

        {/* 导出按钮 */}
        <div className="flex items-center gap-1 ml-2">
          <button
            type="button"
            title="导出 HTML"
            onClick={exportAsHtml}
            className="px-2 py-1 text-sm text-gray-500 hover:bg-gray-200 rounded"
          >
            HTML
          </button>
          <button
            type="button"
            title="导出 Markdown"
            onClick={exportAsMarkdown}
            className="px-2 py-1 text-sm text-gray-500 hover:bg-gray-200 rounded"
          >
            MD
          </button>
          <button
            type="button"
            title="复制为纯文本"
            onClick={copyAsPlainText}
            className="px-2 py-1 text-sm text-gray-500 hover:bg-gray-200 rounded"
          >
            复制
          </button>
        </div>
      </div>

      {/* 编辑器主体 */}
      <div className={`flex flex-1 overflow-hidden ${viewMode === 'split' ? 'flex-row' : 'flex-col'}`}>
        {/* 编辑区 */}
        {(viewMode === 'write' || viewMode === 'split') && (
          <div className={`flex flex-col ${viewMode === 'split' ? 'w-1/2 border-r border-gray-200' : 'flex-1'}`}>
            <textarea
              ref={textareaRef}
              value={content}
              onChange={handleChange}
              onKeyDown={handleKeyDown}
              placeholder={placeholder}
              className="
                flex-1 w-full p-4 resize-none
                font-mono text-sm leading-relaxed
                border-none outline-none
                placeholder:text-gray-400
              "
            />
          </div>
        )}

        {/* 预览区 */}
        {(viewMode === 'preview' || viewMode === 'split') && (
          <div
            className={`
              overflow-auto p-4 bg-gray-50
              ${viewMode === 'split' ? 'w-1/2' : 'flex-1'}
            `}
          >
            <div
              className="prose max-w-none"
              dangerouslySetInnerHTML={{ __html: previewHtml || '<p class="text-gray-400">预览区域</p>' }}
            />
          </div>
        )}
      </div>

      {/* 底部状态栏 */}
      <div className="flex items-center justify-between px-4 py-2 border-t border-gray-200 bg-gray-50 text-xs text-gray-500 shrink-0">
        <div className="flex items-center gap-4">
          <span>{content.length} / {maxLength} 字符</span>
          <span>{content.split(/\s+/).filter(Boolean).length} 单词</span>
          <span>{content.split('\n').filter(Boolean).length} 行</span>
        </div>
        {lastSaved && (
          <span>已自动保存 {lastSaved.toLocaleTimeString()}</span>
        )}
      </div>
    </div>
  );
}
```

## 知识点总结

### 文件上传核心 API

| API | 说明 |
|-----|------|
| `File` | 继承自 Blob，表示文件对象 |
| `FileList` | 类数组对象，包含用户选择的文件 |
| `FileReader` | 异步读取文件内容（文本/二进制） |
| `Blob` | 二进制大数据对象，支持 slice 分割 |
| `URL.createObjectURL()` | 创建 Blob URL 用于预览 |
| `FormData` | 包装表单数据，支持文件上传 |
| `XMLHttpRequest.upload` | 监听上传进度 |

### 富文本编辑器核心 API

| API | 说明 |
|-----|------|
| `contentEditable` | 让元素可编辑 |
| `document.execCommand()` | 执行格式化命令（已废弃但仍可用） |
| `window.getSelection()` | 获取用户选区 |
| `Range` | 表示选中的文本范围 |
| `document.queryCommandState()` | 查询格式化状态 |

### 关键概念

1. **Blob URL vs DataURL**
   - Blob URL：`blob:null/uuid`，高效，需要手动释放
   - DataURL：`data:image/png;base64,...`，可直接使用，无需释放

2. **分片上传原理**
   - 使用 `Blob.slice()` 分割文件
   - 并发上传多个分片
   - 记录已上传分片实现断点续传
   - 所有分片上传完成后合并

3. **XSS 防护**
   - 输入过滤：转义特殊字符
   - 输出编码：将 `< > & " '` 转义为 HTML 实体
   - 标签白名单：只允许安全标签
   - 属性黑名单：移除危险属性如 `onclick`

4. **富文本架构**
   - contentEditable：原生编辑能力
   - 数据驱动（Slate/TipTap）：状态即视图
   - 自定义渲染：完全控制输出

## 常见问题与解决方案

### 1. 大文件上传失败
**问题**：超过服务器限制或浏览器内存限制
**解决**：
- 分片上传
- 压缩图片
- 检查服务器 `client_max_body_size`

### 2. 粘贴图片上传
**问题**：粘贴截图无响应
**解决**：
- 监听 `paste` 事件
- 从 `clipboardData.items` 获取图片文件
- 使用 `item.getAsFile()` 获取 File 对象

### 3. contentEditable 行为不一致
**问题**：不同浏览器表现不同
**解决**：
- 使用成熟的库（Slate、TipTap）
- 或使用纯文本 `textarea` + 自定义渲染

### 4. XSS 攻击
**问题**：用户输入包含恶意脚本
**解决**：
- 始终转义 HTML 特殊字符
- 使用标签白名单
- 避免 `dangerouslySetInnerHTML`
