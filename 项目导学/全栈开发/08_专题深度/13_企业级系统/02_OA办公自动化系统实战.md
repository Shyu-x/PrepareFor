# OA办公自动化系统实战

## 一、系统概述与核心概念

### 1.1 什么是OA系统？

想象一下传统的企业办公场景：

- **请假**：员工填一张纸质假条 → 找主管签字 → 找HR签字 → 找总经理签字 → 交到办公室存档
- **报销**：员工收集发票 → 填报销单 → 找主管审批 → 找财务审核 → 找领导签字 → 出纳付款 → 交到财务存档
- **开会**：行政挨个打电话通知 → 会议室预约 → 打印资料 → 会后整理会议纪要 → 邮件发送

每个流程都要跑好几天，纸张堆积如山，找个文件翻箱倒柜，领导出差就要干等。

**OA（Office Automation，办公自动化）系统** 就是把这些线下流程搬到线上，用电子化的方式处理审批、协作、沟通等日常办公事务。员工在电脑上提交申请，系统自动流转到相关审批人，审批人在手机上就能批准，整个过程可能只需要几分钟。

### 1.2 OA系统的核心价值

| 核心功能 | 解决的问题 | 业务价值 |
|----------|------------|----------|
| **审批流程** | 纸质审批效率低、流程不透明 | 审批效率提升80% |
| **工作流引擎** | 流程定义不灵活、难以调整 | 快速响应业务变化 |
| **表单设计** | 纸质表格难以统一、数据难汇总 | 数据结构化、便于分析 |
| **考勤管理** | 打卡记录混乱、统计工作量大 | 自动统计、实时查看 |
| **会议管理** | 会议室冲突、通知不到位 | 资源高效利用 |

### 1.3 OA系统与其他系统的区别

```
┌─────────────────────────────────────────────────────────────────┐
│                         企业信息系统全家桶                         │
├─────────────┬─────────────┬─────────────┬─────────────┬──────────┤
│     CRM     │     OA      │     ERP     │     HRM    │    SCM   │
│  客户关系   │   办公自动化  │   资源计划   │   人力资源  │   供应链  │
├─────────────┼─────────────┼─────────────┼─────────────┼──────────┤
│ 销售过程   │ 行政审批    │  生产制造   │  招聘入职   │  采购    │
│ 客户跟进   │ 考勤打卡    │  库存管理   │  薪酬核算   │  销售    │
│ 商机管理   │ 日程会议    │  财务成本   │  绩效考核   │  物流    │
└─────────────┴─────────────┴─────────────┴─────────────┴──────────┘
```

OA系统和其他系统的本质区别在于：**OA是以流程为核心，其他系统是以数据为核心**。CRM关注客户数据，ERP关注物料和财务数据，OA关注的是"一件事从开始到完成的整个过程"。

## 三、工作流引擎设计

### 3.1 什么是工作流引擎？

工作流引擎是OA系统的"大脑"，负责定义、执行和管理业务流程。可以把它想象成一个交通警察：

- **定义流程**：规定好这条路怎么走（流程定义）
- **指挥交通**：决定每辆车当前在哪（流程实例）
- **处理违章**：处理审批不通过的情况（流程分支）
- **记录轨迹**：记录每辆车的行驶路线（审批历史）

```typescript
// ============================================
// 工作流引擎 - 核心类型定义
// ============================================

/**
 * 流程节点类型
 *
 * 不同的节点类型有不同的行为：
 * - startEvent：流程开始，只有一个
 * - endEvent：流程结束，可以有多个
 * - userTask：需要用户办理的任务节点
 * - scriptTask：自动执行的脚本节点
 * - serviceTask：调用外部服务的节点
 * - gateway：网关节点，用于分支判断
 */
type NodeType =
  | 'startEvent'      // 开始节点
  | 'endEvent'        // 结束节点
  | 'userTask'        // 用户任务节点
  | 'scriptTask'      // 脚本任务节点
  | 'serviceTask'     // 服务任务节点
  | 'exclusiveGateway' // 单一路由网关（只能选一个分支）
  | 'parallelGateway'  // 并行网关（同时走所有分支）
  | 'inclusiveGateway'  // 包容网关（满足条件的分支都走）;

/**
 * 流程节点定义
 */
interface FlowNode {
  id: string;                     // 节点ID，唯一标识
  name: string;                   // 节点名称
  type: NodeType;                 // 节点类型

  // userTask特有属性
  assignee?: string;             // 指定办理人，支持变量，如 ${manager}
  candidateUsers?: string[];     // 候选办理人列表
  candidateGroups?: string[];    // 候选办理组列表
  assigneeType?: 'user' | 'role' | 'department' | 'formula';  // 办理人类型

  // 网关特有属性
  conditionExpression?: string;   // 条件表达式
  defaultFlow?: boolean;          // 是否为默认分支（网关用）

  // 其他属性
  timeout?: number;              // 超时时间（小时）
  reminderInterval?: number;     // 催办间隔（小时）
  formKey?: string;              // 关联的表单
  properties?: Record<string, any>;  // 其他自定义属性
}

/**
 * 流程连线定义
 *
 * 规定流程从哪里来、到哪里去
 * 可以带条件，满足条件才走这条线
 */
interface FlowLine {
  id: string;                     // 连线ID
  sourceNodeId: string;           // 源节点ID
  targetNodeId: string;          // 目标节点ID
  conditionExpression?: string; // 条件表达式
  isDefault?: boolean;           // 是否为默认连线
}

/**
 * 流程定义
 */
interface WorkflowDefinition {
  id: number;
  code: string;                   // 流程编码
  name: string;                   // 流程名称
  version: number;                // 版本号

  nodes: FlowNode[];              // 所有节点
  lines: FlowLine[];              // 所有连线

  // 起始和结束节点
  startNodeId: string;           // 开始节点ID
  endNodeIds: string[];           // 结束节点ID列表

  // 配置
  config: {
    timeout?: number;             // 默认超时时间
    remindInterval?: number;      // 默认催办间隔
    allowWithdrawal?: boolean;    // 允许撤回
    allowRecall?: boolean;        // 允许撤回
    callbackUrl?: string;         // 回调URL
  };
}

/**
 * 流程实例
 */
interface WorkflowInstance {
  id: number;
  code: string;                   // 实例编号

  definitionId: number;          // 关联的流程定义ID
  definitionCode: string;        // 流程定义编码

  // 申请人
  applicantId: number;
  applicantName: string;

  // 申请数据
  formData: Record<string, any>; // 表单数据

  // 当前状态
  currentNodeId: string;         // 当前所在节点ID
  status: 'running' | 'completed' | 'terminated' | 'cancelled';

  // 历史记录
  history: FlowNodeHistory[];    // 节点执行历史

  // 时间
  startedAt: Date;
  completedAt?: Date;
}

/**
 * 节点执行历史
 */
interface FlowNodeHistory {
  nodeId: string;                // 节点ID
  nodeName: string;              // 节点名称
  nodeType: string;              // 节点类型

  assigneeId?: number;           // 办理人ID
  assigneeName?: string;         // 办理人姓名

  action?: string;               // 执行的操作：submit/approve/reject/withdraw
  comment?: string;              // 审批意见

  startTime: Date;              // 开始时间
  endTime?: Date;                // 结束时间
  duration?: number;             // 耗时（秒）

  variables?: Record<string, any>;  // 节点变量
}
```

### 3.2 工作流引擎核心实现

```typescript
// ============================================
// 工作流引擎 - 核心引擎类
// ============================================

// src/workflow/engine/workflow-engine.ts
import { Injectable } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';

/**
 * 表达式求值器
 *
 * 用于计算流程连线上的条件表达式
 *
 * 支持的表达式格式：
 * - amount > 5000
 * - leaveType == 'annual' && duration > 3
 * - dept.name == '财务部'
 */
class ExpressionEvaluator {
  /**
   * 执行表达式求值
   *
   * @param expression 条件表达式，如 "amount > 5000"
   * @param context 上下文变量，如 { amount: 10000, formData: {...} }
   */
  evaluate(expression: string, context: Record<string, any>): boolean {
    // 简单的表达式解析
    // 实际生产中建议使用专门的表达式引擎库，如 easy-expression

    try {
      // 替换变量引用
      // 把 ${formData.amount} 替换成 context.formData.amount
      let evalExpression = expression.replace(
        /\$\{([^}]+)\}/g,
        (_, path) => this.getNestedValue(context, path)
      );

      // 处理 == 替换成 ===
      evalExpression = evalExpression.replace(/==/g, '===');

      // 使用 Function 构造器执行表达式
      // 注意：这是简化实现，实际生产中需要严格过滤
      const fn = new Function('context', `with(context) { return ${evalExpression}; }`);
      return fn(context);

    } catch (error) {
      console.error('表达式求值失败:', error);
      return false;
    }
  }

  /**
   * 获取嵌套对象的值
   * 支持 a.b.c 的路径格式
   */
  private getNestedValue(obj: any, path: string): any {
    return path.split('.').reduce((current, key) => {
      return current?.[key];
    }, obj);
  }
}

/**
 * 流程引擎服务
 *
 * 这是工作流引擎的核心类
 * 负责：启动流程、执行节点、推进流程、处理审批
 */
@Injectable()
export class WorkflowEngine {
  // 表达式求值器
  private evaluator = new ExpressionEvaluator();

  constructor(
    @InjectRepository(WorkflowInstance)
    private instanceRepository: Repository<WorkflowInstance>,

    @InjectRepository(WorkflowTask)
    private taskRepository: Repository<WorkflowTask>,

    @InjectRepository(WorkflowDefinition)
    private definitionRepository: Repository<WorkflowDefinition>,
  ) {}

  // ============================================
  // 启动流程实例
  // ============================================

  /**
   * 启动一个新的流程实例
   *
   * 当员工提交一个申请时，会调用这个方法
   *
   * @param definitionCode 流程定义编码
   * @param applicantId 申请人ID
   * @param formData 表单数据
   */
  async startProcess(
    definitionCode: string,
    applicantId: number,
    formData: Record<string, any>
  ): Promise<WorkflowInstance> {
    // 1. 获取流程定义
    const definition = await this.definitionRepository.findOne({
      where: { definitionCode, isPublished: true },
      order: { version: 'DESC' }
    });

    if (!definition) {
      throw new Error(`流程定义不存在: ${definitionCode}`);
    }

    // 2. 解析流程定义
    const parsedDefinition = this.parseDefinition(definition);

    // 3. 创建流程实例
    const instance = await this.createInstance(
      definition,
      applicantId,
      formData
    );

    // 4. 获取开始节点并推进到下一个节点
    const startNode = parsedDefinition.nodes.find(
      n => n.type === 'startEvent'
    );

    // 5. 执行开始节点后的第一个节点（通常是userTask）
    const nextNodes = this.getNextNodes(
      parsedDefinition,
      startNode.id,
      { applicantId, formData }
    );

    // 6. 为下一个节点创建任务
    for (const node of nextNodes) {
      await this.createTask(instance, node, { applicantId, formData });
    }

    // 7. 更新实例当前节点
    instance.currentNodeId = nextNodes[0]?.id || startNode.id;
    await this.instanceRepository.save(instance);

    return instance;
  }

  // ============================================
  // 执行用户任务（审批）
  // ============================================

  /**
   * 执行任务审批
   *
   * @param taskId 任务ID
   * @param userId 审批人ID
   * @param action 操作：approve/reject
   * @param comment 审批意见
   */
  async completeTask(
    taskId: number,
    userId: number,
    action: 'approve' | 'reject',
    comment?: string
  ): Promise<WorkflowInstance> {
    // 1. 获取任务
    const task = await this.taskRepository.findOne({
      where: { id: taskId },
      relations: ['instance']
    });

    if (!task) {
      throw new Error(`任务不存在: ${taskId}`);
    }

    if (task.status !== 'pending') {
      throw new Error(`任务状态不是pending: ${taskId}`);
    }

    if (task.assigneeId !== userId) {
      throw new Error(`不是当前任务的办理人`);
    }

    // 2. 完成任务
    task.status = 'completed';
    task.action = action;
    task.comment = comment;
    task.completedAt = new Date();
    await this.taskRepository.save(task);

    // 3. 记录历史
    await this.recordHistory(
      task.instance,
      task.nodeId,
      userId,
      action,
      comment
    );

    // 4. 根据操作类型处理流程
    const instance = task.instance;

    if (action === 'reject') {
      // 驳回：流程终止
      await this.terminateInstance(instance, 'rejected');
    } else {
      // 批准：推进到下一个节点
      await this.moveToNextNodes(instance);
    }

    return instance;
  }

  // ============================================
  // 流程推进核心逻辑
  // ============================================

  /**
   * 推进流程到下一个节点
   *
   * 这是工作流引擎最核心的逻辑
   * 根据当前节点类型和连线条件，决定下一步去哪里
   */
  private async moveToNextNodes(instance: WorkflowInstance): Promise<void> {
    // 1. 获取流程定义
    const definition = await this.definitionRepository.findOne({
      where: { id: instance.definitionId }
    });

    const parsedDefinition = this.parseDefinition(definition);

    // 2. 获取当前节点
    const currentNode = parsedDefinition.nodes.find(
      n => n.id === instance.currentNodeId
    );

    // 3. 获取当前节点的后续连线
    const outgoingLines = parsedDefinition.lines.filter(
      l => l.sourceNodeId === currentNode.id
    );

    if (outgoingLines.length === 0) {
      // 没有后续连线，说明是结束节点
      await this.completeInstance(instance);
      return;
    }

    // 4. 根据连线条件过滤后续节点
    const context = {
      applicantId: instance.applicantId,
      formData: instance.formData,
      variables: instance.variables || {}
    };

    const nextNodes: FlowNode[] = [];

    for (const line of outgoingLines) {
      // 如果没有条件，或者条件满足
      if (!line.conditionExpression ||
          this.evaluator.evaluate(line.conditionExpression, context)) {
        const targetNode = parsedDefinition.nodes.find(
          n => n.id === line.targetNodeId
        );
        if (targetNode) {
          nextNodes.push(targetNode);
        }
      }
    }

    // 5. 根据节点类型处理
    for (const nextNode of nextNodes) {
      await this.executeNode(instance, nextNode, context);
    }
  }

  /**
   * 执行单个节点
   */
  private async executeNode(
    instance: WorkflowInstance,
    node: FlowNode,
    context: Record<string, any>
  ): Promise<void> {
    switch (node.type) {
      case 'userTask':
        // 创建任务，等待用户办理
        await this.createTask(instance, node, context);
        instance.currentNodeId = node.id;
        break;

      case 'scriptTask':
        // 执行脚本任务
        await this.executeScriptTask(instance, node, context);
        // 脚本任务执行完后自动推进
        await this.moveToNextNodes(instance);
        break;

      case 'serviceTask':
        // 调用外部服务
        await this.executeServiceTask(instance, node, context);
        await this.moveToNextNodes(instance);
        break;

      case 'exclusiveGateway':
        // 单一路由网关，直接推进
        instance.currentNodeId = node.id;
        await this.moveToNextNodes(instance);
        break;

      case 'parallelGateway':
        // 并行网关， fork 分支
        instance.currentNodeId = node.id;
        await this.moveToNextNodes(instance);
        break;

      case 'endEvent':
        // 结束节点
        await this.completeInstance(instance);
        break;
    }

    await this.instanceRepository.save(instance);
  }

  /**
   * 创建用户任务
   */
  private async createTask(
    instance: WorkflowInstance,
    node: FlowNode,
    context: Record<string, any>
  ): Promise<WorkflowTask> {
    // 计算办理人
    let assigneeId: number | null = null;
    let assigneeName: string = '';

    if (node.assigneeType === 'user') {
      // 指定办理人
      assigneeId = await this.resolveAssignee(node.assignee, context);
    } else if (node.assigneeType === 'role') {
      // 按角色查找办理人
      assigneeId = await this.findUserByRole(node.assignee);
    } else if (node.assigneeType === 'department') {
      // 按部门查找办理人
      assigneeId = await this.findDeptManager(context.applicantId);
    } else if (node.assigneeType === 'formula') {
      // 按公式计算办理人
      assigneeId = await this.calculateFormulaAssignee(node.assignee, context);
    }

    // 创建任务
    const task = this.taskRepository.create({
      instanceId: instance.id,
      instanceCode: instance.code,
      nodeId: node.id,
      nodeName: node.name,
      nodeType: node.type,
      assigneeId,
      assigneeName,
      status: 'pending',
      applicantId: instance.applicantId,
      applicantName: instance.applicantName,
      formTitle: instance.formData.title || instance.formData.reason || '审批申请',
      dueAt: node.timeout
        ? new Date(Date.now() + node.timeout * 60 * 60 * 1000)
        : null,
      tenantId: instance.tenantId,
    });

    return this.taskRepository.save(task);
  }

  // ============================================
  // 辅助方法
  // ============================================

  /**
   * 解析流程定义
   * 把数据库中存储的JSON格式解析成对象
   */
  private parseDefinition(definition: WorkflowDefinition): any {
    const config = typeof definition.config === 'string'
      ? JSON.parse(definition.config)
      : definition.config;

    return {
      id: definition.id,
      code: definition.definitionCode,
      name: definition.definitionName,
      version: definition.version,
      nodes: config.nodes || [],
      lines: config.lines || [],
      startNodeId: config.nodes?.find(n => n.type === 'startEvent')?.id,
      endNodeIds: config.nodes?.filter(n => n.type === 'endEvent').map(n => n.id),
      config: config.config || {},
    };
  }

  /**
   * 获取后续节点
   */
  private getNextNodes(
    definition: any,
    currentNodeId: string,
    context: Record<string, any>
  ): FlowNode[] {
    // 找到当前节点的后续连线
    const outgoingLines = definition.lines.filter(
      (l: FlowLine) => l.sourceNodeId === currentNodeId
    );

    const nextNodes: FlowNode[] = [];

    for (const line of outgoingLines) {
      // 检查条件
      if (!line.conditionExpression ||
          this.evaluator.evaluate(line.conditionExpression, context)) {
        const targetNode = definition.nodes.find(
          (n: FlowNode) => n.id === line.targetNodeId
        );
        if (targetNode) {
          nextNodes.push(targetNode);
        }
      }
    }

    return nextNodes;
  }

  /**
   * 完成任务实例
   */
  private async completeInstance(instance: WorkflowInstance): Promise<void> {
    instance.status = 'completed';
    instance.completedAt = new Date();
    instance.duration = Math.floor(
      (instance.completedAt.getTime() - instance.startedAt.getTime()) / 1000
    );
    await this.instanceRepository.save(instance);

    // 触发回调（如果有配置）
    if (instance.callbackUrl) {
      await this.triggerCallback(instance);
    }
  }

  /**
   * 终止流程实例
   */
  private async terminateInstance(
    instance: WorkflowInstance,
    reason: string
  ): Promise<void> {
    instance.status = 'terminated';
    instance.terminateReason = reason;
    instance.completedAt = new Date();
    await this.instanceRepository.save(instance);
  }
}
```

```tsx
  data,
  loading,
  onRefresh,
  onItemClick
}: {
  data?: { list: PendingTask[]; total: number };
  loading: boolean;
  onRefresh: () => void;
  onItemClick: (taskId: number) => void;

  // 加载状态

    return (
      <div style={{ padding: '40px 0', textAlign: 'center' }}>
        <Spin />
      </div>
    );
  }

  // 空状态
  if (!data?.list || data.list.length === 0) {
    return (
      <Empty
        description="暂无待审批项"
        image={<CheckCircleOutline style={{ fontSize: 64, color: '#87d068' }} />}
      />
    );

   * 获取优先级标签
   */

    const config = {
      low: { color: '#95de64', text: '低' },
      normal: { color: '#1890ff', text: '普通' },
      high: { color: '#faad14', text: '紧急' },
      urgent: { color: '#ff4d4f', text: '非常紧急' },
    };
    const { color, text } = config[priority as keyof typeof config] || config.normal;
    return <Tag color={color}>{text}</Tag>;
  };

  /**
   * 计算剩余时间
   */
  const getDueTime = (dueAt?: string) => {
    if (!dueAt) return null;
    const now = new Date();
    const due = new Date(dueAt);
    const diffHours = Math.floor((due.getTime() - now.getTime()) / (1000 * 60 * 60));

    if (diffHours < 0) {
      return <span style={{ color: '#ff4d4f' }}>已超时</span>;
    } else if (diffHours < 24) {
      return <span style={{ color: '#faad14' }}>剩余{diffHours}小时</span>;
    } else {
      const diffDays = Math.floor(diffHours / 24);
      return <span style={{ color: '#87d068' }}>剩余{diffDays}天</span>;
    }
  };

  return (
    <div style={{ padding: '8px 0' }}>
      <List>
        {data.list.map((item) => (
          <List.Item
            key={item.taskId}
            onClick={() => onItemClick(item.taskId)}
            extra={

                    fontWeight: 'bold',
                    color: '#333',
                    fontSize: '15px'
                  }}>
                    {item.nodeName}
                  </span>
                  <span style={{ fontSize: '12px', color: '#999' }}>
                    {item.instanceCode}
                  </span>
                </div>

                {/* 申请标题 */}

                  marginBottom: '8px',
                  overflow: 'hidden',
                  textOverflow: 'ellipsis',
                  whiteSpace: 'nowrap'
                }}>
```

```tsx
import React, { useState } from 'react';
import {
  NavBar,
  Card,
  Button,
  TextArea,
  Dialog,
  Spin,
  List,
  Tag,
  Divider
} from 'antd-mobile';
import { useRouter, useParams } from 'next/router';
import { QuestionCircleOutline, CheckOutline, CloseOutline } from 'antd-mobile-icons';
import useSWR from 'swr';

/**
 * 审批详情页面
 *
 * 这个页面展示一个审批任务的完整信息
 * 包括：申请内容、审批历史、当前审批人
 * 同时提供批准和驳回的操作按钮
 */
export default function ApprovalDetailPage() {
  const router = useRouter();
  const { taskId } = useParams();

  // 审批意见
  const [comment, setComment] = useState('');

  // 操作loading状态
  const [actionLoading, setActionLoading] = useState(false);

  // 获取审批详情
  const { data, isLoading, mutate } = useSWR(
    taskId ? `/api/oa/approval/tasks/${taskId}` : null,
    (url) => fetch(url).then(r => r.json())
  );

  /**
   * 处理批准操作
   */
  const handleApprove = async () => {
    // 二次确认
    const confirmed = await Dialog.confirm({
      content: '确认批准该申请？',
      confirmText: '确认批准',
    });

    if (!confirmed) return;

    setActionLoading(true);
    try {
      const response = await fetch(`/api/oa/approval/tasks/${taskId}/approve`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ comment }),
      });

      if (response.ok) {
        // 刷新数据
        mutate();
        // 跳回列表
        router.back();
      } else {
        const result = await response.json();
        Dialog.alert({ content: result.message || '操作失败' });
      }
    } finally {
      setActionLoading(false);
    }
  };

  /**
   * 处理驳回操作
   */
  const handleReject = async () => {
    if (!comment.trim()) {
      Dialog.alert({ content: '请填写驳回原因' });
      return;
    }

    const confirmed = await Dialog.confirm({
      content: '确认驳回该申请？',
      confirmText: '确认驳回',
    });

    if (!confirmed) return;

    setActionLoading(true);
    try {
      const response = await fetch(`/api/oa/approval/tasks/${taskId}/reject`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ comment }),
      });

      if (response.ok) {
        router.back();
      } else {
        const result = await response.json();
        Dialog.alert({ content: result.message || '操作失败' });
      }
    } finally {
      setActionLoading(false);
    }
  };

  // 加载状态
  if (isLoading) {
    return (
      <div style={{ padding: '40px 0', textAlign: 'center' }}>
        <Spin />
      </div>
    );
  }

  // 数据不存在
  if (!data?.task) {
    return (
      <div style={{ padding: '40px 0', textAlign: 'center' }}>
        <div>任务不存在或已处理</div>
      </div>
    );
  }

  const { task, instance, history } = data;

  return (
    <div style={{ minHeight: '100vh', background: '#f5f5f5' }}>
      {/* 顶部导航 */}
      <NavBar
        onBack={() => router.back()}
        rightText="详情"
      >
        审批详情
      </NavBar>

      {/* 审批内容区域 */}
      <div style={{ padding: '12px' }}>
        {/* 申请信息卡片 */}
        <Card style={{ marginBottom: '12px' }}>
          <Card.Header
            title={<span style={{ fontWeight: 'bold' }}>{instance.definitionName}</span>}
            extra={
              <Tag color={instance.status === 'running' ? 'blue' : 'green'}>
                {instance.status === 'running' ? '审批中' : '已完成'}
              </Tag>
            }
          />
          <Card.Body>
            {/* 申请标题 */}
            <div style={{
              fontSize: '16px',
              fontWeight: 'bold',
              marginBottom: '16px',
              color: '#333'
            }}>
              {instance.formData.title || instance.formData.reason || '审批申请'}
            </div>

            {/* 申请信息列表 */}
            <List>
              <List.Item prefix="申请人">
                {instance.applicantName}
              </List.Item>
              <List.Item prefix="申请部门">
                {instance.applicantDeptName || '—'}
              </List.Item>
              <List.Item prefix="申请时间">
                {new Date(instance.startedAt).toLocaleString()}
              </List.Item>
              <List.Item prefix="当前节点">
                {task.nodeName}
              </List.Item>
            </List>
          </Card.Body>
        </Card>

        {/* 表单数据卡片 */}
        <Card title="申请内容" style={{ marginBottom: '12px' }}>
          <Card.Body>
            {/* 根据不同的流程类型，渲染不同的表单内容 */}
            <FormDataRenderer
              definitionCode={instance.definitionCode}
              formData={instance.formData}
            />
          </Card.Body>
        </Card>

        {/* 审批历史卡片 */}
        <Card title="审批历史" style={{ marginBottom: '12px' }}>
          <Card.Body>
            {history && history.length > 0 ? (
              <div style={{ position: 'relative' }}>
                {/* 时间线 */}
                {history.map((item: any, index: number) => (
                  <div
                    key={index}
                    style={{
                      display: 'flex',
                      marginBottom: index < history.length - 1 ? '16px' : 0
                    }}
                  >
                    {/* 时间线圆点 */}
                    <div style={{
                      width: '12px',
                      height: '12px',
                      borderRadius: '50%',
                      background: index === history.length - 1 ? '#1677ff' : '#d9d9d9',
                      marginTop: '4px',
                      marginRight: '12px',
                      flexShrink: 0
                    }} />

                    {/* 内容 */}
                    <div style={{ flex: 1 }}>
                      <div style={{
                        fontWeight: 'bold',
                        marginBottom: '4px'
                      }}>
                        {item.nodeName}
                      </div>
                      <div style={{ fontSize: '13px', color: '#666' }}>
                        办理人：{item.assigneeName}
                      </div>
                      <div style={{ fontSize: '13px', color: '#666' }}>
                        操作：{
                          item.action === 'submit' ? '提交申请' :
                          item.action === 'approve' ? '批准' :
                          item.action === 'reject' ? '驳回' : item.action
                        }
                      </div>
                      {item.comment && (
                        <div style={{
                          fontSize: '13px',
                          color: '#999',
                          marginTop: '4px',
                          padding: '8px',
                          background: '#f5f5f5',
                          borderRadius: '4px'
                        }}>
                          意见：{item.comment}
                        </div>
                      )}
                      <div style={{ fontSize: '12px', color: '#999', marginTop: '4px' }}>

                    </div>
                  </div>
                ))}
              </div>
            ) : (
              <div style={{ textAlign: 'center', color: '#999', padding: '20px 0' }}>
                暂无审批历史
              </div>
            )}
          </Card.Body>
        </Card>

        {/* 审批操作区域 */}
        <Card>
          <Card.Header title="审批操作" />
          <Card.Body>
            {/* 审批意见输入 */}
            <div style={{ marginBottom: '16px' }}>
              <TextArea
                placeholder="请填写审批意见（驳回时必填）"
                value={comment}

/**
 * 表单数据渲染器
 *
 * 根据不同的流程类型，渲染不同的表单内容
 */
function FormDataRenderer({
  definitionCode,
  formData
}: {
  definitionCode: string;
  formData: Record<string, any>;
}) {
  // 根据流程类型选择渲染方式
  switch (definitionCode) {
    case 'leave_request':
      // 请假申请
      return (
        <List>
          <List.Item prefix="请假类型">
            {formData.leaveType === 'annual' ? '年假' :
             formData.leaveType === 'sick' ? '病假' :
             formData.leaveType === 'personal' ? '事假' : formData.leaveType}
          </List.Item>
          <List.Item prefix="开始时间">
            {formData.startDate}
          </List.Item>
          <List.Item prefix="结束时间">
            {formData.endDate}
          </List.Item>
          <List.Item prefix="请假时长">
            {formData.duration} 天
          </List.Item>
          <List.Item prefix="请假原因">
            {formData.reason}
          </List.Item>
          {formData.attachments?.length > 0 && (
            <List.Item prefix="附件">
              {formData.attachments.map((url: string, i: number) => (
                <a key={i} href={url} target="_blank" rel="noopener noreferrer">
                  附件{i + 1}
                </a>
              ))}
            </List.Item>
          )}
        </List>
      );

    case 'expense_claim':
      // 报销申请
      return (
        <List>
          <List.Item prefix="报销类型">
            {formData.expenseType}
          </List.Item>
          <List.Item prefix="报销金额">
            ¥{formData.amount?.toLocaleString()}
          </List.Item>
          <List.Item prefix="费用明细">
            {formData.expenseItems?.map((item: any, i: number) => (
              <div key={i} style={{ marginBottom: '4px' }}>
                {item.type}: ¥{item.amount} ({item.date})
              </div>
            ))}
          </List.Item>
          <List.Item prefix="备注">
            {formData.remark || '—'}
          </List.Item>
        </List>
      );

    default:
      // 默认渲染方式：遍历所有字段
      return (
        <List>
          {Object.entries(formData).map(([key, value]) => (
            <List.Item key={key} prefix={key}>
              {typeof value === 'object' ? JSON.stringify(value) : String(value)}
            </List.Item>
          ))}
        </List>
      );
  }
}
```

```typescript
// ============================================
// 考勤打卡服务
// ============================================

// src/attendance/attendance.service.ts
import { Injectable } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository, Between } from 'typeorm';
import { AttendanceRecord } from './entities/attendance-record.entity';
import { AttendanceRule } from './entities/attendance-rule.entity';

/**
 * 打卡请求
 */
interface CheckInRequest {
  userId: number;
  longitude?: number;     // 经度
  latitude?: number;       // 纬度
  deviceInfo?: string;     // 设备信息
  checkType: 'check_in' | 'check_out';  // 上班打卡/下班打卡
}

/**
 * 打卡结果
 */
interface CheckInResult {
  success: boolean;
  record?: AttendanceRecord;
  message: string;
  status: 'normal' | 'late' | 'early' | 'absence';
  overtime?: boolean;     // 是否加班
}

/**
 * 考勤规则
 */
interface WorkTimeRule {
  checkInStart: string;   // 上班打卡开始时间，如 "08:00"
  checkInEnd: string;      // 上班打卡结束时间，如 "09:30"
  checkOutStart: string;   // 下班打卡开始时间，如 "17:00"
  checkOutEnd: string;     // 下班打卡结束时间，如 "22:00"
  workHours: number;       // 每日工作时长（小时）
  requireLocation: boolean;  // 是否需要定位
  locationRange: number;     // 允许的定位范围（米）
}

@Injectable()
export class AttendanceService {
  constructor(
    @InjectRepository(AttendanceRecord)
    private attendanceRepository: Repository<AttendanceRecord>,

    @InjectRepository(AttendanceRule)
    private ruleRepository: Repository<AttendanceRule>,
  ) {}

  /**
   * 员工打卡
   *
   * @param request 打卡请求
   */
  async checkIn(request: CheckInRequest): Promise<CheckInResult> {
    const { userId, checkType, longitude, latitude } = request;

    // 1. 获取考勤规则
    const rule = await this.getAttendanceRule(userId);

    // 2. 获取今天的日期（只保留日期部分）
    const today = this.getDateOnly(new Date());

    // 3. 检查今天是否已经打过卡
    const existingRecord = await this.attendanceRepository.findOne({
      where: {
        userId,
        checkDate: today,
      }
    });

    // 4. 判断是上班打卡还是下班打卡
    if (checkType === 'check_in') {
      return this.processCheckIn(existingRecord, userId, today, rule, longitude, latitude);
    } else {
      return this.processCheckOut(existingRecord, userId, today, rule, longitude, latitude);
    }
  }

  /**
   * 处理上班打卡
   */
  private async processCheckIn(
    existingRecord: AttendanceRecord | null,
    userId: number,
    checkDate: Date,
    rule: WorkTimeRule,
    longitude?: number,
    latitude?: number
  ): Promise<CheckInResult> {
    // 如果已经有上班打卡记录，说明重复打卡
    if (existingRecord?.checkInTime) {
      return {
        success: false,
        message: '今日已打过上班卡',
        status: 'normal',
      };
    }

    // 获取当前时间
    const now = new Date();
    const currentTimeStr = this.formatTime(now);

    // 判断是否迟到
    // 上班打卡结束时间是09:30，如果超过就算迟到
    let status: 'normal' | 'late' = 'normal';
    if (currentTimeStr > rule.checkInEnd) {
      status = 'late';
    }

    // 创建打卡记录
    const record = this.attendanceRepository.create({
      userId,
      checkDate,
      checkInTime: now,
      checkInStatus: status,
      // 记录打卡位置
      checkInLongitude: longitude,
      checkInLatitude: latitude,
    });

    await this.attendanceRepository.save(record);

    // 返回结果
    const messages = {
      normal: '打卡成功',
      late: `打卡成功，已迟到 ${this.calculateLateMinutes(now, rule)} 分钟`,
    };

    return {
      success: true,
      record,
      message: messages[status],
      status,
    };
  }

  /**
   * 处理下班打卡
   */
  private async processCheckOut(
    existingRecord: AttendanceRecord | null,
    userId: number,
    checkDate: Date,
    rule: WorkTimeRule,
    longitude?: number,
    latitude?: number
  ): Promise<CheckInResult> {
    // 检查是否打过上班卡
    if (!existingRecord?.checkInTime) {
      return {
        success: false,
        message: '请先打上班卡',
        status: 'normal',
      };
    }

    // 如果已经有下班打卡记录，说明重复打卡
    if (existingRecord.checkOutTime) {
      return {
        success: false,
        message: '今日已打过下班卡',
        status: 'normal',
      };
    }

    // 获取当前时间
    const now = new Date();
    const currentTimeStr = this.formatTime(now);

    // 判断是否早退
    // 下班打卡开始时间是17:00，如果提前就算早退
    let status: 'normal' | 'early' = 'normal';
    let overtime = false;

    if (currentTimeStr < rule.checkOutStart) {
      status = 'early';
    } else if (currentTimeStr > rule.checkOutEnd) {
      // 如果超过下班打卡结束时间，算加班
      overtime = true;
    }

    // 计算工作时长
    const workDuration = this.calculateWorkDuration(
      existingRecord.checkInTime,
      now
    );

    // 更新打卡记录
    existingRecord.checkOutTime = now;
    existingRecord.checkOutStatus = status;
    existingRecord.checkOutLongitude = longitude;
    existingRecord.checkOutLatitude = latitude;
    existingRecord.workDuration = workDuration;

    // 判断整体考勤状态
    // 上班迟到 或 下班早退，都算异常
    if (existingRecord.checkInStatus === 'late' || status === 'early') {
      existingRecord.attendanceStatus = existingRecord.checkInStatus === 'late' && status === 'early'
        ? 'late_and_early'
        : existingRecord.checkInStatus === 'late' ? 'late' : 'early';
    }

    await this.attendanceRepository.save(existingRecord);

    // 返回结果
    let message = '打卡成功';
    if (status === 'early') {
      message = `打卡成功，提前 ${this.calculateEarlyMinutes(now, rule)} 分钟下班`;
    } else if (overtime) {
      message = `打卡成功，已加班 ${this.calculateOvertimeMinutes(now, rule)} 分钟`;
    }

    return {
      success: true,
      record: existingRecord,
      message,
      status,
      overtime,
    };
  }

  // ============================================
  // 辅助方法
  // ============================================

  /**
   * 获取考勤规则
   * 优先获取个人规则，没有则获取默认规则
   */
  private async getAttendanceRule(userId: number): Promise<WorkTimeRule> {
    // 实际的实现应该从数据库查询
    // 这里返回默认规则
    return {
      checkInStart: '08:00',
      checkInEnd: '09:30',
      checkOutStart: '17:00',
      checkOutEnd: '22:00',
      workHours: 8,
      requireLocation: true,
      locationRange: 200,
    };
  }

  /**
   * 获取日期部分（去掉时间）
   */
  private getDateOnly(date: Date): Date {
    const d = new Date(date);
    d.setHours(0, 0, 0, 0);
    return d;
  }

  /**
   * 格式化时间为 HH:mm 格式
   */
  private formatTime(date: Date): string {
    const hours = String(date.getHours()).padStart(2, '0');
    const minutes = String(date.getMinutes()).padStart(2, '0');
    return `${hours}:${minutes}`;
  }

  /**
   * 计算迟到分钟数
   */
  private calculateLateMinutes(now: Date, rule: WorkTimeRule): number {
    const nowMinutes = now.getHours() * 60 + now.getMinutes();
    const endMinutes = this.parseTime(rule.checkInEnd);
    return Math.max(0, nowMinutes - endMinutes);
  }

  /**
   * 计算早退分钟数
   */
  private calculateEarlyMinutes(now: Date, rule: WorkTimeRule): number {
    const nowMinutes = now.getHours() * 60 + now.getMinutes();
    const startMinutes = this.parseTime(rule.checkOutStart);
    return Math.max(0, startMinutes - nowMinutes);
  }

  /**
   * 计算加班分钟数
   */
  private calculateOvertimeMinutes(now: Date, rule: WorkTimeRule): number {
    const nowMinutes = now.getHours() * 60 + now.getMinutes();
    const endMinutes = this.parseTime(rule.checkOutEnd);
    return Math.max(0, nowMinutes - endMinutes);
  }

  /**
   * 计算工作时长（分钟）
   */
  private calculateWorkDuration(checkInTime: Date, checkOutTime: Date): number {
    return Math.floor((checkOutTime.getTime() - checkInTime.getTime()) / (1000 * 60));
  }

  /**
   * 解析时间为分钟数
   */
  private parseTime(timeStr: string): number {
    const [hours, minutes] = timeStr.split(':').map(Number);
    return hours * 60 + minutes;
  }
}
```

```typescript
      userId,
      startDate: Between(startDate, endDate),
      status: 'approved'
    }
  });

  // 6. 计算统计数据
  let lateCount = 0;
  let lateMinutes = 0;
  let earlyLeaveCount = 0;
  let earlyLeaveMinutes = 0;
  let actualWorkDays = 0;
  let overtimeHours = 0;

  const dailyDetails: DailyAttendance[] = workdays.map(date => {
    const dateStr = this.formatDate(date);
    const record = records.find(r => this.formatDate(r.checkDate) === dateStr);

    const isWeekend = date.getDay() === 0 || date.getDay() === 6;
    const isHoliday = isHolidayDate(date);
    const isWorkday = !isWeekend && !isHoliday;

    let status = '未打卡';
    let workDuration: number | undefined;

    if (record) {
      if (record.checkInStatus === 'late') {
        lateCount++;
        lateMinutes += this.calculateLateMinutes(record.checkInTime);
        status = '迟到';
      }

      if (record.checkOutStatus === 'early') {
        earlyLeaveCount++;
        earlyLeaveMinutes += this.calculateEarlyMinutes(record.checkOutTime);
        status = status === '迟到' ? '迟到早退' : '早退';
      }

      if (record.workDuration) {
        workDuration = record.workDuration;
        // 超过8小时算加班
        if (record.workDuration > 480) {
          overtimeHours += (record.workDuration - 480) / 60;
        }
        actualWorkDays++;
      }

      if (record.checkInTime && record.checkOutTime) {
        status = status === '未打卡' ? '正常' : status;
      }
    } else if (isWorkday) {
      // 工作日未打卡
      status = '缺勤';
    }

    return {
      date: dateStr,
      dayOfWeek: ['周日', '周一', '周二', '周三', '周四', '周五', '周六'][date.getDay()],
      checkInTime: record?.checkInTime ? this.formatTime(record.checkInTime) : undefined,
      checkOutTime: record?.checkOutTime ? this.formatTime(record.checkOutTime) : undefined,
      workDuration,
      status,
      isWorkday,
    };
  });

  // 7. 汇总请假统计
  const leaveDetails = this.summarizeLeaves(leaves);

  // 8. 计算应出勤天数
  const totalWorkDays = workdays.filter(d => {
    const date = new Date(d);
    const isWeekend = date.getDay() === 0 || date.getDay() === 6;
    const isHoliday = isHolidayDate(date);
    return !isWeekend && !isHoliday;
  }).length;

  return {
    userId,
    userName: user.name,
    deptName: user.deptName,
    month,
    totalWorkDays,
    actualWorkDays,
    lateCount,
    lateMinutes,
    earlyLeaveCount,
    earlyLeaveMinutes,
    absenceCount: totalWorkDays - actualWorkDays,
    leaveDetails,
    overtimeHours: Math.round(overtimeHours * 10) / 10,
    normalDays: actualWorkDays,
    exceptionDays: lateCount + earlyLeaveCount,
    records: dailyDetails,
  };
}
```

```typescript
/**
 * 会议室预约服务

// src/meeting/meeting.service.ts
import { Injectable, ConflictException } from '@nestjs/common';
import { InjectRepository } from '@nestjs/typeorm';
import { Repository, Between, Not } from 'typeorm';
import { Meeting } from './entities/meeting.entity';
import { MeetingRoom } from './entities/meeting-room.entity';

/**
 * 会议室可用时段
 */
interface TimeSlot {
  startTime: string;
  endTime: string;
  available: boolean;
  meeting?: {
    title: string;
    organizer: string;
  };
}

/**
 * 会议室预约请求
 */
interface BookingRequest {
  roomId: number;
  title: string;
  startTime: Date;
  endTime: Date;
  attendees: number[];
  requiredAttendees: number[];
  organizerId: number;
  description?: string;
  isOnline: boolean;
  onlineMeetingLink?: string;
}

/**
 * 会议室预约服务
 */
@Injectable()
export class MeetingService {
  constructor(
    @InjectRepository(Meeting)
    private meetingRepository: Repository<Meeting>,

    @InjectRepository(MeetingRoom)
    private roomRepository: Repository<MeetingRoom>,
  ) {}

  /**
   * 预约会议室
   */
  async bookRoom(request: BookingRequest): Promise<Meeting> {
    // 1. 检查会议室是否存在
    const room = await this.roomRepository.findOne({
      where: { id: request.roomId }
    });

    if (!room) {
      throw new Error('会议室不存在');
    }

    // 2. 检查会议室容量
    if (request.attendees.length > room.capacity) {
      throw new Error(`参会人数超过会议室容量（${room.capacity}人）`);
    }

    // 3. 检查时间冲突
    const hasConflict = await this.checkTimeConflict(
      request.roomId,
      request.startTime,
      request.endTime
    );

    if (hasConflict) {
      throw new ConflictException('该时间段已被预约');
    }

    // 4. 计算会议时长
    const duration = Math.floor(
      (request.endTime.getTime() - request.startTime.getTime()) / (1000 * 60)
    );

    // 5. 创建会议记录
    const meeting = this.meetingRepository.create({
      roomId: request.roomId,
      roomName: room.name,
      meetingTitle: request.title,
      startTime: request.startTime,
      endTime: request.endTime,
      duration,
      organizerId: request.organizerId,
      attendees: request.attendees.map(id => ({ userId: id, status: 'pending' })),
      requiredAttendees: request.requiredAttendees,
      agenda: request.description,
      status: 'scheduled',
      tenantId: request.organizerId,  // 简化处理
    });

    const savedMeeting = await this.meetingRepository.save(meeting);

    // 6. 发送会议通知
    await this.sendMeetingNotification(savedMeeting, request.attendees);

    return savedMeeting;
  }

  /**
   * 检查时间冲突
   */
  private async checkTimeConflict(
    roomId: number,
    startTime: Date,
    endTime: Date
  ): Promise<boolean> {
    // 查询该会议室在同一时间段是否有其他会议
    const conflict = await this.meetingRepository.findOne({
      where: {
        roomId,
        status: 'scheduled',
        // 时间段有重叠的条件：
        // 新会议的结束时间 > 已有会议的开始时间
        // 新会议的开始时间 < 已有会议的结束时间
        startTime: Between(startTime, endTime),
        endTime: Between(startTime, endTime),
      },
    });

    // 另一种查询方式：使用 NOT 条件
    // 即：不是 (新会议结束 <= 已有会议开始) 或 (新会议开始 >= 已有会议结束)
    // 用SQL表示就是：NOT (end_time <= existing_start OR start_time >= existing_end)

    return !!conflict;
  }

  /**
   * 获取会议室可用时段
   */
  async getAvailableSlots(
    roomId: number,
    date: Date
  ): Promise<TimeSlot[]> {
    // 获取工作时间配置（假设8:00-22:00）
    const workStart = 8;
    const workEnd = 22;

    // 获取当天的会议预约
    const dayStart = new Date(date);
    dayStart.setHours(0, 0, 0, 0);
    const dayEnd = new Date(date);
    dayEnd.setHours(23, 59, 59, 999);

    const meetings = await this.meetingRepository.find({
      where: {
        roomId,
        status: 'scheduled',
        startTime: Between(dayStart, dayEnd),
      },
      order: { startTime: 'ASC' }
    });

    // 生成时段列表
    const slots: TimeSlot[] = [];

    for (let hour = workStart; hour < workEnd; hour++) {
      const slotStart = new Date(date);
      slotStart.setHours(hour, 0, 0, 0);

      const slotEnd = new Date(date);
      slotEnd.setHours(hour + 1, 0, 0, 0);

      // 检查这个时段是否被预约
      const booking = meetings.find(m =>
        m.startTime < slotEnd && m.endTime > slotStart
      );

      slots.push({
        startTime: `${hour.toString().padStart(2, '0')}:00`,
        endTime: `${(hour + 1).toString().padStart(2, '0')}:00`,
        available: !booking,
        meeting: booking ? {
          title: booking.meetingTitle,
          organizer: booking.organizerName,
        } : undefined,
      });
    }

    return slots;
  }

  /**
   * 发送会议通知
   */
  private async sendMeetingNotification(
    meeting: Meeting,
    attendeeIds: number[]
  ): Promise<void> {
    // 实际实现应该调用消息通知服务
    // 这里只是占位
    console.log(`发送会议通知给 ${attendeeIds.length} 个参会者`);

    for (const id of attendeeIds) {
      // 1. 站内通知
      await notificationService.send({
        userId: id,
        type: 'meeting',
        title: '会议邀请',
        content: `您被邀请参加会议：${meeting.meetingTitle}`,
        data: { meetingId: meeting.id },
      });

      // 2. 邮件通知
      const user = await getUserById(id);
      if (user.email) {
        await emailService.send({
          to: user.email,
          subject: `【会议邀请】${meeting.meetingTitle}`,
          body: this.renderMeetingEmail(meeting),
        });
      }
    }
  }
}
```
