# 电商系统项目实战完全指南

## 一、项目概述与需求分析

### 1.1 项目背景

电商系统是互联网时代最核心的业务系统之一，承载着商品交易、用户管理、支付结算等关键业务流程。一个完整的电商系统需要解决诸多技术挑战：高并发访问、海量数据处理、交易一致性、安全支付、用户体验优化等。本指南将从零开始，详细讲解如何使用现代Web技术栈构建一个功能完整、性能优异的电商系统。

本项目名为 **"FastShop"**，是一个面向中小型企业的B2C电商平台，核心功能包括用户注册登录、商品浏览与搜索、购物车管理、订单处理、在线支付、用户中心等模块。系统采用前后端分离架构，前端使用Next.js 16 + React 19构建，后端使用Node.js + Express构建，数据库采用PostgreSQL存储核心业务数据，Redis用于缓存和会话管理。

### 1.2 需求分析

#### 1.2.1 功能性需求

**用户模块**

- 用户注册：支持手机号、邮箱注册，需邮箱/短信验证码激活
- 用户登录：支持账号密码登录、短信验证码登录、第三方登录（微信、支付宝）
- 个人信息管理：修改昵称、头像、收货地址管理
- 会员等级体系：根据消费金额划分会员等级，不同等级享有不同折扣

**商品模块**

- 商品展示：首页推荐、商品分类、详情页展示
- 商品搜索：关键词搜索、分类筛选、价格区间筛选、排序
- 商品评论：购买后对商品进行评价打分
- 收藏功能：用户收藏商品到收藏夹

**购物车模块**

- 添加商品到购物车：支持选择规格、数量
- 修改购物车：修改数量、删除商品、选择商品
- 购物车持久化：登录用户购物车数据同步到服务器
- 结算入口：选择收货地址、优惠券、支付方式后发起结算

**订单模块**

- 创建订单：确认订单信息、计算价格、生成订单
- 订单支付：调用支付接口、完成支付、支付结果查询
- 订单列表：待付款、待发货、待收货、已完成、已取消等状态筛选
- 订单详情：展示订单完整信息、物流状态
- 取消订单：用户主动取消、超时自动取消
- 售后退款：申请退款、填写退款原因、客服审核

**支付模块**

- 支付方式：微信支付、支付宝支付、银行卡支付
- 支付流程：创建支付订单、跳转支付页面、支付结果回调
- 支付安全：签名验证、订单状态同步、幂等性处理

#### 1.2.2 非功能性需求

**性能需求**

- 首页加载时间不超过2秒
- 商品搜索响应时间不超过500毫秒
- 支持10000并发用户访问
- 订单处理峰值达到每秒500单

**安全需求**

- 用户密码加密存储（bcrypt）
- 敏感数据传输使用HTTPS
- 防止SQL注入、XSS攻击、CSRF攻击
- 支付接口需签名验证和回调验签

**可用性需求**

- 系统可用性达到99.9%
- 支持灰度发布和快速回滚
- 完整的日志记录和监控告警

### 1.3 技术选型理由

**前端技术选型：Next.js 16 + React 19**

选择Next.js作为前端框架的核心原因在于其优秀的服务端渲染能力。电商网站对SEO有较高要求，商品详情页、分类页等内容需要被搜索引擎索引，服务端渲染可以显著提升这类页面的SEO效果。Next.js 16的App Router模式提供了更直观的项目结构，React Server Components让我们能够在服务端直接获取数据，减少客户端JavaScript体积。Turbopack构建工具的引入使得开发体验更加流畅，热更新速度提升明显。

React 19带来的新特性同样为电商场景带来便利。useOptimistic可以在用户添加购物车时立即更新UI，提供即时反馈；useActionState简化了表单处理流程；React的并发特性确保了复杂交互下的流畅体验。

**后端技术选型：Node.js + Express**

Node.js的非阻塞I/O模型特别适合电商系统这类I/O密集型应用。商品查询、订单处理等操作涉及大量数据库读写，Node.js能够有效利用系统资源处理并发请求。Express作为成熟的Node.js框架，提供了丰富的中间件生态，可以快速构建RESTful API。

选择Express而非NestJS的原因在于教学目的。Express的中间件模式和路由设计更加直观，有助于理解Web开发的基础概念。对于学习者而言，从Express开始更能打好基础，理解HTTP协议、Web服务运行的本质原理。

**数据库选型：PostgreSQL + Redis**

PostgreSQL是功能最强大的开源关系型数据库，支持复杂查询、事务处理、JSON存储等特性。电商系统的核心数据（用户、订单、商品）具有明确的表结构关系，适合使用PostgreSQL存储。其强大的查询优化器和索引机制能够支撑大数据量下的复杂查询场景。

Redis作为内存数据库，主要承担缓存和会话存储的角色。电商系统的热点数据（如商品信息、分类列表）非常适合缓存到Redis中，可以显著降低数据库压力。用户会话、购物车数据、分布式锁等场景也依赖Redis实现。选择Redis还因为其丰富的数据结构（String、Hash、List、Set、Sorted Set），可以满足不同业务场景的需求。

## 三、数据模型设计

### 3.1 数据库选型与设计原则

PostgreSQL作为关系型数据库，采用ER模型进行数据设计。在设计过程中遵循以下原则：

**范式化设计**：减少数据冗余，确保数据一致性。订单商品表需要冗余商品名称和价格，因为商品信息可能会变化，但订单记录需要保持历史状态。

**索引优化**：针对查询场景建立合适索引。用户表建立邮箱、手机号唯一索引；商品表建立分类ID、价格区间索引；订单表建立用户ID、创建时间、状态索引。

**分区表策略**：订单数据按时间分区，历史订单归档到单独表空间，提升查询性能。

### 3.2 核心数据模型

#### 3.2.1 用户模型（User）

```typescript
// types/user.ts - 用户类型定义

/**
 * 用户角色枚举
 * - USER: 普通用户
 * - VIP: 会员用户
 * - ADMIN: 管理员
 */
export enum UserRole {
  USER = 'USER',
  VIP = 'VIP',
  ADMIN = 'ADMIN'
}

/**
 * 用户账号状态枚举
 * - ACTIVE: 正常状态，可正常登录使用
 * - INACTIVE: 未激活，需要邮箱或短信验证
 * - BANNED: 被禁用，账号异常被封禁
 * - DELETED: 已注销，账号已删除
 */
export enum UserStatus {
  ACTIVE = 'ACTIVE',
  INACTIVE = 'INACTIVE',
  BANNED = 'BANNED',
  DELETED = 'DELETED'
}

/**
 * 用户实体接口
 * 包含用户注册和认证的核心信息
 */
export interface User {
  id: string;                    // UUID主键
  email: string;                 // 邮箱地址，唯一索引
  phone?: string;                // 手机号，可选，唯一索引
  password: string;             // 加密后的密码
  nickname: string;              // 用户昵称
  avatar?: string;               // 头像URL
  role: UserRole;                // 用户角色
  status: UserStatus;           // 账号状态
  vipLevel: number;              // 会员等级，0为非会员
  vipExpireTime?: Date;          // 会员过期时间
  totalSpent: number;            // 累计消费金额
  createdAt: Date;               // 创建时间
  updatedAt: Date;                // 更新时间
  lastLoginAt?: Date;             // 最后登录时间
}
```

#### 3.2.2 商品模型（Product）

```typescript
// types/product.ts - 商品类型定义

/**
 * 商品状态枚举
 * - DRAFT: 草稿，未上架
 * - ON_SALE: 在售，可购买
 * - OFF_SALE: 下架
 * - SOLD_OUT: 售罄
 * - DELETED: 已删除
 */
export enum ProductStatus {
  DRAFT = 'DRAFT',
  ON_SALE = 'ON_SALE',
  OFF_SALE = 'OFF_SALE',
  SOLD_OUT = 'SOLD_OUT',
  DELETED = 'DELETED'
}

/**
 * 商品分类实体接口
 */
export interface Category {
  id: string;                    // 分类ID
  name: string;                  // 分类名称
  parentId?: string;             // 父分类ID，支持多级分类
  level: number;                 // 分类层级
  sort: number;                  // 排序权重
  icon?: string;                 // 分类图标
  createdAt: Date;
  updatedAt: Date;
}

/**
 * 商品规格项接口
 * 例如颜色、尺寸等规格的具体值
 */
export interface SKUItem {
  id: string;                    // 规格项ID
  name: string;                  // 规格项名称，如"红色"
  attrName: string;              // 所属规格名称，如"颜色"
  sort: number;                  // 排序
}

/**
 * 商品规格接口
 * 定义商品的规格类型，如颜色、尺寸
 */
export interface ProductSpec {
  id: string;                    // 规格ID
  name: string;                  // 规格名称，如"颜色"
  type: 'color' | 'size' | 'text'; // 规格类型
  isRequired: boolean;           // 是否必选
  values: SKUItem[];             // 规格值列表
}

/**
 * 商品SKU接口
 * 具体可购买的商品规格组合
 */
export interface SKU {
  id: string;                    // SKU ID
  productId: string;             // 所属商品ID
  specs: Record<string, string>; // 规格组合，key为规格ID，value为规格值ID
  price: number;                 // 价格（分）
  originalPrice: number;         // 原价（分）
  stock: number;                 // 库存数量
  skuCode: string;               // SKU编码
  image?: string;                // SKU主图
  weight?: number;               // 重量（克），用于物流计算
  enabled: boolean;              // 是否启用
}

/**
 * 商品实体接口
 * 包含商品的核心信息和销售属性
 */
export interface Product {
  id: string;                    // 商品ID
  name: string;                  // 商品名称
  subtitle?: string;             // 商品副标题/简介
  categoryId: string;            // 所属分类ID
  brand?: string;                // 品牌名称
  description: string;           // 商品详细描述（富文本）
  images: string[];              // 商品图片列表
  tags: string[];                // 商品标签
  specs: ProductSpec[];          // 商品规格定义
  skus: SKU[];                   // 商品SKU列表
  price: number;                 // 最低价（分）
  originalPrice: number;         // 最高原价（分）
  costPrice: number;             // 成本价（分）
  stock: number;                 // 总库存
  soldCount: number;             // 已售数量
  viewCount: number;             // 浏览次数
  favoriteCount: number;         // 收藏次数
  status: ProductStatus;         // 商品状态
  rating: number;                // 平均评分
```

```typescript
// backend/src/services/authService.ts - 认证服务层

import bcrypt from 'bcrypt';
import jwt from 'jsonwebtoken';
import { AppDataSource } from '../config/database';
import { User, UserStatus, UserRole } from '../models/User';
import { redis } from '../config/redis';
import { sendEmail } from '../utils/email';
import { generateToken, verifyToken } from '../utils/jwt';
import { v4 as uuidv4 } from 'uuid';

/**
 * 盐值轮数，安全强度与性能平衡
 * 生产环境建议12以上
 */
const SALT_ROUNDS = 10;

/**
 * Token有效期配置
 */
const TOKEN_EXPIRY = {
  accessToken: '15m',      // 访问令牌15分钟
  refreshToken: '7d',      // 刷新令牌7天
};

/**
 * 认证服务类
 * 处理用户注册、登录、Token管理等核心认证逻辑
 */
export class AuthService {
  private userRepository = AppDataSource.getRepository(User);

  /**
   * 用户注册
   * @param email 邮箱地址
   * @param password 密码（明文）
   * @param nickname 昵称
   * @returns 注册结果，包含用户信息和Token
   */
  async register(email: string, password: string, nickname: string): Promise<{
    user: Partial<User>;
    accessToken: string;
    refreshToken: string;
  }> {
    // 1. 检查邮箱是否已被注册
    const existingUser = await this.userRepository.findOne({
      where: { email }
    });

    if (existingUser) {
      throw new Error('该邮箱已被注册');
    }

    // 2. 密码加密存储
    // 使用bcrypt进行单向哈希加密，即使数据库泄露也无法还原原始密码
    const hashedPassword = await bcrypt.hash(password, SALT_ROUNDS);

    // 3. 创建用户记录
    const user = this.userRepository.create({
      email,
      password: hashedPassword,
      nickname,
      status: UserStatus.INACTIVE,  // 新用户需激活
      role: UserRole.USER,
      vipLevel: 0,
    });

    await this.userRepository.save(user);

    // 4. 生成激活验证码并发送邮件
    const activationCode = this.generateActivationCode(user.id);
    await this.sendActivationEmail(email, activationCode);

    // 5. 生成JWT令牌
    const tokens = this.generateTokens(user);

    return {
      user: this.sanitizeUser(user),
      ...tokens,
    };
  }

  /**
   * 用户登录
   * @param email 邮箱
   * @param password 密码（明文）
   * @returns 登录结果
   */
  async login(email: string, password: string): Promise<{
    user: Partial<User>;
    accessToken: string;
    refreshToken: string;
  }> {
    // 1. 查找用户
    const user = await this.userRepository.findOne({
      where: { email }
    });

    if (!user) {
      throw new Error('邮箱或密码错误');
    }

    // 2. 验证账号状态
    if (user.status === UserStatus.BANNED) {
      throw new Error('账号已被禁用，请联系客服');
    }

    if (user.status === UserStatus.DELETED) {
      throw new Error('账号已注销');
    }

    if (user.status === UserStatus.INACTIVE) {
      throw new Error('账号未激活，请先激活账号');
    }

    // 3. 验证密码
    // bcrypt.compare进行常量时间比较，防止时序攻击
    const isPasswordValid = await bcrypt.compare(password, user.password);

    if (!isPasswordValid) {
      throw new Error('邮箱或密码错误');
    }

    // 4. 更新最后登录时间
    user.lastLoginAt = new Date();
    await this.userRepository.save(user);

    // 5. 生成令牌
    const tokens = this.generateTokens(user);

    return {
      user: this.sanitizeUser(user),
      ...tokens,
    };
  }

  /**
   * 激活账号
   * @param userId 用户ID
   * @param code 激活码
   */
  async activateAccount(userId: string, code: string): Promise<void> {
    // 从Redis获取激活码
    const storedCode = await redis.get(`activation:${userId}`);

    if (!storedCode || storedCode !== code) {
      throw new Error('激活码无效或已过期');
    }

    // 更新用户状态
    await this.userRepository.update(userId, {
      status: UserStatus.ACTIVE
    });

    // 删除激活码
    await redis.del(`activation:${userId}`);
  }

  /**
   * 刷新Token
   * @param refreshToken 刷新令牌
   * @returns 新的令牌对
   */
  async refreshTokens(refreshToken: string): Promise<{
    accessToken: string;
    refreshToken: string;
  }> {
    try {
      // 验证refreshToken
      const decoded = verifyToken(refreshToken, 'refresh') as { userId: string };

      // 检查Token是否在黑名单
      const isBlacklisted = await redis.get(`blacklist:refresh:${refreshToken}`);
      if (isBlacklisted) {
        throw new Error('Token已失效，请重新登录');
      }

      // 获取用户信息
      const user = await this.userRepository.findOne({
        where: { id: decoded.userId }
      });

      if (!user || user.status !== UserStatus.ACTIVE) {
        throw new Error('用户不存在或已禁用');
      }

      // 将旧Token加入黑名单
      await redis.setex(`blacklist:refresh:${refreshToken}`, 7 * 24 * 60 * 60, '1');

      // 生成新令牌
      return this.generateTokens(user);
    } catch (error) {
      throw new Error('无效的刷新Token');
    }
  }

  /**
   * 用户登出
   * 将Token加入黑名单
   */
  async logout(userId: string, accessToken: string, refreshToken: string): Promise<void> {
    // 获取Token的剩余有效期
    const decoded = verifyToken(accessToken) as { exp: number };
    const ttl = decoded.exp - Math.floor(Date.now() / 1000);

    if (ttl > 0) {
      // 将AccessToken加入黑名单
      await redis.setex(`blacklist:access:${accessToken}`, ttl, '1');
    }

    // 将RefreshToken加入黑名单
    await redis.setex(`blacklist:refresh:${refreshToken}`, 7 * 24 * 60 * 60, '1');
  }

  /**
   * 修改密码
   */
  async changePassword(userId: string, oldPassword: string, newPassword: string): Promise<void> {
    const user = await this.userRepository.findOne({
      where: { id: userId }
    });

    if (!user) {
      throw new Error('用户不存在');
    }

    // 验证旧密码
    const isValid = await bcrypt.compare(oldPassword, user.password);
    if (!isValid) {
      throw new Error('原密码错误');
    }

    // 更新密码
    const hashedPassword = await bcrypt.hash(newPassword, SALT_ROUNDS);
    await this.userRepository.update(userId, {
      password: hashedPassword
    });

    // 使所有现有Token失效（可选的安全增强措施）
    await this.invalidateAllUserTokens(userId);
  }

  /**
   * 重置密码（通过邮箱验证码）
   */
  async resetPassword(email: string, code: string, newPassword: string): Promise<void> {
    // 验证验证码
    const storedCode = await redis.get(`reset:${email}`);
    if (!storedCode || storedCode !== code) {
      throw new Error('验证码无效或已过期');
    }

    // 更新密码
    const hashedPassword = await bcrypt.hash(newPassword, SALT_ROUNDS);
    await this.userRepository.update({ email }, {
      password: hashedPassword
    });

    // 删除验证码
    await redis.del(`reset:${email}`);
  }

  /**
   * 生成Token对
   * @param user 用户实体
   * @returns AccessToken和RefreshToken
   */
  private generateTokens(user: User): { accessToken: string; refreshToken: string } {
    // AccessToken包含用户ID和角色信息
    const accessToken = generateToken(
      { userId: user.id, role: user.role },
      'access',
      TOKEN_EXPIRY.accessToken
    );

    // RefreshToken只包含用户ID
    const refreshToken = generateToken(
      { userId: user.id },
      'refresh',
      TOKEN_EXPIRY.refreshToken
    );

    return { accessToken, refreshToken };
  }

  /**
   * 使所有用户Token失效
   * 用于密码修改等安全敏感操作后
   */
  private async invalidateAllUserTokens(userId: string): Promise<void> {
    // 在Redis中记录用户需要Token黑名单验证
    // 这里简化处理，实际可使用用户级别的大版本号
    await redis.incr(`user:token:version:${userId}`);
  }

  /**
   * 生成激活码
   */
  private generateActivationCode(userId: string): string {
    const code = uuidv4().replace(/-/g, '').substring(0, 6).toUpperCase();
    // 激活码有效期15分钟
    redis.setex(`activation:${userId}`, 15 * 60, code);
    return code;
  }

  /**
   * 发送激活邮件
   */
  private async sendActivationEmail(email: string, code: string): Promise<void> {

      to: email,
      subject: 'FastShop账号激活',

        <h1>感谢注册FastShop</h1>

        <p>验证码15分钟内有效，请尽快完成激活。</p>
      `
    });
  }

  /**
   * 去除敏感信息，返回安全的用户对象
   */
  private sanitizeUser(user: User): Partial<User> {
    const { password, ...safeUser } = user;
    return safeUser;
  }
}
```

```typescript
import { Request, Response, NextFunction } from 'express';
import { verifyToken } from '../utils/jwt';
import { redis } from '../config/redis';
import { AppDataSource } from '../config/database';
import { User, UserStatus } from '../models/User';

// 扩展Express Request类型
declare global {
  namespace Express {
    interface Request {
      user?: {
        userId: string;
        role: string;
      };
    }
  }
}

/**
 * 认证中间件
 * 验证请求中的JWT Token
 *
 * 工作流程：
 * 1. 从Authorization Header获取Token
 * 2. 验证Token签名和有效期
 * 3. 检查Token是否在黑名单
 * 4. 将用户信息附加到Request对象
 */
export async function authMiddleware(
  req: Request,
  res: Response,
  next: NextFunction
): Promise<void> {
  try {
    // 1. 获取Token
    const authHeader = req.headers.authorization;

    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      res.status(401).json({ error: '未提供认证令牌' });
      return;
    }

    const token = authHeader.substring(7);

    // 2. 验证Token
    const decoded = verifyToken(token) as { userId: string; role: string };

    // 3. 检查Token黑名单
    const isBlacklisted = await redis.get(`blacklist:access:${token}`);
    if (isBlacklisted) {
      res.status(401).json({ error: '令牌已失效，请重新登录' });
      return;
    }

    // 4. 可选：验证用户Token版本
    // 如果用户修改了密码，Token版本会改变
    const tokenVersion = await redis.get(`user:token:version:${decoded.userId}`);
    if (tokenVersion) {
      const tokenVersionFromPayload = (decoded as any).version;
      if (tokenVersionFromPayload && parseInt(tokenVersion) > tokenVersionFromPayload) {
        res.status(401).json({ error: '令牌已失效，请重新登录' });
        return;
      }
    }

    // 5. 验证用户状态
    const userRepository = AppDataSource.getRepository(User);
    const user = await userRepository.findOne({
      where: { id: decoded.userId }
    });

    if (!user) {
      res.status(401).json({ error: '用户不存在' });
      return;
    }

    if (user.status !== UserStatus.ACTIVE) {
      res.status(401).json({ error: '账号状态异常' });
      return;
    }

    // 6. 将用户信息附加到Request
    req.user = {
      userId: decoded.userId,
      role: decoded.role,
    };

    next();
  } catch (error) {
    if (error.name === 'TokenExpiredError') {
      res.status(401).json({ error: '令牌已过期' });
      return;
    }
    if (error.name === 'JsonWebTokenError') {
      res.status(401).json({ error: '无效的令牌' });
      return;
    }
    res.status(500).json({ error: '认证失败' });
  }
}

/**
 * 角色验证中间件工厂
 * @param allowedRoles 允许的角色数组
 * @returns 中间件函数
 *
 * 使用示例：
 * router.post('/admin', authMiddleware, roleMiddleware(['ADMIN']), handler);
 */
export function roleMiddleware(allowedRoles: string[]) {
  return (req: Request, res: Response, next: NextFunction): void => {
    if (!req.user) {
      res.status(401).json({ error: '未认证' });
      return;
    }

    if (!allowedRoles.includes(req.user.role)) {
      res.status(403).json({ error: '权限不足' });
      return;
    }

    next();
  };
}

/**
 * 可选认证中间件
 * 如果请求带有Token则验证，否则继续
 * 用于需要认证但不是强制的场景
 */
export async function optionalAuthMiddleware(
  req: Request,
  res: Response,
  next: NextFunction
): Promise<void> {
  try {
    const authHeader = req.headers.authorization;

    if (!authHeader || !authHeader.startsWith('Bearer ')) {
      next();
      return;
    }

    const token = authHeader.substring(7);
    const decoded = verifyToken(token) as { userId: string; role: string };

    const isBlacklisted = await redis.get(`blacklist:access:${token}`);
    if (isBlacklisted) {
      next();
      return;
    }

    req.user = {
      userId: decoded.userId,
      role: decoded.role,
    };

    next();
  } catch (error) {
    // Token无效或过期，继续执行
    next();
  }
}
```

```typescript
// backend/src/utils/jwt.ts - JWT工具

import jwt from 'jsonwebtoken';
import { config } from '../config/env';

/**
 * JWT配置
 */
const JWT_SECRET = config.jwt.secret || 'your-secret-key-change-in-production';
const JWT_ISSUER = 'fastshop';

/**
 * Token载荷类型定义
 */
interface AccessTokenPayload {
  userId: string;
  role: string;
}

interface RefreshTokenPayload {
  userId: string;
}

type TokenPayload = AccessTokenPayload | RefreshTokenPayload;

/**
 * 生成JWT Token
 * @param payload Token载荷
 * @param type Token类型，决定签名算法和过期时间
 * @param expiresIn 过期时间字符串
 * @returns 签名的Token字符串
 */
export function generateToken(
  payload: TokenPayload,
  type: 'access' | 'refresh',
  expiresIn?: string
): string {
  const options: jwt.SignOptions = {
    issuer: JWT_ISSUER,
    subject: type === 'access' ? 'access-token' : 'refresh-token',
  };

  // 根据类型设置不同的过期时间和算法
  if (type === 'access') {
    options.expiresIn = expiresIn || '15m';
    options.algorithm = 'HS256';
  } else {
    options.expiresIn = expiresIn || '7d';
    options.algorithm = 'HS512';  // RefreshToken使用更强算法
  }

  return jwt.sign(payload, JWT_SECRET, options);
}

/**
 * 验证JWT Token
 * @param token Token字符串
 * @param type Token类型
 * @returns 解码后的载荷
 */
export function verifyToken(
  token: string,
  type?: 'access' | 'refresh'
): TokenPayload {
  const options: jwt.VerifyOptions = {
    issuer: JWT_ISSUER,
  };

  if (type) {
    options.subject = type === 'access' ? 'access-token' : 'refresh-token';
    options.algorithms = type === 'access' ? ['HS256'] : ['HS512'];
  }

  return jwt.verify(token, JWT_SECRET, options) as TokenPayload;
}

/**
 * 解码Token（不验证签名）
 * 仅用于调试，生产环境慎用
 */
export function decodeToken(token: string): TokenPayload | null {
  return jwt.decode(token) as TokenPayload;
}

/**
 * 获取Token剩余有效期（秒）
 */
export function getTokenTTL(token: string): number {
  try {
    const decoded = jwt.decode(token) as { exp: number };
    if (!decoded || !decoded.exp) {
      return 0;
    }
    return Math.max(0, decoded.exp - Math.floor(Date.now() / 1000));
  } catch {
    return 0;
  }
}
```

```typescript
  /**
   * 获取分类及子分类ID列表
   * 递归查询所有子分类
   */
  private async getCategoryAndChildrenIds(categoryId: string): Promise<string[]> {
    const cacheKey = `category:children:${categoryId}`;
    const cached = await redis.get(cacheKey);

    if (cached) {
      return JSON.parse(cached);
    }

    // 使用递归CTE查询分类树
    const result = await AppDataSource.query(`
      WITH RECURSIVE category_tree AS (
        SELECT id FROM categories WHERE id = $1
        UNION ALL
        SELECT c.id FROM categories c
        INNER JOIN category_tree ct ON c.parent_id = ct.id
      )
      SELECT id FROM category_tree
    `, [categoryId]);

    const ids = result.map((row: any) => row.id);
    await redis.setex(cacheKey, 60 * 60, JSON.stringify(ids)); // 缓存1小时

    return ids;

  /**

    const key = `product:views:${productId}`;

    if (views % 10 === 0) {

  /**

   * 扣减库存时需要确保不会变成负数

   * @param skuId SKU ID
   * @param quantity 变更数量（正数为扣减，负数为退回）
   * @returns 更新后的库存
   */
  async updateStock(skuId: string, quantity: number): Promise<number> {
    // 使用事务确保一致性
    return await AppDataSource.transaction(async manager => {
      const skuRepository = manager.getRepository(Product).metadata;

      // 悲观锁查询当前库存
      const product = await manager
        .createQueryBuilder(Product, 'product')
        .setLock('pessimistic_write')
        .where('JSONB_EXTRACT_PATH(skus, :skuId, "stock") >= :quantity', {
          skuId,
          quantity: Math.abs(quantity)
        })
        .getOne();

      if (!product) {
        throw new Error('库存不足或商品不存在');
      }

      // 更新库存
      // 实际实现中需要解析JSONB中的SKU库存
      const currentStock = product.stock;
      const newStock = currentStock - quantity;

      if (newStock < 0) {
        throw new Error('库存不足');
      }

      await manager.update(Product, product.id, {
        stock: newStock
      });

      // 如果库存变为0，更新商品状态
      if (newStock === 0) {
        await manager.update(Product, product.id, {
          status: ProductStatus.SOLD_OUT
        });
      }

      return newStock;
    });
  }
}

// 导出服务实例
export const productService = new ProductService();
```

### 4.3 购物车模块

#### 4.3.1 购物车服务

购物车是电商系统的核心功能之一，本项目采用Redis存储购物车数据，利用Redis的数据结构优势实现高效的购物车操作。

```typescript
// backend/src/services/cartService.ts - 购物车服务

import { AppDataSource } from '../config/database';
import { CartItem } from '../models/CartItem';
import { Product } from '../models/Product';
import { redis } from '../config/redis';
import { ProductService } from './productService';

/**
 * 购物车项数据结构
 * 存储在Redis Hash中
 */
interface CartItemData {
  id: string;
  productId: string;
  skuId: string;
  quantity: number;
  price: number;
  selected: boolean;
  addedAt: string;
}

/**
 * 购物车统计数据
 */
export interface CartStatistics {
  totalQuantity: number;         // 总商品数量
  selectedQuantity: number;      // 选中商品数量
  totalAmount: number;           // 总金额
  selectedAmount: number;        // 选中商品金额
  discountAmount: number;        // 优惠金额
  shippingFee: number;           // 运费
}

/**
 * 购物车服务类
 * 使用Redis Hash存储购物车数据，支持快速读写和原子操作
 *
 * Redis数据结构设计：
 * - 购物车主Hash: cart:{userId} -> Hash { itemId: JSON(CartItemData) }
 * - 商品信息缓存: product:{productId} -> String (商品信息JSON)
 */
export class CartService {
  private readonly CART_PREFIX = 'cart:';
  private readonly CART_ITEM_TTL = 30 * 24 * 60 * 60; // 购物车项30天过期

  constructor(
    private productService: ProductService
  ) {}

  /**
   * 获取用户购物车
   *
   * @param userId 用户ID
   * @returns 购物车项列表（包含完整商品信息）
   */
  async getCart(userId: string): Promise<CartItemData[]> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    const cartData = await redis.hgetall(cartKey);

    if (!cartData || Object.keys(cartData).length === 0) {
      return [];
    }

    // 解析购物车项数据
    const items: CartItemData[] = [];
    for (const [itemId, itemJson] of Object.entries(cartData)) {
      const item = JSON.parse(itemJson) as CartItemData;
      // 检查商品是否仍然有效
      const isValid = await this.validateCartItem(item);
      if (isValid) {
        items.push(item);
      } else {
        // 商品已下架或不存在，删除购物车项
        await this.removeItem(userId, itemId);
      }
    }

    return items;
  }

  /**
   * 添加商品到购物车
   *
   * @param userId 用户ID
   * @param productId 商品ID
   * @param skuId SKU ID
   * @param quantity 数量
   * @returns 添加后的购物车项
   */
  async addToCart(
    userId: string,
    productId: string,
    skuId: string,
    quantity: number
  ): Promise<CartItemData> {
    // 1. 获取商品信息
    const product = await this.productService.getProductDetail(productId);
    if (!product) {
      throw new Error('商品不存在');
    }

    // 2. 查找对应的SKU
    const sku = product.skus.find(s => s.id === skuId);
    if (!sku) {
      throw new Error('商品规格不存在');
    }

    if (!sku.enabled) {
      throw new Error('该规格已下架');
    }

    if (sku.stock < quantity) {
      throw new Error('库存不足');
    }

    // 3. 检查购物车中是否已有该商品
    const cartKey = `${this.CART_PREFIX}${userId}`;
    const existingItemId = await redis.hget(cartKey, `${productId}:${skuId}`);

    let cartItem: CartItemData;

    if (existingItemId) {
      // 4a. 已存在，增加数量
      const existingItem = JSON.parse(existingItemId) as CartItemData;
      const newQuantity = existingItem.quantity + quantity;

      if (newQuantity > sku.stock) {
        throw new Error('库存不足');
      }

      cartItem = {
        ...existingItem,
        quantity: newQuantity,
      };
    } else {
      // 4b. 不存在，创建新项
      cartItem = {
        id: `${productId}:${skuId}`,
        productId,
        skuId,
        quantity,
        price: sku.price,
        selected: true,
        addedAt: new Date().toISOString(),
      };
    }

    // 5. 保存到Redis
    await redis.hset(cartKey, cartItem.id, JSON.stringify(cartItem));
    await redis.expire(cartKey, this.CART_ITEM_TTL);

    return cartItem;
  }

  /**
   * 更新购物车项数量
   *
   * @param userId 用户ID
   * @param itemId 购物车项ID（productId:skuId格式）
   * @param quantity 新数量
   */
  async updateQuantity(
    userId: string,
    itemId: string,
    quantity: number
  ): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    const itemJson = await redis.hget(cartKey, itemId);

    if (!itemJson) {
      throw new Error('购物车项不存在');
    }

    const item = JSON.parse(itemJson) as CartItemData;

    if (quantity <= 0) {
      // 数量为0或负数，删除该项
      await this.removeItem(userId, itemId);
      return;
    }

    // 验证库存
    const product = await this.productService.getProductDetail(item.productId);
    if (!product) {
      throw new Error('商品已下架');
    }

    const sku = product.skus.find(s => s.id === item.skuId);
    if (!sku || sku.stock < quantity) {
      throw new Error('库存不足');
    }

    // 更新数量
    item.quantity = quantity;
    await redis.hset(cartKey, itemId, JSON.stringify(item));
  }

  /**
   * 选择/取消选择购物车项
   *
   * @param userId 用户ID
   * @param itemId 购物车项ID
   * @param selected 是否选中
   */
  async selectItem(
    userId: string,
    itemId: string,
    selected: boolean
  ): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    const itemJson = await redis.hget(cartKey, itemId);

    if (!itemJson) {
      throw new Error('购物车项不存在');
    }

    const item = JSON.parse(itemJson) as CartItemData;
    item.selected = selected;

    await redis.hset(cartKey, itemId, JSON.stringify(item));
  }

  /**
   * 全选/取消全选
   *
   * @param userId 用户ID
   * @param selected 是否全部选中
   */
  async selectAll(userId: string, selected: boolean): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    const items = await this.getCart(userId);

    const pipeline = redis.pipeline();
    for (const item of items) {
      item.selected = selected;
      pipeline.hset(cartKey, item.id, JSON.stringify(item));
    }
    await pipeline.exec();
  }

  /**
   * 删除购物车项
   *
   * @param userId 用户ID
   * @param itemId 购物车项ID
   */
  async removeItem(userId: string, itemId: string): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    await redis.hdel(cartKey, itemId);
  }

  /**
   * 批量删除购物车项
   *
   * @param userId 用户ID
   * @param itemIds 购物车项ID列表
   */
  async removeItems(userId: string, itemIds: string[]): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    await redis.hdel(cartKey, ...itemIds);
  }

  /**
   * 清空购物车
   *
   * @param userId 用户ID
   */
  async clearCart(userId: string): Promise<void> {
    const cartKey = `${this.CART_PREFIX}${userId}`;
    await redis.del(cartKey);
  }

  /**
   * 获取购物车统计信息
   *
   * @param userId 用户ID
   * @returns 购物车统计
   */
  async getCartStatistics(userId: string): Promise<CartStatistics> {
    const items = await this.getCart(userId);

    let totalQuantity = 0;
    let selectedQuantity = 0;
    let totalAmount = 0;
    let selectedAmount = 0;

    for (const item of items) {
      const itemTotal = item.price * item.quantity;
      totalQuantity += item.quantity;
      totalAmount += itemTotal;

      if (item.selected) {
        selectedQuantity += item.quantity;
        selectedAmount += itemTotal;
      }
    }

    // 计算运费（满99元包邮）
    const shippingFee = selectedAmount >= 9900 ? 0 : 1000;

    return {
      totalQuantity,
      selectedQuantity,
      totalAmount,
      selectedAmount,
      discountAmount: 0,  // TODO: 后续接入优惠券
      shippingFee,
    };
  }

  /**
   * 获取选中的购物车项
   * 用于结算流程
   *
   * @param userId 用户ID
   * @returns 选中的购物车项
   */
  async getSelectedItems(userId: string): Promise<CartItemData[]> {
    const items = await this.getCart(userId);
    return items.filter(item => item.selected);
  }

  /**
   * 验证购物车项有效性
   */
  private async validateCartItem(item: CartItemData): Promise<boolean> {
    try {
      const product = await this.productService.getProductDetail(item.productId);
      if (!product) {
        return false;
      }

      if (product.status !== 'ON_SALE') {
        return false;
      }

      const sku = product.skus.find(s => s.id === item.skuId);
      if (!sku || !sku.enabled) {
        return false;
      }

      return true;
    } catch {
      return false;
    }
  }

  /**
   * 合并离线购物车到用户购物车
   * 用户未登录时将商品加入离线购物车，登录后合并
   *
   * @param userId 用户ID
   * @param guestCart 离线购物车数据
   */
  async mergeCart(userId: string, guestCart: CartItemData[]): Promise<void> {
    for (const item of guestCart) {
      try {
        await this.addToCart(userId, item.productId, item.skuId, item.quantity);
        // 如果原购物车项是选中状态，需要保持
        if (item.selected) {
          const cartKey = `${this.CART_PREFIX}${userId}`;
          const itemJson = await redis.hget(cartKey, item.id);
          if (itemJson) {
            const existingItem = JSON.parse(itemJson) as CartItemData;
            existingItem.selected = true;
            await redis.hset(cartKey, item.id, JSON.stringify(existingItem));
          }
        }
      } catch (error) {
        // 跳过无效的商品
        console.error(`合并购物车项失败: ${item.productId}`, error);
      }
    }
  }
}

// 创建并导出服务实例
export const cartService = new CartService(new ProductService());
```

### 4.4 订单模块

#### 4.4.1 订单服务

订单是电商系统最核心的模块，涉及金额计算、库存扣减、状态流转等多个复杂逻辑。

```typescript
// backend/src/services/orderService.ts - 订单服务

import { AppDataSource } from '../config/database';
import { Order, OrderStatus } from '../models/Order';
import { OrderItem } from '../models/OrderItem';
import { Product } from '../models/Product';
import { CartItem } from '../models/CartItem';
import { redis } from '../config/redis';
import { cartService } from './cartService';
import { productService } from './productService';
import { paymentService } from './paymentService';
import { v4 as uuidv4 } from 'uuid';

/**
 * 创建订单请求参数
 */
export interface CreateOrderParams {
  userId: string;
  items: {
    productId: string;
    skuId: string;
    quantity: number;
  }[];
  shippingAddress: {
    receiverName: string;
    phone: string;
    province: string;
    city: string;
    district: string;
    detailAddress: string;
    postalCode?: string;
  };
  buyerMessage?: string;
  couponId?: string;
  paymentMethod: 'WECHAT_PAY' | 'ALIPAY' | 'BANK_CARD' | 'BALANCE';
}

/**
 * 订单查询参数
 */
export interface OrderQueryParams {
  userId: string;
  status?: OrderStatus;
  page?: number;
  pageSize?: number;
}

/**
 * 订单服务类
 * 处理订单创建、查询、状态变更等核心业务逻辑
 */
export class OrderService {
  private orderRepository = AppDataSource.getRepository(Order);
  private orderItemRepository = AppDataSource.getRepository(OrderItem);

  // 分布式锁前缀
  private readonly LOCK_PREFIX = 'lock:order:';
  // 订单号前缀
  private readonly ORDER_NO_PREFIX = 'FS';
  // 订单自动取消时间（分钟）
  private readonly AUTO_CANCEL_MINUTES = 30;

  /**
   * 创建订单
   * 核心业务流程：
   * 1. 验证商品和库存
   * 2. 计算订单金额
   * 3. 扣减库存
   * 4. 生成订单
   * 5. 记录订单历史
   *
   * @param params 创建订单参数
   * @returns 创建的订单
   */
  async createOrder(params: CreateOrderParams): Promise<Order> {
    const {
      userId,
      items,
      shippingAddress,
      buyerMessage,
      paymentMethod,
    } = params;

    // 1. 验证商品和SKU，获取商品信息
    const orderItems: OrderItem[] = [];
    let totalAmount = 0;
    let totalDiscount = 0;

    // 按商品ID聚合，避免同一商品多次查询
    const productMap = new Map<string, { product: Product; quantity: number }>();

    for (const item of items) {
      const product = await productService.getProductDetail(item.productId);
      if (!product) {
        throw new Error(`商品不存在: ${item.productId}`);
      }

      if (product.status !== 'ON_SALE') {
        throw new Error(`商品已下架: ${product.name}`);
      }

      const sku = product.skus.find(s => s.id === item.skuId);
      if (!sku) {
        throw new Error(`商品规格不存在: ${item.skuId}`);
      }

      if (!sku.enabled) {
        throw new Error(`商品规格已下架: ${sku.skuCode}`);
      }

      if (sku.stock < item.quantity) {
        throw new Error(`商品库存不足: ${product.name} (${sku.skuCode})`);
      }

      // 计算小计金额
      const itemTotal = sku.price * item.quantity;
      totalAmount += itemTotal;

      // 创建订单项快照
      const orderItem = new OrderItem();
      orderItem.productId = product.id;
      orderItem.productName = product.name;
      orderItem.productImage = product.images[0] || '';
      orderItem.skuId = sku.id;
      orderItem.skuCode = sku.skuCode;
      orderItem.specs = sku.specs;
      orderItem.price = sku.price;
      orderItem.quantity = item.quantity;
      orderItem.discountAmount = 0;
      orderItem.totalAmount = itemTotal;

      orderItems.push(orderItem);

      // 记录商品和数量用于后续处理
      productMap.set(item.productId, { product, quantity: item.quantity });
    }

    // 2. 计算运费
    const shippingFee = totalAmount >= 9900 ? 0 : 1000;

    // 3. 计算应付金额
    const payableAmount = totalAmount - totalDiscount + shippingFee;

    // 4. 生成订单号（唯一性保证）
    const orderNo = this.generateOrderNo();

    // 5. 创建订单（使用事务保证一致性）
    const order = await AppDataSource.transaction(async manager => {
      // 5.1 扣减库存（使用悲观锁）
      for (const [productId, { product, quantity }] of productMap.entries()) {
        const sku = product.skus.find(s =>
          items.find(i => i.productId === productId && i.skuId === s.id)
        )!;

        // 扣减库存
        await productService.updateStock(sku.id, -quantity);

        // 增加已售数量
        await manager.increment(Product, { id: productId }, 'soldCount', quantity);
      }

      // 5.2 创建订单主记录
      const newOrder = manager.create(Order, {
        orderNo,
        userId,
        status: OrderStatus.PENDING_PAYMENT,
        totalAmount,
        discountAmount: totalDiscount,
        shippingFee,
        payableAmount,
        shippingAddress,
        buyerMessage,
        paymentMethod,
        statusHistory: [{
          status: OrderStatus.PENDING_PAYMENT,
          remark: '订单已创建，等待付款',
          operator: userId,
          createdAt: new Date(),
        }],
      });

      await manager.save(newOrder);

      // 5.3 创建订单商品记录
      for (const item of orderItems) {
        item.orderId = newOrder.id;
        await manager.save(item);
      }

      // 5.4 清理购物车（移除已购买的商品）
      const itemIds = items.map(i => `${i.productId}:${i.skuId}`);
      await cartService.removeItems(userId, itemIds);

      return newOrder;
    });

    // 6. 设置订单自动取消（延时任务）
    await this.scheduleOrderCancel(order.id, this.AUTO_CANCEL_MINUTES);

    // 7. 返回完整订单信息
    return this.getOrderDetail(order.id) as Promise<Order>;
  }

  /**
   * 获取订单详情
   *
   * @param orderId 订单ID
   * @returns 订单详情（包含订单项）
   */
  async getOrderDetail(orderId: string): Promise<Order | null> {
    const order = await this.orderRepository.findOne({
      where: { id: orderId },
      relations: ['items'],
    });

    return order;
  }

  /**
   * 获取订单详情（通过订单号）
   */
  async getOrderByOrderNo(orderNo: string): Promise<Order | null> {
    const order = await this.orderRepository.findOne({
      where: { orderNo },
      relations: ['items'],
    });

    return order;
  }

  /**
   * 查询用户订单列表
   *
   * @param params 查询参数
   * @returns 分页的订单列表
   */
  async getUserOrders(params: OrderQueryParams): Promise<{
    data: Order[];
    total: number;
    page: number;
    pageSize: number;
    totalPages: number;
  }> {
    const { userId, status, page = 1, pageSize = 10 } = params;

    const queryBuilder = this.orderRepository
      .createQueryBuilder('order')
      .leftJoinAndSelect('order.items', 'items')
      .where('order.userId = :userId', { userId })
      .orderBy('order.createdAt', 'DESC');

    if (status) {
      queryBuilder.andWhere('order.status = :status', { status });
    }

    const [data, total] = await queryBuilder
      .skip((page - 1) * pageSize)
      .take(pageSize)
      .getManyAndCount();

    return {
      data,
      total,
      page,
      pageSize,
      totalPages: Math.ceil(total / pageSize),
    };
  }

  /**
   * 取消订单
   *
   * @param orderId 订单ID
   * @param userId 用户ID（用于权限验证）
   * @param reason 取消原因
   */
  async cancelOrder(orderId: string, userId: string, reason?: string): Promise<void> {
    // 1. 获取订单（使用悲观锁）
    const order = await this.orderRepository.findOne({
      where: { id: orderId },
      relations: ['items'],
    });

    if (!order) {
      throw new Error('订单不存在');
    }

    // 2. 验证权限
    if (order.userId !== userId) {
      throw new Error('无权取消此订单');
    }

    // 3. 验证状态
    if (order.status !== OrderStatus.PENDING_PAYMENT) {
      throw new Error('只有待付款的订单可以取消');
    }

    // 4. 取消订单（事务）
    await AppDataSource.transaction(async manager => {
      // 4.1 恢复库存
      for (const item of order.items) {
        await productService.updateStock(item.skuId, item.quantity);
        // 减少已售数量
        await manager.decrement(Product, { id: item.productId }, 'soldCount', item.quantity);
      }

      // 4.2 更新订单状态
      const newStatusHistory = [...order.statusHistory, {
        status: OrderStatus.CANCELLED,
        remark: reason || '用户取消订单',
        operator: userId,
        createdAt: new Date(),
      }];

      await manager.update(Order, orderId, {
        status: OrderStatus.CANCELLED,
        statusHistory: newStatusHistory,
        cancelledAt: new Date(),
      });
    });

    // 5. 取消延时任务
    await this.cancelScheduledOrderCancel(orderId);
  }

  /**
   * 支付订单
   * 更新订单状态为已支付
   *
   * @param orderId 订单ID
   * @param paymentNo 支付流水号
   */
  async payOrder(orderId: string, paymentNo: string): Promise<void> {
    const order = await this.orderRepository.findOne({
      where: { id: orderId }
    });

    if (!order) {
      throw new Error('订单不存在');
    }

    if (order.status !== OrderStatus.PENDING_PAYMENT) {
      throw new Error('订单状态不允许支付');
    }

    // 更新订单状态
    await this.orderRepository.update(orderId, {
      status: OrderStatus.PAID,
      paymentNo,
      paidAt: new Date(),
      statusHistory: [...order.statusHistory, {
        status: OrderStatus.PAID,
        remark: '支付成功',
        operator: 'system',
        createdAt: new Date(),
      }],
    });

    // 取消延时取消任务
    await this.cancelScheduledOrderCancel(orderId);

    // TODO: 触发后续流程（通知商家、发送邮件等）
  }

  /**

   */
  async shipOrder(

   */
  async completeOrder(orderId: string, userId: string): Promise<void> {
    const order = await this.orderRepository.findOne({
      where: { id: orderId, userId }
    });

    if (!order) {
      throw new Error('订单不存在');
    }

    if (order.status !== OrderStatus.DELIVERED) {
      throw new Error('订单状态不允许完成');
    }

    await this.orderRepository.update(orderId, {
      status: OrderStatus.COMPLETED,
      completedAt: new Date(),
      statusHistory: [...order.statusHistory, {

        remark: '订单已完成',

        createdAt: new Date(),
      }],

  }

  /**
   * 申请退款
   */
  async applyRefund(orderId: string, userId: string, reason: string): Promise<void> {
    const order = await this.orderRepository.findOne({
      where: { id: orderId, userId }
    });

    if (!order) {

    const refundableStatuses = [

      OrderStatus.PROCESSING,
      OrderStatus.SHIPPED,
      OrderStatus.DELIVERED,

    if (!refundableStatuses.includes(order.status)) {

    }

  private generateOrderNo(): string {
    const timestamp = new Date().toISOString().replace(/[-:T]/g, '').substring(0, 14);
    const random = Math.random().toString(36).substring(2, 8).toUpperCase();
    return `${this.ORDER_NO_PREFIX}${timestamp}${random}`;

  private async cancelScheduledOrderCancel(orderId: string): Promise<void> {

  }
```

### 4.5 支付模块

#### 4.5.1 支付服务

支付是电商系统最复杂的模块之一，需要与第三方支付渠道对接，处理支付回调，保证交易安全。

```typescript
// backend/src/services/paymentService.ts - 支付服务

import { AppDataSource } from '../config/database';
import { Payment, PaymentStatus, PaymentMethod } from '../models/Payment';
import { Order, OrderStatus } from '../models/Order';
import { redis } from '../config/redis';
import { orderService } from './orderService';
import { v4 as uuidv4 } from 'uuid';
import crypto from 'crypto';

/**
 * 支付渠道配置
 */
interface PaymentChannelConfig {
  appId: string;
  appSecret: string;
  merchantId: string;
  notifyUrl: string;
}

/**
 * 创建支付订单参数
 */
export interface CreatePaymentParams {
  orderId: string;
  userId: string;
  method: PaymentMethod;
}

/**
 * 支付回调参数
 */
interface PaymentCallbackParams {
  orderNo: string;
  transactionId: string;
  status: 'SUCCESS' | 'FAILED';
  paidAmount: number;
  paidTime: string;
  rawData: Record<string, string>;
}

/**
 * 支付服务类
 * 处理支付订单创建、回调处理、退款等核心流程
 *
 * 安全设计：
 * 1. 支付签名验证 - 确保回调来自真实的支付渠道
 * 2. 幂等性处理 - 防止重复支付
 * 3. 异步处理 - 支付回调异步处理，不阻塞主流程
 */
export class PaymentService {
  private paymentRepository = AppDataSource.getRepository(Payment);
  private orderRepository = AppDataSource.getRepository(Order);

  // 支付渠道配置
  private channelConfigs: Record<PaymentMethod, PaymentChannelConfig> = {
    WECHAT_PAY: {
      appId: process.env.WECHAT_APP_ID!,
      appSecret: process.env.WECHAT_APP_SECRET!,
      merchantId: process.env.WECHAT_MERCHANT_ID!,
      notifyUrl: process.env.WECHAT_NOTIFY_URL!,
    },
    ALIPAY: {
      appId: process.env.ALIPAY_APP_ID!,
      appSecret: process.env.ALIPAY_APP_SECRET!,
      merchantId: process.env.ALIPAY_MERCHANT_ID!,
      notifyUrl: process.env.ALIPAY_NOTIFY_URL!,
    },
    BANK_CARD: {
      appId: '',
      appSecret: '',
      merchantId: '',
      notifyUrl: '',
    },
    BALANCE: {
      appId: '',
      appSecret: '',
      merchantId: '',
      notifyUrl: '',
    },
  };

  // 分布式锁前缀
  private readonly LOCK_PREFIX = 'lock:payment:';

  /**
   * 创建支付订单
   * 根据支付方式不同，返回不同的支付参数（跳转链接/二维码等）
   *
   * @param params 创建支付参数
   * @returns 支付信息（包含跳转链接或二维码）
   */
  async createPayment(params: CreatePaymentParams): Promise<{
    paymentId: string;
    paymentNo: string;
    amount: number;
    payUrl?: string;              // 跳转支付链接
    qrCode?: string;              // 支付二维码（Base64）
    expireTime: Date;             // 过期时间
  }> {
    const { orderId, userId, method } = params;

    // 1. 获取订单信息
    const order = await this.orderRepository.findOne({
      where: { id: orderId, userId }
    });

    if (!order) {
      throw new Error('订单不存在');
    }

    if (order.status !== OrderStatus.PENDING_PAYMENT) {
      throw new Error('订单状态不允许支付');
    }

    if (order.payableAmount <= 0) {
      throw new Error('订单金额无效');
    }

    // 2. 检查是否已有待支付的支付单
    const existingPayment = await this.paymentRepository.findOne({
      where: {
        orderId,
        status: PaymentStatus.PENDING,
      }
    });

    if (existingPayment) {
      // 存在待支付单，检查是否过期
      const expireTime = new Date(existingPayment.createdAt.getTime() + 30 * 60 * 1000);
      if (expireTime > new Date()) {
        // 未过期，返回已有支付单
        return this.buildPaymentResponse(existingPayment, method);
      }
      // 已过期，创建新的支付单
    }

    // 3. 生成支付单号
    const paymentNo = this.generatePaymentNo();

    // 4. 创建支付记录
    const payment = this.paymentRepository.create({
      paymentNo,
      orderId,
      userId,
      method,
      amount: order.payableAmount,
      status: PaymentStatus.PENDING,
    });
    await this.paymentRepository.save(payment);

    // 5. 根据支付方式调用对应渠道
    try {
      const paymentResult = await this.callPaymentChannel(payment, method);

      // 更新支付记录（存储渠道返回的信息）
      payment.channelTradeNo = paymentResult.channelTradeNo;
      payment.rawResponse = paymentResult.rawResponse;
      await this.paymentRepository.save(payment);

      return {
        paymentId: payment.id,
        paymentNo: payment.paymentNo,
        amount: payment.amount,
        payUrl: paymentResult.payUrl,
        qrCode: paymentResult.qrCode,
        expireTime: new Date(payment.createdAt.getTime() + 30 * 60 * 1000),
      };
    } catch (error) {
      // 支付渠道调用失败
      payment.status = PaymentStatus.FAILED;
      payment.rawResponse = { error: (error as Error).message };
      await this.paymentRepository.save(payment);
      throw error;
    }
  }

  /**
   * 处理支付回调
   * 支付渠道服务器回调，通知支付结果
   *
   * @param params 回调参数
   * @returns 处理结果
   */
  async handlePaymentCallback(params: PaymentCallbackParams): Promise<{
    success: boolean;
    message: string;
  }> {
    const { orderNo, transactionId, status, paidAmount, rawData } = params;

    // 1. 获取支付记录
    const payment = await this.paymentRepository.findOne({
      where: { paymentNo: orderNo },
      relations: ['order'],
    });

    if (!payment) {
      return { success: false, message: '支付单不存在' };
    }

    // 2. 验证签名（根据渠道不同）
    const signatureValid = await this.verifyCallbackSignature(payment, rawData);
    if (!signatureValid) {
      return { success: false, message: '签名验证失败' };
    }

    // 3. 幂等性检查
    if (payment.status === PaymentStatus.SUCCESS) {
      return { success: true, message: '已处理' };
    }

    // 4. 验证支付金额
    if (status === 'SUCCESS' && paidAmount !== payment.amount) {
      // 金额不匹配，记录异常但仍处理
      console.error(`支付金额不匹配: expected=${payment.amount}, actual=${paidAmount}`);
    }

    // 5. 更新支付状态
    if (status === 'SUCCESS') {
      await this.paymentRepository.update(payment.id, {
        status: PaymentStatus.SUCCESS,
        channelTradeNo: transactionId,
        paidAt: new Date(),
        rawResponse: rawData,
      });

      // 6. 更新订单状态
      await orderService.payOrder(payment.orderId, transactionId);
    } else {
      await this.paymentRepository.update(payment.id, {
        status: PaymentStatus.FAILED,
        rawResponse: rawData,
      });
    }

    return { success: true, message: '处理成功' };
  }

  /**
   * 查询支付状态
   *
   * @param paymentId 支付单ID

  async getPaymentStatus(paymentId: string): Promise<{

   * 申请退款
   *
   * @param orderId 订单ID
   * @param userId 用户ID
   * @param reason 退款原因
   * @param refundAmount 退款金额（可选，默认全额退款）

  async applyRefund(

  }> {

    }

    // 计算退款金额

    if (actualRefundAmount > order.payableAmount) {
      throw new Error('退款金额超过支付金额');
    }

    // 生成退款单号

  }
```

```typescript
      selectedAmount += item.price * item.quantity;
    }
  }

  // 运费规则：满99元包邮
  const shippingFee = selectedAmount >= 9900 ? 0 : 1000;

  return {
    totalQuantity,
    selectedQuantity,
    totalAmount,
    selectedAmount,
    discountAmount: 0,
    shippingFee,
    payableAmount: selectedAmount + shippingFee,
  };
}

 * 购物车Store
 * 使用Zustand状态管理库，支持持久化存储

export const useCartStore = create<CartState>()(

      items: [],
      loading: false,
      error: null,
      statistics: calculateStatistics([]),

      /**
       * 获取购物车数据
       * 从服务器拉取最新购物车数据
       */
      fetchCart: async () => {
        set({ loading: true, error: null });
        try {
          const response = await fetch('/api/cart', {
            headers: {
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
          });

          if (!response.ok) {
            throw new Error('获取购物车失败');
          }

          const data = await response.json();
          const items = data.items as CartItem[];

          set({
            items,
            loading: false,
            statistics: calculateStatistics(items),
          });
        } catch (error) {
          set({
            loading: false,
            error: (error as Error).message,
          });
        }
      },

      /**
       * 添加商品到购物车
       */
      addItem: async (productId, skuId, quantity) => {
        set({ loading: true, error: null });
        try {
          const response = await fetch('/api/cart', {
            method: 'POST',
            headers: {
              'Content-Type': 'application/json',
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
            body: JSON.stringify({ productId, skuId, quantity }),
          });

          if (!response.ok) {
            const error = await response.json();
            throw new Error(error.message || '添加购物车失败');
          }

          const data = await response.json();

          set(state => {
            // 检查是否已存在
            const existingIndex = state.items.findIndex(
              item => item.productId === productId && item.skuId === skuId
            );

            if (existingIndex >= 0) {
              // 更新数量
              state.items[existingIndex].quantity += quantity;
            } else {
              // 添加新项
              state.items.push(data.item);
            }

            state.loading = false;
            state.statistics = calculateStatistics(state.items);
          });
        } catch (error) {
          set({
            loading: false,
            error: (error as Error).message,
          });
          throw error;
        }
      },

      /**
       * 更新购物车项数量
       */
      updateQuantity: async (itemId, quantity) => {
        // 乐观更新：立即更新UI
        set(state => {
          const item = state.items.find(i => i.id === itemId);
          if (item) {
            item.quantity = quantity;
          }
          state.statistics = calculateStatistics(state.items);
        });

        try {
          const response = await fetch(`/api/cart/${itemId}`, {
            method: 'PATCH',
            headers: {
              'Content-Type': 'application/json',
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
            body: JSON.stringify({ quantity }),
          });

          if (!response.ok) {
            throw new Error('更新数量失败');
          }

          // 重新计算统计
          set(state => {
            state.statistics = calculateStatistics(state.items);
          });
        } catch (error) {
          // 失败回滚
          await get().fetchCart();
          set({ error: (error as Error).message });
        }
      },

      /**
       * 删除购物车项
       */
      removeItem: async (itemId) => {
        // 乐观更新
        set(state => {
          state.items = state.items.filter(i => i.id !== itemId);
          state.statistics = calculateStatistics(state.items);
        });

        try {
          const response = await fetch(`/api/cart/${itemId}`, {
            method: 'DELETE',
            headers: {
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
          });

          if (!response.ok) {
            throw new Error('删除失败');
          }
        } catch (error) {
          // 失败回滚
          await get().fetchCart();
          set({ error: (error as Error).message });
        }
      },

      /**
       * 选择/取消选择购物车项
       */
      selectItem: async (itemId, selected) => {
        // 乐观更新
        set(state => {
          const item = state.items.find(i => i.id === itemId);
          if (item) {
            item.selected = selected;
          }
          state.statistics = calculateStatistics(state.items);
        });

        try {
          await fetch(`/api/cart/${itemId}/select`, {
            method: 'PATCH',
            headers: {
              'Content-Type': 'application/json',
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
            body: JSON.stringify({ selected }),
          });
        } catch (error) {
          await get().fetchCart();
          set({ error: (error as Error).message });
        }
      },

      /**
       * 全选/取消全选
       */
      selectAll: async (selected) => {
        set(state => {
          state.items.forEach(item => {
            item.selected = selected;
          });
          state.statistics = calculateStatistics(state.items);
        });

        try {
          await fetch('/api/cart/select-all', {
            method: 'PATCH',
            headers: {
              'Content-Type': 'application/json',
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
            body: JSON.stringify({ selected }),
          });
        } catch (error) {
          await get().fetchCart();
          set({ error: (error as Error).message });
        }
      },

      /**
       * 清空购物车
       */
      clearCart: async () => {
        set({ items: [], statistics: calculateStatistics([]) });

        try {
          await fetch('/api/cart', {
            method: 'DELETE',
            headers: {
              'Authorization': `Bearer ${localStorage.getItem('accessToken')}`,
            },
          });
        } catch (error) {
          await get().fetchCart();
          set({ error: (error as Error).message });
        }
      },

      /**
       * 获取选中的购物车项
       */
      getSelectedItems: () => {
        return get().items.filter(item => item.selected);
      },
    })),
    {
      name: 'fastshop-cart-storage',       // localStorage key
      storage: createJSONStorage(() => localStorage),
      partialize: (state) => ({ items: state.items }), // 只持久化items
    }
  )
);
```

```typescript
// frontend/src/lib/api.ts - API请求封装

/**
 * API响应基础结构
 */
interface ApiResponse<T> {
  success: boolean;
  data?: T;
  error?: {
    code: string;
    message: string;
  };
  meta?: {
    page?: number;
    pageSize?: number;
    total?: number;
  };
}

/**
 * API请求工具类
 */
class ApiClient {
  private baseURL: string;
  private accessToken: string | null = null;

  constructor(baseURL: string) {
    this.baseURL = baseURL;
  }

  /**
   * 设置访问令牌
   */
  setAccessToken(token: string | null) {
    this.accessToken = token;
  }

  /**
   * 获取请求头
   */
  private getHeaders(): HeadersInit {
    const headers: HeadersInit = {
      'Content-Type': 'application/json',
    };

    if (this.accessToken) {
      headers['Authorization'] = `Bearer ${this.accessToken}`;
    }

    return headers;
  }

  /**
   * 处理响应
   */
  private async handleResponse<T>(response: Response): Promise<T> {
    if (!response.ok) {
      // HTTP错误状态码
      if (response.status === 401) {
        // Token过期，尝试刷新
        const refreshed = await this.refreshToken();
        if (refreshed) {
          // 重新发起请求
          // 实际实现中需要递归或循环
        }
        throw new Error('请重新登录');
      }

      if (response.status === 403) {
        throw new Error('权限不足');
      }

      if (response.status === 404) {
        throw new Error('资源不存在');
      }

      if (response.status >= 500) {
        throw new Error('服务器错误，请稍后重试');
      }

      // 业务错误
      const error = await response.json();
      throw new Error(error.message || '请求失败');
    }

    const result: ApiResponse<T> = await response.json();

    if (!result.success) {
      throw new Error(result.error?.message || '请求失败');
    }

    return result.data as T;
  }

  /**
   * 刷新Token
   */
  private async refreshToken(): Promise<boolean> {
    try {
      const refreshToken = localStorage.getItem('refreshToken');
      if (!refreshToken) {
        return false;
      }

      const response = await fetch(`${this.baseURL}/auth/refresh`, {
        method: 'POST',
        headers: this.getHeaders(),
        body: JSON.stringify({ refreshToken }),
      });

      if (!response.ok) {
        return false;
      }

      const result = await response.json();
      if (result.data) {
        this.setAccessToken(result.data.accessToken);
        localStorage.setItem('refreshToken', result.data.refreshToken);
        return true;
      }

      return false;
    } catch {
      return false;
    }
  }

  /**
   * GET请求
   */
  async get<T>(path: string, params?: Record<string, any>): Promise<T> {
    const url = new URL(`${this.baseURL}${path}`, window.location.origin);
    if (params) {
      Object.entries(params).forEach(([key, value]) => {
        if (value !== undefined && value !== null) {
          url.searchParams.append(key, String(value));
        }
      });
    }

    const response = await fetch(url.toString(), {
      method: 'GET',
      headers: this.getHeaders(),
    });

    return this.handleResponse<T>(response);
  }

  /**
   * POST请求
   */
  async post<T>(path: string, data?: any): Promise<T> {
    const response = await fetch(`${this.baseURL}${path}`, {
      method: 'POST',
      headers: this.getHeaders(),
      body: data ? JSON.stringify(data) : undefined,
    });

    return this.handleResponse<T>(response);
  }

  /**
   * PATCH请求
   */
  async patch<T>(path: string, data?: any): Promise<T> {
    const response = await fetch(`${this.baseURL}${path}`, {
      method: 'PATCH',
      headers: this.getHeaders(),
      body: data ? JSON.stringify(data) : undefined,
    });

    return this.handleResponse<T>(response);
  }

  /**
   * DELETE请求
   */
  async delete<T>(path: string): Promise<T> {
    const response = await fetch(`${this.baseURL}${path}`, {
      method: 'DELETE',
      headers: this.getHeaders(),
    });

    return this.handleResponse<T>(response);
  }
}

// 创建API客户端实例
export const apiClient = new ApiClient('/api/v1');

// 初始化Token
if (typeof window !== 'undefined') {
  apiClient.setAccessToken(localStorage.getItem('accessToken'));
}
```

## 八、技术选型总结

### 8.1 技术栈优势分析

**为什么选择Next.js作为前端框架？**

Next.js的App Router模式为电商场景提供了优化的开发体验。服务端组件允许我们在服务端直接获取数据，这对于商品详情页这类需要SEO的页面非常有利。相比传统的客户端渲染，服务端渲染能够生成完整的HTML，让搜索引擎更好地索引页面内容。客户端组件只需要在真正需要交互的地方使用，如购物车、收藏等状态管理。这种"渐进式"使用React的策略让我们既能享受服务端渲染的优势，又能保留客户端交互的灵活性。

Turbopack的引入是另一个重要改进。电商网站的开发过程中，频繁的热更新对开发体验影响很大。Turbopack基于Rust实现，比Webpack快10倍以上，能够显著缩短开发时的等待时间。

**为什么选择Express而不是NestJS？**

Express的中间件模式更加直观和灵活。电商系统的API往往需要针对不同业务场景进行定制，Express的洋葱模型让请求处理流程清晰可见。对于学习者而言，理解Express的中间件机制有助于建立对Web服务本质的认知。NestJS虽然提供了更完善的企业级特性，但它的依赖注入、装饰器等概念会增加学习曲线。

**为什么选择PostgreSQL而不是MySQL？**

PostgreSQL对JSON类型的原生支持让商品属性、订单扩展信息等数据的存储更加灵活。电商系统中经常需要存储规格参数、用户画像等半结构化数据，JSONB类型既保留了SQL的查询能力，又具备NoSQL的灵活性。PostgreSQL的全文检索功能可以直接支持商品搜索，无需引入额外的Elasticsearch。窗口函数、CTE等高级SQL特性让复杂的数据分析查询更加简洁。

**为什么选择Redis作为缓存层？**

Redis的多种数据结构为不同场景提供了最佳实践。购物车使用Hash结构，按用户ID作为key，购物车项ID作为field，商品信息作为value，可以快速地进行单项更新和删除。商品详情使用String类型缓存，配合合理的过期时间实现自动刷新。分布式锁使用Redisson实现，支持自动续期和公平锁。Sorted Set实现延时任务队列，用于订单自动取消等场景。

### 8.2 性能优化策略

**前端优化**

Next.js的图片优化组件自动对图片进行格式转换和尺寸调整，在不同设备上提供最优的图片资源。动态导入（dynamic import）让非首屏组件实现代码分割，减少首屏加载时间。服务端组件的默认使用减少了客户端需要下载和执行的JavaScript代码量。

**后端优化**

数据库查询使用N+1问题的避免策略，通过join或批量查询减少数据库往返次数。热点数据缓存到Redis，显著降低数据库压力。连接池的使用避免了频繁创建和销毁数据库连接的开销。异步处理将非核心流程（如发送通知、更新统计等）放到消息队列异步执行。

**架构优化**

CDN加速静态资源和图片访问。负载均衡分发请求到多个服务实例。反向代理处理SSL终端和请求分发。数据库读写分离将读请求分散到从库。

### 8.3 安全措施

**数据安全**

用户密码使用bcrypt单向加密存储，即使数据库泄露也无法还原原始密码。敏感操作需要二次验证，如修改密码、绑定手机号等。支付接口使用数字签名验证请求合法性。

**传输安全**

全站使用HTTPS加密传输。关键接口添加请求限流，防止DDoS攻击。使用CSRF Token防止跨站请求伪造。

**业务安全**

订单金额在服务端计算，防止客户端篡改。库存使用悲观锁防止超卖。支付回调验证签名和金额，防止伪造。

## 九、总结

本指南详细介绍了使用Next.js、Express、PostgreSQL和Redis构建完整电商系统的技术方案。从需求分析到架构设计，从数据模型到核心功能实现，涵盖了电商系统开发的主要技术点。

项目采用前后端分离架构，前端利用Next.js的服务端渲染能力提升SEO效果和首屏性能，后端使用Express保持简洁直观的开发体验。PostgreSQL强大的查询能力和JSON支持为复杂业务提供了灵活的数据存储方案，Redis在缓存、会话管理和实时场景中发挥关键作用。

电商系统的核心挑战在于高并发处理、数据一致性保障和交易安全。通过合理的架构设计、数据库优化和缓存策略，系统能够支撑每日数万单的并发访问。分布式事务、幂等性处理、签名验证等机制确保了交易的安全性。

希望本指南能够帮助开发者掌握电商系统的核心技术，为实际项目开发提供参考。
