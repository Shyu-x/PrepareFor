# 实际项目中的网络层封装

## 前言：什么是网络层封装？

想象一下，你在一栋大型写字楼里工作。每天都要和各种部门打交道：
- 要打印文件？找文印室
- 要开会？找会议室管理员
- 要请假？找人事部
- 要修电脑？找IT部门

**如果没有这些前台和秘书，你得记住每个部门在几楼、找谁、怎么说，这得多麻烦！**

网络层封装就是这个"前台秘书"：
- **统一错误处理**：文印室告诉你"打印机坏了"，不用你操心
- **Token自动刷新**：秘书自动帮你续签通行证，不用你追着问
- **请求缓存**：常用文件直接复印好，不用每次都去原件
- **接口看板管理**：像看板一样清晰看到所有接口状态

本章我们将构建一个生产级别的网络层封装系统。

## 第三部分：Token管理器

### 3.1 Token管理器的职责

```
Token管理器就像"门禁卡管理员"：
1. 员工刷卡进门（带Token请求）
2. Token快过期了，自动去前台续签
3. 续签成功，继续正常工作
4. 续签失败（比如辞职了），请员工重新登录
```

### 3.2 完整实现

```typescript
/**
 * Token管理器
 * 负责Token的存储、刷新、自动续期
 */

import { TokenInfo } from './types';

// Token存储的key
const TOKEN_KEY = 'auth_tokens';
// Token提前刷新时间（秒），快过期前10分钟就刷新
const REFRESH_BEFORE_SECONDS = 600;

class TokenManager {
  private tokenInfo: TokenInfo | null = null;
  private refreshing: Promise<TokenInfo> | null = null;
  private listeners: Set<(token: string) => void> = new Set();

  constructor() {
    // 从本地存储恢复token
    this.loadFromStorage();
  }

  /**
   * 从本地存储加载Token
   */
  private loadFromStorage() {
    try {
      const stored = localStorage.getItem(TOKEN_KEY);
      if (stored) {
        this.tokenInfo = JSON.parse(stored);
      }
    } catch (error) {
      console.error('加载Token失败:', error);
      this.tokenInfo = null;
    }
  }

  /**
   * 保存Token到本地存储
   */
  private saveToStorage() {
    if (this.tokenInfo) {
      localStorage.setItem(TOKEN_KEY, JSON.stringify(this.tokenInfo));
    } else {
      localStorage.removeItem(TOKEN_KEY);
    }
  }

  /**
   * 获取当前AccessToken
   */
  getAccessToken(): string | null {
    return this.tokenInfo?.accessToken || null;
  }

  /**
   * 获取刷新Token
   */
  getRefreshToken(): string | null {
    return this.tokenInfo?.refreshToken || null;
  }

  /**
   * 检查Token是否过期
   */
  isExpired(): boolean {
    if (!this.tokenInfo) return true;
    // 提前判断，比如 expiresAt - 10秒
    return Date.now() >= (this.tokenInfo.expiresAt - REFRESH_BEFORE_SECONDS * 1000);
  }

  /**
   * 设置Token
   */
  setToken(tokenInfo: TokenInfo) {
    this.tokenInfo = tokenInfo;
    this.saveToStorage();

    // 通知所有监听器
    this.notifyListeners(tokenInfo.accessToken);
  }

  /**
   * 清除Token
   */
  clearToken() {
    this.tokenInfo = null;
    localStorage.removeItem(TOKEN_KEY);

    // 通知所有监听器
    this.notifyListeners('');
  }

  /**
   * 刷新Token
   */
  async refreshToken(): Promise<TokenInfo> {
    // 如果正在刷新，直接返回
    if (this.refreshing) {
      return this.refreshing;
    }

    // 如果有refreshToken，尝试刷新
    const refreshToken = this.getRefreshToken();
    if (!refreshToken) {
      throw new Error('没有refreshToken，需要重新登录');
    }

    this.refreshing = this.doRefresh(refreshToken);

    try {
      const newTokenInfo = await this.refreshing;
      this.refreshing = null;
      return newTokenInfo;
    } catch (error) {
      this.refreshing = null;
      this.clearToken();
      throw error;
    }
  }

  /**
   * 执行刷新
   */
  private async doRefresh(refreshToken: string): Promise<TokenInfo> {
    // 调用刷新Token的API
    const response = await fetch('/api/auth/refresh', {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({ refreshToken })
    });

    if (!response.ok) {
      throw new Error('刷新Token失败');
    }

    const result = await response.json();

    if (result.code !== 0) {
      throw new Error(result.message || '刷新Token失败');
    }

    const tokenInfo: TokenInfo = {
      accessToken: result.data.accessToken,
      refreshToken: result.data.refreshToken,
      expiresAt: Date.now() + result.data.expiresIn * 1000
    };

    this.setToken(tokenInfo);
    return tokenInfo;
  }

  /**
   * 获取Token（自动刷新如果快过期）
   */
  async getToken(): Promise<string> {
    if (!this.isExpired()) {
      return this.getAccessToken()!;
    }

    const tokenInfo = await this.refreshToken();
    return tokenInfo.accessToken;
  }

  /**
   * 添加Token变更监听器
   */
  addListener(callback: (token: string) => void) {
    this.listeners.add(callback);
  }

  /**
   * 移除Token变更监听器
   */
  removeListener(callback: (token: string) => void) {
    this.listeners.delete(callback);
  }

  /**
   * 通知所有监听器
   */
  private notifyListeners(token: string) {
    this.listeners.forEach(callback => {
      try {
        callback(token);
      } catch (error) {
        console.error('Token监听器执行失败:', error);
      }
    });
  }
}

// 导出单例
export const tokenManager = new TokenManager();
```

### 4.2 缓存管理器实现

```typescript
/**
 * 缓存管理器
 * 负责HTTP请求的缓存管理
 */

import { CacheConfig } from './types';

// 默认缓存配置
const DEFAULT_MAX_SIZE = 100;      // 最大缓存数量
const DEFAULT_TTL = 5 * 60 * 1000; // 默认缓存时间（5分钟）

class CacheManager {
  private cache: Map<string, CacheConfig> = new Map();
  private maxSize: number;
  private defaultTtl: number;
  private cleanupTimer: NodeJS.Timeout | null = null;

  constructor(options = {}) {
    this.maxSize = options.maxSize || DEFAULT_MAX_SIZE;
    this.defaultTtl = options.ttl || DEFAULT_TTL;

    // 启动定时清理
    this.startCleanupTimer();
  }

  /**
   * 生成缓存Key
   */
  generateKey(url: string, params?: Record<string, any>): string {
    const key = params
      ? `${url}:${JSON.stringify(params)}`
      : url;
    return key;
  }

  /**
   * 获取缓存
   */
  get<T = any>(key: string): T | null {
    const cached = this.cache.get(key);

    if (!cached) {
      return null;
    }

    // 检查是否过期
    if (Date.now() > cached.expiry) {
      this.cache.delete(key);
      return null;
    }

    return cached.data as T;
  }

  /**
   * 设置缓存
   */
  set(key: string, data: any, ttl?: number, tags?: string[]) {
    // 如果缓存已满，删除最早的
    if (this.cache.size >= this.maxSize) {
      this.evictOldest();
    }

    const expiry = Date.now() + (ttl || this.defaultTtl);

    this.cache.set(key, {
      key,
      data,
      expiry,
      tags
    });
  }

  /**
   * 删除缓存
   */
  delete(key: string) {
    this.cache.delete(key);
  }

  /**
   * 清除所有缓存
   */
  clear() {
    this.cache.clear();
  }

  /**
   * 清除指定标签的缓存
   */
  clearByTags(tags: string[]) {
    const keysToDelete: string[] = [];

    this.cache.forEach((cached, key) => {
      if (cached.tags && cached.tags.some(tag => tags.includes(tag))) {
        keysToDelete.push(key);
      }
    });

    keysToDelete.forEach(key => this.cache.delete(key));
  }

  /**
   * 检查是否存在
   */
  has(key: string): boolean {
    if (!this.cache.has(key)) {
      return false;
    }

    const cached = this.cache.get(key)!;

    // 检查是否过期
    if (Date.now() > cached.expiry) {
      this.cache.delete(key);
      return false;
    }

    return true;
  }

  /**
   * 删除最早的缓存
   */
  private evictOldest() {
    let oldestKey: string | null = null;
    let oldestTime = Infinity;

    this.cache.forEach((cached, key) => {
      if (cached.expiry < oldestTime) {
        oldestTime = cached.expiry;
        oldestKey = key;
      }
    });

    if (oldestKey) {
      this.cache.delete(oldestKey);
    }
  }

  /**
   * 启动定时清理
   */
  private startCleanupTimer() {
    // 每分钟清理一次过期缓存
    this.cleanupTimer = setInterval(() => {
      this.cleanup();
    }, 60000);
  }

  /**
   * 清理过期缓存
   */
  cleanup() {
    const now = Date.now();
    const keysToDelete: string[] = [];

    this.cache.forEach((cached, key) => {
      if (now > cached.expiry) {
        keysToDelete.push(key);
      }
    });

    keysToDelete.forEach(key => this.cache.delete(key));

    if (keysToDelete.length > 0) {
      console.log(`清理了 ${keysToDelete.length} 个过期缓存`);
    }
  }

  /**
   * 销毁
   */
  destroy() {
    if (this.cleanupTimer) {
      clearInterval(this.cleanupTimer);
      this.cleanupTimer = null;
    }
    this.cache.clear();
  }

  /**
   * 获取统计信息
   */
  getStats() {
    return {
      size: this.cache.size,
      maxSize: this.maxSize,
      keys: Array.from(this.cache.keys())
    };
  }
}

// 导出单例
export const cacheManager = new CacheManager();
```

### 5.2 完整实现

```typescript
/**
 * 错误处理模块
 * 统一的错误处理和用户提示
 */

import { HttpError } from './types';

// 错误码到错误消息的映射
const ERROR_MESSAGES: Record<number, string> = {
  400: '请求参数错误',
  401: '登录已过期，请重新登录',
  403: '您没有权限执行此操作',
  404: '请求的资源不存在',
  408: '请求超时，请重试',
  500: '服务器开小差了，请稍后再试',
  502: '服务器网关错误',
  503: '服务暂时不可用',
  504: '网关超时'
};

// 业务错误码映射
const BUSINESS_ERROR_MESSAGES: Record<string, string> = {
  'TOKEN_EXPIRED': '登录已过期',
  'TOKEN_INVALID': '登录信息无效',
  'USER_NOT_FOUND': '用户不存在',
  'PERMISSION_DENIED': '没有权限',
  'RESOURCE_LOCKED': '资源被锁定',
  'RATE_LIMIT_EXCEEDED': '请求太频繁，请稍后再试'
};

/**
 * 错误处理配置
 */
export interface ErrorHandlerConfig {
  // 是否显示错误提示
  showError?: boolean;
  // 是否显示成功提示
  showSuccess?: boolean;
  // 是否在控制台打印错误
  logError?: boolean;
  // 登录页面路径
  loginUrl?: string;
  // 自定义错误消息
  customMessages?: Record<string, string>;
}

class ErrorHandler {
  private config: ErrorHandlerConfig;
  private listeners: Set<(error: HttpError) => void> = new Set();

  constructor(config: ErrorHandlerConfig = {}) {
    this.config = {
      showError: true,
      showSuccess: false,
      logError: true,
      loginUrl: '/login',
      customMessages: {},
      ...config
    };
  }

  /**
   * 处理错误
   */
  handle(error: HttpError): HttpError {
    // 打印到控制台
    if (this.config.logError) {
      console.error('请求错误:', {
        message: error.message,
        status: error.status,
        code: error.code,
        url: error.config?.url
      });
    }

    // 通知监听器
    this.notifyListeners(error);

    // 显示错误提示
    if (this.config.showError) {
      this.showToast(error);
    }

    // 处理401错误
    if (error.status === 401) {
      this.handleUnauthorized(error);
    }

    return error;
  }

  /**
   * 显示错误提示
   */
  private showToast(error: HttpError) {
    // 获取错误消息
    let message = error.message;

    // 优先使用自定义消息
    if (error.code && this.config.customMessages?.[error.code]) {
      message = this.config.customMessages[error.code]!;
    }
    // 其次使用HTTP状态码消息
    else if (ERROR_MESSAGES[error.status!]) {
      message = ERROR_MESSAGES[error.status!]!;
    }
    // 最后使用业务错误消息
    else if (error.code && BUSINESS_ERROR_MESSAGES[error.code]) {
      message = BUSINESS_ERROR_MESSAGES[error.code]!;
    }

    // 使用UI库显示提示
    // 这里使用Ant Design作为示例
    if (typeof window !== 'undefined' && (window as any).message) {
      (window as any).message.error(message);
    } else {
      // 降级使用alert
      alert(message);
    }
  }

  /**
   * 处理401未授权错误
   */
  private handleUnauthorized(error: HttpError) {
    // 清除Token
    localStorage.removeItem('auth_tokens');

    // 跳转到登录页
    if (window.location.pathname !== this.config.loginUrl) {
      window.location.href = this.config.loginUrl;
    }
  }

  /**
   * 创建HttpError
   */
  createError(status: number, message: string, data?: any): HttpError {
    const error = new Error(message) as HttpError;
    error.status = status;
    error.message = message;
    error.data = data;
    return error;
  }

  /**
   * 解析响应错误
   */
  parseResponseError(response: any, config: any): HttpError {
    let message = '请求失败';
    let code: string | undefined;
    let data: any;

    if (response) {
      // 后端返回的格式可能是 { code, message, data }
      if (typeof response === 'object') {
        code = response.code;
        message = response.message || message;
        data = response.data;
      } else {
        message = String(response);
      }
    }

    const error = new Error(message) as HttpError;
    error.code = code;
    error.message = message;
    error.data = data;
    error.config = config;

    return error;
  }

  /**
   * 添加错误监听器
   */
  addListener(callback: (error: HttpError) => void) {
    this.listeners.add(callback);
  }

  /**
   * 移除错误监听器
   */
  removeListener(callback: (error: HttpError) => void) {
    this.listeners.delete(callback);
  }

  /**
   * 通知所有监听器
   */
  private notifyListeners(error: HttpError) {
    this.listeners.forEach(callback => {
      try {
        callback(error);
      } catch (e) {
        console.error('错误监听器执行失败:', e);
      }
    });
  }
}
```

```typescript
/**
 * HTTP客户端
 * 整合所有功能，提供统一的网络请求接口
 */

import { tokenManager } from './TokenManager';
import { cacheManager } from './CacheManager';
import { errorHandler, createHttpError } from './ErrorHandler';
import type { RequestConfig, HttpResponse, HttpError } from './types';

// 请求计数器（用于生成唯一ID)
let requestIdCounter = 0;

class HttpClient {
  // 默认配置
  private defaults: RequestConfig = {
    method: 'GET',
    timeout: 30000,
    withCredentials: true,
    responseType: 'json'
  };

  // 请求拦截器
  private requestInterceptors: Array<{
    onFulfilled?: (config: RequestConfig) => RequestConfig | Promise<RequestConfig>;
    onRejected?: (error: any) => any;
  }> = [];

  // 响应拦截器
  private responseInterceptors: Array<{
    onFulfilled?: (response: HttpResponse) => HttpResponse | Promise<HttpResponse>;
    onRejected?: (error: HttpError) => any;
  }> = [];

  // 正在进行的请求（用于取消）
  private pendingRequests: Map<string, AbortController> = new Map();

  /**
   * 发送请求
   */
  async request<T = any>(config: RequestConfig): Promise<HttpResponse<T>> {
    // 生成请求ID
    const requestId = `req_${++requestIdCounter}_${Date.now()}`;

    // 合并配置
    const mergedConfig: RequestConfig = {
      ...this.defaults,
      ...config,
      headers: {
        ...this.defaults.headers,
        ...config.headers
      }
    };

    // 生成缓存key
    const cacheKey = cacheManager.generateKey(
      mergedConfig.url,
      mergedConfig.params as Record<string, any>
    );

    // 检查是否需要缓存（仅GET请求）
    const useCache = mergedConfig.method === 'GET' && mergedConfig.cache !== false;

    if (useCache && cacheManager.has(cacheKey)) {
      console.log('命中缓存:', cacheKey);
      const cachedData = cacheManager.get<T>(cacheKey);
      return {
        data: cachedData!,
        status: 200,
        statusText: 'OK (cached)',
        headers: {},
        config: mergedConfig
      };
    }

    // 创建AbortController
    const controller = new AbortController();
    this.pendingRequests.set(requestId, controller);

    // 设置超时
    const timeoutId = setTimeout(() => {
      controller.abort();
    }, mergedConfig.timeout!);

    try {
      // 执行请求拦截器
      let modifiedConfig = mergedConfig;
      for (const interceptor of this.requestInterceptors) {
        if (interceptor.onFulfilled) {
          modifiedConfig = await interceptor.onFulfilled(modifiedConfig);
        }
      }

      // 构建URL
      let url = modifiedConfig.url;
      if (modifiedConfig.params) {
        const searchParams = new URLSearchParams();
        Object.entries(modifiedConfig.params).forEach(([key, value]) => {
          if (value !== undefined && value !== null) {
            searchParams.append(key, String(value));
          }
        });
        const queryString = searchParams.toString();
        if (queryString) {
          url += (url.includes('?') ? '&' : '?') + queryString;
        }
      }

      // 准备请求头
      const headers: Record<string, string> = {
        ...modifiedConfig.headers
      };

      // 如果有Token，添加到请求头
      const token = tokenManager.getAccessToken();
      if (token) {
        headers['Authorization'] = `Bearer ${token}`;
      }

      // 发送请求
      const fetchOptions: RequestInit = {
        method: modifiedConfig.method,
        headers,
        credentials: modifiedConfig.withCredentials ? 'include' : 'same-origin',
        signal: controller.signal
      };

      // 添加请求体
      if (modifiedConfig.data && !['GET', 'HEAD'].includes(modifiedConfig.method!)) {
        if (modifiedConfig.data instanceof FormData) {
          fetchOptions.body = modifiedConfig.data;
          // 不要设置Content-Type，让浏览器自动设置
          delete headers['Content-Type'];
        } else {
          fetchOptions.body = JSON.stringify(modifiedConfig.data);
        }
      }

      // 执行实际请求
      const response = await fetch(url, fetchOptions);

      // 清除超时
      clearTimeout(timeoutId);

      // 检查HTTP状态
      if (!response.ok) {
        throw createHttpError(
          `HTTP ${response.status}: ${response.statusText}`,
          response.status,
          undefined,
          await response.text()
        );
      }

      // 解析响应数据
      let data: T;
      const contentType = response.headers.get('Content-Type') || '';

      if (contentType.includes('application/json')) {
        data = await response.json();
      } else if (contentType.includes('text/')) {
        data = await response.text() as any;
      } else {
        data = await response.blob() as any;
      }

      // 构建响应对象
      const httpResponse: HttpResponse<T> = {
        data,
        status: response.status,
        statusText: response.statusText,
        headers: this.parseHeaders(response.headers),
        config: modifiedConfig
      };

      // 执行响应拦截器
      let finalResponse = httpResponse;
      for (const interceptor of this.responseInterceptors) {
        if (interceptor.onFulfilled) {
          finalResponse = await interceptor.onFulfilled(finalResponse);
        }
      }

      // 缓存结果
      if (useCache && httpResponse.data) {
        cacheManager.set(
          cacheKey,
          httpResponse.data,
          modifiedConfig.cacheTTL,
          modifiedConfig.cacheTags
        );
      }

      return finalResponse;

    } catch (error: any) {
      // 清除超时
      clearTimeout(timeoutId);

      // 处理错误
      if (error.name === 'AbortError') {
        const httpError = createHttpError('请求被取消', 0, 'ABORTED');
        httpError.config = mergedConfig;
        throw httpError;
      }

      if (error.name === 'TypeError' && error.message.includes('fetch')) {
        const httpError = createHttpError('网络错误，请检查您的网络连接', 0, 'NETWORK_ERROR');
        httpError.config = mergedConfig;
        errorHandler.handle(httpError);
        throw httpError;
      }

      // 已经是HttpError，直接抛出
      if (error.status) {
        errorHandler.handle(error);
        throw error;
      }

      // 其他错误
      const httpError = createHttpError(error.message || '未知错误', 0, 'UNKNOWN');
      httpError.config = mergedConfig;
      errorHandler.handle(httpError);
      throw httpError;

    } finally {
      // 移除pending请求
      this.pendingRequests.delete(requestId);
    }
  }

  /**
   * GET请求
   */
  get<T = any>(url: string, config: RequestConfig = {}): Promise<HttpResponse<T>> {
    return this.request<T>({ ...config, url, method: 'GET' });
  }

  /**
   * POST请求
   */
  post<T = any>(url: string, data?: any, config: RequestConfig = {}): Promise<HttpResponse<T>> {
    return this.request<T>({ ...config, url, method: 'POST', data });
  }

  /**
   * PUT请求
   */
  put<T = any>(url: string, data?: any, config: RequestConfig = {}): Promise<HttpResponse<T>> {
    return this.request<T>({ ...config, url, method: 'PUT', data });
  }

  /**
   * PATCH请求
   */
  patch<T = any>(url: string, data?: any, config: RequestConfig = {}): Promise<HttpResponse<T>> {
    return this.request<T>({ ...config, url, method: 'PATCH', data });
  }

  /**
   * DELETE请求
   */
  delete<T = any>(url: string, config: RequestConfig = {}): Promise<HttpResponse<T>> {
    return this.request<T>({ ...config, url, method: 'DELETE' });
  }

  /**
   * 添加请求拦截器
   */
  addRequestInterceptor(interceptor: {
    onFulfilled?: (config: RequestConfig) => RequestConfig | Promise<RequestConfig>;
    onRejected?: (error: any) => any;
  }) {
    this.requestInterceptors.push(interceptor);
  }

  /**
   * 添加响应拦截器
   */
  addResponseInterceptor(interceptor: {
    onFulfilled?: (response: HttpResponse) => HttpResponse | Promise<HttpResponse>;
    onRejected?: (error: HttpError) => any;
  }) {
    this.responseInterceptors.push(interceptor);
  }

  /**
   * 取消所有请求
   */
  cancelAll() {
    this.pendingRequests.forEach((controller, id) => {
      console.log('取消请求:', id);
      controller.abort();
    });
    this.pendingRequests.clear();
  }

  /**
   * 解析响应头
   */
  private parseHeaders(headers: Headers): Record<string, string> {
    const result: Record<string, string> = {};
    headers.forEach((value, key) => {
      result[key.toLowerCase()] = value;
    });
    return result;
  }
}

// 导出单例
export const httpClient = new HttpClient();
```

```typescript
export async function uploadAvatar(
  file: File,
  onProgress?: (percent: number) => void
): Promise<HttpResponse<{ url: string }>> {
  const formData = new FormData();
  formData.append('avatar', file);

  return new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest();

    xhr.open('POST', '/api/users/avatar', true);

    // 上传进度
    xhr.upload.onprogress = (e) => {
      if (e.lengthComputable && onProgress) {
        const percent = Math.round((e.loaded / e.total) * 100);
        onProgress(percent);
      }
    };

    xhr.onload = () => {
      if (xhr.status >= 200 && xhr.status < 300) {
        resolve({
          data: JSON.parse(xhr.responseText),
          status: xhr.status,
          statusText: xhr.statusText,
          headers: {}
        });
      } else {
        reject(new Error(`上传失败: ${xhr.status}`));
      }
    };

    xhr.onerror = () => reject(new Error('网络错误'));

    // 添加token
    const token = localStorage.getItem('auth_tokens');
    if (token) {
      const { accessToken } = JSON.parse(token);
      xhr.setRequestHeader('Authorization', `Bearer ${accessToken}`);
    }

    xhr.send(formData);
  });
}
```

```typescript
  const [loading, setLoading] = useState(false);
  const [error, setError] = useState<Error | null>(null);

  useEffect(() => {
    if (userId === null) {
      setUser(null);
      return;
    }

    let cancelled = false;

    async function fetchUser() {
      setLoading(true);
      setError(null);

      try {
        const response = await getUser(userId);
        if (!cancelled) {
          setUser(response.data);
        }
      } catch (err) {
        if (!cancelled) {
          setError(err as Error);
        }
      } finally {
        if (!cancelled) {
          setLoading(false);
        }
      }
    }

    fetchUser();

    return () => {
      cancelled = true;
    };
  }, [userId]);

  return { user, loading, error };
}
```

## 总结：网络层封装的核心思想

1. **统一错误处理**：所有请求错误都经过同一个错误处理中心
2. **Token自动刷新**：用户无感知地续期，掉线自动跳转登录
3. **请求缓存**：重复请求直接返回缓存，减少服务器压力
4. **请求池管理**：控制并发数量，避免浏览器卡顿
5. **类型安全**：使用TypeScript确保每个API都有完整类型提示

**最佳实践**：
- 所有API都要有类型定义
- 统一使用单例的HttpClient
- 错误处理要区分HTTP错误和业务错误
- 敏感操作（登录、支付）要单独处理
- 做好监控和日志记录
