本文档系统讲解微信小程序从技术架构到性能优化、从交互细节到商业化落地的完整知识体系。通过大量实战代码，帮助开发者掌握小程序精细化开发的核心技能。

## 一、小程序概述

### 1.1 小程序 vs H5 vs App

在移动端开发领域，小程序、H5网页和原生App是三种主要的解决方案。它们各有优劣，适用于不同的场景。

| 对比维度 | 微信小程序 | H5网页 | 原生App |
|---------|-----------|--------|---------|
| **加载速度** | 首次加载后缓存，快 | 每次需网络加载，慢 | 本地加载，最快 |
| **用户体验** | 接近原生，流畅 | 一般，有白屏 | 最优 |
| **开发成本** | 中等 | 低 | 高 |
| **发布周期** | 即时审核通过 | 零成本 | App Store审核 |
| **系统能力** | 受限但丰富 | 受浏览器限制 | 完全开放 |
| **留存率** | 可留在桌面，高 | 收藏率低 | 安装率低 |

**小程序的优势在于：**

首次加载后，小程序会将代码包缓存在本地，后续打开几乎是瞬时的。用户可以"发现"小程序并将其添加到我的小程序，留存率远高于H5。同时小程序借助微信的生态，可以直接分享到朋友圈、微信群，获得社交裂变的机会。

### 1.2 微信小程序技术架构

微信小程序采用了**双线程模型**的技术架构，这是小程序体验接近原生的核心技术保障。

```
┌─────────────────────────────────────────────────────────┐
│                      微信客户端                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ┌─────────────────┐        ┌─────────────────┐        │
│   │   渲染线程      │        │   逻辑线程       │        │
│   │   (WebView)     │  ⬌    │   (V8引擎)      │        │
│   │                 │  数据  │                 │        │
│   │  WXML + WXSS   │  协议  │   JavaScript    │        │
│   │  + JS Core     │        │   + 小程序框架   │        │
│   └─────────────────┘        └─────────────────┘        │
│                                                         │
│   ┌─────────────────┐        ┌─────────────────┐        │
│   │   Native层      │        │   微信SDK       │        │
│   │   (原生能力)     │  ⬌    │   (微信API)     │        │
│   └─────────────────┘        └─────────────────┘        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**渲染线程负责：**
- 解析WXML文件，构建DOM树
- 解析WXSS文件，应用样式
- 渲染页面内容
- 执行动画效果
- 响应用户交互

**逻辑线程负责：**
- 运行小程序的业务逻辑代码
- 管理小程序的生命周期
- 调用微信开放API
- 处理数据请求
- 存储管理

**双线程之间的协作通过WeiXin JS Bridge进行：**

```javascript
// 渲染线程向逻辑线程通信（用户操作）
// 用户点击按钮 → 渲染线程捕获 → 通过bridge发送到逻辑线程
// 逻辑线程处理业务逻辑 → 通过bridge通知渲染线程更新

// 示例：按钮点击事件流程
// 1. 用户点击 <button bindtap="onTap"> 点我 </button>
// 2. 渲染线程捕获点击事件
// 3. 通过 WeixinJSBridge 将事件发送给逻辑线程
// 4. 逻辑线程执行 onTap 函数
// 5. this.setData({ msg: '已点击' }) 更新数据
// 6. 逻辑线程通过 WeixinJSBridge 通知渲染线程
// 7. 渲染线程重新渲染对应的UI
```

### 1.3 双线程模型的深层理解

双线程模型的设计主要是为了**安全**和**性能**两个核心目标。

**安全层面：** 小程序的代码运行在微信客户端内部，用户无法直接访问逻辑线程的JavaScript环境。这意味着开发者可以安全地使用微信支付的密钥、用户敏感信息等，而不必担心被恶意用户通过控制台调试获取。H5网页则完全没有这个保护，任何人都可以打开开发者工具查看所有代码。

**性能层面：** 渲染和逻辑分离后，即使业务逻辑出现阻塞，也不会影响页面的流畅渲染。微信客户端会对渲染线程进行硬件加速，确保动画和滚动操作达到60fps的流畅度。

```javascript
// 小程序中 setData 是异步的，不会阻塞渲染
// 这与React的setState类似，但底层实现不同

Page({
  data: {
    list: []
  },

  onLoad: function() {
    // 模拟数据加载
    wx.showLoading({ title: '加载中' });

    // 同步打印 - 不阻塞UI
    console.log('开始加载数据');

    // setData是异步的，不会立即更新视图
    this.setData({ list: [1, 2, 3] });

    // 立即读取data，值可能还是旧的
    console.log(this.data.list); // []

    // 下一次事件循环才会真正更新视图
    setTimeout(() => {
      console.log(this.data.list); // [1, 2, 3]
    }, 0);
  }
});
```

### 1.4 我的思考：为什么小程序体验好

小程序的优秀体验来自于多个技术细节的精心设计。

**首屏加载优化：** 小程序在首次打开时会有一个启动画面（可以自定义），这段时间微信客户端在后台预加载小程序的代码包。加载完成后，微信会立即展示小程序的首页，而不是让用户看到白屏。这种"伪即开即用"的体验极大地降低了用户的等待感知。

**原生组件封装：** 微信将一些常用组件（如map、video、canvas、input）封装为原生组件，这些组件运行在原生线程中，而不是WebView线程。这意味着即使用户在页面上快速滑动，原生组件也能保持稳定，不会出现卡顿或丢帧。

**智能的数据请求：** 小程序的request API会自动携带登录态、UA标识等信息，微信还会根据网络状况智能选择最快的服务器IP。这些细节让开发者无需关注底层优化，专注于业务逻辑。

**统一的视觉规范：** 小程序的组件库遵循iOS和Android的设计规范，在不同平台会自动适配相应的视觉风格。开发者无需为两个平台分别适配，这大大降低了开发成本。

## 二、精细化布局

### 2.1 响应式单位：rpx

rpx（responsive pixel）是微信小程序独创的响应式长度单位。它的设计思想是让开发者只需要编写一套设计稿，就能自动适配所有屏幕尺寸。

**rpx的原理：** rpx会根据屏幕宽度进行等分。在标准屏幕（750px宽度）下，1rpx等于1px。微信规定iPhone 6的屏幕宽度为750物理像素，所以iPhone 6下1rpx = 1px。

```
屏幕宽度换算公式：
1rpx = 屏幕宽度 / 750

各设备换算示例：
- iPhone 6 (375pt / 750px): 1rpx = 0.5px
- iPhone 6 Plus (414pt / 1242px): 1rpx = 0.552px
- iPhone 12 (390pt / 1170px): 1rpx = 0.52px
```

**实际使用建议：**

```css
/* 设计稿通常基于iPhone 6（750px宽度）*/
/* 如果设计稿上某个元素宽度为375px，在小程序中应该设置为375rpx */

/* 间距使用示例 */
.container {
  padding: 20rpx;        /* 20rpx在iPhone 6上等于20px */
  margin: 40rpx 30rpx;   /* 上下40rpx，左右30rpx */
}

/* 字体大小使用 */
.title {
  font-size: 32rpx;      /* 标题字体 */
}

.content {
  font-size: 28rpx;      /* 正文字体 */
}

.caption {
  font-size: 24rpx;      /* 辅助文字 */
}

/* 组件宽度 */
.button {
  width: 600rpx;         /* 占屏幕80%的按钮 */
  height: 88rpx;         /* 固定高度 */
  border-radius: 44rpx; /* 圆角为高度的一半，胶囊按钮 */
}
```

**rpx vs 百分比 vs vw/vh：**

```css
/* 方案对比 */
.box-rpx {
  width: 50%;  /* 父元素50%，但嵌套时计算复杂 */
}

.box-vw {
  width: 50vw; /* 视口50%，但部分安卓机兼容差 */
}

.box-rpx {
  width: 375rpx; /* 固定值，精确控制 */
}

/* 推荐做法：关键尺寸用rpx，容器用flex自适应 */
.container {
  display: flex;
  width: 750rpx;    /* 固定值 */
  height: 100vh;    /* 视口高度 */
}

.item {
  flex: 1;          /* flex自适应填充剩余空间 */
  margin: 0 20rpx;  /* 固定间距用rpx */
}
```

### 2.2 弹性布局：flex

Flex布局是小程序开发中最常用的布局方式，它能优雅地解决各种排列和对齐问题。

**基础用法：**

```xml
<!-- WXML -->
<view class="flex-row">
  <view class="item">左</view>
  <view class="item center">中</view>
  <view class="item">右</view>
</view>
```

```css
/* WXSS */
.flex-row {
  display: flex;
  flex-direction: row;        /* 默认值，可省略 */
  justify-content: space-between; /* 主轴对齐方式 */
  align-items: center;         /* 交叉轴对齐方式 */
  height: 100rpx;
}

.item {
  width: 100rpx;
  height: 100rpx;
  background: #ddd;
}

.center {
  /* 某个元素特殊处理 */
}
```

**Flex完整属性解析：**

```css
/* 容器的属性 */
.container {
  /* 1. 排列方向 */
  flex-direction: row | row-reverse | column | column-reverse;

  /* 2. 换行方式 */
  flex-wrap: nowrap | wrap | wrap-reverse;

  /* 3. 主轴对齐（水平方向） */
  justify-content: flex-start | flex-end | center | space-between | space-around | space-evenly;

  /* 4. 交叉轴对齐（垂直方向） */
  align-items: flex-start | flex-end | center | baseline | stretch;

  /* 5. 多行对齐（换行后） */
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}

/* 子元素的属性 */
.item {
  /* 1. 放大比例（0表示不放大） */
  flex-grow: 0;

  /* 2. 缩小比例（0表示不缩小） */
  flex-shrink: 1;

  /* 3. 占据空间（可以是固定值或auto） */
  flex-basis: auto | <length>;

  /* 简写形式：放大|缩小|基础 */
  flex: 0 1 auto;

  /* 常用简写 */
  flex: 1;  /* 1 1 0%，占据剩余空间 */
  flex: 0 0 200rpx; /* 固定宽度，不放大不缩小 */

  /* 4. 个别对齐（覆盖容器的align-items） */
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
```

### 2.3 固定定位：吸顶吸底

### 2.4 键盘适配：input固定

键盘弹出时的适配是小程序开发中的一个难点，特别是当页面有fixed定位的输入框时。

**问题场景：** 当用户点击页面底部的固定输入框时，键盘会遮住输入框，导致用户看不到自己输入的内容。

**解决方案：**

```xml
<!-- 聊天页面示例 -->
<view class="chat-page">
  <!-- 消息列表 -->
  <scroll-view
    class="message-list"
    scroll-y
    scroll-into-view="{{scrollToMessage}}"
    bindscroll="onScroll"
  >
    <view id="msg-{{item.id}}" class="message-item" wx:for="{{messages}}" wx:key="id">
      <text>{{item.content}}</text>
    </view>
  </scroll-view>

  <!-- 底部输入区域（关键部分） -->
  <view class="input-area {{keyboardHeight > 0 ? 'keyboard-up' : ''}}">
    <input
      class="message-input"
      value="{{inputValue}}"
      bindinput="onInput"
      bindfocus="onFocus"
      bindblur="onBlur"
      confirm-type="send"
      bindconfirm="onSend"
      placeholder="输入消息..."
    />
    <button class="send-btn" bindtap="onSend">发送</button>
  </view>
</view>
```

### 3.2 上拉加载：onReachBottom

上拉加载用于实现分页加载，当用户滑动到页面底部时自动加载更多数据。

**基础实现：**

```javascript
Page({
  data: {
    list: [],
    page: 1,
    pageSize: 20,
    hasMore: true,
    loading: false
  },

  onLoad: function() {
    this.loadData();
  },

  // 监听上拉触底
  onReachBottom: function() {
    console.log('触发上拉加载');

    // 如果没有更多数据或正在加载中，不再请求
    if (!this.data.hasMore || this.data.loading) {
      return;
    }

    this.loadMore();
  },

  async loadData: function() {
    this.setData({ loading: true });
    wx.showLoading({ title: '加载中...' });

    try {
      const res = await wx.request({
        url: 'https://api.example.com/list',
        data: {
          page: this.data.page,
          pageSize: this.data.pageSize
        }
      });

      const newList = res.data.list || [];

      this.setData({
        list: newList,
        hasMore: newList.length >= this.data.pageSize
      });
    } catch (err) {
      wx.showToast({ title: '加载失败', icon: 'none' });
    } finally {
      this.setData({ loading: false });
      wx.hideLoading();
    }
  },

  async loadMore: function() {
    const nextPage = this.data.page + 1;

    const res = await wx.request({
      url: 'https://api.example.com/list',
      data: {
        page: nextPage,
        pageSize: this.data.pageSize
      }
    });

    const newList = res.data.list || [];

    // 关键：累加数据，而不是替换
    this.setData({
      list: [...this.data.list, ...newList],
      page: nextPage,
      hasMore: newList.length >= this.data.pageSize
    });
  }
});
```

### 3.4 手势操作：滑动、长按

微信小程序支持多种手势操作，包括滑动（touch）、长按（longpress）等。

**基础手势识别：**

```xml<view
  class="gesture-area"
  bindtouchstart="onTouchStart"
  bindtouchmove="onTouchMove"
  bindtouchend="onTouchEnd"
  bindlongpress="onLongPress"
>
  <text>{{gestureText}}</text>
</view>
```

```javascript
Page({
  data: {
    gestureText: '请操作',
    startX: 0,
    startY: 0,
    currentX: 0,
    currentY: 0,
    isLongPress: false
  },

  // 手指触摸开始
  onTouchStart: function(e) {
    const touch = e.touches[0];
    this.setData({
      startX: touch.clientX,
      startY: touch.clientY,
      isLongPress: false,
      gestureText: '触摸开始'
    });
  },

  // 手指移动
  onTouchMove: function(e) {
    const touch = e.touches[0];
    const deltaX = touch.clientX - this.data.startX;
    const deltaY = touch.clientY - this.data.startY;

    // 判断滑动方向
    let direction = '';
    if (Math.abs(deltaX) > Math.abs(deltaY)) {
      // 水平滑动
      if (deltaX > 50) {
        direction = '向右滑动';
      } else if (deltaX < -50) {
        direction = '向左滑动';
      }
    } else {
      // 垂直滑动
      if (deltaY > 50) {
        direction = '向下滑动';
      } else if (deltaY < -50) {
        direction = '向上滑动';
      }
    }

    if (direction) {
      this.setData({ gestureText: direction });
    }

    this.setData({
      currentX: touch.clientX,
      currentY: touch.clientY
    });
  },

  // 手指触摸结束
  onTouchEnd: function(e) {
    if (!this.data.isLongPress) {
      this.setData({ gestureText: '触摸结束' });
    }
  },

  // 长按触发（300ms后触发）
  onLongPress: function(e) {
    this.setData({
      isLongPress: true,
      gestureText: '长按触发！'
    });

    // 长按操作
    wx.showToast({
      title: '长按删除',
      icon: 'none'
    });
  }
});
```

```javascript
// 手势密码核心逻辑
Page({
  data: {
    isSetting: true,           // 是否在设置模式
    isVerifying: false,        // 是否在验证模式
    errorMsg: '',
    canConfirm: false,
    firstPassword: [],         // 第一次绘制的密码
    currentPassword: [],       // 当前绘制的密码
    password: [],              // 已选择的点
    dots: [],                  // 9个点位的状态
    canvasWidth: 300,
    canvasHeight: 300,
    lastDot: null              // 最后一个点
  },

  onLoad: function(options) {
    this.initDots();
    this.initCanvas();
  },

  // 初始化9个点位
  initDots: function() {
    const dots = [];
    const gap = 100;  // 点位间距
    const offsetX = 50;
    const offsetY = 50;

    for (let i = 0; i < 9; i++) {
      const row = Math.floor(i / 3);
      const col = i % 3;
      dots.push({
        index: i,
        x: offsetX + col * gap,
        y: offsetY + row * gap,
        selected: false
      });
    }

    this.setData({ dots });
  },

  // 初始化画布
  initCanvas: function() {
    // 获取屏幕宽度来计算canvas尺寸
    const screenWidth = wx.getSystemInfoSync().windowWidth;
    const canvasSize = screenWidth - 60;
    this.setData({
      canvasWidth: canvasSize,
      canvasHeight: canvasSize
    });
  },

  // 触摸开始
  onTouchStart: function(e) {
    const touch = e.touches[0];
    const dot = this.hitTestDot(touch.clientX, touch.clientY);

    if (dot !== null) {
      this.selectDot(dot);
      this.setData({ lastDot: dot });
    }
  },

  // 触摸移动
  onTouchMove: function(e) {
    if (this.data.lastDot === null) return;

    const touch = e.touches[0];
    const dot = this.hitTestDot(touch.clientX, touch.clientY);

    // 绘制连接线
    this.drawLine(touch);

    if (dot !== null && !dot.selected) {
      // 检查是否需要跨点
      this.checkCrossDot(dot);
      this.selectDot(dot);
      this.setData({ lastDot: dot });
    }
  },

  // 触摸结束
  onTouchEnd: function() {
    const password = this.data.password;
    const passwordStr = password.join('');

    if (password.length < 4) {
      this.setData({ errorMsg: '至少连接4个点' });
      this.clearPassword();
      return;
    }

    if (this.data.isSetting) {
      if (!this.data.firstPassword.length) {
        // 第一次设置
        this.setData({
          firstPassword: password,
          errorMsg: '请再次绘制确认'
        });
        this.clearPassword();
      } else {
        // 确认密码
        if (passwordStr === this.data.firstPassword.join('')) {
          // 保存密码到本地
          wx.setStorageSync('gesturePassword', passwordStr);
          wx.showToast({ title: '设置成功' });
          // 跳转到其他页面或进入验证模式
        } else {
          this.setData({ errorMsg: '两次密码不一致，请重试' });
          this.clearPassword();
        }
      }
    } else {
      // 验证密码
      const savedPassword = wx.getStorageSync('gesturePassword');
      if (passwordStr === savedPassword) {
        wx.showToast({ title: '验证成功' });
        // 验证成功后的逻辑
      } else {
        this.setData({ errorMsg: '密码错误' });
        wx.vibrateShort({ type: 'heavy' });  // 震动提示
        this.clearPassword();
      }
    }
  },

  // 检测触碰点
  hitTestDot: function(x, y) {
    const dots = this.data.dots;
    const hitRadius = 50;  // 点击半径

    for (let i = 0; i < dots.length; i++) {
      const dot = dots[i];
      // 实际项目中需要考虑canvas的偏移量
      const dx = x - dot.x;
      const dy = y - dot.y;
      const distance = Math.sqrt(dx * dx + dy * dy);

      if (distance < hitRadius) {
        return dot;
      }
    }
    return null;
  },

  // 选择点位
  selectDot: function(dot) {
    const dots = this.data.dots;
    const password = this.data.password;

    dots[dot.index].selected = true;
    password.push(dot.index);

    this.setData({
      dots: dots,
      password: password,
      canConfirm: password.length >= 4
    });
  },

  // 检查跨点
  checkCrossDot: function(targetDot) {
    const password = this.data.password;
    if (password.length < 1) return;

    const lastDotIndex = password[password.length - 1];

    // 跨点规则：两个点之间的中点如果有未选中的点，自动选中它
    const crossPoints = {
      '0-2': 1, '0-6': 3, '0-8': 4,
      '1-7': 4, '2-6': 4, '2-8': 5,
      '3-5': 4, '6-8': 7
    };

    const key1 = `${lastDotIndex}-${targetDot.index}`;
    const key2 = `${targetDot.index}-${lastDotIndex}`;
    const crossIndex = crossPoints[key1] || crossPoints[key2];

    if (crossIndex !== undefined) {
      const crossDot = this.data.dots[crossIndex];
      if (!crossDot.selected) {
        this.selectDot(crossDot);
      }
    }
  },

  // 绘制连接线
  drawLine: function(touch) {
    // 使用canvas绘制，这里简化处理
    // 实际项目中需要在canvas上绘制线条
    const ctx = wx.createCanvasContext('gestureLine');
    ctx.setStrokeStyle('#ff6b6b');
    ctx.setLineWidth(6);
    ctx.lineCap = 'round';
    ctx.lineJoin = 'round';

    // 绘制从起点到当前点的线
    // 实际实现需要维护所有已选点和当前触摸位置
    ctx.draw();
  },

  // 清除密码
  clearPassword: function() {
    const dots = this.data.dots;
    dots.forEach(dot => dot.selected = false);

    setTimeout(() => {
      this.setData({
        dots: dots,
        password: [],
        lastDot: null,
        canConfirm: false
      });
    }, 500);  // 延迟清除，让用户看到最终路径
  },

  onCancel: function() {
    this.setData({
      firstPassword: [],
      errorMsg: ''
    });
    this.clearPassword();
  },

  onConfirm: function() {
    // 确认设置
  },

  onResetPassword: function() {
    wx.removeStorageSync('gesturePassword');
    wx.showToast({ title: '密码已重置' });
    this.clearPassword();
  }
});
```

```javascript
const imageUtils = {
  // 根据设备像素比选择合适尺寸
  getOptimalImageUrl: function(url, pixelRatio) {
    if (!url) return '';

    // 如果URL已经包含尺寸参数，直接返回
    if (url.includes('?w=') || url.includes('&w=')) {
      return url;
    }

    // 添加图片尺寸参数（CDN支持的情况下）
    const separator = url.includes('?') ? '&' : '?';
    return `${url}${separator}w=${750 * pixelRatio}&q=80`;
  },

  // 生成缩略图URL
  getThumbnailUrl: function(url, size = 200) {
    if (!url) return '';
    // 不同CDN的缩略图参数不同，这里以七牛为例
    const separator = url.includes('?') ? '&' : '?';
    return `${url}${separator}imageView2/1/w/${size}/h/${size}`;
  },

  // 渐进式加载（先加载小图，再加载大图）
  progressiveLoad: function(thumbnailUrl, fullUrl) {
    return new Promise((resolve, reject) => {
      // 先加载缩略图
      wx.getImageInfo({
        src: thumbnailUrl,
        success: () => {
          // 缩略图加载成功后显示
          resolve({ url: thumbnailUrl, isFull: false });

          // 再预加载完整图
          wx.getImageInfo({
            src: fullUrl,
            success: () => {
              resolve({ url: fullUrl, isFull: true });
            }
          });
        },
        fail: reject
      });
    });
  }
};
```

### 4.4 数据缓存：localStorage

合理使用本地存储可以减少网络请求，提升用户体验。

**localStorage API封装：**

```javascript
// 缓存工具类
class StorageCache {
  constructor(prefix = 'app_') {
    this.prefix = prefix;
    this.defaultExpire = 7 * 24 * 60 * 60 * 1000; // 默认7天
  }

  // 生成带前缀的key
  getKey(key) {
    return `${this.prefix}${key}`;
  }

  // 设置缓存（支持过期时间）
  set(key, value, expire = this.defaultExpire) {
    const data = {
      value,
      expire: expire ? Date.now() + expire : 0
    };
    wx.setStorageSync(this.getKey(key), data);
  }

  // 获取缓存
  get(key, defaultValue = null) {
    const data = wx.getStorageSync(this.getKey(key));

    if (!data) {
      return defaultValue;
    }

    // 检查是否过期
    if (data.expire && Date.now() > data.expire) {
      this.remove(key);
      return defaultValue;
    }

    return data.value;
  }

  // 删除缓存
  remove(key) {
    wx.removeStorageSync(this.getKey(key));
  }

  // 清空所有缓存
  clear() {
    wx.clearStorageSync();
  }

  // 异步封装
  async setAsync(key, value, expire) {
    return new Promise((resolve, reject) => {
      wx.setStorage({
        key: this.getKey(key),
        data: {
          value,
          expire: expire ? Date.now() + expire : 0
        },
        success: resolve,
        fail: reject
      });
    });
  }

  async getAsync(key, defaultValue = null) {
    return new Promise((resolve) => {
      wx.getStorage({
        key: this.getKey(key),
        success: (res) => {
          const data = res.data;
          if (!data) {
            resolve(defaultValue);
            return;
          }
          if (data.expire && Date.now() > data.expire) {
            this.remove(key);
            resolve(defaultValue);
            return;
          }
          resolve(data.value);
        },
        fail: () => {
          resolve(defaultValue);
        }
      });
    });
  }
}

// 创建实例
const cache = new StorageCache('mall_');

// 使用示例
Page({
  data: {
    userInfo: null,
    goodsList: []
  },

  onLoad: function() {
    this.loadGoodsList();
    this.loadUserInfo();
  },

  async loadGoodsList() {
    // 尝试从缓存获取
    const cachedList = cache.get('goodsList');

    if (cachedList) {
      this.setData({ goodsList: cachedList });
      // 后台静默刷新
      this.refreshGoodsList();
    } else {
      await this.refreshGoodsList();
    }
  },

  async refreshGoodsList() {
    const res = await wx.request({
```

**多级缓存策略：**

```javascript
// 多级缓存管理器
class MultiLevelCache {
  constructor() {
    this.memoryCache = new Map(); // 内存缓存
    this.memoryExpire = new Map(); // 内存过期时间
    this.checkInterval = null;
  }

  // 启动过期检查
  startExpireCheck(interval = 60000) {
    this.checkInterval = setInterval(() => {
      const now = Date.now();
      for (const [key, expire] of this.memoryExpire.entries()) {
        if (now > expire) {
          this.memoryCache.delete(key);
          this.memoryExpire.delete(key);
        }
      }
    }, interval);
  }

  // 停止检查
  stopExpireCheck() {
    if (this.checkInterval) {
      clearInterval(this.checkInterval);
    }
  }

  // 设置内存缓存
  setMemory(key, value, expire = 300000) {
    this.memoryCache.set(key, value);
    this.memoryExpire.set(key, Date.now() + expire);
  }

  // 获取内存缓存
  getMemory(key) {
    const expire = this.memoryExpire.get(key);
    if (expire && Date.now() > expire) {
      this.memoryCache.delete(key);
      this.memoryExpire.delete(key);
      return null;
    }
    return this.memoryCache.get(key);
  }

  // L1（内存）-> L2（本地）-> L3（网络）
  async get(key, options = {}) {
    const { networkFn, localExpire = 3600000, memoryExpire = 300000 } = options;

    // L1检查
    const memoryData = this.getMemory(key);
    if (memoryData !== null) {
      console.log('L1缓存命中', key);
      return memoryData;
    }

    // L2检查
    const localData = wx.getStorageSync(`cache_${key}`);
    if (localData && (!localData.expire || Date.now() < localData.expire)) {
      console.log('L2缓存命中', key);
      // 升级到L1
      this.setMemory(key, localData.value, memoryExpire);
      return localData.value;
    }

    // L3网络获取
    if (networkFn) {
      console.log('缓存未命中，网络获取', key);
      const data = await networkFn();
      // 写入L2
      wx.setStorageSync(`cache_${key}`, {
        value: data,
        expire: Date.now() + localExpire
      });
      // 写入L1
      this.setMemory(key, data, memoryExpire);
      return data;
    }

    return null;
  }

  // 清除所有缓存
  clear() {
    this.memoryCache.clear();
    this.memoryExpire.clear();
    // 注意：不清除本地缓存
  }
}

const multiCache = new MultiLevelCache();
```

### 4.5 我的思考：小程序性能是关键

小程序性能优化不是单一维度的改进，而是需要系统性的思考。

**性能指标体系：**

| 指标 | 定义 | 目标值 |
|------|------|--------|
| 首次渲染时间 | 启动到首屏内容可见 | < 1.5s |
| 可交互时间 | 启动到页面可响应 | < 3s |
| 页面切换时间 | 页面间跳转耗时 | < 300ms |
| setData耗时 | 数据更新到渲染完成 | < 16ms |
| 列表滚动帧率 | scroll-view滚动流畅度 | 60fps |

**性能优化的核心原则：**

减少setData的频率和数量比减少setData的次数更重要。setData会触发页面重渲染，如果每次setData传递大量数据，即使次数少也会有性能问题。正确的做法是精确setData，只传递变化的最小数据。

```javascript
// 错误做法：每次都传递全部数据
this.setData({
  list: newList  // 假设list有1000项，但只变化了1项
});

// 正确做法：精确传递变化的数据
this.setData({
  'list[0].title': newList[0].title  // 精确到具体字段
});

// 更正确的做法：使用key路径批量更新
const updatedItems = {};
newList.forEach((item, index) => {
  if (item.title !== this.data.list[index].title) {
    updatedItems[`list[${index}].title`] = item.title;
  }
});
if (Object.keys(updatedItems).length > 0) {
  this.setData(updatedItems);
}
```

**海报生成器实现：**

```javascript
// 海报生成器类
class PosterGenerator {
  constructor(options = {}) {
    this.width = options.width || 750;        // 海报宽度
    this.height = options.height || 1100;      // 海报高度
    this.backgroundColor = options.backgroundColor || '#ffffff';
    this.padding = options.padding || 40;      // 内边距
    this.pixelRatio = options.pixelRatio || wx.getSystemInfoSync().pixelRatio;

    // 画布上下文（延迟初始化）
    this.ctx = null;
    this.canvasId = 'posterCanvas';
  }

  // 初始化画布
  initCanvas() {
    // 创建canvas节点
    return new Promise((resolve, reject) => {
      const query = wx.createSelectorQuery();
      query.select(`#${this.canvasId}`)
        .fields({ node: true, size: true })
        .exec((res) => {
          if (!res[0]) {
            reject(new Error('Canvas not found'));
            return;
          }

          const canvas = res[0].node;
          const ctx = canvas.getContext('2d');

          // 设置canvas尺寸（需要乘以像素比）
          canvas.width = this.width * this.pixelRatio;
          canvas.height = this.height * this.pixelRatio;
          ctx.scale(this.pixelRatio, this.pixelRatio);

          this.ctx = ctx;
          this.canvas = canvas;
          resolve(ctx);
        });
    });
  }

  // 绘制背景
  drawBackground(color) {
    this.ctx.fillStyle = color || this.backgroundColor;
    this.ctx.fillRect(0, 0, this.width, this.height);
  }

  // 绘制图片
  async drawImage(imgUrl, x, y, w, h, options = {}) {
    return new Promise((resolve, reject) => {
      wx.getImageInfo({
        src: imgUrl,
        success: (res) => {
          const img = this.canvas.createImage();
          img.onload = () => {
            // 计算图片裁剪
            let sx = 0, sy = 0, sw = res.width, sh = res.height;

            if (options.mode === 'aspectFill') {
              // 裁剪到目标比例
              const targetRatio = w / h;
              const imgRatio = res.width / res.height;

              if (imgRatio > targetRatio) {
                sw = res.height * targetRatio;
                sx = (res.width - sw) / 2;
              } else {
                sh = res.width / targetRatio;
                sy = (res.height - sh) / 2;
              }
            }

            this.ctx.drawImage(img, sx, sy, sw, sh, x, y, w, h);
            resolve();
          };
          img.onerror = reject;
          img.src = res.path;
        },
        fail: reject
      });
    });
  }

  // 绘制圆角矩形
  drawRoundedRect(x, y, w, h, r, fillColor, strokeColor) {
    this.ctx.beginPath();
    this.ctx.moveTo(x + r, y);
    this.ctx.lineTo(x + w - r, y);
    this.ctx.quadraticCurveTo(x + w, y, x + w, y + r);
    this.ctx.lineTo(x + w, y + h - r);
    this.ctx.quadraticCurveTo(x + w, y + h, x + w - r, y + h);
    this.ctx.lineTo(x + r, y + h);
    this.ctx.quadraticCurveTo(x, y + h, x, y + h - r);
    this.ctx.lineTo(x, y + r);
    this.ctx.quadraticCurveTo(x, y, x + r, y);
    this.ctx.closePath();

    if (fillColor) {
      this.ctx.fillStyle = fillColor;
      this.ctx.fill();
    }
    if (strokeColor) {
      this.ctx.strokeStyle = strokeColor;
      this.ctx.stroke();
    }
  }

  // 绘制文字（单行）
  drawText(text, x, y, options = {}) {
    const {
      fontSize = 28,
      color = '#333333',
      align = 'left',
      baseline = 'top',
      fontWeight = 'normal',
      maxWidth = null
    } = options;

    this.ctx.font = `${fontWeight} ${fontSize}px sans-serif`;
    this.ctx.fillStyle = color;
    this.ctx.textAlign = align;
    this.ctx.textBaseline = baseline;

    this.ctx.fillText(text, x, y, maxWidth);
  }

  // 绘制文字（多行，自动换行）
  drawTextWrap(text, x, y, maxWidth, lineHeight, options = {}) {
    const { fontSize = 28, color = '#333333', maxLines = 0 } = options;

    this.ctx.font = `${fontSize}px sans-serif`;
    this.ctx.fillStyle = color;

    let line = '';
    let lines = [];
    let currentY = y;

    for (let i = 0; i < text.length; i++) {
      const char = text[i];
      const testLine = line + char;
      const metrics = this.ctx.measureText(testLine);

      if (metrics.width > maxWidth && i > 0) {
        lines.push(line);
        line = char;
      } else {
        line = testLine;
      }
    }
    lines.push(line);

    // 限制最大行数
    if (maxLines > 0 && lines.length > maxLines) {
      lines = lines.slice(0, maxLines);
      lines[lines.length - 1] = lines[lines.length - 1].replace(/…$/, '');
    }

    // 绘制每行
    lines.forEach((l, index) => {
      this.ctx.fillText(l, x, currentY + index * lineHeight);
    });

    return lines.length * lineHeight;
  }

  // 保存海报
  save(filename = 'poster') {
    return new Promise((resolve, reject) => {
      wx.canvasToTempFilePath({
        canvas: this.canvas,
        fileType: 'png',
        quality: 1,
        success: (res) => {
          // 保存到相册
          wx.saveImageToPhotosAlbum({
            filePath: res.tempFilePath,
            success: () => {
              resolve(res.tempFilePath);
            },
            fail: (err) => {
              if (err.errMsg.includes('auth deny')) {
                // 需要授权
                wx.showModal({
                  title: '提示',
                  content: '需要您授权保存图片到相册',
                  success: (modalRes) => {
                    if (modalRes.confirm) {
                      wx.openSetting();
                    }
                  }
                });
              }
              reject(err);
            }
          });
        },
        fail: reject
      });
    });
  }
}
```

```javascript
class SharePosterGenerator {
  constructor() {
    this.pixelRatio = wx.getSystemInfoSync().pixelRatio;
  }

  // 绘制主图（商品图或活动图）
  async drawMainImage(ctx, url, x, y, width, height) {
    return new Promise((resolve, reject) => {
      wx.getImageInfo({
        src: url,
        success: (res) => {
          const img = wx.createCanvasContext ? ctx.canvas.createImage() : null;
          if (!img) {
            resolve();
            return;
          }
          img.onload = () => {
            // 绘制带圆角的图片
            this.drawRoundedImage(ctx, img, x, y, width, height, 16);
            resolve();
          };
          img.onerror = reject;
          img.src = res.path;
        },
        fail: reject
      });
    });
  }

  // 绘制圆角图片
  drawRoundedImage(ctx, img, x, y, width, height, radius) {
    ctx.beginPath();
    ctx.moveTo(x + radius, y);
    ctx.lineTo(x + width - radius, y);
    ctx.arcTo(x + width, y, x + width, y + radius, radius);
    ctx.lineTo(x + width, y + height - radius);
    ctx.arcTo(x + width, y + height, x + width - radius, y + height, radius);
    ctx.lineTo(x + radius, y + height);
    ctx.arcTo(x, y + height, x, y + height - radius, radius);
    ctx.lineTo(x, y + radius);
    ctx.arcTo(x, y, x + radius, y, radius);
    ctx.closePath();
    ctx.clip();
    ctx.drawImage(img, x, y, width, height);
  }

  // 绘制带描边的圆角矩形
  drawRoundedRect(ctx, x, y, width, height, radius, fillColor, strokeColor, lineWidth) {
    ctx.beginPath();
    ctx.moveTo(x + radius, y);
    ctx.lineTo(x + width - radius, y);
    ctx.quadraticCurveTo(x + width, y, x + width, y + radius);
    ctx.lineTo(x + width, y + height - radius);
    ctx.quadraticCurveTo(x + width, y + height, x + width - radius, y + height);
    ctx.lineTo(x + radius, y + height);
    ctx.quadraticCurveTo(x, y + height, x, y + height - radius);
    ctx.lineTo(x, y + radius);
    ctx.quadraticCurveTo(x, y, x + radius, y);
    ctx.closePath();

    if (fillColor) {
      ctx.fillStyle = fillColor;
      ctx.fill();
    }
    if (strokeColor) {
      ctx.strokeStyle = strokeColor;
      ctx.lineWidth = lineWidth || 2;
      ctx.stroke();
    }
  }

  // 绘制文字（支持省略号）
  drawTextEllipsis(ctx, text, x, y, maxWidth, fontSize) {
    let currentText = '';
    for (let i = 0; i < text.length; i++) {
      const testText = currentText + text[i];
      const metrics = ctx.measureText(testText);
      if (metrics.width > maxWidth && i > 0) {
        currentText += '...';
        break;
      }
      currentText = testText;
    }
    ctx.fillText(currentText, x, y);
    return currentText;
  }

      type = 'goods',      // goods | activity | invite
      background = '#ffffff',
      mainImage,
      title,
      subtitle,
      price,
      originalPrice,
      qrcode,
      width = 600,
      height = 960
    } = options;

    // 创建画布
    const canvas = wx.createCanvasContext('shareCanvas');
    const dpr = this.pixelRatio;

    canvas.width = width * dpr;
    canvas.height = height * dpr;
    canvas.scale(dpr, dpr);

    // 绘制背景
    canvas.fillStyle = background;
    canvas.fillRect(0, 0, width, height);

    if (type === 'goods') {
      // 商品海报
      await this.drawGoodsPoster(canvas, {

        title,
        subtitle,
        qrcode,
        width,
        height
      });
    } else if (type === 'invite') {
      // 邀请海报
      await this.drawInvitePoster(canvas, {
        mainImage,
        title,
        qrcode,
        width,
        height
      });
    }

    // 绘制
    await new Promise(resolve => canvas.draw(false, resolve));

    // 导出图片
    const posterPath = await new Promise((resolve, reject) => {
      wx.canvasToTempFilePath({
        canvasId: 'shareCanvas',
        fileType: 'png',
        quality: 1,
        success: resolve,
        fail: reject
      });
    });

    return posterPath;
  }

    // 价格
```

### 6.1 微信支付：wx.requestPayment

微信支付是小程序商业化的核心能力，wx.requestPayment是调起支付的API。

**支付流程：**

```
用户下单 → 服务器创建订单 → 服务器调用微信支付统一下单
        → 返回prepay_id → 小程序调用wx.requestPayment → 用户支付
        → 微信服务器回调服务器 → 订单状态更新
```

**支付实现：**

```javascript
// 支付服务
class PaymentService {
  constructor() {
    this.apiBase = 'https://api.example.com/pay';
  }

  // 创建订单并获取支付参数
  async createOrder(goodsList, address) {
    const res = await wx.request({
      url: `${this.apiBase}/create`,
      method: 'POST',
      data: {
        goodsList,
        address,
        openId: wx.getStorageSync('openId'),
        totalFee: this.calculateTotal(goodsList)
      }
    });

    if (res.data.code !== 0) {
      throw new Error(res.data.message);
    }

    return res.data.data;
  }

  // 调起微信支付
  async requestPayment(payInfo) {
    const {
      timeStamp,
      nonceStr,
      package: packageStr,
      paySign,
      signType = 'MD5'
    } = payInfo;

    return new Promise((resolve, reject) => {
      wx.requestPayment({
        timeStamp,
        nonceStr,
        package: packageStr,
        signType,
        paySign,
        success: (res) => {
          console.log('支付成功', res);
          resolve(res);
        },
        fail: (err) => {
          console.log('支付失败', err);
          // 用户取消也走这里
          if (err.errMsg === 'requestPayment:fail cancel') {
            reject(new Error('cancel'));
          } else {
            reject(new Error(err.errMsg));
          }
        }
      });
    });
  }

  // 支付结果查询
  async queryOrder(orderId) {
    const res = await wx.request({
      url: `${this.apiBase}/query/${orderId}`
    });
    return res.data;
  }

  // 计算订单总价
  calculateTotal(goodsList) {
    return goodsList.reduce((sum, item) => {
      return sum + item.price * item.quantity;
    }, 0);
  }
}

    }
```

### 6.2 授权登录：静默授权

静默授权是指用户无感知的情况下获取用户的基本信息，用于识别用户身份。

**静默授权流程：**

```javascript
// 授权服务
class AuthService {
  constructor() {
    this.storageKey = 'user_auth';
  }

  // 检查登录态是否有效
  isLoginValid() {
    const auth = wx.getStorageSync(this.storageKey);
    if (!auth || !auth.openId) {
      return false;
    }

    // 检查是否过期（7天有效期）
    const expireTime = 7 * 24 * 60 * 60 * 1000;
    if (Date.now() - auth.timestamp > expireTime) {
      return false;
    }

    return true;
  }

  // 静默登录
  async silentLogin() {
    // 如果本地登录态有效，直接返回
    if (this.isLoginValid()) {
      return wx.getStorageSync(this.storageKey);
    }

    // 调用后端接口，使用微信静默登录
    const res = await this.code2Session();

    if (res.openId) {
      const auth = {
        openId: res.openId,
        sessionKey: res.session_key,
        timestamp: Date.now()
      };
      wx.setStorageSync(this.storageKey, auth);
      return auth;
    }

    throw new Error('静默登录失败');
  }

  // 小程序登录Code换取Session
  async code2Session() {
    return new Promise((resolve, reject) => {
      // 获取登录凭证
      wx.login({
        success: async (loginRes) => {
          if (!loginRes.code) {
            reject(new Error('获取code失败'));
            return;
          }

          try {
            // 发送到服务器换取openId
            const res = await wx.request({
              url: 'https://api.example.com/auth/code2session',
              method: 'POST',
              data: { code: loginRes.code }
            });

            resolve(res.data);
          } catch (err) {
            reject(err);
          }
        },
        fail: reject
      });
    });
  }

  // 主动登录（需要用户授权）
  async login() {
    return new Promise((resolve, reject) => {
      // 1. 获取用户信息
      wx.getUserProfile({
        desc: '用于完善会员资料',
        success: async (userRes) => {
          try {
            // 2. 静默登录获取openId
            const auth = await this.silentLogin();

            // 3. 上报用户信息到服务器
            await wx.request({
              url: 'https://api.example.com/auth/updateUser',
              method: 'POST',
              data: {
                openId: auth.openId,
                userInfo: userRes.userInfo
              }
            });

            // 4. 保存用户信息
            wx.setStorageSync('userInfo', userRes.userInfo);

            resolve({
              ...auth,
              userInfo: userRes.userInfo
            });

          } catch (err) {
            reject(err);
          }
        },
        fail: (err) => {
          if (err.errMsg.includes('cancel')) {
            reject(new Error('cancel'));
          } else {
            reject(err);
          }
        }
      });
    });
  }
}

const authService = new AuthService();

// 应用启动时自动登录
App({
  onLaunch: async function() {
    // 自动静默登录
    try {
      const auth = await authService.silentLogin();
      console.log('静默登录成功', auth.openId);
    } catch (err) {
      console.log('静默登录失败', err);
    }
  }
});

// 页面中使用
Page({
  data: {
    userInfo: null,
    isLogin: false
  },

  onLoad: function() {
    this.checkLogin();
  },

  checkLogin: function() {
    const userInfo = wx.getStorageSync('userInfo');
    const isLogin = authService.isLoginValid();

    this.setData({
      userInfo,
      isLogin
    });
  },

  // 用户点击登录按钮
  async onLogin() {
    wx.showLoading({ title: '登录中...' });

    try {
      const result = await authService.login();
      this.setData({
        userInfo: result.userInfo,
        isLogin: true
      });
      wx.showToast({ title: '登录成功' });
    } catch (err) {
      if (err.message !== 'cancel') {
        wx.showToast({ title: '登录失败', icon: 'none' });
      }
    } finally {
      wx.hideLoading();
    }
  }
});
```

```javascript
// WxBizDataCrypt 实现
class WxBizDataCrypt {
  constructor(appId, sessionKey) {
    this.appId = appId;
    this.sessionKey = Buffer.from(sessionKey, 'base64');
  }

  decryptData(encryptedData, iv) {
    // 解码
    const encryptedString = Buffer.from(encryptedData, 'base64');
    const ivString = Buffer.from(iv, 'base64');

    // AES-128-CBC 解密
    const decipher = crypto.createDecipheriv(
      'aes-128-cbc',
      this.sessionKey,
      ivString
    );

    // 设置自动填充
    decipher.setAutoPadding(true);

    // 解密
    let decoded = decipher.update(encryptedString);
    decoded = Buffer.concat([decoded, decipher.final()]);

    // 解析JSON
    const data = JSON.parse(decoded.toString('utf8'));

    // 验证watermark
    if (data.watermark.appid !== this.appId) {
      throw new Error('Invalid watermark');
    }

    return data;
  }
}
```

### 6.4 我的思考：支付是商业闭环

小程序支付绝不仅仅是调用一个API那么简单，它涉及用户信任、资金安全、合规等多个维度。

**支付体验优化要点：**

支付失败后的引导至关重要。当支付失败时，应该清晰告知用户失败原因（余额不足、网络问题、风控拦截等），并提供明确的解决路径。余额不足时，可以引导用户充值；网络问题时，可以提供重试按钮；风控拦截时，应该建议用户稍后再试或联系客服。

支付安全需要多重验证。客户端应该对订单金额进行签名校验，防止篡改；服务端应该验证订单金额与实际金额一致，防止0.01元购买高价商品；风控系统应该监控异常支付行为，如短时间内大量小额支付。

```javascript
// 订阅消息服务
class SubscribeService {
  constructor() {
    this.templateIds = {
      orderPaid: 'TEMPLATE_ID_ORDER_PAID',      // 订单支付成功
      orderShipped: 'TEMPLATE_ID_ORDER_SHIPPED', // 订单发货
      deliveryRemind: 'TEMPLATE_ID_DELIVERY',   // 收货提醒
      groupSuccess: 'TEMPLATE_ID_GROUP_SUCCESS', // 拼团成功
      refundNotice: 'TEMPLATE_ID_REFUND'         // 退款通知
    };
  }

  // 请求订阅消息授权
  async requestSubscribe(templateKey) {
    const templateId = this.templateIds[templateKey];
    if (!templateId) {
      throw new Error('未知的模板');
    }

    return new Promise((resolve, reject) => {
      wx.requestSubscribeMessage({
        tmplIds: [templateId],
        success: (res) => {
          if (res[templateId] === 'accept') {
            resolve('accepted');
          } else if (res[templateId] === 'reject') {
            resolve('rejected');
          } else if (res[templateId] === 'ban') {
            reject(new Error('消息发送权限被封禁'));
          }
        },
        fail: (err) => {
          reject(err);
        }
      });
    });
  }

  // 发送订阅消息（需服务端调用微信API）
  async sendSubscribeMessage(params) {
    const { templateKey, openId, data, page, miniprogramState } = params;

    const templateId = this.templateIds[templateKey];
    if (!templateId) {
      throw new Error('未知的模板');
    }

    // 发送到自己的服务器，由服务器调用微信发送消息API
    const res = await wx.request({
      url: 'https://api.example.com/message/subscribe',
      method: 'POST',
      data: {
        templateId,
        openId,
        data,
        page,
        miniprogramState: miniprogramState || 'developer' // 小程序版本
      }
    });

    return res.data;
  }

  // 批量发送（用户主动订阅多个模板）
  async requestMultipleSubscribes(keys) {
    const templateIds = keys.map(key => this.templateIds[key]).filter(Boolean);

    return new Promise((resolve, reject) => {
      wx.requestSubscribeMessage({
        tmplIds: templateIds,
        success: (res) => {
          const results = {};
          for (const key of keys) {
            const templateId = this.templateIds[key];
            results[key] = res[templateId] || 'unknown';
          }
          resolve(results);
        },
        fail: reject
      });
    });
  }
}
```

### 7.3 消息类型：即时、延时

根据业务场景选择合适的推送时机和方式。

```javascript
// 消息推送服务
class PushService {
  constructor() {
    this.queue = [];         // 消息队列
    this.processing = false;  // 是否正在处理
  }

  // 即时推送（用户操作触发）
  async pushInstant(templateKey, data) {
    // 先请求订阅权限
    const result = await subscribeService.requestSubscribe(templateKey);

    if (result === 'accepted') {
      // 立即发送
      await subscribeService.sendSubscribeMessage(data);
    }
  }

  // 延时推送（加入队列，定期发送）
  pushDelay(templateKey, data, delayTime = 60000) {
    const message = {
      templateKey,
      data,
      executeTime: Date.now() + delayTime
    };

    this.queue.push(message);
    this.scheduleProcess();
  }

  // 批量发送
  async pushBatch(templateKey, openIds, data) {
    // 分批发送，避免频率限制
    const batchSize = 100;
    const batches = [];

    for (let i = 0; i < openIds.length; i += batchSize) {
      batches.push(openIds.slice(i, i + batchSize));
    }

    for (const batch of batches) {
      for (const openId of batch) {
        try {
          await subscribeService.sendSubscribeMessage({
            ...data,
            openId
          });
        } catch (err) {
          console.error(`发送失败: ${openId}`, err);
        }
      }
      // 批次间延迟
      await this.sleep(1000);
    }
  }

  // 计划处理队列
  scheduleProcess() {
    if (this.processing) return;

    this.processing = true;
    this.processQueue();
  }

  async processQueue() {
    const now = Date.now();

    // 找出到期的消息
    const toProcess = this.queue.filter(msg => msg.executeTime <= now);
    this.queue = this.queue.filter(msg => msg.executeTime > now);

    // 处理消息
    for (const message of toProcess) {
      try {
        await subscribeService.sendSubscribeMessage(message.data);
      } catch (err) {
        console.error('发送失败', err);
      }
    }

    // 继续处理
    if (this.queue.length > 0) {
      const nextTime = Math.min(...this.queue.map(msg => msg.executeTime));
      setTimeout(() => this.processQueue(), nextTime - now);
    } else {
      this.processing = false;
    }
  }

  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

const pushService = new PushService();

// 订单完成后延时提醒收货
Page({
  data: {
    orderId: ''
  },

  onOrderComplete(orderId) {

      openId: this.data.openId,

        orderId: orderId
      }

        orderId: orderId
```

```javascript
  getLocation(forceRefresh = false) {
    // 缓存5分钟内的位置
    if (!forceRefresh &&
        this.lastLocation &&
        Date.now() - this.lastLocationTime < 5 * 60 * 1000) {
      return Promise.resolve(this.lastLocation);
    }

    return new Promise((resolve, reject) => {
      wx.getLocation({
        type: 'gcj02',  // 国测局坐标，腾讯地图使用的坐标
        altitude: false,
        success: (res) => {
          this.lastLocation = {
            latitude: res.latitude,
            longitude: res.longitude,
            accuracy: res.accuracy,
            speed: res.speed,
            timestamp: Date.now()
          };
          this.lastLocationTime = Date.now();
          resolve(this.lastLocation);
        },
        fail: (err) => {
          // 根据错误类型提供友好的错误提示
          if (err.errMsg.includes('auth deny')) {
            wx.showModal({
              title: '定位权限',
              content: '需要获取您的位置信息，请在设置中开启',
              success: (res) => {
                if (res.confirm) {
                  wx.openSetting();
                }
              }
            });
          }
          reject(err);
        }
      });
    });
  }

  // 计算两点之间的距离（单位：米）
  calculateDistance(lat1, lon1, lat2, lon2) {
    const R = 6371000; // 地球半径（米）
    const dLat = this.toRad(lat2 - lat1);
    const dLon = this.toRad(lon2 - lon1);

    const a = Math.sin(dLat / 2) * Math.sin(dLat / 2) +
              Math.cos(this.toRad(lat1)) * Math.cos(this.toRad(lat2)) *
              Math.sin(dLon / 2) * Math.sin(dLon / 2);

    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
    return R * c;
  }

  toRad(deg) {
    return deg * (Math.PI / 180);
  }

  // 格式化距离显示
  formatDistance(meters) {
    if (meters < 1000) {
      return `${Math.round(meters)}m`;
    } else {
      return `${(meters / 1000).toFixed(1)}km`;
    }
  }
}
```

### 9.1 音频播放：innerAudioContext

innerAudioContext是用于播放音频的API，适合背景音乐、语音播报等场景。

```javascript
// 音频服务
class AudioService {
  constructor() {
    this.audio = null;
    this.playlist = [];
    this.currentIndex = 0;
    this.onendedCallback = null;
  }

  // 初始化
  init() {
    if (!this.audio) {
      this.audio = wx.createInnerAudioContext();

      // 音频自然结束
      this.audio.onEnded(() => {
        this.next();
        if (this.onendedCallback) {
          this.onendedCallback();
        }
      });

      // 音频播放出错
      this.audio.onError((err) => {
        console.error('音频播放错误', err);
        wx.showToast({ title: '音频播放失败', icon: 'none' });
      });
    }
  }

  // 播放
  play(src) {
    this.init();
    this.audio.src = src;
    this.audio.play();
  }

  // 暂停
  pause() {
    if (this.audio) {
      this.audio.pause();
    }
  }

  // 停止
  stop() {
    if (this.audio) {
      this.audio.stop();
    }
  }

  // 跳转到指定位置
  seek(time) {
    if (this.audio) {
      this.audio.seek(time);
    }
  }

  // 设置音量
  setVolume(volume) {
    if (this.audio) {
      this.audio.volume = Math.max(0, Math.min(1, volume));
    }
  }

  // 获取当前播放时间
  getCurrentTime() {
    return this.audio ? this.audio.currentTime : 0;
  }

  // 获取总时长
  getDuration() {
    return this.audio ? this.audio.duration : 0;
  }

  // 播放列表
  playList(list, startIndex = 0) {
    this.playlist = list;
    this.currentIndex = startIndex;
    this.play(list[startIndex].src);
  }

  // 下一首
  next() {
    if (this.playlist.length === 0) return;

    this.currentIndex = (this.currentIndex + 1) % this.playlist.length;
    this.play(this.playlist[this.currentIndex].src);
  }

  // 上一首

  }

  // 销毁
  destroy() {
    if (this.audio) {
      this.audio.destroy();
      this.audio = null;
    }
  }
}
```

```javascript
class TRTCService {
  constructor() {

    this.localStream = null;

  }

  // 初始化TRTC
  async init(params) {
    const {
      userId,
      roomId,
      sdkAppId,
      userSig,
      scene = 'video_call' // video_call | live
    } = params;

    // 引入TRTC SDK（需在app.json中配置）

    this.trtc = await TRTCCloud.create();
    this.userId = userId;
    this.roomId = roomId;

    // 设置事件监听
    this.setEventListeners();

    // 进入房间

      userId,
      userSig,
      roomId,
      scene
    });
  }

  // 设置事件监听
  setEventListeners() {
    // 远端用户进房
    this.trtc.on(TRTCCloud.EVENT.REMOTE_USER_ENTER, (userId) => {
      console.log('远端用户进房', userId);
    });

    // 远端用户退房
    this.trtc.on(TRTCCloud.EVENT.REMOTE_USER_EXIT, (userId) => {
      console.log('远端用户退房', userId);
      this.remoteStreams.delete(userId);
    });

    // 远端用户视频流首帧
    this.trtc.on(TRTCCloud.EVENT.REMOTE_VIDEO_FIRST_FRAME, (userId) => {
      console.log('远端视频首帧', userId);
    });

    // 远端用户音频首帧
    this.trtc.on(TRTCCloud.EVENT.REMOTE_AUDIO_FIRST_FRAME, (userId) => {
      console.log('远端音频首帧', userId);
    });

    // 网络状态变化
    this.trtc.on(TRTCCloud.EVENT.NETWORK_QUALITY, (networkInfo) => {
      console.log('网络质量', networkInfo);
    });

    // 错误
    this.trtc.on(TRTCCloud.EVENT.ERROR, (error) => {
      console.error('TRTC错误', error);
    });
  }

  // 进入房间
  async enterRoom(params) {
    return new Promise((resolve, reject) => {
      this.trtc.enterRoom(params, (err) => {
        if (err) {
          reject(err);
        } else {
          resolve();
        }
      });
    });
  }

  // 退出房间
  async exitRoom() {
    if (this.trtc) {
      await this.trtc.exitRoom();
      this.trtc.destroy();
      this.trtc = null;
    }
  }

  // 开启本地视频
  async startLocalVideo(params = {}) {
    const {
      viewId,       // 渲染的video组件id
      quality = 1,   // 1: 360p, 2: 540p, 3: 720p
      bitrate = 600
    } = params;

    this.localStream = this.trtc.startLocalVideo({
      viewId,
      quality,
      bitrate
    });

    return this.localStream;
  }

  // 停止本地视频
  stopLocalVideo() {
    if (this.trtc) {
      this.trtc.stopLocalVideo();
      this.localStream = null;
    }
  }

  // 开启本地音频
  startLocalAudio() {
    if (this.trtc) {
      this.trtc.startLocalAudio();
    }
  }

  // 停止本地音频
  stopLocalAudio() {
    if (this.trtc) {
      this.trtc.stopLocalAudio();
    }
  }

  // 订阅远端视频流
  async subscribeRemoteVideo(userId, viewId) {
    if (this.trtc) {
      await this.trtc.startRemoteView({
        userId,
        viewId,
        streamType: 'main' // main: 主码流, sub: 辅码流
      });
      this.remoteStreams.set(userId, viewId);
    }
  }

  // 取消订阅
  unsubscribeRemoteVideo(userId) {
    if (this.trtc) {
      this.trtc.stopRemoteView({ userId });
      this.remoteStreams.delete(userId);
    }
  }

  // 切换摄像头
  switchCamera() {
    if (this.trtc) {
      this.trtc.switchCamera();
    }
  }

  // 设置网络质量
  setNetworkQosParam(params) {
    if (this.trtc) {
      this.trtc.setNetworkQosParam({
        preference: params.preference || 'smooth' // clear: 流畅, smooth: 卡顿
      });
    }
  }
}
```

```javascript
class VoiceChatService {
  constructor() {
    this.trtc = null;
    this.roomId = null;
    this.userId = null;
    this.members = new Map();
    this.isMuted = false;
    this.onMemberChangeCallback = null;
  }

  async initRoom(roomId, userId) {
    this.roomId = roomId;
    this.userId = userId;

    // 创建TRTC实例
    const { TRTCCloud } = requirePlugin('trtc');
    this.trtc = await TRTCCloud.create();

    // 设置事件监听
    this.setupEventListeners();

    // 进入房间
    await this.enterRoom();
  }

  // 设置事件监听
  setupEventListeners() {
    // 用户进入
    this.trtc.on(TRTCCloud.EVENT.REMOTE_USER_ENTER, (userId) => {
      console.log('用户进入', userId);
      this.members.set(userId, {
        userId,
        isSpeaking: false,
        isMuted: false
      });
      if (this.onMemberChangeCallback) {
        this.onMemberChangeCallback(Array.from(this.members.values()));
      }
    });

    // 用户退出
    this.trtc.on(TRTCCloud.EVENT.REMOTE_USER_EXIT, (userId) => {
      console.log('用户退出', userId);
      this.members.delete(userId);
      if (this.onMemberChangeCallback) {
        this.onMemberChangeCallback(Array.from(this.members.values()));
      }
    });

    // 音频音量
    this.trtc.on(TRTCCloud.EVENT.USER_VOLUME_UPDATE, (userId, volume) => {
      const member = this.members.get(userId);
      if (member) {
        member.isSpeaking = volume > 0;
        if (this.onMemberChangeCallback) {
          this.onMemberChangeCallback(Array.from(this.members.values()));
        }
      }
    });
  }

  // 进入房间
  async enterRoom() {
    const userSig = await this.getUserSig();

    await this.trtc.enterRoom({
      sdkAppId: 1400000000,
      userId: this.userId,
      userSig,
      roomId: this.roomId,
      scene: 'live'
    });

    // 开启麦克风
    await this.trtc.startLocalAudio({
      quality: 1,
      echoCancellation: true,  // 回声消除
      noiseSuppression: true   // 噪音抑制
    });

  // 离开房间
  async exitRoom() {
    if (this.trtc) {
      await this.trtc.exitRoom();
      this.trtc.destroy();
      this.trtc = null;
    }
  }

  // 静音/取消静音
  mute(muted) {
    if (this.trtc) {
      this.trtc.muteLocalAudio(muted);
      this.isMuted = muted;
    }
  }

  // 获取用户签名
  async getUserSig() {
    const res = await wx.request({
      url: 'https://api.example.com/trtc/usersig',
      data: { userId: this.userId }
    });
    return res.data.userSig;
  }

  // 监听成员变化
  onMemberChange(callback) {
    this.onMemberChangeCallback = callback;
  }
}

      });
    });
```

```javascript
  getRunData() {
    return new Promise((resolve, reject) => {
      wx.getWeRunData({
        success: async (res) => {
          try {
            // 解密数据
            const decryptRes = await this.decryptData(res);
            resolve(decryptRes);
          } catch (err) {
            reject(err);
          }
        },
        fail: reject
      });
    });
  }

  // 解密微信运动数据
  async decryptData(encryptedData) {

    const res = await wx.request({
      url: 'https://api.example.com/werun/decrypt',
      method: 'POST',
      data: {
        encryptedData: encryptedData.encryptedData,
        iv: encryptedData.iv
      }
    });

    return res.data;
  }

  // 同步步数到服务器
  async syncStepCount() {
    try {
      // 检查授权
      const supported = await this.checkSupport();
      if (!supported) {
        const authorized = await this.authorize();
        if (!authorized) {
          throw new Error('未授权');
        }
      }

      // 获取步数数据
      const runData = await this.getRunData();

      // 上报到服务器
      await wx.request({
        url: 'https://api.example.com/werun/sync',
        method: 'POST',
        data: {
          stepInfoList: runData.stepInfoList
        }
      });

      return runData.stepInfoList;

    } catch (err) {
      console.error('同步步数失败', err);
      throw err;
    }
  }

    const data = await this.syncStepCount();

      const today = data[data.length - 1];
```

```javascript
  constructor() {

          } else {
            reject(new Error('蓝牙适配器状态异常'));
          }
        }
      });
    });
  }

  // 搜索蓝牙设备
  searchDevices() {
    return new Promise((resolve, reject) => {
      // 开始搜索
      wx.startBluetoothDevicesDiscovery({
        services: [], // 指定服务UUID，不指定则搜索所有
        success: (res) => {
          console.log('开始搜索设备');
        },
        fail: reject
      });

      // 监听发现设备
      wx.onBluetoothDeviceFound((res) => {
        const devices = res.devices;
        console.log('发现设备', devices);
      });

      // 5秒后停止搜索
      setTimeout(() => {
        wx.stopBluetoothDevicesDiscovery({
          success: () => console.log('停止搜索')
        });
        resolve();
      }, 5000);
    });
  }

  // 连接设备
  connect(deviceId) {
    return new Promise((resolve, reject) => {
      wx.createBLEConnection({
        deviceId,
        success: () => {
          this.deviceId = deviceId;
          this.isConnected = true;
          this.getDeviceServices(deviceId).then(resolve).catch(reject);
        },
        fail: reject
      });
    });
  }

  // 获取设备服务
  async getDeviceServices(deviceId) {
    const res = await wx.getBLEDeviceServices({ deviceId });

    // 找到打印服务
    const printService = res.services.find(s =>
      s.uuid.includes('4953') // 常见蓝牙打印机的服务UUID
    );

    if (printService) {
      this.serviceId = printService.uuid;
      await this.getCharacteristics(deviceId, printService.uuid);
    }
  }

  // 获取服务特征
  async getCharacteristics(deviceId, serviceId) {
    const res = await wx.getBLEDeviceCharacteristics({
      deviceId,
      serviceId
    });

    // 找到写特征
    const writeChar = res.characteristics.find(c =>
      c.properties.write || c.properties.writeNoResponse
    );

    if (writeChar) {
      this.characteristicId = writeChar.uuid;
    }
  }

  // 打印文本
  printText(text) {
    if (!this.isConnected) {
      throw new Error('设备未连接');
    }

    // 将文本转换为字节流（ESC/POS指令）
    const encoder = new TextEncoder();
    const data = encoder.encode(text);

    // 发送打印数据
    return wx.writeBLECharacteristicValue({
      deviceId: this.deviceId,
      serviceId: this.serviceId,
      characteristicId: this.characteristicId,
      value: data.buffer,
      success: () => console.log('打印成功'),
      fail: (err) => console.error('打印失败', err)
    });
  }

  // 打印小票
  printReceipt(receiptData) {
    // ESC/POS指令
    const commands = [];

    // 初始化打印机
    commands.push(0x1B, 0x40);

    // 设置居中
    commands.push(0x1B, 0x61, 0x01);

    // 打印标题
    commands.push(...this.textToBytes(receiptData.title));
    commands.push(0x0A);

    // 设置左对齐
    commands.push(0x1B, 0x61, 0x00);

    // 打印分割线
    commands.push(...this.textToBytes('--------------------------------'));
    commands.push(0x0A);

    // 打印商品明细
    for (const item of receiptData.items) {
      const line = this.formatLine(item.name, item.price);
      commands.push(...this.textToBytes(line));
      commands.push(0x0A);
    }

    // 分割线
    commands.push(...this.textToBytes('--------------------------------'));
    commands.push(0x0A);

    // 打印总计
    const totalLine = this.formatLine('合计', receiptData.total);
    commands.push(...this.textToBytes(totalLine));
    commands.push(0x0A);

    // 切纸指令
    commands.push(0x1D, 0x56, 0x00);

    // 发送数据
    const buffer = new Uint8Array(commands).buffer;
    wx.writeBLECharacteristicValue({
      deviceId: this.deviceId,
      serviceId: this.serviceId,
      characteristicId: this.characteristicId,
      value: buffer
    });
  }

  // 文本转字节
  textToBytes(text) {
    const encoder = new TextEncoder();
    return Array.from(encoder.encode(text));
  }

  // 格式化行
  formatLine(left, right) {
    const spaces = 32 - left.length - right.length;
    return left + ' '.repeat(Math.max(1, spaces)) + right;
  }

  // 断开连接
  disconnect() {
    if (this.deviceId) {
      wx.closeBLEConnection({ deviceId: this.deviceId });
      this.deviceId = null;
      this.isConnected = false;
    }
  }
}
```

### 10.5 我的思考：开放能力扩展边界

小程序的开放能力让开发者可以充分利用微信生态的优势，但也需要谨慎使用。

**开放能力的使用原则：**

尊重用户隐私是第一位。每次请求用户授权时，应该明确告知用户这个权限的用途，并且只在必要时才请求。如果用户拒绝授权，不应该反复弹窗打扰，而应该提供替代方案。

考虑降级方案。并非所有用户的设备都支持所有开放能力，比如蓝牙打印功能在低端安卓手机上可能不稳定。开发时应该做好能力检测，提供手动输入等替代方案。

### 11.1 数据加密：接口签名

接口签名是防止请求被篡改和伪造的核心手段。

```javascript
// 签名服务
class SignService {
  constructor() {
    this.appSecret = 'your-app-secret'; // 应用密钥
    this.appId = 'your-app-id';         // 应用ID
  }

  // 生成签名
  generateSign(params) {
    // 1. 排序参数
    const sortedKeys = Object.keys(params).sort();
    const sortedParams = {};

    sortedKeys.forEach(key => {
      sortedParams[key] = params[key];
    });

    // 2. 拼接参数字符串
    const paramString = sortedKeys
      .map(key => `${key}=${params[key]}`)
      .join('&');

    // 3. 拼接密钥
    const signString = `${this.appSecret}${paramString}${this.appSecret}`;

    // 4. MD5哈希
    return this.md5(signString);
  }

  // MD5实现
  md5(string) {
    function safeAdd(x, y) {
      const lsw = (x & 0xffff) + (y & 0xffff);
      const msw = (x >> 16) + (y >> 16) + (lsw >> 16);
      return (msw << 16) | (lsw & 0xffff);
    }

    function bitRotateLeft(num, cnt) {
      return (num << cnt) | (num >>> (32 - cnt));
    }

    function md5cmn(q, a, b, x, s, t) {
      return safeAdd(bitRotateLeft(safeAdd(safeAdd(a, q), safeAdd(x, t)), s), b);
    }

    function md5ff(a, b, c, d, x, s, t) {
      return md5cmn((b & c) | ((~b) & d), a, b, x, s, t);
    }

    function md5gg(a, b, c, d, x, s, t) {
      return md5cmn((b & d) | (c & (~d)), a, b, x, s, t);
    }

    function md5hh(a, b, c, d, x, s, t) {
      return md5cmn(b ^ c ^ d, a, b, x, s, t);
    }

    function md5ii(a, b, c, d, x, s, t) {
      return md5cmn(c ^ (b | (~d)), a, b, x, s, t);
    }

    // 简化版MD5，实际使用请引入成熟库
    return this.simpleMd5(string);
  }

  simpleMd5(str) {
    // 实际项目中应使用crypto-js或类似的MD5库
    // 这里返回简化版演示
    let hash = 0;
    for (let i = 0; i < str.length; i++) {
      const char = str.charCodeAt(i);
      hash = ((hash << 5) - hash) + char;
      hash = hash & hash;
    }
    return Math.abs(hash).toString(16).padStart(32, '0');
  }

  // 生成请求参数（带签名）
  generateRequestParams(data) {
    const timestamp = Date.now();
    const nonce = this.generateNonce();

    return {
      ...data,
      appId: this.appId,
      timestamp,
      nonce,
      sign: this.generateSign({
        ...data,
        timestamp,
        nonce
      })
    };
  }

  // 生成随机字符串
  generateNonce(length = 16) {
    const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789';
    let result = '';
    for (let i = 0; i < length; i++) {
      result += chars.charAt(Math.floor(Math.random() * chars.length));
    }
    return result;
  }
}

const signService = new SignService();
```

### 11.2 签名算法：时间戳+nonce

时间戳和nonce的组合可以有效防止重放攻击。

```javascript
// 防重放攻击服务
class AntiReplayService {
  constructor() {
    this.cache = new Map();       // 存储已使用的nonce
    this.cacheExpire = 5 * 60 * 1000; // 缓存5分钟
    this.checkInterval = null;
  }

  // 启动过期检查
  startCleanup() {
    this.checkInterval = setInterval(() => {
      const now = Date.now();
      for (const [key, time] of this.cache.entries()) {
        if (now - time > this.cacheExpire) {
          this.cache.delete(key);
        }
      }
    }, this.cacheExpire);
  }

  // 停止清理
  stopCleanup() {
    if (this.checkInterval) {
      clearInterval(this.checkInterval);
    }
  }

  // 检查nonce是否已使用
  isNonceUsed(nonce) {
    return this.cache.has(nonce);
  }

  // 记录nonce
  markNonce(nonce) {
    this.cache.set(nonce, Date.now());
  }

  // 生成带nonce的签名
  generateSign(params) {
    const timestamp = params.timestamp;
    const nonce = params.nonce;

    // 检查nonce是否有效（时间戳±5分钟内）
    const now = Date.now();
    const timeDiff = Math.abs(now - timestamp);

    if (timeDiff > 5 * 60 * 1000) {
      throw new Error('请求已过期');
    }

    // 检查nonce是否已使用
    if (this.isNonceUsed(nonce)) {
      throw new Error('重复请求');
    }

    // 生成签名
    const signString = this.buildSignString(params);
    const sign = this.hash(signString);

    // 记录nonce
    this.markNonce(nonce);

    return sign;
  }

  // 构建签名字符串
  buildSignString(params) {
    const keys = Object.keys(params).sort();
    return keys
      .filter(key => key !== 'sign')
      .map(key => `${key}=${params[key]}`)
      .join('&');
  }

  // 哈希函数
  hash(str) {
    // 实际使用MD5或SHA256
    let hash = 0;
    for (let i = 0; i < str.length; i++) {
      hash = ((hash << 5) - hash) + str.charCodeAt(i);
      hash = hash & hash;
    }
    return Math.abs(hash).toString(16);
  }
}
```

### 11.3 爬虫识别：行为分析

通过分析用户行为识别爬虫和恶意请求。

```javascript
// 风控服务
class RiskControlService {
  constructor() {
    this.userBehaviors = new Map(); // 用户行为记录
    this.riskLevels = {
      NORMAL: 0,
      SUSPICIOUS: 1,
      BLOCKED: 2
    };
  }

  // 记录用户行为
  recordBehavior(userId, action, metadata = {}) {
    if (!this.userBehaviors.has(userId)) {
      this.userBehaviors.set(userId, []);
    }

    const behaviors = this.userBehaviors.get(userId);
    behaviors.push({
      action,
      timestamp: Date.now(),
      metadata
    });

    // 只保留最近100条记录
    if (behaviors.length > 100) {
      behaviors.shift();
    }
  }

  // 评估用户风险等级
  evaluateRisk(userId) {
    const behaviors = this.userBehaviors.get(userId) || [];
    const now = Date.now();

    // 检查请求频率
    const recentBehaviors = behaviors.filter(b => now - b.timestamp < 60000);
    if (recentBehaviors.length > 100) {
      return this.riskLevels.BLOCKED; // 请求过于频繁
    }

    // 检查是否有规律性访问
    const timeGaps = this.calculateTimeGaps(recentBehaviors);
    if (this.hasRegularPattern(timeGaps)) {
      return this.riskLevels.SUSPICIOUS; // 可能是机器访问
    }

    // 检查是否访问了大量不存在的资源
    const notFoundCount = behaviors.filter(b =>
      b.action === 'NOT_FOUND' &&
      now - b.timestamp < 300000
    ).length;

    if (notFoundCount > 20) {
      return this.riskLevels.SUSPICIOUS; // 可能是爬虫
    }

    return this.riskLevels.NORMAL;
  }

  // 计算时间间隔
  calculateTimeGaps(behaviors) {
    if (behaviors.length < 2) return [];

    const sorted = [...behaviors].sort((a, b) => a.timestamp - b.timestamp);
    const gaps = [];

    for (let i = 1; i < sorted.length; i++) {
      gaps.push(sorted[i].timestamp - sorted[i - 1].timestamp);
    }

    return gaps;
  }

  // 检测是否有规律模式
  hasRegularPattern(gaps) {
    if (gaps.length < 10) return false;

    // 计算平均间隔
    const avg = gaps.reduce((a, b) => a + b, 0) / gaps.length;

    // 检查是否有大量间隔接近平均值的请求
    const similarCount = gaps.filter(gap =>
      Math.abs(gap - avg) < avg * 0.1
    ).length;

    // 如果超过80%的请求间隔都接近平均值，认为有规律
    return similarCount / gaps.length > 0.8;
  }

  // 获取风险等级描述
  getRiskMessage(level) {
    switch (level) {
      case this.riskLevels.SUSPICIOUS:
        return '检测到异常行为，请稍后重试';
      case this.riskLevels.BLOCKED:
        return '请求过于频繁，请稍后再试';
      default:
        return null;
    }
  }
}

const riskService = new RiskControlService();

// 中间件：在请求前检查
function checkRiskBeforeRequest(userId) {
  const riskLevel = riskService.evaluateRisk(userId);
  const message = riskService.getRiskMessage(riskLevel);

  if (riskLevel >= riskService.riskLevels.SUSPICIOUS) {
    return { allowed: false, message };
  }

  return { allowed: true };
}

// 页面中使用
Page({
  data: {
    isBlocked: false,
    blockMessage: ''
  },

  // 封装的请求方法
  async safeRequest(url, data = {}) {
    const userId = wx.getStorageSync('userId');

    // 风控检查
    const check = checkRiskBeforeRequest(userId);
    if (!check.allowed) {
      this.setData({
        isBlocked: true,
        blockMessage: check.message
      });
      return;
    }

    // 记录行为
    riskService.recordBehavior(userId, 'API_REQUEST', { url });

    // 发送请求
    return wx.request({ url, data });
  }
});
```

```javascript
// 频率限制服务
class RateLimitService {
  constructor() {
    this.limits = {
      // 接口调用限制
      api: {
        maxCount: 100,      // 100次
        windowMs: 60000     // 1分钟内
      },
      // 搜索限制
      search: {
        maxCount: 20,
        windowMs: 60000
      },
      // 评论限制
      comment: {
        maxCount: 10,
        windowMs: 60000
      },
      // 点赞限制
      like: {
        maxCount: 50,
        windowMs: 60000
      }
    };

    this.requests = new Map();
  }

  // 检查是否超过限制
  isLimited(userId, action) {
    const limit = this.limits[action];
    if (!limit) return false;

    const key = `${userId}:${action}`;
    const now = Date.now();

    if (!this.requests.has(key)) {
      this.requests.set(key, []);
    }

    const requests = this.requests.get(key);

    // 清理过期的请求记录
    const validRequests = requests.filter(
      time => now - time < limit.windowMs
    );

    if (validRequests.length >= limit.maxCount) {
      return true;
    }

    // 记录新请求
    validRequests.push(now);
    this.requests.set(key, validRequests);

    return false;
  }

  // 获取剩余请求次数
  getRemainingCount(userId, action) {
    const limit = this.limits[action];
    if (!limit) return Infinity;

    const key = `${userId}:${action}`;
    const now = Date.now();

    const requests = this.requests.get(key) || [];
    const validCount = requests.filter(
      time => now - time < limit.windowMs
    ).length;

    return Math.max(0, limit.maxCount - validCount);
  }

  // 获取重置时间
  getResetTime(userId, action) {
    const limit = this.limits[action];
    if (!limit) return 0;

    const key = `${userId}:${action}`;
    const requests = this.requests.get(key) || [];

    if (requests.length === 0) return 0;

    const oldestRequest = Math.min(...requests);
    return oldestRequest + limit.windowMs;
  }

  // 重置限制
  reset(userId, action) {
    const key = `${userId}:${action}`;
    this.requests.delete(key);
  }
}

const rateLimitService = new RateLimitService();

// 页面中使用
Page({
  data: {
    canLike: true,
    likeCount: 0
  },

  onLike() {
    const userId = wx.getStorageSync('userId');

    if (rateLimitService.isLimited(userId, 'like')) {
      const remaining = rateLimitService.getRemainingCount(userId, 'like');
      const resetTime = rateLimitService.getResetTime(userId, 'like');

      const seconds = Math.ceil((resetTime - Date.now()) / 1000);

      wx.showToast({
        title: `操作太频繁，请${seconds}秒后重试`,
        icon: 'none'
      });
      return;
    }

    // 执行点赞
    this.doLike();
  },

  async doLike() {
    const userId = wx.getStorageSync('userId');
    riskService.recordBehavior(userId, 'LIKE');

    // 调用API...
    this.setData({ likeCount: this.data.likeCount + 1 });
  }
});
```

### 11.5 我的思考：小程序也要安全

小程序的安全性常常被开发者忽视，这是一个危险的误区。

**安全建设的必要性：**

小程序的代码虽然运行在微信的沙箱环境中，但这不意味着它是绝对安全的。业务接口、用户数据、商业逻辑都运行在后端服务器上，如果接口没有做好签名验证和权限控制，很容易被恶意用户利用。

**常见的安全漏洞：**

水平越权是最常见的问题。如果一个用户可以通过修改ID参数访问其他用户的订单、评论等资源，说明存在越权漏洞。正确的做法是所有查询都基于当前登录用户的ID，而不是用户可控制的参数。

签名验证形同虚设也是常见问题。有些接口虽然要求签名，但签名算法过于简单或者密钥泄露，导致签名验证可以被绕过。

### 12.1 问题：首屏加载3秒

让我们通过一个真实案例来学习如何系统性地优化小程序性能。

**问题背景：**
一个电商小程序的首屏加载时间达到了3秒，用户反馈体验很差。通过性能分析发现问题主要集中在以下几个方面：

1. 代码包体积过大（2.5MB）
2. 图片资源未优化（首屏加载20+张图片）
3. 接口请求串行执行
4. 无骨架屏或加载状态

### 12.2 分析：代码包大、图片多

**详细分析：**

代码包分析发现，2.5MB的包体积主要来自：
- 3个营销小游戏（总共800KB）
- 一套完整的富文本编辑器（400KB）
- 多套主题配置资源（300KB）
- 公共库代码（600KB，未做tree-shaking）

图片资源分析：
- 首屏有20+张图片
- 商品图没有做尺寸适配（大图直接显示小缩略图）
- 没有懒加载
- 没有CDN加速

接口请求分析：
- 首页加载顺序：先获取用户信息 → 再获取首页配置 → 再获取商品列表
- 3个接口串行执行，总耗时1.5秒

### 12.5 我的思考：优化要系统性

性能优化不是一蹴而就的事情，也不是单一维度的改进。

**系统性优化的思路：**

从用户视角出发，性能优化的最终目标是让用户感到"快"和"流畅"。3秒和1秒的差距在技术上只有2秒，但用户体验的差距却是巨大的。用户研究表明，超过3秒的加载时间会导致大量用户流失，而1秒内的响应则让用户感到应用是"即时"的。

分层优化才能事半功倍。网络层面的优化（CDN、缓存、压缩）是基础设施，决定了理论最快速度。代码层面的优化（分包、懒加载、算法）是执行效率，决定了能否达到理论速度。用户感知层面的优化（骨架屏、进度反馈）是体验包装，让用户感受到的等待更短。

持续监控是优化的保障。优化不是一劳永逸的，随着业务迭代，性能可能会再次退化。建议建立性能监控体系，持续跟踪关键指标，及时发现和解决问题。
