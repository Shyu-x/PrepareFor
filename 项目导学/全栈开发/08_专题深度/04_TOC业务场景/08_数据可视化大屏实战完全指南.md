## 一、数据大屏概述

### 1.1 数据大屏的价值

数据大屏作为数据可视化的重要载体，具有以下核心价值：

**信息密度高**：一块大屏可以同时展示数十个关键指标，数据间相互关联，形成完整的数据画像。传统报表需要翻阅多页才能获取的信息，在大屏上一目了然。

**决策效率提升**：当双十一销售额实时跳动、当生产线良率突然下降时，大屏能够第一时间将异常信息传递给决策者，实现"数据驱动决策"。

**数据讲故事**：大屏不仅仅是数据的堆砌，更是一种数据叙事的艺术。通过合理的布局、配色、动效，将枯燥的数字转化为有说服力的故事。

### 1.4 大屏设计原则

**我的思考**：大屏设计不仅仅是技术实现，更是一种设计艺术。一个好的大屏应该做到"一目了然"，即用户在 3 秒内就能获取核心信息。

大屏设计的核心原则：

**分层展示**：按照信息的重要程度进行分层。顶部展示核心 KPI，中间展示主要图表，底部展示详细数据或次要信息。这种"倒金字塔"结构符合人类的阅读习惯。

**色彩心理**：深色背景（蓝色、紫色、黑色）能够减少屏幕眩光，让图表更加突出。同时，深色背景给人以"高科技感"，适合数据密集型场景。

**动效克制**：动效能够吸引注意力，但过度使用会分散用户注意力。关键数据的变化可以使用动效强调，但装饰性动效应该尽量简化。

**数据真实**：大屏展示的数据必须真实可靠，虚假的繁荣只会误导决策。

## 二、大屏布局系统

### 2.1 响应式布局策略

大屏通常运行在固定分辨率的显示设备上，如 1920x1080、2560x1440、甚至 4K 屏幕。但开发时我们需要在各种尺寸的屏幕上预览，因此需要一套响应式布局策略。

```typescript
/**
 * 大屏布局配置
 */
interface ScreenLayout {
  // 设计稿基准分辨率
  baseWidth: number;
  baseHeight: number;

  // 当前屏幕实际分辨率
  actualWidth: number;
  actualHeight: number;

  // 缩放比例
  scale: number;

  // 边距
  padding: { top: number; right: number; bottom: number; left: number };
}

/**
 * 响应式缩放 Hook
 * 核心原理：根据屏幕分辨率计算缩放比例，保持设计稿的比例关系
 */
function useScreenScale(baseWidth = 1920, baseHeight = 1080) {
  const [layout, setLayout] = useState<ScreenLayout>({
    baseWidth,
    baseHeight,
    actualWidth: baseWidth,
    actualHeight: baseHeight,
    scale: 1,
    padding: { top: 40, right: 40, bottom: 40, left: 40 }
  });

  useEffect(() => {
    // 计算实际缩放比例
    const calculateScale = () => {
      // 使用较长的边作为基准，避免超出屏幕
      const scaleX = window.innerWidth / baseWidth;
      const scaleY = window.innerHeight / baseHeight;
      const scale = Math.min(scaleX, scaleY);

      setLayout(prev => ({
        ...prev,
        actualWidth: window.innerWidth,
        actualHeight: window.innerHeight,
        scale,
        padding: {
          top: Math.round(40 * scale),
          right: Math.round(40 * scale),
          bottom: Math.round(40 * scale),
          left: Math.round(40 * scale)
        }
      }));
    };

    calculateScale();
    window.addEventListener('resize', calculateScale);

    return () => window.removeEventListener('resize', calculateScale);
  }, [baseWidth, baseHeight]);

  return layout;
}
```

### 2.2 栅格系统

栅格系统是大屏布局的骨架。24 列栅格是业界最常用的方案，它能够被 2、3、4、6、8、12 整除，适配大多数布局需求。

```typescript
/**
 * 栅格系统配置
 */
interface GridConfig {
  columns: number;        // 列数，默认 24
  gap: number;            // 栅格间距，单位 px
  rowHeight: number;      // 行高，单位 px
}

/**
 * 栅格组件 Props
 */
interface GridProps {
  config?: Partial<GridConfig>;
  children: React.ReactNode;
  className?: string;
}

/**
 * 24 列栅格系统组件
 *
 * 设计原理：
 * - 将屏幕宽度分为 24 等份
 * - 每个组件可以占据 1-24 列
 * - 组件之间通过 gap 设置间距
 *
 * 使用示例：
 * <Grid>
 *   <GridItem columns={6}>左侧边栏</GridItem>
 *   <GridItem columns={12}>中间主区域</GridItem>
 *   <GridItem columns={6}>右侧边栏</GridItem>
 * </Grid>
 */
const Grid: React.FC<GridProps> = ({ config, children, className }) => {
  const { columns = 24, gap = 16, rowHeight = 60 } = config || {};

  return (
    <div
      className={className}
      style={{
        display: 'grid',
        gridTemplateColumns: `repeat(${columns}, 1fr)`,
        gap: `${gap}px`,
        height: '100%',
        width: '100%'
      }}
    >
      {children}
    </div>
  );
};

/**
 * GridItem：栅格中的单项组件
 *
 * @param columns 占据的列数（1-24）
 * @param rows 占据的行数（可选）
 * @param offset 左侧偏移列数（可选）
 */
interface GridItemProps {
  columns: number;
  rows?: number;
  offset?: number;
  children: React.ReactNode;
  style?: React.CSSProperties;
  className?: string;
}

const GridItem: React.FC<GridItemProps> = ({
  columns,
  rows = 1,
  offset = 0,
  children,
  style,
  className
}) => {
  return (
    <div
      className={className}
      style={{
        gridColumn: `${offset + 1} / span ${columns}`,
        gridRow: `span ${rows}`,
        minHeight: 0,        // 防止内容溢出
        minWidth: 0,        // 防止内容溢出
        ...style
      }}
    >
      {children}
    </div>
  );
};
```

### 2.3 主题系统

大屏主题通常分为暗色主题和亮色主题两大类。暗色主题是大屏的主流选择，原因如下：

- **减少眩光**：在展会或指挥中心，环境光线通常较暗，深色背景能够减少屏幕对眼睛的刺激
- **突出数据**：深色背景上的彩色图表更加醒目，数据对比度更高
- **降低功耗**：对于 LCD 屏幕，深色背景能够降低背光功耗

### 2.5 完整布局方案实战

下面是一个完整的电商数据大屏布局示例：

```typescript
/**
 * 电商数据大屏完整布局
 *
 * 布局结构（1920x1080）：
 *
 * ┌─────────────────────────────────────────────────────────────────┐
 * │                         头部标题区域                              │
 * │  (GMV 实时数据 + 时间)                                          │
 * ├────────────┬───────────────────────────────────┬────────────────┤
 * │            │                                   │                │
 * │   左侧边栏   │           中心主区域              │    右侧边栏     │
 * │  (用户画像)  │  ┌─────────┬─────────┬─────────┐ │  (实时订单)   │
 * │            │  │ 成交趋势  │  品类排行 │  销售占比 │ │              │
 * │            │  ├─────────┴─────────┴─────────┤ │                │
 * │            │  │         全国销售地图         │ │                │
 * │            │  └─────────────────────────────┘ │                │
 * ├────────────┴───────────────────────────────────┴────────────────┤
 * │                         底部数据条                                │
 * │  (物流数据、客服数据、库存数据等)                                  │
 * └─────────────────────────────────────────────────────────────────┘
 */

import React, { useState, useEffect, useRef } from 'react';
import { Grid, GridItem } from './GridSystem';
import { useScreenScale, useTheme } from './hooks';

// KPI 卡片组件
interface KPICardProps {
  title: string;
  value: number;
  unit: string;
  prefix?: string;
  change: number;
  icon?: string;
}

const KPICard: React.FC<KPICardProps> = ({ title, value, unit, prefix, change }) => {
  const theme = useTheme();
  const [displayValue, setDisplayValue] = useState(0);
  const targetValue = value;

  // 数字滚动动画
  useEffect(() => {
    const duration = 1500;
    const startTime = Date.now();
    const startValue = 0;

    const animate = () => {
      const elapsed = Date.now() - startTime;
      const progress = Math.min(elapsed / duration, 1);

      // 使用 easeOutExpo 缓动函数
      const easeProgress = 1 - Math.pow(1 - progress, 4);
      const current = startValue + (targetValue - startValue) * easeProgress;
```

```typescript
const useChartResize = (chartRef: React.RefObject<HTMLDivElement>) => {
  const [dimensions, setDimensions] = useState({ width: 0, height: 0 });

  useEffect(() => {
    if (!chartRef.current) return;

/**
```

```typescript
interface WebSocketConfig {

  reconnect?: boolean;         // 是否自动重连
  reconnectInterval?: number; // 重连间隔（毫秒）

/**
 * WebSocket Hook
 * 管理 WebSocket 连接状态
 */
const useWebSocket = (config: WebSocketConfig) => {
  const {
    url,
    reconnect = true,
    reconnectInterval = 3000,
    heartbeat = true,
    heartbeatInterval = 30000
  } = config;

  const [data, setData] = useState<any>(null);
  const [status, setStatus] = useState<'connecting' | 'connected' | 'disconnected' | 'error'>('disconnected');
  const [error, setError] = useState<Error | null>(null);

  const wsRef = useRef<WebSocket | null>(null);
  const reconnectTimerRef = useRef<number | null>(null);
  const heartbeatTimerRef = useRef<number | null>(null);

  // 清除定时器
  const clearTimers = () => {
    if (reconnectTimerRef.current) {
      clearInterval(reconnectTimerRef.current);
      reconnectTimerRef.current = null;
    }
    if (heartbeatTimerRef.current) {
      clearInterval(heartbeatTimerRef.current);
      heartbeatTimerRef.current = null;
    }
  };

  // 连接 WebSocket
  const connect = useCallback(() => {
    if (wsRef.current?.readyState === WebSocket.OPEN) return;

    setStatus('connecting');
    setError(null);

    try {
      wsRef.current = new WebSocket(url);

      wsRef.current.onopen = () => {
        setStatus('connected');
        console.log('WebSocket 连接成功');

        // 启动心跳
        if (heartbeat) {
          heartbeatTimerRef.current = window.setInterval(() => {
            if (wsRef.current?.readyState === WebSocket.OPEN) {
              wsRef.current.send(JSON.stringify({ type: 'heartbeat' }));
            }
          }, heartbeatInterval);
        }
      };

      wsRef.current.onmessage = (event) => {
        try {
          const message = JSON.parse(event.data);
          handleMessage(message);
        } catch (e) {
          console.error('消息解析失败:', e);
        }
      };

      wsRef.current.onerror = (event) => {
        setStatus('error');
        setError(new Error('WebSocket 连接错误'));
        console.error('WebSocket 错误:', event);
      };

      wsRef.current.onclose = (event) => {
        setStatus('disconnected');
        console.log('WebSocket 关闭:', event.code, event.reason);

        // 清除心跳
        if (heartbeatTimerRef.current) {
          clearInterval(heartbeatTimerRef.current);
          heartbeatTimerRef.current = null;
        }

        // 自动重连
        if (reconnect && event.code !== 1000) {
          reconnectTimerRef.current = window.setTimeout(() => {
            connect();
          }, reconnectInterval);
        }
      };
    } catch (e) {
      setStatus('error');
      setError(e as Error);
    }
  }, [url, reconnect, reconnectInterval, heartbeat, heartbeatInterval]);

  // 处理消息
  const handleMessage = useCallback((message: any) => {
    // 根据消息类型处理
    switch (message.type) {
      case 'data':
        setData(message.payload);
        break;
      case 'kpi':
        // 处理 KPI 数据更新
        setData(prev => ({
          ...prev,
          kpi: message.payload
        }));
        break;
      case 'chart':
        // 处理图表数据更新
        setData(prev => ({
          ...prev,
          charts: {
            ...prev?.charts,
            [message.chartId]: message.payload
          }
        }));
        break;
      case 'heartbeat':
        // 心跳响应
        break;
      default:
        console.warn('未知消息类型:', message.type);
    }
  }, []);

  // 断开连接
  const disconnect = useCallback(() => {
    clearTimers();
    if (wsRef.current) {
      wsRef.current.close(1000, '主动关闭');
      wsRef.current = null;
    }
  }, []);

  // 发送消息
  const send = useCallback((data: any) => {
    if (wsRef.current?.readyState === WebSocket.OPEN) {
      wsRef.current.send(JSON.stringify(data));
    } else {
      console.warn('WebSocket 未连接，无法发送消息');
    }
  }, []);

  // 初始化连接
  useEffect(() => {
    connect();
    return () => {
      disconnect();
    };
  }, [connect, disconnect]);

  return {
    data,
    status,
    error,
    send,
    connect,
    disconnect
  };
};
```

```typescript
const useDataChange = <T,>(data: T, compare?: (prev: T, next: T) => boolean) => {
  const prevDataRef = useRef<T>(data);
  const [changedData, setChangedData] = useState<T>(data);

  useEffect(() => {
    const isEqual = compare
      ? compare(prevDataRef.current, data)
      : deepEqual(prevDataRef.current, data);

    if (!isEqual) {
      setChangedData(data);
      prevDataRef.current = data;
    }
  }, [data, compare]);

  return changedData;
};

/**
 * 深度相等比较
 */
const deepEqual = (a: any, b: any): boolean => {
  if (a === b) return true;
  if (a == null || b == null) return false;
  if (typeof a !== 'object' || typeof b !== 'object') return false;

  const keysA = Object.keys(a);
  const keysB = Object.keys(b);

  if (keysA.length !== keysB.length) return false;

  for (const key of keysA) {
    if (!keysB.includes(key)) return false;
    if (!deepEqual(a[key], b[key])) return false;
  }

  return true;
};

/**
 * 数据 diff 工具
 */
const diffData = <T extends Record<string, any>>(
  oldData: T,
  newData: T
): { added: Partial<T>; removed: Partial<T>; changed: Partial<T> } => {
  const added: Partial<T> = {};
  const removed: Partial<T> = {};
  const changed: Partial<T> = {};

  // 检查新增和变化的字段
  for (const key of Object.keys(newData)) {
    if (!(key in oldData)) {
      added[key] = newData[key];
    } else if (oldData[key] !== newData[key]) {
      changed[key] = newData[key];
    }
  }

  // 检查删除的字段
  for (const key of Object.keys(oldData)) {
    if (!(key in newData)) {
      removed[key] = oldData[key];
    }
  }

  return { added, removed, changed };
};

/**
```
```typescript
 * 数字滚动 Hook
 */
const useCountUp = (config: CountUpProps) => {
  const {
    end,
    start = 0,
    duration = 2000,
    decimals = 0,
    separator = ',',
    useGrouping = true,
    easingFn = easeOutExpo,
    onEnd
  } = config;

  const [value, setValue] = useState(start);
  const hasEnded = useRef(false);

  useEffect(() => {
    hasEnded.current = false;

    const startTime = performance.now();
    const startValue = start;
    const endValue = end;

    const animate = (currentTime: number) => {
      const elapsed = currentTime - startTime;
      const progress = Math.min(elapsed / duration, 1);

      // 应用缓动函数
      const easeProgress = easingFn(progress);

      // 计算当前值
      const currentValue = startValue + (endValue - startValue) * easeProgress;

      setValue(currentValue);

      if (progress < 1) {
        requestAnimationFrame(animate);
      } else if (!hasEnded.current) {
        hasEnded.current = true;
        onEnd?.();
      }
    };

    requestAnimationFrame(animate);
  }, [end, start, duration, easingFn, onEnd]);

  // 格式化数字
  const formatValue = (val: number): string => {
    // 处理小数
    const fixedValue = decimals > 0 ? val.toFixed(decimals) : Math.floor(val).toString();

    if (!useGrouping) return fixedValue;

    // 添加千分位
    const parts = fixedValue.split('.');
    parts[0] = parts[0].replace(/\B(?=(\d{3})+(?!\d))/g, separator);

    return parts.join('.');
  };

  return {
    value,
    formattedValue: formatValue(value)
  };
};

/**
 * 缓动函数
 */
const easeOutExpo = (t: number): number => {
  return t === 1 ? 1 : 1 - Math.pow(2, -10 * t);
};

const easeOutElastic = (t: number): number => {
  if (t === 0 || t === 1) return t;
  return Math.pow(2, -10 * t) * Math.sin((t * 10 - 0.75) * ((2 * Math.PI) / 3)) + 1;
};

const easeOutBounce = (t: number): number => {
  const n1 = 7.5625;
  const d1 = 2.75;

  if (t < 1 / d1) {
    return n1 * t * t;
  } else if (t < 2 / d1) {
    return n1 * (t -= 1.5 / d1) * t + 0.75;
  } else if (t < 2.5 / d1) {
    return n1 * (t -= 2.25 / d1) * t + 0.9375;
  } else {
    return n1 * (t -= 2.625 / d1) * t + 0.984375;
  }
};
```

```typescript
const exportToCSV = (data: any[], filename: string, columns: { key: string; label: string }[]) => {
  // 生成 CSV 头部
  const header = columns.map(c => `"${c.label}"`).join(',');

  // 生成 CSV 数据行
  const rows = data.map(item =>
    columns.map(c => {
      const value = item[c.key];
      // 处理包含逗号或引号的值
      if (typeof value === 'string' && (value.includes(',') || value.includes('"'))) {
        return `"${value.replace(/"/g, '""')}"`;
      }
      return `"${value}"`;
    }).join(',')
  );

  // 合并头部和数据
  const csvContent = [header, ...rows].join('\n');

  // 创建 Blob 并下载
  const blob = new Blob(['\ufeff' + csvContent], { type: 'text/csv;charset=utf-8;' });
  const link = document.createElement('a');
  link.href = URL.createObjectURL(blob);
  link.download = `${filename}_${formatDate(new Date())}.csv`;
  link.click();
  URL.revokeObjectURL(link.href);
```

```typescript
      radius: number;
      alpha: number;
    }[] = [];

    const particleCount = 80;
    const connectionDistance = 150;
    const mouseRadius = 100;

    let mouseX = 0;
    let mouseY = 0;

    // 初始化粒子
    for (let i = 0; i < particleCount; i++) {
      particles.push({
        x: Math.random() * canvas.width,
        y: Math.random() * canvas.height,
        vx: (Math.random() - 0.5) * 0.5,
        vy: (Math.random() - 0.5) * 0.5,
        radius: Math.random() * 2 + 1,
        alpha: Math.random() * 0.5 + 0.2
      });
    }

      mouseX = e.clientX;
      mouseY = e.clientY;
    };
    window.addEventListener('mousemove', handleMouseMove);

    // 动画循环
    const animate = () => {
      ctx.clearRect(0, 0, canvas.width, canvas.height);

      // 更新和绘制粒子
      particles.forEach((p, i) => {
        // 更新位置
        p.x += p.vx;
        p.y += p.vy;

        // 边界检测
        if (p.x < 0 || p.x > canvas.width) p.vx *= -1;
        if (p.y < 0 || p.y > canvas.height) p.vy *= -1;

        // 鼠标交互
        const dx = mouseX - p.x;
        const dy = mouseY - p.y;
        const dist = Math.sqrt(dx * dx + dy * dy);

        if (dist < mouseRadius) {
          p.x -= dx * 0.02;
          p.y -= dy * 0.02;
        }

        // 绘制粒子
        ctx.beginPath();
        ctx.arc(p.x, p.y, p.radius, 0, Math.PI * 2);
        ctx.fillStyle = `rgba(0, 212, 255, ${p.alpha})`;
        ctx.fill();

        // 连接线
        particles.slice(i + 1).forEach(p2 => {
          const dx = p.x - p2.x;
          const dy = p.y - p2.y;
          const dist = Math.sqrt(dx * dx + dy * dy);

            ctx.beginPath();
            ctx.moveTo(p.x, p.y);
            ctx.lineTo(p2.x, p2.y);
            ctx.strokeStyle = `rgba(0, 212, 255, ${0.2 * (1 - dist / connectionDistance)})`;
            ctx.stroke();
          }
        });
      });

      requestAnimationFrame(animate);
    };

    animate();

    return () => {
      window.removeEventListener('resize', resize);
      window.removeEventListener('mousemove', handleMouseMove);
    };
  }, []);

  return (
    <canvas
      ref={canvasRef}
      style={{
        position: 'absolute',
        top: 0,
        left: 0,
        width: '100%',
        height: '100%',
        pointerEvents: 'none',
        opacity: 0.6
      }}
    />
  );
```

```typescript
/**
 * 数据聚合策略
 */
const aggregateData = <T extends Record<string, any>>(
  data: T[],
  groupBy: keyof T,
  aggregate: 'sum' | 'avg' | 'max' | 'min' | 'count'
): Record<string, number> => {
  const grouped: Record<string, T[]> = {};

  // 分组
  data.forEach(item => {
    const key = String(item[groupBy]);
    if (!grouped[key]) grouped[key] = [];
    grouped[key].push(item);
  });

  // 聚合计算
  const result: Record<string, number> = {};

  Object.entries(grouped).forEach(([key, items]) => {
    switch (aggregate) {
      case 'sum':
        result[key] = items.reduce((sum, item) => {
          const value = Number(item.value || 0);
          return sum + value;
        }, 0);
        break;

      case 'avg':
        result[key] = items.reduce((sum, item) => {
          return sum + Number(item.value || 0);
        }, 0) / items.length;
        break;

      case 'max':
        result[key] = Math.max(...items.map(item => Number(item.value || 0)));
        break;

      case 'min':
        result[key] = Math.min(...items.map(item => Number(item.value || 0)));
        break;

      case 'count':
        result[key] = items.length;
        break;
    }
  });

  return result;
};

/**
 * 数据采样策略
 */
const sampleData = <T>(data: T[], sampleSize: number, method: 'random' | 'lttb' = 'random'): T[] => {
  if (data.length <= sampleSize) return data;

  if (method === 'random') {
    // 随机采样
    const shuffled = [...data].sort(() => Math.random() - 0.5);
    return shuffled.slice(0, sampleSize);
  } else {
    // LTTB (Largest Triangle Three Buckets) - 保留峰值
    return lttbSample(data, sampleSize);
  }
};

/**
 * LTTB 降采样算法
 * 保持数据趋势的同时减少数据点
 */
const lttbSample = <T>(data: T[], threshold: number): T[] => {
  if (data.length <= threshold) return data;

  const sampled: T[] = [];
  const bucketSize = (data.length - 2) / (threshold - 2);

  // 保留第一个点
  sampled.push(data[0]);

  let a = 0; // 上一保留点索引

  for (let i = 0; i < threshold - 2; i++) {
    // 计算当前桶和下一个桶的平均值
    const avgRangeStart = Math.floor((i + 1) * bucketSize) + 1;
    const avgRangeEnd = Math.min(Math.floor((i + 2) * bucketSize) + 1, data.length);

    let avgX = 0;
    let avgY = 0;
    const avgRangeLength = avgRangeEnd - avgRangeStart;

    for (let j = avgRangeStart; j < avgRangeEnd; j++) {
      avgX += j;
      avgY += Number(data[j].value || 0);
    }
    avgX /= avgRangeLength;
    avgY /= avgRangeLength;

    // 当前桶的起始和结束
    const rangeStart = Math.floor(i * bucketSize) + 1;
    const rangeEnd = Math.floor((i + 1) * bucketSize) + 1;

    // 在当前桶中找到与前一个保留点和平均值形成最大三角形的点
    const pointAX = a;
    const pointAY = Number(data[a].value || 0);

    let maxArea = -1;
    let maxAreaIndex = rangeStart;

    for (let j = rangeStart; j < rangeEnd; j++) {
      const area = Math.abs(
        (pointAX - avgX) * (Number(data[j].value || 0) - pointAY) -
        (pointAX - j) * (avgY - pointAY)
      ) * 0.5;

      if (area > maxArea) {
        maxArea = area;
        maxAreaIndex = j;
      }
    }

    sampled.push(data[maxAreaIndex]);
    a = maxAreaIndex;
  }

  // 保留最后一个点
  sampled.push(data[data.length - 1]);

  return sampled;
};
```

```typescript
  get(key: string): T | null {
    const item = this.cache.get(key);

    if (!item) return null;

    // 检查是否过期
    if (Date.now() - item.timestamp > item.ttl) {
      this.cache.delete(key);
      return null;
    }

    return item.data;
  }

  clear() {
    this.cache.clear();
  }

  // 清理过期数据
  cleanExpired() {
    const now = Date.now();
    this.cache.forEach((item, key) => {
      if (now - item.timestamp > item.ttl) {
        this.cache.delete(key);
      }
    });
  }
}

// 创建全局缓存实例
const dataCache = new DataCache<any>();

// 定期清理过期缓存
setInterval(() => {
  dataCache.cleanExpired();
}, 60 * 1000);  // 每分钟清理一次
```

```typescript
interface DataMonitorConfig {

class DataLatencyMonitor {
  private lastDataTime: number = 0;
  private delays: number[] = [];
  private maxDelay: number;
  private onDelayExceeded?: (delay: number) => void;
  private onError?: (error: Error) => void;

  constructor(config: DataMonitorConfig) {
    this.maxDelay = config.maxDelay || 5000;
    this.onDelayExceeded = config.onDelayExceeded;
    this.onError = config.onError;
  }

  recordData() {
    const now = Date.now();
    const delay = this.lastDataTime ? now - this.lastDataTime : 0;

    if (this.lastDataTime && delay > 0) {
      this.delays.push(delay);

      // 保持最近 100 个样本
      if (this.delays.length > 100) {
        this.delays.shift();
      }

      // 检查是否超过最大延迟
      if (delay > this.maxDelay) {
        this.onDelayExceeded?.(delay);
      }
    }

    this.lastDataTime = now;
  }

  getStats() {
    if (this.delays.length === 0) {
      return { avg: 0, min: 0, max: 0, p95: 0 };
    }

    const sorted = [...this.delays].sort((a, b) => a - b);
    const sum = sorted.reduce((a, b) => a + b, 0);

    return {
      avg: sum / sorted.length,
      min: sorted[0],
      max: sorted[sorted.length - 1],
      p95: sorted[Math.floor(sorted.length * 0.95)]
    };
  }

  reset() {
    this.lastDataTime = 0;
    this.delays = [];
  }
}
```

```typescript
    const checkStale = () => {
```

```typescript
const useTrendPrediction = (historicalData: number[]) => {
  // 简单的线性回归预测
  const predict = (periods: number) => {
    const n = historicalData.length;
    const sumX = (n * (n - 1)) / 2;
    const sumY = historicalData.reduce((a, b) => a + b, 0);
    const sumXY = historicalData.reduce((sum, y, x) => sum + x * y, 0);
    const sumX2 = (n * (n - 1) * (2 * n - 1)) / 6;

    const slope = (n * sumXY - sumX * sumY) / (n * sumX2 - sumX * sumX);
    const intercept = (sumY - slope * sumX) / n;

    // 预测未来值
    return Array.from({ length: periods }, (_, i) => {
      return slope * (n + i) + intercept;
    });
  };
```

1. **布局系统**：响应式布局、栅格系统、间距规范

大屏开发不仅仅是技术实现，更是设计艺术和用户体验的结合。只有在功能性和美观性之间找到平衡，才能打造出优秀的大屏应用。
