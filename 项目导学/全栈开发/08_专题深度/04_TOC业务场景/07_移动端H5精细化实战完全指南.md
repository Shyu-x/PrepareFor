## 一、移动端H5概述

### 1.1 H5与小程序、App的对比分析

在移动端开发领域，开发者通常有三种主要的技术选择：H5页面、小程序和原生App。这三种技术各有优劣，适用于不同的业务场景。理解它们之间的区别和联系，是选择正确技术方案的前提。

| 技术方案 | 开发成本 | 用户体验 | 分发能力 | 更新能力 | 设备API |
|----------|----------|----------|----------|----------|---------|
| **H5页面** | 低 | 一般 | 强（通过URL） | 即时 | 有限 |
| **小程序** | 中等 | 较好 | 强（平台流量） | 即时 | 中等 |
| **原生App** | 高 | 最佳 | 一般 | 需审核 | 完整 |

H5页面最大的优势在于其极高的灵活性和便捷性。用户无需下载安装，只需通过浏览器或扫码即可访问，这大大降低了用户的使用门槛。同时，H5页面可以通过URL进行分享，具有天然的社交传播能力，这对于营销活动和内容分发来说是非常重要的优势。此外，H5页面的开发成本相对较低，一套代码可以在多个平台运行，大大提高了开发效率。

然而，H5页面也存在一些明显的局限性。首先，由于运行在浏览器环境中，H5页面对设备硬件的访问能力有限，无法像原生App那样完整地使用设备的所有功能。其次，H5页面的性能受限于浏览器引擎，在复杂的动画和交互场景下可能不如原生App流畅。最后，H5页面的用户留存率通常较低，因为用户需要记住网址或频繁扫码才能再次访问。

小程序则是介于H5和原生App之间的一种技术方案。它解决了H5的一些痛点问题，比如需要扫码访问、用户留存困难等。小程序依托于各大平台（微信、支付宝、抖音等），具有天然的平台流量入口，用户可以通过搜索、扫码、分享等多种方式发现和使用小程序。小程序还提供了一些设备API的封装，使得开发者能够实现一些H5无法完成的功能，如蓝牙、NFC等。小程序的更新也是即时的，开发者可以随时发布新版本，用户下次打开时自动使用最新版本。

原生App是移动端开发中用户体验最好的技术方案，它可以完整地访问设备的所有硬件能力，包括摄像头、GPS、传感器等，运行性能也是最佳的。但是，原生App的开发成本最高，需要同时开发iOS和Android两个版本，而且需要用户下载安装，用户获取成本较高。App的更新也需要用户主动下载安装新版本，这在一定程度上影响了功能的快速迭代。

### 1.2 移动端H5的核心特点

移动端H5页面有几个区别于桌面端Web开发的核心特点，开发者必须深入理解这些特点，才能开发出优秀的移动端H5应用。

**触摸交互是移动端区别于桌面端最显著的特点**。在桌面端，用户使用鼠标进行点击、悬停等操作，而在移动端，用户通过手指进行触摸、滑动、捏合等操作。这意味着传统的hover效果在移动端没有意义，取而代之的是触摸反馈、滑动切换、手势识别等交互方式。触摸交互要求开发者重新思考信息架构和交互流程，确保所有交互元素足够大（通常建议至少44x44像素），让用户能够方便地点击和操作。

**有限的屏幕空间是移动端开发面临的另一个重要挑战**。移动设备的屏幕尺寸虽然越来越大，但相对于桌面显示器仍然非常有限。开发者必须在有限的屏幕空间内呈现用户需要的信息和功能，这就要求界面设计更加紧凑和高效。响应式布局在移动端变得更加重要，页面需要能够适应不同尺寸和分辨率的设备。同时，导航设计也需要更加精简，层级不宜过深，通常建议不超过三层。

**设备多样性和兼容性问题也是移动端开发的重要特点**。移动设备市场碎片化严重，各种品牌、型号、系统版本共存。Android设备有众多屏幕尺寸和分辨率，iOS设备也有多种尺寸（iPhone SE、iPhone标准版、iPhone Pro Max、iPad等）。不同的浏览器内核（特别是Android WebView）带来了各种兼容性问题。开发者需要进行大量的兼容性测试，确保页面在主流设备上都能正常显示和运行。

### 1.3 移动端H5开发面临的技术挑战

移动端H5开发面临着多方面的技术挑战，这些挑战需要开发者在技术选型、架构设计、性能优化等多个层面进行深入考虑。

**性能问题是移动端H5开发最核心的挑战之一**。移动设备的计算能力相对桌面设备较弱，而移动端Web应用的复杂度却越来越高。用户对页面加载速度和交互流畅度的要求也很高，任何卡顿都会严重影响用户体验。研究表明，页面加载时间超过3秒，流失率会大幅上升。因此，性能优化是移动端H5开发中永恒的主题，需要从网络请求、资源加载、代码执行、渲染表现等多个环节进行优化。

**浏览器兼容性问题是另一个不可忽视的挑战**。虽然现代浏览器对H5标准的支持越来越好，但在移动端，情况要复杂得多。iOS Safari和Android Chrome是两大主流移动浏览器，它们对H5 API的支持程度较高，但仍然存在一些差异。更麻烦的是Android设备上的各种定制浏览器和WebView，这些组件的版本和实现各异，对H5标准的支持参差不齐。开发者需要使用特性检测而非浏览器检测来处理兼容性问题，并准备相应的降级方案。

**网络环境的不稳定性也是移动端需要重点考虑的因素**。移动用户可能在地铁、电梯等弱网环境下使用应用，网络中断、频繁切换网络（4G切换WiFi）的情况时有发生。H5应用需要具备良好的离线支持能力，在网络不可用时仍能提供基本的功能体验。同时，对于网络请求，需要考虑超时处理、重试机制、数据压缩等问题，以适应不稳定的网络环境。

**电池消耗也是移动端开发需要关注的问题**。虽然这个问题对于单个H5页面来说可能不太明显，但如果H5应用需要长时间运行（如小游戏、实时聊天等），就需要考虑各种操作对电池消耗的影响。比如，频繁的网络请求、持续的动画播放、高频的位置更新等操作都会加速电池消耗。开发者需要在功能和能耗之间找到平衡点。

### 1.4 H5的不可替代性：我的深度思考

尽管小程序和原生App在很多场景下能够提供更好的用户体验，但H5页面仍然具有其不可替代的价值。作为一个有着多年移动端开发经验的开发者，我对此有深刻的体会。

**H5的最大价值在于其极致的开放性和便捷性**。H5页面通过URL访问，可以嵌入到任何支持WebView的环境中。你可以在微信、微博、钉钉、邮件、短信等任何地方分享一个H5链接，用户点击即可访问。这种跨平台的开放性是小程序和App无法比拟的。小程序需要用户先打开特定的应用（微信、支付宝等），App需要用户下载安装，而在很多场景下，用户可能并没有安装你的App，也可能不在微信环境中。这时，H5就是唯一的选择。

**H5是快速迭代和敏捷开发的最佳选择**。在互联网行业，速度就是生命。一个营销活动可能只有几天的窗口期，需要快速上线、随时调整。H5页面的发布不需要审核（除了某些特殊平台），可以做到分钟级的更新上线。而小程序的审核可能需要数小时甚至数天，原生App的审核更是需要数周。对于需要快速试错、快速迭代的业务场景，H5是不可替代的选择。

**H5也是实现部分功能复用的有效方式**。很多企业同时有App和H5业务，有些功能需要在两端同时提供。如果为App和H5各开发一套代码，维护成本很高。通过H5嵌入App的技术方案（如iOS的WKWebView、Android的WebView），可以实现核心功能的统一维护，只开发一次，即可在两端使用。当然，这里需要注意H5和原生之间的通信机制设计。

**H5在数据分析方面也有独特的优势**。由于H5页面运行在浏览器环境中，可以直接使用Web的各种分析工具，如Google Analytics、神策分析等。这些工具功能完善、配置灵活，是App和小程序难以匹敌的。对于需要精细化运营的业务来说，H5的数据采集能力是一大优势。

综上所述，H5、小程序和App各有其适用场景，不存在谁取代谁的问题。作为开发者，我们需要根据具体的业务需求和约束条件，选择最合适的技术方案。H5在很多场景下仍然是最佳甚至唯一的选择，掌握H5开发技术是前端开发者的必备技能。

## 二、视口与布局系统

### 2.1 viewport配置详解

viewport是移动端H5开发中最基础也是最重要的配置之一。正确配置viewport可以确保页面在移动设备上正确显示，避免出现缩放异常、布局混乱等问题。

viewport是通过HTML的meta标签进行配置的，它告诉浏览器如何处理页面的缩放和布局。以下是标准的viewport配置：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <!-- 标准viewport配置 -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no, viewport-fit=cover">
  <title>移动端H5页面</title>
</head>
<body>
  <div id="app"></div>
</body>
</html>
```

下面对viewport的各个配置项进行详细解释：

**width=device-width** 表示页面的宽度等于设备的宽度，这是最常用的配置。如果设置为具体的数值（如width=320），则页面宽度固定为320像素，可能会在较宽的设备上显示不正常。

**initial-scale=1.0** 表示页面的初始缩放比例为1倍，即不缩放。值为1.0意味着1个CSS像素对应1个设备像素。需要注意的是，这个比例是相对于逻辑像素而非物理像素的。

**maximum-scale=1.0, minimum-scale=1.0** 分别表示允许的最大和最小缩放比例。将这两个值都设置为1.0，配合user-scalable=no，可以完全禁止用户缩放页面。这在某些不希望用户缩放的场景下很有用，但也要谨慎使用，因为这可能影响无障碍访问。

**user-scalable=no** 禁止用户通过手势缩放页面。这个配置需要权衡使用，一方面可以防止用户误操作导致页面布局错乱，另一方面也可能会影响需要放大阅读的场景（如图文详情页）。我的建议是，对于内容驱动的页面，可以允许适度缩放；对于应用型页面，可以禁止缩放。

**viewport-fit=cover** 是iPhone X及以上设备需要的配置，用于控制页面是否延伸到屏幕的安全区域。设置为cover表示页面内容延伸到屏幕边缘，包括刘海区域。这是实现全面屏适配的重要配置。

在实际项目中，viewport配置可能需要根据业务需求进行调整。以下是一些常见的配置场景：

```html
<!-- 游戏场景：禁止缩放，优化全屏体验 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no, viewport-fit=cover">

<!-- 内容阅读场景：允许适度缩放，方便阅读 -->
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=2.0, user-scalable=yes">

<!-- 固定宽度场景：某些特殊页面可能需要固定宽度 -->
<meta name="viewport" content="width=750, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

### 2.2 设备像素比与物理像素

设备像素比（Device Pixel Ratio，简称DPR）是移动端开发中一个非常重要的概念，它描述了物理像素和CSS像素之间的比例关系。理解DPR对于实现高清显示和处理1px边框问题至关重要。

在桌面端，屏幕的一个像素通常对应一个物理像素。但在移动端，情况要复杂得多。为了在有限的物理尺寸内提供更高的显示清晰度，移动设备普遍采用了高密度屏幕。一个CSS像素可能对应多个物理像素，这就是DPR存在的意义。

```javascript
// 获取设备像素比
const dpr = window.devicePixelRatio;
console.log(`当前设备像素比为: ${dpr}`);

// 常见的设备像素比：
// iPhone 4/5/6/7/8: 2
// iPhone Plus/Pro Max: 2.6 或 3
// iPhone X/XS/11 Pro: 3
// iPad Pro 12.9: 2.5
// Android 高端机: 3 或 4
// Android 中低端机: 2
```

DPR的存在对开发工作有重要影响。首先，在设置1px边框时，如果直接使用CSS的border: 1px solid #ccc，在高DPR设备上显示的实际边框厚度会是2px或3px，看起来比较粗。这就需要使用一些技巧来实现真正的1px边框。

其次，在处理图片时，高DPR设备需要更高分辨率的图片才能显示清晰。如果使用普通图片，在高DPR设备上会显得模糊。这就需要准备多套图片资源，或者使用SVG矢量图、Icon Font等方案。

### 2.3 真正1px边框的实现方案

实现真正的1px边框是移动端开发中的一个经典问题。由于DPR的存在，直接使用border: 1px会导致边框显示过粗。以下是几种常见的解决方案：

**方案一：使用transform缩放**

这是最常见的解决方案，通过设置0.5像素的边框然后用transform缩放来实现：

```css
/* 1px边框基础样式 */
.border-1px {
  position: relative;
}

.border-1px::after {
  content: '';
  position: absolute;
  left: 0;
  top: 0;
  width: 200%;
  height: 200%;
  border: 1px solid #e0e0e0;
  transform-origin: left top;
  transform: scale(0.5);
  pointer-events: none;
}

/* 针对不同方向的边框优化 */
.border-top-1px::before {
  content: '';
  position: absolute;
  left: 0;
  top: 0;
  width: 100%;
  height: 0;
  border-top: 1px solid #e0e0e0;
  transform: scaleY(0.5);
  transform-origin: left top;
}

.border-bottom-1px::after {
  content: '';
  position: absolute;
  left: 0;
  bottom: 0;
  width: 100%;
  height: 0;
  border-bottom: 1px solid #e0e0e0;
  transform: scaleY(0.5);
  transform-origin: left bottom;
}
```

```html
<div class="border-1px">使用1px边框的容器</div>
<div class="border-top-1px">只有顶部边框</div>
<div class="border-bottom-1px">只有底部边框</div>
```

**方案二：使用渐变实现边框**

利用CSS渐变可以创建非常细的边框效果：

```css
/* 使用线性渐变实现0.5px边框 */
.border-gradient {
  background:
    linear-gradient(to bottom, #e0e0e0 50%, transparent 50%) left / 100% 1px no-repeat,
    linear-gradient(to right, #e0e0e0 50%, transparent 50%) top / 1px 100% no-repeat,
    linear-gradient(to bottom, #e0e0e0 50%, transparent 50%) left / 100% 1px no-repeat,
    linear-gradient(to right, #e0e0e0 50%, transparent 50%) bottom / 1px 100% no-repeat;
  background-position: top, right, bottom, left;
}
```

**方案三：使用box-shadow模拟边框**

对于内侧边框，可以使用box-shadow实现：

```css
/* 使用box-shadow实现内边框 */
.inner-border {
  box-shadow: inset 0 -1px 1px 0 #e0e0e0;
}
```

**方案四：使用PostCSS插件自动处理**

在实际项目中，可以使用postcss-write-svg等插件自动处理1px边框：

```javascript
// postcss.config.js 配置
module.exports = {
  plugins: {
    'postcss-write-svg': {
      // 插件会自动处理1px边框
    }
  }
}
```

### 2.4 安全区域适配

随着全面屏手机的普及，安全区域（Safe Area）成为了移动端开发中必须考虑的问题。安全区域是指屏幕中不会被刘海、圆角或系统导航栏遮挡的区域。H5页面需要正确处理安全区域，确保内容不会被遮挡。

**viewport-fit=cover配置**

首先，需要在viewport meta标签中设置viewport-fit=cover：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover">
```

**使用env()函数获取安全区域**

CSS提供了env()函数来获取安全区域的尺寸：

```css
/* 定义安全区域变量 */
:root {
  --safe-area-inset-top: env(safe-area-inset-top);
  --safe-area-inset-right: env(safe-area-inset-right);
  --safe-area-inset-bottom: env(safe-area-inset-bottom);
  --safe-area-inset-left: env(safe-area-inset-left);
}

/* 使用安全区域 */
.safe-top {
  padding-top: var(--safe-area-inset-top);
}

.safe-bottom {
  padding-bottom: var(--safe-area-inset-bottom);
}

/* 全屏适配布局 */
.full-screen-layout {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  padding-top: var(--safe-area-inset-top);
  padding-bottom: var(--safe-area-inset-bottom);
  padding-left: var(--safe-area-inset-left);
  padding-right: var(--safe-area-inset-right);
}
```

**JavaScript中获取安全区域**

如果需要在JavaScript中获取安全区域信息，可以通过计算：

```javascript
// 检测是否为全面屏iPhone
function isIPhoneX() {
  return /iphone/gi.test(navigator.userAgent) && window.screen.height >= 812;
}

// 获取安全区域尺寸
function getSafeArea() {
  const style = getComputedStyle(document.documentElement);
  return {
    top: parseInt(style.getPropertyValue('--safe-area-inset-top')) || 0,
    right: parseInt(style.getPropertyValue('--safe-area-inset-right')) || 0,
    bottom: parseInt(style.getPropertyValue('--safe-area-inset-bottom')) || 0,
    left: parseInt(style.getPropertyValue('--safe-area-inset-left')) || 0
  };
}
```

**实战：完整的移动端适配框架**

以下是一个完整的移动端适配框架示例：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover, user-scalable=no">
  <title>全面屏适配示例</title>
  <style>
```

## 三、触摸交互系统

### 3.1 触摸事件详解

触摸事件是移动端H5开发的基础，掌握好触摸事件才能实现各种丰富的交互效果。移动端主要有四种触摸事件：touchstart、touchmove、touchend和touchcancel。

**touchstart事件**在手指触摸屏幕时触发，此时手指刚刚接触到屏幕，可以在这个事件中记录触摸的起始位置。

**touchmove事件**在手指在屏幕上滑动时持续触发，可以通过这个事件获取手指的实时位置，计算滑动距离和方向。

**touchend事件**在手指离开屏幕时触发，此时可以判断滑动的终止位置，分析滑动手势。

**touchcancel事件**在触摸被中断时触发，比如在触摸过程中突然接到电话或弹出一个系统弹窗。

以下是一个完整的触摸事件监听示例：

```javascript
// 触摸事件处理类
class TouchHandler {
  constructor(element, options = {}) {
    this.element = element;
    this.options = {
      onTouchStart: null,
      onTouchMove: null,
      onTouchEnd: null,
      onSwipeLeft: null,
      onSwipeRight: null,
      onSwipeUp: null,
      onSwipeDown: null,
      threshold: 50, // 触发滑动的最小距离
      ...options
    };

    // 触摸起始位置
    this.startX = 0;
    this.startY = 0;

    // 触摸当前位置
    this.currentX = 0;
    this.currentY = 0;

    // 触摸起始时间
    this.startTime = 0;

    // 是否已触发滑动
    this.isSwiped = false;

    this.bindEvents();
  }

  bindEvents() {
    // 绑定触摸事件
    this.element.addEventListener('touchstart', this.handleTouchStart.bind(this), { passive: false });
    this.element.addEventListener('touchmove', this.handleTouchMove.bind(this), { passive: false });
    this.element.addEventListener('touchend', this.handleTouchEnd.bind(this), { passive: false });
    this.element.addEventListener('touchcancel', this.handleTouchCancel.bind(this), { passive: false });
  }

  handleTouchStart(event) {
    // 如果是多指触摸，只处理第一个手指
    if (event.touches.length > 1) return;

    const touch = event.touches[0];
    this.startX = touch.clientX;
    this.startY = touch.clientY;
    this.currentX = touch.clientX;
    this.currentY = touch.clientY;
    this.startTime = Date.now();
    this.isSwiped = false;

    // 调用自定义回调
    if (this.options.onTouchStart) {
      this.options.onTouchStart({
        x: touch.clientX,
        y: touch.clientY,
        event
      });
    }

    // 阻止默认行为，防止页面滚动
    event.preventDefault();
  }

  handleTouchMove(event) {
    if (event.touches.length > 1) return;

    const touch = event.touches[0];
    this.currentX = touch.clientX;
    this.currentY = touch.clientY;

    // 计算滑动距离
    const deltaX = this.currentX - this.startX;
    const deltaY = this.currentY - this.startY;

    // 调用自定义回调
    if (this.options.onTouchMove) {
      this.options.onTouchMove({
        startX: this.startX,
        startY: this.startY,
        currentX: this.currentX,
        currentY: this.currentY,
        deltaX,
        deltaY,
        event
      });
    }

    event.preventDefault();
  }

  handleTouchEnd(event) {
    const deltaX = this.currentX - this.startX;
    const deltaY = this.currentY - this.startY;
    const deltaTime = Date.now() - this.startTime;

    // 判断滑动方向
    const absDeltaX = Math.abs(deltaX);
    const absDeltaY = Math.abs(deltaY);

    // 如果滑动距离超过阈值，触发滑动事件
    if (absDeltaX > this.options.threshold || absDeltaY > this.options.threshold) {
      this.isSwiped = true;

      // 确定滑动方向
      if (absDeltaX > absDeltaY) {
        // 水平滑动
        if (deltaX > 0) {
          this.triggerSwipe('right', deltaX, deltaTime);
        } else {
          this.triggerSwipe('left', deltaX, deltaTime);
        }
      } else {
        // 垂直滑动
        if (deltaY > 0) {
          this.triggerSwipe('down', deltaY, deltaTime);
        } else {
          this.triggerSwipe('up', deltaY, deltaTime);
        }
      }
    }

    // 调用自定义回调
    if (this.options.onTouchEnd) {
      this.options.onTouchEnd({
        startX: this.startX,
        startY: this.startY,
        endX: this.currentX,
        endY: this.currentY,
        deltaX,
        deltaY,
        deltaTime,
        isSwiped: this.isSwiped,
        event
      });
    }
  }

  handleTouchCancel(event) {
    console.log('触摸被中断');
    if (this.options.onTouchCancel) {
      this.options.onTouchCancel(event);
    }
  }

  triggerSwipe(direction, distance, duration) {
    const speed = Math.abs(distance) / duration; // 计算滑动速度

    const eventMap = {
      'left': this.options.onSwipeLeft,
      'right': this.options.onSwipeRight,
      'up': this.options.onSwipeUp,
      'down': this.options.onSwipeDown
    };

    const callback = eventMap[direction];
    if (callback) {
      callback({
        direction,
        distance,
        speed,
        duration
      });
    }
  }

  destroy() {
    this.element.removeEventListener('touchstart', this.handleTouchStart);
    this.element.removeEventListener('touchmove', this.handleTouchMove);
    this.element.removeEventListener('touchend', this.handleTouchEnd);
    this.element.removeEventListener('touchcancel', this.handleTouchCancel);
  }
}

// 使用示例
const container = document.querySelector('.swipe-container');
const touchHandler = new TouchHandler(container, {
  onSwipeLeft: (data) => {
    console.log('向左滑动:', data);
    // 执行左滑操作，如切换到下一页
  },
  onSwipeRight: (data) => {
    console.log('向右滑动:', data);
    // 执行右滑操作，如返回上一页
  },
  onSwipeUp: (data) => {
    console.log('向上滑动:', data);
    // 加载更多内容
  },
  threshold: 50
});
```

### 3.2 点击延迟问题与解决方案

移动端H5页面中存在一个经典问题：点击延迟。在早期的移动浏览器中，为了区分单击和双击手势，浏览器在touchend事件后会等待300毫秒，如果在这个时间内没有再次触摸，才触发click事件。这个设计虽然解决了双击问题，但也导致了页面响应迟钝的问题。

**问题原因分析**

点击延迟的存在主要有以下原因：

1. **区分单击和双击**：浏览器需要等待一段时间，确认用户是单击还是双击
2. **缩放检测**：浏览器需要确认用户是否在执行双指缩放操作
3. **历史兼容**：早期的移动网站没有视口meta标签，浏览器需要通过延迟来适应桌面站点

**解决方案一：设置viewport禁用缩放**

最简单的方法是在viewport中禁用缩放：

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

这样浏览器就知道页面不需要缩放功能，可以立即触发click事件。

**解决方案二：使用touch-action CSS属性**

CSS的touch-action属性可以告诉浏览器该元素应该如何处理触摸手势：

```css
/* 禁止浏览器处理触摸手势 */
.no-delay-click {
  touch-action: manipulation;
}

/* manipulation值的作用：
   1. 禁用双击缩放
   2. 允许平移和缩放操作（但双击缩放不行）
   3. 消除点击延迟
*/
```

**解决方案三：使用CSS transform: translateZ(0)**

触发GPU加速可以加快渲染速度，间接减少延迟：

```css

  transform: translateZ(0);
  -webkit-transform: translateZ(0);
}
```

**解决方案四：手动实现即时响应**

在需要极致体验的场景下，可以手动处理touch事件实现即时响应：

```javascript
// 即时点击处理
class FastClick {
  constructor(element, onClick) {
    this.element = element;
    this.onClick = onClick;
    this.startX = 0;
    this.startY = 0;
    this.startTime = 0;

    this.bindEvents();
  }

  bindEvents() {
    this.element.addEventListener('touchstart', this.handleTouchStart.bind(this), { passive: false });
    this.element.addEventListener('touchend', this.handleTouchEnd.bind(this), { passive: false });
    this.element.addEventListener('touchmove', this.handleTouchMove.bind(this), { passive: false });
  }

  handleTouchStart(event) {
    if (event.touches.length !== 1) return;

    this.startX = event.touches[0].clientX;
    this.startY = event.touches[0].clientY;
    this.startTime = Date.now();
    this.isMoving = false;
  }

  handleTouchMove(event) {
    if (event.touches.length !== 1) return;

    const deltaX = event.touches[0].clientX - this.startX;
    const deltaY = event.touches[0].clientY - this.startY;

    // 如果移动距离超过10px，认为是在滑动而非点击
    if (Math.abs(deltaX) > 10 || Math.abs(deltaY) > 10) {
      this.isMoving = true;
    }
  }

  handleTouchEnd(event) {
    // 如果是滑动操作，不触发点击
    if (this.isMoving) return;

    // 计算触摸时长
    const duration = Date.now() - this.startTime;

    // 如果触摸时间过长（超过500ms），不视为点击
    if (duration > 500) return;

    // 创建模拟的click事件
    const clickEvent = new MouseEvent('click', {
      bubbles: true,
      cancelable: true,
      view: window
    });

    this.element.dispatchEvent(clickEvent);

    // 执行回调
    if (this.onClick) {
      this.onClick(event);
    }

    event.preventDefault();
  }
}

// 使用示例
const button = document.querySelector('.fast-button');
new FastClick(button, (event) => {
  console.log('按钮被点击');
});
```

```javascript
function createRipple(event) {
  const container = event.currentTarget;
  const rect = container.getBoundingClientRect();

  // 获取触摸点
  const touch = event.touches[0];
  const x = touch.clientX - rect.left;
```

### 3.4 实战：滑动切换组件

滑动切换是移动端常见的交互模式，如轮播图、页面切换等。以下是一个完整的滑动切换组件实现：

```javascript
// 滑动切换组件
class Swiper {
  constructor(container, options = {}) {
    this.container = container;
    this.wrapper = container.querySelector('.swiper-wrapper');
    this.slides = Array.from(this.wrapper.children);

    this.options = {
      initialIndex: 0,
      autoplay: false,
      autoplayInterval: 3000,
      loop: true,
      threshold: 50,
      duration: 300,
      onChange: null,
      ...options
    };

    // 状态
    this.currentIndex = this.options.initialIndex;
    this.totalSlides = this.slides.length;
    this.isAnimating = false;
    this.startX = 0;
    this.startY = 0;
    this.currentX = 0;
    this.deltaX = 0;
    this.deltaY = 0;
    this.isHorizontalSwipe = null;

    // 自动播放定时器
    this.autoplayTimer = null;

    this.init();
  }

  init() {
    // 设置初始样式
    this.wrapper.style.display = 'flex';
    this.wrapper.style.transition = 'none';

    // 初始化位置
    this.goTo(this.currentIndex, false);

    // 绑定触摸事件
    this.bindTouchEvents();

    // 启动自动播放
    if (this.options.autoplay) {
      this.startAutoplay();
    }

    // 窗口大小改变时重新计算
    window.addEventListener('resize', this.handleResize.bind(this));
  }

  bindTouchEvents() {
    this.container.addEventListener('touchstart', this.handleTouchStart.bind(this), { passive: false });
    this.container.addEventListener('touchmove', this.handleTouchMove.bind(this), { passive: false });
    this.container.addEventListener('touchend', this.handleTouchEnd.bind(this), { passive: false });
  }

  handleTouchStart(event) {
    if (this.isAnimating) return;
    if (event.touches.length !== 1) return;

    // 停止自动播放
    if (this.options.autoplay) {
      this.stopAutoplay();
    }

    const touch = event.touches[0];
    this.startX = touch.clientX;
    this.startY = touch.clientY;
    this.currentX = touch.clientX;
    this.deltaX = 0;
    this.deltaY = 0;
    this.isHorizontalSwipe = null;

    // 移除过渡动画
    this.wrapper.style.transition = 'none';
  }

  handleTouchMove(event) {
    if (this.isAnimating) return;
    if (event.touches.length !== 1) return;

    const touch = event.touches[0];
    this.deltaX = touch.clientX - this.startX;
    this.deltaY = touch.clientY - this.startY;
    this.currentX = touch.clientX;

    // 首次移动时判断滑动方向
    if (this.isHorizontalSwipe === null) {
      const absDeltaX = Math.abs(this.deltaX);
      const absDeltaY = Math.abs(this.deltaY);

      // 如果垂直移动距离大于水平，不处理
      if (absDeltaY > absDeltaX && absDeltaY > 10) {
        this.isHorizontalSwipe = false;
        return;
      }

      // 水平滑动距离大于10px时，确定为水平滑动
      if (absDeltaX > 10) {
        this.isHorizontalSwipe = true;
      }
    }

    // 如果不是水平滑动，不处理
    if (!this.isHorizontalSwipe) return;

    event.preventDefault();

    // 实时更新位置
    const slideWidth = this.container.offsetWidth;
    const offset = -this.currentIndex * slideWidth + this.deltaX;
    this.wrapper.style.transform = `translateX(${offset}px)`;
  }

  handleTouchEnd(event) {
    if (this.isAnimating) return;
    if (!this.isHorizontalSwipe) return;

    const slideWidth = this.container.offsetWidth;
    const threshold = this.options.threshold;

    // 判断是否需要切换
    if (this.deltaX < -threshold && this.currentIndex < this.totalSlides - 1) {
      // 向左滑，下一页
      this.next();
    } else if (this.deltaX > threshold && this.currentIndex > 0) {
      // 向右滑，上一页
      this.prev();
    } else {
      // 回弹到当前位置
      this.goTo(this.currentIndex, true);
    }

    // 重新开始自动播放
    if (this.options.autoplay) {
      setTimeout(() => {
        this.startAutoplay();
      }, this.options.autoplayInterval);
    }
  }

  handleResize() {
    this.goTo(this.currentIndex, false);
  }

  goTo(index, animate = true) {
    if (this.isAnimating) return;

    // 处理循环模式
    if (this.options.loop) {
      // 循环模式下的边界处理
      if (index < 0) {
        index = this.totalSlides - 1;
      } else if (index >= this.totalSlides) {
        index = 0;
      }
    } else {
      // 非循环模式
      index = Math.max(0, Math.min(index, this.totalSlides - 1));
    }

    this.currentIndex = index;

    const slideWidth = this.container.offsetWidth;
    const offset = -index * slideWidth;

    if (animate) {
      this.isAnimating = true;
      this.wrapper.style.transition = `transform ${this.options.duration}ms ease`;

      setTimeout(() => {
        this.isAnimating = false;
      }, this.options.duration);
    }

    this.wrapper.style.transform = `translateX(${offset}px)`;

    // 触发回调
    if (this.options.onChange) {
      this.options.onChange(index);
    }
  }

  next() {
    this.goTo(this.currentIndex + 1, true);
  }

  prev() {
    this.goTo(this.currentIndex - 1, true);
  }

  startAutoplay() {
    this.stopAutoplay();
    this.autoplayTimer = setInterval(() => {
      this.next();
    }, this.options.autoplayInterval);
  }

  stopAutoplay() {
    if (this.autoplayTimer) {
      clearInterval(this.autoplayTimer);
      this.autoplayTimer = null;
    }
  }

  destroy() {
    this.stopAutoplay();
    window.removeEventListener('resize', this.handleResize);
  }
}
```

```javascript
// 输入框聚焦管理类
class InputFocusManager {
  constructor(options = {}) {
    this.options = {
      container: document.body,
      offset: 0, // 额外的偏移量
      animate: true,
      ...options
    };

    this.scrollY = 0;
    this.activeInput = null;

    this.bindEvents();
  }

  bindEvents() {
    // 监听所有输入框的聚焦事件
    document.querySelectorAll('input, textarea, select').forEach(input => {
      input.addEventListener('focus', this.handleFocus.bind(this), true);
      input.addEventListener('blur', this.handleBlur.bind(this), true);
    });

    // 监听键盘弹出/收起
    window.addEventListener('resize', this.handleResize.bind(this));

    // 监听滚动事件，防止输入框被键盘遮挡
    this.options.container.addEventListener('scroll', this.handleScroll.bind(this), true);
  }

  handleFocus(event) {
    this.activeInput = event.target;
    this.scrollY = window.scrollY;

    // 延迟处理，等待键盘完全弹出
    setTimeout(() => {
      this.scrollToInput();
    }, 300);
  }

  handleBlur(event) {
    this.activeInput = null;

    // 延迟隐藏键盘后再恢复位置
    setTimeout(() => {
      window.scrollTo(0, this.scrollY);
    }, 100);
  }

  handleResize() {
    // 检测视口高度变化，判断键盘是否弹出
    const visualHeight = window.visualViewport?.height || window.innerHeight;
    const isKeyboardVisible = visualHeight < window.innerHeight;

    if (this.activeInput && isKeyboardVisible) {
      this.scrollToInput();
    }
  }

  handleScroll(event) {
    // 如果有输入框聚焦，阻止滚动
    if (this.activeInput) {
      event.preventDefault();
    }
  }

  scrollToInput() {
    if (!this.activeInput) return;

    const inputRect = this.activeInput.getBoundingClientRect();
    const keyboardHeight = window.innerHeight - (window.visualViewport?.height || window.innerHeight);
    const targetY = inputRect.bottom + this.options.offset - (window.innerHeight - keyboardHeight);

    if (this.options.animate) {
      window.scrollTo({
        top: targetY,
        behavior: 'smooth'
      });
    } else {
      window.scrollTo(0, targetY);
    }
  }
}

// 使用Visual Viewport API的增强版本
class EnhancedInputFocusManager {
  constructor() {
    this.scrollY = 0;
    this.init();
  }

  init() {
    // 使用Visual Viewport API获取更精确的视口信息
    if (window.visualViewport) {
      window.visualViewport.addEventListener('resize', this.handleViewportChange.bind(this));
      window.visualViewport.addEventListener('scroll', this.handleViewportChange.bind(this));
    }
  }

  handleViewportChange() {
    const visualViewport = window.visualViewport;
    if (!visualViewport) return;

    const { height, offsetTop } = visualViewport;

    // 计算键盘高度
    const keyboardHeight = window.innerHeight - height;

    // 如果键盘高度大于100px，说明键盘弹出了
    if (keyboardHeight > 100) {
      // 找到当前聚焦的输入框

      if (activeElement && (activeElement.tagName === 'INPUT' || activeElement.tagName === 'TEXTAREA')) {
        this.adjustPosition(activeElement, keyboardHeight);
      }
    }

    const inputRect = input.getBoundingClientRect();
    const visibleBottom = window.visualViewport.height - keyboardHeight;

    if (inputRect.bottom > visibleBottom) {

      window.scrollTo({
        top: window.scrollY + offset,
        behavior: 'smooth'
      });
    }
  }
```

### 4.2 键盘处理与自动推起

移动端键盘的处理是一个复杂的问题，不同的浏览器和操作系统有不同的行为。以下是一个完整的键盘处理方案：

```javascript
// 键盘状态管理
class KeyboardManager {
  constructor() {
    this.isKeyboardVisible = false;
    this.keyboardHeight = 0;
    this.listeners = [];

    this.init();
  }

  init() {
    // 方法1：监听window resize（传统方法）
    let lastHeight = window.innerHeight;
    window.addEventListener('resize', () => {
      const currentHeight = window.innerHeight;
      if (currentHeight < lastHeight) {
        // 键盘弹出
        this.keyboardHeight = lastHeight - currentHeight;
        this.isKeyboardVisible = true;
        this.notify('show', this.keyboardHeight);
      } else if (currentHeight > lastHeight) {
        // 键盘收起
        this.isKeyboardVisible = false;
        this.keyboardHeight = 0;
        this.notify('hide', 0);
      }
      lastHeight = currentHeight;
    });

    // 方法2：使用Visual Viewport API（更精确）
    if (window.visualViewport) {
      window.visualViewport.addEventListener('resize', () => {
        const height = window.visualViewport.height;
        const windowHeight = window.innerHeight;
        const newKeyboardHeight = windowHeight - height;

        if (newKeyboardHeight > 100) {
          this.keyboardHeight = newKeyboardHeight;
          this.isKeyboardVisible = true;
          this.notify('show', this.keyboardHeight);
        } else {
          this.isKeyboardVisible = false;
          this.keyboardHeight = 0;
          this.notify('hide', 0);
        }
      });
    }

    // 监听输入框聚焦，自动调整位置
    document.querySelectorAll('input, textarea').forEach(input => {
      input.addEventListener('focus', () => {
        this.scrollToInput(input);
      });
    });
  }

  scrollToInput(input) {
    // 等待键盘完全弹出
    setTimeout(() => {
      const rect = input.getBoundingClientRect();
      const keyboardVisibleHeight = window.visualViewport?.height || window.innerHeight - this.keyboardHeight;

      // 计算需要滚动的距离
      const inputBottom = rect.bottom;
      const visibleTop = window.scrollY;
      const visibleBottom = visibleTop + keyboardVisibleHeight;

      // 如果输入框在键盘下方，需要滚动
      if (inputBottom > visibleBottom - 20) {
        const scrollTop = inputBottom - keyboardVisibleHeight + 20;
        window.scrollTo({
          top: scrollTop,
          behavior: 'smooth'
        });
      }
    }, 300);
  }

  on(event, callback) {
    this.listeners.push({ event, callback });
  }

  notify(event, height) {
    this.listeners.forEach(({ event: e, callback }) => {
      if (e === event) {
        callback(height);
      }
    });
  }
}
```

### 4.3 即时表单验证

表单验证是用户体验的重要环节，即时验证可以减少用户提交表单后的等待时间，提供更好的反馈：

```javascript
// 表单验证器类
class FormValidator {
  constructor(form, options = {}) {
    this.form = form;
    this.fields = new Map();
    this.options = {
      validateOnBlur: true,
      validateOnInput: false,
      showErrorType: 'inline', // 'inline' | 'toast' | 'alert'
      ...options
    };

    this.init();
  }

  init() {
    // 获取所有需要验证的字段
    const inputs = this.form.querySelectorAll('[data-validate]');
    inputs.forEach(input => {
      const rules = this.parseRules(input.dataset.validate);
      this.fields.set(input, {
        input,
        rules,
        errors: [],
        isValid: false
      });

      // 绑定验证事件
      if (this.options.validateOnBlur) {
        input.addEventListener('blur', () => this.validateField(input));
      }

      if (this.options.validateOnInput) {
        input.addEventListener('input', () => this.validateField(input));
      }
    });

    // 绑定表单提交
    this.form.addEventListener('submit', (e) => {
      if (!this.validateAll()) {
        e.preventDefault();
      }
    });
  }

  parseRules(rulesString) {
    const rules = [];
    const ruleParts = rulesString.split(',');

    ruleParts.forEach(part => {
      const [ruleName, param] = part.trim().split(':');
      rules.push({ name: ruleName, param });
    });

    return rules;
  }

  validateField(input) {
    const field = this.fields.get(input);
    if (!field) return true;

    field.errors = [];
    const value = input.value.trim();

    field.rules.forEach(rule => {
      const error = this.executeRule(rule, value, input);
      if (error) {
        field.errors.push(error);
      }
    });

    field.isValid = field.errors.length === 0;
    this.updateUI(input, field);

    return field.isValid;
  }

  executeRule(rule, value, input) {
    switch (rule.name) {
      case 'required':
        if (!value) return '此字段必填';
        break;

      case 'minLength':
        if (value.length < parseInt(rule.param)) {
          return `至少需要${rule.param}个字符`;
        }
        break;

      case 'maxLength':
        if (value.length > parseInt(rule.param)) {
          return `最多只能输入${rule.param}个字符`;
        }
        break;

      case 'pattern':
        const regex = new RegExp(rule.param);
        if (!regex.test(value)) {
          return '格式不正确';
        }
        break;

      case 'email':
        const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
        if (!emailRegex.test(value)) {
          return '请输入有效的邮箱地址';
        }
        break;

      case 'phone':
        const phoneRegex = /^1[3-9]\d{9}$/;
        if (!phoneRegex.test(value)) {
          return '请输入有效的手机号码';
        }
        break;

      case 'idCard':
        const idRegex = /(^\d{15}$)|(^\d{18}$)|(^\d{17}(\d|X|x)$)/;
        if (!idRegex.test(value)) {
          return '请输入有效的身份证号码';
        }
        break;

      case 'custom':
        // 自定义验证函数
        const customValidator = input.dataset.validator;
        if (customValidator && typeof window[customValidator] === 'function') {
          const error = window[customValidator](value);
          if (error) return error;
        }
        break;
    }

    return null;
  }

  updateUI(input, field) {
    const errorElement = this.form.querySelector(`[data-error-for="${input.name}"]`)
                      || input.parentElement.querySelector('.error-message');

    // 移除之前的错误样式
    input.classList.remove('error', 'valid');

    if (field.errors.length > 0) {
      input.classList.add('error');
      if (errorElement) {
        errorElement.textContent = field.errors[0];
        errorElement.style.display = 'block';
      }
    } else if (value) {
      input.classList.add('valid');
      if (errorElement) {
        errorElement.style.display = 'none';
      }
    }
  }

  validateAll() {
    let isAllValid = true;

    this.fields.forEach((field, input) => {
      if (!this.validateField(input)) {
        isAllValid = false;
      }
    });

    if (!isAllValid) {
      // 滚动到第一个错误字段
      const firstError = this.form.querySelector('.error');
      if (firstError) {
        firstError.scrollIntoView({ behavior: 'smooth', block: 'center' });
        firstError.focus();
      }
    }

    return isAllValid;
  }
}
```

### 4.4 防止误触与确认操作

移动端的误触是一个常见问题，特别是在列表页面中，用户可能会不小心点击到某个按钮。以下是几种防止误触的策略：

**方案一：长按确认**

```javascript
// 长按确认组件
class LongPressConfirm {
  constructor(element, options = {}) {
    this.element = element;

  init() {

        </svg>
      </div>

        left: 50%;
        transform: translate(-50%, -50%);

        flex-direction: column;
        align-items: center;

      }

      }

    event.preventDefault();

    this.isLongPressing = true;
    this.progress = 0;

    this.progressIndicator.style.opacity = '1';

    // 启动进度动画
    const startTime = Date.now();
    const duration = this.options.duration;

    this.interval = setInterval(() => {
      const elapsed = Date.now() - startTime;
      this.progress = Math.min(elapsed / duration, 1);

      // 更新进度条
      const circumference = 283; // 2 * PI * 45
      const offset = circumference * (1 - this.progress);
      const progressBar = this.progressIndicator.querySelector('.progress-bar');
      progressBar.style.strokeDashoffset = offset;

      // 达到100%时触发确认
      if (this.progress >= 1) {
        this.triggerConfirm();
      }
    }, 16);

    // 如果手指移出了目标元素，取消长按
    const touch = event.touches[0];
    const rect = this.element.getBoundingClientRect();

    if (touch.clientX < rect.left || touch.clientX > rect.right ||
        touch.clientY < rect.top || touch.clientY > rect.bottom) {
      this.cancelLongPress();
    }
  }

  handleTouchEnd() {
    if (this.isLongPressing && this.progress < 1) {
      this.cancelLongPress();
      if (this.options.onCancel) {
        this.options.onCancel();
      }
    }
  }

  handleTouchCancel() {
    this.cancelLongPress();
  }

  cancelLongPress() {
    this.isLongPressing = false;
    clearInterval(this.interval);
    this.progressIndicator.style.opacity = '0';

    // 重置进度
    const progressBar = this.progressIndicator.querySelector('.progress-bar');
    progressBar.style.strokeDashoffset = 283;
  }

  triggerConfirm() {
    this.isLongPressing = false;
    clearInterval(this.interval);

    // 触发点击效果
    this.element.classList.add('active');
    setTimeout(() => {
      this.element.classList.remove('active');
    }, 150);

    // 隐藏进度指示器
    this.progressIndicator.style.opacity = '0';

    // 执行确认回调
    if (this.options.onConfirm) {
      this.options.onConfirm();
    }
  }
}
```

```javascript
  }

  bindEvents() {
    this.thumb.addEventListener('touchstart', this.handleTouchStart.bind(this), { passive: false });
    document.addEventListener('touchmove', this.handleTouchMove.bind(this), { passive: false });
    document.addEventListener('touchend', this.handleTouchEnd.bind(this), { passive: false });

    window.addEventListener('resize', () => this.updateDimensions());
  }

  handleTouchStart(event) {
    if (this.isConfirmed) return;

    event.preventDefault();
    this.isSliding = true;
    this.startX = event.touches[0].clientX;
    this.thumb.style.transition = 'none';
    this.track.style.transition = 'none';
  }

  handleTouchMove(event) {
    if (!this.isSliding || this.isConfirmed) return;

    event.preventDefault();
    this.currentX = event.touches[0].clientX;
    let deltaX = this.currentX - this.startX;

    // 限制滑动范围
    deltaX = Math.max(0, Math.min(deltaX, this.maxSlide));

    this.thumb.style.transform = `translateX(${deltaX}px)`;

    // 更新透明度
    const progress = deltaX / this.maxSlide;
    this.track.style.opacity = 1 - progress * 0.5;
    this.text.style.opacity = 1 - progress;
  }

  handleTouchEnd() {
    if (!this.isSliding || this.isConfirmed) return;

    this.isSliding = false;

    const deltaX = this.currentX - this.startX;
    const progress = deltaX / this.maxSlide;

    if (progress >= 0.95) {
      // 滑动到位，触发确认
      this.thumb.style.transform = `translateX(${this.maxSlide}px)`;
      this.text.textContent = this.options.successText;
      this.track.classList.add('confirmed');
      this.isConfirmed = true;

      if (this.options.onConfirm) {
        this.options.onConfirm();
      }
    } else {
      // 滑动不到位，回弹
      this.thumb.style.transition = 'transform 0.3s ease';
      this.thumb.style.transform = 'translateX(0)';
      this.track.style.transition = 'opacity 0.3s ease';
      this.track.style.opacity = 1;
      this.text.style.transition = 'opacity 0.3s ease';
      this.text.style.opacity = 1;
    }
  }
}
```

## 五、列表性能优化

### 5.1 长列表与懒渲染

移动端H5应用中，列表是最常见的UI组件。当列表数据较多时，一次性渲染所有列表项会导致严重的性能问题。懒渲染（Lazy Rendering）是一种优化长列表性能的核心技术，它只渲染用户可见区域的列表项，大大减少了DOM节点数量和内存占用。

**懒渲染的核心原理**

懒渲染的核心思想是只渲染当前可见区域的内容。当用户滚动列表时，动态更新可见区域内的列表项。具体实现方式有多种：基于滚动位置的计算、基于Intersection Observer API等。

```javascript
// 懒渲染列表组件
class LazyList {
  constructor(container, options = {}) {
    this.container = container;
    this.options = {
      itemHeight: 60,           // 预估的列表项高度
      bufferCount: 3,           // 缓冲区域（上下各渲染几个额外的项）
      renderItem: null,          // 渲染函数
      onScroll: null,           // 滚动回调
      onLoadMore: null,         // 加载更多回调
      loadMoreThreshold: 100,    // 距离底部多少像素触发加载更多
      ...options
    };

    // 状态
    this.items = [];
    this.scrollTop = 0;
    this.containerHeight = 0;
    this.renderedRange = { start: 0, end: 0 };

    this.init();
  }

  init() {
    // 获取容器尺寸
    this.updateDimensions();

    // 创建列表容器
    this.scrollContainer = this.container;
    this.scrollContainer.style.overflowY = 'auto';
    this.scrollContainer.style.webkitOverflowScrolling = 'touch';

    // 创建内容占位元素（用于维持滚动高度）
    this.spacer = document.createElement('div');
    this.spacer.className = 'list-spacer';
    this.spacer.style.cssText = `
      position: relative;
      width: 100%;
      will-change: transform;
    `;

    // 创建可视区域容器
    this.viewport = document.createElement('div');
    this.viewport.className = 'list-viewport';
    this.viewport.style.cssText = `
      position: absolute;
      top: 0;
      left: 0;
      right: 0;
      transform: translateZ(0);
      will-change: transform;
    `;

    this.spacer.appendChild(this.viewport);
    this.scrollContainer.appendChild(this.spacer);

    // 绑定滚动事件
    this.scrollContainer.addEventListener('scroll', this.handleScroll.bind(this), { passive: true });

    // 窗口大小改变时更新尺寸
    window.addEventListener('resize', this.handleResize.bind(this));
  }

  updateDimensions() {
    this.containerHeight = this.scrollContainer.offsetHeight;
  }

  handleResize() {
    this.updateDimensions();
    this.render();
  }

  handleScroll() {
    // 使用requestAnimationFrame优化滚动性能
    if (this.scrollRAF) return;

    this.scrollRAF = requestAnimationFrame(() => {
      this.scrollTop = this.scrollContainer.scrollTop;
      this.render();

      // 检查是否需要加载更多
      const scrollBottom = this.scrollTop + this.containerHeight;
      const contentHeight = this.spacer.offsetHeight;

      if (contentHeight - scrollBottom < this.options.loadMoreThreshold) {
        if (this.options.onLoadMore && !this.isLoadingMore) {
          this.loadMore();
        }
      }

      if (this.options.onScroll) {
        this.options.onScroll(this.scrollTop, this.scrollTop + this.containerHeight);
      }

      this.scrollRAF = null;
    });
  }

  setItems(items) {
    this.items = items;
    this.spacer.style.height = `${items.length * this.options.itemHeight}px`;
    this.render();
  }

  appendItems(items) {
    const startIndex = this.items.length;
    this.items = this.items.concat(items);
    this.spacer.style.height = `${this.items.length * this.options.itemHeight}px`;

    // 只渲染新增的项目
    this.renderRange(startIndex, this.items.length);
  }

  render() {
    const { itemHeight, bufferCount } = this.options;

    // 计算可见范围
    const startIndex = Math.max(0, Math.floor(this.scrollTop / itemHeight) - bufferCount);
    const endIndex = Math.min(
      this.items.length,
      Math.ceil((this.scrollTop + this.containerHeight) / itemHeight) + bufferCount
    );

    this.renderRange(startIndex, endIndex);
  }

  renderRange(startIndex, endIndex) {
    // 如果范围没有变化，不重新渲染
    if (startIndex === this.renderedRange.start && endIndex === this.renderedRange.end) {
      return;
    }

    this.renderedRange = { start: startIndex, end: endIndex };

    // 清除现有的列表项
    this.viewport.innerHTML = '';

    // 渲染范围内的列表项
    for (let i = startIndex; i < endIndex; i++) {
      const item = this.items[i];
      if (!item) continue;

      const itemElement = this.options.renderItem(item, i);
      itemElement.style.cssText = `
        position: absolute;
        top: ${i * this.options.itemHeight}px;
        left: 0;
        right: 0;
        height: ${this.options.itemHeight}px;
      `;

      this.viewport.appendChild(itemElement);
    }
  }

  async loadMore() {
    if (this.isLoadingMore) return;

    this.isLoadingMore = true;

    try {
      const newItems = await this.options.onLoadMore();
      if (newItems && newItems.length > 0) {
        this.appendItems(newItems);
      }
    } finally {
      this.isLoadingMore = false;
    }
  }

  scrollToIndex(index) {
    const top = index * this.options.itemHeight;
    this.scrollContainer.scrollTo({ top, behavior: 'smooth' });
  }

  destroy() {
```

### 5.2 图片懒加载

图片懒加载是移动端性能优化的另一个关键技术。很多H5页面中，图片占据了大量的带宽资源，一次性加载所有图片会导致首屏加载缓慢、流量浪费等问题。图片懒加载可以让图片在进入可视区域时才加载，提升首屏性能和用户体验。

**基于Intersection Observer的实现**

```javascript
// 图片懒加载组件
class LazyImage {
  constructor(options = {}) {
    this.options = {
      root: null,                    // 视口容器，null表示浏览器视口
      rootMargin: '50px',           // 提前加载的距离
      threshold: 0,                 // 触发时机
      placeholder: 'data:image/svg+xml;base64,...', // 占位图
      onLoad: null,                 // 加载成功回调
      onError: null,               // 加载失败回调
      ...options
    };

    this.images = [];
    this.observer = null;

    this.init();
  }

  init() {
    // 创建Intersection Observer
    if ('IntersectionObserver' in window) {
      this.observer = new IntersectionObserver(
        this.handleIntersect.bind(this),
        {
          root: this.options.root,
          rootMargin: this.options.rootMargin,
          threshold: this.options.threshold
        }
      );
    } else {
      // 降级处理：直接加载所有图片
      this.loadAllImages();
    }
  }

  // 添加图片
  addImage(element) {
    if (!element) return;

    // 保存原始src
    const src = element.dataset.src || element.dataset.original || element.src;
    element.dataset.src = src;

    // 设置占位图
    if (this.options.placeholder) {
      element.src = this.options.placeholder;
    }

    // 添加loaded类
    element.classList.add('lazy-image');

    // 监听图片加载完成
    element.addEventListener('load', () => {
      element.classList.add('loaded');
      if (this.options.onLoad) {
        this.options.onLoad(element);
      }
    });

    element.addEventListener('error', () => {
      element.classList.add('error');
      if (this.options.onError) {
        this.options.onError(element);
      }
    });

    // 开始观察
    if (this.observer) {
      this.observer.observe(element);
    }
  }

  // 批量添加图片
  addImages(elements) {
    elements.forEach(element => this.addImage(element));
  }

  handleIntersect(entries, observer) {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        this.loadImage(entry.target);
        observer.unobserve(entry.target);
      }
    });
  }

  loadImage(element) {
    const src = element.dataset.src;
    if (!src) return;

    // 创建新图片对象预加载
    const img = new Image();
    img.onload = () => {
      element.src = src;
      element.classList.add('loaded');
    };
    img.onerror = () => {
      element.classList.add('error');
    };
    img.src = src;
  }

  loadAllImages() {
    this.images.forEach(img => this.loadImage(img));
  }

  destroy() {
    if (this.observer) {
      this.observer.disconnect();
    }
  }
}

// 便捷函数：自动处理所有图片
function setupLazyImages(options = {}) {
  const lazyImage = new LazyImage(options);

  // 查找所有需要懒加载的图片
  const images = document.querySelectorAll('img[data-src], img[data-original]');
  lazyImage.addImages(images);

  return lazyImage;
}
```

```javascript
// 下拉刷新组件
class PullToRefresh {
  constructor(container, options = {}) {
    this.container = container;
    this.options = {
      pullThreshold: 80,           // 触发刷新的下拉距离
      maxPullDistance: 120,        // 最大下拉距离
      refreshTip: '下拉刷新',
      releaseTip: '释放刷新',
      refreshingTip: '刷新中...',
      onRefresh: null,             // 刷新回调

    this.isPulling = false;
    this.pullDistance = 0;

    this.init();
  }

  init() {
    // 创建提示元素
    this.createTipElement();

    // 绑定触摸事件
    this.bindTouchEvents();
  }

  createTipElement() {
    this.tipElement = document.createElement('div');
    this.tipElement.className = 'pull-to-refresh-tip';
    this.tipElement.innerHTML = `
      <div class="tip-content">
        <div class="tip-arrow"></div>
        <div class="tip-text">${this.options.refreshTip}</div>
      </div>
    `;

    const style = document.createElement('style');
    style.textContent = `
      .pull-to-refresh-tip {
        position: absolute;
        top: -50px;
        left: 0;
        right: 0;
        height: 50px;
        display: flex;
        align-items: center;
        justify-content: center;
        transition: transform 0.2s;
      }

      .tip-content {
        display: flex;
        flex-direction: column;
        align-items: center;
      }

      .tip-arrow {
        width: 16px;
        height: 16px;
        border: 2px solid #999;
        border-top: none;
        border-right: none;
        transform: rotate(-135deg);
        transition: transform 0.2s;
      }

      .tip-arrow.pulled {
        transform: rotate(45deg);
      }

      .tip-text {
        font-size: 12px;
        color: #666;
        margin-top: 4px;
      }

      .pull-to-refresh-tip.refreshing .tip-arrow {
        animation: rotate 1s linear infinite;
      }

      @keyframes rotate {
        from { transform: rotate(0deg); }
        to { transform: rotate(360deg); }
      }
    `;

    document.head.appendChild(style);

    // 插入到容器顶部
    this.container.style.position = 'relative';
    this.container.insertBefore(this.tipElement, this.container.firstChild);
  }

  bindTouchEvents() {
    let startY = 0;
    let startScrollTop = 0;

    this.container.addEventListener('touchstart', (e) => {
      startY = e.touches[0].clientY;
      startScrollTop = this.container.scrollTop;

      // 只有在顶部时才启用下拉刷新
      if (startScrollTop === 0) {
        this.isPulling = true;
      }
    }, { passive: true });

    this.container.addEventListener('touchmove', (e) => {
      if (!this.isPulling) return;
      if (this.isRefreshing) return;

      const currentY = e.touches[0].clientY;
      const deltaY = currentY - startY;

      // 只有下拉时才处理
      if (deltaY > 0) {
        // 计算下拉距离（带阻尼效果）
        this.pullDistance = Math.min(
          this.options.maxPullDistance,
          Math.pow(deltaY, 0.8)
        );

        // 更新UI
        this.updateUI();

        // 阻止默认滚动
        if (this.pullDistance > 10) {
          e.preventDefault();
        }
      }
    }, { passive: false });

    this.container.addEventListener('touchend', () => {
      if (!this.isPulling) return;

      this.isPulling = false;

      if (this.pullDistance >= this.options.pullThreshold) {
        // 触发刷新
        this.startRefresh();
      } else {
        // 回弹
        this.reset();
      }
    }, { passive: true });
  }

  updateUI() {
    const { pullThreshold, refreshTip, releaseTip } = this.options;

    // 更新提示文字和箭头方向
    const arrow = this.tipElement.querySelector('.tip-arrow');
    const text = this.tipElement.querySelector('.tip-text');

    if (this.pullDistance >= pullThreshold) {
      text.textContent = releaseTip;
      arrow.classList.add('pulled');
    } else {
      text.textContent = refreshTip;
      arrow.classList.remove('pulled');
    }

    // 更新位置
    this.tipElement.style.transform = `translateY(${this.pullDistance}px)`;
  }

  async startRefresh() {
    this.isRefreshing = true;
    this.pullDistance = this.options.pullThreshold;
    this.updateUI();

    // 添加刷新状态样式
    this.tipElement.classList.add('refreshing');
    const text = this.tipElement.querySelector('.tip-text');
    text.textContent = this.options.refreshingTip;

    // 执行刷新回调
    if (this.options.onRefresh) {
      try {
        await this.options.onRefresh();
      } catch (error) {
        console.error('刷新失败:', error);
      }
    }

    // 完成后重置
    setTimeout(() => {
      this.reset();
    }, 500);
  }

  reset() {
    this.isRefreshing = false;
    this.pullDistance = 0;

    this.tipElement.classList.remove('refreshing');
    this.tipElement.style.transform = 'translateY(0)';

    const text = this.tipElement.querySelector('.tip-text');
    text.textContent = this.options.refreshTip;

    const arrow = this.tipElement.querySelector('.tip-arrow');
    arrow.classList.remove('pulled');
  }
}
```

## 六、动画性能优化

### 6.1 动画性能基础

移动端H5动画的性能直接影响用户体验。在开发过程中，需要时刻关注动画的流畅度，目标通常是60fps（每秒60帧）。为了达到这个目标，需要深入理解浏览器渲染机制和动画性能优化的核心原则。

**浏览器渲染流程**

浏览器的渲染流程主要包括以下几个步骤：JavaScript执行 → 样式计算（Style）→ 布局（Layout）→ 绘制（Paint）→ 合成（Composite）。每个步骤都可能成为动画性能瓶颈。

动画性能优化的核心原则是**避免触发重排（Reflow）和重绘（Repaint）**，直接进行合成层操作。这是因为重排和重绘会触发大量的计算和绘图操作，非常消耗性能，而合成操作只需要GPU参与，相对轻量。

**哪些属性可以触发合成**

合成层操作主要涉及`transform`和`opacity`两个属性，因为这两个属性的变化不会触发重排和重绘，只会触发合成：

- `transform: translateX/Y/Z`, `scale()`, `rotate()` 等
- `opacity`

```css
/* 正确的动画方式 */
.good-animation {
  transform: translateX(100px);
  transition: transform 0.3s ease;
}

/* 错误的动画方式（会触发重排） */
.bad-animation {
  left: 100px;
  transition: left 0.3s ease;
}

/* 错误的动画方式（会触发重绘） */
.bad-animation-2 {
  background-color: red;
  transition: background-color 0.3s ease;
}
```

### 6.2 requestAnimationFrame控制

`requestAnimationFrame`（简称rAF）是浏览器提供的用于动画渲染的API，它会在下一帧渲染前调用回调函数，确保动画在最佳的时机进行渲染，实现流畅的动画效果。

**rAF基础用法**

```javascript
// 基本的rAF动画
function animate(element, from, to, duration) {
  const startTime = performance.now();

  function update(currentTime) {
    const elapsed = currentTime - startTime;
    const progress = Math.min(elapsed / duration, 1);

    // 使用easing函数让动画更自然
    const easedProgress = easeOutCubic(progress);

    // 计算当前值
    const currentValue = from + (to - from) * easedProgress;
    element.style.transform = `translateX(${currentValue}px)`;

    if (progress < 1) {
      requestAnimationFrame(update);
    }
  }

  requestAnimationFrame(update);
}

// 常用的easing函数
function easeOutCubic(t) {
  return 1 - Math.pow(1 - t, 3);
}

function easeInOutQuad(t) {
  return t < 0.5
    ? 2 * t * t
    : 1 - Math.pow(-2 * t + 2, 2) / 2;
}

function easeOutElastic(t) {
  const c4 = (2 * Math.PI) / 3;
  return t === 0
    ? 0
    : t === 1
    ? 1
    : Math.pow(2, -10 * t) * Math.sin((t * 10 - 0.75) * c4) + 1;
}
```

**rAF封装类**

```javascript
// 动画控制器类

    this.animations = new Map();

      element,

      startTime: null,

    return id;
  }

  start(animation) {
    animation.startTime = performance.now();
    animation.isPlaying = true;
    this.tick(animation);
  }

  tick(animation) {
    if (!animation.isPlaying) return;

    const elapsed = performance.now() - animation.startTime;
    const progress = Math.min(elapsed / animation.duration, 1);
    const easedProgress = animation.easing(progress);

    // 计算当前值
    const currentValue = animation.from + (animation.to - animation.from) * easedProgress;

    // 应用到元素
    if (animation.property === 'transform') {
      animation.element.style.transform = `translateX(${currentValue}px)`;
    } else if (animation.property === 'opacity') {
      animation.element.style.opacity = currentValue;
    } else if (animation.property === 'scale') {
      animation.element.style.transform = `scale(${currentValue})`;
    }

    // 触发回调
    if (animation.onUpdate) {
      animation.onUpdate(currentValue, progress);
    }

    if (progress < 1) {
      requestAnimationFrame(() => this.tick(animation));
    } else {
      animation.isPlaying = false;
      if (animation.onComplete) {
        animation.onComplete();
      }
      this.animations.delete(animation.id);
    }
  }

  // 停止动画
  stop(id) {
    const animation = this.animations.get(id);
    if (animation) {
      animation.isPlaying = false;
      this.animations.delete(id);
    }
  }

  // 停止所有动画
  stopAll() {
    this.animations.forEach((animation) => {
      animation.isPlaying = false;
    });
    this.animations.clear();
  }

  // 暂停/恢复（需要修改tick逻辑支持）
  pause(id) {
    const animation = this.animations.get(id);
    if (animation) {
      animation.isPlaying = false;
    }
  }
}
```

### 6.3 CSS动画与JS动画的选择

在实际开发中，需要根据场景选择CSS动画还是JS动画。

**CSS动画适用场景**

- 简单的状态切换（hover、active等）
- 只需要触发一次的标准过渡效果
- 对性能要求不高的动画
- 只需要正向播放的动画

**JS动画适用场景**

- 复杂的动画序列
- 需要精细控制的动画
- 需要暂停、倒退、逐帧控制的动画
- 与用户交互紧密的动画
- 需要动态计算起始值的动画

```javascript
// 手势驱动的动画示例（JS动画）
class GestureAnimation {
  constructor(element) {
    this.element = element;
    this.isDragging = false;
    this.startX = 0;
    this.currentX = 0;
    this.velocity = 0;
    this.lastX = 0;
    this.lastTime = 0;

    this.bindEvents();
  }

  bindEvents() {
    this.element.addEventListener('touchstart', this.onTouchStart.bind(this), { passive: false });
    document.addEventListener('touchmove', this.onTouchMove.bind(this), { passive: false });
    document.addEventListener('touchend', this.onTouchEnd.bind(this), { passive: false });
  }

  onTouchStart(e) {
    this.isDragging = true;
    this.startX = e.touches[0].clientX;
    this.lastX = this.startX;
    this.lastTime = performance.now();
    this.velocity = 0;

    e.preventDefault();
  }

  onTouchMove(e) {
    if (!this.isDragging) return;

    const currentTime = performance.now();
    const currentX = e.touches[0].clientX;

    // 计算速度
    const deltaTime = currentTime - this.lastTime;
    if (deltaTime > 0) {
      this.velocity = (currentX - this.lastX) / deltaTime;
    }

    this.currentX = currentX;
    this.lastX = currentX;
    this.lastTime = currentTime;

    // 直接更新transform（不触发重排）
    const deltaX = currentX - this.startX;
    this.element.style.transform = `translateX(${deltaX}px)`;

    e.preventDefault();
  }

  onTouchEnd() {
    if (!this.isDragging) return;
    this.isDragging = false;

    // 根据速度决定最终位置
    const targetX = this.currentX + this.velocity * 200;
    const threshold = window.innerWidth / 3;

    let finalX;
    if (Math.abs(targetX) > threshold) {
      // 切换到下一页
      finalX = targetX > 0 ? window.innerWidth : -window.innerWidth;
    } else {
      // 回弹
      finalX = 0;
    }

    // 使用CSS transition平滑过渡
    this.element.style.transition = 'transform 0.3s ease-out';
    this.element.style.transform = `translateX(${finalX}px)`;

    // 动画结束后清除transition
    setTimeout(() => {
      this.element.style.transition = 'none';
    }, 300);
  }
}
```

### 6.4 节流与防抖在滚动中的应用

滚动事件是移动端最常见的高频事件之一，如果不进行节流处理，每次滚动都会触发大量的事件处理函数，严重影响性能。

**节流（Throttle）**

节流是限制函数的执行频率，在一定时间间隔内只执行一次。

```javascript
// 节流函数
function throttle(func, limit) {
  let inThrottle = false;

  return function(...args) {
    if (!inThrottle) {
      func.apply(this, args);
      inThrottle = true;
      setTimeout(() => {
        inThrottle = false;
      }, limit);
    }
  };
}

// 使用示例
const handleScroll = throttle(() => {
  console.log('滚动位置:', window.scrollY);
  // 执行滚动相关的操作
}, 100);

window.addEventListener('scroll', handleScroll, { passive: true });
```

**防抖（Debounce）**

防抖是延迟函数的执行，只有在事件停止触发一段时间后才执行。

```javascript
// 防抖函数
function debounce(func, delay) {
  let timeoutId = null;

  return function(...args) {
    clearTimeout(timeoutId);
    timeoutId = setTimeout(() => {
      func.apply(this, args);
    }, delay);
  };
}

// 使用示例
const handleResize = debounce(() => {
  console.log('窗口大小:', window.innerWidth, window.innerHeight);
  // 执行尺寸相关的操作
}, 300);

window.addEventListener('resize', handleResize);
```

**requestAnimationFrame节流**

这是处理滚动动画的最佳方式，结合了rAF的刷新同步和节流的性能优化：

```javascript
// rAF节流
function rafThrottle(func) {
  let ticking = false;

  return function(...args) {
    if (!ticking) {
      requestAnimationFrame(() => {
        func.apply(this, args);
        ticking = false;
      });
      ticking = true;
    }
  };
}

// 使用示例
const updateScrollIndicator = rafThrottle(() => {
  const scrollTop = window.scrollY;
  const scrollHeight = document.documentElement.scrollHeight - window.innerHeight;
  const progress = scrollTop / scrollHeight;

  // 更新滚动指示器
  const indicator = document.querySelector('.scroll-indicator');
  if (indicator) {
    indicator.style.height = `${progress * 100}%`;
  }
});

window.addEventListener('scroll', updateScrollIndicator, { passive: true });
```

### 6.5 60fps目标：我的思考

在移动端H5开发中，追求60fps的流畅动画是每个开发者都应该努力的目标。这不仅是技术要求，更是对用户体验的尊重。

**为什么是60fps？**

60fps意味着每帧的渲染时间只有约16.67毫秒。如果动画的帧率低于这个值，人眼就能感受到明显的卡顿，影响体验。特别是在移动设备上，由于硬件性能相对有限，更需要精心优化才能达到流畅的动画效果。

**如何确保60fps？**

首先，必须使用Chrome DevTools的Performance面板进行性能分析，找出导致卡顿的瓶颈所在。常见的性能问题包括：频繁触发重排的JavaScript、低效的CSS选择器、未使用硬件加速的动画等。

其次，合理使用CSS transform和opacity进行动画。这两个属性的变化会跳过Layout和Paint阶段，直接进入Compositing阶段，由GPU完成合成，大大提高了渲染效率。

第三，避免在动画过程中创建新的JavaScript对象或进行复杂的计算。这些操作会占用主线程的时间，影响动画的流畅度。如果确实需要进行复杂的计算，考虑使用Web Worker在后台线程执行。

第四，对于长列表等复杂UI组件，务必使用懒渲染技术。只渲染可见区域的内容，可以大大减少DOM节点数量，降低渲染压力。

最后，动画应该尊重用户的偏好设置。如果用户在系统设置中开启了"减弱动画效果"，应该适当减少或禁用某些动画。这不仅是更好的用户体验，也是Accessibility的基本要求。

### 7.1 Service Worker离线缓存

```javascript
const CACHE_NAME = 'my-app-cache-v1';
const RUNTIME_CACHE = 'runtime-cache-v1';

const PRECACHE_URLS = [
  '/',

  '/scripts/main.js',
  '/images/logo.png',
  '/fonts/iconfont.woff2'

  console.log('Service Worker: 安装中');

  event.waitUntil(
    caches.open(CACHE_NAME)
      .then((cache) => {
        console.log('Service Worker: 预缓存资源');
        return cache.addAll(PRECACHE_URLS);
      })
      .then(() => {
        console.log('Service Worker: 跳过等待');
        return self.skipWaiting();
      })
  );
});

// 激活事件 - 清理旧缓存
self.addEventListener('activate', (event) => {
  console.log('Service Worker: 激活中');

  event.waitUntil(
    caches.keys()
      .then((cacheNames) => {
        return Promise.all(
          cacheNames
            .filter((name) => name !== CACHE_NAME && name !== RUNTIME_CACHE)
            .map((name) => {
              console.log('Service Worker: 删除旧缓存', name);
              return caches.delete(name);
            })
        );
      })
      .then(() => {
        console.log('Service Worker: 接管所有页面');
        return self.clients.claim();
      })
  );
});

// 抓取事件 - 实现缓存策略
self.addEventListener('fetch', (event) => {
  const { request } = event;
  const url = new URL(request.url);

  // 只处理同源请求
  if (url.origin !== location.origin) {
    return;
  }

  // 根据请求类型使用不同的缓存策略
  if (request.mode === 'navigate') {
    // 导航请求：优先网络，失败时回退到缓存
    event.respondWith(networkFirst(request));
  } else if (request.destination === 'image' || request.destination === 'font') {
    // 图片和字体：优先缓存
    event.respondWith(cacheFirst(request));
  } else {
    // 其他请求：缓存优先，回退到网络
    event.respondWith(staleWhileRevalidate(request));
  }
});

// 缓存优先策略
async function cacheFirst(request) {
  const cachedResponse = await caches.match(request);
  if (cachedResponse) {
    return cachedResponse;
  }

  try {
    const networkResponse = await fetch(request);
    if (networkResponse.ok) {
      const cache = await caches.open(RUNTIME_CACHE);
      cache.put(request, networkResponse.clone());
    }
    return networkResponse;
  } catch (error) {
    console.error('缓存优先策略失败:', error);
    return new Response('离线资源不可用', { status: 503 });
  }
}

// 网络优先策略
async function networkFirst(request) {
  try {
    const networkResponse = await fetch(request);
    if (networkResponse.ok) {
      const cache = await caches.open(CACHE_NAME);
      cache.put(request, networkResponse.clone());
    }
    return networkResponse;
  } catch (error) {
    console.log('网络请求失败，使用缓存');
    const cachedResponse = await caches.match(request);
    if (cachedResponse) {
      return cachedResponse;
    }

    // 如果是导航请求，返回离线页面
    if (request.mode === 'navigate') {
      return caches.match('/offline.html');
    }

    return new Response('离线资源不可用', { status: 503 });
  }
}

// 缓存时重新验证策略
async function staleWhileRevalidate(request) {
  const cachedResponse = await caches.match(request);

  const fetchPromise = fetch(request)
    .then((networkResponse) => {
      if (networkResponse.ok) {
        caches.open(RUNTIME_CACHE)
          .then((cache) => cache.put(request, networkResponse.clone()));
      }
      return networkResponse;
    })
    .catch(() => cachedResponse);

  return cachedResponse || fetchPromise;
}
```

### 7.4 离线优先策略

离线优先是一种开发模式，假设网络不可用，优先从本地获取数据，同时在后台同步到服务器。

```javascript
// 离线优先数据管理器
class OfflineFirstManager {
  constructor(options = {}) {
    this.apiBase = options.apiBase || '/api';
    this.db = options.db;
    this.syncManager = options.syncManager;
    this.cacheStrategy = new CacheStrategy(options.cacheOptions);
  }

  // 获取数据（离线优先）
  async fetchData(resource, params = {}) {
    const cacheKey = `${resource}:${JSON.stringify(params)}`;

    // 1. 首先尝试从缓存获取
    const cachedData = await this.db.get('cache', cacheKey);
    if (cachedData && !this.isStale(cachedData)) {
      // 缓存命中且未过期
      this.refreshInBackground(resource, params, cacheKey);
      return cachedData.data;
    }

    // 2. 如果没有缓存或缓存过期，尝试网络请求
    if (navigator.onLine) {
      try {
        const response = await fetch(
          `${this.apiBase}/${resource}?${new URLSearchParams(params)}`
        );

        if (response.ok) {
          const data = await response.json();

          // 更新缓存
          await this.db.put('cache', {
            key: cacheKey,
            data,
            timestamp: Date.now(),
            ttl: 5 * 60 * 1000 // 5分钟缓存
          });

          return data;
        }
      } catch (error) {
        console.error('网络请求失败:', error);
      }
    }

    // 3. 网络不可用或请求失败，返回过期缓存
    if (cachedData) {
      console.log('使用过期缓存');
      return cachedData.data;
    }

    // 4. 没有任何可用数据，抛出错误
    throw new Error('无可用数据');
  }

  // 后台刷新
  async refreshInBackground(resource, params, cacheKey) {
    if (!navigator.onLine) return;

    try {
      const response = await fetch(
        `${this.apiBase}/${resource}?${new URLSearchParams(params)}`
      );

      if (response.ok) {
        const data = await response.json();
        await this.db.put('cache', {
          key: cacheKey,
          data,
          timestamp: Date.now(),
          ttl: 5 * 60 * 1000
        });
      }
    } catch (error) {
      // 后台刷新失败，忽略
    }
  }

  // 检查缓存是否过期
  isStale(cachedData) {
    return Date.now() - cachedData.timestamp > cachedData.ttl;
  }

  // 保存数据（离线队列）
  async saveData(resource, data) {
    const localId = `local-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;

    // 1. 先保存到本地
    const localData = {
      id: localId,
      ...data,
      createdAt: Date.now(),
      syncStatus: 'pending'
    };

    await this.db.put(resource, localData);

    // 2. 尝试立即同步
    if (navigator.onLine) {
      try {
        const response = await fetch(`${this.apiBase}/${resource}`, {
          method: 'POST',
          body: JSON.stringify(data)
        });

        if (response.ok) {
          // 同步成功，标记为已同步
          localData.syncStatus = 'synced';
          localData.serverId = (await response.json()).id;
          await this.db.put(resource, localData);
          return localData;
        }
      } catch (error) {
        console.error('同步失败:', error);
      }
    }

    // 3. 同步失败，数据将在后台同步
    return localData;
  }
}
```

```javascript
// 弱网模拟器
class NetworkThrottler {
  constructor() {
    this.latency = 0;
    this.downloadSpeed = Infinity; // KB/s
    this.uploadSpeed = Infinity;   // KB/s
    this.isActive = false;
  }

  enable(options = {}) {
    const {
      latency = 100,        // 延迟（毫秒）
      downloadSpeed = 500,  // 下载速度（KB/s）
      uploadSpeed = 200     // 上传速度（KB/s）
    } = options;

    this.latency = latency;
    this.downloadSpeed = downloadSpeed;
    this.uploadSpeed = uploadSpeed;
    this.isActive = true;

    this.interceptFetch();
    this.interceptXHR();

    console.log(`弱网模拟已启用: 延迟${latency}ms, 下载${downloadSpeed}KB/s, 上传${uploadSpeed}KB/s`);
  }

  disable() {
    this.isActive = false;
    console.log('弱网模拟已禁用');
  }

  interceptFetch() {
    const originalFetch = window.fetch;

    window.fetch = async (input, init) => {
      if (!this.isActive) {
        return originalFetch(input, init);
      }

      // 添加延迟
      await this.sleep(this.latency);

      const request = input instanceof Request ? input : new Request(input, init);
      const body = init?.body || request.clone().body;

      // 如果有请求体，模拟上传速度
      if (body) {
        const bodySize = await this.getBodySize(body);
        const uploadTime = (bodySize / this.uploadSpeed) * 1000;
        await this.sleep(uploadTime);
      }

      // 发起请求
      const response = await originalFetch(request);

      // 模拟下载速度
      const responseBody = response.clone().body;
      const reader = responseBody.getReader();
      const chunks = [];

      while (true) {
        const { done, value } = await reader.read();
        if (done) break;

        // 模拟下载延迟
        const downloadTime = (value.length / this.downloadSpeed) * 1000;
        await this.sleep(downloadTime);
        chunks.push(value);
      }

      // 重新构建响应
      const body = new ReadableStream({
        start(controller) {
          chunks.forEach(chunk => controller.enqueue(chunk));
          controller.close();
        }
      });

      return new Response(body, {
        status: response.status,
        statusText: response.statusText,
        headers: response.headers
      });
    };
  }

  interceptXHR() {
    const originalOpen = XMLHttpRequest.prototype.open;

    XMLHttpRequest.prototype.open = function(method, url, async, user, password) {
      this._throttleMethod = method;
      this._throttleUrl = url;
      return originalOpen.apply(this, arguments);
    };

    const originalSend = XMLHttpRequest.prototype.send;

    XMLHttpRequest.prototype.send = async function(data) {
      if (!NetworkThrottler.isActive) {
        return originalSend.apply(this, arguments);
      }

      // 添加延迟
      await NetworkThrottler.sleep(NetworkThrottler.latency);

      // 模拟上传
      if (data) {
        const size = new Blob([data]).size;
        const uploadTime = (size / NetworkThrottler.uploadSpeed) * 1000;
        await NetworkThrottler.sleep(uploadTime);
      }

      // 调用原send
      return originalSend.apply(this, arguments);
    };
  }

  async getBodySize(body) {
    if (body instanceof Blob) {
      return body.size;
    }
    if (body instanceof FormData) {
      // 估算FormData大小
      return 1024; // 默认1KB
    }
    if (typeof body === 'string') {
      return new Blob([body]).size;
    }
    return 0;
  }

  sleep(ms) {
    return new Promise(resolve => setTimeout(resolve, ms));
  }

  static get isActive() {
    return window.__networkThrottler__?.isActive;
  }

  static get latency() {
    return window.__networkThrottler__?.latency || 0;
  }

  static get uploadSpeed() {
    return window.__networkThrottler__?.uploadSpeed || Infinity;
  }
}

// 创建全局实例
window.__networkThrottler__ = new NetworkThrottler();

// 快捷方法
window.enableSlowNetwork = (preset = '3g') => {
  const presets = {
    'slow-2g': { latency: 500, downloadSpeed: 50, uploadSpeed: 30 },
    '2g': { latency: 300, downloadSpeed: 100, uploadSpeed: 50 },
    '3g': { latency: 100, downloadSpeed: 400, uploadSpeed: 200 },
    '4g': { latency: 50, downloadSpeed: 2000, uploadSpeed: 1000 }
  };

  window.__networkThrottler__.enable(presets[preset] || presets['3g']);
};

window.disableSlowNetwork = () => {
  window.__networkThrottler__.disable();
};

// 使用示例
/*
在浏览器控制台中执行：

disableSlowNetwork()     // 禁用网络模拟
*/
```

### 9.1 iOS Safari兼容

iOS Safari是移动端浏览器中份额最大的之一，但它也以各种奇怪的兼容性问题著称。以下是常见的iOS Safari兼容问题及其解决方案。

**iOS Safari特有的问题**

1. **安全区域问题**：iPhone X及以上的全面屏设备，需要处理刘海和圆角区域
2. **橡皮筋效果**：iOS Safari在页面滚动到边界时的弹性效果
3. **输入框聚焦问题**：键盘弹出时页面的处理
4. **300ms点击延迟**：早期iOS Safari的点击延迟问题
5. **position:fixed在输入框聚焦时失效**：iOS Safari的一个经典bug

```javascript
// iOS Safari兼容处理
const IOSFix = {
  // 检测是否为iOS设备
  isIOS() {
    return /iPad|iPhone|iPod/.test(navigator.userAgent) && !window.MSStream;
  },

  // 检测是否为iOS Safari
  isSafari() {
    return /iP(ad|od|hone)/.test(navigator.userAgent) &&
           /Safari/.test(navigator.userAgent) &&
           !/Chrome/.test(navigator.userAgent);
  },

  // 修复iOS Safari position:fixed在输入框聚焦时的问题
  fixPositionFixedInInput() {
    if (!this.isIOS()) return;

    const inputs = document.querySelectorAll('input, textarea');
    const fixedElements = document.querySelectorAll('.fixed-top, .fixed-bottom, .modal');

    let originalStyles = new Map();

    inputs.forEach(input => {
      input.addEventListener('focus', () => {
        fixedElements.forEach(el => {
          originalStyles.set(el, el.style.position);
          el.style.position = 'absolute';
        });
      });

      input.addEventListener('blur', () => {
        fixedElements.forEach(el => {
          el.style.position = originalStyles.get(el) || 'fixed';
        });
      });
    });
  },

  // 修复iOS Safari橡皮筋效果导致的scroll事件异常
  fixRubberBandScroll(callback) {
    if (!this.isIOS()) return;

    let lastScrollY = window.scrollY;
    let lastScrollTop = 0;

    window.addEventListener('scroll', () => {
      const scrollY = window.scrollY;
      const scrollTop = document.documentElement.scrollTop;

      // 检测是否为橡皮筋效果（滚动超出边界）
      if ((scrollY < 0 && lastScrollY <= 0) ||
          (scrollTop < 0 && lastScrollTop <= 0) ||
          (scrollY > document.documentElement.scrollHeight - window.innerHeight &&
           lastScrollY >= document.documentElement.scrollHeight - window.innerHeight)) {
        // 触发边界回弹回调
        if (callback) callback('rubber-band');
      }

      lastScrollY = scrollY;
      lastScrollTop = scrollTop;
    }, { passive: true });
  },

  // 禁用iOS Safari的双击缩放
  disableDoubleTapZoom() {
    if (!this.isIOS()) return;

    let lastTouchEnd = 0;

    document.addEventListener('touchend', (e) => {
      const now = Date.now();
      if (now - lastTouchEnd <= 300) {
        e.preventDefault();
      }
      lastTouchEnd = now;
    }, { passive: false });

    document.addEventListener('touchstart', () => {
      lastTouchEnd = Date.now();
    }, { passive: true });
  },

  // 修复iOS Safari输入框自动放大
  preventAutoZoom() {
    if (!this.isIOS()) return;

    // iOS Safari会在输入框聚焦时自动放大页面
    // 解决方案是将输入框的font-size设置为至少16px
    const inputs = document.querySelectorAll('input, textarea, select');

    inputs.forEach(input => {
      const fontSize = parseFloat(getComputedStyle(input).fontSize);
      if (fontSize < 16) {
        input.style.fontSize = '16px';
      }
    });
  }
};

// 在页面加载后应用修复
document.addEventListener('DOMContentLoaded', () => {
  IOSFix.fixPositionFixedInInput();
  IOSFix.preventAutoZoom();
});
```

```javascript
// Android WebView兼容处理
const AndroidFix = {
  // 常见Android WebView问题及解决方案
  knownIssues: {
    // Android 5.0以下不支持WebView中的flexbox
    'android-5.0': {
      issue: 'Flexbox兼容性问题',

    },

    'android-4.4': {
      issue: 'CSS transition/transform问题',
      solution: '使用translateZ(0)触发GPU加速'
    }
  },

  // 应用兼容性修复
  applyPolyfills() {
    // 检测是否需要polyfill
    if (this.needFlexboxPolyfill()) {
      this.loadFlexboxPolyfill();
    }

    if (this.needIntersectionObserverPolyfill()) {
      this.loadIntersectionObserverPolyfill();
    }
  },

  needFlexboxPolyfill() {
    const version = WebViewDetector.getAndroidVersion();
    return version && version < 5.0;
  },

  needIntersectionObserverPolyfill() {
    return typeof IntersectionObserver === 'undefined';
  },

  loadFlexboxPolyfill() {
    // 加载flexibility.js
    const script = document.createElement('script');
    script.src = 'https://cdn.jsdelivr.net/npm/flexibility@2.0.1/flexibility.js';
    document.head.appendChild(script);
  },

  loadIntersectionObserverPolyfill() {
    const script = document.createElement('script');
    script.src = 'https://cdn.jsdelivr.net/npm/intersection-observer@0.12.0/intersection-observer.js';
    document.head.appendChild(script);
  },

  // 修复Android WebView中的click事件延迟
  fixClickDelay() {
    // Android WebView默认有300ms的点击延迟
    // 通过设置viewport和touch-action来修复
    const meta = document.querySelector('meta[name="viewport"]');
    if (meta) {
      const content = meta.getAttribute('content') || '';
      if (!content.includes('user-scalable=no')) {
        meta.setAttribute('content', content + ', user-scalable=no');
      }
    }
```

### 10.1 摇一摇功能

DeviceMotion API可以检测设备的加速度变化，利用这个特性可以实现摇一摇功能。

```javascript
// 摇一摇检测器
class ShakeDetector {
  constructor(options = {}) {
    this.options = {
      threshold: 15,          // 加速度阈值
      timeout: 1000,          // 两次摇动之间的最小间隔（毫秒）
      onShake: null,          // 摇动回调
      onShakeStart: null,     // 开始摇动回调
      onShakeEnd: null,       // 结束摇动回调
      ...options
    };

    this.lastShakeTime = 0;
    this.lastAcceleration = { x: 0, y: 0, z: 0 };
    this.isShaking = false;
    this.shakeCount = 0;

    this.init();
  }

  init() {
    // 检查DeviceMotion支持
    if (!('DeviceMotionEvent' in window)) {
      console.warn('DeviceMotion API不支持');
      return;
    }

    // iOS 13+需要请求权限
    this.requestPermission();
  }

  async requestPermission() {
    // iOS 13+需要用户授权
    if (typeof DeviceMotionEvent !== 'undefined' &&
        typeof DeviceMotionEvent.requestPermission === 'function') {
      try {
        const permission = await DeviceMotionEvent.requestPermission();
        if (permission === 'granted') {
          this.bindEvents();
        } else {
          console.warn('DeviceMotion权限被拒绝');
        }
      } catch (error) {
        console.error('请求DeviceMotion权限失败:', error);
      }
    } else {
      // 非iOS或旧版iOS
      this.bindEvents();
    }
  }

  bindEvents() {
    window.addEventListener('devicemotion', this.handleMotion.bind(this), { passive: true });
  }

  handleMotion(event) {
    const acceleration = event.accelerationIncludingGravity;
    if (!acceleration) return;

    const { x, y, z } = acceleration;

    // 计算加速度变化
    const deltaX = Math.abs(x - this.lastAcceleration.x);
    const deltaY = Math.abs(y - this.lastAcceleration.y);
    const deltaZ = Math.abs(z - this.lastAcceleration.z);

    const totalDelta = deltaX + deltaY + deltaZ;

    // 判断是否超过阈值
    if (totalDelta > this.options.threshold) {
      const now = Date.now();

      // 如果距离上次摇动时间超过timeout
      if (now - this.lastShakeTime > this.options.timeout) {
        this.triggerShake(now);
      }
    }

    this.lastAcceleration = { x, y, z };
  }

  triggerShake(time) {
    this.lastShakeTime = time;
    this.shakeCount++;

    if (!this.isShaking) {
      this.isShaking = true;
      if (this.options.onShakeStart) {
        this.options.onShakeStart(this.shakeCount);
      }
    }

    if (this.options.onShake) {
      this.options.onShake(this.shakeCount);
    }

    // 触发振动反馈
    if (navigator.vibrate) {
      navigator.vibrate(100);
    }

    // 重置状态
    clearTimeout(this.shakeTimeout);
    this.shakeTimeout = setTimeout(() => {
      this.isShaking = false;
      if (this.options.onShakeEnd) {
        this.options.onShakeEnd(this.shakeCount);
      }
    }, this.options.timeout);
  }

  destroy() {
    window.removeEventListener('devicemotion', this.handleMotion);
    clearTimeout(this.shakeTimeout);
  }
}
```

### 10.2 陀螺仪与设备方向

DeviceOrientation API可以获取设备相对于地球坐标系的方向，常用于实现AR效果、赛车游戏等场景。

```javascript
// 陀螺仪管理器
class GyroscopeManager {
  constructor(options = {}) {
    this.options = {
      onOrientation: null,    // 方向变化回调
      onTilt: null,           // 倾斜回调
      enableCompass: false,   // 是否启用电子罗盘
      throttle: 16,           // 节流间隔（毫秒）
      ...options
    };

    this.alpha = 0;  // 绕Z轴旋转（指南针方向）
    this.beta = 0;   // 绕X轴旋转（前后倾斜）
    this.gamma = 0;  // 绕Y轴旋转（左右倾斜）

    this.lastUpdate = 0;
    this.isListening = false;

    this.init();
  }

  init() {
    if (!('DeviceOrientationEvent' in window)) {
      console.warn('DeviceOrientation API不支持');
      return;
    }

    this.requestPermission();
  }

  async requestPermission() {
    // iOS 13+需要用户授权
    if (typeof DeviceOrientationEvent !== 'undefined' &&
        typeof DeviceOrientationEvent.requestPermission === 'function') {
      try {
        const permission = await DeviceOrientationEvent.requestPermission();
        if (permission === 'granted') {
          this.bindEvents();
        }
      } catch (error) {
        console.error('请求陀螺仪权限失败:', error);
      }
    } else {
      this.bindEvents();
    }
  }

  bindEvents() {
    this.isListening = true;
    window.addEventListener('deviceorientation', this.handleOrientation.bind(this), { passive: true });
  }

  handleOrientation(event) {
    const now = Date.now();

    // 节流
    if (now - this.lastUpdate < this.options.throttle) return;
    this.lastUpdate = now;

    const { alpha, beta, gamma } = event;

    // 更新当前方向
    this.alpha = alpha || 0;
    this.beta = beta || 0;
    this.gamma = gamma || 0;

    // 触发方向回调
    if (this.options.onOrientation) {
      this.options.onOrientation({
        alpha: this.alpha,  // 0-360, 绕Z轴
        beta: this.beta,    // -180-180, 绕X轴
        gamma: this.gamma   // -90-90, 绕Y轴
      });
    }

    // 触发倾斜回调
    if (this.options.onTilt) {
      this.options.onTilt(this.calculateTilt());
    }
  }

  // 计算倾斜方向
  calculateTilt() {
    const { beta, gamma } = this;

    let tilt = 'flat';

    if (beta > 45) {
      tilt = 'up';      // 屏幕朝上
    } else if (beta < -45) {
      tilt = 'down';    // 屏幕朝下
    } else if (gamma > 30) {
      tilt = 'right';   // 向右倾斜
    } else if (gamma < -30) {
      tilt = 'left';    // 向左倾斜
    }

    return {
      tilt,
      angle: {
        x: beta,
        y: gamma
      }
    };
  }

  // 获取当前方向（用于初始化）
  getCurrentOrientation() {
    return {
      alpha: this.alpha,
      beta: this.beta,
      gamma: this.gamma
    };
  }

  stop() {
    this.isListening = false;
    window.removeEventListener('deviceorientation', this.handleOrientation);
  }

  resume() {
    if (!this.isListening) {
      this.bindEvents();
    }
  }
}
```

```javascript
// 拍照上传组件
class CameraUpload {
  constructor(options = {}) {
    this.options = {
      accept: 'image/*',           // 接受的文件类型
      capture: 'environment',     // 优先使用后置相机
      maxSize: 5 * 1024 * 1024,   // 最大文件大小（5MB）

      maxHeight: 1920,            // 最大高度
      quality: 0.8,              // 图片质量
      onSelect: null,            // 选择图片回调
      onError: null,             // 错误回调
      ...options
    };

    this.fileInput = null;
    this.previewElement = null;

    this.init();
  }

  init() {
    this.createFileInput();
  }

  createFileInput() {
    // 创建隐藏的文件输入框
    this.fileInput = document.createElement('input');
    this.fileInput.type = 'file';
    this.fileInput.accept = this.options.accept;
    this.fileInput.capture = this.options.capture;
    this.fileInput.style.display = 'none';

    // 监听文件选择
    this.fileInput.addEventListener('change', this.handleFileSelect.bind(this));

    document.body.appendChild(this.fileInput);
  }

  // 触发文件选择
  select() {
    this.fileInput.click();
  }

  // 处理文件选择
  async handleFileSelect(event) {
    const file = event.target.files[0];
    if (!file) return;

    try {
      // 验证文件
      this.validateFile(file);

      // 处理图片
      const processedFile = await this.processImage(file);

      // 生成预览
      const preview = await this.generatePreview(processedFile);

      // 触发回调
      if (this.options.onSelect) {
        this.options.onSelect({
          originalFile: file,
          processedFile,
          preview,
          url: URL.createObjectURL(processedFile)
        });
      }
    } catch (error) {
      if (this.options.onError) {
        this.options.onError(error);
      }
    }

    // 清空input，允许重新选择同一文件
    this.fileInput.value = '';
  }

  // 验证文件
  validateFile(file) {
    // 检查文件类型
    if (!file.type.startsWith('image/')) {
      throw new Error('请选择图片文件');
    }

    // 检查文件大小
    if (file.size > this.options.maxSize) {
      const maxMB = this.options.maxSize / (1024 * 1024);
      throw new Error(`图片大小不能超过${maxMB}MB`);
    }

    return true;
  }

  // 处理图片（压缩、缩放）
  async processImage(file) {
    return new Promise((resolve, reject) => {
      const img = new Image();
      img.onload = () => {
        // 计算新尺寸
        let { width, height } = img;

        if (width > this.options.maxWidth) {
          height = height * (this.options.maxWidth / width);
          width = this.options.maxWidth;
        }

        if (height > this.options.maxHeight) {
          width = width * (this.options.maxHeight / height);
          height = this.options.maxHeight;
        }

        // 创建Canvas进行压缩
        const canvas = document.createElement('canvas');
        canvas.width = width;
        canvas.height = height;

        const ctx = canvas.getContext('2d');
        ctx.drawImage(img, 0, 0, width, height);

        // 转换为Blob
        canvas.toBlob(
          (blob) => {
            if (blob) {
              resolve(blob);
            } else {
              reject(new Error('图片处理失败'));
            }
          },
          'image/jpeg',
          this.options.quality
        );
      };

      img.onerror = () => reject(new Error('图片加载失败'));
      img.src = URL.createObjectURL(file);
    });
  }

  // 生成预览
  async generatePreview(file) {
    return new Promise((resolve, reject) => {
      const reader = new FileReader();
      reader.onload = (e) => resolve(e.target.result);
      reader.onerror = reject;
      reader.readAsDataURL(file);
    });
  }

  // 上传到服务器
  async upload(file, uploadUrl, extraData = {}) {
    const formData = new FormData();
    formData.append('file', file);

    Object.entries(extraData).forEach(([key, value]) => {
      formData.append(key, value);
    });

    const response = await fetch(uploadUrl, {
      method: 'POST',
      body: formData
    });

    if (!response.ok) {
      throw new Error(`上传失败: ${response.status}`);
    }

    return response.json();
  }

  destroy() {
    if (this.fileInput) {
      this.fileInput.remove();
    }
  }
}
```

### 11.2 URL Scheme唤起App

URL Scheme是一种自定义协议，可以在H5页面中通过特定格式的链接唤起App的特定页面。

```javascript
// URL Scheme唤起管理器
class URLSchemeManager {
  constructor(options = {}) {
    this.options = {
      scheme: '',                // 应用的URL Scheme，如 'myapp://'
      universalLink: '',         // iOS通用链接
      appStoreLink: '',          // App Store链接
      androidPackage: '',        // Android包名
      fallbackUrl: '',           // 降级跳转URL
      downloadTip: 'App未安装，是否下载？',
      ...options
    };

    this.iosDevice = /iPad|iPhone|iPod/.test(navigator.userAgent);
    this.androidDevice = /Android/.test(navigator.userAgent);
  }

  // 唤起App
  async openApp() {
    // 优先尝试唤起
    const opened = await this.tryOpen();

    if (!opened) {
      // 唤起失败，判断是否需要下载
      if (this.shouldPromptDownload()) {
        this.promptDownload();
      } else {
        // 跳转到降级页面
        if (this.options.fallbackUrl) {
          window.location.href = this.options.fallbackUrl;
        }
      }
    }
  }

  // 尝试打开App
  async tryOpen() {
    if (this.iosDevice) {
      return this.openIOS();
    } else if (this.androidDevice) {
      return this.openAndroid();
    }

    return false;
  }

  // iOS唤起
  openIOS() {
    // 如果支持Universal Link，优先使用
    if (this.options.universalLink && typeof location !== 'undefined') {
      // iOS 9+ Universal Link
      location.href = this.options.universalLink;

      // 设置超时回退
      setTimeout(() => {
        if (!document.webkitHidden) {
          location.href = this.options.appStoreLink;
        }
      }, 1500);

      return true;
    }

    // URL Scheme方式
    const scheme = this.getIOSScheme();
    location.href = scheme;

    // iOS URL Scheme唤起检测（不太可靠）
    setTimeout(() => {
      if (!document.webkitHidden) {
        location.href = this.options.appStoreLink;
      }
    }, 1500);

    return true;
  }

  // Android唤起
  openAndroid() {
    const scheme = this.getAndroidScheme();

    // Chrome 25+支持直接通过intent协议唤起
    if (this.supportsIntent()) {
      const intentUrl = `intent://${this.options.androidPackage}#Intent;scheme=${this.options.scheme};package=${this.options.androidPackage};end`;
      location.href = intentUrl;
    } else {
      // 直接使用scheme
      const iframe = document.createElement('iframe');
      iframe.style.display = 'none';
      iframe.src = scheme;
      document.body.appendChild(iframe);

      // 清理iframe
      setTimeout(() => {
        iframe.remove();
      }, 1000);
    }

    // 设置超时回退
    setTimeout(() => {
      if (!document.hidden) {
        this.promptDownload();
      }
    }, 1500);

    return true;
  }

  // 获取iOS URL Scheme
  getIOSScheme() {
    return this.options.scheme;
  }

  // 获取Android URL Scheme
  getAndroidScheme() {
    return this.options.scheme;
  }

  // 检测是否支持Intent协议
  supportsIntent() {
    return /Chrome/.test(navigator.userAgent);
  }

  // 判断是否应该提示下载
  shouldPromptDownload() {
    if (this.iosDevice && this.options.appStoreLink) {
      return true;
    }
    if (this.androidDevice && this.options.androidPackage) {
      return true;
    }
    return false;
  }

  // 提示下载
  promptDownload() {
    if (confirm(this.options.downloadTip)) {
      if (this.iosDevice) {
        location.href = this.options.appStoreLink;
      } else if (this.androidDevice) {
        // 尝试通过应用宝微下载
        if (this.options.fallbackUrl) {
          location.href = this.options.fallbackUrl;
        } else {
          location.href = `https://www.example.com/download/${this.options.androidPackage}`;
        }
      }
    }
  }
}
```

### 12.1 上下滑动无缝切换

抖音的核心交互是上下滑动切换视频，这个功能实现起来涉及多个技术点：视频预加载、滑动检测、页面切换动画等。

```javascript
// 抖音式滑动切换组件
class DouyinSwiper {
  constructor(container, options = {}) {
    this.container = container;
    this.options = {
      initialIndex: 0,
      autoplay: false,
      autoplayInterval: 3000,
      onChange: null,
      onVideoPlay: null,
      onVideoPause: null,
      ...options
    };

    // 状态
    this.currentIndex = this.options.initialIndex;
    this.slides = [];
    this.isTransitioning = false;
    this.touchStartY = 0;
    this.touchDeltaY = 0;
    this.currentTranslateY = 0;

    this.init();
  }

  init() {
    // 获取所有幻灯片
    this.slides = Array.from(this.container.querySelectorAll('.video-slide'));

    // 设置容器样式
    this.container.style.cssText = `
      position: relative;
      width: 100%;
      height: 100vh;
      overflow: hidden;
      background: #000;
    `;

    // 初始化幻灯片
    this.slides.forEach((slide, index) => {
      slide.style.cssText = `
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
        transition: transform 0.4s ease-out, opacity 0.4s ease-out;
        transform: translateY(${index === this.currentIndex ? 0 : 100}%);
        opacity: ${index === this.currentIndex ? 1 : 0};
        z-index: ${index === this.currentIndex ? 1 : 0};
      `;
      slide.dataset.index = index;
    });

    // 绑定触摸事件
    this.bindTouchEvents();

    // 预加载视频
    this.preloadVideos();

    // 自动播放第一个视频
    this.playCurrentVideo();
  }

  bindTouchEvents() {
    let startY = 0;
    let startX = 0;
    let isVerticalSwipe = false;

    this.container.addEventListener('touchstart', (e) => {
      if (this.isTransitioning) return;

      startY = e.touches[0].clientY;
      startX = e.touches[0].clientX;
      isVerticalSwipe = false;
    }, { passive: true });

    this.container.addEventListener('touchmove', (e) => {
      if (this.isTransitioning) return;

      const deltaY = e.touches[0].clientY - startY;
      const deltaX = e.touches[0].clientX - startX;

      // 判断滑动方向
      if (!isVerticalSwipe && Math.abs(deltaY) > 10) {
        isVerticalSwipe = Math.abs(deltaY) > Math.abs(deltaX);
      }

      if (!isVerticalSwipe) return;

      // 阻止页面滚动
      e.preventDefault();

      // 实时更新位置
      this.updateSlidePositions(deltaY, false);
    }, { passive: false });

    this.container.addEventListener('touchend', (e) => {
      if (this.isTransitioning) return;

      const deltaY = e.changedTouches[0].clientY - startY;
      const threshold = window.innerHeight * 0.15;

      if (deltaY < -threshold && this.currentIndex < this.slides.length - 1) {
        // 上滑：下一个
        this.goTo(this.currentIndex + 1);
      } else if (deltaY > threshold && this.currentIndex > 0) {
        // 下滑：上一个
        this.goTo(this.currentIndex - 1);
      } else {
        // 回弹
        this.updateSlidePositions(0, true);
      }
    }, { passive: true });
  }

  updateSlidePositions(deltaY, animate) {
    const transition = animate ? 'transform 0.3s ease-out' : 'none';

    this.slides.forEach((slide, index) => {
      let translateY = deltaY;
      let opacity = 1;
      let zIndex = 0;

      if (index === this.currentIndex) {
        // 当前幻灯片
        translateY = deltaY;
        opacity = 1 - Math.abs(deltaY) / window.innerHeight * 0.5;
        zIndex = 1;
      } else if (index === this.currentIndex + 1 && deltaY < 0) {
        // 下一个幻灯片
        translateY = window.innerHeight + deltaY;
        opacity = 0.8;
        zIndex = 0;
      } else if (index === this.currentIndex - 1 && deltaY > 0) {
        // 上一个幻灯片
        translateY = -window.innerHeight + deltaY;
        opacity = 0.8;
        zIndex = 0;
      } else {
        translateY = index < this.currentIndex ? -window.innerHeight : window.innerHeight;
        opacity = 0;
        zIndex = 0;
      }

      slide.style.transition = transition;
      slide.style.transform = `translateY(${translateY}px)`;
      slide.style.opacity = opacity;
      slide.style.zIndex = zIndex;
    });
  }

  goTo(index, animate = true) {
    if (index === this.currentIndex) return;
    if (index < 0 || index >= this.slides.length) return;

    this.isTransitioning = true;

    // 暂停当前视频
    this.pauseCurrentVideo();

    const oldIndex = this.currentIndex;
    this.currentIndex = index;

    // 触发回调
    if (this.options.onChange) {
      this.options.onChange(index, oldIndex);
    }

    // 更新位置
    this.updateSlidePositions(0, animate);

    // 播放新视频
    setTimeout(() => {
      this.playCurrentVideo();
      this.isTransitioning = false;
    }, animate ? 400 : 0);
  }

  next() {
    this.goTo(this.currentIndex + 1);
  }

  prev() {
    this.goTo(this.currentIndex - 1);
  }

  preloadVideos() {
    // 预加载当前视频及前后各一个
    const preloadIndexes = [
      this.currentIndex - 1,
      this.currentIndex,
      this.currentIndex + 1
    ].filter(i => i >= 0 && i < this.slides.length);

    preloadIndexes.forEach(index => {
      const video = this.slides[index].querySelector('video');
      if (video && !video.src) {
        video.preload = 'auto';
      }
    });
  }

  playCurrentVideo() {
    const currentSlide = this.slides[this.currentIndex];
    if (!currentSlide) return;

    const video = currentSlide.querySelector('video');
    if (!video) return;

    video.play().catch(() => {
      // 自动播放被阻止，等待用户交互
      this.waitForPlay(video);
    });

    if (this.options.onVideoPlay) {
      this.options.onVideoPlay(this.currentIndex);
    }
  }

  pauseCurrentVideo() {
    const currentSlide = this.slides[this.currentIndex];
    if (!currentSlide) return;

    const video = currentSlide.querySelector('video');
    if (!video) return;

    video.pause();

    if (this.options.onVideoPause) {
      this.options.onVideoPause(this.currentIndex);
    }
  }

  waitForPlay(video) {
    const handler = () => {
      video.play();
      video.removeEventListener('touchstart', handler);
      video.removeEventListener('click', handler);
    };

    video.addEventListener('touchstart', handler, { once: true });
    video.addEventListener('click', handler, { once: true });
  }
}
```

### 12.2 视频预加载与流畅播放

为了实现流畅的播放体验，视频预加载至关重要。以下是一个完整的视频预加载管理器：

```javascript
// 视频预加载管理器
class VideoPreloader {
  constructor(options = {}) {
    this.options = {
      maxPreload: 3,           // 最大同时预加载数量
      preloadTimeout: 5000,    // 预加载超时时间
      qualityOrder: ['hd', 'sd', 'ld'], // 清晰度优先级
      ...options
    };

    this.preloadQueue = [];
    this.preloadingVideos = new Map();
    this.loadedVideos = new Map();
  }

  // 添加视频到预加载队列
  add(videoInfo) {
    const { url, id, quality = 'hd' } = videoInfo;

    // 检查是否已加载或正在加载
    if (this.loadedVideos.has(url) || this.preloadingVideos.has(url)) {
      return;
    }

    this.preloadQueue.push({ url, id, quality });
    this.processQueue();
  }

  // 处理预加载队列
  processQueue() {
    // 如果正在预加载的数量已达到上限，等待
    if (this.preloadingVideos.size >= this.options.maxPreload) {
      return;
    }

    // 取出队首的待预加载项
    const item = this.preloadQueue.shift();
    if (!item) return;

    this.preloadVideo(item);
  }

  // 预加载视频
  preloadVideo({ url, id, quality }) {
    this.preloadingVideos.set(url, { id, quality, status: 'loading' });

    const video = document.createElement('video');
    video.preload = 'auto';
    video.muted = true;
    video.playsInline = true;
    video.crossOrigin = 'anonymous';

    // 超时处理
    const timeout = setTimeout(() => {
      if (this.preloadingVideos.has(url)) {
        console.warn(`视频预加载超时: ${url}`);
        this.preloadingVideos.delete(url);
        this.processQueue();
      }
    }, this.options.preloadTimeout);

    video.oncanplaythrough = () => {
      clearTimeout(timeout);
      this.preloadingVideos.delete(url);
      this.loadedVideos.set(url, video);
      console.log(`视频预加载完成: ${url}`);
      this.processQueue();
    };

    video.onerror = () => {
      clearTimeout(timeout);
      console.error(`视频预加载失败: ${url}`);
      this.preloadingVideos.delete(url);
      this.processQueue();
    };

    video.src = url;
  }

  // 获取预加载的视频
  getVideo(url) {
    return this.loadedVideos.get(url);
  }

  // 检查视频是否已加载
  isLoaded(url) {
    return this.loadedVideos.has(url);
  }

  // 检查视频是否正在加载
  isLoading(url) {
    return this.preloadingVideos.has(url);
  }

  // 清理已加载的视频（释放内存）
  clear() {
    this.loadedVideos.forEach(video => {
      video.src = '';
      video.load();
    });
    this.loadedVideos.clear();
  }
}
```

### 12.3 手势操作：双击点赞

双击点赞是抖音的经典交互，以下是实现代码：

```javascript
// 双击点赞检测器
class DoubleTapLike {
  constructor(element, options = {}) {
    this.element = element;
    this.options = {
      threshold: 300,         // 两次点击的最大间隔（毫秒）
      radius: 100,            // 爱心显示的半径
      duration: 1000,         // 动画持续时间
      maxLikes: 3,            // 最多同时显示的爱心数量
      onDoubleTap: null,      // 双击回调
      ...options
    };

    this.lastTapTime = 0;
    this.lastTapX = 0;
    this.lastTapY = 0;
    this.likeCount = 0;
    this.likes = [];

    this.bindEvents();
  }

  bindEvents() {
    this.element.addEventListener('touchend', this.handleTap.bind(this), { passive: false });
  }

  handleTap(e) {
    const now = Date.now();
    const touch = e.changedTouches[0];

    // 计算与上次点击的距离
    const deltaX = touch.clientX - this.lastTapX;
    const deltaY = touch.clientY - this.lastTapY;
    const distance = Math.sqrt(deltaX * deltaX + deltaY * deltaY);

    // 判断是否为双击（时间间隔短且距离不太远）
    if (now - this.lastTapTime < this.options.threshold && distance < 50) {
      this.triggerDoubleTap(touch.clientX, touch.clientY);
      this.lastTapTime = 0; // 重置，防止三次点击被误判
    } else {
      this.lastTapTime = now;
      this.lastTapX = touch.clientX;
      this.lastTapY = touch.clientY;
    }
  }
```

### 12.4 性能优化：首屏秒开

首屏秒开是短视频应用的关键指标，以下是实现方案：

```javascript
// 首屏秒开优化管理器
class FirstScreenOptimizer {
  constructor() {
    this.metrics = {
      domContentLoaded: 0,
      firstPaint: 0,
      firstContentfulPaint: 0,
      firstMeaningfulPaint: 0,
      domComplete: 0
    };

    this.init();
  }

  init() {
    // 监听Performance API
    if (PerformanceObserver) {
      this.observePaint();
    }

    // 记录关键时间点
    if (document.readyState === 'complete') {
      this.recordMetrics();
    } else {
      window.addEventListener('load', () => this.recordMetrics());
    }
  }

  observePaint() {
    // 观察首次内容绘制
    const paintObserver = new PerformanceObserver((list) => {
      list.getEntries().forEach(entry => {
        if (entry.name === 'first-contentful-paint') {
          this.metrics.firstContentfulPaint = entry.startTime;
        }
        if (entry.name === 'first-paint') {
          this.metrics.firstPaint = entry.startTime;
        }
      });
    });

    paintObserver.observe({ entryTypes: ['paint'] });

    // 观察长任务
    const longTaskObserver = new PerformanceObserver((list) => {
      list.getEntries().forEach(entry => {
        console.warn('长任务检测:', entry.duration, 'ms');
      });
    });

    longTaskObserver.observe({ entryTypes: ['longtask'] });
  }

  recordMetrics() {
    const timing = performance.timing;

    this.metrics.domContentLoaded = timing.domContentLoadedEventEnd - timing.navigationStart;
    this.metrics.domComplete = timing.domComplete - timing.navigationStart;

    // 输出性能报告
    console.log('性能指标:', this.metrics);

    // 发送性能数据到分析服务
    this.reportMetrics();
  }

  reportMetrics() {
    // 实际项目中发送到分析服务
    if (navigator.sendBeacon) {
      const data = JSON.stringify({
        ...this.metrics,
        url: location.href,
        timestamp: Date.now()
      });

      navigator.sendBeacon('/api/analytics/performance', data);
    }
  }

  // 获取性能建议
  getSuggestions() {
    const suggestions = [];

    if (this.metrics.firstContentfulPaint > 2000) {
      suggestions.push({
        type: 'warning',
        message: '首次内容绘制时间过长，建议优化关键渲染路径'
      });
    }

    if (this.metrics.domComplete > 3000) {
      suggestions.push({
        type: 'warning',
        message: 'DOM完成时间过长，建议减少资源体积或延迟加载'
      });
    }

    return suggestions;
  }
}

// 首屏优化技术清单
const FirstScreenOptimization = {
  // 1. DNS预解析
  dnsPrefetch() {
    const domains = [
      'https://your-cdn.com',
      'https://your-api.com'
    ];

    domains.forEach(domain => {
      const link = document.createElement('link');
      link.rel = 'dns-prefetch';
      link.href = domain;
      document.head.appendChild(link);
    });
  },

  // 2. 预连接
  preconnect() {
    const domains = [
      'https://your-cdn.com',
      'https://your-api.com'
    ];

    domains.forEach(domain => {
      const link = document.createElement('link');
      link.rel = 'preconnect';
      link.href = domain;
      link.crossOrigin = 'anonymous';
      document.head.appendChild(link);
    });
  },

  // 3. 预加载关键资源
  preload(key, url) {
    const link = document.createElement('link');
    link.rel = 'preload';
    link.as = key;
    link.href = url;
    document.head.appendChild(link);
  },

  // 4. 异步加载JavaScript
  asyncScript(src) {
    const script = document.createElement('script');
    script.src = src;
    script.async = true;
    document.head.appendChild(script);
  },

  // 5. 延迟执行非关键代码
  deferExec(fn, delay = 100) {
    if ('requestIdleCallback' in window) {
      requestIdleCallback(() => fn(), { timeout: 3000 });
    } else {
      setTimeout(fn, delay);
    }
  }
};
```
