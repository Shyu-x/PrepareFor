## 一、电商应用概述

### 1.1 电商的核心本质

电商系统，本质上是一个**连接买家和卖家的交易平台**。无论淘宝、京东、拼多多还是各类垂直电商，其核心业务都可以归纳为四个字：**"人、货、场、钱"**。

- **人**：用户系统、会员体系、用户画像
- **货**：商品系统、库存管理、供应链
- **场**：交易场景、营销活动、内容运营
- **钱**：支付系统、结算系统、对账系统

### 1.2 核心功能模块

### 1.3 技术挑战

电商系统面临的技术挑战，是所有互联网业务中最为复杂的之一：

| 挑战类型 | 具体问题 | 解决方案 |
|---------|---------|---------|
| **高并发** | 双十一瞬时流量洪峰 | 分层限流、缓存、异步化 |
| **一致性** | 分布式事务问题 | TCC、Saga、本地消息表 |
| **超卖** | 库存扣减不一致 | 乐观锁、Redis原子操作 |
| **防刷** | 恶意下单、薅羊毛 | 验证码、风控系统、限流 |
| **性能** | 页面加载缓慢 | CDN、SSR、静态化 |

### 1.4 我的思考：电商系统的复杂度

很多人认为电商就是"CRUD"，实际上电商系统的复杂度远超想象：

1. **业务逻辑复杂度**：促销叠加规则、优惠券互斥、会员折扣、满减活动等，涉及大量的条件判断和计算。

2. **数据一致性要求高**：下单、支付、库存三个操作必须强一致，任何一个环节出问题都会导致资损。

3. **性能要求极端**：既要支撑双十一这样的流量洪峰，又要保证每次请求的响应时间在毫秒级。

4. **容灾要求高**：电商系统停机一分钟，损失可能就是天文数字。

接下来，我们将逐个模块进行深入讲解。

## 二、商品系统

### 2.1 SKU与SPU概念

在电商系统中，**SPU（Standard Product Unit，标准化产品单元）** 和 **SKU（Stock Keeping Unit，库存量单位）** 是两个核心概念。

- **SPU**：商品聚合的最小单位，是商品信息聚合的载体。例如："iPhone 15 Pro 256GB"是一个SPU。

- **SKU**：库存管理的最小单位，是具体可售商品的唯一标识。例如："iPhone 15 Pro 256GB 深空灰色"是一个SKU。

```
SPU: iPhone 15 Pro 256GB
├── SKU: iPhone 15 Pro 256GB 钛金色
├── SKU: iPhone 15 Pro 256GB 深空灰色
├── SKU: iPhone 15 Pro 256GB 白色钛金属
└── SKU: iPhone 15 Pro 256GB 原色钛金属
```

### 2.3 商品搜索实现

商品搜索是电商系统的核心功能之一，通常使用Elasticsearch来实现：

```typescript
// 商品搜索服务
import { Client } from '@elastic/elasticsearch';

class ProductSearchService {
  private client: Client;
  private indexName = 'products';

  constructor() {
    this.client = new Client({
      node: process.env.ELASTICSEARCH_URL,
    });
  }

  /**
   * 构建商品搜索查询
   * 支持关键词搜索、分类筛选、品牌筛选、价格区间、排序等功能
   */
  async search(params: SearchParams): Promise<SearchResult> {
    const {
      keyword,           // 搜索关键词
      categoryId,        // 分类ID
      brandIds,           // 品牌ID列表
      minPrice,          // 最低价格
      maxPrice,          // 最高价格
      sortField = 'salesCount', // 排序字段
      sortOrder = 'desc',       // 排序方向
      page = 1,            // 页码
      pageSize = 20,        // 每页数量
    } = params;

    // 构建布尔查询
    const must: any[] = [];
    const filter: any[] = [];

    // 关键词搜索 - 使用multi_match查询多个字段
    if (keyword) {
      must.push({
        multi_match: {
          query: keyword,
          fields: ['name^3', 'subtitle^2', 'description', 'brandName'],
          type: 'best_match',
          fuzziness: 'AUTO',  // 自动模糊匹配，处理拼写错误
        },
      });
    }

    // 分类筛选 - 支持多级分类
    if (categoryId) {
      filter.push({
        term: { categoryId: categoryId },
      });
    }

    // 品牌筛选
    if (brandIds && brandIds.length > 0) {
      filter.push({
        terms: { brandId: brandIds },
      });
    }

    // 价格区间筛选
    if (minPrice !== undefined || maxPrice !== undefined) {
      const range: any = {};
      if (minPrice !== undefined) range.gte = minPrice;
      if (maxPrice !== undefined) range.lte = maxPrice;
      filter.push({ range: { price: range } });
    }

    // 构建完整的搜索请求
    const searchQuery = {
      index: this.indexName,
      body: {
        query: {
          bool: {
            must: must.length > 0 ? must : [{ match_all: {} }],
            filter: filter,
          },
        },
        // 聚合查询，用于获取筛选面板的统计数据
        aggs: {
          // 品牌聚合
          brands: {
            terms: { field: 'brandId', size: 50 },
            aggs: {
              brandName: { terms: { field: 'brandName.keyword', size: 1 } },
            },
          },
          // 价格区间聚合
          priceRanges: {
            range: {
              field: 'price',
              ranges: [
                { key: '0-100', to: 100 },
                { key: '100-500', from: 100, to: 500 },
                { key: '500-1000', from: 500, to: 1000 },
                { key: '1000-3000', from: 1000, to: 3000 },
                { key: '3000-5000', from: 3000, to: 5000 },
                { key: '5000+', from: 5000 },
              ],
            },
          },
          // 分类聚合
          categories: {
            terms: { field: 'categoryId', size: 100 },
          },
        },
        // 分页
        from: (page - 1) * pageSize,
        size: pageSize,
        // 排序
        sort: [
          { [sortField]: { order: sortOrder } },
          { _score: { order: 'desc' } },  // 相关性得分
        ],
        // 高亮显示
        highlight: {
          fields: {
            name: {},
            subtitle: {},
          },
          pre_tags: ['<em>'],
          post_tags: ['</em>'],
        },
      },
    };

    const result = await this.client.search(searchQuery);

    // 处理搜索结果
    const hits = result.hits.hits.map((hit: any) => ({
      id: hit._source.id,
      name: hit._source.name,
      subtitle: hit._source.subtitle,
      price: hit._source.price,
      maxPrice: hit._source.maxPrice,
      image: hit._source.images?.[0],
      salesCount: hit._source.salesCount,
      score: hit._source.score,
      // 高亮处理
      highlights: hit.highlight || {},
    }));

    // 提取聚合结果
    const aggregations = {
      brands: result.aggregations?.brands?.buckets?.map((b: any) => ({
        id: b.key,
        name: b.brandName?.buckets?.[0]?.key,
        count: b.doc_count,
      })) || [],
      priceRanges: result.aggregations?.priceRanges?.buckets || [],
      categories: result.aggregations?.categories?.buckets?.map((c: any) => ({
        id: c.key,
        count: c.doc_count,
      })) || [],
    };

    return {
      list: hits,
      total: result.hits.total.value,
      page,
      pageSize,
      totalPages: Math.ceil(result.hits.total.value / pageSize),
      aggregations,
    };
  }

  /**
   * 搜索建议（自动补全）
   * 用于搜索框的实时联想功能
   */
  async suggest(keyword: string, limit = 10): Promise<SuggestResult[]> {
    const result = await this.client.search({
      index: this.indexName,
      body: {
        query: {
          bool: {
            should: [
              // 前缀匹配 - 优先级最高
              {
                prefix: {
                  'name.keyword': {
                    value: keyword,
                    boost: 2,  // 加权
                  },
                },
              },
              // 包含匹配
              {
                match: {
                  name: {
                    query: keyword,
                    fuzziness: 'AUTO',
                  },
                },
              },
            ],
          },
        },
        _source: ['id', 'name', 'categoryId', 'price', 'image'],
        size: limit,
      },
    });

    return result.hits.hits.map((hit: any) => ({
      id: hit._source.id,
      name: hit._source.name,
      categoryId: hit._source.categoryId,
      price: hit._source.price,
      image: hit._source.image,
    }));
  }
}
```

### 2.4 商品详情页静态化

商品详情页是电商流量的入口，通过静态化可以大幅提升访问速度：

```typescript
// 商品详情页服务
class ProductDetailService {
  private cache: Map<number, ProductDetail>;  // 本地缓存
  private redis: Redis;

  constructor() {
    this.cache = new Map();
  }

  /**
   * 获取商品详情
   * 采用多级缓存策略：本地缓存 -> Redis -> 数据库
   */
  async getProductDetail(skuId: number): Promise<ProductDetail | null> {
    // 1. 先检查本地缓存（热点数据）
    const localCache = this.cache.get(skuId);
    if (localCache) {
      return localCache;
    }

    // 2. 查询Redis缓存
    const cacheKey = `product:detail:${skuId}`;
    const cached = await this.redis.get(cacheKey);
    if (cached) {
      const detail = JSON.parse(cached);
      // 写入本地缓存
      this.cache.set(skuId, detail);
      return detail;
    }

    // 3. 查询数据库
    const detail = await this.loadFromDatabase(skuId);
    if (!detail) {
      return null;
    }

    // 4. 写入Redis缓存（设置过期时间）
    await this.redis.setex(cacheKey, 3600, JSON.stringify(detail)); // 1小时过期

    // 5. 写入本地缓存
    this.cache.set(skuId, detail);

    return detail;
  }

  /**
   * 从数据库加载商品详情
   */
  private async loadFromDatabase(skuId: number): Promise<ProductDetail | null> {
    // 查询SKU信息
    const sku = await db.query(
      'SELECT * FROM sku WHERE id = ? AND status = 1',
      [skuId]
    );
    if (!sku || sku.length === 0) {
      return null;
    }

    // 查询SPU信息
    const spu = await db.query(
      'SELECT * FROM spu WHERE id = ?',
      [sku[0].spuId]
    );

    // 查询SKU列表（同SPU下的所有规格）
    const skuList = await db.query(
      'SELECT id, specValues, price, stock, skuCode FROM sku WHERE spuId = ? AND status = 1',
      [sku[0].spuId]
    );

    // 查询商品详情（富文本描述）
    const description = await db.query(
      'SELECT content FROM product_description WHERE spuId = ?',
      [sku[0].spuId]
    );

    // 查询商品参数
    const attrs = await db.query(
      'SELECT attributeName, attributeValue FROM product_attribute WHERE spuId = ?',
      [sku[0].spuId]
    );

    // 查询商品评价统计
    const commentStats = await db.query(
      `SELECT
        COUNT(*) as total,
        AVG(score) as avgScore,
        SUM(CASE WHEN score >= 4 THEN 1 ELSE 0 END) as goodRate
       FROM product_comment WHERE skuId = ?`,
      [skuId]
    );

    return {
      sku: sku[0],
      spu: spu[0],
      skuList,
      description: description[0]?.content || '',
      attrs: this.formatAttrs(attrs),
      comments: {
        total: commentStats[0].total,
        avgScore: commentStats[0].avgScore || 0,
        goodRate: commentStats[0].total > 0
          ? (commentStats[0].goodRate / commentStats[0].total * 100).toFixed(1)
          : 0,
      },
    };
  }

  /**
   * 格式化属性列表为Map
   */
  private formatAttrs(attrs: any[]): Record<string, string> {
    const result: Record<string, string> = {};
    for (const attr of attrs) {
      result[attr.attributeName] = attr.attributeValue;
    }
    return result;
  }
}
```

### 2.5 价格系统设计

电商系统的价格计算是一个复杂的过程，涉及原价、活动价、会员价、促销价等多种价格：

```typescript
/**
 * 价格计算服务
 * 负责计算商品的最终售价，考虑多种价格影响因素
 */
class PriceCalculator {
  /**
   * 计算商品最终价格
   * @param skuId SKU ID
   * @param userId 用户ID（用于计算会员价）
   * @param quantity 购买数量
   * @param channel 渠道（如APP、小程序、PC）
   */
  async calculatePrice(
    skuId: number,
    userId: number,
    quantity: number,
    channel: string
  ): Promise<PriceDetail> {
    // 1. 获取SKU原价
    const sku = await this.getSkuPrice(skuId);
    const originPrice = sku.price;

    // 2. 获取用户会员等级
    const memberLevel = await this.getMemberLevel(userId);

    // 3. 计算会员折扣价
    const memberDiscount = this.getMemberDiscount(memberLevel);
    const memberPrice = this.applyDiscount(originPrice, memberDiscount);

    // 4. 查询可用活动（促销、优惠券等）
    const activities = await this.getAvailableActivities(
      skuId,
      userId,
      channel,
      quantity
    );

    // 5. 计算活动优惠价
    let activityPrice = memberPrice;
    let appliedPromotions: AppliedPromotion[] = [];

    for (const activity of activities) {
      const result = this.calculatePromotion(
        activity,
        activityPrice,
        quantity
      );
      if (result.savings > 0) {
        activityPrice = result.finalPrice;
        appliedPromotions.push({
          activityId: activity.id,
          activityName: activity.name,
          discountAmount: result.savings,
        });
      }
    }

    // 6. 计算优惠券优惠
    const couponDiscount = await this.calculateCouponDiscount(
      userId,
      skuId,
      activityPrice * quantity
    );
    const finalPrice = activityPrice - couponDiscount;

    // 7. 返回完整价格详情
    return {
      skuId,
      originPrice,           // 原价
      memberPrice,           // 会员价
      memberDiscount,         // 会员折扣率
      activityPrice,         // 活动价
      appliedPromotions,     // 已应用的活动
      couponDiscount,        // 优惠券抵扣
      finalPrice: Math.max(0, finalPrice), // 最终价（不能为负）
      totalAmount: Math.max(0, finalPrice) * quantity, // 总价
      savings: originPrice * quantity - finalPrice * quantity, // 节省金额
    };
  }

  /**
   * 获取SKU价格信息
   */
  private async getSkuPrice(skuId: number): Promise<{ price: number; costPrice: number }> {
    const result = await db.query(
      'SELECT price, costPrice FROM sku WHERE id = ?',
      [skuId]
    );
    return result[0];
  }

  /**
   * 获取用户会员等级
   */
  private async getMemberLevel(userId: number): Promise<number> {
    const result = await db.query(
      'SELECT level FROM user WHERE id = ?',
      [userId]
    );
    return result[0]?.level || 0;
  }

  /**
   * 获取会员折扣率
   * 不同会员等级享受不同折扣
   */
  private getMemberDiscount(level: number): number {
    const discounts: Record<number, number> = {
      0: 1.0,     // 普通会员：不打折
      1: 0.98,    // 银牌会员：98折
      2: 0.95,    // 金牌会员：95折
      3: 0.92,    // 钻石会员：92折
      4: 0.90,    // 黑金会员：9折
    };
    return discounts[level] || 1.0;
  }

  /**
   * 应用折扣计算新价格
   */
  private applyDiscount(price: number, discount: number): number {
    // 价格保留2位小数
    return Math.round(price * discount) / 100 * 100;
  }

  /**
   * 获取可用活动列表
   */
  private async getAvailableActivities(
    skuId: number,
    userId: number,
    channel: string,
    quantity: number
  ): Promise<Activity[]> {
    const now = Date.now();
    return await db.query(`
      SELECT a.* FROM activity a
      INNER JOIN activity_sku asa ON a.id = asa.activityId
      WHERE asa.skuId = ?
        AND a.startTime <= ?
        AND a.endTime >= ?
        AND a.status = 1
        AND (a.channels = '' OR FIND_IN_SET(?, a.channels))
        AND a.minQuantity <= ?
      ORDER BY a.priority DESC, a.discount DESC
    `, [skuId, now, now, channel, quantity]);
  }

  /**
   * 计算活动优惠
   */
  private calculatePromotion(
    activity: Activity,
    currentPrice: number,
    quantity: number
  ): { finalPrice: number; savings: number } {
    switch (activity.type) {
      case 'discount': // 折扣活动
        const discount = activity.discount / 100; // 如：85折 = 85
        const finalPrice = this.applyDiscount(currentPrice, discount);
        const savings = (currentPrice - finalPrice) * quantity;
        return { finalPrice, savings };

      case 'special': // 特价活动（如第二件半价）
        if (quantity >= 2) {
          const discountItems = Math.floor(quantity / 2);
          const halfPriceItems = quantity % 2;
          const total = discountItems * currentPrice * 0.5 + halfPriceItems * currentPrice;
          return {
            finalPrice: total / quantity,
            savings: (currentPrice - total / quantity) * quantity,
          };
        }
        return { finalPrice: currentPrice, savings: 0 };

      case 'reduction': // 满减活动
        const totalAmount = currentPrice * quantity;
        if (totalAmount >= activity.minAmount) {
          const reduction = activity.reductionAmount;
          return {
            finalPrice: currentPrice,
            savings: reduction,
          };
        }
        return { finalPrice: currentPrice, savings: 0 };

      default:
        return { finalPrice: currentPrice, savings: 0 };
    }
  }

  /**
   * 计算优惠券抵扣
   */
  private async calculateCouponDiscount(
    userId: number,
    skuId: number,
    orderAmount: number
  ): Promise<number> {
    // 获取用户最优优惠券
    const coupon = await this.getBestCoupon(userId, skuId, orderAmount);
    if (!coupon) return 0;

    // 计算优惠金额
    if (coupon.type === 'amount') {
      // 满减券：直接抵扣
      return Math.min(coupon.discountAmount, orderAmount);
    } else if (coupon.type === 'percent') {
      // 折扣券：按比例抵扣
      return Math.floor(orderAmount * coupon.discountPercent) / 100;
    }
    return 0;
  }
}
```

```
"iPhone" -> [文档1, 文档3, 文档5, ...]
"手机" -> [文档1, 文档2, 文档3, ...]
"5G" -> [文档1, 文档4, ...]
```

### 3.3 搜索建议服务

搜索建议是提升用户体验的重要功能：

```typescript
/**
 * 搜索建议服务
 * 实现前缀匹配、拼音搜索、热词推荐等功能
 */
class SearchSuggestionService {
  private redis: Redis;
  private esClient: Client;

  /**
   * 获取搜索建议列表
   * @param keyword 用户输入的关键词
   * @param limit 返回数量限制
   */
  async getSuggestions(keyword: string, limit = 10): Promise<Suggestion[]> {
    const suggestions: Suggestion[] = [];

    // 1. 查询热搜词（从Redis Sorted Set获取）
    const hotSearches = await this.getHotSearches(keyword, limit);
    suggestions.push(...hotSearches);

    // 2. 查询历史搜索（用户个性化）
    const historySearches = await this.getHistorySearches(keyword, limit);
    suggestions.push(...historySearches);

    // 3. 查询商品联想
    const productSuggestions = await this.getProductSuggestions(keyword, limit);
    suggestions.push(...productSuggestions);

    // 4. 查询品牌联想
    const brandSuggestions = await this.getBrandSuggestions(keyword, limit);
    suggestions.push(...brandSuggestions);

    // 5. 去重并返回
    return this.deduplicateAndSort(suggestions, limit);
  }

  /**
   * 从热搜榜获取建议
   */
  private async getHotSearches(keyword: string, limit: number): Promise<Suggestion[]> {
    // 使用Redis ZSet存储热搜词，按搜索次数排序
    const hotWords = await this.redis.zrevrange('search:hot', 0, limit - 1);

    // 过滤匹配前缀的热搜词
    return hotWords
      .filter(word => word.startsWith(keyword))
      .slice(0, limit)
      .map(word => ({
        type: 'hot',
        text: word,
        score: 1, // 热搜词权重较高
      }));
  }

  /**
   * 获取用户历史搜索
   */
  private async getHistorySearches(
    keyword: string,
    userId: string,
    limit: number
  ): Promise<Suggestion[]> {
    // 从Redis List获取用户搜索历史
    const history = await this.redis.lrange(`search:history:${userId}`, 0, 99);

    // 过滤匹配关键词的记录
    return history
      .filter(word => word.includes(keyword))
      .slice(0, limit)
      .map(word => ({
        type: 'history',
        text: word,
        score: 0.9,
      }));
  }

  /**
   * 获取商品联想
   */
  private async getProductSuggestions(
    keyword: string,
    limit: number
  ): Promise<Suggestion[]> {
    const result = await this.esClient.search({
      index: 'products',
      body: {
        query: {
          bool: {
            should: [
              // 精确前缀匹配（最高优先级）
              {
                prefix: {
                  'name.keyword': {
                    value: keyword,
                    boost: 3,
                  },
                },
              },
              // IK分词匹配
              {
                match: {
                  name: {
                    query: keyword,
                    minimum_should_match: '50%',
                    boost: 2,
                  },
                },
              },
              // 拼音首字母匹配
              {
                match: {
                  pinyinInitials: {
                    query: keyword,
                    boost: 1.5,
                  },
                },
              },
            ],
          },
        },
        _source: ['id', 'name', 'price', 'image'],
        size: limit,
      },
    });

    return result.hits.hits.map((hit: any) => ({
      type: 'product',
      text: hit._source.name,
      productId: hit._source.id,
      price: hit._source.price,
      image: hit._source.image,
      score: hit._score,
    }));
  }

  /**
   * 获取品牌联想
   */
  private async getBrandSuggestions(
    keyword: string,
    limit: number
  ): Promise<Suggestion[]> {
    const result = await this.esClient.search({
      index: 'brands',
      body: {
        query: {
          bool: {
            should: [
              { prefix: { 'name.keyword': keyword } },
              { match: { name: keyword } },
              { match: { pinyinInitials: keyword } },
            ],
          },
        },
        _source: ['id', 'name', 'logo'],
        size: limit,
      },
    });

    return result.hits.hits.map((hit: any) => ({
      type: 'brand',
      text: hit._source.name,
      brandId: hit._source.id,
      logo: hit._source.logo,
      score: hit._score,
    }));
  }

  /**
   * 记录搜索热词
   */
  async recordSearchKeyword(keyword: string): Promise<void> {
    if (!keyword || keyword.length < 2) return;

    // 增加热搜词计数
    await this.redis.zincrby('search:hot', 1, keyword);

    // 设置热搜词过期时间（7天）
    await this.redis.expire('search:hot', 7 * 24 * 3600);
  }

  /**
   * 记录用户搜索历史
   */
  async recordSearchHistory(userId: string, keyword: string): Promise<void> {
    if (!keyword || keyword.length < 2) return;

    const key = `search:history:${userId}`;

    // 先移除已存在的相同关键词（避免重复）
    await this.redis.lrem(key, 0, keyword);

    // 添加到历史列表开头
    await this.redis.lpush(key, keyword);

    // 只保留最近20条
    await this.redis.ltrim(key, 0, 19);

    // 设置过期时间（30天）
    await this.redis.expire(key, 30 * 24 * 3600);
  }
}
```

### 3.4 推荐系统设计

推荐系统是电商提升GMV的核心武器，主要有两类推荐算法：

```typescript
/**
 * 推荐系统服务
 * 结合协同过滤和内容推荐，为用户提供个性化商品推荐
 */
class RecommendationService {
  private redis: Redis;
  private userBehaviorRepo: UserBehaviorRepository;

  /**
   * 获取首页推荐列表
   * 结合多种推荐策略
   */
  async getHomeRecommendations(
    userId: string,
    limit = 20
  ): Promise<RecommendedProduct[]> {
    // 1. 猜你喜欢（协同过滤）
    const collaborativeRecs = await this.getCollaborativeRecommendations(
      userId,
      Math.floor(limit * 0.4)
    );

    // 2. 相似商品（内容推荐）
    const contentRecs = await this.getContentBasedRecommendations(
      userId,
      Math.floor(limit * 0.3)
    );

    // 3. 热销商品（补充推荐）
    const hotRecs = await this.getHotProducts(limit - collaborativeRecs.length - contentRecs.length);

    // 4. 实时加权混合
    return this.hybridSort([
      ...collaborativeRecs.map(r => ({ ...r, weight: 0.4 })),
      ...contentRecs.map(r => ({ ...r, weight: 0.3 })),
      ...hotRecs.map(r => ({ ...r, weight: 0.3 })),
    ], limit);
  }

  /**
   * 协同过滤推荐
   * 基于用户行为相似度，推荐相似用户喜欢的商品
   */
  private async getCollaborativeRecommendations(
    userId: string,
    limit: number
  ): Promise<RecommendedProduct[]> {
    // 1. 找到与当前用户行为相似的K个用户
    const similarUsers = await this.findSimilarUsers(userId, 50);

    // 2. 获取相似用户喜欢的商品（但当前用户未购买的）
    const userBuyHistory = await this.getUserBuyHistory(userId);
    const excludeProductIds = new Set(userBuyHistory.map(p => p.productId));

    // 3. 聚合相似用户的商品喜好
    const productScores = new Map<string, number>();

    for (const similarUser of similarUsers) {
      const products = await this.getUserBuyHistory(similarUser.userId);

      for (const product of products) {
        if (excludeProductIds.has(product.productId)) continue;

        // 加权计分：相似度 * 行为权重
        const score = similarUser.similarity * this.getBehaviorWeight(product.behavior);
        const existing = productScores.get(product.productId) || 0;
        productScores.set(product.productId, existing + score);
      }
    }

    // 4. 排序并返回Top N
    const topProducts = Array.from(productScores.entries())
      .sort((a, b) => b[1] - a[1])
      .slice(0, limit);

    return this.getProductDetails(topProducts.map(([id]) => id));
  }

  /**
   * 查找相似用户
   * 使用余弦相似度计算用户行为相似度
   */
  private async findSimilarUsers(
    userId: string,
    limit: number
  ): Promise<SimilarUser[]> {
    // 获取当前用户的行为向量
    const userVector = await this.buildUserBehaviorVector(userId);

    // 从Redis获取所有用户的向量（实际生产中会用向量数据库）
    const allUserVectors = await this.redis.hgetall('user:behavior:vectors');

    const similarities: SimilarUser[] = [];

    for (const [otherId, vectorStr] of Object.entries(allUserVectors)) {
      if (otherId === userId) continue;

      const otherVector = JSON.parse(vectorStr);
      const similarity = this.cosineSimilarity(userVector, otherVector);

      similarities.push({
        userId: otherId,
        similarity,
      });
    }

    // 返回最相似的N个用户
    return similarities
      .sort((a, b) => b.similarity - a.similarity)
      .slice(0, limit);
  }

  /**
   * 构建用户行为向量
   * 将用户的行为（浏览、收藏、购买）转换为向量
   */
  private async buildUserBehaviorVector(
    userId: string
  ): Promise<Record<string, number>> {
    const behaviors = await this.userBehaviorRepo.getUserBehaviors(userId);

    const vector: Record<string, number> = {};

    for (const behavior of behaviors) {
      const key = `category:${behavior.categoryId}`;
      const weight = this.getBehaviorWeight(behavior.type);
      vector[key] = (vector[key] || 0) + weight;
    }

    return vector;
  }

  /**
   * 计算余弦相似度
   */
  private cosineSimilarity(
    vecA: Record<string, number>,
    vecB: Record<string, number>
  ): number {
    // 计算点积
    let dotProduct = 0;
    for (const key of Object.keys(vecA)) {
      if (vecB[key] !== undefined) {
        dotProduct += vecA[key] * vecB[key];
      }
    }

    // 计算模长
    const normA = Math.sqrt(
      Object.values(vecA).reduce((sum, val) => sum + val * val, 0)
    );
    const normB = Math.sqrt(
      Object.values(vecB).reduce((sum, val) => sum + val * val, 0)
    );

    if (normA === 0 || normB === 0) return 0;

    return dotProduct / (normA * normB);
  }

  /**
   * 获取行为权重
   */
  private getBehaviorWeight(behavior: string): number {
    const weights: Record<string, number> = {
      view: 1,      // 浏览
      favorite: 3,  // 收藏
      cart: 5,      // 加购
      buy: 10,      // 购买
    };
    return weights[behavior] || 1;
  }

  /**
   * 内容推荐
   * 基于用户历史购买的商品特征，推荐相似商品
   */
  private async getContentBasedRecommendations(
    userId: string,
    limit: number
  ): Promise<RecommendedProduct[]> {
    // 获取用户购买历史中的商品特征
    const buyHistory = await this.getUserBuyHistory(userId);
    if (buyHistory.length === 0) return [];

    // 提取特征：品牌、分类、价格区间
    const features = this.extractProductFeatures(buyHistory);

    // 从ES查询相似特征的商品
    const similarProducts = await this.esClient.search({
      index: 'products',
      body: {
        query: {
          bool: {
            should: [
              // 同品牌
              { terms: { brandId: features.brandIds } },
              // 同分类
              { terms: { categoryId: features.categoryIds } },
              // 价格区间相近
              {
                range: {
                  price: {
                    gte: features.avgPrice * 0.8,
                    lte: features.avgPrice * 1.2,
                  },
                },
              },
            ],
            minimum_should_match: 2,
          },
        },
        size: limit,
        sort: [{ salesCount: 'desc' }],
      },
    });

    return similarProducts.hits.hits.map((hit: any) => ({
      productId: hit._source.id,
      name: hit._source.name,
      price: hit._source.price,
      score: hit._score,
      reason: '相似商品',
    }));
  }

  /**
   * 提取商品特征
   */
  private extractProductFeatures(
    products: { productId: string; categoryId: number; brandId: number; price: number }[]
  ): {
    brandIds: number[];
    categoryIds: number[];
    avgPrice: number;
  } {
    const brandIds = [...new Set(products.map(p => p.brandId))];
    const categoryIds = [...new Set(products.map(p => p.categoryId))];
    const avgPrice = products.reduce((sum, p) => sum + p.price, 0) / products.length;

    return { brandIds, categoryIds, avgPrice };
  }

  /**
   * 获取热销商品
   */
  private async getHotProducts(limit: number): Promise<RecommendedProduct[]> {
    const cacheKey = 'recommend:hot';
    const cached = await this.redis.get(cacheKey);

    if (cached) {
      return JSON.parse(cached);
    }

    // 从ES查询热销商品
    const result = await this.esClient.search({
      index: 'products',
      body: {
        query: { match_all: {} },
        sort: [{ salesCount: 'desc' }],
        size: limit,
      },
    });

    const products = result.hits.hits.map((hit: any) => ({
      productId: hit._source.id,
      name: hit._source.name,
      price: hit._source.price,
      image: hit._source.images?.[0],
      score: 1,
      reason: '热销爆款',
    }));

    // 缓存1小时
    await this.redis.setex(cacheKey, 3600, JSON.stringify(products));

    return products;
  }

  /**
   * 加权混合排序
   */
  private hybridSort(
    items: (RecommendedProduct & { weight: number })[],
    limit: number
  ): RecommendedProduct[] {
    // 去重：同一商品只保留权重最高的
    const uniqueMap = new Map<string, RecommendedProduct & { weight: number }>();

    for (const item of items) {
      const existing = uniqueMap.get(item.productId);
      if (!existing || item.weight > existing.weight) {
        uniqueMap.set(item.productId, item);
      }
    }

    // 按权重排序
    return Array.from(uniqueMap.values())
      .sort((a, b) => b.weight - a.weight)
      .slice(0, limit)
      .map(({ weight, ...product }) => product);
  }
}
```

### 3.5 我的思考：搜索与推荐的区别

很多人容易把搜索和推荐混淆，但实际上它们有本质区别：

| 维度 | 搜索 | 推荐 |
|-----|-----|-----|
| **意图** | 用户有明确需求 | 用户无明确需求 |
| **触发** | 用户主动输入关键词 | 系统主动推送 |
| **优化目标** | 相关性 | 点击率/转化率 |
| **技术重点** | 文本匹配、相关性排序 | 用户理解、物品理解 |
| **评估指标** | Precision@K, NDCG | CTR, CVR, GMV |

### 4.1 购物车状态管理

购物车是电商转化率的关键节点，一个好的购物车系统需要考虑：

```typescript
/**
 * 购物车服务
 * 负责购物车的增删改查、价格计算等核心功能
 */
class CartService {
  private redis: Redis;
  private priceCalculator: PriceCalculator;

  /**
   * 添加商品到购物车
   * @param userId 用户ID
   * @param skuId SKU ID
   * @param quantity 数量
   */
  async addToCart(
    userId: string,
    skuId: number,
    quantity: number
  ): Promise<CartResult> {
    // 1. 验证SKU有效性
    const sku = await this.validateSku(skuId, quantity);
    if (!sku.valid) {
      return { success: false, error: sku.error };
    }

    // 2. 构建购物车Key
    const cartKey = `cart:${userId}`;

    // 3. 检查购物车中是否已存在该商品
    const existingItem = await this.redis.hget(cartKey, String(skuId));

    if (existingItem) {
      // 已存在，累加数量
      const item = JSON.parse(existingItem);
      const newQuantity = item.quantity + quantity;

      // 再次验证库存
      if (newQuantity > sku.stock) {
        return { success: false, error: '库存不足' };
      }

      // 更新数量
      item.quantity = newQuantity;
      item.selected = item.selected || false;
      item.updatedAt = Date.now();

      await this.redis.hset(cartKey, String(skuId), JSON.stringify(item));
    } else {
      // 新增商品
      const item: CartItem = {
        skuId,
        quantity,
        price: sku.price,
        originalPrice: sku.originPrice,
        selected: false,
        addedAt: Date.now(),
        updatedAt: Date.now(),
      };

      await this.redis.hset(cartKey, String(skuId), JSON.stringify(item));
    }

    // 4. 设置购物车过期时间（30天）
    await this.redis.expire(cartKey, 30 * 24 * 3600);

    // 5. 返回购物车统计
    return {
      success: true,
      cartCount: await this.getCartCount(userId),
    };
  }

  /**
   * 获取购物车详情
   */
  async getCartDetail(userId: string): Promise<CartDetail> {
    const cartKey = `cart:${userId}`;
    const cartData = await this.redis.hgetall(cartKey);

    if (!cartData || Object.keys(cartData).length === 0) {
      return {
        items: [],
        totalCount: 0,
        totalAmount: 0,
        totalDiscount: 0,
      };
    }

    const items: CartItemDetail[] = [];
    let totalAmount = 0;
    let totalOriginalAmount = 0;

    for (const [skuId, itemStr] of Object.entries(cartData)) {
      const item: CartItem = JSON.parse(itemStr);

      // 获取商品最新信息（价格、库存）
      const skuInfo = await this.getSkuInfo(Number(skuId));

      // 计算该商品的价格
      const priceDetail = await this.priceCalculator.calculatePrice(
        Number(skuId),
        userId,
        item.quantity,
        'APP'
      );

      // 检查商品状态
      let itemStatus: 'valid' | 'stockout' | 'offline' | 'price_changed' = 'valid';

      if (!skuInfo) {
        itemStatus = 'offline'; // 商品已下架
      } else if (skuInfo.stock < item.quantity) {
        itemStatus = 'stockout'; // 库存不足
      } else if (skuInfo.price !== item.price) {
        itemStatus = 'price_changed'; // 价格变动
      }

      items.push({
        skuId: Number(skuId),
        skuInfo,
        quantity: item.quantity,
        price: priceDetail.finalPrice,
        originalPrice: priceDetail.originPrice,
        selected: item.selected,
        subtotal: priceDetail.finalPrice * item.quantity,
        savings: (priceDetail.originPrice - priceDetail.finalPrice) * item.quantity,
        status: itemStatus,
        promotion: priceDetail.appliedPromotions,
      });

      totalAmount += priceDetail.finalPrice * item.quantity;
      totalOriginalAmount += priceDetail.originPrice * item.quantity;
    }

    return {
      items,
      totalCount: items.reduce((sum, item) => sum + item.quantity, 0),
      totalAmount,
      totalOriginalAmount,
      totalDiscount: totalOriginalAmount - totalAmount,
    };
  }

  /**
   * 选择购物车商品（用于结算）
   */
  async selectItems(
    userId: string,
    skuIds: number[],
    selected: boolean
  ): Promise<void> {
    const cartKey = `cart:${userId}`;

    for (const skuId of skuIds) {
      const itemStr = await this.redis.hget(cartKey, String(skuId));
      if (itemStr) {
        const item = JSON.parse(itemStr);
        item.selected = selected;
        item.updatedAt = Date.now();
        await this.redis.hset(cartKey, String(skuId), JSON.stringify(item));
      }
    }
  }

  /**
   * 全选/取消全选
   */
  async selectAll(userId: string, selected: boolean): Promise<void> {
    const cartKey = `cart:${userId}`;
    const cartData = await this.redis.hgetall(cartKey);

    for (const [skuId, itemStr] of Object.entries(cartData)) {
      const item = JSON.parse(itemStr);
      item.selected = selected;
      item.updatedAt = Date.now();
      await this.redis.hset(cartKey, skuId, JSON.stringify(item));
    }
  }

  /**
   * 更新商品数量
   */
  async updateQuantity(
    userId: string,
    skuId: number,
    quantity: number
  ): Promise<CartResult> {
    // 验证库存
    const sku = await this.validateSku(skuId, quantity);
    if (!sku.valid) {
      return { success: false, error: sku.error };
    }

    const cartKey = `cart:${userId}`;
    const itemStr = await this.redis.hget(cartKey, String(skuId));

    if (!itemStr) {
      return { success: false, error: '商品不在购物车中' };
    }

    const item = JSON.parse(itemStr);
    item.quantity = quantity;
    item.updatedAt = Date.now();

    await this.redis.hset(cartKey, String(skuId), JSON.stringify(item));

    return {
      success: true,
      cartCount: await this.getCartCount(userId),
    };
  }

  /**
   * 删除购物车商品
   */
  async removeItems(userId: string, skuIds: number[]): Promise<void> {
    const cartKey = `cart:${userId}`;

    for (const skuId of skuIds) {
      await this.redis.hdel(cartKey, String(skuId));
    }
  }

  /**
   * 清空购物车
   */
  async clearCart(userId: string): Promise<void> {
    const cartKey = `cart:${userId}`;
    await this.redis.del(cartKey);
  }

  /**
   * 验证SKU
   */
  private async validateSku(
    skuId: number,
    quantity: number
  ): Promise<{ valid: boolean; error?: string; stock?: number; price?: number }> {
    const skuInfo = await this.getSkuInfo(skuId);

    if (!skuInfo) {
      return { valid: false, error: '商品不存在或已下架' };
    }

    if (skuInfo.status !== 1) {
      return { valid: false, error: '商品已下架' };
    }

    if (skuInfo.stock < quantity) {
      return { valid: false, error: '库存不足' };
    }

    return {
      valid: true,
      stock: skuInfo.stock,
      price: skuInfo.price,
    };
  }

  /**
   * 获取SKU信息
   */
  private async getSkuInfo(skuId: number): Promise<SkuInfo | null> {
    const cacheKey = `sku:${skuId}`;
    const cached = await this.redis.get(cacheKey);

    if (cached) {
      return JSON.parse(cached);
    }

    // 从数据库查询
    const result = await db.query(`
      SELECT s.*, sp.name as spuName, sp.images
      FROM sku s
      INNER JOIN spu sp ON s.spuId = sp.id
      WHERE s.id = ?
    `, [skuId]);

    if (!result || result.length === 0) {
      return null;
    }

    const sku = result[0];
    const skuInfo: SkuInfo = {
      skuId: sku.id,
      spuName: sku.spuName,
      price: sku.price,
      originPrice: sku.originPrice,
      stock: sku.stock,
      status: sku.status,
      image: JSON.parse(sku.images || '[]')[0],
    };

    // 缓存1小时
    await this.redis.setex(cacheKey, 3600, JSON.stringify(skuInfo));

    return skuInfo;
  }

  /**
   * 获取购物车商品数量
   */
  private async getCartCount(userId: string): Promise<number> {
    const cartKey = `cart:${userId}`;
    const cartData = await this.redis.hgetall(cartKey);

    let count = 0;
    for (const itemStr of Object.values(cartData)) {
      const item = JSON.parse(itemStr);
      count += item.quantity;
    }
    return count;
  }
}
```

### 4.3 购物车价格计算

购物车的价格计算需要考虑多种优惠的叠加：

```typescript
/**
 * 购物车价格计算
 * 处理促销叠加、优惠券等问题
 */
class CartPriceCalculator {
  private promotionService: PromotionService;
  private couponService: CouponService;

  /**
   * 计算购物车价格
   */
  async calculateCartPrice(
    userId: string,
    cartDetail: CartDetail,
    params: PriceCalcParams
  ): Promise<CartPriceResult> {
    // 1. 分离选中和非选中商品
    const selectedItems = cartDetail.items.filter(item => item.selected);
    const unselectedItems = cartDetail.items.filter(item => !item.selected);

    if (selectedItems.length === 0) {
      return {
        selectedItems: [],
        unselectedItems,
        totalAmount: 0,
        totalDiscount: 0,
        payAmount: 0,
      };
    }

    // 2. 计算商品小计
    let totalAmount = selectedItems.reduce(
      (sum, item) => sum + item.price * item.quantity,
      0
    );
    let totalDiscount = 0;

    // 3. 计算促销优惠（满减、折扣等）
    const promotionResult = await this.calculatePromotionDiscount(
      userId,
      selectedItems,
      totalAmount
    );
    totalDiscount += promotionResult.discount;
    totalAmount -= promotionResult.discount;

    // 4. 计算优惠券优惠
    const couponResult = await this.calculateCouponDiscount(
      userId,
      params.couponId,
      selectedItems,
      totalAmount
    );
    totalDiscount += couponResult.discount;
    totalAmount -= couponResult.discount;

    // 5. 应用会员折扣
    const memberDiscount = await this.calculateMemberDiscount(
      userId,
      totalAmount
    );
    totalDiscount += memberDiscount;
    totalAmount -= memberDiscount;

    // 6. 最终金额（不能为负）
    const payAmount = Math.max(0, totalAmount);

    return {
      selectedItems,
      unselectedItems,
      totalAmount: cartDetail.totalAmount, // 商品总价
      promotionDiscount: promotionResult.discount,
      couponDiscount: couponResult.discount,
      memberDiscount,
      totalDiscount,
      payAmount,
      coupon: couponResult.coupon,
    };
  }

  /**
   * 计算促销优惠
   */
  private async calculatePromotionDiscount(
    userId: string,
    items: CartItemDetail[],
    totalAmount: number
  ): Promise<{ discount: number; appliedPromotions: AppliedPromotion[] }> {
    // 获取可用促销活动
    const activities = await this.promotionService.getAvailableActivities(
      items.map(item => item.skuId)
    );

    let discount = 0;
    const appliedPromotions: AppliedPromotion[] = [];

    for (const activity of activities) {
      // 检查是否满足活动条件
      if (!this.checkActivityCondition(activity, totalAmount, items)) {
        continue;
      }

      // 计算优惠金额
      const activityDiscount = this.calculateActivityDiscount(
        activity,
        totalAmount,
        items
      );

      if (activityDiscount > 0) {
        discount += activityDiscount;
        appliedPromotions.push({
          activityId: activity.id,
          activityName: activity.name,
          discountAmount: activityDiscount,
        });
      }
    }

    return { discount, appliedPromotions };
  }

  /**
   * 检查活动条件
   */
  private checkActivityCondition(
    activity: Activity,
    totalAmount: number,
    items: CartItemDetail[]
  ): boolean {
    switch (activity.type) {
      case 'reduction':
        // 满减活动：检查是否满足门槛
        return totalAmount >= activity.minAmount;

      case 'discount':
        // 折扣活动：通常无门槛
        return true;

      case 'special':
        // 特价活动：检查商品是否在活动范围内
        const activitySkuIds = new Set(activity.skuIds);
        return items.some(item => activitySkuIds.has(item.skuId));

      default:
        return false;
    }
  }

  /**
   * 计算活动优惠金额
   */
  private calculateActivityDiscount(
    activity: Activity,
    totalAmount: number,
    items: CartItemDetail[]
  ): number {
    switch (activity.type) {
      case 'reduction':
        // 满减：直接返回减免金额
        return Math.min(activity.reductionAmount, totalAmount);

      case 'discount':
        // 折扣：按比例计算
        return Math.floor(totalAmount * (1 - activity.discount / 100));

      case 'special':
        // 特价：计算特价与原价的差额
        const activitySkuIds = new Set(activity.skuIds);
        let savings = 0;
        for (const item of items) {
          if (activitySkuIds.has(item.skuId)) {
            const specialPrice = activity.specialPrice;
            savings += (item.price - specialPrice) * item.quantity;
          }
        }
        return savings;

      default:
        return 0;
    }
  }

  /**
   * 计算优惠券优惠
   */
  private async calculateCouponDiscount(
    userId: string,
    couponId: string | undefined,
    items: CartItemDetail[],
    totalAmount: number
  ): Promise<{ discount: number; coupon: Coupon | null }> {
    if (!couponId) {
      return { discount: 0, coupon: null };
    }

    // 获取优惠券详情
    const coupon = await this.couponService.getCoupon(couponId, userId);
    if (!coupon) {
      return { discount: 0, coupon: null };
    }

    // 检查优惠券是否可用
    if (!this.checkCouponAvailable(coupon, items, totalAmount)) {
      return { discount: 0, coupon: null };
    }

    // 计算优惠金额
    let discount = 0;
    if (coupon.type === 'amount') {
      discount = coupon.discountAmount;
    } else if (coupon.type === 'percent') {
      discount = Math.floor(totalAmount * coupon.discountPercent / 100);
    }

    return {
      discount: Math.min(discount, totalAmount), // 优惠金额不能超过订单金额
      coupon,
    };
  }

  /**
   * 检查优惠券是否可用
   */
  private checkCouponAvailable(
    coupon: Coupon,
    items: CartItemDetail[],
    totalAmount: number
  ): boolean {
    // 检查使用门槛
    if (coupon.minAmount > totalAmount) {
      return false;
    }

    // 检查适用范围（指定分类/商品）
    if (coupon.scopeType === 'category') {
      const categoryIds = new Set(items.map(item => item.skuInfo.categoryId));
      if (![...categoryIds].some(id => coupon.scopeIds.includes(id))) {
        return false;
      }
    } else if (coupon.scopeType === 'product') {
      const skuIds = new Set(items.map(item => item.skuId));
      if (![...skuIds].some(id => coupon.scopeIds.includes(id))) {
        return false;
      }
    }

    return true;
  }

  /**
   * 计算会员折扣
   */
  private async calculateMemberDiscount(
    userId: string,
    totalAmount: number
  ): Promise<number> {
    const memberLevel = await this.getMemberLevel(userId);

    // 不同会员等级对应不同折扣
    const discountRates: Record<number, number> = {
      0: 0,      // 普通会员：无额外折扣
      1: 0.02,   // 银牌：额外2%
      2: 0.05,   // 金牌：额外5%
      3: 0.08,   // 钻石：额外8%
      4: 0.10,   // 黑金：额外10%
    };

    const discountRate = discountRates[memberLevel] || 0;
    return Math.floor(totalAmount * discountRate);
  }

  private async getMemberLevel(userId: string): Promise<number> {
    const result = await db.query(
      'SELECT level FROM user WHERE id = ?',
      [userId]
    );
    return result[0]?.level || 0;
  }
}
```

### 5.3 创建订单核心流程

创建订单是电商系统最核心的流程之一，涉及多个系统的协调：

```typescript
/**
 * 订单服务
 * 负责订单创建、查询、取消等核心功能
 */
class OrderService {
  private orderRepo: OrderRepository;
  private inventoryService: InventoryService;
  private paymentService: PaymentService;
  private promotionService: PromotionService;
  private messageQueue: MQ;

  /**
   * 创建订单
   * 这是电商系统最核心的接口之一，需要保证原子性
   */
  async createOrder(userId: number, params: CreateOrderParams): Promise<OrderCreateResult> {
    const { items, addressId, couponId, buyerMessage, channel } = params;

    // 1. 校验地址
    const address = await this.validateAddress(userId, addressId);
    if (!address) {
      return { success: false, error: '收货地址不存在' };
    }

    // 2. 锁定库存（预扣库存）
    const lockResult = await this.lockInventory(items);
    if (!lockResult.success) {
      return { success: false, error: lockResult.error };
    }

    // 3. 获取商品信息并计算价格
    const productDetails = await this.getProductDetails(
      items.map(item => item.skuId)
    );

    // 4. 计算订单金额
    const priceResult = await this.calculateOrderPrice(
      userId,
      productDetails,
      couponId
    );

    // 5. 扣减库存（实扣）
    const deductResult = await this.inventoryService.deductStock(
      items.map(item => ({
        skuId: item.skuId,
        quantity: item.quantity,
        orderNo: '', // 先占位，后面更新
      }))
    );
    if (!deductResult.success) {
      // 释放锁定的库存
      await this.releaseInventoryLock(lockResult.lockIds);
      return { success: false, error: deductResult.error };
    }

    // 6. 生成订单号
    const orderNo = this.generateOrderNo(userId);

    // 7. 创建订单（使用事务）
    try {
      const order = await this.createOrderWithTransaction({
        orderNo,
        userId,
        items: productDetails,
        address,
        priceResult,
        buyerMessage,
        channel,
        lockIds: lockResult.lockIds,
      });

      // 8. 发送订单创建消息（用于异步处理）
      await this.messageQueue.send('order:created', {
        orderId: order.id,
        orderNo: order.orderNo,
        userId,
        createdAt: Date.now(),
      });

      return {
        success: true,
        orderId: order.id,
        orderNo: order.orderNo,
        payAmount: order.payAmount,
        expireTime: order.expireTime,
      };
    } catch (error) {
      // 回滚库存
      await this.inventoryService.rollbackStock(
        items.map(item => ({
          skuId: item.skuId,
          quantity: item.quantity,
        }))
      );
      throw error;
    }
  }

  /**
   * 锁定库存（预扣）
   * 在下单时先锁定库存，防止超卖
   */
  private async lockInventory(
    items: { skuId: number; quantity: number }[]
  ): Promise<{ success: boolean; lockIds?: string[]; error?: string }> {
    const lockIds: string[] = [];

    for (const item of items) {
      // 使用Redis SETNX实现分布式锁
      const lockKey = `inventory:lock:${item.skuId}`;
      const lockId = `lock:${item.skuId}:${Date.now()}:${Math.random()}`;

      // 尝试获取锁（设置10秒过期）
      const locked = await this.redis.set(lockKey, lockId, 'NX', 'EX', 10);

      if (!locked) {
        // 获取锁失败，回滚已锁定的库存
        await this.releaseInventoryLock(lockIds);
        return { success: false, error: `商品${item.skuId}库存锁定失败` };
      }

      lockIds.push(lockId);

      // 检查库存
      const stock = await this.getSkuStock(item.skuId);
      if (stock < item.quantity) {
        await this.releaseInventoryLock(lockIds);
        return { success: false, error: `商品${item.skuId}库存不足` };
      }
    }

    return { success: true, lockIds };
  }

  /**
   * 释放库存锁
   */
  private async releaseInventoryLock(lockIds: string[]): Promise<void> {
    for (const lockId of lockIds) {
      const match = lockId.match(/lock:(\d+):/);
      if (match) {
        const skuId = match[1];
        const lockKey = `inventory:lock:${skuId}`;
        await this.redis.del(lockKey);
      }
    }
  }

  /**
   * 获取SKU库存
   */
  private async getSkuStock(skuId: number): Promise<number> {
    const cacheKey = `sku:stock:${skuId}`;
    const cached = await this.redis.get(cacheKey);

    if (cached !== null) {
      return parseInt(cached, 10);
    }

    // 从数据库查询
    const result = await db.query(
      'SELECT stock FROM sku WHERE id = ?',
      [skuId]
    );

    const stock = result[0]?.stock || 0;

    // 缓存5秒
    await this.redis.setex(cacheKey, 5, String(stock));

    return stock;
  }

  /**
   * 计算订单价格
   */
  private async calculateOrderPrice(
    userId: number,
    productDetails: ProductDetail[],
    couponId?: string
  ): Promise<OrderPriceResult> {
    // 计算商品总金额
    let totalAmount = 0;
    let freightAmount = 0;
    const itemDetails: OrderItem[] = [];

    for (const product of productDetails) {
      const subtotal = product.price * product.quantity;
      totalAmount += subtotal;

      itemDetails.push({
        skuId: product.skuId,
        spuName: product.spuName,
        skuSpec: JSON.stringify(product.specs),
        image: product.image,
        price: product.price,
        quantity: product.quantity,
        subtotal,
        freightAmount: 0,
        promotionAmount: 0,
        couponAmount: 0,
        实际支付金额: subtotal,
      });
    }

    // 计算运费（满99包邮）
    freightAmount = totalAmount >= 9900 ? 0 : 1000; // 10元运费

    // 计算优惠
    let discountAmount = 0;
    let couponAmount = 0;
    let promotionDetails: PromotionDetail[] = [];

    // 促销活动优惠
    const promotionResult = await this.promotionService.calculatePromotions(
      productDetails.map(p => p.skuId)
    );
    discountAmount += promotionResult.totalDiscount;
    promotionDetails = promotionResult.details;

    // 优惠券优惠
    if (couponId) {
      const couponResult = await this.couponService.applyCoupon(
        couponId,
        userId,
        totalAmount - discountAmount
      );
      if (couponResult.success) {
        couponAmount = couponResult.discount;
      }
    }

    // 实付金额
    const payAmount = totalAmount + freightAmount - discountAmount - couponAmount;

    return {
      totalAmount,
      freightAmount,
      discountAmount,
      couponAmount,
      payAmount: Math.max(0, payAmount),
      itemDetails,
      promotionDetails,
    };
  }

  /**
   * 使用事务创建订单
   */
  private async createOrderWithTransaction(
    params: CreateOrderParamsInternal
  ): Promise<Order> {
    const {
      orderNo,
      userId,
      items,
      address,
      priceResult,
      buyerMessage,
      channel,
    } = params;

    // 生成订单过期时间（15分钟）
    const expireTime = new Date(Date.now() + 15 * 60 * 1000);

    // 构建订单对象
    const order: Order = {
      id: 0, // 自增ID
      orderNo,
      userId,
      status: OrderStatus.Pending,
      totalAmount: priceResult.totalAmount,
      freightAmount: priceResult.freightAmount,
      discountAmount: priceResult.discountAmount,
      couponAmount: priceResult.couponAmount,
      payAmount: priceResult.payAmount,
      receiverName: address.name,
      receiverPhone: address.phone,
      receiverProvince: address.province,
      receiverCity: address.city,
      receiverDistrict: address.district,
      receiverAddress: address.detail,
      receiverZipCode: address.zipCode,
      expireTime,
      buyerMessage,
      promotionDetails: JSON.stringify(priceResult.promotionDetails),
      channel,
      createdAt: new Date(),
      updatedAt: new Date(),
    };

    // 创建订单主表
    const orderId = await this.orderRepo.create(order);

    // 创建订单商品表
    for (const item of priceResult.itemDetails) {
      await this.orderRepo.createOrderItem({
        ...item,
        orderId,
      });
    }

    // 记录订单日志
    await this.orderRepo.createOrderLog({
      orderId,
      action: 'create',
      operator: 'user',
      operatorId: userId,
      content: '用户创建订单',
    });

    return { ...order, id: orderId };
  }

  /**
   * 支付订单
   */
  async payOrder(orderId: number, userId: number, payChannel: string): Promise<PayResult> {
    // 1. 查询订单
    const order = await this.orderRepo.findById(orderId);
    if (!order) {
      return { success: false, error: '订单不存在' };
    }

    // 2. 验证订单状态
    if (order.status !== OrderStatus.Pending) {
      return { success: false, error: '订单状态不允许支付' };
    }

    // 3. 验证订单是否过期
    if (new Date() > order.expireTime) {
      // 取消订单
      await this.cancelOrder(orderId, userId, '订单已过期');
      return { success: false, error: '订单已过期' };
    }

    // 4. 调用支付服务
    const paymentResult = await this.paymentService.createPayment({
      orderNo: order.orderNo,
      amount: order.payAmount,
      channel: payChannel,
      userId,
    });

    if (!paymentResult.success) {
      return { success: false, error: paymentResult.error };
    }

    return {
      success: true,
      paymentId: paymentResult.paymentId,
      payUrl: paymentResult.payUrl,
    };
  }

  /**
   * 支付回调
   * 处理支付成功/失败的回调
   */
  async handlePayCallback(
    paymentId: string,
    status: 'success' | 'failed',
    callbackData: any
  ): Promise<void> {
    // 查询支付记录
    const payment = await this.paymentService.getPayment(paymentId);
    if (!payment) {
      return;
    }

    // 根据支付状态更新订单
    if (status === 'success') {
      await this.orderRepo.updateStatus(
        payment.orderId,
        OrderStatus.Paid,
        { payTime: new Date() }
      );

      // 发送订单支付成功消息
      await this.messageQueue.send('order:paid', {
        orderId: payment.orderId,
        payTime: Date.now(),
      });
    } else {
      // 支付失败，记录日志
      await this.orderRepo.createOrderLog({
        orderId: payment.orderId,
        action: 'pay_failed',
        operator: 'payment',
        operatorId: 0,
        content: `支付失败：${callbackData.errorMsg || '未知错误'}`,
      });
    }
  }

  /**
   * 取消订单
   */
  async cancelOrder(orderId: number, userId: number, reason: string): Promise<void> {
    const order = await this.orderRepo.findById(orderId);

    if (!order) {
      throw new Error('订单不存在');
    }

    // 只有待支付状态的订单可以取消
    if (order.status !== OrderStatus.Pending) {
      throw new Error('订单状态不允许取消');
    }

    // 更新订单状态
    await this.orderRepo.updateStatus(orderId, OrderStatus.Cancelled, {
      cancelTime: new Date(),
    });

    // 释放库存
    const orderItems = await this.orderRepo.getOrderItems(orderId);
    await this.inventoryService.releaseStock(
      orderItems.map(item => ({
        skuId: item.skuId,
        quantity: item.quantity,
      }))
    );

    // 记录日志
    await this.orderRepo.createOrderLog({
      orderId,
      action: 'cancel',
      operator: 'user',
      operatorId: userId,
      content: `用户取消订单：${reason}`,
    });
  }

  /**
   * 生成订单号
   * 格式：时间戳(13位) + 随机数(4位) + 用户ID后4位
   */
  private generateOrderNo(userId: number): string {
    const timestamp = Date.now();
    const random = Math.floor(Math.random() * 10000).toString().padStart(4, '0');
    const userSuffix = String(userId).slice(-4).padStart(4, '0');
    return `${timestamp}${random}${userSuffix}`;
  }
}
```

### 5.5 我的思考：订单系统的核心

订单系统是电商系统中最复杂的部分，有几个关键点：

1. **库存一致性**：下单、支付、发货三个阶段都涉及库存，任何一步出错都可能导致超卖或资损。

2. **事务一致性**：创建订单涉及库存扣减、优惠券使用、余额扣减等多个操作，需要保证要么全部成功，要么全部失败。

3. **幂等性**：网络超时、重试等场景下，同一个订单可能被多次创建，需要保证幂等。

4. **可追溯性**：订单的每一步操作都需要记录日志，便于排查问题和数据分析。

## 六、库存系统

### 6.1 库存扣减策略

库存系统是电商防超卖的核心，常见策略有两种：

| 策略 | 原理 | 优点 | 缺点 |
|-----|-----|-----|-----|
| **预扣库存** | 下单时锁定库存，支付后实扣 | 防止恶意下单占用库存 | 需要处理超时释放 |
| **实扣库存** | 支付后直接扣库存 | 逻辑简单 | 可能导致超卖 |

推荐使用**预扣库存**策略，在下单时预扣，支付后确认，超时后释放。

### 6.2 Redis原子操作实现

```typescript
/**
 * 库存服务
 * 使用Redis实现高性能库存管理
 */
class InventoryService {
  private redis: Redis;
  private db: Database;

  /**
   * 扣减库存（乐观锁 + Redis原子操作）
   * @param skuId SKU ID
   * @param quantity 扣减数量
   * @param orderNo 订单号（用于追踪）
   */
  async deductStock(
    params: { skuId: number; quantity: number; orderNo: string }[]
  ): Promise<{ success: boolean; error?: string }> {
    // 使用Redis事务保证原子性
    const pipeline = this.redis.multi();

    for (const { skuId, quantity } of params) {
      // 获取当前库存
      const stockKey = `sku:stock:${skuId}`;

      // 使用Lua脚本保证原子性
      // 脚本逻辑：
      // 1. 检查库存是否充足
      // 2. 如果充足，扣减库存并返回新库存
      // 3. 如果不足，返回错误
      pipeline.eval(`
        local stock = tonumber(redis.call('GET', KEYS[1]) or 0)
        local quantity = tonumber(ARGV[1])
        if stock >= quantity then
          return redis.call('DECRBY', KEYS[1], quantity)
        else
          return -1
        end
      `, 1, stockKey, quantity);
    }

    const results = await pipeline.exec();

    // 检查执行结果
    for (let i = 0; i < results.length; i++) {
      const [err, newStock] = results[i];
      if (err || newStock === -1) {
        // 回滚已扣减的库存
        await this.rollbackStock(params.slice(0, i));
        return {
          success: false,
          error: `商品${params[i].skuId}库存不足`,
        };
      }
    }

    // 扣减成功，记录库存流水（异步写入数据库）
    await this.recordStockFlow(params, 'deduct');

    return { success: true };
  }

  /**
   * 回滚库存
   * 当订单创建失败时，释放已扣减的库存
   */
  async rollbackStock(
    params: { skuId: number; quantity: number }[]
  ): Promise<void> {
    const pipeline = this.redis.multi();

    for (const { skuId, quantity } of params) {
      const stockKey = `sku:stock:${skuId}`;
      pipeline.incrby(stockKey, quantity);
    }

    await pipeline.exec();

    // 记录库存流水
    await this.recordStockFlow(params, 'rollback');
  }

  /**
   * 库存预占
   * 在购物车或下单时预占库存，防止超卖
   */
  async preoccupyStock(
    userId: string,
    items: { skuId: number; quantity: number }[]
  ): Promise<{ success: boolean; lockId?: string; error?: string }> {
    const lockId = `lock:${Date.now()}:${Math.random()}`;

    // 检查是否已经预占过
    const existingLock = await this.redis.get(`user:stock:lock:${userId}`);
    if (existingLock) {
      return { success: false, error: '您有未支付的订单，请先处理' };
    }

    const pipeline = this.redis.multi();
    const lockKeys: string[] = [];

    for (const { skuId, quantity } of items) {
      const lockKey = `stock:pre:${skuId}`;
      const stockKey = `sku:stock:${skuId}`;

      // 锁定库存（减少可用库存，增加预占库存）
      pipeline.decrby(stockKey, quantity);
      pipeline.hincrby(lockKey, lockId, quantity);

      lockKeys.push(lockKey);
    }

    const results = await pipeline.exec();

    // 检查是否有库存不足
    for (let i = 0; i < results.length; i++) {
      const [err, newStock] = results[i];
      if (err || (newStock !== null && newStock < 0)) {
        // 回滚预占
        await this.releasePreoccupy(lockKeys, lockId);
        return {
          success: false,
          error: `商品${items[i].skuId}库存不足`,
        };
      }
    }

    // 记录预占信息（带过期时间，防止死锁）
    await this.redis.setex(
      `user:stock:lock:${userId}`,
      900, // 15分钟过期
      JSON.stringify({ lockId, items, createdAt: Date.now() })
    );

    // 记录预占日志
    await this.recordStockFlow(items, 'preoccupy', lockId);

    return { success: true, lockId };
  }

  /**
   * 释放预占库存
   * 取消订单或超时后释放预占的库存
   */
  async releasePreoccupy(
    lockKeys: string[],
    lockId: string
  ): Promise<void> {
    const pipeline = this.redis.multi();

    for (const lockKey of lockKeys) {
      // 获取该锁占用的库存数量
      const quantity = await this.redis.hget(lockKey, lockId);
      if (quantity) {
        // 提取SKU ID
        const skuId = lockKey.split(':')[2];
        const stockKey = `sku:stock:${skuId}`;

        // 释放库存
        pipeline.incrby(stockKey, parseInt(quantity, 10));
        pipeline.hdel(lockKey, lockId);
      }
    }

    await pipeline.exec();
  }

  /**
   * 确认预占（转实扣）
   * 支付成功后，将预占库存转为实际扣减
   */
  async confirmPreoccupy(
    userId: string,
    orderNo: string
  ): Promise<void> {
    const lockData = await this.redis.get(`user:stock:lock:${userId}`);
    if (!lockData) {
      return; // 没有预占记录
    }

    const { lockId, items } = JSON.parse(lockData);

    // 预占确认，删除预占记录
    // 实际库存已在预占时扣减，无需再次操作
    await this.redis.del(`user:stock:lock:${userId}`);

    // 更新库存流水状态
    await this.updateStockFlowStatus(orderNo, 'confirmed');

    // 记录确认日志
    await this.recordStockFlow(items, 'confirm', lockId, orderNo);
  }

  /**
   * 获取实时库存
   */
  async getRealTimeStock(skuId: number): Promise<number> {
    const cacheKey = `sku:stock:${skuId}`;
    const cached = await this.redis.get(cacheKey);

    if (cached !== null) {
      return parseInt(cached, 10);
    }

    // 从数据库查询
    const result = await this.db.query(
      'SELECT stock FROM sku WHERE id = ?',
      [skuId]
    );

    const stock = result[0]?.stock || 0;

    // 缓存5秒
    await this.redis.setex(cacheKey, 5, String(stock));

    return stock;
  }

  /**
   * 初始化库存缓存
   * 系统启动或库存变动时调用
   */
  async initStockCache(skuIds: number[]): Promise<void> {
    const pipeline = this.redis.multi();

    for (const skuId of skuIds) {
      const result = await this.db.query(
        'SELECT stock FROM sku WHERE id = ?',
        [skuId]
      );
      const stock = result[0]?.stock || 0;

      pipeline.set(`sku:stock:${skuId}`, stock);
    }

    await pipeline.exec();
  }

  /**
   * 记录库存流水
   */
  private async recordStockFlow(
    items: { skuId: number; quantity: number }[],
    type: string,
    lockId?: string,
    orderNo?: string
  ): Promise<void> {
    const flows = items.map(item => ({
      skuId: item.skuId,
      quantity: item.quantity,
      type,
      lockId,
      orderNo,
      createdAt: new Date(),
    }));

    // 异步写入数据库
    this.db.query(
      'INSERT INTO stock_flow (sku_id, quantity, type, lock_id, order_no, created_at) VALUES ?',
      [flows]
    ).catch(err => {
      console.error('记录库存流水失败:', err);
    });
  }

  /**
   * 更新库存流水状态
   */
  private async updateStockFlowStatus(
    orderNo: string,
    status: string
  ): Promise<void> {
    await this.db.query(
      'UPDATE stock_flow SET status = ? WHERE order_no = ?',
      [status, orderNo]
    );
  }
}
```

### 7.2 支付服务实现

```typescript
/**
 * 支付服务
 * 负责对接第三方支付渠道，处理支付下单、回调等核心功能
 */
class PaymentService {
  private db: Database;
  private redis: Redis;
  private mq: MQ;

  // 支付渠道配置
  private channels = {
    alipay: { appId: 'xxx', privateKey: 'xxx', alipayPublicKey: 'xxx' },
    wxpay: { appId: 'xxx', mchId: 'xxx', apiKey: 'xxx' },
  };

  /**
   * 创建支付
   * @param params 支付参数
   */
  async createPayment(params: CreatePaymentParams): Promise<PaymentResult> {
    const { orderNo, amount, channel, userId } = params;

    // 1. 查询订单信息
    const order = await this.getOrder(orderNo);
    if (!order) {
      return { success: false, error: '订单不存在' };
    }

    // 2. 检查订单状态
    if (order.status !== OrderStatus.Pending) {
      return { success: false, error: '订单状态不允许支付' };
    }

    // 3. 检查订单是否已支付（幂等性检查）
    const existingPayment = await this.getPaymentByOrderNo(orderNo);
    if (existingPayment && existingPayment.status === 'paid') {
      return { success: false, error: '订单已支付' };
    }

    // 4. 生成支付单号
    const paymentId = this.generatePaymentId();

    // 5. 调用第三方支付
    let payUrl: string;
    let payParams: any;

    if (channel === 'alipay') {
      const result = await this.alipayPay(paymentId, orderNo, amount);
      payUrl = result.payUrl;
      payParams = result.params;
    } else if (channel === 'wxpay') {
      const result = await this.wxpayPay(paymentId, orderNo, amount, userId);
      payUrl = result.payUrl;
      payParams = result.params;
    } else {
      return { success: false, error: '不支持的支付渠道' };
    }

    // 6. 创建支付记录
    await this.createPaymentRecord({
      paymentId,
      orderNo,
      amount,
      channel,
      status: 'pending',
      payParams,
    });

    return {
      success: true,
      paymentId,
      payUrl,
      payParams,
    };
  }

  /**
   * 支付宝支付
   */
  private async alipayPay(
    paymentId: string,
    orderNo: string,
    amount: number
  ): Promise<{ payUrl: string; params: any }> {
    const { appId, privateKey } = this.channels.alipay;

    // 构建支付宝支付参数
    const bizContent = {
      method: 'alipay.trade.wap.pay',
      bizContent: {
        out_trade_no: paymentId,        // 商户订单号
        total_amount: (amount / 100).toFixed(2),  // 金额（元）
        subject: '商品订单支付',           // 订单标题
        product_code: 'QUICK_WAP_WAY',   // 产品码
      },
    };

    // 使用支付宝SDK签名
    // 实际项目中应使用官方SDK：@alipay/alipay-sdk
    const sign = this.alipaySign(bizContent, privateKey);

    // 构建支付链接
    const payUrl = `https://openapi.alipay.com/gateway.do?${querystring.stringify({
      app_id: appId,
      method: 'POST',
      charset: 'utf-8',
      sign_type: 'RSA2',
      timestamp: this.formatDate(new Date()),
      version: '1.0',
      biz_content: JSON.stringify(bizContent.bizContent),
      sign,
    })}`;

    return {
      payUrl,
      params: { appId, paymentId, orderNo },
    };
  }

  /**
   * 微信支付
   */
  private async wxpayPay(
    paymentId: string,
    orderNo: string,
    amount: number,
    userId: string
  ): Promise<{ payUrl: string; params: any }> {
    const { appId, mchId, apiKey } = this.channels.wxpay;

    // 统一下单参数
    const params = {
      appid: appId,
      mch_id: mchId,
      nonce_str: this.generateNonce(),
      body: '商品订单支付',
      out_trade_no: paymentId,
      total_fee: amount,  // 金额（分）
      spbill_create_ip: '127.0.0.1',
      notify_url: `${process.env.PAY_NOTIFY_URL}/wxpay`,
      trade_type: 'NATIVE',
    };

    // 生成签名
    params.sign = this.wxpaySign(params, apiKey);

    // 调用微信统一下单接口
    const response = await this.httpClient.post(
      'https://api.mch.weixin.qq.com/pay/unifiedorder',
      params
    );

    const result = this.parseXml(response);

    if (result.return_code !== 'SUCCESS') {
      throw new Error(`微信支付创建失败：${result.return_msg}`);
    }

    // 生成支付二维码链接
    const payUrl = result.code_url;

    return {
      payUrl,
      params: { appId, paymentId, orderNo, codeUrl: payUrl },
    };
  }

  /**
   * 支付回调处理
   * 这是支付系统最关键的接口，需要保证幂等性和安全性
   */
  async handleNotify(
    channel: string,
    notifyData: any,
    sign: string
  ): Promise<{ success: boolean; error?: string }> {
    // 1. 验证签名
    if (!this.verifyNotifySign(channel, notifyData, sign)) {
      return { success: false, error: '签名验证失败' };
    }

    // 2. 解析回调数据
    let paymentId: string;
    let status: string;
    let amount: number;

    if (channel === 'alipay') {
      paymentId = notifyData.out_trade_no;
      status = notifyData.trade_status;
      amount = Math.round(parseFloat(notifyData.total_amount) * 100);
    } else if (channel === 'wxpay') {
      paymentId = notifyData.out_trade_no;
      status = notifyData.result_code === 'SUCCESS' ? 'SUCCESS' : 'FAIL';
      amount = parseInt(notifyData.total_fee, 10);
    } else {
      return { success: false, error: '不支持的支付渠道' };
    }

    // 3. 幂等性检查：使用分布式锁
    const lockKey = `payment:notify:${paymentId}`;
    const lock = await this.redis.set(lockKey, '1', 'NX', 'EX', 30);

    if (!lock) {
      // 已经在处理中
      return { success: true };
    }

    try {
      // 4. 查询支付记录
      const payment = await this.getPayment(paymentId);
      if (!payment) {
        return { success: false, error: '支付记录不存在' };
      }

      // 5. 检查是否已处理
      if (payment.status === 'paid') {
        return { success: true }; // 已处理，直接返回成功
      }

      // 6. 验证金额
      if (payment.amount !== amount) {
        return { success: false, error: '金额不匹配' };
      }

      // 7. 更新支付状态
      if (status === 'SUCCESS' || status === 'TRADE_SUCCESS') {
        await this.updatePaymentStatus(paymentId, 'paid', notifyData);

        // 发送支付成功消息
        await this.mq.send('payment:success', {
          paymentId,
          orderNo: payment.orderNo,
          amount,
          channel,
          paidAt: new Date(),
        });
      } else {
        // 支付失败
        await this.updatePaymentStatus(paymentId, 'failed', notifyData);
      }

      return { success: true };
    } finally {
      await this.redis.del(lockKey);
    }
  }

  /**
   * 验证回调签名
   */
  private verifyNotifySign(
    channel: string,
    data: any,
    sign: string
  ): boolean {
    if (channel === 'alipay') {
      // 支付宝：使用公钥验签
      const { alipayPublicKey } = this.channels.alipay;
      return this.alipayVerify(data, sign, alipayPublicKey);
    } else if (channel === 'wxpay') {
      // 微信：重新签名比对
      const { apiKey } = this.channels.wxpay;
      const calculatedSign = this.wxpaySign(data, apiKey);
      return calculatedSign === sign;
    }
    return false;
  }

  /**
   * 查询支付状态
   */
  async queryPayment(paymentId: string): Promise<PaymentStatus> {
    const payment = await this.getPayment(paymentId);
    if (!payment) {
      return { status: 'not_found' };
    }

    // 如果是pending状态，尝试查询第三方
    if (payment.status === 'pending') {
      const channelStatus = await this.queryChannelStatus(
        payment.channel,
        paymentId
      );

      if (channelStatus === 'paid') {
        // 第三方已支付，更新状态
        await this.updatePaymentStatus(paymentId, 'paid', {});
        return { status: 'paid' };
      } else if (channelStatus === 'expired') {
        // 已过期
        await this.updatePaymentStatus(paymentId, 'expired', {});
        return { status: 'expired' };
      }
    }

    return { status: payment.status };
  }

  /**
   * 申请退款
   */
  async refund(
    paymentId: string,
    amount: number,
    reason: string,
    operatorId: number
  ): Promise<{ success: boolean; error?: string }> {
    const payment = await this.getPayment(paymentId);

    if (!payment) {
      return { success: false, error: '支付记录不存在' };
    }

    if (payment.status !== 'paid') {
      return { success: false, error: '仅已支付的订单可以退款' };
    }

    // 调用第三方退款接口
    let refundResult: any;

    if (payment.channel === 'alipay') {
      refundResult = await this.alipayRefund(paymentId, amount);
    } else if (payment.channel === 'wxpay') {
      refundResult = await this.wxpayRefund(paymentId, amount);
    }

    if (refundResult.success) {
      // 创建退款记录
      await this.createRefundRecord({
        paymentId,
        amount,
        reason,
        operatorId,
        refundNo: refundResult.refundNo,
        status: 'pending',
      });

      // 发送退款消息
      await this.mq.send('payment:refund', {
        paymentId,
        orderNo: payment.orderNo,
        amount,
      });

      return { success: true };
    } else {
      return { success: false, error: refundResult.error };
    }
  }

  /**
   * 生成支付单号
   */
  private generatePaymentId(): string {
    const timestamp = Date.now();
    const random = Math.floor(Math.random() * 1000000).toString().padStart(6, '0');
    return `P${timestamp}${random}`;
  }

  /**
   * 生成随机字符串
   */
  private generateNonce(): string {
    return Math.random().toString(36).substring(2, 15);
  }

  /**
   * 格式化日期
   */
  private formatDate(date: Date): string {
    return date.format('yyyy-MM-dd hh:mm:ss');
  }

  /**
   * 微信支付签名
   */
  private wxpaySign(params: any, key: string): string {
    const sortedKeys = Object.keys(params).sort();
    const signStr = sortedKeys
      .filter(k => k !== 'sign' && params[k])
      .map(k => `${k}=${params[k]}`)
      .join('&');
    const sign = crypto
      .createHmac('sha256', key)
      .update(signStr + '&key=' + key)
      .digest('hex')
      .toUpperCase();
    return sign;
  }
}
```

### 7.4 我的思考：支付系统的严肃性

支付系统是电商系统中最不能出错的模块，有几个关键点：

1. **资金安全第一**：任何漏洞都可能导致资损，必须使用成熟的支付框架。

2. **幂等性必须保证**：网络超时、重试等都可能触发重复支付，必须保证幂等。

3. **对账必不可少**：每天日终必须对账，确保平台和渠道的数据一致。

4. **回调要验证签名**：安全验证是一切的前提。

### 8.1 促销类型概述

电商促销系统主要包含以下几种促销类型：

| 类型 | 说明 | 示例 |
|-----|-----|-----|
| **优惠券** | 用户主动领取的优惠凭证 | 满100减20券、新用户专享券 |
| **满减** | 订单金额达到门槛自动减免 | 满300减50 |
| **折扣** | 按比例打折 | 8折、5折起 |
| **特价** | 指定商品特殊价格 | 第二件半价 |
| **秒杀** | 限时限量低价抢购 | 整点秒杀 |
| **拼团** | 多人成团享受优惠 | 3人成团立减100 |
| **砍价** | 邀请好友砍价 | 0元拿商品 |

### 8.3 促销计算引擎

```typescript
/**
 * 促销计算引擎
 * 负责计算订单可以享受的促销优惠
 */
class PromotionEngine {
  private activityRepo: ActivityRepository;
  private couponRepo: CouponRepository;
  private redis: Redis;

  /**
   * 计算最优促销方案
   * 促销叠加规则：
   * 1. 优惠券和活动优惠可以同时享受
   * 2. 多个活动只能享受一个（取最优）
   * 3. 活动之间互斥，按优先级选择
   */
  async calculateBestPromotion(
    userId: number,
    items: { skuId: number; price: number; quantity: number }[],
    couponId?: string
  ): Promise<PromotionResult> {
    const skuIds = items.map(item => item.skuId);
    const totalAmount = items.reduce(
      (sum, item) => sum + item.price * item.quantity,
      0
    );

    // 1. 获取用户可用的促销活动
    const activities = await this.getAvailableActivities(userId, skuIds);

    // 2. 计算每个活动的优惠金额
    const activityResults = activities.map(activity => ({
      activity,
      discount: this.calculateActivityDiscount(activity, items, totalAmount),
    }));

    // 3. 选择最优活动（互斥）
    const bestActivity = this.selectBestActivity(activityResults);

    // 4. 计算优惠券优惠
    let couponDiscount = 0;
    let coupon: Coupon | null = null;

    if (couponId) {
      const couponResult = await this.calculateCouponDiscount(
        userId,
        couponId,
        totalAmount - (bestActivity?.discount || 0)
      );
      if (couponResult.success) {
        couponDiscount = couponResult.discount;
        coupon = couponResult.coupon;
      }
    }

    // 5. 计算最终优惠
    const totalDiscount = (bestActivity?.discount || 0) + couponDiscount;

    return {
      activity: bestActivity?.activity || null,
      activityDiscount: bestActivity?.discount || 0,
      coupon,
      couponDiscount,
      totalDiscount,
      finalAmount: Math.max(0, totalAmount - totalDiscount),
    };
  }

  /**
   * 获取可用的促销活动
   */
  private async getAvailableActivities(
    userId: number,
    skuIds: number[]
  ): Promise<Activity[]> {
    const now = Date.now();

    // 从Redis缓存获取活动列表
    const cacheKey = 'activities:active';
    const cached = await this.redis.get(cacheKey);

    let activities: Activity[] = cached
      ? JSON.parse(cached)
      : await this.loadActivities();

    // 过滤符合条件的活动
    return activities.filter(activity => {
      // 检查时间
      if (now < activity.startTime.getTime()) return false;
      if (now > activity.endTime.getTime()) return false;

      // 检查状态
      if (activity.status !== 2) return false;

      // 检查范围
      if (!this.isInScope(activity, skuIds)) return false;

      // 检查限制
      if (activity.totalLimit > 0 && activity.usedCount >= activity.totalLimit) {
        return false;
      }

      return true;
    });
  }

  /**
   * 检查商品是否在活动范围内
   */
  private isInScope(activity: Activity, skuIds: number[]): boolean {
    if (activity.scopeType === 'all') return true;

    if (activity.scopeType === 'sku') {
      return skuIds.some(id => activity.scopeIds.includes(id));
    }

    if (activity.scopeType === 'product' || activity.scopeType === 'category') {
      // 查询商品所属分类/商品
      const productInfos = await this.getProductInfos(skuIds);

      for (const product of productInfos) {
        if (activity.scopeType === 'category' && activity.scopeIds.includes(product.categoryId)) {
          return true;
        }
        if (activity.scopeType === 'product' && activity.scopeIds.includes(product.spuId)) {
          return true;
        }
      }
    }

    return false;
  }

  /**
   * 计算活动优惠金额
   */
  private calculateActivityDiscount(
    activity: Activity,
    items: { skuId: number; price: number; quantity: number }[],
    totalAmount: number
  ): number {
    for (const rule of activity.rules) {
      // 检查是否满足条件
      let satisfied = false;

      if (rule.condition.type === 'amount') {
        satisfied = totalAmount >= rule.condition.value;
      } else if (rule.condition.type === 'quantity') {
        const totalQuantity = items.reduce((sum, item) => sum + item.quantity, 0);
        satisfied = totalQuantity >= rule.condition.value;
      }

      if (!satisfied) continue;

      // 计算优惠
      switch (rule.action.type) {
        case '减钱':
          return rule.action.value;

        case '打折':
          return Math.floor(totalAmount * (1 - rule.action.value / 100));

        case '一口价':
          // 计算节省金额
          const originalAmount = items.reduce(
            (sum, item) => sum + item.price * item.quantity,
            0
          );
          return originalAmount - rule.action.value * items.reduce((sum, item) => sum + item.quantity, 0);

        default:
          return 0;
      }
    }

    return 0;
  }

  /**
   * 选择最优活动（互斥）
   */
  private selectBestActivity(
    activityResults: { activity: Activity; discount: number }[]
  ): { activity: Activity; discount: number } | null {
    if (activityResults.length === 0) return null;

    // 按优惠金额降序排序
    activityResults.sort((a, b) => b.discount - a.discount);

    // 返回最优的
    return activityResults[0];
  }

  /**
   * 计算优惠券优惠
   */
  private async calculateCouponDiscount(
    userId: number,
    couponId: string,
    orderAmount: number
  ): Promise<{ success: boolean; discount: number; coupon?: Coupon }> {
    // 获取用户优惠券
    const userCoupon = await this.couponRepo.findByIdAndUser(
      couponId,
      userId
    );

    if (!userCoupon) {
      return { success: false, discount: 0 };
    }

    // 检查优惠券状态
    if (userCoupon.status !== 'unused') {
      return { success: false, discount: 0 };
    }

    // 检查有效期
    const now = Date.now();
    if (now < userCoupon.validStartTime.getTime()) {
      return { success: false, discount: 0 };
    }
    if (now > userCoupon.validEndTime.getTime()) {
      return { success: false, discount: 0 };
    }

    // 检查使用门槛
    const coupon = await this.couponRepo.getCouponById(userCoupon.couponId);
    if (!coupon || orderAmount < coupon.minAmount) {
      return { success: false, discount: 0 };
    }

    // 计算优惠金额
    let discount = 0;
    if (coupon.type === 'amount') {
      discount = coupon.discountAmount;
    } else if (coupon.type === 'percent') {
      discount = Math.floor(orderAmount * coupon.discountPercent / 100);
    }

    // 优惠金额不能超过订单金额
    discount = Math.min(discount, orderAmount);

    return { success: true, discount, coupon };
  }

  /**
   * 验证并使用优惠券
   */
  async useCoupon(
    userId: number,
    couponId: string,
    orderNo: string
  ): Promise<{ success: boolean; error?: string }> {
    const userCoupon = await this.couponRepo.findByIdAndUser(couponId, userId);

    if (!userCoupon) {
      return { success: false, error: '优惠券不存在' };
    }

    if (userCoupon.status !== 'unused') {
      return { success: false, error: '优惠券已使用或已过期' };
    }

    // 更新优惠券状态
    await this.couponRepo.updateStatus(couponId, userId, 'used', orderNo);

    return { success: true };
  }

  /**
   * 退还优惠券（订单取消时）
   */
  async returnCoupon(userId: number, couponId: string): Promise<void> {
    await this.couponRepo.updateStatus(couponId, userId, 'unused');
  }
}
```

### 9.1 商品收藏功能

收藏是用户表达购买意愿的重要行为，也是精准营销的数据基础：

```typescript
/**
 * 收藏服务
 */
class FavoriteService {
  private redis: Redis;
  private db: Database;

  /**
   * 添加收藏
   */
  async addFavorite(userId: number, skuId: number): Promise<FavoriteResult> {
    // 检查商品是否存在
    const product = await this.getProductInfo(skuId);
    if (!product) {
      return { success: false, error: '商品不存在' };
    }

    // 检查是否已收藏
    const existing = await this.isFavorited(userId, skuId);
    if (existing) {
      return { success: false, error: '已收藏过该商品' };
    }

    // 添加收藏
    const now = Date.now();
    await this.db.query(
      `INSERT INTO user_favorite (user_id, sku_id, created_at)
       VALUES (?, ?, ?)`,
      [userId, skuId, now]
    );

    // 更新缓存
    await this.redis.lpush(`favorite:${userId}`, skuId);

    return { success: true };
  }

  /**
   * 取消收藏
   */
  async removeFavorite(userId: number, skuId: number): Promise<void> {
    await this.db.query(
      'DELETE FROM user_favorite WHERE user_id = ? AND sku_id = ?',
      [userId, skuId]
    );

    // 更新缓存
    await this.redis.lrem(`favorite:${userId}`, 0, skuId);
  }

  /**
   * 获取收藏列表
   */
  async getFavorites(
    userId: number,
    page = 1,
    pageSize = 20
  ): Promise<FavoriteListResult> {
    const offset = (page - 1) * pageSize;

    // 查询收藏列表
    const favorites = await this.db.query(
      `SELECT f.id, f.sku_id, f.created_at as favoriteTime,
              s.price, s.origin_price, s.stock, s.status,
              sp.name as productName, sp.images
       FROM user_favorite f
       INNER JOIN sku s ON f.sku_id = s.id
       INNER JOIN spu sp ON s.spu_id = sp.id
       WHERE f.user_id = ?
       ORDER BY f.created_at DESC
       LIMIT ? OFFSET ?`,
      [userId, pageSize, offset]
    );

    // 统计总数
    const totalResult = await this.db.query(
      'SELECT COUNT(*) as total FROM user_favorite WHERE user_id = ?',
      [userId]
    );
    const total = totalResult[0].total;

    // 处理商品状态
    const items = favorites.map(fav => ({
      id: fav.id,
      skuId: fav.sku_id,
      productName: fav.productName,
      image: JSON.parse(fav.images || '[]')[0],
      price: fav.price,
      originPrice: fav.origin_price,
      favoriteTime: fav.favoriteTime,
      status: this.getProductStatus(fav),
    }));

    return {
      items,
      total,
      page,
      pageSize,
      totalPages: Math.ceil(total / pageSize),
    };
  }

  /**
   * 检查是否已收藏
   */
  private async isFavorited(userId: number, skuId: number): Promise<boolean> {
    const result = await this.db.query(
      'SELECT 1 FROM user_favorite WHERE user_id = ? AND sku_id = ?',
      [userId, skuId]
    );
    return result.length > 0;
  }

  /**
   * 获取商品状态描述
   */
  private getProductStatus(sku: any): string {
    if (sku.status !== 1) return '已下架';
    if (sku.stock <= 0) return '暂时缺货';
    return '在售';
  }
}
```

### 9.2 足迹记录

用户浏览历史是推荐系统的重要数据来源：

```typescript
/**
 * 足迹服务
 */
class FootprintService {
  private redis: Redis;
  private db: Database;

  /**
   * 记录访问足迹
   */
  async addFootprint(userId: number, skuId: number): Promise<void> {
    const now = Date.now();

    // 写入Redis List（保留最近100条）
    const key = `footprint:${userId}`;
    const data = JSON.stringify({ skuId, time: now });

    // 先移除可能存在的旧记录
    const existing = await this.redis.lrange(key, 0, -1);
    for (const item of existing) {
      const parsed = JSON.parse(item);
      if (parsed.skuId === skuId) {
        await this.redis.lrem(key, 1, item);
      }
    }

    // 添加到列表头部
    await this.redis.lpush(key, data);
    await this.redis.ltrim(key, 0, 99); // 只保留100条

    // 设置过期时间（30天）
    await this.redis.expire(key, 30 * 24 * 3600);

    // 异步写入数据库（去重）
    this.syncToDatabase(userId, skuId, now).catch(err => {
      console.error('同步足迹失败:', err);
    });
  }

  /**
   * 获取访问足迹
   */
  async getFootprints(
    userId: number,
    page = 1,
    pageSize = 20
  ): Promise<FootprintListResult> {
    const key = `footprint:${userId}`;
    const start = (page - 1) * pageSize;
    const end = start + pageSize - 1;

    // 从Redis获取
    const items = await this.redis.lrange(key, start, end);

    if (items.length === 0) {
      return { items: [], total: 0, page, pageSize };
    }

    // 解析并获取商品信息
    const skuIds = items.map(item => JSON.parse(item).skuId);
    const products = await this.getProductInfos(skuIds);

    const result = items.map(item => {
      const { skuId, time } = JSON.parse(item);
      const product = products.find(p => p.skuId === skuId);
      return {
        skuId,
        time,
        productName: product?.productName,
        image: product?.image,
        price: product?.price,
        status: product?.status,
      };
    });

    // 获取总数
    const total = await this.redis.llen(key);

    return {
      items: result,
      total,
      page,
      pageSize,
      totalPages: Math.ceil(total / pageSize),
    };
  }

  /**
   * 清除全部足迹
   */
  async clearFootprints(userId: number): Promise<void> {
    const key = `footprint:${userId}`;
    await this.redis.del(key);
  }

  /**
   * 异步同步到数据库
   */
  private async syncToDatabase(
    userId: number,
    skuId: number,
    time: number
  ): Promise<void> {
    // 使用REPLACE INTO保证去重
    await this.db.query(
      `REPLACE INTO user_footprint (user_id, sku_id, created_at)
       VALUES (?, ?, ?)`,
      [userId, skuId, new Date(time)]
    );
  }
}
```

```typescript
    if (userBought && parseInt(userBought, 10) >= 1) {

      else
        return 0
      end
```

```typescript
/**
 * 限流服务
 * 实现多种限流算法
 */
class RateLimitService {
  private redis: Redis;

  /**
   * 令牌桶限流
   * 适合突发流量场景
   */
  async tokenBucket(
    key: string,
    capacity: number,    // 桶容量
    refillRate: number,  // 每秒补充的令牌数
  ): Promise<{ allowed: boolean; remaining: number }> {
    const now = Date.now() / 1000;
    const bucketKey = `ratelimit:bucket:${key}`;

    // 获取桶状态
    const bucket = await this.redis.hgetall(bucketKey);
    let tokens = parseFloat(bucket.tokens || String(capacity));
    let lastRefill = parseFloat(bucket.lastRefill || String(now));

    // 补充令牌
    const secondsPassed = now - lastRefill;
    tokens = Math.min(capacity, tokens + secondsPassed * refillRate);
    lastRefill = now;

    // 保存桶状态
    await this.redis.hmset(bucketKey, {
      tokens: String(tokens),
      lastRefill: String(lastRefill),
    });
    await this.redis.expire(bucketKey, 60);

    // 尝试获取令牌
    if (tokens >= 1) {
      tokens -= 1;
      return { allowed: true, remaining: Math.floor(tokens) };
    }

    return { allowed: false, remaining: 0 };
  }

  /**
   * 滑动窗口限流
   * 精确控制时间窗口内的请求数
   */
  async slidingWindow(
    key: string,
    limit: number,       // 窗口内最大请求数
    windowMs: number,     // 窗口大小（毫秒）
  ): Promise<{ allowed: boolean; remaining: number }> {
    const now = Date.now();
    const windowKey = `ratelimit:window:${key}`;

    // 删除过期的请求记录
    await this.redis.zremrangebyscore(windowKey, 0, now - windowMs);

    // 统计当前窗口内的请求数
    const count = await this.redis.zcard(windowKey);

    if (count < limit) {
      // 添加当前请求
      await this.redis.zadd(windowKey, now, `${now}:${Math.random()}`);
      await this.redis.expire(windowKey, Math.ceil(windowMs / 1000));
      return { allowed: true, remaining: limit - count - 1 };
    }

    return { allowed: false, remaining: 0 };
  }

  /**
   * 计数器限流（简单粗暴）
   */
  async counter(
    key: string,
    limit: number,
    windowSec: number,
  ): Promise<{ allowed: boolean; remaining: number; resetIn: number }> {
    const now = Date.now();
    const keyWithWindow = `${key}:${Math.floor(now / (windowSec * 1000))}`;

    const current = await this.redis.incr(keyWithWindow);

    if (current === 1) {
      await this.redis.expire(keyWithWindow, windowSec);
    }

    const remaining = Math.max(0, limit - current);
    const ttl = await this.redis.ttl(keyWithWindow);

    return {
      allowed: current <= limit,
      remaining,
      resetIn: ttl > 0 ? ttl : windowSec,
    };
  }
}
```

### 12.2 核心问题

#### 问题1：库存超卖

**现象**：订单数量远超库存数量，导致大量订单无法发货。

**原因**：
- 数据库行锁竞争严重，性能下降
- 库存扣减和订单创建不在同一事务

**解决方案**：
- 分层库存：Redis库存（预扣） + 数据库库存（实扣）
- 使用Lua脚本保证Redis原子性
- 异步化订单创建，解耦库存扣减

#### 问题2：支付失败率高

**现象**：下单后支付失败，用户体验极差。

**原因**：
- 支付渠道限流
- 异步回调处理慢
- 库存预占超时释放

**解决方案**：
- 支付前置：下单即占库存，支付时直接扣款
- 分层限流：CDN -> 应用层 -> 业务层
- 异步回调优化：批量处理、统一应答

**现象**：某个服务故障导致整体不可用。

**原因**：
- 服务强依赖
- 没有降级策略
- 缓存集体失效

**解决方案**：
- 服务拆分，隔离故障
- 降级熔断：Sentinel/Hystrix
- 多级缓存：本地 -> Redis -> 数据库

### 12.5 反思与总结

双十一的经历让我深刻理解：

1. **预案比预测更重要**：再完美的预测也可能出错，完备的预案才是底气。

2. **可观测性是基础**：没有监控，就像蒙着眼睛开车。

3. **容灾设计要落地**：不能只存在于PPT，必须实际演练过。

4. **团队协同是关键**：大促是团队的战役，不是个人英雄主义。
