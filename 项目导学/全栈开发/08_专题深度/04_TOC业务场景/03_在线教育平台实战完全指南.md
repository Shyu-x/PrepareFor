# 在线教育平台实战完全指南

## 一、在线教育概述

### 1.1 教育场景分类

在线教育平台根据教学模式的差异，主要分为三种核心场景，每种场景都有其独特的技术需求和用户体验设计要点。

**直播课场景**是实时性最强的教学模式，教师通过摄像头和麦克风进行实时授课，学生可以同步观看并参与互动。直播课的核心优势在于其强互动性和高参与度，学生能够获得类似线下课堂的学习体验。技术实现上需要支持实时音视频流传输、聊天室互动、屏幕共享等功能。直播课适合职业技能培训、升学辅导、兴趣爱好培养等场景，典型平台包括得到直播、腾讯课堂直播版、网易云课堂直播等。

**录播课场景**采用预先录制的视频内容，学生可以随时随地进行学习。录播课的核心优势在于学习时间和进度的灵活性，学生可以根据自己的节奏安排学习计划。技术实现上重点关注视频播放体验、进度保存、笔记标注等功能。录播课适合系统性知识学习、兴趣爱好培养、企业内部培训等场景，典型平台包括中国大学MOOC、慕课网、极客时间等。

**一对一辅导场景**提供最高程度的个性化教学服务，一位教师对应一位学生进行针对性辅导。这种模式能够最大化教学效果，但成本也最高。一对一场景的技术实现需要支持高清视频通话、屏幕共享、文档协作等功能。典型平台包括VIPKID、51Talk、掌门一对一等。

### 1.2 核心功能模块

在线教育平台的功能模块设计需要围绕教学全流程展开，涵盖课前、课中、课后三个阶段。

**课程管理模块**是平台的基础模块，负责课程的创建、编辑、上下架等操作。课程通常采用章-节-知识点的三级结构组织，每节包含视频、文档、课后作业等内容。教师可以通过后台管理系统发布课程、设置价格、管理学生名单。企业内部培训平台还需要支持课程分配、学习期限设置等功能。

**视频播放模块**直接决定了学生的学习体验。核心功能包括多清晰度切换、播放速度控制、进度保存、弹幕评论等。技术实现上需要考虑视频加密、数字水印、防录屏等版权保护措施。视频播放器通常基于HTML5的video标签或使用video.js、Player等成熟播放器框架。

**互动系统模块**用于增强学习过程中的社交属性。常见功能包括实时问答、弹幕评论、学习笔记、讨论区等。直播课场景下还需要支持送礼、点赞、连麦等互动玩法。互动系统能够显著提升学生参与度和完课率，是在线教育平台的核心竞争力之一。

**作业考试模块**用于检验学习效果和巩固知识。题型通常包括单选题、多选题、判断题、填空题、问答题等。客观题支持系统自动批改，主观题需要教师手动批改。考试系统还需要具备防作弊机制，如切屏检测、禁止复制粘贴、时间限制等。

**学习进度模块**用于追踪学生的学习轨迹。核心指标包括观看时长、完播率、作业正确率、考勤记录等。平台通常会生成可视化的学习报告，帮助学生了解自己的学习状态。同时配合勋章系统、等级系统等激励措施，提升学习动力。

### 1.3 技术挑战分析

在线教育平台面临诸多技术挑战，这些挑战直接影响了用户体验和业务发展。

**视频流畅度**是首要挑战。视频卡顿会严重破坏学习体验，导致学生流失。造成卡顿的原因多种多样，包括用户网络带宽不足、CDN分发不合理、视频编码参数不当、服务器负载过高等。解决方案通常包括：采用HLS或DASH协议实现自适应码率流媒体播放，部署多级CDN加速网络，实施视频预加载和缓冲策略，优化编码参数减少视频体积。

**实时互动延迟**是直播课场景的核心挑战。在线课堂需要将教师讲解、屏幕共享等内容实时传输给学生，同时学生的问题和反馈也需要及时传达给教师。延迟过大会严重影响课堂节奏，降低互动效果。WebRTC是目前实现低延迟实时通信的主流技术，但部署成本较高，需要专业的技术团队维护。

**大规模并发**是教育平台促销活动期间面临的压力。开学季、高考季等活动期间，平台流量可能激增数十倍。如果系统不具备弹性扩展能力，将出现服务不可用的情况。解决方案包括：采用微服务架构分离核心业务，引入消息队列削峰填谷，使用云原生技术实现弹性伸缩。

**内容版权保护**是在线教育的生命线。视频内容是平台的核心资产，一旦泄露将造成重大经济损失。技术保护手段包括DRM数字版权管理、视频加密、数字水印、防下载限制等。同时需要配合法律手段和运营策略进行综合保护。

### 1.4 我的思考：教育产品的使命感

作为一名在线教育产品的开发者，我始终认为教育产品承载着比其他互联网产品更重的社会责任。与电商、社交等产品不同，教育产品的用户是在寻求成长和进步，他们的每一分钟投入都期望有所收获。这要求我们在设计产品时，必须把用户体验和教学效果放在首位，而不是单纯追求流量和变现。

一个好的教育产品应该做到：让学习困难的学生能够跟上进度，让优秀的学生能够走得更快，让每一个努力的人都能看到成果。技术只是手段，教育的本质是点燃火焰、唤醒潜能。在线教育最大的价值在于打破教育资源的不平衡，让偏远地区的孩子也能享受到优质的教育资源。

作为一名开发者，在设计每一个功能时，都应该思考：这个功能是否真正帮助了学生的学习？是否提升了教学效率？是否让教育更加公平？只有怀着这样的使命感，才能打造出真正有价值的教育产品。

## 二、课程系统设计

### 2.1 课程结构设计

课程是教育平台的核心内容载体，其结构设计直接影响到教学效果和用户学习体验。一个设计良好的课程结构应该既符合教学规律，又便于技术实现和维护。

**三级结构模型**是最常见的课程组织方式：课程（Course）- 章节（Chapter）- 小节（Lesson）。以一门《React进阶实战》课程为例，其结构可能如下：

```
课程：React进阶实战
├── 第一章：React核心原理
│   ├── 1.1 虚拟DOM与Diff算法
│   ├── 1.2 Fiber架构详解
│   └── 1.3 调度器工作原理
├── 第二章：状态管理进阶
│   ├── 2.1 Redux中间件机制
│   ├── 2.2 MobX响应式原理
│   └── 2.3 Zustand源码解析
└── 第三章：性能优化实践
    ├── 3.1 React Profiler使用指南
    ├── 3.2 代码分割与懒加载
    └── 3.3 真实案例性能优化
```

**知识点原子化设计**是一种更精细的课程组织方式。每个知识点是独立的最小教学单元，包含视频、文档、练习三个部分。学生可以根据自己的需求选择性学习，而不必从头到尾观看整个课程。这种设计特别适合技能类课程，因为学生的学习目标往往非常明确。

**学习路径设计**关注课程之间的逻辑关系和先后顺序。通过定义前置课程和后置课程，帮助学生构建系统的知识体系。学习路径通常以可视化的方式呈现，让学生对整个学习旅程有清晰的认知。

### 2.2 课程类型与商业模式

根据变现方式的不同，课程可以分为多种类型，每种类型对应不同的商业模式和技术实现。

**免费课程**是平台引流的重要手段。免费课程降低了用户的学习门槛，能够有效提升平台注册用户数和活跃度。免费课程的盈利主要通过以下方式实现：广告变现、转化付费课程、增值服务收费。技术实现上免费课程与付费课程没有本质区别，但在访问控制上需要进行区分。

**付费单课**是最直接的变现方式，用户一次性支付固定金额即可永久观看该课程。付费单课适合单价较高的精品课程，如职业技能培训、升学考试辅导等。技术实现上需要集成支付系统，并确保支付安全性和交易准确性。

**会员订阅**是近年来兴起的商业模式，用户按月或按年支付订阅费，可以在会员有效期内观看平台所有课程或指定分类下的课程。会员模式能够提供稳定的现金流，同时鼓励用户多学习提升用户粘性。技术实现上需要设计会员等级体系、权益管理系统、续费提醒机制等。

**课程组合包**将多门相关课程打包销售，通常给予一定折扣。组合包适合系统性学习场景，如"全栈工程师训练营"、"数据科学入门到精通"等。技术实现上需要支持套装定价、课程关联、订单优惠计算等功能。

### 2.3 课件管理系统

课件是课程的核心教学内容，通常包括视频、文档、PPT等多种格式。课件管理系统负责课件的上传、存储、转码、分发等全生命周期管理。

**视频处理流程**是课件管理的核心环节。当教师上传原始视频后，系统需要依次执行以下处理步骤：

```typescript
// 视频上传与处理流程
// types/video.ts

/**
 * 视频处理任务状态
 */
interface VideoProcessingTask {
  id: string;                          // 任务唯一标识
  videoId: string;                      // 视频ID
  originalUrl: string;                  // 原始视频URL
  status: 'pending' | 'processing' | 'completed' | 'failed';
  progress: number;                     // 处理进度 0-100
  outputs: VideoOutput[];               // 输出格式列表
  error?: string;                       // 错误信息
  createdAt: Date;                      // 创建时间
  completedAt?: Date;                   // 完成时间
}

/**
 * 视频输出格式
 */
interface VideoOutput {
  id: string;                           // 输出格式ID
  quality: '240p' | '360p' | '480p' | '720p' | '1080p' | '4k';
  format: 'mp4' | 'hls';               // 格式：MP4或HLS自适应流
  resolution: string;                   // 分辨率如 1920x1080
  bitrate: number;                      // 码率 kbps
  url?: string;                        // 存储URL
  fileSize?: number;                   // 文件大小字节
}

/**
 * 视频实体
 */
interface Video {
  id: string;
  title: string;                        // 视频标题
  description?: string;                 // 视频描述
  duration: number;                     // 时长（秒）
  coverImage: string;                   // 封面图URL
  teacherId: string;                    // 上传教师ID
  processingTask?: VideoProcessingTask; // 处理任务
  isPublished: boolean;                // 是否发布
  viewCount: number;                   // 播放次数
  createdAt: Date;
  updatedAt: Date;

// 视频上传服务
// services/videoUpload.ts

import { v4 as uuidv4 } from 'uuid';
import { ossClient } from './oss';
import { videoQueue } from './queue';

class VideoUploadService {
  /**
   * 创建视频上传任务
   * @param file 视频文件
   * @param metadata 元数据
   * @returns 上传预签名URL
   */
  async createUploadTask(
    file: File,
    metadata: {
      title: string;
      courseId: string;
      chapterId: string;
      teacherId: string;
    }
  ): Promise<{
    uploadUrl: string;
    videoId: string;
    taskId: string;
  }> {
    // 生成唯一标识
    const videoId = uuidv4();
    const taskId = uuidv4();
    const fileExt = file.name.split('.').pop();
    const objectKey = `videos/${metadata.courseId}/${videoId}.${fileExt}`;

    // 创建视频记录
    const video = await db.videos.create({
      id: videoId,
      title: metadata.title,
      courseId: metadata.courseId,
      chapterId: metadata.chapterId,
      teacherId: metadata.teacherId,
      originalUrl: objectKey,
      status: 'uploading',
    });

    // 获取上传预签名URL
    const uploadUrl = await ossClient.generatePresignedUrl(objectKey, {
      method: 'PUT',
      expiresIn: 3600,                  // 1小时有效期
      contentType: file.type,
    });

    // 创建处理任务记录
    await db.processingTasks.create({
      id: taskId,
      videoId: videoId,
      status: 'pending',
      outputs: [],
    });

    return { uploadUrl, videoId, taskId };
  }

  /**
   * 上传完成回调
   * 触发视频处理流程
   */
  async onUploadComplete(videoId: string): Promise<void> {
    // 更新视频状态为处理中
    await db.videos.update(videoId, { status: 'processing' });

    // 获取视频信息
    const video = await db.videos.findById(videoId);
    const objectKey = video.originalUrl;

    // 创建多清晰度转码任务
    const transcodeTasks = this.generateTranscodeTasks(objectKey, videoId);

    // 将转码任务加入队列
    for (const task of transcodeTasks) {
      await videoQueue.add('transcode', task);
    }

    // 创建HLS切片任务
    await videoQueue.add('hls', {
      videoId,
      inputUrl: objectKey,
      qualities: ['240p', '360p', '480p', '720p', '1080p'],
    });
  }

  /**
   * 生成转码任务列表
   */
  private generateTranscodeTasks(
    inputUrl: string,
    videoId: string
  ): TranscodeTask[] {
    const qualities = [
      { name: '240p', width: 426, height: 240, bitrate: 400 },
      { name: '360p', width: 640, height: 360, bitrate: 800 },
      { name: '480p', width: 854, height: 480, bitrate: 1400 },
      { name: '720p', width: 1280, height: 720, bitrate: 2800 },
      { name: '1080p', width: 1920, height: 1080, bitrate: 5000 },
    ];

    return qualities.map(q => ({
      videoId,
      inputUrl,
      outputQuality: q.name,
      outputPath: `videos/${videoId}/${q.name}/index.mp4`,
      settings: {
        width: q.width,
        height: q.height,
        bitrate: q.bitrate,
        codec: 'h264',
        audioCodec: 'aac',
        audioBitrate: 128,
      },
    }));
  }
}
```

**文档处理流程**相对简单，主要涉及PDF、Word等格式的转换和预览。系统需要支持在线预览，让学生无需下载即可查看文档内容。

### 2.4 实战：课程数据模型设计

以下是一个完整的课程数据模型设计，包含所有核心实体和关联关系：

```typescript
// 数据模型定义
// models/course.ts

import {
  Entity,
  PrimaryGeneratedColumn,
  Column,
  CreateDateColumn,
  UpdateDateColumn,
  ManyToOne,
  OneToMany,
  ManyToMany,
  JoinTable,
  JoinColumn,
} from 'typeorm';

/**
 * 课程分类实体
 * 用于课程分类管理，如：前端开发、后端开发、运维等
 */
@Entity('categories')
export class Category {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column({ length: 100 })
  name: string;                         // 分类名称

  @Column({ length: 500, nullable: true })
  description: string;                  // 分类描述

  @Column({ nullable: true })
  parentId: string;                     // 父分类ID，支持多级分类

  @Column({ default: 0 })
  sortOrder: number;                    // 排序顺序

  @Column({ default: true })
  isActive: boolean;                   // 是否启用

  @OneToMany(() => Category, (category) => category.parent)
  children: Category[];                 // 子分类

  @ManyToOne(() => Category, (category) => category.children)
  @JoinColumn({ name: 'parentId' })
  parent: Category;

  @CreateDateColumn()
  createdAt: Date;

  @UpdateDateColumn()
  updatedAt: Date;
}

/**
 * 课程实体
 * 核心业务实体，代表一门完整的课程
 */
@Entity('courses')
export class Course {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column({ length: 200 })
  title: string;                        // 课程标题

  @Column({ type: 'text', nullable: true })
  description: string;                  // 课程描述

  @Column({ nullable: true })
  coverImage: string;                   // 封面图片URL

  @Column({
    type: 'enum',
    enum: ['free', 'paid', 'member'],
    default: 'free',
  })
  pricingType: 'free' | 'paid' | 'member';

  @Column({ type: 'decimal', precision: 10, scale: 2, default: 0 })
  price: number;                        // 课程价格（分）

  @Column({ nullable: true })
  originalPrice: number;                // 原价（用于显示折扣）

  @Column({
    type: 'enum',
    enum: ['draft', 'published', 'archived'],
    default: 'draft',
  })
  status: 'draft' | 'published' | 'archived';

  @Column({ type: 'int', default: 0 })
  viewCount: number;                    // 浏览次数

  @Column({ type: 'int', default: 0 })
  studentCount: number;                // 学习人数

  @Column({ type: 'int', default: 0 })
  totalDuration: number;               // 总时长（秒）

  @Column({ type: 'int', default: 5 })
  rating: number;                       // 平均评分 1-5

  @Column({ type: 'int', default: 0 })
  reviewCount: number;                  // 评价数量

  @ManyToOne(() => Category)
  @JoinColumn({ name: 'categoryId' })
  category: Category;

  @Column({ nullable: true })
  categoryId: string;

  @ManyToOne(() => User)
  @JoinColumn({ name: 'teacherId' })
  teacher: User;                        // 授课教师

  @Column()
  teacherId: string;

  @OneToMany(() => Chapter, (chapter) => chapter.course)
  chapters: Chapter[];                 // 章节列表

  @ManyToMany(() => User)
  @JoinTable({
    name: 'course_students',
    joinColumn: { name: 'courseId' },
    inverseJoinColumn: { name: 'userId' },
  })
  students: User[];                     // 已报名学生

  @CreateDateColumn()
  createdAt: Date;

  @UpdateDateColumn()
  updatedAt: Date;
}

/**
 * 章节实体
 * 课程下的章，一个课程包含多个章节
 */
@Entity('chapters')
export class Chapter {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column()
  courseId: string;                     // 所属课程ID

  @ManyToOne(() => Course, (course) => course.chapters)
  @JoinColumn({ name: 'courseId' })
  course: Course;

  @Column({ length: 200 })
  title: string;                         // 章节标题

  @Column({ type: 'int', default: 0 })
  sortOrder: number;                    // 排序顺序

  @Column({ type: 'int', default: 0 })
  duration: number;                      // 章节总时长

  @OneToMany(() => Lesson, (lesson) => lesson.chapter)
  lessons: Lesson[];                     // 小节列表

  @CreateDateColumn()
  createdAt: Date;

  @UpdateDateColumn()
  updatedAt: Date;
}

/**
 * 小节实体
 * 章节下的小节，是视频和文档的容器
 */
@Entity('lessons')
export class Lesson {
  @PrimaryGeneratedColumn('uuid')
  id: string;

  @Column()
  chapterId: string;                    // 所属章节ID

  @ManyToOne(() => Chapter, (chapter) => chapter.lessons)
  @JoinColumn({ name: 'chapterId' })
  chapter: Chapter;

  @Column({ length: 200 })
  title: string;                         // 小节标题

  @Column({ type: 'int', default: 0 })
  sortOrder: number;                     // 排序顺序

  @Column({ type: 'int', default: 0 })
  duration: number;                      // 视频时长（秒）

  @Column({
    type: 'enum',
    enum: ['video', 'document', 'quiz'],
    default: 'video',
  })
```

## 三、视频播放系统

### 3.1 播放器核心开发

视频播放器是在线教育平台最核心的前端组件，其性能直接影响用户体验。一个优秀的教育视频播放器需要具备以下核心功能：多格式支持、播放控制、清晰度切换、进度保存、键盘快捷键等。

**基础播放器实现**基于HTML5 video标签，结合React的状态管理能力：

```typescript
// 视频播放器核心组件
// components/VideoPlayer/VideoPlayer.tsx

"use client";

import React, {
  useRef,
  useState,
  useEffect,
  useCallback,
} from 'react';
import {
  Play,
  Pause,
  Volume2,
  VolumeX,
  Maximize,
  Minimize,
  Settings,
  PictureInPicture,
} from 'lucide-react';
import { Slider } from '@/components/ui/slider';
import { useProgressStore } from '@/store/progressStore';
import type { Video } from '@/types/video';

/**
 * 播放器组件属性
 */
interface VideoPlayerProps {
  video: Video;                         // 视频信息
  onProgress?: (progress: number) => void; // 播放进度回调
  onComplete?: () => void;              // 播放完成回调
  autoPlay?: boolean;                   // 自动播放
  startTime?: number;                   // 起始播放时间（用于断点续播）
}

/**
 * 播放器状态
 */
interface PlayerState {
  isPlaying: boolean;                    // 是否正在播放
  currentTime: number;                   // 当前播放时间（秒）
  duration: number;                      // 视频总时长
  volume: number;                        // 音量 0-1
  isMuted: boolean;                       // 是否静音
  isFullscreen: boolean;                 // 是否全屏
  isLoading: boolean;                   // 是否正在加载
  buffered: number;                      // 缓冲进度 0-1
  playbackRate: number;                  // 播放速度
  quality: string;                       // 当前清晰度
  showControls: boolean;                // 是否显示控制栏
  error: string | null;                   // 错误信息
}

/**
 * 视频播放器核心组件
 * 提供完整的播放控制、进度管理、清晰度切换等功能
 */
export const VideoPlayer: React.FC<VideoPlayerProps> = ({
  video,
  onProgress,
  onComplete,
  autoPlay = false,
  startTime = 0,
}) => {
  // Refs
  const containerRef = useRef<HTMLDivElement>(null);
  const videoRef = useRef<HTMLVideoElement>(null);
  const controlsTimeoutRef = useRef<NodeJS.Timeout>();

  // 状态管理
  const [state, setState] = useState<PlayerState>({
    isPlaying: false,
    currentTime: startTime,
    duration: 0,
    volume: 1,
    isMuted: false,
    isFullscreen: false,
    isLoading: true,
    buffered: 0,
    playbackRate: 1,
    quality: '720p',
    showControls: true,
    error: null,
  });

  // 进度存储
  const { saveProgress, getProgress } = useProgressStore();

  /**
   * 格式化时间显示
   * @param seconds 秒数
   * @returns MM:SS 或 HH:MM:SS 格式
   */
  const formatTime = useCallback((seconds: number): string => {
    if (isNaN(seconds) || !isFinite(seconds)) return '00:00';

    const hrs = Math.floor(seconds / 3600);
    const mins = Math.floor((seconds % 3600) / 60);
    const secs = Math.floor(seconds % 60);

    if (hrs > 0) {
      return `${hrs.toString().padStart(2, '0')}:${mins
        .toString()
        .padStart(2, '0')}:${secs.toString().padStart(2, '0')}`;
    }
    return `${mins.toString().padStart(2, '0')}:${secs
      .toString()
      .padStart(2, '0')}`;
  }, []);

  /**
   * 播放/暂停切换
   */
  const togglePlay = useCallback(() => {
    if (!videoRef.current) return;

    if (state.isPlaying) {
      videoRef.current.pause();
    } else {
      videoRef.current.play();
    }
  }, [state.isPlaying]);

  /**
   * 跳转到指定时间
   */
  const seek = useCallback((time: number) => {
    if (!videoRef.current) return;
    videoRef.current.currentTime = Math.max(
      0,
      Math.min(time, state.duration)
    );
  }, [state.duration]);

  /**
   * 音量控制
   */
  const setVolume = useCallback((volume: number) => {
    if (!videoRef.current) return;
    const clampedVolume = Math.max(0, Math.min(1, volume));
    videoRef.current.volume = clampedVolume;
    setState((prev) => ({ ...prev, volume: clampedVolume, isMuted: clampedVolume === 0 }));
  }, []);

  /**
   * 静音切换
   */
  const toggleMute = useCallback(() => {
    if (!videoRef.current) return;
    const newMuted = !state.isMuted;
    videoRef.current.muted = newMuted;
    setState((prev) => ({ ...prev, isMuted: newMuted }));
  }, [state.isMuted]);

  /**
   * 播放速度控制
   */
  const setPlaybackRate = useCallback((rate: number) => {
    if (!videoRef.current) return;
    videoRef.current.playbackRate = rate;
    setState((prev) => ({ ...prev, playbackRate: rate }));
  }, []);

  /**
   * 全屏切换
   */
  const toggleFullscreen = useCallback(async () => {
    if (!containerRef.current) return;

    try {
      if (!document.fullscreenElement) {
        await containerRef.current.requestFullscreen();
        setState((prev) => ({ ...prev, isFullscreen: true }));
      } else {
        await document.exitFullscreen();
        setState((prev) => ({ ...prev, isFullscreen: false }));
      }
    } catch (error) {
      console.error('全屏切换失败:', error);
    }
  }, []);

  /**
   * 画中画模式
   */
  const togglePictureInPicture = useCallback(async () => {
    if (!videoRef.current) return;

    try {
      if (document.pictureInPictureElement) {
        await document.exitPictureInPicture();
      } else {
        await videoRef.current.requestPictureInPicture();
      }
    } catch (error) {
      console.error('画中画切换失败:', error);
    }
  }, []);

  // 视频事件处理
  useEffect(() => {
    const video = videoRef.current;
    if (!video) return;

    const handlePlay = () => setState((prev) => ({ ...prev, isPlaying: true }));
    const handlePause = () => setState((prev) => ({ ...prev, isPlaying: false }));
    const handleTimeUpdate = () => {
      const currentTime = video.currentTime;
      setState((prev) => ({ ...prev, currentTime }));

      // 保存播放进度（每30秒保存一次）
      if (Math.floor(currentTime) % 30 === 0) {
        saveProgress(video.id, currentTime);
      }

      onProgress?.(currentTime);
    };

    const handleDurationChange = () => {
      setState((prev) => ({ ...prev, duration: video.duration }));
    };

    const handleLoadStart = () => setState((prev) => ({ ...prev, isLoading: true }));
    const handleCanPlay = () => setState((prev) => ({ ...prev, isLoading: false }));
    const handleWaiting = () => setState((prev) => ({ ...prev, isLoading: true }));
    const handlePlaying = () => setState((prev) => ({ ...prev, isLoading: false }));

    const handleProgress = () => {
      if (video.buffered.length > 0) {
        const bufferedEnd = video.buffered.end(video.buffered.length - 1);
        const buffered = bufferedEnd / video.duration;
        setState((prev) => ({ ...prev, buffered }));
      }
    };

    const handleEnded = () => {
      setState((prev) => ({ ...prev, isPlaying: false }));
      saveProgress(video.id, 0); // 重置进度
      onComplete?.();
    };

    const handleError = () => {
      setState((prev) => ({
        ...prev,
        error: '视频加载失败，请刷新重试',
        isLoading: false,
      }));
    };

    // 绑定事件
    video.addEventListener('play', handlePlay);
    video.addEventListener('pause', handlePause);
    video.addEventListener('timeupdate', handleTimeUpdate);
    video.addEventListener('durationchange', handleDurationChange);
    video.addEventListener('loadstart', handleLoadStart);
    video.addEventListener('canplay', handleCanPlay);
    video.addEventListener('waiting', handleWaiting);
    video.addEventListener('playing', handlePlaying);
    video.addEventListener('progress', handleProgress);
    video.addEventListener('ended', handleEnded);
    video.addEventListener('error', handleError);

    return () => {
      video.removeEventListener('play', handlePlay);
      video.removeEventListener('pause', handlePause);
      video.removeEventListener('timeupdate', handleTimeUpdate);
      video.removeEventListener('durationchange', handleDurationChange);
      video.removeEventListener('loadstart', handleLoadStart);
      video.removeEventListener('canplay', handleCanPlay);
      video.removeEventListener('waiting', handleWaiting);
      video.removeEventListener('playing', handlePlaying);
      video.removeEventListener('progress', handleProgress);
      video.removeEventListener('ended', handleEnded);
      video.removeEventListener('error', handleError);
    };
  }, [video.id, onProgress, onComplete, saveProgress]);

  // 键盘快捷键
  useEffect(() => {
    const handleKeyDown = (e: KeyboardEvent) => {
      // 防止在输入框中触发
      if (e.target instanceof HTMLInputElement || e.target instanceof HTMLTextAreaElement) {
        return;
      }

      switch (e.key) {
        case ' ':
        case 'k':
          e.preventDefault();
          togglePlay();
          break;
        case 'ArrowLeft':
          e.preventDefault();
          seek(state.currentTime - 10);
          break;
        case 'ArrowRight':
          e.preventDefault();
          seek(state.currentTime + 10);
          break;
        case 'ArrowUp':
          e.preventDefault();
          setVolume(state.volume + 0.1);
          break;
        case 'ArrowDown':
          e.preventDefault();
          setVolume(state.volume - 0.1);
          break;
        case 'm':
          e.preventDefault();
          toggleMute();
          break;
        case 'f':
          e.preventDefault();
          toggleFullscreen();
          break;
        case 'j':
          e.preventDefault();
          setPlaybackRate(Math.max(0.5, state.playbackRate - 0.25));
          break;
        case 'l':
          e.preventDefault();
          setPlaybackRate(Math.min(2, state.playbackRate + 0.25));
          break;
      }
    };

    window.addEventListener('keydown', handleKeyDown);
    return () => window.removeEventListener('keydown', handleKeyDown);
  }, [togglePlay, seek, setVolume, toggleMute, toggleFullscreen, state, setPlaybackRate]);

  // 控制栏自动隐藏
  const hideControls = useCallback(() => {
    if (state.isPlaying) {
      setState((prev) => ({ ...prev, showControls: false }));
    }
  }, [state.isPlaying]);

  const showControls = useCallback(() => {
    setState((prev) => ({ ...prev, showControls: true }));
    if (controlsTimeoutRef.current) {
      clearTimeout(controlsTimeoutRef.current);
    }
    controlsTimeoutRef.current = setTimeout(hideControls, 3000);
  }, [hideControls]);

  // 计算进度百分比
  const progressPercent = state.duration > 0
    ? (state.currentTime / state.duration) * 100
    : 0;

  return (
    <div
      ref={containerRef}
      className="video-player relative w-full h-full bg-black"
      onMouseMove={showControls}
      onMouseLeave={hideControls}
    >
      {/* 视频元素 */}
      <video
        ref={videoRef}
        src={video.url}
        poster={video.coverImage}
        autoPlay={autoPlay}
        onClick={togglePlay}
        className="w-full h-full"
      />

      {/* 加载指示器 */}
      {state.isLoading && (
        <div className="absolute inset-0 flex items-center justify-center bg-black/50">
          <div className="w-12 h-12 border-4 border-white/30 border-t-white rounded-full animate-spin" />
        </div>
      )}

      {/* 错误提示 */}
      {state.error && (
        <div className="absolute inset-0 flex items-center justify-center bg-black/80">
          <div className="text-center text-white">
            <p className="text-lg mb-2">{state.error}</p>
            <button
              onClick={() => window.location.reload()}
              className="px-4 py-2 bg-white/20 rounded hover:bg-white/30"
            >
              刷新页面
            </button>
          </div>
        </div>
      )}

      {/* 控制栏 */}
      <div
        className={`absolute bottom-0 left-0 right-0 bg-gradient-to-t from-black/80 to-transparent p-4 transition-opacity duration-300 ${
          state.showControls ? 'opacity-100' : 'opacity-0 pointer-events-none'
        }`}
      >
        {/* 进度条 */}
        <div className="relative h-1 bg-white/30 rounded-full mb-4 cursor-pointer group">
          {/* 缓冲进度 */}
          <div
            className="absolute h-full bg-white/30 rounded-full"
            style={{ width: `${state.buffered * 100}%` }}
          />
          {/* 播放进度 */}
          <div
            className="absolute h-full bg-blue-500 rounded-full"
            style={{ width: `${progressPercent}%` }}
          />
          {/* 拖动点 */}
          <div
            className="absolute top-1/2 -translate-y-1/2 w-3 h-3 bg-white rounded-full opacity-0 group-hover:opacity-100 transition-opacity"
            style={{ left: `calc(${progressPercent}% - 6px)` }}
          />
        </div>

        {/* 控制按钮组 */}
        <div className="flex items-center justify-between">
          {/* 左侧控制 */}
          <div className="flex items-center gap-4">
            {/* 播放/暂停 */}
            <button
              onClick={togglePlay}
              className="text-white hover:text-blue-400 transition-colors"
            >
              {state.isPlaying ? (
                <Pause className="w-6 h-6" />
              ) : (
                <Play className="w-6 h-6" />
              )}
            </button>

            {/* 音量控制 */}
```

### 3.2 多清晰度切换实现

多清晰度切换是视频播放的基本需求，不同网络环境的用户需要不同码率的视频。以下是基于HLS协议的自适应清晰度实现：

```typescript
// HLS多清晰度播放器
// components/VideoPlayer/HLSPlayer.tsx

"use client";

import React, { useRef, useEffect, useState } from 'react';
import Hls, { Level } from 'hls.js';

/**
 * HLS播放器属性
 */
interface HLSPlayerProps {
  src: string;                           // HLS播放地址
  poster?: string;                        // 封面图
  onLevelChange?: (level: Level) => void; // 清晰度切换回调
  autoLevelEnabled?: boolean;            // 是否启用自动清晰度
}

/**
 * 清晰度选项
 */
interface QualityOption {
  height: number;                         // 高度
  label: string;                          // 显示标签
  bitrate: number;                        // 码率
  url: string;                            // 播放地址
}

export const HLSPlayer: React.FC<HLSPlayerProps> = ({
  src,
  poster,
  onLevelChange,
  autoLevelEnabled = true,
}) => {
  const videoRef = useRef<HTMLVideoElement>(null);
  const hlsRef = useRef<Hls | null>(null);

  const [levels, setLevels] = useState<QualityOption[]>([]);
  const [currentLevel, setCurrentLevel] = useState<number>(-1); // -1 表示自动
  const [isAuto, setIsAuto] = useState(true);

  /**
   * 初始化HLS
   */
  useEffect(() => {
    const video = videoRef.current;
    if (!video || !src) return;

    // 检查浏览器是否支持HLS
    if (Hls.isSupported()) {
      const hls = new Hls({
        enableWorker: true,               // 启用Web Worker
        lowLatencyMode: false,            // 关闭低延迟模式（教育场景以稳定性为主）
        backBufferLength: 90,              // 缓冲90秒
        maxBufferLength: 30,               // 最大缓冲30秒
        maxMaxBufferLength: 600,          // 最大缓冲10分钟
        maxBufferSize: 60 * 1000 * 1000,   // 最大缓冲60MB
        maxBufferHole: 0.5,                // 缓冲空洞容忍度
        autoLevelEnabled: autoLevelEnabled, // 自动清晰度
      });

      hls.loadSource(src);
      hls.attachMedia(video);

      // HLS初始化完成
      hls.on(Hls.Events.MANIFEST_PARSED, (_, data) => {
        console.log('HLS清单解析完成:', data);

        // 提取清晰度选项
        const qualityLevels: QualityOption[] = data.levels.map((level, index) => ({
          height: level.height,
          label: `${level.height}p`,
          bitrate: level.bitrate,
          url: level.url,
        }));

        // 按高度排序（从低到高）
        qualityLevels.sort((a, b) => a.height - b.height);
        setLevels(qualityLevels);

        // 如果是自动模式，选择中等清晰度作为初始值
        if (autoLevelEnabled) {
          const midLevel = Math.floor(qualityLevels.length / 2);
          hls.startLevel = midLevel;
        }
      });

      // 清晰度切换完成
      hls.on(Hls.Events.LEVEL_SWITCHED, (_, data) => {
        const level = data.level;
        const levelInfo = hls.levels[level];
        setCurrentLevel(level);

        if (levelInfo) {
          onLevelChange?.({
            height: levelInfo.height,
            bitrate: levelInfo.bitrate,
          } as Level);
        }
      });

      // 错误处理
      hls.on(Hls.Events.ERROR, (_, data) => {
        if (data.fatal) {
          switch (data.type) {
            case Hls.ErrorTypes.NETWORK_ERROR:
              console.error('网络错误，尝试恢复...');
              hls.startLoad();
              break;
            case Hls.ErrorTypes.MEDIA_ERROR:
              console.error('媒体错误，尝试恢复...');
              hls.recoverMediaError();
              break;
            default:
              console.error('致命错误，销毁HLS');
              hls.destroy();
              break;
          }
        }
      });

      hlsRef.current = hls;

      return () => {
        hls.destroy();
      };
    } else if (video.canPlayType('application/vnd.apple.mpegurl')) {
      // Safari原生支持HLS
      video.src = src;
    }
  }, [src, autoLevelEnabled, onLevelChange]);

  /**
   * 切换清晰度
   */
  const switchLevel = (index: number) => {
    const hls = hlsRef.current;
    if (!hls) return;

    if (index === -1) {
      // 自动模式
      hls.currentLevel = -1;
      setIsAuto(true);
    } else {
      // 手动选择清晰度
      hls.currentLevel = index;
      setIsAuto(false);
    }
  };

  return (
    <div className="relative">
```

断点续播是在线教育平台的基础功能，用户中断学习后能够从上次位置继续。以下是完整的进度保存实现：

```typescript

// store/progressStore.ts

 * 单个视频的播放进度

/**
 * 播放进度状态
 */
interface ProgressState {
  // 进度数据 Map<videoId, VideoProgress>
  progressMap: Record<string, VideoProgress>;

  // 正在观看的视频
  currentVideoId: string | null;
  currentTime: number;

  // 加载状态
  isLoading: boolean;
}

/**
 * 播放进度Store
 */
export const useProgressStore = create<ProgressState>()(
  persist(
    (set, get) => ({
      progressMap: {},
      currentVideoId: null,
      currentTime: 0,
      isLoading: false,

      /**
       * 保存播放进度
       * @param videoId 视频ID
       * @param currentTime 当前播放时间
       */
      saveProgress: (videoId: string, currentTime: number) => {
        set((state) => {
          const video = state.progressMap[videoId];
          const completed = video
            ? currentTime / video.duration > 0.9
            : false;

          return {
            progressMap: {
              ...state.progressMap,
              [videoId]: {
                videoId,
                currentTime,
                duration: video?.duration || 0,
                percent: video?.duration
                  ? (currentTime / video.duration) * 100
                  : 0,
                lastWatchTime: Date.now(),
                completed,
              },
            },
          };
        });

        // 同步到服务器（防抖处理）
        debouncedSyncToServer(videoId, currentTime);
      },

      /**
       * 更新视频总时长
       */
      updateDuration: (videoId: string, duration: number) => {
        set((state) => {
          const existing = state.progressMap[videoId];

          return {
            progressMap: {
              ...state.progressMap,
              [videoId]: {
                videoId,
                currentTime: existing?.currentTime || 0,
                duration,
                percent: existing?.duration
                  ? ((existing.currentTime / duration) * 100)
                  : 0,
                lastWatchTime: existing?.lastWatchTime || Date.now(),
                completed: existing?.currentTime
                  ? existing.currentTime / duration > 0.9
                  : false,
              },
            },
          };
        });
      },

      /**
       * 获取播放进度
       */
      getProgress: (videoId: string): VideoProgress | null => {
        return get().progressMap[videoId] || null;
      },

      /**
       * 获取起始播放时间
       * 用于断点续播
       */
      getResumeTime: (videoId: string): number => {
        const progress = get().progressMap[videoId];
        if (!progress) return 0;

        // 如果已完成，从头开始
        if (progress.completed) return 0;

        // 如果进度超过90%，从头开始（可能是重看）
        if (progress.percent > 90) return 0;

        // 否则从上次位置继续
        return Math.max(0, progress.currentTime - 5); // 退后5秒帮助回忆
      },

      /**
       * 标记视频为已完成
       */
      markCompleted: (videoId: string) => {
        set((state) => {
          const video = state.progressMap[videoId];
          if (!video) return state;

          return {
            progressMap: {
              ...state.progressMap,
              [videoId]: {
                ...video,
                percent: 100,
                completed: true,
                lastWatchTime: Date.now(),
              },
            },
          };
        });
      },

      /**
       * 获取课程学习进度
       */
      getCourseProgress: (courseId: string, lessonIds: string[]): number => {
        const { progressMap } = get();
        const completedLessons = lessonIds.filter(
          (id) => progressMap[id]?.completed
        );
        return lessonIds.length > 0
          ? (completedLessons.length / lessonIds.length) * 100
          : 0;
      },

      /**
       * 清空所有进度
       */
      clearAllProgress: () => {
        set({ progressMap: {}, currentVideoId: null, currentTime: 0 });
      },
    }),
    {
      name: 'edu-progress-storage',
      storage: createJSONStorage(() => localStorage),
      // 只同步这些字段
      partialize: (state) => ({
        progressMap: state.progressMap,
      }),
    }
  )
);

    }

        videoId,

        currentTime,
        duration: video.duration,

      },
```

```typescript
export class WatermarkOverlay {
  private canvas: HTMLCanvasElement;

    // 创建Canvas

      left: 0;
      width: 100%;
      height: 100%;
      pointer-events: none;
      z-index: 10;
    `;
    container.appendChild(this.canvas);

    this.ctx = this.canvas.getContext('2d')!;

    // 设置Canvas尺寸

    window.addEventListener('resize', () => this.resize());

    // 开始绘制
    this.startRender();
  }

  /**
   * 调整Canvas尺寸
   */
  private resize(): void {
    const rect = this.canvas.parentElement!.getBoundingClientRect();
    this.canvas.width = rect.width * window.devicePixelRatio;
    this.canvas.height = rect.height * window.devicePixelRatio;
    this.ctx.scale(window.devicePixelRatio, window.devicePixelRatio);
  }

  /**
   * 计算水印位置
   */
  private getPosition(
    index: number,
    total: number
  ): { x: number; y: number } {
    const width = this.canvas.width / window.devicePixelRatio;
    const height = this.canvas.height / window.devicePixelRatio;

    // 根据位置配置计算
    switch (this.config.position) {
      case 'top-left':
        return { x: 20, y: 20 + index * 40 };
      case 'top-right':
        return { x: width - 200, y: 20 + index * 40 };
      case 'bottom-left':
        return { x: 20, y: height - 30 - index * 40 };
      case 'bottom-right':
        return { x: width - 200, y: height - 30 - index * 40 };
      case 'center':
      default:
        return { x: width / 2 - 100, y: height / 2 - 50 + index * 40 };
    }
  }

  /**
   * 绘制单帧水印
   */
  private drawFrame(): void {
    const width = this.canvas.width / window.devicePixelRatio;
    const height = this.canvas.height / window.devicePixelRatio;

    // 清空画布
    this.ctx.clearRect(0, 0, width, height);

    // 水印内容
    const watermark = generateDynamicWatermark(this.context, this.config);

    // 设置样式
    this.ctx.globalAlpha = this.config.opacity;
    this.ctx.font = `${this.config.fontSize || 14}px Arial`;
    this.ctx.fillStyle = this.config.color || 'white';
    this.ctx.rotate((this.config.rotate || 0) * Math.PI / 180);

    // 绘制多个水印（矩阵排列）
    const cols = Math.ceil(width / 200);
    const rows = Math.ceil(height / 80);

    for (let row = 0; row < rows; row++) {
      for (let col = 0; col < cols; col++) {
        const x = col * 200 + 20;
        const y = row * 80 + 30;

        // 添加轻微的随机偏移，防止规律性太强被移除
        const offsetX = (Math.random() - 0.5) * 10;
        const offsetY = (Math.random() - 0.5) * 5;

        this.ctx.fillText(watermark, x + offsetX, y + offsetY);
      }
    }

    // 重置变换
    this.ctx.setTransform(1, 0, 0, 1, 0, 0);
  }

  /**
   * 开始渲染
   */
  private startRender(): void {
    const render = () => {
      this.drawFrame();
      this.animationFrame = requestAnimationFrame(render);
    };
    render();
  }

  /**
   * 销毁
   */
  public destroy(): void {
    if (this.animationFrame) {
      cancelAnimationFrame(this.animationFrame);
    }
    this.canvas.remove();
  }
}
```

### 3.5 我的思考：视频体验的重要性

视频体验是在线教育平台的生命线。一个播放卡顿、画质差的视频会严重打击学生的学习积极性。根据统计数据，视频加载时间每增加1秒，就有约7%的用户选择离开。

我认为好的视频体验包含以下几个层次：

**第一层是流畅性**。学生能够顺利观看视频，不出现卡顿、加载失败等问题。这需要做好CDN加速、自适应码率、预加载等工作。

**第二层是便捷性**。学生能够方便地控制播放进度、调整播放速度、切换清晰度。断点续播功能让学生无需担心中断后要从头开始。

**第三层是互动性**。视频不再是单向输出，而是能够与学生互动的学习工具。弹幕评论、划线笔记、学习打卡等功能让学习变得更加有趣。

**第四层是安全性**。视频内容得到有效保护，学生的隐私（如水印信息）得到尊重。防录屏、防下载等机制让版权得到保护。

## 四、直播系统架构

### 4.1 直播流媒体协议

直播是在线教育的核心场景之一，实时性和稳定性是直播系统的核心指标。目前主流的直播协议有RTMP、HLS、FLV和WebRTC四种。

**RTMP（Real-Time Messaging Protocol）**是Adobe开发的实时消息传输协议，长期以来是直播行业的事实标准。RTMP基于TCP协议，优点是延迟较低（通常3-5秒）、稳定性好，缺点是需要专用推流服务器、与现代Web技术兼容性差。随着Flash的淘汰，RTMP正在逐渐被其他协议取代。

**HLS（HTTP Live Streaming）**是Apple开发的基于HTTP的自适应流媒体协议。HLS将视频切分成小TS片段，通过M3U8播放列表管理。优点是兼容性极好（原生支持于iOS/Safari）、基于HTTP便于CDN分发，缺点是延迟较高（通常10-30秒）。HLS适合对延迟要求不高的场景，如游戏直播、赛事直播等。

**FLV（Flash Video）**是基于RTMP协议的封装格式，通过HTTP-FLV可以在Web端实现低延迟直播。优点是延迟可以做到3秒以内、实现简单，缺点是依赖Flash（已被淘汰）。目前常用的方案是通过flv.js实现HTTP-FLV播放。

**WebRTC（Web Real-Time Communication）**是W3C定义的浏览器实时通信标准。WebRTC支持端到端的音视频传输，延迟可以做到1秒以内，适合需要强互动的场景如在线课堂、视频会议等。缺点是技术复杂度高、服务器资源消耗大。

**协议对比表：**

| 特性 | RTMP | HLS | HTTP-FLV | WebRTC |
|------|------|-----|----------|--------|
| 延迟 | 3-5秒 | 10-30秒 | 3秒以内 | 1秒以内 |
| 兼容性 | 差 | 优秀 | 一般 | 一般（需ICE） |
| 复杂度 | 低 | 低 | 中 | 高 |
| 服务器成本 | 中 | 低 | 中 | 高 |
| 互动支持 | 一般 | 差 | 一般 | 优秀 |
| 适用场景 | 秀场直播 | 点播/赛事 | 电商直播 | 在线课堂 |

### 4.3 直播聊天室设计

聊天室是直播间的重要功能，支持弹幕、礼物、点赞等互动形式。以下是完整的聊天室实现：

```typescript
// 直播间聊天室
// components/LiveRoom/ChatRoom.tsx

"use client";

import React, {
  useState,
  useEffect,
  useRef,
  useCallback,
} from 'react';
import { Gift, ThumbsUp, Send, Emoji } from 'lucide-react';
import { useLiveStore } from '@/store/liveStore';
import { useUserStore } from '@/store/userStore';
import { chatWebSocketService } from '@/lib/chat-websocket';
import type { ChatMessage, GiftMessage, SystemMessage } from '@/types/live';

/**
 * 消息类型
 */
type MessageType = ChatMessage | GiftMessage | SystemMessage;

/**
 * 聊天室组件
 */
export const ChatRoom: React.FC<{
  roomId: string;
  liveId: string;
}> = ({ roomId, liveId }) => {
  // 状态
  const [messages, setMessages] = useState<MessageType[]>([]);
  const [inputValue, setInputValue] = useState('');
  const [isConnected, setIsConnected] = useState(false);
  const [unreadCount, setUnreadCount] = useState(0);

  // Refs
  const messagesEndRef = useRef<HTMLDivElement>(null);
  const containerRef = useRef<HTMLDivElement>(null);
  const inputRef = useRef<HTMLInputElement>(null);

  // Store
  const { user } = useUserStore();
  const { addGift, incrementLikes } = useLiveStore();

  /**
   * 连接聊天室
   */
  useEffect(() => {
    // 连接WebSocket
    chatWebSocketService.connect(roomId, {
      onOpen: () => setIsConnected(true),
      onClose: () => setIsConnected(false),
      onMessage: handleNewMessage,
      onError: (error) => console.error('聊天室连接错误:', error),
    });

    // 加入房间
    chatWebSocketService.joinRoom(roomId, {
      userId: user?.id || '',
      userName: user?.name || '匿名用户',
      avatar: user?.avatar || '',
    });

    return () => {
      chatWebSocketService.leaveRoom(roomId);
      chatWebSocketService.disconnect();
    };
  }, [roomId, user]);

  /**
   * 处理新消息
   */
  const handleNewMessage = useCallback((message: MessageType) => {
    setMessages((prev) => [...prev.slice(-100), message]); // 保留最近100条

    // 如果滚动区域不可见，增加未读计数
    if (containerRef.current) {
      const { scrollTop, clientHeight, scrollHeight } = containerRef.current;
      const isNearBottom = scrollHeight - scrollTop - clientHeight < 100;

      if (!isNearBottom) {
        setUnreadCount((prev) => prev + 1);
      }
    }
  }, []);

  /**
   * 滚动到底部
   */
  const scrollToBottom = useCallback(() => {
    messagesEndRef.current?.scrollIntoView({ behavior: 'smooth' });
    setUnreadCount(0);
  }, []);

  /**
   * 发送聊天消息
   */
  const sendMessage = useCallback(() => {
    const text = inputValue.trim();
    if (!text || !user) return;

    const message: ChatMessage = {
      id: generateId(),
      type: 'chat',
      roomId,
      userId: user.id,
      userName: user.name,
      userAvatar: user.avatar,
      content: text,
      timestamp: Date.now(),
    };

    chatWebSocketService.sendMessage(message);
    setInputValue('');

    // 清空后聚焦输入框
    inputRef.current?.focus();
  }, [inputValue, user, roomId]);

  /**
   * 发送礼物
   */
  const sendGift = useCallback((gift: Gift) => {
    if (!user) return;

    const message: GiftMessage = {
      id: generateId(),
      type: 'gift',
      roomId,
      userId: user.id,
      userName: user.name,
      userAvatar: user.avatar,
      gift,
      timestamp: Date.now(),
    };

    chatWebSocketService.sendMessage(message);

    // 更新本地礼物计数
    addGift(gift.id);
  }, [user, roomId, addGift]);

  /**
   * 点赞
   */
  const sendLike = useCallback(() => {
    chatWebSocketService.sendMessage({
      type: 'like',
      roomId,
      count: 1,
    });

    incrementLikes();
  }, [roomId, incrementLikes]);

  /**
   * 格式化时间戳
   */
  const formatTime = (timestamp: number): string => {
    const date = new Date(timestamp);
    return date.toLocaleTimeString('zh-CN', {
      hour: '2-digit',
      minute: '2-digit',
    });
  };

  /**
   * 渲染消息
   */
  const renderMessage = (message: MessageType) => {
    switch (message.type) {
      case 'chat':
        return (
          <div key={message.id} className="flex gap-3 py-2 hover:bg-gray-50">
            {/* 用户头像 */}
            <img
              src={message.userAvatar || '/default-avatar.png'}
              alt={message.userName}
              className="w-8 h-8 rounded-full flex-shrink-0"
            />
            {/* 消息内容 */}
            <div className="flex-1 min-w-0">
              <div className="flex items-baseline gap-2">
                <span className="font-medium text-blue-600 text-sm">
                  {message.userName}
                </span>
                <span className="text-xs text-gray-400">
                  {formatTime(message.timestamp)}
                </span>
              </div>
              <p className="text-gray-700 text-sm break-all">
                {message.content}
              </p>
            </div>
          </div>
        );

      case 'gift':
        return (
          <div key={message.id} className="flex items-center gap-2 py-2 px-3 bg-gradient-to-r from-yellow-50 to-orange-50 rounded-lg border border-yellow-200">
            <img
              src={message.userAvatar || '/default-avatar.png'}
              alt={message.userName}
              className="w-6 h-6 rounded-full"
            />
            <span className="text-sm">
              <span className="font-medium text-orange-600">
                {message.userName}
              </span>
              <span className="text-gray-500 mx-1">送了</span>
              <span className="font-bold text-red-500">
                {message.gift.name}
              </span>
            </span>
            <img
              src={message.gift.icon}
              alt={message.gift.name}
              className="w-6 h-6"
            />
          </div>
        );

      case 'system':
        return (
          <div
            key={message.id}
            className="text-center text-xs text-gray-400 py-2"
          >
            {message.content}
          </div>
        );

      case 'like':
        return null; // 点赞不显示在消息列表

      default:
        return null;
    }
  };

  return (
    <div className="flex flex-col h-full bg-white rounded-lg shadow">
      {/* 消息区域 */}
      <div
        ref={containerRef}
        className="flex-1 overflow-y-auto p-4"
        onScroll={() => {
          if (containerRef.current) {
            const { scrollTop, clientHeight, scrollHeight } =
              containerRef.current;
            const isNearBottom =
              scrollHeight - scrollTop - clientHeight < 100;
            if (isNearBottom) {
              setUnreadCount(0);
            }
          }
        }}
      >
        {messages.map(renderMessage)}
        <div ref={messagesEndRef} />
      </div>

      {/* 未读提示 */}
      {unreadCount > 0 && (
        <button
          onClick={scrollToBottom}
          className="absolute bottom-20 left-1/2 -translate-x-1/2 px-4 py-2 bg-blue-500 text-white rounded-full text-sm shadow-lg animate-bounce"
        >
          有{unreadCount}条新消息
        </button>
      )}

      {/* 互动按钮 */}
      <div className="flex items-center gap-2 p-4 border-t">
        <button
          onClick={sendLike}
          className="flex items-center gap-1 px-3 py-1.5 text-pink-500 hover:bg-pink-50 rounded-full transition-colors"
        >
          <ThumbsUp className="w-4 h-4" />
          <span className="text-sm">点赞</span>
        </button>

        <button
          onClick={() => {/* 打开礼物选择器 */}}
          className="flex items-center gap-1 px-3 py-1.5 text-orange-500 hover:bg-orange-50 rounded-full transition-colors"
        >
          <Gift className="w-4 h-4" />
          <span className="text-sm">礼物</span>
        </button>

        <Emoji
          className="w-5 h-5 text-gray-400 cursor-pointer hover:text-gray-600"
          onClick={() => {/* 打开表情选择器 */}}
        />
      </div>

      {/* 输入区域 */}
      <div className="p-4 border-t">
        <div className="flex gap-2">
          <input
            ref={inputRef}
            type="text"
            value={inputValue}
            onChange={(e) => setInputValue(e.target.value)}
            onKeyDown={(e) => e.key === 'Enter' && sendMessage()}
            placeholder={isConnected ? '说点什么...' : '连接中...'}
            disabled={!isConnected}
            className="flex-1 px-4 py-2 border border-gray-200 rounded-full focus:outline-none focus:border-blue-500 disabled:bg-gray-100"
          />
          <button
            onClick={sendMessage}
            disabled={!isConnected || !inputValue.trim()}
            className="px-4 py-2 bg-blue-500 text-white rounded-full hover:bg-blue-600 disabled:bg-gray-300 disabled:cursor-not-allowed transition-colors"
          >
            <Send className="w-4 h-4" />
          </button>
        </div>
      </div>
    </div>
  );
};

/**
 * WebSocket聊天服务
 */
class ChatWebSocketService {
  private ws: WebSocket | null = null;
  private reconnectAttempts = 0;
  private maxReconnectAttempts = 5;
  private reconnectDelay = 1000;
  private heartbeatInterval: NodeJS.Timeout | null = null;
  private listeners: Map<string, Function> = new Map();

  /**
   * 连接服务器
   */
  connect(roomId: string, options: {
    onOpen?: () => void;
    onClose?: () => void;
    onMessage?: (message: MessageType) => void;
    onError?: (error: Event) => void;
  }): void {
    const wsUrl = `${process.env.NEXT_PUBLIC_WS_URL}/chat/${roomId}`;

    this.ws = new WebSocket(wsUrl);

    this.ws.onopen = () => {
      console.log('聊天室连接成功');
      this.reconnectAttempts = 0;
      this.startHeartbeat();
      options.onOpen?.();
    };

    this.ws.onclose = () => {
      console.log('聊天室连接关闭');
      this.stopHeartbeat();
      options.onClose?.();
      this.reconnect(roomId, options);
    };

    this.ws.onerror = (error) => {
      console.error('聊天室WebSocket错误:', error);
      options.onError?.(error);
    };

    this.ws.onmessage = (event) => {
      try {
        const message = JSON.parse(event.data);
        options.onMessage?.(message);
        this.listeners.get('message')?.(message);
      } catch (error) {
        console.error('消息解析失败:', error);
      }
    };
  }

  /**
   * 断开连接
   */
  disconnect(): void {
    this.stopHeartbeat();
    this.ws?.close();
    this.ws = null;
  }

  /**
   * 加入房间
   */
  joinRoom(roomId: string, user: {
    userId: string;
    userName: string;
    avatar: string;
  }): void {
    this.send({
      type: 'join',
      roomId,
      ...user,
    });
  }

  /**
   * 离开房间
   */
  leaveRoom(roomId: string): void {
    this.send({
      type: 'leave',
      roomId,
    });
  }

  /**
   * 发送消息
   */
  sendMessage(message: any): void {
    this.send({
      type: 'message',
      ...message,
    });
  }

  /**
   * 发送数据
   */
  private send(data: any): void {
    if (this.ws?.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }

  /**
   * 心跳
   */
  private startHeartbeat(): void {
    this.heartbeatInterval = setInterval(() => {
      this.send({ type: 'ping' });
    }, 30000);
  }

  private stopHeartbeat(): void {
    if (this.heartbeatInterval) {
      clearInterval(this.heartbeatInterval);
      this.heartbeatInterval = null;
    }
  }

  /**
   * 重连
   */
  private reconnect(roomId: string, options: any): void {
    if (this.reconnectAttempts >= this.maxReconnectAttempts) {
      console.error('达到最大重连次数');
      return;
    }

    this.reconnectAttempts++;
    const delay = this.reconnectDelay * Math.pow(2, this.reconnectAttempts - 1);

    console.log(`${delay}ms后尝试重连...`);

    setTimeout(() => {
      this.connect(roomId, options);
    }, delay);
  }
}

export const chatWebSocketService = new ChatWebSocketService();
```

### 4.4 连麦互动实现

连麦是在线课堂的重要功能，让学生能够与老师进行实时语音或视频交流。以下是基于WebRTC的连麦实现：

```typescript
// WebRTC连麦服务
// lib/webrtc-connect.ts

/**
 * 连麦用户信息
 */
interface ConnectedUser {
  id: string;
  userName: string;
  avatar?: string;
  stream?: MediaStream;
  isMuted: boolean;
  isVideoOff: boolean;
}

/**
 * 连麦配置
 */
interface ConnectConfig {
  roomId: string;                        // 房间ID
  userId: string;                        // 用户ID
  userName: string;                      // 用户名
  onUserJoined?: (user: ConnectedUser) => void;
  onUserLeft?: (userId: string) => void;
  onRemoteStream?: (userId: string, stream: MediaStream) => void;
  onError?: (error: Error) => void;
}

/**
 * WebRTC连麦服务
 * 使用WebRTC实现低延迟的实时音视频通信
 */
export class WebRTCConnectService {
  private config: ConnectConfig;
  private localStream: MediaStream | null = null;
  private remoteUsers: Map<string, ConnectedUser> = new Map();
  private peerConnections: Map<string, RTCPeerConnection> = new Map();

  // ICE服务器配置
  private readonly iceServers: RTCIceServer[] = [
    { urls: 'stun:stun.l.google.com:19302' },
    { urls: 'stun:stun1.l.google.com:19302' },
    // TURN服务器配置（用于穿透）
    {
      urls: process.env.NEXT_PUBLIC_TURN_URL || '',
      username: process.env.NEXT_PUBLIC_TURN_USER || '',
      credential: process.env.NEXT_PUBLIC_TURN_CREDENTIAL || '',
    },
  ];

  constructor(config: ConnectConfig) {
    this.config = config;
  }

  /**
   * 初始化本地音视频
   */
  async initializeLocalStream(
    videoEnabled: boolean = true,
    audioEnabled: boolean = true
  ): Promise<MediaStream> {
    try {
      this.localStream = await navigator.mediaDevices.getUserMedia({
        video: videoEnabled
          ? {
              width: { ideal: 1280 },
              height: { ideal: 720 },
              frameRate: { ideal: 30 },
            }
          : false,
        audio: audioEnabled
          ? {
              echoCancellation: true,     // 回声消除
              noiseSuppression: true,     // 降噪
              autoGainControl: true,      // 自动增益
            }
          : false,
      });

      return this.localStream;
    } catch (error) {
      console.error('获取本地媒体流失败:', error);
      this.config.onError?.(error as Error);
      throw error;
    }
  }

  /**
   * 加入连麦房间
   */
  async join(): Promise<void> {
    // 获取本地流
    if (!this.localStream) {
      await this.initializeLocalStream();
    }

    // 连接信令服务器
    const wsUrl = `${process.env.NEXT_PUBLIC_WS_URL}/webrtc/${this.config.roomId}`;
    const ws = new WebSocket(wsUrl);

    ws.onopen = () => {
      console.log('WebRTC信令服务器连接成功');

      // 发送加入房间消息
      ws.send(
        JSON.stringify({
          type: 'join',
          roomId: this.config.roomId,
          userId: this.config.userId,
          userName: this.config.userName,
        })
      );
    };

    ws.onmessage = async (event) => {
      const message = JSON.parse(event.data);
      await this.handleSignalingMessage(message, ws);
    };

    ws.onerror = (error) => {
      console.error('WebRTC信令服务器错误:', error);
      this.config.onError?.(error);
    };
  }

  /**
   * 处理信令消息
   */
  private async handleSignalingMessage(
    message: any,
    ws: WebSocket
  ): Promise<void> {
    const { type, from, to, sdp, candidate } = message;

    // 只处理发给自己的消息
    if (to && to !== this.config.userId) return;

    switch (type) {
      case 'offer':
        // 收到其他用户的邀请，建立连接
        await this.handleOffer(from, sdp, ws);
        break;

      case 'answer':
        // 收到对方应答
        await this.handleAnswer(from, sdp);
        break;

      case 'candidate':
        // 收到ICE候选
        await this.handleCandidate(from, candidate);
        break;

      case 'user-joined':
        // 新用户加入，主动发起呼叫
        if (this.localStream) {
          await this.createPeerConnection(from, ws);
          const offer = await this.peerConnections.get(from)!.createOffer();
          await this.peerConnections.get(from)!.setLocalDescription(offer);

          ws.send(
            JSON.stringify({
              type: 'offer',
              to: from,
              from: this.config.userId,
              sdp: offer,
            })
          );
        }
        break;

      case 'user-left':
        // 用户离开
        this.removePeerConnection(from);
        this.config.onUserLeft?.(from);
        break;
    }
  }

  /**
   * 创建对等连接
   */
  private async createPeerConnection(
    peerId: string,
    ws: WebSocket
  ): Promise<RTCPeerConnection> {
    const pc = new RTCPeerConnection({
      iceServers: this.iceServers,
    });

    // 添加本地流
    if (this.localStream) {
      this.localStream.getTracks().forEach((track) => {
        pc.addTrack(track, this.localStream!);
      });
    }

    // 监听远程流
    pc.ontrack = (event) => {
      console.log('收到远程轨道:', peerId);
      const [remoteStream] = event.streams;

      const user = this.remoteUsers.get(peerId) || {
        id: peerId,
        userName: peerId,
        isMuted: false,
        isVideoOff: false,
      };

      this.remoteUsers.set(peerId, {
        ...user,
        stream: remoteStream,
      });

      this.config.onRemoteStream?.(peerId, remoteStream);
    };

    // 监听ICE候选
    pc.onicecandidate = (event) => {
      if (event.candidate) {
        ws.send(
          JSON.stringify({
            type: 'candidate',
            to: peerId,
            from: this.config.userId,
            candidate: event.candidate,
          })
        );
      }
    };

    // 监听连接状态变化
    pc.onconnectionstatechange = () => {
      console.log(`与${peerId}的连接状态:`, pc.connectionState);

      if (
        pc.connectionState === 'failed' ||
        pc.connectionState === 'disconnected'
      ) {
        console.warn(`与${peerId}的连接失败或断开`);
        // 可以尝试重连
        this.reconnectPeer(peerId, ws);
      }
    };

    this.peerConnections.set(peerId, pc);
    return pc;
  }

  /**
   * 处理offer
   */
  private async handleOffer(
    from: string,
    sdp: RTCSessionDescriptionInit,
    ws: WebSocket
  ): Promise<void> {
    const pc = await this.createPeerConnection(from, ws);
    await pc.setRemoteDescription(new RTCSessionDescription(sdp));

    const answer = await pc.createAnswer();
    await pc.setLocalDescription(answer);

    ws.send(
      JSON.stringify({
        type: 'answer',
        to: from,
        from: this.config.userId,
        sdp: answer,
      })
    );
  }

  /**
   * 处理answer
   */
  private async handleAnswer(
    from: string,
    sdp: RTCSessionDescriptionInit
  ): Promise<void> {
    const pc = this.peerConnections.get(from);
    if (pc) {
      await pc.setRemoteDescription(new RTCSessionDescription(sdp));
    }
  }

  /**
   * 处理ICE候选
   */
  private async handleCandidate(
    from: string,
    candidate: RTCIceCandidateInit
  ): Promise<void> {
    const pc = this.peerConnections.get(from);
    if (pc) {
      await pc.addIceCandidate(new RTCIceCandidate(candidate));
    }
  }

  /**
   * 重连对等连接
   */
  private async reconnectPeer(peerId: string, ws: WebSocket): Promise<void> {
    const pc = this.peerConnections.get(peerId);
    if (pc) {
      pc.close();
      this.peerConnections.delete(peerId);
    }

    await this.createPeerConnection(peerId, ws);
    const offer = await this.peerConnections.get(peerId)!.createOffer();
    await this.peerConnections.get(peerId)!.setLocalDescription(offer);

    ws.send(
      JSON.stringify({
        type: 'offer',
        to: peerId,
        from: this.config.userId,
        sdp: offer,
      })
    );
  }

  /**
   * 移除对等连接
   */
  private removePeerConnection(peerId: string): void {
    const pc = this.peerConnections.get(peerId);
    if (pc) {
      pc.close();
      this.peerConnections.delete(peerId);
    }

    this.remoteUsers.delete(peerId);
  }

  /**
   * 切换本地音频
   */
  toggleAudio(enabled: boolean): void {
    if (this.localStream) {
      this.localStream.getAudioTracks().forEach((track) => {
        track.enabled = enabled;
      });
    }
  }

  /**
   * 切换本地视频
   */
  toggleVideo(enabled: boolean): void {
    if (this.localStream) {
      this.localStream.getVideoTracks().forEach((track) => {
        track.enabled = enabled;
      });
    }
  }

  /**
   * 离开连麦
   */
  leave(): void {
    // 关闭所有对等连接
    this.peerConnections.forEach((pc) => pc.close());
    this.peerConnections.clear();
    this.remoteUsers.clear();

    // 停止本地流
    if (this.localStream) {
      this.localStream.getTracks().forEach((track) => track.stop());
      this.localStream = null;
    }
  }
}
```

### 4.5 直播回放系统

直播回放允许学生在直播结束后观看录制的课程。以下是完整的直播录制和回放方案：

```typescript
// services/recordService.ts

import { PrismaClient } from '@prisma/client';

const prisma = new PrismaClient();

/**
 * 录制片段
 */
interface RecordSegment {
  startTime: number;                     // 开始时间（秒）
  endTime: number;                       // 结束时间（秒）
  filename: string;                      // 文件名
  duration: number;                      // 时长（秒）
}

/**
 * 录制任务
 */
interface RecordingTask {
  liveId: string;                        // 直播ID
  roomId: string;                         // 房间ID
  segments: RecordSegment[];             // 录制片段
  status: 'recording' | 'processing' | 'completed' | 'failed';
  startTime: Date;                       // 开始录制时间
  endTime?: Date;                        // 结束录制时间
}

/**
 * 直播录制服务
 * 边直播边录制，生成回放视频
 */
export class LiveRecordService {
  private recordingTasks: Map<string, RecordingTask> = new Map();

  /**
   * 开始录制
   */
  async startRecording(liveId: string, roomId: string): Promise<void> {
    const task: RecordingTask = {
      liveId,
      roomId,
      segments: [],
      status: 'recording',
      startTime: new Date(),
    };

    this.recordingTasks.set(liveId, task);

    // 更新数据库状态
    await prisma.live.update({
      where: { id: liveId },
      data: { isRecording: true },
    });

    console.log(`开始录制直播: ${liveId}`);
  }

  /**
   * 结束录制
   */
  async stopRecording(liveId: string): Promise<void> {
    const task = this.recordingTasks.get(liveId);
    if (!task) return;

    task.status = 'processing';
    task.endTime = new Date();

    // 合并录制片段
    const finalVideo = await this.mergeSegments(task);

    // 创建回放记录
    await this.createReplayRecord(liveId, finalVideo, task);

    task.status = 'completed';

    // 更新数据库状态
    await prisma.live.update({
      where: { id: liveId },
      data: {
        isRecording: false,
        replayUrl: finalVideo,
        replayDuration: this.calculateTotalDuration(task.segments),
      },
    });

    console.log(`直播录制完成: ${liveId}`);
  }

  /**
   * 添加录制片段
   */
  async addSegment(
    liveId: string,
    segment: RecordSegment
  ): Promise<void> {
    const task = this.recordingTasks.get(liveId);
    if (!task || task.status !== 'recording') return;

    task.segments.push(segment);

    console.log(
      `添加录制片段: ${segment.filename}, 时长: ${segment.duration}s`
    );
  }

  /**
   * 合并录制片段
   * 使用FFmpeg合并多个TS片段为完整视频
   */
  private async mergeSegments(task: RecordingTask): Promise<string> {
    const { liveId, segments } = task;

    if (segments.length === 0) {
      throw new Error('没有录制片段');
    }

    // 生成输出文件名
    const outputFilename = `replay_${liveId}_${Date.now()}.mp4`;
    const outputPath = `recordings/${outputFilename}`;

    // 构造FFmpeg命令
    // 首先创建文件列表
    const listFile = `/tmp/ffmpeg_list_${liveId}.txt`;
    const listContent = segments
      .map((seg) => `file '${seg.filename}'`)
      .join('\n');

    // 写入文件列表
    await this.writeFile(listFile, listContent);

    // 执行FFmpeg合并
    const ffmpegCommand = [
      'ffmpeg',
      '-f', 'concat',
      '-safe', '0',
      '-i', listFile,
      '-c', 'copy',          // 直接复制，不重新编码
      '-y',                  // 覆盖输出文件
      outputPath,
    ];

    await this.execCommand(ffmpegCommand.join(' '));

    // 清理临时文件
    await this.deleteFile(listFile);

    return outputPath;
  }

  /**
   * 创建回放记录
   */
  private async createReplayRecord(
    liveId: string,
    videoUrl: string,
    task: RecordingTask
  ): Promise<void> {
    const duration = this.calculateTotalDuration(task.segments);

    await prisma.liveReplay.create({
      data: {
        liveId,
        videoUrl,
        duration,
        coverImage: await this.generateCover(videoUrl),
        segments: task.segments,
        recordedAt: task.startTime,
        publishedAt: new Date(),
      },
    });
  }

  /**
   * 计算总时长
   */
  private calculateTotalDuration(segments: RecordSegment[]): number {
    return segments.reduce((sum, seg) => sum + seg.duration, 0);
  }

  /**
   * 生成封面图
   */
  private async generateCover(videoUrl: string): Promise<string> {
    const coverFilename = `cover_${Date.now()}.jpg`;
    const coverPath = `covers/${coverFilename}`;

    const ffmpegCommand = [
      'ffmpeg',
      '-i', videoUrl,
      '-ss', '00:00:05',       // 取第5秒的帧作为封面
      '-vframes', '1',
      '-y',
      coverPath,
    ];

    await this.execCommand(ffmpegCommand.join(' '));

    return coverPath;
  }

  /**
   * 执行命令行
   */
  private execCommand(command: string): Promise<string> {
    return new Promise((resolve, reject) => {
      exec(command, (error, stdout, stderr) => {
        if (error) {
          reject(error);
        } else {
          resolve(stdout);
        }
      });
    });
  }

  private writeFile(path: string, content: string): Promise<void> {
    return writeFile(path, content) as any;
  }

  private deleteFile(path: string): Promise<void> {
    return unlink(path) as any;
  }
}
```

## 五、学习进度系统

### 5.1 进度追踪架构

学习进度系统是在线教育平台的核心模块，负责记录、追踪和展示学生的学习情况。良好的进度系统能够激励学生持续学习，同时也为教师和家长提供教学反馈。

```typescript
// 学习进度追踪
// store/learningProgressStore.ts

import { create } from 'zustand';
import { persist } from 'zustand/middleware';

/**
 * 单个小节的学习记录
 */
interface LessonProgress {
  lessonId: string;
  status: 'not_started' | 'in_progress' | 'completed';
  watchedTime: number;                  // 观看时长（秒）
  totalTime: number;                     // 总时长
  percent: number;                       // 完成百分比
  lastWatchTime: number;                 // 最后观看时间
  watchCount: number;                    // 观看次数
  notes: string[];                       // 笔记列表
  questions: string[];                   // 提问列表
}

/**
 * 单个课程的学习进度
 */
interface CourseProgress {
  courseId: string;
  userId: string;
  enrolledAt: Date;                       // 报名时间
  lastStudyTime: Date;                   // 最后学习时间
  totalStudyTime: number;                // 总学习时长（秒）

}

/**
 * 学习报告
 */
interface LearningReport {
  userId: string;
  period: {
    start: Date;
    end: Date;
  };
  totalStudyTime: number;               // 本周期学习时长
  activeDays: number;                    // 活跃天数
  completedCourses: number;              // 完成的课程数
  completedLessons: number;              // 完成的小节数
  avgDailyTime: number;                 // 日均学习时长
  streak: number;                        // 连续学习天数
  strengths: string[];                   // 擅长领域
  weaknesses: string[];                  // 薄弱领域
  suggestions: string[];                // 学习建议
}

/**
 * 勋章定义
 */
interface Badge {
  id: string;
  name: string;                          // 名称
  description: string;                    // 描述
  icon: string;                          // 图标
  earnedAt: Date;                        // 获得时间
  criteria: string;                      // 获得条件
}

/**
 * 证书定义
 */
interface Certificate {
  id: string;
  courseId: string;
  courseName: string;
  earnedAt: Date;                        // 获得时间
  certificateUrl: string;                // 证书URL
  verificationCode: string;             // 验证码
}

/**
 * 进度状态
 */
interface LearningProgressState {
  // 当前用户的所有课程进度
  courseProgressMap: Record<string, CourseProgress>;

  // 当前学习会话
  currentSession: {
    courseId: string | null;
    lessonId: string | null;
    startTime: number | null;
    lastActivityTime: number | null;
  };

  // 加载状态
  isLoading: boolean;

  // 动作
  enrollCourse: (courseId: string) => Promise<void>;
  updateLessonProgress: (courseId: string, lessonId: string, progress: Partial<LessonProgress>) => void;
  getCourseProgress: (courseId: string) => CourseProgress | null;
  getLessonProgress: (courseId: string, lessonId: string) => LessonProgress | null;

export const useLearningProgressStore = create<LearningProgressState>()(
  persist(
    (set, get) => ({
      courseProgressMap: {},
      currentSession: {
        courseId: null,
        lessonId: null,
        startTime: null,
        lastActivityTime: null,
      },
      isLoading: false,

      /**
       * 报名课程
       */
      enrollCourse: async (courseId: string) => {
        const { user } = useUserStore.getState();
        if (!user) return;

        // 创建课程进度记录
        const newProgress: CourseProgress = {
          courseId,
          userId: user.id,
          enrolledAt: new Date(),
          lastStudyTime: new Date(),
          totalStudyTime: 0,
          completedLessons: 0,
          totalLessons: 0,
          percent: 0,
          currentLessonId: null,
          lessonProgress: {},
          certificates: [],
          badges: [],
        };

        set((state) => ({
          courseProgressMap: {
            ...state.courseProgressMap,
            [courseId]: newProgress,
          },
        }));

        // 同步到服务器
        await fetch('/api/progress/enroll', {
          method: 'POST',
          headers: { 'Content-Type': 'application/json' },
          body: JSON.stringify({ courseId }),
        });
      },

      /**
       * 更新小节进度
       */
      updateLessonProgress: (
        courseId: string,
        lessonId: string,
        progress: Partial<LessonProgress>
      ) => {
        set((state) => {
          const courseProgress = state.courseProgressMap[courseId];
          if (!courseProgress) return state;

          // 更新或创建小节进度
          const existingLesson = courseProgress.lessonProgress[lessonId] || {
            lessonId,
            status: 'not_started' as const,
            watchedTime: 0,
            totalTime: 0,
            percent: 0,
            lastWatchTime: Date.now(),
            watchCount: 0,
            notes: [],
            questions: [],
          };

          const updatedLesson: LessonProgress = {
            ...existingLesson,
            ...progress,
            lastWatchTime: Date.now(),
          };

          // 判断是否完成（观看超过90%）
          if (
            updatedLesson.totalTime > 0 &&
            updatedLesson.watchedTime / updatedLesson.totalTime >= 0.9
          ) {
            updatedLesson.status = 'completed';
          } else if (updatedLesson.watchedTime > 0) {
            updatedLesson.status = 'in_progress';
          }

          // 更新课程进度
          const completedLessons = Object.values({
            ...courseProgress.lessonProgress,
            [lessonId]: updatedLesson,
          }).filter((l) => l.status === 'completed').length;

          const totalStudyTime =
            courseProgress.totalStudyTime +
            (progress.watchedTime || 0);

          const updatedCourseProgress: CourseProgress = {
            ...courseProgress,
            lastStudyTime: new Date(),
            totalStudyTime,
            completedLessons,
            percent:
              courseProgress.totalLessons > 0
                ? (completedLessons / courseProgress.totalLessons) * 100
                : 0,
            currentLessonId: lessonId,
            lessonProgress: {
              ...courseProgress.lessonProgress,
              [lessonId]: updatedLesson,
            },
          };

          return {
            courseProgressMap: {
              ...state.courseProgressMap,
              [courseId]: updatedCourseProgress,
            },
            currentSession: {
              ...state.currentSession,
              courseId,
              lessonId,
              lastActivityTime: Date.now(),
            },
          };
        });
      },

      /**
       * 获取课程进度
       */
      getCourseProgress: (courseId: string) => {
        return get().courseProgressMap[courseId] || null;
      },

      /**
       * 获取小节进度
       */
      getLessonProgress: (courseId: string, lessonId: string) => {
        const courseProgress = get().courseProgressMap[courseId];
        return courseProgress?.lessonProgress[lessonId] || null;
      },

      /**
       * 计算整体学习报告
       */
      calculateOverallProgress: (): LearningReport => {
        const { courseProgressMap } = get();
        const progressList = Object.values(courseProgressMap);

        // 计算各项指标
        const totalStudyTime = progressList.reduce(
          (sum, p) => sum + p.totalStudyTime,
          0
        );

        const completedCourses = progressList.filter(
          (p) => p.percent >= 100
        ).length;

        const completedLessons = progressList.reduce(
          (sum, p) => sum + p.completedLessons,
          0
        );

        // 计算活跃天数
        const studyDays = new Set(
          progressList.flatMap((p) =>
            Object.values(p.lessonProgress).map(
              (l) => new Date(l.lastWatchTime).toDateString()
            )
          )
        );

        // 计算连续学习天数
        const streak = calculateStreak(
          Array.from(studyDays)
            .map((d) => new Date(d))
            .sort((a, b) => b.getTime() - a.getTime())
        );

        // 生成学习建议
        const suggestions = generateSuggestions({
          totalStudyTime,
          completedCourses,
          completedLessons,
          streak,
        });

        return {
          userId: useUserStore.getState().user?.id || '',
          period: {
            start: new Date(Date.now() - 30 * 24 * 60 * 60 * 1000),
            end: new Date(),
          },
          totalStudyTime,
          activeDays: studyDays.size,
          completedCourses,
          completedLessons,
          avgDailyTime: totalStudyTime / 30,
          streak,
          strengths: [],
          weaknesses: [],
          suggestions,
        };
      },

      /**
       * 同步到服务器
       */
      syncToServer: async () => {
        const { courseProgressMap, currentSession } = get();

        try {
          await fetch('/api/progress/sync', {
            method: 'POST',
            headers: { 'Content-Type': 'application/json' },
            body: JSON.stringify({
              courseProgressMap,
              currentSession,
            }),
          });
        } catch (error) {
          console.error('进度同步失败:', error);
        }
      },
    }),
    {
      name: 'edu-learning-progress',
      partialize: (state) => ({
        courseProgressMap: state.courseProgressMap,
      }),
    }
  )
);

/**
 * 计算连续学习天数
 */
function calculateStreak(sortedDates: Date[]): number {
  if (sortedDates.length === 0) return 0;

  let streak = 1;
  const today = new Date();
  today.setHours(0, 0, 0, 0);

  // 检查今天或昨天是否学习（保证连续性）
  const lastStudyDate = sortedDates[0];
  lastStudyDate.setHours(0, 0, 0, 0);

  const diffDays = Math.floor(
    (today.getTime() - lastStudyDate.getTime()) / (24 * 60 * 60 * 1000)
  );

  if (diffDays > 1) return 0; // 超过1天未学习，连续中断

  // 向前计算连续天数
  for (let i = 1; i < sortedDates.length; i++) {
    const current = new Date(sortedDates[i]);
    current.setHours(0, 0, 0, 0);

    const prev = new Date(sortedDates[i - 1]);
    prev.setHours(0, 0, 0, 0);

    const diff = Math.floor(
      (prev.getTime() - current.getTime()) / (24 * 60 * 60 * 1000)
    );

    if (diff === 1) {
      streak++;
    } else {
      break;
    }
  }

  return streak;
}
```

### 5.2 完播率统计分析

完播率是衡量学习效果的重要指标，以下是完整的完播率统计系统：

```typescript
// analytics/completionRate.ts

/**
 * 完播率统计数据
 */
interface CompletionRateStats {
  courseId: string;
  lessonId: string;
  totalViews: number;                    // 总观看次数
  completedViews: number;                // 完成观看次数
  completionRate: number;                // 完播率
  avgWatchTime: number;                  // 平均观看时长
  avgProgress: number;                   // 平均观看进度
  dropOffPoints: DropOffPoint[];         // 流失点分析
}

/**
 * 流失点
 */
interface DropOffPoint {
  time: number;                          // 时间点（秒）
  dropOffCount: number;                  // 在此时间点流失的人数
  dropOffRate: number;                   // 流失率
  possibleReasons: string[];            // 可能原因
}

/**
 * 计算完播率
 */
export function calculateCompletionRate(
  views: ViewRecord[]
): CompletionRateStats {
  const totalViews = views.length;

  // 完成观看的定义：观看时长 >= 90%的视频时长
  const completedViews = views.filter(
    (v) => v.watchedTime / v.totalTime >= 0.9
  ).length;

  const completionRate = totalViews > 0
    ? (completedViews / totalViews) * 100
    : 0;

  // 计算平均观看时长
  const avgWatchTime = views.reduce(
    (sum, v) => sum + v.watchedTime,
    0
  ) / totalViews;

  // 计算平均进度
  const avgProgress = views.reduce(
    (sum, v) => sum + (v.watchedTime / v.totalTime) * 100,
    0
  ) / totalViews;

  // 分析流失点
  const dropOffPoints = analyzeDropOffPoints(views);

  return {
    courseId: views[0]?.courseId || '',
    lessonId: views[0]?.lessonId || '',
    totalViews,
    completedViews,
    completionRate,
    avgWatchTime,
    avgProgress,
    dropOffPoints,
  };
}

/**
 * 分析流失点
 */
function analyzeDropOffPoints(views: ViewRecord[]): DropOffPoint[] {
  if (views.length === 0) return [];

  const totalTime = views[0].totalTime;
  const timeSegments = 20; // 将视频分成20段

  const dropOffBySegment: Map<number, number> = new Map();

  // 统计每个时间段的流失人数
  views.forEach((view) => {
    if (view.watchedTime < view.totalTime * 0.9) {
      // 计算在哪个时间段停止观看
      const segmentIndex = Math.min(
        Math.floor((view.watchedTime / view.totalTime) * timeSegments),
        timeSegments - 1
      );
      dropOffBySegment.set(
        segmentIndex,
        (dropOffBySegment.get(segmentIndex) || 0) + 1
      );
    }
  });

  // 转换为流失点列表
  const dropOffPoints: DropOffPoint[] = [];

  dropOffBySegment.forEach((count, index) => {
    const time = (index / timeSegments) * totalTime;
    const dropOffRate = (count / views.length) * 100;

    dropOffPoints.push({
      time,
      dropOffCount: count,
      dropOffRate,
      possibleReasons: inferDropOffReasons(time, totalTime, dropOffRate),
    });
  });

  // 按流失率排序
  return dropOffPoints.sort((a, b) => b.dropOffRate - a.dropOffRate);
}

/**
 * 推断流失原因
 */
function inferDropOffReasons(
  time: number,
  totalTime: number,
  dropOffRate: number
): string[] {
  const reasons: string[] = [];
  const percent = time / totalTime;

  // 如果在前10%流失，可能是视频开头不吸引人
  if (percent < 0.1) {
    reasons.push('视频开头不够吸引人');
    reasons.push('内容与学生预期不符');
  }

  // 如果在中间流失，可能是内容太难或太简单
  if (percent >= 0.1 && percent <= 0.8) {
    if (dropOffRate > 10) {
      reasons.push('可能存在难以理解的知识点');
      reasons.push('视频节奏可能需要调整');
    }
  }

  // 如果在最后流失（但未完成），可能是疲劳
  if (percent > 0.8 && percent < 1) {
    reasons.push('视频可能过长');
    reasons.push('学生可能已经获得所需信息');
  }

  return reasons;
}

/**
 * 观看记录
 */
interface ViewRecord {
  odId: string;
  lessonId: string;
  userId: string;
  watchedTime: number;                  // 观看时长（秒）
  totalTime: number;                    // 视频总时长
  watchedAt: Date;                      // 观看时间
}

/**

 */
export function generateLearningReport(

  summary: {

  // 获取该用户在该时间段的所有学习记录

  // 计算汇总数据
```

```typescript
  async random抽题(
    quizId: string,
    config: {
      totalCount: number;
      difficultyDistribution?: {
        easy: number;
        medium: number;
        hard: number;
      };
      tags?: string[];
    }
  ): Promise<Question[]> {
    const questions: Question[] = [];
    const distribution = config.difficultyDistribution || {
      easy: Math.ceil(config.totalCount * 0.3),
      medium: Math.ceil(config.totalCount * 0.5),
      hard: Math.floor(config.totalCount * 0.2),
    };

    // 按难度分别抽取
    for (const [difficulty, count] of Object.entries(distribution)) {
      const difficultyEnum = difficulty as unknown as Difficulty;

      const tagFilter = config.tags?.length
        ? { tags: { hasSome: config.tags } }
        : {};

      const availableQuestions = await prisma.question.findMany({
        where: {
          quizId: null, // 只抽取未分配的题目
          difficulty: difficultyEnum,
          ...tagFilter,
        },
        take: count,
        orderBy: {
          statistics: {
            correctRate: 'asc', // 优先抽取正确率低的（增加区分度）
          },
        },
      });

      questions.push(...availableQuestions);

      // 标记这些题目已被使用
      await prisma.question.updateMany({
        where: {
          id: { in: availableQuestions.map((q) => q.id) },
        },
        data: { quizId },
      });
    }

    return questions;
  }
}
```

### 6.3 自动批改与成绩分析

```typescript
// 自动批改服务
// services/gradingService.ts

import { PrismaClient } from '@prisma/client';
import { QuestionType } from '@/types/question';

const prisma = new PrismaClient();

/**
 * 答案记录
 */
interface Answer {
  questionId: string;
  selectedOptionId?: string;
  textAnswer?: string;
  flagged?: boolean;
}

/**
 * 批改结果
 */
interface GradingResult {
  questionId: string;
  isCorrect: boolean;
  score: number;
  maxScore: number;
  feedback?: string;
  correctAnswer?: string | string[];
  explanation?: string;
}

/**
 * 自动批改服务
 */
export class GradingService {
  /**
   * 批改客观题
   */
  async gradeObjective(
    answers: Answer[],
    quizId: string
  ): Promise<GradingResult[]> {
    // 获取题目信息
    const questionIds = answers.map((a) => a.questionId);
    const questions = await prisma.question.findMany({
      where: { id: { in: questionIds } },
    });

    const questionMap = new Map(questions.map((q) => [q.id, q]));
    const results: GradingResult[] = [];

    for (const answer of answers) {
      const question = questionMap.get(answer.questionId);
      if (!question) continue;

      let isCorrect = false;
      let feedback: string | undefined;

      switch (question.type) {
        case QuestionType.SINGLE_CHOICE:
        case QuestionType.TRUE_FALSE:
          isCorrect =
            answer.selectedOptionId === question.correctAnswer;
          feedback = isCorrect ? '回答正确' : '回答错误';
          break;

        case QuestionType.MULTIPLE_CHOICE:
          const correctAnswers = Array.isArray(question.correctAnswer)
            ? question.correctAnswer
            : [question.correctAnswer];
          const selectedAnswers = answer.selectedOptionId?.split(',') || [];

          // 多选题：全部选对才算正确
          isCorrect =
            correctAnswers.length === selectedAnswers.length &&
            correctAnswers.every((a) => selectedAnswers.includes(a));
          feedback = isCorrect
            ? '回答正确'
            : `正确答案是：${correctAnswers.join(', ')}`;
          break;

        case QuestionType.FILL_BLANK:
          // 填空题：忽略空格比较
          const normalizedCorrect = question.correctAnswer
            ?.toString()
            .replace(/\s+/g, '')
            .toLowerCase();
          const normalizedAnswer = answer.textAnswer
            ?.replace(/\s+/g, '')
            .toLowerCase();
          isCorrect = normalizedCorrect === normalizedAnswer;
          feedback = isCorrect ? '回答正确' : `正确答案是：${question.correctAnswer}`;
          break;

        default:
          // 其他题型需要手动批改
          break;
      }

      results.push({
        questionId: answer.questionId,
        isCorrect,
        score: isCorrect ? question.points : 0,
        maxScore: question.points,
        feedback,
        correctAnswer: question.correctAnswer,
        explanation: question.explanation,
      });
    }

    return results;
  }

  /**
   * 提交答案并获取结果
   */
  async submitAndGrade(
    odId: string,
    userId: string,
    answers: Answer[]
  ): Promise<{
    results: GradingResult[];
    totalScore: number;
    maxScore: number;
    correctCount: number;
    totalCount: number;
  }> {
    // 批改客观题
    const results = await this.gradeObjective(answers, odId);

    // 计算总分
    const totalScore = results.reduce((sum, r) => sum + r.score, 0);
    const maxScore = results.reduce((sum, r) => sum + r.maxScore, 0);
    const correctCount = results.filter((r) => r.isCorrect).length;
    const totalCount = results.length;

    // 保存答案记录
    await prisma.quizAttempt.create({
      data: {
        quizId: odId,
        userId,
        answers: answers as any,
        results: results as any,
        score: totalScore,
        maxScore,
        correctCount,
        completedAt: new Date(),
      },
    });

    // 更新用户积分
    await this.updateUserPoints(userId, totalScore);

    return {
      results,
      totalScore,
      maxScore,
      correctCount,
      totalCount,
    };
  }

  /**
   * 更新用户积分
   */
  private async updateUserPoints(
    userId: string,
    score: number
  ): Promise<void> {
    await prisma.user.update({
      where: { id: userId },
      data: {
        points: { increment: score },
      },
    });
  }

  /**
   * 生成错题本
   */
  async generateMistakeBook(
    userId: string,
    options?: { limit?: number; category?: string }
  ): Promise<{
    mistakes: {
      question: any;
      wrongCount: number;
      lastWrongAt: Date;
    }[];
  }> {
    // 获取用户所有的错题记录
    const attempts = await prisma.quizAttempt.findMany({
      where: { userId },
      include: {
        quiz: true,
        results: true,
      },
    });

    // 统计每道错题的错误次数
    const mistakeMap = new Map<
      string,
      { wrongCount: number; lastWrongAt: Date }
    >();

    for (const attempt of attempts) {
      for (const result of attempt.results) {
        if (!result.isCorrect) {
          const existing = mistakeMap.get(result.questionId);
          if (existing) {
            existing.wrongCount++;
            if (attempt.completedAt > existing.lastWrongAt) {
              existing.lastWrongAt = attempt.completedAt;
            }
          } else {
            mistakeMap.set(result.questionId, {
              wrongCount: 1,
              lastWrongAt: attempt.completedAt,
            });
          }
        }
      }
    }

    // 获取题目详情
    const mistakeIds = Array.from(mistakeMap.keys());
    const questions = await prisma.question.findMany({
      where: { id: { in: mistakeIds } },
    });

    const questionMap = new Map(questions.map((q) => [q.id, q]));

    // 构建错题本
    const mistakes = mistakeIds
      .map((id) => ({
        question: questionMap.get(id),
        ...mistakeMap.get(id)!,
      }))
      .filter((m) => m.question)
      .sort((a, b) => b.wrongCount - a.wrongCount)
      .slice(0, options?.limit || 50);

    return { mistakes };
  }
}
```

```typescript
    startTime: number;
    endTime: number;
    selectedText: string;

      if (!selection || selection.isCollapsed) return;

      const selectedText = selection.toString().trim();
      if (!selectedText) return;

      // 获取当前播放时间
      const currentTime = video.currentTime;
```

```typescript
 * 用户等级服务

  /**
   * 根据积分计算等级
   */
  static calculateLevel(points: number): LevelDefinition {
    for (let i = LEVEL_DEFINITIONS.length - 1; i >= 0; i--) {
      if (points >= LEVEL_DEFINITIONS[i].minPoints) {
        return LEVEL_DEFINITIONS[i];
      }
    }
    return LEVEL_DEFINITIONS[0];
  }

  /**
   * 获取当前等级进度
   */
  static getLevelProgress(
    points: number
  ): {
    currentLevel: LevelDefinition;
    nextLevel: LevelDefinition | null;
    progress: number;                      // 0-100
    pointsToNext: number;
  } {
    const currentLevel = this.calculateLevel(points);
    const currentIndex = LEVEL_DEFINITIONS.findIndex(
      (l) => l.level === currentLevel.level
    );

    // 已是最高等级
    if (currentIndex === LEVEL_DEFINITIONS.length - 1) {
      return {
        currentLevel,
        nextLevel: null,
        progress: 100,
        pointsToNext: 0,
      };
    }

    const nextLevel = LEVEL_DEFINITIONS[currentIndex + 1];
    const pointsInLevel = points - currentLevel.minPoints;
    const pointsNeeded = nextLevel.minPoints - currentLevel.minPoints;

    return {
      currentLevel,
      nextLevel,
      progress: Math.min((pointsInLevel / pointsNeeded) * 100, 100),
      pointsToNext: nextLevel.minPoints - points,
    };
  }

  /**
   * 获取用户特权列表
   */
  static getUserPrivileges(points: number): string[] {
    const level = this.calculateLevel(points);
    return level.privileges;
  }

  /**
   * 检查用户是否拥有某项特权
   */
  static hasPrivilege(points: number, privilege: string): boolean {
    const privileges = this.getUserPrivileges(points);
    return privileges.includes(privilege);
  }
}

    STREAK_30: 200,                       // 连续30天
  };

  /**
   * 增加用户积分
   */
  static async addPoints(
    userId: string,
    eventType: keyof typeof this.EVENTS,
    metadata?: Record<string, any>
  ): Promise<{ newPoints: number; levelUp?: LevelDefinition }> {
    const points = this.EVENTS[eventType];

    // 更新用户积分
    const user = await prisma.user.update({
      where: { id: userId },
      data: {
        points: { increment: points },
      },
    });

    // 检查是否升级
    const currentLevel = UserLevelService.calculateLevel(
      user.points - points
    );
    const newLevel = UserLevelService.calculateLevel(user.points);

    let levelUp: LevelDefinition | undefined;
    if (newLevel.level > currentLevel.level) {
      levelUp = newLevel;

      // 创建升级通知
      await prisma.notification.create({
        data: {
          userId,
          type: 'level_up',
          title: '恭喜升级！',
          content: `您已升级为${newLevel.name}，解锁了新的特权！`,
          metadata: { level: newLevel.level },
        },
      });
    }

    // 记录积分日志
    await prisma.pointsLog.create({
      data: {
        userId,
        eventType,
        points,
        balance: user.points,
        metadata,
      },
    });

    return { newPoints: user.points, levelUp };
  }
```

```typescript
    if (currentIndex === -1 || currentIndex >= playlist.length - 1) {
      return;
    }

    const nextVideo = playlist[currentIndex + 1];

    // 创建预加载链接
    const link = document.createElement('link');
    link.rel = 'preload';
    link.as = 'video';
    link.href = nextVideo.url;
    document.head.appendChild(link);

    // 5分钟后自动清理
    setTimeout(() => {
      link.remove();
    }, 5 * 60 * 1000);
  }

  /**
   * 智能缓冲策略
   * 根据网络状况动态调整缓冲时长
   */
  static getSmartBufferSize(
    bandwidth: number
  ): { bufferSize: number; preloadSize: number } {
    // 带宽单位：kbps
    if (bandwidth > 5000) {
      // 高速网络：缓冲30秒，预加载60秒
      return { bufferSize: 30, preloadSize: 60 };
    } else if (bandwidth > 2000) {
      // 中速网络：缓冲60秒，预加载90秒
      return { bufferSize: 60, preloadSize: 90 };
    } else if (bandwidth > 1000) {
      // 较慢网络：缓冲120秒，不预加载
      return { bufferSize: 120, preloadSize: 0 };
    } else {
      // 低速网络：尽可能多缓冲
      return { bufferSize: 180, preloadSize: 0 };
    }
  }

  /**
   * 自适应清晰度选择
   * 根据网络状况自动选择合适的清晰度
   */
  static selectAdaptiveQuality(
    bandwidth: number,
    availableQualities: { height: number; bitrate: number }[]
  ): { height: number; bitrate: number } {
    // 排序（从高到低）
    const sorted = [...availableQualities].sort(
      (a, b) => b.bitrate - a.bitrate
    );

    // 选择码率小于带宽60%的最高清晰度
    for (const quality of sorted) {
      if (quality.bitrate < bandwidth * 0.6) {
        return quality;
      }
    }

    // 如果都太高，选择最低的
    return sorted[sorted.length - 1];
  }
}

 * 视频缓存管理

export class VideoCacheManager {
  private cache: Map<string, {
    url: string;
    blob: Blob;
    lastAccess: number;
  }> = new Map();

  private maxCacheSize = 3;               // 最多缓存3个视频
  private maxCacheAge = 24 * 60 * 60 * 1000; // 缓存有效期24小时

  /**
   * 添加到缓存
   */
  async addToCache(videoId: string, url: string): Promise<void> {
    try {
      const response = await fetch(url);
      const blob = await response.blob();

      // 如果缓存已满，删除最旧的
      if (this.cache.size >= this.maxCacheSize) {
        this.evictOldest();
      }

      this.cache.set(videoId, {
        url,
        blob,
        lastAccess: Date.now(),
      });

      console.log(`视频已缓存: ${videoId}`);
    } catch (error) {
      console.error('视频缓存失败:', error);
    }
  }

  /**
   * 从缓存获取
   */
  async getFromCache(videoId: string): Promise<string | null> {
    const cached = this.cache.get(videoId);

    if (!cached) return null;

    // 检查是否过期
    if (Date.now() - cached.lastAccess > this.maxCacheAge) {
      this.cache.delete(videoId);
      return null;
    }

    // 更新访问时间
    cached.lastAccess = Date.now();

    return URL.createObjectURL(cached.blob);
  }

  /**
   * 清理最旧的缓存
   */
  private evictOldest(): void {
    let oldest: string | null = null;
    let oldestTime = Infinity;

    this.cache.forEach((value, key) => {
      if (value.lastAccess < oldestTime) {
        oldestTime = value.lastAccess;
        oldest = key;
      }
    });

    if (oldest) {
      const cached = this.cache.get(oldest);
      if (cached) {
        // 释放Blob URL
        URL.revokeObjectURL(cached.url);
      }
      this.cache.delete(oldest);
      console.log(`清理过期缓存: ${oldest}`);
    }
  }

  /**
   * 清空所有缓存
   */
  clear(): void {
    this.cache.forEach((cached) => {
      URL.revokeObjectURL(cached.url);
    });
    this.cache.clear();
  }
}
```

```typescript
interface DRMConfig {

/**
 * DRM管理器
 */
export class DRMManager {
  private config: DRMConfig;

  constructor(config: DRMConfig) {
    this.config = config;
  }

  /**
   * 初始化DRM
   */
  async initialize(
    videoElement: HTMLVideoElement,
    contentId: string
  ): Promise<MediaDecryptor | null> {
    if (!('requestMediaKeySystemAccess' in navigator)) {
      console.warn('浏览器不支持DRM');
      return null;
    }

    try {
      // 创建密钥系统访问
      const keySystemAccess = await navigator.requestMediaKeySystemAccess(
        this.config.provider,
        this.getKeySystemConfig()
      );

      // 创建媒体密钥
      const mediaKeys = await keySystemAccess.createMediaKeys();

      // 设置视频元素
      await videoElement.setMediaKeys(mediaKeys);

      // 监听密钥请求
      videoElement.addEventListener('encrypted', async (event) => {
        const { initDataType, initData } = event as any;

        await this.handleKeyRequest(
          mediaKeys,
          contentId,
          initDataType,
          initData
        );
      });

      return mediaKeys;
    } catch (error) {
      console.error('DRM初始化失败:', error);
      return null;
    }
  }

  /**
   * 处理密钥请求
   */
  private async handleKeyRequest(
    mediaKeys: MediaKeys,
    contentId: string,
    initDataType: string,
    initData: ArrayBuffer
  ): Promise<void> {
    // 获取许可证
    const license = await this.fetchLicense(contentId, initData);

    // 创建密钥会话
    const session = mediaKeys.createSession();
    await session.generateRequest(initDataType, initData);

    // 加载许可证
    await session.update(license);
  }

  /**
   * 从许可证服务器获取许可证
   */
  private async fetchLicense(
    contentId: string,
    initData: ArrayBuffer
  ): Promise<ArrayBuffer> {
    const response = await fetch(this.config.licenseServer, {
      method: 'POST',
      headers: {
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({
        contentId,
        initData: Array.from(new Uint8Array(initData)),
        provider: this.config.provider,
      }),
    });

    const { license } = await response.json();
    return new Uint8Array(license).buffer;
  }

 * 在服务端对视频URL进行签名，防止盗链
 */
export class VideoUrlSigner {
  private secret: string;

  constructor(secret: string) {
    this.secret = secret;
  }

  /**
   * 生成带签名的视频URL
   */
  signUrl(videoUrl: string, expiresIn: number = 3600): string {
    const expiry = Math.floor(Date.now() / 1000) + expiresIn;
    const signature = this.generateSignature(videoUrl, expiry);

    const url = new URL(videoUrl);
    url.searchParams.set('expires', expiry.toString());
    url.searchParams.set('signature', signature);

    return url.toString();
  }

  /**
   * 验证签名
   */
  verifyUrl(url: string, signature: string, expires: number): boolean {
    // 检查是否过期
    if (Math.floor(Date.now() / 1000) > expires) {
      return false;
    }

    // 重新计算签名进行验证
    const expectedSignature = this.generateSignature(url, expires);
    return signature === expectedSignature;
  }

  /**
   * 生成签名
   */
  private generateSignature(url: string, expiry: number): string {
    const data = `${url}:${expiry}`;
    return crypto.createHmac('sha256', this.secret).update(data).digest('hex');
  }
}
```

```typescript
/**

 */
export class AntiCheatSystem {
  private config: AntiCheatConfig;
  private tabSwitchCount = 0;
  private warningCallbacks: Array<(message: string) => void> = [];
  private violationCallbacks: Array<(type: string, details: any) => void> = [];

  constructor(config: Partial<AntiCheatConfig> = {}) {
    this.config = {
      enableScreenRecordDetection: true,
      enableTabSwitchDetection: true,
      enableCopyPasteDetection: true,
      enableRightClickDetection: true,
      maxTabSwitches: 5,
      tabSwitchWarningThreshold: 3,
      ...config,
    };
  }

  /**
   * 开始监控
   */
  start(): void {
    if (this.config.enableTabSwitchDetection) {
      this.monitorTabSwitch();
    }

    if (this.config.enableScreenRecordDetection) {
      this.monitorScreenRecord();
    }

    if (this.config.enableCopyPasteDetection) {
      this.monitorCopyPaste();
    }

    if (this.config.enableRightClickDetection) {
      this.monitorRightClick();
    }
  }

  /**
   * 停止监控
   */
  stop(): void {
    document.removeEventListener('visibilitychange', this.handleVisibilityChange);
    document.removeEventListener('copy', this.handleCopy);
    document.removeEventListener('cut', this.handleCopy);
    document.removeEventListener('contextmenu', this.handleRightClick);
  }

  /**
   * 注册警告回调
   */
  onWarning(callback: (message: string) => void): void {
    this.warningCallbacks.push(callback);
  }

  /**
   * 注册违规回调
   */
  onViolation(callback: (type: string, details: any) => void): void {
    this.violationCallbacks.push(callback);
  }

  /**
   * 监控标签页切换
   */
  private monitorTabSwitch(): void {
    const handleVisibilityChange = () => {
      if (document.hidden) {
        this.tabSwitchCount++;

        // 检查是否超过阈值
        if (this.tabSwitchCount >= this.config.maxTabSwitches) {
          this.violationCallbacks.forEach((cb) =>
            cb('tab_switch_overflow', {
              count: this.tabSwitchCount,
              threshold: this.config.maxTabSwitches,
            })
          );
        } else if (this.tabSwitchCount >= this.config.tabSwitchWarningThreshold) {
          const remaining = this.config.maxTabSwitches - this.tabSwitchCount;
          const message = `检测到切换标签页行为，剩余警告次数：${remaining}`;
          this.warningCallbacks.forEach((cb) => cb(message));
        }

        // 记录日志
        this.logViolation('tab_switch', {
          count: this.tabSwitchCount,
          timestamp: Date.now(),
        });
      }
    };

    this.handleVisibilityChange = handleVisibilityChange;
    document.addEventListener('visibilitychange', handleVisibilityChange);
  }

  private handleVisibilityChange: () => void = () => {};

  /**
   * 监控录屏
   */
  private monitorScreenRecord(): void {
    // 检测媒体查询
    const presentationQuery = window.matchMedia('(presentation: container)');

    if (presentationQuery.matches) {
      this.violationCallbacks.forEach((cb) =>
        cb('screen_recording', { type: 'presentation_mode' })
      );
    }

    // 定时检测
    setInterval(() => {
      // 检测页面是否被隐藏（可能是被覆盖）
      if (document.hidden) {
        this.logViolation('possible_recording', {
          pageHidden: true,
          timestamp: Date.now(),
        });
      }
    }, 5000);
  }

  /**
   * 监控复制粘贴
   */
  private monitorCopyPaste(): void {
    const handleCopy = (e: ClipboardEvent) => {
      e.preventDefault();
      this.warningCallbacks.forEach((cb) =>
        cb('考试中禁止复制粘贴内容')
      );

      this.logViolation('copy_attempt', {
        timestamp: Date.now(),
      });
    };

    this.handleCopy = handleCopy;
    document.addEventListener('copy', handleCopy);
    document.addEventListener('cut', handleCopy);
  }

  /**
   * 监控右键菜单
   */
  private monitorRightClick(): void {
    const handleRightClick = (e: Event) => {
      e.preventDefault();
      this.warningCallbacks.forEach((cb) =>
        cb('考试中禁止使用右键菜单')
      );

      this.logViolation('right_click_attempt', {
        timestamp: Date.now(),
      });
    };

    this.handleRightClick = handleRightClick;
    document.addEventListener('contextmenu', handleRightClick);
  }

  /**
   * 记录违规日志
   */
  private logViolation(type: string, details: any): void {
    console.warn('考试违规:', type, details);

    // 发送到服务器
    fetch('/api/quiz/violation', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        type,
        details,
        timestamp: Date.now(),
        url: window.location.href,
      }),
    }).catch(console.error);
  }

  /**
   * 获取违规统计
   */
  getStatistics(): { tabSwitches: number } {
    return {
      tabSwitches: this.tabSwitchCount,
```

```typescript
function selectStreamTier(
  bandwidth: number,
  userPreference?: string
): string {
  // 用户手动选择优先
  if (userPreference && STREAM_TIERS.some((t) => t.name === userPreference)) {
    return userPreference;
  }

  // 根据带宽自动选择
  for (const tier of STREAM_TIERS) {
    if (bandwidth >= tier.bitrate * 1.5) {
      // 留50%余量
      return tier.name;
    }
  }

  // 最低清晰度
  return STREAM_TIERS[0].name;
}
```

### 11.5 我的思考：技术支撑业务

这次直播优化案例让我深刻认识到技术架构对业务的重要性。在活动开始前，技术团队应该：

**第一，做好容量规划。** 根据活动的预期参与人数，提前评估系统容量，确保关键技术指标有足够的冗余。

**第二，设计弹性架构。** 系统应该能够根据负载自动扩展，而不是在峰值时崩溃。多级CDN、分级分发都是应对高并发的有效手段。

**第三，建立监控告警。** 实时监控系统各项指标，在问题扩大之前及时发现和处理。

**第四，准备应急预案。** 即使做了充分准备，仍可能出现问题。提前准备降级方案和应急响应流程，确保问题发生时能够快速恢复。

技术是手段，业务是目的。好的技术架构应该能够支撑业务的快速发展，而不是成为业务的瓶颈。作为技术负责人，我们既要懂技术，也要懂业务，才能做出真正有价值的决策。

## 总结

在线教育平台是一个复杂的系统，涉及课程管理、视频播放、直播系统、学习追踪、作业考试、互动社交等多个模块。本指南从实际项目经验出发，详细讲解了各个模块的设计原理和实现方案。

**核心要点回顾：**

1. **课程结构设计**：采用章-节-知识点的三级结构，支持多种课程类型和定价模式。

2. **视频播放系统**：基于HTML5 video标签，实现多清晰度切换、进度保存、水印叠加等功能。

3. **直播系统**：根据业务场景选择合适的流媒体协议（HLS/WebRTC），实现聊天室、连麦互动、直播回放等功能。

4. **学习进度**：完整追踪学生的学习轨迹，生成可视化报告，配合勋章系统激励学习。

5. **作业考试**：支持多种题型，自动批改客观题，生成错题本帮助学生巩固知识。

6. **性能优化**：通过CDN加速、视频预加载、骨架屏等技术手段提升用户体验。

7. **安全版权**：采用DRM数字版权管理、视频加密、防作弊机制保护内容安全。

在线教育是一个充满挑战和机遇的领域。希望本指南能够帮助开发者快速上手教育产品的开发，为用户提供更好的学习体验。
