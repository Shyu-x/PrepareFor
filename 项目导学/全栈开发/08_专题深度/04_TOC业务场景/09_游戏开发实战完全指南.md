## 一、游戏开发概述

### 1.1 H5游戏 vs 小游戏 vs 原生

在移动游戏领域，HTML5游戏（简称H5游戏）、微信小游戏和原生应用构成了三种主要的游戏形态。它们在技术架构、性能表现、发布渠道等方面存在显著差异。

**H5游戏**是基于Web技术开发的游戏，运行在浏览器环境中。H5游戏的最大优势是跨平台兼容性——一次开发即可在PC浏览器、移动浏览器、小程序等任何支持HTML5的环境中运行。H5游戏通过Canvas或WebGL进行渲染，借助JavaScript实现游戏逻辑。在技术实现上，H5游戏可以使用原生的Canvas 2D API进行2D渲染，也可以使用Phaser、Egret等游戏引擎来加速开发。H5游戏适合休闲类、益智类、社交分享类游戏，其性能受限于浏览器环境和设备硬件，但在中轻度游戏中表现良好。

**小游戏**特指运行在特定平台内嵌浏览器中的轻量级游戏，以微信小游戏为代表。小游戏使用JavaScript开发，但运行在双线程模型上——逻辑线程和渲染线程分离，这使得小游戏在性能上优于传统H5游戏。小游戏提供了丰富的平台API，如微信支付的虚拟支付、好友排行榜、分享到群聊等功能。小游戏的包体限制（通常为4-8MB）要求开发者对资源进行精细管理，采用CDN动态加载资源的方式来解决这一问题。小游戏适合需要社交传播、支付变现的中轻度游戏，如棋牌、休闲益智、io类游戏。

**原生应用**使用平台原生语言开发（如iOS的Swift/Objective-C、Android的Java/Kotlin），性能最优，可调用全部系统API，但开发成本高、维护难度大。原生应用需要通过应用商店分发，用户获取成本较高。对于需要高性能、高体验的游戏（如大型3D游戏、射击游戏），原生应用仍是最佳选择。

| 维度 | H5游戏 | 小游戏 | 原生应用 |
|------|--------|--------|----------|
| **开发语言** | JavaScript/TypeScript | JavaScript/TypeScript | Swift/Kotlin/Java |
| **渲染方式** | Canvas 2D / WebGL | Canvas 2D / WebGL | OpenGL ES / Metal |
| **性能** | 中等 | 中上 | 最优 |
| **包体限制** | 无 | 4-8MB | 无 |
| **发布渠道** | Web渠道 | 平台内分发 | 应用商店 |
| **社交能力** | 依赖Web API | 平台深度集成 | 需自行实现 |
| **更新方式** | 无需审核热更新 | 需审核热更新 | 版本更新 |

### 1.2 游戏引擎介绍

游戏引擎是游戏开发的核心工具，它封装了游戏开发中常用的功能，包括渲染引擎、物理引擎、音效系统、动画系统、资源管理等。选择合适的游戏引擎可以大幅提升开发效率。

**Phaser**是目前最流行的HTML5 2D游戏引擎之一。Phaser基于Pixi.js进行渲染，提供了丰富的2D游戏开发功能，包括精灵管理、动画系统、粒子效果、物理引擎集成等。Phaser的设计理念是"开箱即用"，它内置了Phaser.Physics子系统，可以集成Matter.js或Arcade物理引擎。Phaser的API设计清晰，文档完善，社区活跃，是学习HTML5游戏开发的优秀选择。Phaser适合开发休闲游戏、平台跳跃游戏、射击游戏、RPG游戏等2D游戏。

**Egret Engine**（白鹭引擎）是国内知名的HTML5游戏引擎，提供了完整的产品线，包括Egret Engine（2D引擎）、Egret 3D（3D引擎）、Egret Wing（可视化编辑器）等。Egret提供了完善的TypeScript支持，API设计符合国内开发者的习惯，提供了丰富的发布渠道支持，可发布到微信小游戏、抖音小游戏、H5等平台。Egret的配套工具链完善，包括EgretUIEditor（界面编辑器）、Res Depot（资源管理器）等。Egret适合需要发布到多个小游戏平台的国内开发者。

**Cocos Creator**是Cocos引擎的下一代产品，支持2D和3D游戏开发。Cocos Creator基于Cocos2d-x和Cocos3d-x引擎，提供了可视化的场景编辑器、组件化的开发模式、内置的物理引擎等。Cocos Creator支持发布到微信小游戏、抖音小游戏、百度小游戏、H5、iOS、Android等平台。Cocos Creator的TypeScript/API设计优秀，文档完善，是国内最受欢迎的HTML5游戏引擎之一。

**选择建议**：对于初学者，推荐从Phaser开始学习，因为它语法简洁、文档优秀、社区活跃。对于需要发布到微信小游戏的国内项目，可以考虑Egret或Cocos Creator，它们对国内平台的支持更完善。对于WebGL 3D游戏，Three.js是最流行的选择（将在后文详细介绍）。

### 1.3 游戏核心概念

无论使用哪种游戏引擎，都有一些核心概念是相通的，理解这些概念是游戏开发的基础。

**场景（Scene）**是游戏中的基本容器，用于组织游戏内容。一个游戏可以包含多个场景，如开始场景、游戏场景、结束场景等。场景管理是游戏状态控制的核心，开发者需要设计合理的场景切换逻辑。在Phaser中，场景通过继承Phaser.Scene类来实现，每个场景有自己的preload（预加载）、create（创建）、update（更新）生命周期方法。

```typescript
// Phaser中的场景定义示例
class BootScene extends Phaser.Scene {
    // 预加载资源
    preload(): void {
        this.load.image('logo', 'assets/logo.png');
        this.load.audio('bgm', 'assets/bgm.mp3');
    }

    // 创建场景内容
    create(): void {
        // 添加背景图
        this.add.image(400, 300, 'logo');

        // 播放背景音乐
        this.sound.play('bgm', { loop: true });

        // 点击进入游戏场景
        this.input.on('pointerdown', () => {
            this.scene.start('GameScene');
        });
    }
}

class GameScene extends Phaser.Scene {
    create(): void {
        this.add.text(400, 300, '游戏进行中', {
            fontSize: '48px',
            color: '#ffffff'
        }).setOrigin(0.5);
    }
}

// 游戏配置
const config: Phaser.Types.Core.GameConfig = {
    type: Phaser.AUTO,
    width: 800,
    height: 600,
    scene: [BootScene, GameScene], // 注册多个场景
    physics: {
        default: 'arcade',
        arcade: { gravity: { y: 300 }, debug: false }
    }
};

const game = new Phaser.Game(config);
```

**精灵（Sprite）**是游戏中最基本的可视对象，相当于电影中的演员。精灵是一张图片或者一帧动画，可以在游戏世界中移动、旋转、缩放、改变透明度等。精灵是游戏表现的核心载体，玩家角色、NPC、道具、背景元素等都可以用精灵来表示。在Phaser中，精灵通过this.add.sprite()或this.physics.add.sprite()创建，后者会为精灵添加物理特性。

```typescript
// 在GameScene中创建带物理特性的玩家精灵
class GameScene extends Phaser.Scene {
    private player!: Phaser.Physics.Arcade.Sprite;

    create(): void {
        // 创建玩家精灵（带物理引擎支持）
        this.player = this.physics.add.sprite(100, 450, 'player');

        // 设置精灵属性
        this.player.setBounce(0.2);       // 弹性
        this.player.setCollideWorldBounds(true); // 边界碰撞

        // 创建敌人精灵组
        this.enemies = this.physics.add.group();

        // 每隔1秒生成一个敌人
        this.time.addEvent({
            delay: 1000,
            callback: this.spawnEnemy,
            callbackScope: this,
            loop: true
        });

        // 设置碰撞检测
        this.physics.add.overlap(
            this.player,
            this.enemies,
            this.onHitEnemy,
            undefined,
            this
        );
    }

    spawnEnemy(): void {
        const enemy = this.enemies.create(
            Phaser.Math.Between(50, 750),
            -50,
            'enemy'
        );
        enemy.setVelocityY(200);
    }

    onHitEnemy(
        player: Phaser.Physics.Arcade.Sprite,
        enemy: Phaser.Physics.Arcade.Sprite
    ): void {
        // 闪烁效果
        this.tweens.add({
            targets: player,
            alpha: 0.5,
            duration: 100,
            yoyo: true,
            repeat: 3
        });

        // 销毁敌人
        enemy.destroy();
    }
}
```

**物理（Physics）**系统是游戏引擎中用于模拟现实物理效果的模块。物理系统可以处理重力、碰撞、摩擦力、弹性等效果，大大简化了游戏开发中对这些效果的实现。主流游戏引擎都内置了物理引擎，Phaser集成了Arcade Physics（轻量级）和Matter.js（高级）两种物理系统。Arcade Physics适合平台跳跃、发射弹丸等简单物理场景，Matter.js则适合需要精确物理模拟的复杂场景。

```typescript
// Phaser Arcade Physics配置示例
const config: Phaser.Types.Core.GameConfig = {
    type: Phaser.AUTO,
    width: 800,
    height: 600,
    physics: {
        // 指定使用的物理系统
        default: 'arcade',
        arcade: {
            // 重力加速度（像素/秒²）
            gravity: { y: 800 },
            // 是否显示调试信息
            debug: false
        }
    },
    scene: { preload, create, update }
};

// 创建具有物理特性的平台
function create(): void {
    // 静态物理体（不受重力影响，不会移动）
    const platforms = this.physics.add.staticGroup();

    // 地面
    platforms.create(400, 568, 'ground');

    // 悬浮平台
    platforms.create(600, 400, 'platform');
    platforms.create(100, 250, 'platform');

    // 玩家（动态物理体，受重力影响）
    player = this.physics.add.sprite(100, 450, 'player');
    player.setBounce(0.2);  // 跳跃时的弹性
    player.setCollideWorldBounds(true); // 与世界边界碰撞

    // 让玩家与平台产生碰撞
    this.physics.add.collider(player, platforms);
}
```

### 1.4 我的思考：游戏是前端的练兵场

游戏开发是前端技术集大成者的说法毫不夸张。一款游戏需要综合运用图形渲染、性能优化、网络通信、数据存储、用户交互等多种技术，这些技术能力都可以迁移到其他前端领域。

在图形渲染方面，游戏开发者需要深入理解Canvas API、WebGL着色器、帧率控制、精灵动画等，这些知识在数据可视化、图表库、交互式Web应用等领域同样重要。一个合格的前端工程师如果能够开发出流畅的60FPS游戏，那么开发普通的Web应用必然游刃有余。

在性能优化方面，游戏开发是性能敏感型应用，必须关注内存管理、CPU占用、GPU渲染、网络延迟等多个维度。游戏开发者养成的性能意识和方法论，可以直接应用于Web应用的性能优化。无论是React组件渲染优化，还是大型Web应用的首屏加载优化，游戏开发中积累的经验都能派上用场。

在架构设计方面，游戏需要管理复杂的状态、场景、角色、道具、网络同步等对象，这要求开发者具备良好的面向对象设计能力和架构能力。游戏中的设计模式（如ECS架构、状态机模式、对象池模式）在前端开发中同样广泛应用。

游戏开发的反馈循环极快——修改代码后立即可以看到效果，这种即时反馈对于学习新技术、调试复杂逻辑非常有帮助。很多前端开发者都是通过开发游戏入门前端的，游戏开发是学习前端技术的最佳起点。

## 二、物理引擎详解

### 2.1 Matter.js介绍

Matter.js是一个轻量级、模块化的2D物理引擎，使用纯JavaScript编写，可以独立运行，也可以集成到Phaser、PlayCanvas等游戏引擎中。Matter.js提供了刚体动力学、碰撞检测、约束系统等核心功能，API设计清晰，文档完善。

Matter.js的核心概念包括：

- **Engine（引擎）**：物理世界的主控制器，管理所有物理对象和模拟循环
- **World（世界）**：物理对象的容器，包含重力设置和所有刚体
- **Body（刚体）**：物理世界中的实体对象，可以是静态的（不移动）或动态的（受力和重力影响）
- **Constraint（约束）**：连接两个刚体的物理连接，如弹簧、绳索
- **Collision（碰撞）**：刚体之间的碰撞检测和响应

```typescript
// Matter.js基本使用示例
import Matter from 'matter-js';

// 创建引擎（物理世界的主控制器）
const engine = Matter.Engine.create({
    // 开启休眠优化，不活动的物体会进入休眠状态节省性能
    enableSleeping: true
});

// 获取物理世界
const world = engine.world;

// 设置世界重力（x方向0，y方向1，即正常向下）
world.gravity.y = 1;

// 获取渲染模块（用于调试，实际项目可用Canvas自己渲染）
const render = Matter.Render.create({
    element: document.getElementById('game-container')!,
    engine: engine,
    options: {
        width: 800,
        height: 600,
        wireframes: false,  // 使用彩色渲染而非线框
        background: '#1a1a2e'
    }
});

// 创建静态刚体（地面和墙壁）
const ground = Matter.Bodies.rectangle(400, 580, 800, 40, {
    isStatic: true,  // 静态刚体，不受重力影响
    render: { fillStyle: '#4a4a6a' }
});

const leftWall = Matter.Bodies.rectangle(20, 300, 40, 600, {
    isStatic: true,
    render: { fillStyle: '#4a4a6a' }
});

const rightWall = Matter.Bodies.rectangle(780, 300, 40, 600, {
    isStatic: true,
    render: { fillStyle: '#4a4a6a' }
});

// 创建动态刚体（可以移动的小球）
const ball = Matter.Bodies.circle(400, 100, 30, {
    restitution: 0.8,  // 弹性系数，0-1之间，越大越弹
    friction: 0.01,    // 摩擦系数
    density: 0.001,    // 密度，影响质量
    render: {
        fillStyle: '#ff6b6b',
        strokeStyle: '#ffffff',
        lineWidth: 2
    }
});

// 创建矩形刚体（木板）
const plank = Matter.Bodies.rectangle(400, 400, 200, 20, {
    angle: Math.PI * 0.1,  // 初始角度（弧度）
    render: {
        fillStyle: '#4ecdc4',
        strokeStyle: '#ffffff',
        lineWidth: 2
    }
});

// 将所有刚体添加到世界
Matter.World.add(world, [ground, leftWall, rightWall, ball, plank]);

// 添加鼠标控制（可以拖拽物体）
const mouse = Matter.Mouse.create(render.canvas);
const mouseConstraint = Matter.MouseConstraint.create(engine, {
    mouse: mouse,
    constraint: {
        stiffness: 0.2,  // 约束刚度
        render: { visible: false }
    }
});
Matter.World.add(world, mouseConstraint);

// 保持鼠标与渲染同步
render.mouse = mouse;

// 启动物理模拟循环
Matter.Runner.run(engine);

// 启动渲染循环
Matter.Render.run(render);

// 监听碰撞事件
Matter.Events.on(engine, 'collisionStart', (event) => {
    event.pairs.forEach((pair) => {
        const bodyA = pair.bodyA;
        const bodyB = pair.bodyB;

        // 判断是否是球与地面的碰撞
        if ((bodyA.label === 'ball' && bodyB.isStatic) ||
            (bodyB.label === 'ball' && bodyA.isStatic)) {
            console.log('球落地了！');
        }
    });
});

// 自定义渲染（替代Matter.js内置渲染，更灵活）
function customRender(): void {
    const canvas = render.canvas;
    const ctx = render.context;
    const bodies = Matter.Composite.allBodies(world);

    // 清除画布
    ctx.fillStyle = '#1a1a2e';
    ctx.fillRect(0, 0, canvas.width, canvas.height);

    // 绘制所有刚体
    bodies.forEach((body) => {
        ctx.beginPath();
        const vertices = body.vertices;
        ctx.moveTo(vertices[0].x, vertices[0].y);

        for (let i = 1; i < vertices.length; i++) {
            ctx.lineTo(vertices[i].x, vertices[i].y);
        }

        ctx.closePath();

        // 根据刚体类型设置不同的填充色
        if (body.isStatic) {
            ctx.fillStyle = '#4a4a6a';
        } else if (body.label === 'ball') {
            ctx.fillStyle = '#ff6b6b';
        } else {
            ctx.fillStyle = '#4ecdc4';
        }

        ctx.fill();
        ctx.strokeStyle = '#ffffff';
        ctx.lineWidth = 2;
        ctx.stroke();
    });

    requestAnimationFrame(customRender);
}

customRender();
```

### 2.2 碰撞检测

碰撞检测是物理引擎的核心功能之一，用于判断两个刚体是否发生接触或重叠。Matter.js提供了多种碰撞检测机制，包括事件监听方式和主动查询方式。

**事件监听方式**通过订阅物理引擎的碰撞事件来检测碰撞，这种方式适合需要响应碰撞并执行特定逻辑的场景，如角色受伤、道具收集、触发机关等。

```typescript
// Matter.js碰撞检测完整示例
import Matter from 'matter-js';

// 初始化引擎和世界
const engine = Matter.Engine.create();
const world = engine.world;

// 玩家刚体
const player = Matter.Bodies.circle(100, 100, 25, {
    label: 'player',  // 标签，用于识别刚体类型
    restitution: 0.2, // 弹性
    friction: 0.8,    // 摩擦力
    render: { fillStyle: '#3498db' }
});

// 敌人刚体
const enemy = Matter.Bodies.circle(300, 100, 25, {
    label: 'enemy',
    restitution: 0.2,
    friction: 0.8,
    render: { fillStyle: '#e74c3c' }
});

// 金币刚体（传感器模式，不产生物理碰撞，只检测重叠）
const coin = Matter.Bodies.circle(500, 100, 15, {
    label: 'coin',
    isSensor: true,  // 传感器模式，不阻挡其他物体
    render: { fillStyle: '#f1c40f' }
});

// 地面
const ground = Matter.Bodies.rectangle(400, 580, 800, 40, {
    isStatic: true,
    label: 'ground',
    render: { fillStyle: '#2ecc71' }
});

// 钉子（造成伤害的障碍物）
const spike = Matter.Bodies.polygon(600, 550, 3, 30, {
    isStatic: true,
    label: 'spike',
    render: { fillStyle: '#e74c3c' }
});

// 将刚体添加到世界
Matter.World.add(world, [player, enemy, coin, ground, spike]);

// 碰撞事件监听
Matter.Events.on(engine, 'collisionStart', (event) => {
    const pairs = event.pairs;

    for (const pair of pairs) {
        const bodyA = pair.bodyA;
        const bodyB = pair.bodyB;

        // 玩家与金币碰撞 - 收集金币
        if ((bodyA.label === 'player' && bodyB.label === 'coin') ||
            (bodyB.label === 'player' && bodyA.label === 'coin')) {
            const coinBody = bodyA.label === 'coin' ? bodyA : bodyB;
            collectCoin(coinBody);
        }

        // 玩家与敌人碰撞 - 受伤
        if ((bodyA.label === 'player' && bodyB.label === 'enemy') ||
            (bodyB.label === 'player' && bodyA.label === 'enemy')) {
            playerHit();
        }

        // 玩家与钉子碰撞 - 严重受伤
        if ((bodyA.label === 'player' && bodyB.label === 'spike') ||
            (bodyB.label === 'player' && bodyA.label === 'spike')) {
            playerHurt();
        }
    }
});

// 碰撞持续检测（每帧都会触发）
Matter.Events.on(engine, 'collisionActive', (event) => {
    const pairs = event.pairs;

    for (const pair of pairs) {
        const bodyA = pair.bodyA;
        const bodyB = pair.bodyB;

        // 玩家站在地面上（持续触发）
        if ((bodyA.label === 'player' && bodyB.label === 'ground') ||
            (bodyB.label === 'player' && bodyA.label === 'ground')) {
            player.grounded = true;
        }
    }
});

// 碰撞结束检测
Matter.Events.on(engine, 'collisionEnd', (event) => {
    const pairs = event.pairs;

    for (const pair of pairs) {
        const bodyA = pair.bodyA;
        const bodyB = pair.bodyB;

        // 玩家离开地面
        if ((bodyA.label === 'player' && bodyB.label === 'ground') ||
            (bodyB.label === 'player' && bodyA.label === 'ground')) {
            player.grounded = false;
        }
    }
});

// 碰撞响应处理函数
function collectCoin(coin: Matter.Body): void {

        Matter.Composite.allBodies(world),
        region
    );
```

### 2.3 刚体动力学

刚体动力学是研究刚体在力作用下的运动规律的学科，在游戏物理中用于模拟物体的真实运动。Matter.js提供了丰富的刚体动力学功能，包括力、扭矩、速度、加速度等物理量的控制。

```typescript
// Matter.js刚体动力学详解
import Matter from 'matter-js';

const engine = Matter.Engine.create();
const world = engine.world;

// 创建各种类型的刚体
// 1. 圆形刚体 - 适合球形对象
const circle = Matter.Bodies.circle(100, 100, 40, {
    label: 'circle',
    // 质量相关
    mass: 1,
    density: 0.001,  // 密度，密度越大质量越大
    // 运动相关
    restitution: 0.8,  // 弹性系数
    friction: 0.01,     // 表面摩擦
    frictionAir: 0.001, // 空气阻力
    frictionStatic: 0.5, // 静摩擦
    // 初始状态
    isStatic: false,
    isSensor: false,  // 是否为传感器
    isSleeping: false, // 是否处于休眠状态
    sleepThreshold: 60, // 休眠阈值（静止多少帧后休眠）
    render: { fillStyle: '#e74c3c' }
});

// 2. 矩形刚体 - 适合箱子、平台
const box = Matter.Bodies.rectangle(300, 100, 80, 80, {
    label: 'box',
    // 角度相关
    angle: Math.PI / 4,  // 初始旋转角度（弧度）
    // 碰撞筛选
    collisionFilter: {
        group: 0,      // 碰撞组，相同组内物体不碰撞
        category: 0x0001, // 碰撞类别
        mask: 0xFFFF   // 碰撞掩码，决定可以和哪些类别碰撞
    },
    render: { fillStyle: '#3498db' }
});

// 3. 多边形刚体 - 三角钉、六边形等
const triangle = Matter.Bodies.polygon(500, 100, 3, 40, {
    label: 'triangle',
    render: { fillStyle: '#2ecc71' }
});

// 4. 梯形刚体
const trapezoid = Matter.Bodies.trapezoid(700, 100, 100, 60, 0.5, {
    label: 'trapezoid',
    render: { fillStyle: '#9b59b6' }
});

// 添加到世界
Matter.World.add(world, [circle, box, triangle, trapezoid]);

// ====== 力与运动控制 ======

// 施加重力（每个刚体可以单独设置重力系数）
circle.gravityScale = 2;  // 2倍重力

// 对刚体施加力（向量形式）
// Force.vector(x, y) - 创建一个力向量
Matter.Body.applyForce(circle, circle.position, { x: 0.05, y: 0 });

// 施加力到指定点（产生扭矩效果）
Matter.Body.applyForce(
    box,
    { x: box.position.x + 20, y: box.position.y },  // 力的作用点
    { x: 0, y: -0.1 }  // 力的向量
);

// 设置速度（直接设置，绕过力和质量）
Matter.Body.setVelocity(circle, { x: 10, y: 0 });
Matter.Body.setVelocity(box, { x: 0, y: 5 });

// 设置角速度（旋转速度）
Matter.Body.setAngularVelocity(box, 0.1);  // 正值为顺时针旋转

// 设置角度
Matter.Body.setAngle(box, Math.PI / 2);  // 旋转到90度
Matter.Body.rotate(box, Math.PI / 4);    // 当前角度基础上再旋转45度

// 设置位置
Matter.Body.setPosition(circle, { x: 200, y: 200 });
Matter.Body.setPosition(box, { x: 300, y: 300 }, true);  // 第二个参数为true会同时设置角度

// 设置质量和惯性
Matter.Body.setMass(circle, 2);  // 设置质量为2
Matter.Body.setInertia(box, 1000);  // 设置转动惯性

// ====== 约束系统 ======

// 创建弹簧约束
const spring = Matter.Constraint.create({
    bodyA: circle,      // 连接刚体A
    bodyB: box,         // 连接刚体B
    stiffness: 0.01,   // 刚度（0-1），越小越像弹簧
    damping: 0.1,       // 阻尼
    length: 100,        // 静止时的长度
    render: {
        visible: true,
        lineWidth: 2,
        strokeStyle: '#f39c12'
    }
});

// 创建固定距离约束
const rope = Matter.Constraint.create({
    bodyA: circle,
    pointA: { x: 0, y: 0 },   // 在A上的连接点（本地坐标）
    bodyB: box,
    pointB: { x: 0, y: 0 },   // 在B上的连接点（本地坐标）
    stiffness: 1,   // 刚度为1，即固定距离
    length: 150,
    render: {
        visible: true,
        lineWidth: 4,
        strokeStyle: '#7f8c8d'
    }
});

// 创建钉子约束（固定点）
const pin = Matter.Constraint.create({
    bodyA: triangle,
    pointA: { x: 0, y: 0 },
    pointB: { x: 500, y: 200 },  // 固定在画布某点
    stiffness: 1,
    length: 0,  // 长度为0表示固定点
    render: {
        visible: true,
        lineWidth: 2,
        strokeStyle: '#e74c3c'
    }
});

Matter.World.add(world, [spring, rope, pin]);

// ====== 刚体状态查询 ======

// 获取刚体当前速度
console.log('Circle速度:', circle.velocity);
console.log('Box速度:', box.velocity);

// 获取刚体当前角速度
console.log('Circle角速度:', circle.angularVelocity);
console.log('Box角速度:', box.angularVelocity);

// 获取刚体当前动能（可用于实现破坏效果等）
const kineticEnergy = Matter.Body.getKineticEnergy(circle);
console.log('Circle动能:', kineticEnergy);

// 获取刚体的惯性张量
const inertia = Matter.Body.getInertia(box);
console.log('Box惯性:', inertia);

// 判断刚体是否处于睡眠状态（静止）
if (circle.isSleeping) {
    console.log('Circle正在休眠');
}

// 强制唤醒刚体
Matter.Body.set(circle, 'isSleeping', false);
Matter.Sleeping.run(circle);

// ====== 复合刚体 ======

// 创建一个由多个刚体组成的复合物体（如交通工具）
const vehicle = Matter.Body.create({
    parts: [
        // 车体（矩形）
        Matter.Bodies.rectangle(0, 0, 160, 40, { render: { fillStyle: '#34495e' } }),
        // 车轮安装位1
        Matter.Bodies.circle(-50, 25, 20, { render: { fillStyle: '#2c3e50' } }),
        // 车轮安装位2
        Matter.Bodies.circle(50, 25, 20, { render: { fillStyle: '#2c3e50' } })
    ],
    label: 'vehicle'
});

// 给车轮添加实际的旋转刚体
const wheelL = Matter.Bodies.circle(-50, 30, 20, {
    friction: 0.9,
    render: { fillStyle: '#1abc9c' }
});

const wheelR = Matter.Bodies.circle(50, 30, 20, {
    friction: 0.9,
    render: { fillStyle: '#1abc9c' }
});

// 添加悬架约束
const suspensionL = Matter.Constraint.create({
    bodyA: vehicle,
    pointA: { x: -50, y: 20 },
    bodyB: wheelL,
    stiffness: 0.5,
    damping: 0.3,
    length: 10
});

const suspensionR = Matter.Constraint.create({
    bodyA: vehicle,
    pointA: { x: 50, y: 20 },
    bodyB: wheelR,
    stiffness: 0.5,
    damping: 0.3,
    length: 10
});

Matter.World.add(world, [vehicle, wheelL, wheelR, suspensionL, suspensionR]);

// 车辆控制 - 通过轮子施加扭矩
function accelerate(): void {
    // 给右轮施加扭矩（向前）
    Matter.Body.setAngularVelocity(wheelR, 0.3);
    Matter.Body.setAngularVelocity(wheelL, 0.3);
}

function brake(): void {
    // 降低轮子角速度
    Matter.Body.setAngularVelocity(wheelR, wheelR.angularVelocity * 0.9);
    Matter.Body.setAngularVelocity(wheelL, wheelL.angularVelocity * 0.9);
}

function steer(direction: 'left' | 'right'): void {
    // 差速转向
    if (direction === 'left') {
        Matter.Body.setAngularVelocity(wheelL, wheelL.angularVelocity - 0.1);
        Matter.Body.setAngularVelocity(wheelR, wheelR.angularVelocity + 0.05);
    } else {
        Matter.Body.setAngularVelocity(wheelL, wheelL.angularVelocity + 0.05);
        Matter.Body.setAngularVelocity(wheelR, wheelR.angularVelocity - 0.1);
    }
}
```

### 2.4 实战：物理小球

通过一个完整的物理小球示例来巩固Matter.js的使用。本示例将创建一个带有重力、碰撞、弹跳效果的物理小球，并实现用户交互功能。

```typescript
// 物理小球实战 - 完整的物理模拟演示
import Matter from 'matter-js';

// ==================== 配置 ====================
const CONFIG = {
    width: 800,
    height: 600,
    gravity: 1,
    ballRadius: 25,
    ballRestitution: 0.8,  // 弹性系数
    ballFriction: 0.01,
    ballDensity: 0.001,
    colors: {
        background: '#0f0f23',
        ground: '#1a1a2e',
        platform: '#16213e',
        ball: '#e94560',
        ballStroke: '#ffffff',
        text: '#ffffff'
    }
};

// ==================== 初始化 ====================
class PhysicsBallGame {
    private engine!: Matter.Engine;
    private render!: Matter.Render;
    private runner!: Matter.Runner;
    private world!: Matter.World;
    private ball!: Matter.Body;
    private platforms!: Matter.Body[];
    private score: number = 0;
    private lives: number = 3;
    private isGameOver: boolean = false;
    private canvas!: HTMLCanvasElement;
    private ctx!: CanvasRenderingContext2D;

    constructor() {
        this.initEngine();
        this.createWorld();
        this.createBall();
        this.createObstacles();
        this.setupControls();
        this.startGameLoop();
        this.setupUI();
    }

    // 初始化物理引擎
    private initEngine(): void {
        // 创建引擎，enableSleeping优化性能
        this.engine = Matter.Engine.create({
            enableSleeping: true
        });
        this.world = this.engine.world;

        // 设置重力
        this.world.gravity.y = CONFIG.gravity;

        // 创建自定义渲染器
        this.canvas = document.createElement('canvas');
        this.canvas.width = CONFIG.width;
        this.canvas.height = CONFIG.height;
        this.canvas.style.background = CONFIG.colors.background;
        this.canvas.style.borderRadius = '12px';
        this.canvas.style.boxShadow = '0 0 40px rgba(233, 69, 96, 0.3)';
        document.getElementById('game-container')?.appendChild(this.canvas);
        this.ctx = this.canvas.getContext('2d')!;

        // 创建运行器
        this.runner = Matter.Runner.create();
    }

    // 创建游戏世界（边界）
    private createWorld(): void {
        const thickness = 50;

        // 地面
        const ground = Matter.Bodies.rectangle(
            CONFIG.width / 2,
            CONFIG.height - thickness / 2,
            CONFIG.width,
            thickness,
            {
                isStatic: true,
                label: 'ground',
                render: { fillStyle: CONFIG.colors.ground }
            }
        );

        // 左墙
        const leftWall = Matter.Bodies.rectangle(
            -thickness / 2,
            CONFIG.height / 2,
            thickness,
            CONFIG.height,
            {
                isStatic: true,
                label: 'wall',
                render: { fillStyle: CONFIG.colors.ground }
            }
        );

        // 右墙
        const rightWall = Matter.Bodies.rectangle(
            CONFIG.width + thickness / 2,
            CONFIG.height / 2,
            thickness,
            CONFIG.height,
            {
                isStatic: true,
                label: 'wall',
                render: { fillStyle: CONFIG.colors.ground }
            }
        );

        Matter.World.add(this.world, [ground, leftWall, rightWall]);
    }

    // 创建小球
    private createBall(): void {
        this.ball = Matter.Bodies.circle(
            CONFIG.width / 2,
            100,
            CONFIG.ballRadius,
            {
                label: 'ball',
                restitution: CONFIG.ballRestitution,
                friction: CONFIG.ballFriction,
                density: CONFIG.ballDensity,
                render: {
                    fillStyle: CONFIG.colors.ball,
                    strokeStyle: CONFIG.colors.ballStroke,
                    lineWidth: 3
                }
            }
        );

        Matter.World.add(this.world, this.ball);

        // 监听碰撞事件
        Matter.Events.on(this.engine, 'collisionStart', (event) => {
            event.pairs.forEach((pair) => {
                const { bodyA, bodyB } = pair;

                // 球与障碍物碰撞
                if ((bodyA.label === 'ball' && bodyB.label === 'obstacle') ||
                    (bodyB.label === 'ball' && bodyA.label === 'obstacle')) {

                    // 根据碰撞速度增加分数
                    const speed = Math.sqrt(
                        this.ball.velocity.x ** 2 +
                        this.ball.velocity.y ** 2
                    );

                    if (speed > 5) {
                        this.addScore(Math.floor(speed * 2));
                        this.createParticles(this.ball.position.x, this.ball.position.y);
                    }
                }

                // 球落地
                if ((bodyA.label === 'ball' && bodyB.label === 'ground') ||
                    (bodyB.label === 'ball' && bodyA.label === 'ground')) {
                    this.onBallHitGround();
                }
            });
        });
    }

    // 创建障碍物
    private createObstacles(): void {
        this.platforms = [];

        // 固定平台
        const platformPositions = [
            { x: 200, y: 450, w: 150, h: 20, angle: 0 },
            { x: 600, y: 450, w: 150, h: 20, angle: 0 },
            { x: 400, y: 350, w: 200, h: 20, angle: Math.PI / 6 },
            { x: 150, y: 280, w: 120, h: 20, angle: -Math.PI / 8 },
            { x: 650, y: 280, w: 120, h: 20, angle: Math.PI / 8 },
            { x: 400, y: 180, w: 180, h: 20, angle: 0 }
        ];

        platformPositions.forEach((pos, index) => {
            const platform = Matter.Bodies.rectangle(
                pos.x, pos.y, pos.w, pos.h,
                {
                    isStatic: true,
                    angle: pos.angle,
                    label: 'platform',
                    chamfer: { radius: 5 },  // 圆角
                    render: {
                        fillStyle: CONFIG.colors.platform,
                        strokeStyle: '#0f3460',
                        lineWidth: 2
                    }
                }
            );
            this.platforms.push(platform);
        });

        // 移动平台
        const movingPlatform = Matter.Bodies.rectangle(
            400, 520, 100, 15,
            {
                isStatic: true,
                label: 'movingPlatform',
                chamfer: { radius: 5 },
                render: {
                    fillStyle: '#533483',
                    strokeStyle: '#e94560',
                    lineWidth: 2
                }
            }
        );
        this.platforms.push(movingPlatform);

        Matter.World.add(this.world, this.platforms);

        // 移动平台动画
        this.animateMovingPlatform(movingPlatform);
    }

    // 移动平台动画
    private animateMovingPlatform(platform: Matter.Body): void {
        let direction = 1;
        const speed = 2;
        const range = 200;

        Matter.Events.on(this.engine, 'beforeUpdate', () => {
            const x = platform.position.x + speed * direction;

            if (x > 400 + range || x < 400 - range) {
                direction *= -1;
            }

            Matter.Body.setPosition(platform, { x, y: platform.position.y });
        });
    }

    // 设置鼠标/触摸控制
    private setupControls(): void {
        Matter.Events.on(this.engine, 'beforeUpdate', () => {
            // 持续向右的力（模拟重力或风）
            // Matter.Body.applyForce(this.ball, this.ball.position, { x: 0.0001, y: 0 });
        });

        // 点击屏幕给球施加向上的力
        this.canvas.addEventListener('click', (e) => {
            if (this.isGameOver) {
                this.restart();
                return;
            }

            const rect = this.canvas.getBoundingClientRect();
            const x = e.clientX - rect.left;
            const y = e.clientY - rect.top;

            // 计算方向
            const dx = x - this.ball.position.x;
            const dy = y - this.ball.position.y;
            const distance = Math.sqrt(dx * dx + dy * dy);

            // 施加冲力
            const forceMagnitude = 0.05;
            Matter.Body.applyForce(this.ball, this.ball.position, {
                x: (dx / distance) * forceMagnitude,
                y: (dy / distance) * forceMagnitude
            });

            this.createParticles(x, y, 5);
        });

        // 触摸支持
        this.canvas.addEventListener('touchstart', (e) => {
            e.preventDefault();
            const touch = e.touches[0];
            const rect = this.canvas.getBoundingClientRect();
            const x = touch.clientX - rect.left;
            const y = touch.clientY - rect.top;

            const dx = x - this.ball.position.x;
            const dy = y - this.ball.position.y;
            const distance = Math.sqrt(dx * dx + dy * dy);

            const forceMagnitude = 0.08;
            Matter.Body.applyForce(this.ball, this.ball.position, {
                x: (dx / distance) * forceMagnitude,
                y: (dy / distance) * forceMagnitude
            });
        });
    }

    // 粒子效果
    private particles: Array<{ x: number; y: number; vx: number; vy: number; life: number; color: string }> = [];

    private createParticles(x: number, y: number, count: number = 10): void {
        for (let i = 0; i < count; i++) {
            this.particles.push({
                x,
                y,
                vx: (Math.random() - 0.5) * 10,
                vy: (Math.random() - 0.5) * 10,
                life: 1,
                color: Math.random() > 0.5 ? CONFIG.colors.ball : '#ffffff'
            });
        }
    }

    private updateParticles(): void {
        this.particles = this.particles.filter(p => {
            p.x += p.vx;
            p.y += p.vy;
            p.vy += 0.2;  // 重力
            p.life -= 0.02;
            return p.life > 0;
        });
    }

    // 开始游戏循环
    private startGameLoop(): void {
        Matter.Runner.run(this.runner, this.engine);
        this.render();
    }

    // 渲染循环
    private render(): void {
        requestAnimationFrame(() => this.render());

        const ctx = this.ctx;
        const bodies = Matter.Composite.allBodies(this.world);

        // 清除画布
        ctx.fillStyle = CONFIG.colors.background;
        ctx.fillRect(0, 0, CONFIG.width, CONFIG.height);

        // 绘制所有刚体
        bodies.forEach((body) => {
            if (body.label === 'ball') {
                // 绘制小球（带渐变）
                const gradient = ctx.createRadialGradient(
                    body.position.x - 5, body.position.y - 5, 0,
                    body.position.x, body.position.y, CONFIG.ballRadius
                );
                gradient.addColorStop(0, '#ff8a8a');
                gradient.addColorStop(1, CONFIG.colors.ball);

                ctx.beginPath();
                ctx.arc(body.position.x, body.position.y, CONFIG.ballRadius, 0, Math.PI * 2);
                ctx.fillStyle = gradient;
                ctx.fill();
                ctx.strokeStyle = CONFIG.colors.ballStroke;
                ctx.lineWidth = 3;
                ctx.stroke();

                // 绘制速度指示器
                this.drawVelocityIndicator(ctx, body);
            } else if (body.label === 'platform' || body.label === 'movingPlatform') {
                // 绘制平台
                ctx.save();
                ctx.translate(body.position.x, body.position.y);
                ctx.rotate(body.angle);

                const width = 150;  // 简化处理
                const height = 20;

                ctx.beginPath();
                ctx.roundRect(-width / 2, -height / 2, width, height, 5);
                ctx.fillStyle = body.render.fillStyle;
                ctx.fill();
                ctx.strokeStyle = body.render.strokeStyle;
                ctx.lineWidth = 2;
                ctx.stroke();

                ctx.restore();
            } else if (body.label === 'ground') {
                // 绘制地面
                ctx.fillStyle = CONFIG.colors.ground;
                ctx.fillRect(0, CONFIG.height - 50, CONFIG.width, 50);
            }
        });

        // 绘制粒子
        this.updateParticles();
        this.particles.forEach(p => {
            ctx.beginPath();
```

## 三、Canvas游戏开发

### 3.1 游戏循环

游戏循环是游戏的核心，它驱动着游戏的每一帧更新和渲染。一个典型的游戏循环包括以下阶段：处理输入、更新游戏状态、渲染画面。游戏循环需要以稳定的帧率运行，通常是60FPS（每帧约16.67毫秒）。

```typescript
// 基础游戏循环实现
class GameLoop {
    // 帧率配置
    private targetFPS: number = 60;
    private frameInterval: number = 1000 / this.targetFPS;  // 每帧时间（毫秒）

    // 时间追踪
    private lastTime: number = 0;       // 上一帧的时间戳
    private deltaTime: number = 0;     // 两帧之间的时间差
    private accumulator: number = 0;  // 时间累加器（用于固定时间步长）

    // 运行状态
    private isRunning: boolean = false;
    private animationFrameId: number = 0;

    // 回调函数
    private updateCallback: (deltaTime: number) => void = () => {};
    private renderCallback: () => void = () => {};

    // 固定的物理更新时间步长（防止高速碰撞穿透）
    private fixedTimeStep: number = 1000 / 120;  // 120Hz物理更新
    private maxSubSteps: number = 5;  // 最大子步数

    constructor() {
        // 绑定this
        this.gameLoop = this.gameLoop.bind(this);
    }

    // 设置更新回调
    onUpdate(callback: (deltaTime: number) => void): void {
        this.updateCallback = callback;
    }

    // 设置渲染回调
    onRender(callback: () => void): void {
        this.renderCallback = callback;
    }

    // 启动游戏循环
    start(): void {
        if (this.isRunning) return;

        this.isRunning = true;
        this.lastTime = performance.now();
        this.accumulator = 0;

        // 启动循环
        this.animationFrameId = requestAnimationFrame(this.gameLoop);

        console.log(`游戏循环启动，目标帧率: ${this.targetFPS}FPS`);
    }

    // 停止游戏循环
    stop(): void {
        this.isRunning = false;
        if (this.animationFrameId) {
            cancelAnimationFrame(this.animationFrameId);
        }
        console.log('游戏循环已停止');
    }

    // 主游戏循环
    private gameLoop(currentTime: number): void {
        if (!this.isRunning) return;

        // 计算时间差
        this.deltaTime = currentTime - this.lastTime;
        this.lastTime = currentTime;

        // 防止时间差过大（浏览器标签页切换时可能出现）
        if (this.deltaTime > 250) {
            this.deltaTime = 250;
        }

        // 累加时间
        this.accumulator += this.deltaTime;

        // 固定时间步长的物理更新
        // 这样可以保证物理模拟的确定性，不受帧率影响
        let steps = 0;
        while (this.accumulator >= this.fixedTimeStep && steps < this.maxSubSteps) {
            // 以固定的步长更新游戏逻辑
            this.updateCallback(this.fixedTimeStep / 1000);  // 转换为秒
            this.accumulator -= this.fixedTimeStep;
            steps++;
        }

        // 清空剩余时间（可选，也可以保留用于插值）
        // this.accumulator = 0;

        // 计算插值因子用于平滑渲染
        const alpha = this.accumulator / this.fixedTimeStep;

        // 渲染（使用插值使动画更平滑）
        this.renderCallback(alpha);

        // 调度下一帧
        this.animationFrameId = requestAnimationFrame(this.gameLoop);
    }

    // 获取当前FPS
    getFPS(): number {
        return this.deltaTime > 0
            ? Math.round(1000 / this.deltaTime)
            : this.targetFPS;
    }
}

// ==================== 游戏类示例 ====================

interface Vector2D {
    x: number;
    y: number;
}

interface GameObject {
    position: Vector2D;
    velocity: Vector2D;
    size: Vector2D;
    update(dt: number): void;
    render(ctx: CanvasRenderingContext2D): void;

// 玩家类
class Player implements GameObject {
    position: Vector2D;
    velocity: Vector2D;
    size: Vector2D = { x: 50, y: 50 };
    color: string = '#3498db';
    speed: number = 300;  // 像素/秒

    // 输入状态
    private keys: Set<string> = new Set();

    constructor(x: number, y: number) {
        this.position = { x, y };
        this.velocity = { x: 0, y: 0 };

        // 绑定输入处理
        this.setupInput();
    }

    setupInput(): void {
        window.addEventListener('keydown', (e) => {
            this.keys.add(e.key.toLowerCase());
        });

        window.addEventListener('keyup', (e) => {
            this.keys.delete(e.key.toLowerCase());
        });
    }

    update(dt: number): void {
        // 根据输入计算速度
        this.velocity.x = 0;
        this.velocity.y = 0;

        if (this.keys.has('w') || this.keys.has('arrowup')) {
            this.velocity.y = -this.speed;
        }
        if (this.keys.has('s') || this.keys.has('arrowdown')) {
            this.velocity.y = this.speed;
        }
        if (this.keys.has('a') || this.keys.has('arrowleft')) {
            this.velocity.x = -this.speed;
        }
        if (this.keys.has('d') || this.keys.has('arrowright')) {
            this.velocity.x = this.speed;
        }

        // 对角线移动时归一化速度
        if (this.velocity.x !== 0 && this.velocity.y !== 0) {
            const factor = 1 / Math.sqrt(2);
            this.velocity.x *= factor;
            this.velocity.y *= factor;
        }

        // 更新位置
        this.position.x += this.velocity.x * dt;
        this.position.y += this.velocity.y * dt;
    }

    render(ctx: CanvasRenderingContext2D): void {
        ctx.fillStyle = this.color;
        ctx.fillRect(
            this.position.x - this.size.x / 2,
            this.position.y - this.size.y / 2,
            this.size.x,
            this.size.y
        );

        // 绘制方向指示器
        if (this.velocity.x !== 0 || this.velocity.y !== 0) {
            const len = 20;
            const nx = this.velocity.x / Math.sqrt(this.velocity.x ** 2 + this.velocity.y ** 2);
            const ny = this.velocity.y / Math.sqrt(this.velocity.x ** 2 + this.velocity.y ** 2);

            ctx.beginPath();
            ctx.moveTo(this.position.x, this.position.y);
            ctx.lineTo(
                this.position.x + nx * len,
                this.position.y + ny * len
            );
            ctx.strokeStyle = '#ffffff';
            ctx.lineWidth = 3;
            ctx.stroke();
        }
    }
}

            this.position.y,

            Math.PI * 2
        );
        ctx.fill();
    }

        // 更新敌人

        // 检测碰撞

        // 更新分数（存活时间）

    checkCollisions(): void {
        // 简化的碰撞检测（矩形碰撞）
        const playerRect = {
            left: this.player.position.x - this.player.size.x / 2,
            right: this.player.position.x + this.player.size.x / 2,
            top: this.player.position.y - this.player.size.y / 2,
            bottom: this.player.position.y + this.player.size.y / 2
        };

            const enemyRect = {
                left: enemy.position.x - enemy.size.x / 2,
                right: enemy.position.x + enemy.size.x / 2,
                top: enemy.position.y - enemy.size.y / 2,
                bottom: enemy.position.y + enemy.size.y / 2
            };

            // AABB碰撞检测
            if (playerRect.left < enemyRect.right &&
                playerRect.right > enemyRect.left &&
                playerRect.top < enemyRect.bottom &&
                playerRect.bottom > enemyRect.top) {
```

### 3.2 精灵绘制与动画系统

精灵是游戏中最基本的视觉元素，而动画则是让游戏角色活起来的关键。精灵动画通常由多帧图片组成，通过快速切换帧来产生动画效果。

```typescript
// 精灵与动画系统实现
interface SpriteFrame {
    image: HTMLImageElement | HTMLCanvasElement;
    duration: number;  // 帧持续时间（毫秒）
}

interface AnimationConfig {
    name: string;
    frames: SpriteFrame[];
    loop: boolean;
    onComplete?: () => void;
}

// 精灵类
class Sprite {
    public position: { x: number; y: number };
    public scale: { x: number; y: number } = { x: 1, y: 1 };
    public rotation: number = 0;  // 弧度
    public opacity: number = 1;
    public anchor: { x: number; y: number } = { x: 0.5, y: 0.5 };  // 锚点（0-1）
    public flipX: boolean = false;
    public flipY: boolean = false;
    public visible: boolean = true;

    private currentFrame: number = 0;
    private image: HTMLImageElement;
    private frameWidth: number;
    private frameHeight: number;

    constructor(imageSrc: string, frameWidth: number, frameHeight: number) {
        this.position = { x: 0, y: 0 };
        this.frameWidth = frameWidth;
        this.frameHeight = frameHeight;

        // 加载图片
        this.image = new Image();
        this.image.src = imageSrc;
    }

    setFrame(frame: number): void {
        this.currentFrame = Math.max(0, Math.min(frame, this.getTotalFrames() - 1));
    }

    getTotalFrames(): number {
        // 假设图片是水平排列的帧
        return Math.floor(this.image.width / this.frameWidth);
    }

    update(dt: number): void {
        // 子类重写
    }

    render(ctx: CanvasRenderingContext2D): void {
        if (!this.visible || !this.image.complete) return;

        ctx.save();

        // 应用变换
        ctx.translate(this.position.x, this.position.y);
        ctx.rotate(this.rotation);
        ctx.scale(
            this.scale.x * (this.flipX ? -1 : 1),
            this.scale.y * (this.flipY ? -1 : 1)
        );
        ctx.globalAlpha = this.opacity;

        // 计算帧的位置
        const frameX = this.currentFrame * this.frameWidth;

        // 绘制精灵（以锚点为中心）
        ctx.drawImage(
            this.image,
            frameX, 0,                           // 源图像中的位置
            this.frameWidth, this.frameHeight,    // 源图像中要使用的区域
            -this.frameWidth * this.anchor.x,    // 目标画布位置
            -this.frameHeight * this.anchor.y,
            this.frameWidth, this.frameHeight
        );

        ctx.restore();
    }
}

// 动画播放器
class AnimationPlayer {
    private animations: Map<string, AnimationConfig> = new Map();
    private currentAnimation: AnimationConfig | null = null;
    private currentFrameIndex: number = 0;
    private elapsedTime: number = 0;
    private isPlaying: boolean = false;

    // 添加动画
    addAnimation(config: AnimationConfig): void {
        this.animations.set(config.name, config);
    }

    // 播放动画
    play(animationName: string, forceRestart: boolean = false): void {
        if (!this.animations.has(animationName)) {
            console.warn(`动画 "${animationName}" 不存在`);
            return;
        }

        const animation = this.animations.get(animationName)!;

        // 如果是同一个动画且正在播放，不重新开始
        if (this.currentAnimation === animation && !forceRestart) {
            return;
        }

        this.currentAnimation = animation;
        this.currentFrameIndex = 0;
        this.elapsedTime = 0;
        this.isPlaying = true;
    }

    // 停止动画
    stop(): void {
        this.isPlaying = false;
    }

    // 暂停动画
    pause(): void {
        this.isPlaying = false;
    }

    // 继续播放
    resume(): void {
        if (this.currentAnimation) {
            this.isPlaying = true;
        }
    }

    // 更新动画
    update(dt: number): void {
        if (!this.isPlaying || !this.currentAnimation) return;

        const animation = this.currentAnimation;
        const currentFrame = animation.frames[this.currentFrameIndex];

        // 累加时间
        this.elapsedTime += dt * 1000;  // 转换为毫秒

        // 检查是否需要切换到下一帧
        if (this.elapsedTime >= currentFrame.duration) {
            this.elapsedTime = 0;
            this.currentFrameIndex++;

            // 检查动画是否结束
            if (this.currentFrameIndex >= animation.frames.length) {
                if (animation.loop) {
                    // 循环播放
                    this.currentFrameIndex = 0;
                } else {
                    // 动画结束
                    this.currentFrameIndex = animation.frames.length - 1;
                    this.isPlaying = false;

                    // 调用完成回调
                    animation.onComplete?.();
                }
            }
        }
    }

    // 获取当前帧
    getCurrentFrame(): SpriteFrame | null {
        if (!this.currentAnimation) return null;
        return this.currentAnimation.frames[this.currentFrameIndex];
    }

    // 获取当前动画名称
    getCurrentAnimationName(): string | null {
        return this.currentAnimation?.name || null;
    }
}

// 带动画的精灵
class AnimatedSprite extends Sprite {
    private animator: AnimationPlayer = new AnimationPlayer();

    constructor(imageSrc: string, frameWidth: number, frameHeight: number) {
        super(imageSrc, frameWidth, frameHeight);
    }

    addAnimation(config: AnimationConfig): void {
        this.animator.addAnimation(config);
    }

    play(animationName: string, forceRestart: boolean = false): void {
        this.animator.play(animationName, forceRestart);
    }

    stop(): void {
        this.animator.stop();
    }

    update(dt: number): void {
        super.update(dt);
        this.animator.update(dt);
    }

    render(ctx: CanvasRenderingContext2D): void {
        if (!this.visible) return;

        const frame = this.animator.getCurrentFrame();
        if (!frame) {
            super.render(ctx);
            return;
        }

        ctx.save();

        // 应用变换
        ctx.translate(this.position.x, this.position.y);
        ctx.rotate(this.rotation);
        ctx.scale(
            this.scale.x * (this.flipX ? -1 : 1),
            this.scale.y * (this.flipY ? -1 : 1)
        );
        ctx.globalAlpha = this.opacity;

        // 绘制当前帧
        ctx.drawImage(
            frame.image,
            -this.frameWidth * this.anchor.x,
            -this.frameHeight * this.anchor.y,
            this.frameWidth,
            this.frameHeight
        );

        ctx.restore();
    }
}

// ==================== 精灵表加载器 ====================
class SpriteSheet {
    private image: HTMLImageElement;
    private loaded: boolean = false;

    constructor(imageSrc: string) {
        this.image = new Image();
        this.image.onload = () => {
            this.loaded = true;
        };
        this.image.src = imageSrc;
    }

    isLoaded(): boolean {
        return this.loaded;
    }

    // 从精灵表中提取单个帧
    getFrame(x: number, y: number, width: number, height: number): HTMLCanvasElement {
        const canvas = document.createElement('canvas');
        canvas.width = width;
        canvas.height = height;
        const ctx = canvas.getContext('2d')!;

        ctx.drawImage(
            this.image,
            x, y, width, height,  // 源区域
            0, 0, width, height   // 目标区域
        );

        return canvas;
    }

    // 批量获取帧
    getFrames(
        row: number,
        colStart: number,
        colEnd: number,
        frameWidth: number,
        frameHeight: number
    ): HTMLCanvasElement[] {
        const frames: HTMLCanvasElement[] = [];
        const y = row * frameHeight;

        for (let x = colStart; x <= colEnd; x++) {
            frames.push(this.getFrame(x * frameWidth, y, frameWidth, frameHeight));
        }

        return frames;
    }
}

// ==================== 粒子系统 ====================
interface Particle {
    x: number;
    y: number;
    vx: number;
    vy: number;
    life: number;
    maxLife: number;
    size: number;
    color: string;
    gravity: number;
    friction: number;
}

class ParticleSystem {
    private particles: Particle[] = [];
    private maxParticles: number = 1000;

    update(dt: number): void {
        // 更新每个粒子
        for (let i = this.particles.length - 1; i >= 0; i--) {
            const p = this.particles[i];

            // 应用重力
            p.vy += p.gravity * dt;

            // 应用摩擦力
            p.vx *= p.friction;
            p.vy *= p.friction;

            // 更新位置
            p.x += p.vx * dt;
            p.y += p.vy * dt;

            // 减少生命值
            p.life -= dt;

            // 移除死亡的粒子
            if (p.life <= 0) {
                this.particles.splice(i, 1);
            }
        }
    }

    render(ctx: CanvasRenderingContext2D): void {
        for (const p of this.particles) {
            const alpha = p.life / p.maxLife;
            ctx.globalAlpha = alpha;
            ctx.fillStyle = p.color;
            ctx.beginPath();
            ctx.arc(p.x, p.y, p.size * alpha, 0, Math.PI * 2);
            ctx.fill();
        }
        ctx.globalAlpha = 1;
    }

    emit(
        x: number,
        y: number,
        count: number,
        config: Partial<Particle> = {}
    ): void {
        for (let i = 0; i < count; i++) {
            if (this.particles.length >= this.maxParticles) break;

            const angle = config.vx !== undefined
                ? Math.atan2(config.vy || 0, config.vx || 0) + (Math.random() - 0.5) * Math.PI
                : Math.random() * Math.PI * 2;

            const speed = config.vx !== undefined
                ? Math.sqrt((config.vx || 0) ** 2 + (config.vy || 0) ** 2)
                : 100 + Math.random() * 200;

            this.particles.push({
                x,
                y,
                vx: Math.cos(angle) * speed,
                vy: Math.sin(angle) * speed,
                life: config.life || 1,
                maxLife: config.life || 1,
                size: config.size || 5,
                color: config.color || '#ffffff',
                gravity: config.gravity !== undefined ? config.gravity : 500,
                friction: config.friction !== undefined ? config.friction : 0.98
            });
        }
    }

    clear(): void {
        this.particles = [];
    }

    getParticleCount(): number {
        return this.particles.length;
    }
}
```

### 3.3 实战：Flappy Bird

通过一个完整的Flappy Bird游戏来巩固Canvas游戏开发的知识。本示例将实现一个可玩的Flappy Bird克隆版。

```typescript
// Flappy Bird游戏完整实现
class FlappyBird {
    // 游戏配置
    private readonly CONFIG = {
        CANVAS_WIDTH: 400,
        CANVAS_HEIGHT: 600,
        GRAVITY: 1200,           // 重力加速度（像素/秒²）
        JUMP_FORCE: -450,       // 跳跃力度（负值向上）
        PIPE_SPEED: 200,        // 管道移动速度
        PIPE_GAP: 180,          // 管道间隙
        PIPE_WIDTH: 60,
        PIPE_SPAWN_INTERVAL: 1500,  // 管道生成间隔（毫秒）
        BIRD_SIZE: 34,
        BIRD_X: 80
    };

    // 游戏状态
    private gameState: 'ready' | 'playing' | 'gameover' = 'ready';
    private score: number = 0;
    private highScore: number = 0;

    // 游戏对象
    private bird!: Bird;
    private pipes: Pipe[] = [];
    private particles: ParticleSystem = new ParticleSystem();

    // Canvas和渲染
    private canvas!: HTMLCanvasElement;
    private ctx!: CanvasRenderingContext2D;

    // 时间追踪
    private lastTime: number = 0;
    private lastPipeSpawn: number = 0;

    // 资源
    private spriteSheet!: HTMLImageElement;
    private spritesLoaded: boolean = false;

    // ==================== 初始化 ====================

    constructor() {
        this.createCanvas();
        this.loadSprites();
        this.initGame();
        this.setupInput();
        this.startGameLoop();
    }

    private createCanvas(): void {
        this.canvas = document.createElement('canvas');
        this.canvas.width = this.CONFIG.CANVAS_WIDTH;
        this.canvas.height = this.CONFIG.CANVAS_HEIGHT;
        this.canvas.style.background = '#70c5ce';
        this.canvas.style.borderRadius = '8px';
        this.canvas.style.boxShadow = '0 10px 40px rgba(0,0,0,0.3)';
        document.getElementById('game-container')?.appendChild(this.canvas);
        this.ctx = this.canvas.getContext('2d')!;
    }

    private loadSprites(): void {
        // 创建雪碧图（使用Canvas绘制替代真实图片）
        this.spriteSheet = new Image();
        this.spriteSheet.src = 'data:image/png;base64,iVBORw0KGgoAAAANSUhEUgAAAAEAAAABCAYAAAAfFcSJAAAADUlEQVR42mNk+M9QDwADhgGAWjR9awAAAABJRU5ErkJggg==';
        this.spritesLoaded = true;
    }

    private initGame(): void {
        // 初始化小鸟
        this.bird = new Bird(
            this.CONFIG.BIRD_X,
            this.CONFIG.CANVAS_HEIGHT / 2,
            this.CONFIG.BIRD_SIZE
        );

        // 清空管道和粒子
        this.pipes = [];
        this.particles.clear();

        // 重置状态
        this.score = 0;
        this.lastPipeSpawn = 0;
    }

    private setupInput(): void {
        // 点击/空格跳跃
        const handleJump = () => {
            if (this.gameState === 'ready') {
                this.gameState = 'playing';
            }

            if (this.gameState === 'playing') {
                this.bird.jump(this.CONFIG.JUMP_FORCE);
                this.particles.emit(
                    this.bird.x,
                    this.bird.y + this.CONFIG.BIRD_SIZE / 2,
                    5,
                    {
                        vx: 0,
                        vy: -200,
                        color: '#ffffff',
                        size: 4,
                        life: 0.3,
                        gravity: 200
                    }
                );
            }

            if (this.gameState === 'gameover') {
                this.restart();
            }
        };

        this.canvas.addEventListener('click', handleJump);
        this.canvas.addEventListener('touchstart', (e) => {
            e.preventDefault();
            handleJump();
        });

        window.addEventListener('keydown', (e) => {
            if (e.code === 'Space' || e.code === 'ArrowUp') {
                e.preventDefault();
                handleJump();
            }
        });
    }

    private restart(): void {
        this.initGame();
        this.gameState = 'ready';
    }

    // ==================== 游戏循环 ====================

    private startGameLoop(): void {
        requestAnimationFrame((time) => this.gameLoop(time));
    }

    private gameLoop(currentTime: number): void {
        // 计算时间差
        const dt = Math.min((currentTime - this.lastTime) / 1000, 0.1);
        this.lastTime = currentTime;

        // 更新
        this.update(dt, currentTime);

        // 渲染
        this.render();

        // 继续循环
        requestAnimationFrame((time) => this.gameLoop(time));
    }

    private update(dt: number, currentTime: number): void {
        if (this.gameState === 'playing') {
            // 更新小鸟
            this.bird.update(dt, this.CONFIG.GRAVITY);

            // 生成管道
            if (currentTime - this.lastPipeSpawn > this.CONFIG.PIPE_SPAWN_INTERVAL) {
                this.spawnPipe();
                this.lastPipeSpawn = currentTime;
            }

            // 更新管道
            for (let i = this.pipes.length - 1; i >= 0; i--) {
                const pipe = this.pipes[i];
                pipe.update(dt, this.CONFIG.PIPE_SPEED);

                // 检测碰撞
                if (this.checkCollision(pipe)) {
                    this.gameOver();
                }

                // 检测得分
                if (!pipe.scored && pipe.x + this.CONFIG.PIPE_WIDTH < this.bird.x) {
                    pipe.scored = true;
                    this.score++;
                    this.onScore();
                }

                // 移除离开屏幕的管道
                if (pipe.x + this.CONFIG.PIPE_WIDTH < 0) {
                    this.pipes.splice(i, 1);
                }
            }

            // 检测边界碰撞
            if (this.bird.y + this.CONFIG.BIRD_SIZE / 2 > this.CONFIG.CANVAS_HEIGHT - 112 ||
                this.bird.y - this.CONFIG.BIRD_SIZE / 2 < 0) {
                this.gameOver();
            }
        } else if (this.gameState === 'ready') {
            // 准备状态下小鸟上下浮动
            this.bird.updateIdle(dt);
        }

        // 更新粒子
        this.particles.update(dt);
    }

    private spawnPipe(): void {
        // 随机间隙位置
        const minY = 100;
        const maxY = this.CONFIG.CANVAS_HEIGHT - this.CONFIG.PIPE_GAP - 112 - 100;
        const gapY = minY + Math.random() * (maxY - minY);

        this.pipes.push(new Pipe(
            this.CONFIG.CANVAS_WIDTH,
            gapY,
            this.CONFIG.PIPE_WIDTH,
            this.CONFIG.PIPE_GAP
        ));
    }

    private checkCollision(pipe: Pipe): boolean {
        const birdLeft = this.bird.x - this.CONFIG.BIRD_SIZE / 2 + 4;
        const birdRight = this.bird.x + this.CONFIG.BIRD_SIZE / 2 - 4;
        const birdTop = this.bird.y - this.CONFIG.BIRD_SIZE / 2 + 4;
        const birdBottom = this.bird.y + this.CONFIG.BIRD_SIZE / 2 - 4;

        const pipeLeft = pipe.x;
        const pipeRight = pipe.x + pipe.width;
        const pipeTop = pipe.gapY;
        const pipeBottom = pipe.gapY + pipe.gap;

        return birdRight > pipeLeft &&
               birdLeft < pipeRight &&
               (birdTop < pipeTop || birdBottom > pipeBottom);
    }

    private onScore(): void {
        // 得分时的特效
        this.particles.emit(
            this.bird.x + this.CONFIG.BIRD_SIZE,
            this.bird.y,
            10,
            {
                vx: 100,
                vy: 0,
                color: '#FFD700',
                size: 6,
                life: 0.5,
                gravity: 300
            }
        );
    }

    private gameOver(): void {
        this.gameState = 'gameover';

        if (this.score > this.highScore) {
            this.highScore = this.score;
        }

        // 爆炸粒子效果
        this.particles.emit(
            this.bird.x,
            this.bird.y,
            30,
            {
                vx: 0,
                vy: 0,
                color: '#ff6b6b',
                size: 8,
                life: 1,
                gravity: 800
            }
        );
    }

    // ==================== 渲染 ====================

    private render(): void {
        const ctx = this.ctx;

        // 绘制背景
        this.drawBackground();

        // 绘制管道
        this.pipes.forEach(pipe => pipe.render(ctx));

        // 绘制地面
        this.drawGround();

        // 绘制小鸟
        if (this.gameState !== 'gameover') {
            this.bird.render(ctx);
        }

        // 绘制粒子
        this.particles.render(ctx);

        // 绘制UI
        this.drawUI();
    }

    private drawBackground(): void {
        const ctx = this.ctx;

        // 天空渐变
        const gradient = ctx.createLinearGradient(0, 0, 0, this.CONFIG.CANVAS_HEIGHT);
        gradient.addColorStop(0, '#87CEEB');
        gradient.addColorStop(0.5, '#70c5ce');
        gradient.addColorStop(1, '#9ED8F0');
        ctx.fillStyle = gradient;
        ctx.fillRect(0, 0, this.CONFIG.CANVAS_WIDTH, this.CONFIG.CANVAS_HEIGHT);

        // 云朵
        ctx.fillStyle = 'rgba(255, 255, 255, 0.8)';
        this.drawCloud(50, 100, 60);
        this.drawCloud(200, 60, 80);
        this.drawCloud(320, 120, 50);
    }

    private drawCloud(x: number, y: number, size: number): void {
        const ctx = this.ctx;

        ctx.arc(x, y, size * 0.5, 0, Math.PI * 2);

// ==================== 小鸟类 ====================
class Bird {
    x: number;
    y: number;
    private size: number;
    rotation: number = 0;
    velocity: number = 0;
    private animTime: number = 0;
    private frame: number = 0;

    constructor(x: number, y: number, size: number) {
        this.x = x;
        this.y = y;
        this.size = size;
    }

    jump(force: number): void {
        this.velocity = force;
    }

    update(dt: number, gravity: number): void {
        // 应用重力
        this.velocity += gravity * dt;
        this.y += this.velocity * dt;

        // 根据速度旋转
        this.rotation = Math.min(
            Math.max(this.velocity / 400 * Math.PI / 4, -Math.PI / 4),
            Math.PI / 2
        );

        // 翅膀动画
        this.animTime += dt;
        if (this.animTime > 0.1) {
            this.frame = (this.frame + 1) % 3;
            this.animTime = 0;
        }
    }

    updateIdle(dt: number): void {
        // 上下浮动
        this.y += Math.sin(Date.now() / 200) * 0.5;
        this.animTime += dt;
        if (this.animTime > 0.15) {
            this.frame = (this.frame + 1) % 3;
            this.animTime = 0;
        }
    }

    render(ctx: CanvasRenderingContext2D): void {
        ctx.save();
        ctx.translate(this.x, this.y);
        ctx.rotate(this.rotation);

        // 绘制鸟身
        ctx.fillStyle = '#f7dc6f';
        ctx.beginPath();
        ctx.ellipse(0, 0, this.size / 2, this.size / 2 * 0.8, 0, 0, Math.PI * 2);
        ctx.fill();

        // 绘制眼睛
        ctx.fillStyle = '#ffffff';
        ctx.beginPath();
        ctx.arc(8, -5, 8, 0, Math.PI * 2);
        ctx.fill();

        ctx.fillStyle = '#000000';
        ctx.beginPath();
        ctx.arc(10, -5, 4, 0, Math.PI * 2);
        ctx.fill();

        // 绘制嘴巴
        ctx.fillStyle = '#e67e22';
        ctx.beginPath();
        ctx.moveTo(this.size / 2, 0);
        ctx.lineTo(this.size / 2 + 10, 3);
        ctx.lineTo(this.size / 2, 6);
        ctx.closePath();
        ctx.fill();

        // 绘制翅膀（根据帧动画）
        ctx.fillStyle = '#f39c12';
        const wingOffset = [0, -5, -2][this.frame];
        ctx.beginPath();
        ctx.ellipse(-5, 5 + wingOffset, 10, 6, -0.3, 0, Math.PI * 2);
        ctx.fill();

        ctx.restore();
    }
}

// ==================== 管道类 ====================
class Pipe {
    x: number;
    gapY: number;
    width: number;
    gap: number;
    scored: boolean = false;

    constructor(x: number, gapY: number, width: number, gap: number) {
        this.x = x;
        this.gapY = gapY;
        this.width = width;
        this.gap = gap;
    }

    update(dt: number, speed: number): void {
        this.x -= speed * dt;
    }

    render(ctx: CanvasRenderingContext2D): void {
        const canvasHeight = 600;
        const groundHeight = 112;

        // 管道颜色
        const pipeColor = '#73bf2e';
        const pipeBorder = '#4a8a1c';
        const pipeHighlight = '#8fd94a';

        // 绘制上管道
        this.drawPipe(ctx, this.x, 0, this.width, this.gapY, pipeColor, pipeBorder, pipeHighlight);

        // 绘制下管道
        const bottomY = this.gapY + this.gap;
        const bottomHeight = canvasHeight - groundHeight - bottomY;
        this.drawPipe(ctx, this.x, bottomY, this.width, bottomHeight, pipeColor, pipeBorder, pipeHighlight);
    }

    private drawPipe(
        ctx: CanvasRenderingContext2D,
        x: number,
        y: number,
        width: number,
        height: number,
        color: string,
        borderColor: string,
        highlightColor: string
    ): void {
        if (height <= 0) return;

        const capHeight = 30;
        const capOverhang = 5;

        // 管道主体
        ctx.fillStyle = color;
        ctx.fillRect(x, y, width, height);

        // 管道边框
        ctx.strokeStyle = borderColor;
        ctx.lineWidth = 3;
        ctx.strokeRect(x, y, width, height);

        // 高光
        ctx.fillStyle = highlightColor;
        ctx.fillRect(x + 5, y, 8, height);

        // 管道帽
        if (y < 300) {
            // 上管道帽（在下端）
            ctx.fillStyle = color;
            ctx.fillRect(x - capOverhang, y + height - capHeight, width + capOverhang * 2, capHeight);
            ctx.strokeStyle = borderColor;
            ctx.strokeRect(x - capOverhang, y + height - capHeight, width + capOverhang * 2, capHeight);
            ctx.fillStyle = highlightColor;
            ctx.fillRect(x - capOverhang + 5, y + height - capHeight, 8, capHeight);
        } else {
            // 下管道帽（在上端）
            ctx.fillStyle = color;
            ctx.fillRect(x - capOverhang, y, width + capOverhang * 2, capHeight);
            ctx.strokeStyle = borderColor;
            ctx.strokeRect(x - capOverhang, y, width + capOverhang * 2, capHeight);
            ctx.fillStyle = highlightColor;
            ctx.fillRect(x - capOverhang + 5, y, 8, capHeight);
        }
    }
}
```

## 四、WebGL游戏开发

### 4.1 Three.js基础

Three.js是目前最流行的WebGL 3D图形库，它封装了WebGL的复杂操作，提供了简洁的API来创建3D场景、相机、灯光、模型等。使用Three.js可以快速开发3D游戏、可视化应用、交互式Web体验等。

Three.js的核心概念包括：

- **Scene（场景）**：3D世界的容器，包含所有3D对象
- **Camera（相机）**：决定观察角度，分为透视相机和正交相机
- **Renderer（渲染器）**：负责将场景渲染到Canvas
- **Mesh（网格）**：由几何体和材质组成的基本3D对象
- **Geometry（几何体）**：形状数据，如BoxGeometry、CylinderGeometry等
- **Material（材质）**：外观数据，如颜色、纹理、光泽度等
- **Light（灯光）**：照亮场景的光源

```typescript

            });

            // 渲染场景
            this.renderer.render(this.scene, this.camera);
        };

        animate();
    }
```

通过一个完整的3D迷宫游戏来巩固Three.js的使用。本示例将实现一个第一人称视角的3D迷宫探索游戏。

```typescript
import * as THREE from 'three';

// ==================== 迷宫生成器 ====================
class MazeGenerator {
    // 使用递归回溯算法生成迷宫
    static generate(width: number, height: number): number[][] {
        // 初始化迷宫（全为墙）
        const maze: number[][] = [];
        for (let y = 0; y < height; y++) {
            maze[y] = [];
            for (let x = 0; x < width; x++) {
                maze[y][x] = 1;  // 1表示墙
            }
        }

        // 从(1,1)开始生成路径
        const stack: [number, number][] = [];
        const startX = 1;
        const startY = 1;

        maze[startY][startX] = 0;  // 0表示路
        stack.push([startX, startY]);

        while (stack.length > 0) {
            const [cx, cy] = stack[stack.length - 1];

            // 获取可用的邻居（距离为2）
            const neighbors: [number, number][] = [];
            const directions = [
                [0, -2], [0, 2], [-2, 0], [2, 0]
            ];

            for (const [dx, dy] of directions) {
                const nx = cx + dx;
                const ny = cy + dy;

                if (nx > 0 && nx < width - 1 && ny > 0 && ny < height - 1) {
                    if (maze[ny][nx] === 1) {
                        neighbors.push([nx, ny]);
                    }
                }
            }

            if (neighbors.length > 0) {
                // 选择随机邻居
                const [nx, ny] = neighbors[Math.floor(Math.random() * neighbors.length)];

                // 打通墙壁
                const wallX = cx + (nx - cx) / 2;
                const wallY = cy + (ny - cy) / 2;
                maze[wallY][wallX] = 0;
                maze[ny][nx] = 0;

                stack.push([nx, ny]);
            } else {
                // 回溯
                stack.pop();
            }
        }

        // 设置起点和终点
        maze[1][1] = 0;  // 起点
        maze[height - 2][width - 2] = 0;  // 终点

        return maze;
    }
}

    private checkCollision(position: THREE.Vector3): THREE.Vector3 {
        const playerRadius = 0.5;
        const newPosition = position.clone();

        // 检查与墙壁的碰撞
        for (const wall of this.walls) {
            const wallBox = new THREE.Box3().setFromObject(wall);
            const playerBox = new THREE.Box3();

            // 创建玩家包围盒
            playerBox.min.set(
                newPosition.x - playerRadius,
                newPosition.y - this.CONFIG.PLAYER_HEIGHT,
                newPosition.z - playerRadius
            );
            playerBox.max.set(
                newPosition.x + playerRadius,
                newPosition.y,
                newPosition.z + playerRadius
            );

            // 检测碰撞
            if (wallBox.intersectsBox(playerBox)) {
                // 计算推回方向
                const wallCenter = new THREE.Vector3();
                wallBox.getCenter(wallCenter);

                const pushDir = new THREE.Vector3()
                    .subVectors(newPosition, wallCenter)
                    .normalize();

                // 找到最小穿透深度
                const distances = [
                    wallBox.max.x - playerBox.min.x,
                    playerBox.max.x - wallBox.min.x,
                    wallBox.max.z - playerBox.min.z,
                    playerBox.max.z - wallBox.min.z
                ];

                const minDist = Math.min(...distances);

                if (minDist === distances[0]) {
                    newPosition.x = wallBox.max.x + playerRadius;
                } else if (minDist === distances[1]) {
                    newPosition.x = wallBox.min.x - playerRadius;
                } else if (minDist === distances[2]) {
                    newPosition.z = wallBox.max.z + playerRadius;
                } else {
                    newPosition.z = wallBox.min.z - playerRadius;
                }
            }
        }

        return newPosition;
    }

    private updateMovement(deltaTime: number): void {
        if (!this.controls.isLocked) return;

        this.direction.z = Number(this.moveForward) - Number(this.moveBackward);
        this.direction.x = Number(this.moveRight) - Number(this.moveLeft);
        this.direction.normalize();

        // 应用移动

        if (this.moveForward || this.moveBackward) {
            this.velocity.z = -this.direction.z * speed;
        } else {
            this.velocity.z = 0;
        }

        if (this.moveLeft || this.moveRight) {
            this.velocity.x = this.direction.x * speed;
        } else {
            this.velocity.x = 0;
        }

        // 应用相机方向
        this.controls.moveRight(this.velocity.x);
        this.controls.moveForward(-this.velocity.z);

        // 碰撞检测
        const newPosition = this.checkCollision(this.camera.position);
        this.camera.position.x = newPosition.x;
        this.camera.position.z = newPosition.z;

        // 检查是否到达终点
        this.checkGoal();
    }

    private animate(): void {
        requestAnimationFrame(() => this.animate());

        const deltaTime = 1 / 60;  // 固定时间步长

        if (this.gameState === 'playing') {
            this.updateMovement(deltaTime);
        }

        this.renderer.render(this.scene, this.camera);
    }
}
```

### 5.1 WebSocket同步

实时对战是游戏开发中的高级话题，WebSocket是实现实时通信的首选技术。相比HTTP轮询，WebSocket提供全双工通信，可以在服务器和客户端之间持续传输数据，非常适合游戏中的实时同步。

```typescript
// WebSocket网络通信模块
import { Matter } from 'matter-js';

type MessageHandler = (data: any) => void;

interface NetworkMessage {
    type: string;
    payload: any;
    timestamp: number;
    sequence: number;
}

class GameNetwork {
    private ws: WebSocket | null = null;
    private url: string;
    private handlers: Map<string, MessageHandler[]> = new Map();
    private messageQueue: NetworkMessage[] = [];
    private sequence: number = 0;
    private latency: number = 0;
    private lastPingTime: number = 0;
    private reconnectAttempts: number = 0;
    private maxReconnectAttempts: number = 5;
    private reconnectDelay: number = 1000;
    private isConnected: boolean = false;

    // 玩家数据
    private playerId: string = '';
    private players: Map<string, PlayerState> = new Map();

    constructor(url: string) {
        this.url = url;
    }

    // 连接服务器
    connect(): Promise<void> {
        return new Promise((resolve, reject) => {
            try {
                this.ws = new WebSocket(this.url);

                this.ws.onopen = () => {
                    console.log('WebSocket连接已建立');
                    this.isConnected = true;
                    this.reconnectAttempts = 0;
                    this.startPing();
                    this.flushMessageQueue();
                    resolve();
                };

                this.ws.onmessage = (event) => {
                    this.handleMessage(JSON.parse(event.data));
                };

                this.ws.onclose = () => {
                    console.log('WebSocket连接已关闭');
                    this.isConnected = false;
                    this.attemptReconnect();
                };

                this.ws.onerror = (error) => {
                    console.error('WebSocket错误:', error);
                    reject(error);
                };
            } catch (error) {
                reject(error);
            }
        });
    }

    // 断开连接
    disconnect(): void {
        if (this.ws) {
            this.ws.close();
            this.ws = null;
        }
    }

    // 发送消息
    send(type: string, payload: any): void {
        const message: NetworkMessage = {
            type,
            payload,
            timestamp: Date.now(),
            sequence: ++this.sequence
        };

        if (this.isConnected && this.ws?.readyState === WebSocket.OPEN) {
            this.ws.send(JSON.stringify(message));
        } else {
            // 离线模式，缓存消息
            this.messageQueue.push(message);
        }
    }

    // 注册消息处理器
    on(type: string, handler: MessageHandler): void {
        if (!this.handlers.has(type)) {
            this.handlers.set(type, []);
        }
        this.handlers.get(type)!.push(handler);
    }

    // 移除消息处理器
    off(type: string, handler: MessageHandler): void {
        const handlers = this.handlers.get(type);
        if (handlers) {
            const index = handlers.indexOf(handler);
            if (index > -1) {
                handlers.splice(index, 1);
            }
        }
    }

    // 处理接收到的消息
    private handleMessage(message: NetworkMessage): void {
        const { type, payload } = message;

        switch (type) {
            case 'pong':
                this.latency = Date.now() - this.lastPingTime;
                break;

            case 'welcome':
                this.playerId = payload.playerId;
                console.log(`已连接到服务器，玩家ID: ${this.playerId}`);
                break;

            case 'player_join':
                this.players.set(payload.playerId, payload.state);
                console.log(`玩家 ${payload.playerId} 加入游戏`);
                break;

            case 'player_leave':
                this.players.delete(payload.playerId);
                console.log(`玩家 ${payload.playerId} 离开游戏`);
                break;

            case 'state_sync':
                this.handleStateSync(payload);
                break;

            case 'input_ack':
                this.handleInputAck(payload);
                break;

            case 'game_event':
                this.handleGameEvent(payload);
                break;
        }

        // 调用注册的处理器
        const handlers = this.handlers.get(type);
        if (handlers) {
            handlers.forEach(handler => handler(payload));
        }

        // 调用所有类型的处理器
        const allHandlers = this.handlers.get('*');
        if (allHandlers) {
            allHandlers.forEach(handler => handler({ type, payload }));
        }
    }

    // 处理状态同步
    private handleStateSync(payload: any): void {
        // 更新其他玩家的状态
        for (const playerState of payload.players) {
            if (playerState.playerId !== this.playerId) {
                const existing = this.players.get(playerState.playerId);
                if (existing) {
                    // 插值更新位置
                    existing.position = this.lerp(
                        existing.position,
                        playerState.position,
                        0.2
                    );
                    existing.velocity = playerState.velocity;
                    existing.rotation = playerState.rotation;
                } else {
                    this.players.set(playerState.playerId, playerState);
                }
            }
        }
    }

    // 处理输入确认
    private handleInputAck(payload: any): void {
        console.log(`输入已确认，服务器序列: ${payload.serverSequence}`);
    }

    // 处理游戏事件
    private handleGameEvent(payload: any): void {
        console.log(`游戏事件: ${payload.event}`);
    }

    // 发送玩家输入
    sendInput(input: PlayerInput): void {
        this.send('player_input', {
            playerId: this.playerId,
            input,
            sequence: this.sequence
        });
    }

    // 发送玩家状态
    sendState(state: Partial<PlayerState>): void {
        this.send('player_state', {
            playerId: this.playerId,
            state
        });
    }

    // 发送聊天消息
    sendChat(message: string): void {
        this.send('chat', {
            playerId: this.playerId,
            message,
            timestamp: Date.now()
        });
    }

    // 心跳检测
    private startPing(): void {
        setInterval(() => {
            if (this.isConnected) {
                this.lastPingTime = Date.now();
                this.send('ping', {});
            }
        }, 1000);
    }

    // 重连机制
    private attemptReconnect(): void {
        if (this.reconnectAttempts >= this.maxReconnectAttempts) {
            console.log('达到最大重连次数');
            return;
        }

        this.reconnectAttempts++;
        const delay = this.reconnectDelay * Math.pow(2, this.reconnectAttempts - 1);

        console.log(`${delay / 1000}秒后尝试重连...`);

        setTimeout(() => {
            this.connect().catch(() => {
                this.attemptReconnect();
            });
        }, delay);
    }

    // 发送缓存的消息
    private flushMessageQueue(): void {
        while (this.messageQueue.length > 0) {
            const message = this.messageQueue.shift()!;
            this.ws?.send(JSON.stringify(message));
        }
    }

    // 线性插值
    private lerp(a: number, b: number, t: number): number {
        return a + (b - a) * t;
    }

    // 获取其他玩家
    getOtherPlayers(): Map<string, PlayerState> {

    }

    // 获取延迟
    getLatency(): number {
        return this.latency;
    }

    // 是否已连接

        return this.isConnected;
    }
}
```

### 5.2 帧同步与状态同步

帧同步和状态同步是网络游戏中最常见的两种同步方式，各有优缺点，适用于不同的游戏类型。

**帧同步**的核心思想是所有客户端运行相同的游戏逻辑，服务器只负责转发玩家的输入指令。客户端将输入发送给服务器，服务器广播给所有客户端，所有客户端基于相同的初始状态和输入序列进行计算，最终得到相同的结果。帧同步的优势在于带宽占用小（只传输输入，不传输状态），可以支持大量玩家同时在线，服务器压力小。但帧同步的缺点是需要保证所有客户端的计算结果完全一致，这要求使用确定性的物理引擎和随机数生成器，开发难度较高。

**状态同步**的核心思想是服务器作为游戏状态的权威来源，客户端只负责渲染。客户端发送输入到服务器，服务器计算游戏状态，然后将状态广播给所有客户端。客户端根据接收到的状态更新本地表现。状态同步的优势是开发简单，客户端不需要完整的游戏逻辑，只需要根据服务器状态进行渲染。但状态同步的缺点是带宽占用大，服务器压力大。

```typescript
// 帧同步实现示例
class FrameSync {
    private frameCount: number = 0;
    private inputs: Map<string, PlayerInput[]> = new Map();
    private serverFrameRate: number = 20;  // 服务器帧率
    private clientFrameRate: number = 60;   // 客户端帧率
    private frameBuffer: GameFrame[] = [];
    private localPlayerId: string = '';

    // 锁定步长（服务器帧间隔）
    private readonly FIXED_DELTA_TIME = 1 / this.serverFrameRate;

    // 初始化
    init(playerId: string): void {
        this.localPlayerId = playerId;
    }

    // 存储输入
    storeInput(playerId: string, input: PlayerInput): void {
        if (!this.inputs.has(playerId)) {
            this.inputs.set(playerId, []);
        }

        // 存储到当前帧
        const frame = Math.floor(this.frameCount);
        const inputs = this.inputs.get(playerId)!;

        // 确保数组足够长
        while (inputs.length <= frame) {
            inputs.push(this.getDefaultInput());
        }

        inputs[frame] = input;
    }

    // 获取指定帧的所有输入
    getInputsForFrame(frame: number): Map<string, PlayerInput> {
        const result = new Map<string, PlayerInput>();

        this.inputs.forEach((playerInputs, playerId) => {
            if (playerInputs[frame]) {
                result.set(playerId, playerInputs[frame]);
            } else {
                result.set(playerId, this.getDefaultInput());
            }
        });

        return result;
    }

    // 模拟一帧游戏逻辑（确定性）
    simulateFrame(frame: number, gameState: GameState): GameState {
        const inputs = this.getInputsForFrame(frame);
        const newState = this.cloneGameState(gameState);

        // 应用每个玩家的输入
        inputs.forEach((input, playerId) => {
            this.applyInput(newState, playerId, input);
        });

        // 物理更新（确定性）
        this.updatePhysics(newState, this.FIXED_DELTA_TIME);

        // 碰撞检测（确定性）
        this.checkCollisions(newState);

        // 规则检查（确定性）
        this.checkRules(newState);

        return newState;
    }

    // 应用输入
    private applyInput(state: GameState, playerId: string, input: PlayerInput): void {
        const player = state.players.get(playerId);
        if (!player) return;

        // 移动
        const speed = 5;
        if (input.forward) player.velocity.y = -speed;
        else if (input.backward) player.velocity.y = speed;
        else player.velocity.y = 0;

        if (input.left) player.velocity.x = -speed;
        else if (input.right) player.velocity.x = speed;
        else player.velocity.x = 0;

        // 跳跃
        if (input.jump && player.onGround) {
            player.velocity.y = 10;
            player.onGround = false;
        }

        // 旋转（跟随鼠标）
        player.rotation = input.mouseX * 0.01;
    }

    // 物理更新（使用确定性随机数）
    private updatePhysics(state: GameState, dt: number): void {
        const gravity = 9.8;

        state.players.forEach((player) => {
            // 应用重力
            player.velocity.y -= gravity * dt;

            // 更新位置
            player.position.x += player.velocity.x * dt;
            player.position.y += player.velocity.y * dt;

            // 地面检测
            if (player.position.y <= 0) {
                player.position.y = 0;
                player.velocity.y = 0;
                player.onGround = true;
            }

            // 世界边界
            player.position.x = Math.max(0, Math.min(100, player.position.x));
        });
    }

    // 碰撞检测
    private checkCollisions(state: GameState): void {
        const players = Array.from(state.players.values());

        // 玩家之间的碰撞
        for (let i = 0; i < players.length; i++) {
            for (let j = i + 1; j < players.length; j++) {
                const p1 = players[i];
                const p2 = players[j];

                const dx = p2.position.x - p1.position.x;
                const dy = p2.position.y - p1.position.y;
                const dist = Math.sqrt(dx * dx + dy * dy);

                if (dist < 2) {  // 碰撞半径
                    // 推开
                    const overlap = 2 - dist;
                    const nx = dx / dist;
                    const ny = dy / dist;

                    p1.position.x -= nx * overlap / 2;
                    p1.position.y -= ny * overlap / 2;
                    p2.position.x += nx * overlap / 2;
                    p2.position.y += ny * overlap / 2;
                }
            }
        }
    }

    // 规则检查
    private checkRules(state: GameState): void {
        // 检查得分
        state.players.forEach((player) => {
            if (player.position.y > 50) {
                player.score += 1;
                player.position = { x: 10, y: 0 };
            }
        });
    }

    // 克隆游戏状态（用于帧回放）
    private cloneGameState(state: GameState): GameState {
        const cloned = {
            players: new Map<string, Player>(),
            projectiles: new Map<string, Projectile>(),
            items: new Map<string, Item>()
        };

        state.players.forEach((player, id) => {
            cloned.players.set(id, { ...player });
        });

        return cloned;
    }

    // 获取默认输入
    private getDefaultInput(): PlayerInput {
        return {
            forward: false,
            backward: false,
            left: false,
            right: false,
            jump: false,
            attack: false,
            mouseX: 0,
            mouseY: 0
        };
    }

    // 增加帧数
    advanceFrame(): number {
        return ++this.frameCount;
    }

    getFrameCount(): number {
        return this.frameCount;
    }
}

// 状态同步实现示例
class StateSync {
    private serverState: GameState | null = null;
    private clientState: GameState | null = null;
    private interpolationDelay: number = 3;  // 插值延迟（帧）
    private stateBuffer: GameState[] = [];

    // 收到服务器状态时调用
    onServerState(state: GameState): void {
        this.stateBuffer.push(state);

        // 保持缓冲区大小
        if (this.stateBuffer.length > this.interpolationDelay * 2) {
            this.stateBuffer.shift();
        }

        this.serverState = state;
    }

    // 获取平滑插值后的状态
    getInterpolatedState(): GameState | null {
        if (this.stateBuffer.length < 2) {
            return this.serverState;
        }

        // 使用最早和最晚的状态进行插值
        const oldest = this.stateBuffer[0];
        const newest = this.stateBuffer[this.stateBuffer.length - 1];
        const t = 0.5;  // 插值因子

        return this.interpolateState(oldest, newest, t);
    }

    // 插值两个状态
    private interpolateState(a: GameState, b: GameState, t: number): GameState {
        const result: GameState = {
            players: new Map(),
            projectiles: new Map(),
            items: new Map()
        };

        // 插值玩家状态
        b.players.forEach((player, id) => {
            const playerA = a.players.get(id);
            if (playerA) {
                result.players.set(id, {
                    ...player,
                    position: {
                        x: playerA.position.x + (player.position.x - playerA.position.x) * t,
                        y: playerA.position.y + (player.position.y - playerA.position.y) * t
                    },
                    velocity: {
                        x: playerA.velocity.x + (player.velocity.x - playerA.velocity.x) * t,
                        y: playerA.velocity.y + (player.velocity.y - playerA.velocity.y) * t
                    }
                });
            } else {
                result.players.set(id, player);
            }
        });

        return result;
    }

    // 预测本地玩家位置（客户端预测）
    predictLocalPlayer(
        localId: string,
        currentState: GameState,
        input: PlayerInput
    ): PlayerState {
        const player = currentState.players.get(localId);
        if (!player) {
            throw new Error(`玩家 ${localId} 不存在`);
        }

        // 复制玩家状态
        const predicted = { ...player };

        // 应用输入预测
        const speed = 5;
        if (input.forward) predicted.velocity.y = -speed;
        if (input.backward) predicted.velocity.y = speed;
        if (input.left) predicted.velocity.x = -speed;
        if (input.right) predicted.velocity.x = speed;

        // 更新位置
        predicted.position.x += predicted.velocity.x * (1 / 60);
        predicted.position.y += predicted.velocity.y * (1 / 60);

        return predicted;
    }

    // 校正预测错误（服务器状态到来时）
    reconcile(
        localId: string,
        serverState: GameState,
        predictedState: PlayerState
    ): void {
        const serverPlayer = serverState.players.get(localId);
        if (!serverPlayer) return;

        // 计算误差
        const errorX = Math.abs(serverPlayer.position.x - predictedState.position.x);
        const errorY = Math.abs(serverPlayer.position.y - predictedState.position.y);

        // 如果误差过大，需要校正
        if (errorX > 1 || errorY > 1) {
            console.log(`预测误差过大，校正玩家位置`);
            // 在实际应用中，这里会平滑地将玩家从当前位置移动到服务器位置
        }
    }
}
```

### 5.3 延迟补偿

在网络游戏中，延迟是不可避免的问题。延迟补偿技术可以缓解网络延迟带来的游戏体验问题。

常见的延迟补偿技术包括：

1. **客户端预测**：客户端在收到服务器确认之前就预测玩家的动作，提供即时反馈
2. **服务器回溯**：服务器回溯到客户端发送指令时的游戏状态来验证操作
3. **延迟平滑**：使用插值技术平滑显示网络延迟造成的位置跳动
4. **延迟隐藏**：使用动画和特效掩盖网络延迟

```typescript
// 延迟补偿系统
class LagCompensation {
    // 存储历史状态（用于服务器回溯）
    private stateHistory: Array<{
        timestamp: number;
        frame: number;
        state: GameState;
    }> = [];

    private historyDuration: number = 5000;  // 保存5秒历史
    private serverFrameRate: number = 20;

    // 存储玩家的延迟估计
    private playerLatencies: Map<string, number> = new Map();

    // 更新延迟估计
    updateLatency(playerId: string, latency: number): void {
        this.playerLatencies.set(playerId, latency);
    }

    // 获取平均延迟
    getAverageLatency(): number {
        if (this.playerLatencies.size === 0) return 0;

        let total = 0;
        this.playerLatencies.forEach(latency => {
            total += latency;
        });

        return total / this.playerLatencies.size;
    }

    // 保存状态到历史
    saveState(frame: number, state: GameState): void {
        const now = Date.now();

        // 移除过期的历史
        this.stateHistory = this.stateHistory.filter(
            entry => now - entry.timestamp < this.historyDuration
        );

        // 添加新状态
        this.stateHistory.push({
            timestamp: now,
            frame,
            state: this.cloneState(state)
        });
    }

    // 获取过去某个时间点的状态（用于服务器回溯）
    getStateAtTime(timestamp: number): GameState | null {
        // 找到最接近目标时间的状态
        let closest: typeof this.stateHistory[0] | null = null;
        let closestDiff = Infinity;

        for (const entry of this.stateHistory) {
            const diff = Math.abs(entry.timestamp - timestamp);
            if (diff < closestDiff) {
                closestDiff = diff;
                closest = entry;
            }
        }

        return closest ? this.cloneState(closest.state) : null;
    }

    // 获取指定帧的状态
    getStateAtFrame(frame: number): GameState | null {
        for (const entry of this.stateHistory) {
            if (entry.frame === frame) {
                return this.cloneState(entry.state);
            }
        }
        return null;
    }

    // 服务器回溯检测命中
    serverSideRaycast(
        playerId: string,
        origin: { x: number; y: number },
        direction: { x: number; y: number },
        range: number,
        timestamp: number
    ): { hit: boolean; targetId: string | null; hitPoint: { x: number; y: number } | null } {
        // 回溯到客户端发送指令时的服务器状态
        const historicalState = this.getStateAtTime(timestamp);

        if (!historicalState) {
            console.warn('无法获取历史状态进行回溯检测');
            return { hit: false, targetId: null, hitPoint: null };
        }

        // 执行射线检测
        return this.raycastInState(historicalState, playerId, origin, direction, range);
    }

    // 在指定状态中执行射线检测
    private raycastInState(
        state: GameState,
        shooterId: string,
        origin: { x: number; y: number },
        direction: { x: number; y: number },
        range: number
    ): { hit: boolean; targetId: string | null; hitPoint: { x: number; y: number } | null } {
        let closestHit: { dist: number; targetId: string; point: { x: number; y: number } } | null = null;

        state.players.forEach((player, playerId) => {
            // 跳过自己
            if (playerId === shooterId) return;

            // 射线与球体相交检测
            const hit = this.raycastSphere(
                origin,
                direction,
                range,
                player.position,
                1  // 玩家半径
            );

            if (hit && (!closestHit || hit.dist < closestHit.dist)) {
                closestHit = {
                    dist: hit.dist,
                    targetId: playerId,
                    point: hit.point
                };
            }
        });

        if (closestHit) {
            return {
                hit: true,
                targetId: closestHit.targetId,
                hitPoint: closestHit.point
            };
        }

        return { hit: false, targetId: null, hitPoint: null };
    }

    // 射线与球体相交检测
    private raycastSphere(
        rayOrigin: { x: number; y: number },
        rayDir: { x: number; y: number },
        rayLength: number,
        sphereCenter: { x: number; y: number },
        sphereRadius: number
    ): { hit: boolean; dist: number; point: { x: number; y: number } } | null {
        // 从原点到球心的向量
        const oc = {
            x: rayOrigin.x - sphereCenter.x,
            y: rayOrigin.y - sphereCenter.y
        };

        // 二次方程系数
        const a = rayDir.x * rayDir.x + rayDir.y * rayDir.y;
        const b = 2 * (oc.x * rayDir.x + oc.y * rayDir.y);
        const c = oc.x * oc.x + oc.y * oc.y - sphereRadius * sphereRadius;

        // 计算判别式
        const discriminant = b * b - 4 * a * c;

        if (discriminant < 0) {
            return null;  // 没有交点
        }

        // 计算交点距离
        const t1 = (-b - Math.sqrt(discriminant)) / (2 * a);
        const t2 = (-b + Math.sqrt(discriminant)) / (2 * a);

        // 取最近的交点（在射线范围内）
        let t = -1;
        if (t1 > 0 && t1 < rayLength) t = t1;
        else if (t2 > 0 && t2 < rayLength) t = t2;
        else return null;

        // 计算交点
        const point = {
            x: rayOrigin.x + rayDir.x * t,
            y: rayOrigin.y + rayDir.y * t
        };

        return { hit: true, dist: t, point };
    }

    // 克隆状态
    private cloneState(state: GameState): GameState {
        const cloned: GameState = {
            players: new Map(),
            projectiles: new Map(),
            items: new Map()
        };

        state.players.forEach((player, id) => {
            cloned.players.set(id, { ...player });
        });

        state.projectiles.forEach((proj, id) => {
            cloned.projectiles.set(id, { ...proj });
        });

        state.items.forEach((item, id) => {
            cloned.items.set(id, { ...item });
        });

        return cloned;
    }

    // 客户端预测本地玩家移动
    predictLocalMovement(
        currentState: GameState,
        input: PlayerInput,
        latency: number
    ): GameState {
        const predictedState = this.cloneState(currentState);

        // 预测每个玩家（使用延迟作为步数）
        const framesToPredict = Math.ceil((latency / 1000) * this.serverFrameRate);

        for (let i = 0; i < framesToPredict; i++) {
            predictedState.players.forEach((player, playerId) => {
                // 应用输入和物理
                this.simulatePlayer(player, input, 1 / this.serverFrameRate);
            });
        }

        return predictedState;
    }

    // 模拟玩家移动（简化版）
    private simulatePlayer(
        player: Player,
        input: PlayerInput,
        dt: number
    ): void {
        const speed = 5;

        // 水平移动
        if (input.forward) player.velocity.y = -speed;
        else if (input.backward) player.velocity.y = speed;
        else player.velocity.y = 0;

        if (input.left) player.velocity.x = -speed;
        else if (input.right) player.velocity.x = speed;
        else player.velocity.x = 0;

        // 应用速度和重力
        player.position.x += player.velocity.x * dt;
        player.position.y += player.velocity.y * dt;

        // 边界检测
        player.position.x = Math.max(0, Math.min(100, player.position.x));
        if (player.position.y < 0) {
            player.position.y = 0;
            player.onGround = true;
        }
    }

    // 平滑校正（从错误位置渐变到正确位置）
    smoothCorrection(
        currentPos: { x: number; y: number },
        targetPos: { x: number; y: number },
        correctionSpeed: number
    ): { x: number; y: number } {
        const dx = targetPos.x - currentPos.x;
        const dy = targetPos.y - currentPos.y;

        // 渐变移动
        return {
            x: currentPos.x + dx * correctionSpeed,
            y: currentPos.y + dy * correctionSpeed
        };
    }
}
```

## 六、排行榜系统

### 6.1 分数存储与排行算法

排行榜是游戏中常见的功能模块，用于展示玩家的成绩排名。实现一个高效的排行榜需要考虑数据存储、排行算法、查询优化等多个方面。

```typescript
// 排行榜系统实现
import { createClient, RedisClientType } from 'redis';

// 排行榜条目
interface LeaderboardEntry {
    playerId: string;
    playerName: string;
    score: number;
    rank: number;
    timestamp: number;
    metadata?: Record<string, any>;
}

// 玩家分数
interface PlayerScore {
    playerId: string;
    score: number;
    name: string;
}

// 排行榜服务
class LeaderboardService {
    private redis!: RedisClientType;
    private leaderboardKey: string = 'game:leaderboard';
    private playerDataKey: string = 'game:player:';

    // 初始化Redis连接
    async init(): Promise<void> {
        this.redis = createClient({
            url: 'redis://localhost:6379'
        });

        this.redis.on('error', (err) => {
            console.error('Redis错误:', err);
        });

        await this.redis.connect();
    }

    // 提交分数
    async submitScore(
        playerId: string,
        playerName: string,
        score: number,
        metadata?: Record<string, any>
    ): Promise<LeaderboardEntry> {
        // 更新玩家分数（使用ZSET，score作为分数）
        await this.redis.zAdd(this.leaderboardKey, {
            score: score,
            value: playerId
        });

        // 存储玩家数据
        await this.redis.hSet(`${this.playerDataKey}${playerId}`, {
            name: playerName,
            score: score.toString(),
            timestamp: Date.now().toString(),
            ...(metadata ? JSON.stringify(metadata) : '{}')
        });

        // 获取排名
        const rank = await this.getRank(playerId);

        return {
            playerId,
            playerName,
            score,
            rank: rank !== null ? rank + 1 : 0,
            timestamp: Date.now(),
            metadata
        };
    }

    // 获取玩家排名
    async getRank(playerId: string): Promise<number | null> {
        const rank = await this.redis.zRank(this.leaderboardKey, playerId);
        return rank;
    }

    // 获取玩家分数
    async getScore(playerId: string): Promise<number | null> {
        const score = await this.redis.zScore(this.leaderboardKey, playerId);
        return score;
    }

    // 获取TOP N排行榜
    async getTopN(count: number = 10): Promise<LeaderboardEntry[]> {
        // 获取最高分的成员（降序）
        const results = await this.redis.zRangeWithScores(
            this.leaderboardKey,
            0,
            count - 1,
            { REV: true }
        );

        const entries: LeaderboardEntry[] = [];

        for (let i = 0; i < results.length; i++) {
            const result = results[i];
            const playerId = result.value;
            const playerData = await this.redis.hGetAll(`${this.playerDataKey}${playerId}`);

            entries.push({
                playerId,
                playerName: playerData.name || 'Unknown',
                score: result.score,
                rank: i + 1,
                timestamp: parseInt(playerData.timestamp || '0'),
                metadata: playerData.metadata ? JSON.parse(playerData.metadata) : undefined
            });
        }

        return entries;
    }

    // 获取玩家周围的排名
    async getAroundPlayer(
        playerId: string,
        range: number = 5
    ): Promise<LeaderboardEntry[]> {
        const rank = await this.getRank(playerId);

        if (rank === null) {
            return [];
        }

        // 计算范围
        const start = Math.max(0, rank - range);
        const stop = rank + range;

        // 获取范围内的成员
        const results = await this.redis.zRangeWithScores(
            this.leaderboardKey,
            start,
            stop,
            { REV: true }
        );

        const entries: LeaderboardEntry[] = [];

        for (let i = 0; i < results.length; i++) {
            const result = results[i];
            const pid = result.value;
            const playerData = await this.redis.hGetAll(`${this.playerDataKey}${pid}`);

            entries.push({
                playerId: pid,
                playerName: playerData.name || 'Unknown',
                score: result.score,
                rank: start + i + 1,
                timestamp: parseInt(playerData.timestamp || '0'),
                metadata: playerData.metadata ? JSON.parse(playerData.metadata) : undefined
            });
        }

        return entries;
    }

    // 分页获取排行榜
    async getByPage(
        page: number,
        pageSize: number = 10
    ): Promise<{ entries: LeaderboardEntry[]; total: number; totalPages: number }> {
        const start = (page - 1) * pageSize;
        const stop = start + pageSize - 1;

        // 获取总数
        const total = await this.redis.zCard(this.leaderboardKey);

        // 获取分页数据
        const results = await this.redis.zRangeWithScores(
            this.leaderboardKey,
            start,
            stop,
            { REV: true }
        );

        const entries: LeaderboardEntry[] = [];

        for (let i = 0; i < results.length; i++) {
            const result = results[i];
            const playerId = result.value;
            const playerData = await this.redis.hGetAll(`${this.playerDataKey}${playerId}`);

            entries.push({
                playerId,
                playerName: playerData.name || 'Unknown',
                score: result.score,
                rank: start + i + 1,
                timestamp: parseInt(playerData.timestamp || '0'),
                metadata: playerData.metadata ? JSON.parse(playerData.metadata) : undefined
            });
        }

        return {
            entries,
            total,
            totalPages: Math.ceil(total / pageSize)
        };
    }

    // 获取玩家的分数百分比排名
    async getPercentile(playerId: string): Promise<number | null> {
        const rank = await this.getRank(playerId);
        const total = await this.redis.zCard(this.leaderboardKey);

        if (rank === null || total === 0) {
            return null;
        }

        // 百分比 = (总人数 - 排名) / 总人数 * 100
        return ((total - rank - 1) / total) * 100;
    }

    // 删除玩家分数
    async removePlayer(playerId: string): Promise<void> {
        await this.redis.zRem(this.leaderboardKey, playerId);
        await this.redis.del(`${this.playerDataKey}${playerId}`);
    }

    // 清理旧数据（保留TOP N）
    async cleanupOldEntries(keepTopN: number = 1000): Promise<number> {
        const total = await this.redis.zCard(this.leaderboardKey);

        if (total <= keepTopN) {
            return 0;
        }

        // 删除TOP N之外的玩家
        const removed = await this.redis.zRemRange(
            this.leaderboardKey,
            0,
            total - keepTopN - 1
        );

        return removed;
    }

    // 关闭连接
    async close(): Promise<void> {
        await this.redis.quit();
    }
}

// 简化版本地排行榜（用于客户端）
class LocalLeaderboard {
    private entries: LeaderboardEntry[] = [];
    private maxEntries: number = 100;
    private storageKey: string = 'local_leaderboard';

    constructor() {
        this.loadFromStorage();
    }

    // 添加分数
    add(entry: Omit<LeaderboardEntry, 'rank'>): LeaderboardEntry {
        const newEntry: LeaderboardEntry = {
            ...entry,
            rank: 0
        };

        // 添加新分数
        this.entries.push(newEntry);

        // 按分数排序
        this.entries.sort((a, b) => b.score - a.score);

        // 限制数量
        if (this.entries.length > this.maxEntries) {
            this.entries = this.entries.slice(0, this.maxEntries);
        }

        // 更新排名
        this.entries.forEach((e, index) => {
            e.rank = index + 1;
        });

        // 保存到本地存储
        this.saveToStorage();

        // 返回新条目的排名
        return this.entries.find(e => e.playerId === entry.playerId)!;
    }

    // 获取TOP N
    getTopN(count: number = 10): LeaderboardEntry[] {
        return this.entries.slice(0, count);
    }

    // 获取玩家排名
    getRank(playerId: string): LeaderboardEntry | null {
        return this.entries.find(e => e.playerId === playerId) || null;
    }

    // 获取玩家周围的排名
    getAroundPlayer(playerId: string, range: number = 5): LeaderboardEntry[] {
        const index = this.entries.findIndex(e => e.playerId === playerId);

        if (index === -1) {
            return [];
        }

        const start = Math.max(0, index - range);
        const stop = Math.min(this.entries.length - 1, index + range);

        return this.entries.slice(start, stop + 1);
    }

    // 分页获取
    getByPage(page: number, pageSize: number = 10): LeaderboardEntry[] {
        const start = (page - 1) * pageSize;
        const stop = start + pageSize;

        return this.entries.slice(start, stop);
    }

    // 获取总数
    getTotal(): number {
        return this.entries.length;
    }

    // 清除所有数据
    clear(): void {
        this.entries = [];
        localStorage.removeItem(this.storageKey);
    }

    // 从本地存储加载
    private loadFromStorage(): void {
        try {
            const data = localStorage.getItem(this.storageKey);
            if (data) {
                this.entries = JSON.parse(data);
            }
        } catch (e) {
            console.error('加载排行榜失败:', e);
            this.entries = [];
        }
    }

    // 保存到本地存储
    private saveToStorage(): void {
        try {
            localStorage.setItem(this.storageKey, JSON.stringify(this.entries));
        } catch (e) {
            console.error('保存排行榜失败:', e);
        }
    }
}
```

```typescript
                </button>
```

```typescript
// 存档管理器
class SaveManager {
    private saveKey: string = 'game_save';
    private backupKey: string = 'game_save_backup';
    private encryptionKey?: string;
    private currentVersion: string = '1.0.0';

    constructor(encryptionKey?: string) {
        this.encryptionKey = encryptionKey;
    }

    // 保存游戏
    async saveGame(saveData: Partial<GameSave>): Promise<boolean> {
        try {
            // 创建完整存档
            const fullSave: GameSave = {
                version: this.currentVersion,
                timestamp: Date.now(),
                playerId: saveData.playerId || 'default',
                gameState: saveData.gameState || this.getDefaultGameState(),
                settings: saveData.settings || this.getDefaultSettings(),
                achievements: saveData.achievements || [],
                inventory: saveData.inventory || [],
                statistics: saveData.statistics || this.getDefaultStatistics()
            };

            // 序列化为JSON
            let saveString = JSON.stringify(fullSave);

            // 加密（如果设置了密钥）
            if (this.encryptionKey) {
                saveString = this.encrypt(saveString);
            }

            // 添加校验和
            const checksum = this.calculateChecksum(saveString);
            const savePackage = {
                data: saveString,
                checksum,
                version: this.currentVersion
            };

            // 保存到本地存储
            localStorage.setItem(this.saveKey, JSON.stringify(savePackage));

            // 创建备份
            localStorage.setItem(this.backupKey, JSON.stringify(savePackage));

            console.log('游戏已保存', fullSave);
            return true;
        } catch (error) {
            console.error('保存失败:', error);
            return false;
        }
    }

    // 加载游戏
    async loadGame(): Promise<GameSave | null> {
        try {
            const savePackage = localStorage.getItem(this.saveKey);

            if (!savePackage) {
                console.log('没有找到存档');
                return null;
            }

            const { data, checksum, version } = JSON.parse(savePackage);

            // 验证校验和
            if (!this.verifyChecksum(data, checksum)) {
                console.error('存档校验失败，尝试使用备份...');
                return this.loadFromBackup();
            }

            // 解密
            let saveString = data;
            if (this.encryptionKey) {
                saveString = this.decrypt(saveString);
            }

            // 解析存档
            const saveData = JSON.parse(saveString) as GameSave;

            // 处理版本迁移
            if (version !== this.currentVersion) {
                const migratedData = this.migrateData(saveData, version);
                return migratedData;
            }

            return saveData;
        } catch (error) {
            console.error('加载失败:', error);
            return this.loadFromBackup();
        }
    }

    // 从备份加载
    private async loadFromBackup(): Promise<GameSave | null> {
        try {
            const backupPackage = localStorage.getItem(this.backupKey);

            if (!backupPackage) {
                console.log('没有备份存档');
                return null;
            }

            const { data, checksum } = JSON.parse(backupPackage);

            if (!this.verifyChecksum(data, checksum)) {
                console.error('备份校验也失败了');
                return null;
            }

            let saveString = data;
            if (this.encryptionKey) {
                saveString = this.decrypt(saveString);
            }

            return JSON.parse(saveString);
        } catch (error) {
            console.error('备份加载失败:', error);
            return null;
        }
    }

    // 删除存档
    deleteGame(): boolean {
        try {
            localStorage.removeItem(this.saveKey);
            localStorage.removeItem(this.backupKey);
            return true;
        } catch (error) {
            console.error('删除存档失败:', error);
            return false;
        }
    }

    // 检查存档是否存在
    hasSave(): boolean {
        return localStorage.getItem(this.saveKey) !== null;
    }

    // 获取存档信息（不加载完整数据）
    getSaveInfo(): { timestamp: number; version: string; playTime: number } | null {
        try {
            const savePackage = localStorage.getItem(this.saveKey);

            if (!savePackage) {
                return null;
            }

            const { data } = JSON.parse(savePackage);
            let saveString = data;

            if (this.encryptionKey) {
                saveString = this.decrypt(saveString);
            }

            return {
                timestamp: saveData.timestamp,
                version: saveData.version,
                playTime: saveData.statistics?.totalPlayTime || 0
            };
        } catch {
            return null;
        }
    }

    // 导出存档（用于云同步或迁移）
    exportSave(): string | null {
        try {
            const savePackage = localStorage.getItem(this.saveKey);
            if (!savePackage) return null;

            // Base64编码
            return btoa(savePackage);
        } catch (error) {
            console.error('导出存档失败:', error);
            return null;
        }
    }

    // 导入存档
    async importSave(base64Data: string): Promise<boolean> {
        try {
            const savePackage = JSON.parse(atob(base64Data));
            localStorage.setItem(this.saveKey, JSON.stringify(savePackage));
            return true;
        } catch (error) {
            console.error('导入存档失败:', error);
            return false;
        }
    }

    // 加密数据
    private encrypt(data: string): string {
        return CryptoJS.AES.encrypt(data, this.encryptionKey!).toString();
    }

    // 解密数据
    private decrypt(encryptedData: string): string {
        const bytes = CryptoJS.AES.decrypt(encryptedData, this.encryptionKey!);
        return bytes.toString(CryptoJS.enc.Utf8);
    }

    // 计算校验和
    private calculateChecksum(data: string): string {
        return CryptoJS.SHA256(data).toString();
    }

    // 验证校验和
    private verifyChecksum(data: string, checksum: string): boolean {
        return this.calculateChecksum(data) === checksum;
    }

    // 版本迁移
    private migrateData(saveData: GameSave, fromVersion: string): GameSave {
        console.log(`迁移存档从 ${fromVersion} 到 ${this.currentVersion}`);

        // 版本迁移逻辑
        if (this.compareVersion(fromVersion, '1.0.0') < 0) {
            // 1.0.0之前的版本迁移
        }

        if (this.compareVersion(fromVersion, '1.1.0') < 0) {
            // 迁移到1.1.0
        }

        // 更新版本号
        saveData.version = this.currentVersion;

        return saveData;
    }

    // 比较版本号
    private compareVersion(v1: string, v2: string): number {
        const parts1 = v1.split('.').map(Number);
        const parts2 = v2.split('.').map(Number);

        for (let i = 0; i < Math.max(parts1.length, parts2.length); i++) {
            const p1 = parts1[i] || 0;
            const p2 = parts2[i] || 0;

            if (p1 > p2) return 1;
            if (p1 < p2) return -1;
        }

        return 0;
    }

    // 获取默认游戏状态
    private getDefaultGameState(): GameState {
        return {
            currentLevel: 1,
            position: { x: 0, y: 0, z: 0 },
            health: 100,
            maxHealth: 100,
            energy: 100,
            maxEnergy: 100,
            experience: 0,
            level: 1,
            gold: 0,

        };
    }

        };
    }

    }

    // 上传存档到云端

        try {
            const response = await fetch(`${this.apiBaseUrl}/saves/upload`, {
                method: 'POST',
                headers: {
                    'Content-Type': 'application/json',
                    'Authorization': `Bearer ${this.authToken}`
                },
                body: JSON.stringify({
                    playerId: saveData.playerId,
                    saveData: saveData,
                    timestamp: Date.now()
                })
            });

            return response.ok;
        } catch (error) {
            console.error('云端上传失败:', error);
            return false;
        }
    }

    // 从云端下载存档
    async downloadFromCloud(playerId: string): Promise<GameSave | null> {
        try {
            const response = await fetch(
                `${this.apiBaseUrl}/saves/download/${playerId}`,
                {
                    headers: {
                        'Authorization': `Bearer ${this.authToken}`
                    }
                }
            );

            if (!response.ok) {
                return null;
            }

            const data = await response.json();
            return data.saveData;
        } catch (error) {
            console.error('云端下载失败:', error);
            return null;
        }
    }

    // 同步存档（比较本地和云端，选择最新的）
    async syncSaves(localSave: GameSave): Promise<GameSave | null> {
        const cloudSave = await this.downloadFromCloud(localSave.playerId);

        if (!cloudSave) {
            // 云端没有存档，上传本地
            await this.uploadToCloud(localSave);
            return localSave;
        }

        if (cloudSave.timestamp > localSave.timestamp) {
            // 云端更新，下载并返回
            return cloudSave;
        } else {
            // 本地更新，上传到云端
            await this.uploadToCloud(localSave);
            return localSave;
        }
    }
```

### 9.1 60FPS保障

游戏性能优化的核心目标是保持稳定的60FPS（每帧约16.67毫秒）。任何导致帧率下降的代码都需要优化。

```typescript
// 游戏性能优化系统
class PerformanceOptimizer {
    // 帧时间追踪
    private frameTimes: number[] = [];
    private maxFrameTimeSamples: number = 60;

    // FPS计算
    private lastFrameTime: number = 0;
    private fps: number = 60;
    private frameCount: number = 0;
    private fpsUpdateInterval: number = 500;  // 每500ms更新一次FPS
    private lastFpsUpdate: number = 0;

    // 对象池
    private objectPools: Map<string, any[]> = new Map();
    private poolSizes: Map<string, number> = new Map();

    // 脏矩形渲染
    private dirtyRectangles: Set<string> = new Set();
    private lastRenderedAreas: Map<string, { x: number; y: number; w: number; h: number }> = new Map();

    // 更新帧时间
    updateFrameTime(currentTime: number): void {
        if (this.lastFrameTime > 0) {
            const frameTime = currentTime - this.lastFrameTime;
            this.frameTimes.push(frameTime);

            if (this.frameTimes.length > this.maxFrameTimeSamples) {
                this.frameTimes.shift();
            }
        }

        this.lastFrameTime = currentTime;

        // 更新FPS显示
        this.frameCount++;
        if (currentTime - this.lastFpsUpdate > this.fpsUpdateInterval) {
            this.fps = Math.round(this.frameCount * 1000 / (currentTime - this.lastFpsUpdate));
            this.frameCount = 0;
            this.lastFpsUpdate = currentTime;
        }
    }

    // 获取当前FPS
    getFPS(): number {
        return this.fps;
    }

    // 获取平均帧时间
    getAverageFrameTime(): number {
        if (this.frameTimes.length === 0) return 0;

        const sum = this.frameTimes.reduce((a, b) => a + b, 0);
        return sum / this.frameTimes.length;
    }

    // 获取最大帧时间
    getMaxFrameTime(): number {
        return Math.max(...this.frameTimes, 0);
    }

    // 获取帧时间抖动（用于检测卡顿）
    getFrameTimeJitter(): number {
        if (this.frameTimes.length < 2) return 0;

        const avg = this.getAverageFrameTime();
        const variance = this.frameTimes.reduce((sum, t) => sum + (t - avg) ** 2, 0) / this.frameTimes.length;

        return Math.sqrt(variance);
    }

    // 检测是否需要优化
    needsOptimization(): boolean {
        return this.getFPS() < 55 || this.getFrameTimeJitter() > 3;
    }

    // 对象池 - 获取对象
    getFromPool<T>(poolName: string, factory: () => T): T {
        const pool = this.objectPools.get(poolName);

        if (pool && pool.length > 0) {
            return pool.pop()!;
        }

        return factory();
    }

    // 对象池 - 回收对象
    returnToPool<T>(poolName: string, obj: T): void {
        if (!this.objectPools.has(poolName)) {
            this.objectPools.set(poolName, []);
        }

        const pool = this.objectPools.get(poolName)!;
        const maxSize = this.poolSizes.get(poolName) || 100;

        if (pool.length < maxSize) {
            pool.push(obj);
        }
    }

    // 设置对象池大小
    setPoolSize(poolName: string, size: number): void {
        this.poolSizes.set(poolName, size);
    }

    // 清理未使用的对象池
    cleanPools(): void {
        const usedMemory = this.getUsedMemory();

        // 如果内存使用过高，清理对象池
        if (usedMemory > 100 * 1024 * 1024) {  // 100MB
            this.objectPools.forEach((pool, name) => {
                if (pool.length > 10) {
                    // 只保留10个
                    this.objectPools.set(name, pool.slice(-10));
                }
            });
        }
    }

    // 标记脏矩形
    markDirty(id: string, x: number, y: number, w: number, h: number): void {
        this.dirtyRectangles.add(id);
        this.lastRenderedAreas.set(id, { x, y, w, h });
    }

    // 清除脏矩形
    clearDirty(id: string): void {
        this.dirtyRectangles.delete(id);
        this.lastRenderedAreas.delete(id);
    }

    // 获取所有脏矩形
    getDirtyRectangles(): { x: number; y: number; w: number; h: number }[] {
        const rects: { x: number; y: number; w: number; h: number }[] = [];

        this.lastRenderedAreas.forEach((rect, id) => {
            if (this.dirtyRectangles.has(id)) {
                rects.push(rect);
            }
        });

        return rects;
    }

    // 内存使用检测
    getUsedMemory(): number {
        if (typeof performance !== 'undefined' && 'memory' in performance) {
            return (performance as any).memory.usedJSHeapSize;
        }
        return 0;
    }

    // 垃圾回收提示（仅在需要时调用）
    suggestGC(): void {
        const usedMemory = this.getUsedMemory();

        if (usedMemory > 200 * 1024 * 1024) {  // 200MB
            console.warn('内存使用过高，建议手动触发垃圾回收');
        }
    }

    // 节流函数工厂
    throttle<T extends (...args: any[]) => any>(
        func: T,
        limit: number
    ): (...args: Parameters<T>) => void {
        let inThrottle: boolean = false;

        return function(this: any, ...args: Parameters<T>) {
            if (!inThrottle) {
                func.apply(this, args);
                inThrottle = true;
                setTimeout(() => inThrottle = false, limit);
            }
        };
    }

    // 防抖函数工厂
    debounce<T extends (...args: any[]) => any>(
        func: T,
        wait: number
    ): (...args: Parameters<T>) => void {
        let timeout: number | null = null;

        return function(this: any, ...args: Parameters<T>) {
            if (timeout !== null) {
                clearTimeout(timeout);
            }

            timeout = window.setTimeout(() => {
                func.apply(this, args);
            }, wait);
        };
    }

    // 帧同步函数（确保在指定帧率下运行）
    createFrameSync(callback: () => void, targetFPS: number = 60): () => void {
        let accumulated = 0;
        const frameTime = 1000 / targetFPS;
        let lastTime = 0;

        return (currentTime: number) => {
            if (lastTime === 0) {
                lastTime = currentTime;
            }

            const deltaTime = currentTime - lastTime;
            lastTime = currentTime;
            accumulated += deltaTime;

            while (accumulated >= frameTime) {
                callback();
                accumulated -= frameTime;
            }
        };
    }

    // 性能报告
    getPerformanceReport(): {
        fps: number;
        avgFrameTime: number;
        maxFrameTime: number;
        jitter: number;
        usedMemory: number;
        poolStats: { name: string; size: number }[];
    } {
        const poolStats: { name: string; size: number }[] = [];
        this.objectPools.forEach((pool, name) => {
            poolStats.push({ name, size: pool.length });
        });

        return {
            fps: this.getFPS(),
            avgFrameTime: this.getAverageFrameTime(),
            maxFrameTime: this.getMaxFrameTime(),
            jitter: this.getFrameTimeJitter(),
            usedMemory: this.getUsedMemory(),
            poolStats
        };
    }
}

// 游戏循环优化示例
class OptimizedGameLoop {
    private optimizer: PerformanceOptimizer = new PerformanceOptimizer();
    private updateCallback: (dt: number) => void = () => {};
    private renderCallback: () => void = () => {};
    private isRunning: boolean = false;
    private animationFrameId: number = 0;

    // 固定时间步长
    private readonly FIXED_TIME_STEP = 1000 / 60;  // 60Hz
    private readonly MAX_FRAME_SKIP = 5;  // 最大跳帧数

    // 时间追踪
    private lastTime: number = 0;
    private accumulator: number = 0;

    constructor() {
        this.loop = this.loop.bind(this);
    }

    // 设置更新回调
    onUpdate(callback: (dt: number) => void): void {
        this.updateCallback = callback;
    }

    // 设置渲染回调
    onRender(callback: () => void): void {
        this.renderCallback = callback;
    }

    // 开始循环
    start(): void {
        if (this.isRunning) return;

        this.isRunning = true;
        this.lastTime = performance.now();
        this.accumulator = 0;

        this.animationFrameId = requestAnimationFrame(this.loop);
    }

    // 停止循环
    stop(): void {
        this.isRunning = false;
        cancelAnimationFrame(this.animationFrameId);
    }

    // 主循环
    private loop(currentTime: number): void {
        if (!this.isRunning) return;

        // 计算时间差
        let deltaTime = currentTime - this.lastTime;
        this.lastTime = currentTime;

        // 防止时间差过大（标签页切换等）
        if (deltaTime > 250) {
            deltaTime = 250;
        }

        // 累加时间
        this.accumulator += deltaTime;

        // 固定时间步长的逻辑更新
        let updates = 0;
        while (this.accumulator >= this.FIXED_TIME_STEP && updates < this.MAX_FRAME_SKIP) {
            this.updateCallback(this.FIXED_TIME_STEP / 1000);  // 转换为秒
            this.accumulator -= this.FIXED_TIME_STEP;
            updates++;
        }

        // 渲染（使用剩余时间进行插值）
        const alpha = this.accumulator / this.FIXED_TIME_STEP;
        this.renderCallback();

        // 更新性能统计
        this.optimizer.updateFrameTime(currentTime);

        // 检查是否需要GC
        this.optimizer.suggestGC();

        // 调度下一帧
        this.animationFrameId = requestAnimationFrame(this.loop);
    }

    // 获取性能报告
    getPerformanceReport(): any {
        return this.optimizer.getPerformanceReport();
    }
}
```

```typescript
class ImageCache {
    private cache: Map<string, HTMLImageElement> = new Map();
    private loading: Map<string, Promise<HTMLImageElement>> = new Map();
    private maxCacheSize: number = 50;

    // 加载图片
    async load(url: string): Promise<HTMLImageElement> {
        // 检查缓存
        if (this.cache.has(url)) {
            return this.cache.get(url)!;
        }

        // 检查是否正在加载
        if (this.loading.has(url)) {
            return this.loading.get(url)!;
        }

        // 加载图片
        const loadPromise = new Promise<HTMLImageElement>((resolve, reject) => {
            const img = new Image();

            img.onload = () => {
                // 添加到缓存
                if (this.cache.size >= this.maxCacheSize) {
                    // 移除最旧的图片
                    const firstKey = this.cache.keys().next().value;
                    if (firstKey) {
                        this.cache.delete(firstKey);
                    }
                }

                this.cache.set(url, img);
                this.loading.delete(url);
                resolve(img);
            };

            img.onerror = reject;
            img.src = url;
        });

        this.loading.set(url, loadPromise);
        return loadPromise;
    }

    // 预加载多张图片
    async preload(urls: string[]): Promise<void> {
        await Promise.all(urls.map(url => this.load(url)));
    }

    // 获取缓存的图片
    get(url: string): HTMLImageElement | undefined {
        return this.cache.get(url);
    }

    // 检查图片是否已缓存
    has(url: string): boolean {
        return this.cache.has(url);
    }

    // 清理缓存
    clear(): void {
        this.cache.forEach(img => {
            img.src = '';  // 释放图片资源
        });
        this.cache.clear();
    }

    // 设置缓存大小
    setMaxSize(size: number): void {

class BatchRenderer {
    private ctx: CanvasRenderingContext2D;
    private batchedDraws: Array<{
        type: 'rect' | 'circle' | 'image' | 'text';
        data: any;
        style?: any;
    }> = [];

    constructor(ctx: CanvasRenderingContext2D) {
        this.ctx = ctx;
    }

    // 添加矩形绘制
    addRect(x: number, y: number, w: number, h: number, style?: any): void {
        this.batchedDraws.push({
            type: 'rect',
            data: { x, y, w, h },
            style
        });
    }

    // 添加圆形绘制
    addCircle(x: number, y: number, r: number, style?: any): void {
        this.batchedDraws.push({
            type: 'circle',
            data: { x, y, r },
            style
        });
    }

    // 添加图片绘制
    addImage(img: HTMLImageElement, x: number, y: number, w?: number, h?: number): void {
        this.batchedDraws.push({
            type: 'image',
            data: { img, x, y, w, h }
        });
    }

    // 添加文本绘制
    addText(text: string, x: number, y: number, style?: any): void {
        this.batchedDraws.push({
            type: 'text',
            data: { text, x, y },
            style
        });
    }

    // 执行批量渲染
    flush(): void {
        const ctx = this.ctx;

        for (const draw of this.batchedDraws) {
            switch (draw.type) {
                case 'rect':
                    ctx.fillStyle = draw.style?.fill || '#000';
                    ctx.fillRect(draw.data.x, draw.data.y, draw.data.w, draw.data.h);
                    break;

                case 'circle':
                    ctx.beginPath();
                    ctx.arc(draw.data.x, draw.data.y, draw.data.r, 0, Math.PI * 2);
                    ctx.fillStyle = draw.style?.fill || '#000';
                    ctx.fill();
                    break;

                case 'image':
                    if (draw.data.img.complete) {
                        ctx.drawImage(
                            draw.data.img,
                            draw.data.x,
                            draw.data.y,
                            draw.data.w,
                            draw.data.h
                        );
                    }
                    break;

                case 'text':
                    ctx.font = draw.style?.font || '16px Arial';
                    ctx.fillStyle = draw.style?.fill || '#000';
                    ctx.fillText(draw.data.text, draw.data.x, draw.data.y);
                    break;
            }
        }

        // 清空批处理队列
        this.batchedDraws = [];
    }

// 四叉树实现
class QuadTree {
    private bounds: { x: number; y: number; width: number; height: number };
    private maxObjects: number;
    private maxLevels: number;
    private level: number;
    private objects: Array<{ x: number; y: number; width: number; height: number; data: any }> = [];
    private nodes: QuadTree[] = [];
    private isSplit: boolean = false;

    constructor(
        bounds: { x: number; y: number; width: number; height: number },
        maxObjects: number = 10,
        maxLevels: number = 5,
        level: number = 0
    ) {
        this.bounds = bounds;
        this.maxObjects = maxObjects;
        this.maxLevels = maxLevels;
        this.level = level;
    }

    // 分割节点
    private split(): void {
        const halfWidth = this.bounds.width / 2;
        const halfHeight = this.bounds.height / 2;
        const x = this.bounds.x;
        const y = this.bounds.y;

        // 右上
        this.nodes[0] = new QuadTree(
            { x: x + halfWidth, y, width: halfWidth, height: halfHeight },
            this.maxObjects,
            this.maxLevels,
            this.level + 1
        );

        // 左上
        this.nodes[1] = new QuadTree(
            { x, y, width: halfWidth, height: halfHeight },
            this.maxObjects,
            this.maxLevels,
            this.level + 1
        );

        // 右下
        this.nodes[2] = new QuadTree(
            { x: x + halfWidth, y: y + halfHeight, width: halfWidth, height: halfHeight },
            this.maxObjects,
            this.maxLevels,
            this.level + 1
        );

        // 左下
        this.nodes[3] = new QuadTree(
            { x, y: y + halfHeight, width: halfWidth, height: halfHeight },
            this.maxObjects,
            this.maxLevels,
            this.level + 1
        );

        this.isSplit = true;
    }

    // 获取对象所属的象限
    private getIndices(obj: { x: number; y: number; width: number; height: number }): number[] {
        const indices: number[] = [];
        const verticalMidpoint = this.bounds.x + this.bounds.width / 2;
        const horizontalMidpoint = this.bounds.y + this.bounds.height / 2;

        const topQuadrant = obj.y < horizontalMidpoint;
        const bottomQuadrant = obj.y + obj.height > horizontalMidpoint;
        const leftQuadrant = obj.x < verticalMidpoint;
        const rightQuadrant = obj.x + obj.width > verticalMidpoint;

        if (topQuadrant) {
            if (rightQuadrant) indices.push(0);
            if (leftQuadrant) indices.push(1);
        }
        if (bottomQuadrant) {
            if (rightQuadrant) indices.push(2);
            if (leftQuadrant) indices.push(3);
        }

        return indices;
    }

    // 插入对象
    insert(obj: { x: number; y: number; width: number; height: number; data?: any }): void {
        if (this.isSplit) {
            const indices = this.getIndices(obj);
            for (const index of indices) {
                this.nodes[index].insert(obj);
            }
            return;
        }

        this.objects.push(obj);

        if (this.objects.length > this.maxObjects && this.level < this.maxLevels) {
            if (!this.isSplit) {
                this.split();
            }

            // 重新分配对象
            for (const existingObj of this.objects) {
                const indices = this.getIndices(existingObj);
                for (const index of indices) {
                    this.nodes[index].insert(existingObj);
                }
            }

            this.objects = [];
        }
    }

    // 查询范围内所有对象
    query(range: { x: number; y: number; width: number; height: number }): any[] {
        const found: any[] = [];

        if (!this.intersects(range)) {
            return found;
        }

        for (const obj of this.objects) {
            if (this.objectIntersects(obj, range)) {
                found.push(obj.data || obj);
            }
        }

        if (this.isSplit) {
            for (const node of this.nodes) {
                found.push(...node.query(range));
            }
        }

        return found;
    }

    // 检测范围是否相交
    private intersects(range: { x: number; y: number; width: number; height: number }): boolean {
        return !(
            range.x > this.bounds.x + this.bounds.width ||
            range.x + range.width < this.bounds.x ||
            range.y > this.bounds.y + this.bounds.height ||
            range.y + range.height < this.bounds.y
        );
    }

    // 检测对象是否相交
    private objectIntersects(
        a: { x: number; y: number; width: number; height: number },
        b: { x: number; y: number; width: number; height: number }
    ): boolean {
        return !(
            a.x > b.x + b.width ||
            a.x + a.width < b.x ||
            a.y > b.y + b.height ||
            a.y + a.height < b.y
        );
    }

    // 清空四叉树
    clear(): void {
        this.objects = [];
        for (const node of this.nodes) {
            node.clear();
        }
        this.nodes = [];
        this.isSplit = false;
    }
}
```

## 十、案例：消除游戏

### 10.1 核心算法

消除游戏的核心算法包括：匹配检测、方块下落、行列消除动画、连锁反应检测等。

```typescript
// 消除游戏核心实现
class Match3Game {
    // 游戏配置
    private readonly CONFIG = {
        ROWS: 8,
        COLS: 8,
        CELL_SIZE: 60,
        BLOCK_TYPES: 6,  // 不同颜色/类型的方块
        MIN_MATCH: 3,    // 最少匹配数量
        ANIMATION_DURATION: 300,  // 动画时长（毫秒）
    };

    // 方块类型
    private readonly BLOCK_COLORS = [
        '#e74c3c',  // 红
        '#3498db',  // 蓝
        '#2ecc71',  // 绿
        '#f1c40f',  // 黄
        '#9b59b6',  // 紫
        '#e67e22',  // 橙
    ];

    // 游戏状态
    private grid: number[][] = [];  // 0表示空格，其他表示方块类型
    private selectedCell: { row: number; col: number } | null = null;
    private isAnimating: boolean = false;
    private score: number = 0;
    private combo: number = 0;

    // Canvas渲染
    private canvas!: HTMLCanvasElement;
    private ctx!: CanvasRenderingContext2D;

    // 动画相关
    private animations: Animation[] = [];
    private particles: Particle[] = [];

    constructor(containerId: string) {
        this.initCanvas(containerId);
        this.initGrid();
        this.render();
        this.setupInput();
    }

    // 初始化画布
    private initCanvas(containerId: string): void {
        this.canvas = document.createElement('canvas');
        this.canvas.width = this.CONFIG.COLS * this.CONFIG.CELL_SIZE;
        this.canvas.height = this.CONFIG.ROWS * this.CONFIG.CELL_SIZE;
        this.canvas.style.background = '#1a1a2e';
        this.canvas.style.borderRadius = '8px';

        document.getElementById(containerId)?.appendChild(this.canvas);
        this.ctx = this.canvas.getContext('2d')!;
    }

    // 初始化网格
    private initGrid(): void {
        this.grid = [];

        for (let row = 0; row < this.CONFIG.ROWS; row++) {
            this.grid[row] = [];
            for (let col = 0; col < this.CONFIG.COLS; col++) {
                // 生成随机方块，确保初始没有匹配
                let type: number;
                do {
                    type = Math.floor(Math.random() * this.CONFIG.BLOCK_TYPES) + 1;
                } while (this.wouldMatch(row, col, type));

                this.grid[row][col] = type;
            }
        }
    }

    // 检查生成该类型方块是否会形成匹配
    private wouldMatch(row: number, col: number, type: number): boolean {
        // 检查横向
        if (col >= 2 &&
            this.grid[row][col - 1] === type &&
            this.grid[row][col - 2] === type) {
            return true;
        }

        // 检查纵向
        if (row >= 2 &&
            this.grid[row - 1]?.[col] === type &&
            this.grid[row - 2]?.[col] === type) {
            return true;
        }

        return false;
    }

    // 设置输入
    private setupInput(): void {
        this.canvas.addEventListener('click', (e) => {
            if (this.isAnimating) return;

            const rect = this.canvas.getBoundingClientRect();
            const x = e.clientX - rect.left;
            const y = e.clientY - rect.top;

            const col = Math.floor(x / this.CONFIG.CELL_SIZE);
            const row = Math.floor(y / this.CONFIG.CELL_SIZE);

            this.handleCellClick(row, col);
        });
    }

    // 处理格子点击
    private handleCellClick(row: number, col: number): void {
        if (row < 0 || row >= this.CONFIG.ROWS || col < 0 || col >= this.CONFIG.COLS) {
            return;
        }

        if (this.grid[row][col] === 0) {
            return;
        }

        if (this.selectedCell === null) {
            // 第一次选择
            this.selectedCell = { row, col };
        } else {
            // 第二次选择，检查是否相邻
            const dr = Math.abs(row - this.selectedCell.row);
            const dc = Math.abs(col - this.selectedCell.col);

            if ((dr === 1 && dc === 0) || (dr === 0 && dc === 1)) {
                // 相邻，尝试交换
                this.trySwap(this.selectedCell.row, this.selectedCell.col, row, col);
            }

            // 清除选择
            this.selectedCell = null;
        }

        this.render();
    }

    // 尝试交换两个方块
    private async trySwap(row1: number, col1: number, row2: number, col2: number): Promise<void> {
        this.isAnimating = true;
        this.combo = 0;

        // 交换方块
        this.swap(row1, col1, row2, col2);

        // 播放交换动画
        await this.animateSwap(row1, col1, row2, col2);

        // 检查是否有匹配
        const matches = this.findAllMatches();

        if (matches.length === 0) {
            // 没有匹配，交换回来
            this.swap(row1, col1, row2, col2);
            await this.animateSwap(row1, col1, row2, col2);
        } else {
            // 有匹配，处理消除
            await this.processMatches(matches);
        }

        this.isAnimating = false;
        this.render();
    }

    // 交换两个方块
    private swap(row1: number, col1: number, row2: number, col2: number): void {
        const temp = this.grid[row1][col1];
        this.grid[row1][col1] = this.grid[row2][col2];
        this.grid[row2][col2] = temp;
    }

    // 动画：交换
    private animateSwap(row1: number, col1: number, row2: number, col2: number): Promise<void> {
        return new Promise((resolve) => {
            const startTime = performance.now();
            const duration = this.CONFIG.ANIMATION_DURATION;

            const animate = (currentTime: number) => {
                const elapsed = currentTime - startTime;
                const progress = Math.min(elapsed / duration, 1);

                // 这里简化处理，实际应该使用easing函数
                if (progress < 1) {
                    requestAnimationFrame(animate);
                } else {
                    resolve();
                }
            };

            requestAnimationFrame(animate);
        });
    }

    // 查找所有匹配
    private findAllMatches(): { row: number; col: number }[] {
        const matches = new Set<string>();

        // 检查横向匹配
        for (let row = 0; row < this.CONFIG.ROWS; row++) {
            for (let col = 0; col < this.CONFIG.COLS - 2; col++) {
                const type = this.grid[row][col];
                if (type === 0) continue;

                if (this.grid[row][col + 1] === type &&
                    this.grid[row][col + 2] === type) {
                    // 找到匹配，继续扩展
                    let endCol = col + 2;
                    while (endCol + 1 < this.CONFIG.COLS &&
                           this.grid[row][endCol + 1] === type) {
                        endCol++;
                    }

                    for (let c = col; c <= endCol; c++) {
                        matches.add(`${row},${c}`);
                    }
                }
            }
        }

        // 检查纵向匹配
        for (let col = 0; col < this.CONFIG.COLS; col++) {
            for (let row = 0; row < this.CONFIG.ROWS - 2; row++) {
                const type = this.grid[row][col];
                if (type === 0) continue;

                if (this.grid[row + 1]?.[col] === type &&
                    this.grid[row + 2]?.[col] === type) {
                    // 找到匹配，继续扩展
                    let endRow = row + 2;
                    while (endRow + 1 < this.CONFIG.ROWS &&
                           this.grid[endRow + 1]?.[col] === type) {
                        endRow++;
                    }

                    for (let r = row; r <= endRow; r++) {
                        matches.add(`${r},${col}`);
                    }
                }
            }
        }

        // 转换为坐标数组
        return Array.from(matches).map(key => {
            const [row, col] = key.split(',').map(Number);
            return { row, col };
        });
    }

    // 处理匹配（递归消除和下落）
    private async processMatches(matches: { row: number; col: number }[]): Promise<void> {
        while (matches.length > 0) {
            this.combo++;

            // 计算得分
            const baseScore = matches.length * 10 * this.combo;
            this.score += baseScore;

            console.log(`Combo x${this.combo}! +${baseScore} 分`);

            // 播放消除动画
            await this.animateMatchRemoval(matches);

            // 消除方块
            for (const { row, col } of matches) {
                this.grid[row][col] = 0;
            }

            // 下落并生成新方块
            await this.dropBlocks();

            // 检查新的匹配
            matches = this.findAllMatches();
        }

        // 重置连击
        this.combo = 0;
    }

    // 动画：匹配消除
    private animateMatchRemoval(matches: { row: number; col: number }[]): Promise<void> {
        return new Promise((resolve) => {
            const startTime = performance.now();
            const duration = this.CONFIG.ANIMATION_DURATION;

            // 创建粒子效果
            for (const { row, col } of matches) {
                this.createParticles(
                    col * this.CONFIG.CELL_SIZE + this.CONFIG.CELL_SIZE / 2,
                    row * this.CONFIG.CELL_SIZE + this.CONFIG.CELL_SIZE / 2,
                    this.grid[row][col]
                );
            }

            const animate = (currentTime: number) => {
                const elapsed = currentTime - startTime;
                const progress = Math.min(elapsed / duration, 1);

                // 缩小动画效果
                this.render();

                const ctx = this.ctx;
                for (const { row, col } of matches) {
                    const x = col * this.CONFIG.CELL_SIZE + this.CONFIG.CELL_SIZE / 2;
                    const y = row * this.CONFIG.CELL_SIZE + this.CONFIG.CELL_SIZE / 2;
                    const scale = 1 - progress;
                    const alpha = 1 - progress;

                    ctx.save();
                    ctx.globalAlpha = alpha;
                    ctx.translate(x, y);
                    ctx.scale(scale, scale);

                    ctx.fillStyle = this.BLOCK_COLORS[this.grid[row][col] - 1];
                    ctx.fillRect(
                        -this.CONFIG.CELL_SIZE / 2,
                        -this.CONFIG.CELL_SIZE / 2,
                        this.CONFIG.CELL_SIZE,
                        this.CONFIG.CELL_SIZE
                    );

                    ctx.restore();
                }

                // 渲染粒子
                this.renderParticles(progress);

                if (progress < 1) {
                    requestAnimationFrame(animate);
                } else {
                    resolve();
                }
            };

            requestAnimationFrame(animate);
        });
    }

    // 方块下落
    private async dropBlocks(): Promise<void> {
        const drops: { fromRow: number; toRow: number; col: number; type: number }[] = [];

        // 从右到左处理每列
        for (let col = 0; col < this.CONFIG.COLS; col++) {
            let emptyRow = this.CONFIG.ROWS - 1;

            // 从底部向上找空位
            for (let row = this.CONFIG.ROWS - 1; row >= 0; row--) {
                if (this.grid[row][col] !== 0) {
                    if (row !== emptyRow) {
                        drops.push({
                            fromRow: row,
                            toRow: emptyRow,
                            col,
                            type: this.grid[row][col]
                        });
                        this.grid[emptyRow][col] = this.grid[row][col];
                        this.grid[row][col] = 0;
                    }
                    emptyRow--;
                }
            }

            // 填充新方块
            for (let row = emptyRow; row >= 0; row--) {
                const type = Math.floor(Math.random() * this.CONFIG.BLOCK_TYPES) + 1;
                this.grid[row][col] = type;

                drops.push({
                    fromRow: -1 - (emptyRow - row),  // 从屏幕外下落
                    toRow: row,
                    col,
                    type
                });
            }
        }

        // 播放下落动画
        if (drops.length > 0) {
            await this.animateDrops(drops);
        }
    }

    // 动画：方块下落
    private animateDrops(drops: { fromRow: number; toRow: number; col: number; type: number }[]): Promise<void> {
        return new Promise((resolve) => {
            const startTime = performance.now();
            const duration = this.CONFIG.ANIMATION_DURATION;

            const animate = (currentTime: number) => {
                const elapsed = currentTime - startTime;
                const progress = Math.min(elapsed / duration, 1);
                const easedProgress = this.easeOutBounce(progress);

                this.render();

                const ctx = this.ctx;

                for (const drop of drops) {
                    const currentRow = drop.fromRow + (drop.toRow - drop.fromRow) * easedProgress;
                    const x = drop.col * this.CONFIG.CELL_SIZE;
                    const y = currentRow * this.CONFIG.CELL_SIZE;

                    ctx.fillStyle = this.BLOCK_COLORS[drop.type - 1];
                    this.drawBlock(ctx, x, y, this.CONFIG.CELL_SIZE, drop.type);
                }

                if (progress < 1) {
                    requestAnimationFrame(animate);
                } else {
                    resolve();
                }
            };

            requestAnimationFrame(animate);
        });
    }

    // 缓动函数：弹性
    private easeOutBounce(t: number): number {
        const n1 = 7.5625;
        const d1 = 2.75;

        if (t < 1 / d1) {
            return n1 * t * t;
        } else if (t < 2 / d1) {
            return n1 * (t -= 1.5 / d1) * t + 0.75;
        } else if (t < 2.5 / d1) {
            return n1 * (t -= 2.25 / d1) * t + 0.9375;
        } else {
            return n1 * (t -= 2.625 / d1) * t + 0.984375;
        }
    }

    // 创建粒子效果
    private createParticles(x: number, y: number, type: number): void {
        const color = this.BLOCK_COLORS[type - 1];

        for (let i = 0; i < 8; i++) {
            const angle = (i / 8) * Math.PI * 2;
            const speed = 2 + Math.random() * 3;

            this.particles.push({
                x,
                y,
                vx: Math.cos(angle) * speed,
                vy: Math.sin(angle) * speed,
                life: 1,
                color,
                size: 4 + Math.random() * 4
            });
        }
    }

    // 渲染粒子
    private renderParticles(progress: number): void {
        const ctx = this.ctx;

        for (let i = this.particles.length - 1; i >= 0; i--) {
            const p = this.particles[i];

            p.x += p.vx;
            p.y += p.vy;
            p.vy += 0.1;  // 重力
            p.life -= 0.02;

            if (p.life <= 0) {
                this.particles.splice(i, 1);
                continue;
            }

            ctx.globalAlpha = p.life;
            ctx.fillStyle = p.color;
            ctx.beginPath();
            ctx.arc(p.x, p.y, p.size * p.life, 0, Math.PI * 2);
            ctx.fill();
        }

        ctx.globalAlpha = 1;
    }

    // 渲染游戏
    private render(): void {
        const ctx = this.ctx;

        // 清空画布
        ctx.fillStyle = '#1a1a2e';
        ctx.fillRect(0, 0, this.canvas.width, this.canvas.height);

        // 绘制网格
        this.drawGrid();

        // 绘制方块
        for (let row = 0; row < this.CONFIG.ROWS; row++) {
            for (let col = 0; col < this.CONFIG.COLS; col++) {
                if (this.grid[row][col] !== 0) {
                    const x = col * this.CONFIG.CELL_SIZE;
                    const y = row * this.CONFIG.CELL_SIZE;

                    this.drawBlock(ctx, x, y, this.CONFIG.CELL_SIZE, this.grid[row][col]);

                    // 高亮选中方块
                    if (this.selectedCell?.row === row && this.selectedCell?.col === col) {
                        ctx.strokeStyle = '#ffffff';
                        ctx.lineWidth = 3;
                        ctx.strokeRect(x + 2, y + 2, this.CONFIG.CELL_SIZE - 4, this.CONFIG.CELL_SIZE - 4);
                    }
                }
            }
        }

        // 绘制分数
        ctx.fillStyle = '#ffffff';
        ctx.font = 'bold 24px Arial';
        ctx.textAlign = 'left';
        ctx.fillText(`分数: ${this.score}`, 10, 30);
    }

    // 绘制网格线
    private drawGrid(): void {
        const ctx = this.ctx;

        ctx.strokeStyle = 'rgba(255, 255, 255, 0.1)';
        ctx.lineWidth = 1;

        // 垂直线
        for (let col = 0; col <= this.CONFIG.COLS; col++) {
            const x = col * this.CONFIG.CELL_SIZE;
            ctx.beginPath();
            ctx.moveTo(x, 0);
            ctx.lineTo(x, this.canvas.height);
            ctx.stroke();
        }

        // 水平线
        for (let row = 0; row <= this.CONFIG.ROWS; row++) {
            const y = row * this.CONFIG.CELL_SIZE;
            ctx.beginPath();
            ctx.moveTo(0, y);
            ctx.lineTo(this.canvas.width, y);
            ctx.stroke();
        }
    }

    // 绘制方块
    private drawBlock(
        ctx: CanvasRenderingContext2D,
        x: number,
        y: number,
        size: number,
        type: number
    ): void {
        const color = this.BLOCK_COLORS[type - 1];
        const padding = 4;
        const radius = 8;

        // 主体
        ctx.fillStyle = color;
        this.roundRect(ctx, x + padding, y + padding, size - padding * 2, size - padding * 2, radius);
        ctx.fill();

        // 高光
        const gradient = ctx.createLinearGradient(x, y, x, y + size);
        gradient.addColorStop(0, 'rgba(255, 255, 255, 0.3)');
        gradient.addColorStop(0.5, 'rgba(255, 255, 255, 0.1)');
        gradient.addColorStop(1, 'rgba(0, 0, 0, 0.2)');

        ctx.fillStyle = gradient;
        this.roundRect(ctx, x + padding, y + padding, size - padding * 2, size - padding * 2, radius);
        ctx.fill();

        // 内阴影
        ctx.strokeStyle = 'rgba(0, 0, 0, 0.3)';
        ctx.lineWidth = 2;
        this.roundRect(ctx, x + padding, y + padding, size - padding * 2, size - padding * 2, radius);
        ctx.stroke();
    }

    // 绘制圆角矩形
    private roundRect(
        ctx: CanvasRenderingContext2D,
        x: number,
        y: number,
        w: number,
        h: number,
        r: number
    ): void {
        ctx.beginPath();
        ctx.moveTo(x + r, y);
        ctx.lineTo(x + w - r, y);
        ctx.quadraticCurveTo(x + w, y, x + w, y + r);
        ctx.lineTo(x + w, y + h - r);
        ctx.quadraticCurveTo(x + w, y + h, x + w - r, y + h);
        ctx.lineTo(x + r, y + h);
        ctx.quadraticCurveTo(x, y + h, x, y + h - r);
        ctx.lineTo(x, y + r);
        ctx.quadraticCurveTo(x, y, x + r, y);
        ctx.closePath();
    }

    // 获取分数
    getScore(): number {
        return this.score;
    }
}

    y: number;
    vx: number;

    color: string;
    size: number;

interface Animation {

}
```

## 十一、总结

本文档系统地介绍了Web前端游戏开发的完整技术体系，涵盖了从基础的Canvas游戏开发到高级的WebGL 3D游戏、网络实时对战等核心技能。

### 核心要点回顾

**第一，游戏开发是前端技术的练兵场。** 游戏开发需要综合运用图形渲染、性能优化、网络通信、数据存储、用户交互等多种技术，这些能力可以迁移到任何前端领域。通过游戏开发，能够深入理解浏览器的工作原理、JavaScript的执行模型、CSS的渲染流程等底层知识。

**第二，选择合适的工具事半功倍。** 2D游戏开发推荐使用Phaser引擎，它提供了完整的游戏开发框架；3D游戏开发推荐使用Three.js，它是WebGL的事实标准；物理引擎推荐使用Matter.js，轻量且易用。对于微信小游戏开发，需要熟悉微信的开放能力，如登录、分享、支付、排行榜等。

**第三，性能优化是游戏开发的核心。** 游戏必须保持60FPS才能提供流畅的用户体验。性能优化包括：使用对象池减少内存分配、使用脏矩形渲染减少重绘、使用空间分区数据结构优化碰撞检测、使用请求动画帧实现稳定的游戏循环等。

**第四，网络同步是实时游戏的关键。** 帧同步和状态同步各有优劣，需要根据游戏类型选择合适的同步方案。延迟补偿技术可以缓解网络延迟带来的体验问题，包括客户端预测、服务器回溯、插值平滑等。

**第五，游戏开发的乐趣在于创造。** 从消除游戏到3D迷宫，从物理小球到实时对战，每一种游戏类型都有其独特的魅力和挑战。通过本文档的学习，希望读者能够掌握游戏开发的核心技能，并在实践中体会到创造的乐趣。
