# 前端监控体系完全指南

## 前言：为什么你的Web应用需要"监控"？

想象一下，你是一位医生，负责照顾一群病人。如果你不知道每个病人的身体状况（体温、血压、心率），你就无法判断谁需要治疗、谁正在康复、谁情况恶化了。

Web应用监控就是这个道理。如果你不监控你的Web应用：
- 你不知道用户遇到了什么错误
- 你不知道你的网站加载有多慢
- 你不知道哪些功能用户从来不用
- 你不知道服务器是否快要扛不住了

前端监控就像给Web应用装上了一系列"体检设备"，让运维人员能够7x24小时了解网站的健康状况，及时发现和解决问题。

## 二、性能监控（Performance Monitoring）

### 2.1 为什么性能监控如此重要？

研究表明，页面加载时间对用户行为和转化率有巨大影响：

| 加载时间 | 用户反应 |
|----------|----------|
| 0-2秒 | 用户感觉很流畅 |
| 2-5秒 | 用户开始不耐烦 |
| 5-10秒 | 用户很可能离开 |
| >10秒 | 超过50%的用户会离开 |

### 2.2 Core Web Vitals：谷歌的核心指标

谷歌推出了Core Web Vitals（核心网络指标），这是评估网页性能的关键指标：

```
┌─────────────────────────────────────────────────────────────┐
│                    Core Web Vitals 三大指标                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  LCP (Largest Contentful Paint)                      │   │
│  │  最大内容绘制时间                                     │   │
│  │                                                       │   │
│  │  衡量：页面主要内容加载速度                           │   │
│  │  目标：< 2.5秒                                        │   │
│  │                                                       │   │
│  │  什么是"主要内容"？                                   │   │
│  │  → 最大的图片或文本块                                 │   │
│  │  → 通常是Hero Banner或首图                            │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  FID (First Input Delay)                             │   │
│  │  首次输入延迟                                         │   │
│  │                                                       │   │
│  │  衡量：页面响应用户操作的速度                         │   │
│  │  目标：< 100毫秒                                      │   │
│  │                                                       │   │
│  │  为什么重要？                                         │   │
│  │  → 用户点击按钮，页面要多久才响应                     │   │
│  │  → 如果延迟太长，用户会觉得"点击没用"                 │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │  CLS (Cumulative Layout Shift)                       │   │
│  │  累计布局偏移                                         │   │
│  │                                                       │   │
│  │  衡量：页面视觉稳定性                                 │   │
│  │  目标：< 0.1                                          │   │
│  │                                                       │   │
│  │  什么是"布局偏移"？                                   │   │
│  │  → 用户正在阅读，突然内容跳走了                       │   │
│  │  → 或者点击按钮时，按钮突然"漂移"了                   │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 Performance API 详解

浏览器提供了`Performance` API来获取详细的性能数据：

```javascript
// performance API 示例 - 获取关键性能指标

// 等待页面加载完成
window.addEventListener("load", () => {
  // 使用 requestIdleCallback 在浏览器空闲时采集数据
  requestIdleCallback(() => {
    // 获取 Performance 对象
    const perf = performance;

    // 1. 获取关键时间点
    const timing = perf.timing;

    // DNS解析耗时
    const dnsTime = timing.domainLookupEnd - timing.domainLookupStart;

    // TCP连接耗时
    const tcpTime = timing.connectEnd - timing.connectStart;

    // SSL连接耗时（如果是HTTPS）
    const sslTime = timing.connectEnd - timing.secureConnectionStart;

    // TTFB (Time To First Byte) - 首字节时间
    const ttfb = timing.responseStart - timing.requestStart;

    // DOM解析耗时
    const domParseTime = timing.domComplete - timing.domInteractive;

    // 页面总加载时间
    const pageLoadTime = perf.loadEventEnd - perf.navigationStart;

    // 2. 获取资源加载详情
    const resources = perf.getEntriesByType("resource");

    // 遍历所有资源，分析加载时间
    resources.forEach((resource) => {
      console.log("资源URL:", resource.name);
      console.log("资源类型:", resource.initiatorType); // link, script, img, xmlhttprequest 等
      console.log("耗时:", resource.duration, "ms");

      // 如果是图片，还可以获取尺寸信息
      if (resource.initiatorType === "img" && resource.fetchStart) {
        console.log("图片加载时间:", resource.responseEnd - resource.fetchStart);
      }
    });

    // 3. 获取 Navigation Timing 数据
    const navigation = perf.getEntriesByType("navigation")[0];

    if (navigation) {
      // 构建关键时间线
      const timeline = {
        // 重定向时间
        redirect: navigation.redirectEnd - navigation.redirectStart,

        // 应用缓存时间
        appCache: navigation.domainLookupStart - navigation.fetchStart,

        // DNS查询时间
        dns: navigation.domainLookupEnd - navigation.domainLookupStart,

        // TCP连接时间
        tcp: navigation.connectEnd - navigation.connectStart,

        // SSL握手时间
        ssl: navigation.connectEnd - navigation.secureConnectionStart,

        // TTFB
        ttfb: navigation.responseStart - navigation.requestStart,

        // 内容下载时间
        download: navigation.responseEnd - navigation.responseStart,

        // DOM解析时间
        domProcessing: navigation.domComplete - navigation.domLoading,

        // DOM加载完成时间
        domReady: navigation.domContentLoadedEventEnd - navigation.navigationStart,

        // 页面完全加载时间
        loaded: navigation.loadEventEnd - navigation.navigationStart,
      };

      console.log("性能时间线:", timeline);

      // 上报性能数据到监控服务器
      reportPerformance(timeline);
    }
  });
});

// 性能数据上报函数
function reportPerformance(data) {
  // 使用 sendBeacon 确保页面关闭时也能发送数据
  navigator.sendBeacon("/api/performance", JSON.stringify(data));
}
```

### 2.4 使用 PerformanceObserver 实时监控

`PerformanceObserver`允许我们订阅性能事件，实时获取性能数据：

```javascript
// PerformanceObserver 示例 - 实时监控性能指标

// 1. 观察长任务（Long Tasks）- 主线程被阻塞超过50ms的任务
const longTaskObserver = new PerformanceObserver((list) => {
  // 获取所有长任务
  const longTasks = list.getEntries();

  longTasks.forEach((task) => {
    console.log("检测到长任务:", task.name);
    console.log("任务持续时间:", task.duration, "ms");
    console.log("任务开始时间:", task.startTime, "ms");

    // 长任务告警：如果任务超过100ms，需要优化
    if (task.duration > 100) {
      console.warn("警告：主线程被阻塞超过100ms！");
      // 可以上报到监控系统
      reportLongTask(task);
    }
  });
});

// 开始观察长任务
longTaskObserver.observe({ entryTypes: ["longtask"] });

// 2. 观察首次输入延迟 (FID)
const fidObserver = new PerformanceObserver((list) => {
  list.getEntries().forEach((entry) => {
    // entry.processingStart 是主线程开始处理的时间
    // entry.startTime 是用户首次交互的时间
    const fid = entry.processingStart - entry.startTime;

    console.log("FID (首次输入延迟):", fid, "ms");

    // FID 理想值应该小于 100ms
    if (fid > 100) {
      console.warn("FID 过高，建议优化主线程任务");
    }
  });
});

fidObserver.observe({ type: "first-input", buffered: true });

// 3. 观察最大内容绘制 (LCP)
let lcpValue = 0;

const lcpObserver = new PerformanceObserver((list) => {
  // 获取最后一项（最新的LCP）
  const entries = list.getEntries();
  const lastEntry = entries[entries.length - 1];

  console.log("LCP (最大内容绘制):", lastEntry.startTime, "ms");
  console.log("LCP 元素:", lastEntry.element);

  lcpValue = lastEntry.startTime;

  // LCP 理想值应该小于 2.5秒
  if (lcpValue > 2500) {
    console.warn("LCP 过高，建议优化首屏加载");
  }
});

lcpObserver.observe({ type: "largest-contentful-paint", buffered: true });

// 4. 观察布局偏移 (CLS)
let clsValue = 0;

const clsObserver = new PerformanceObserver((list) => {
  for (const entry of list.getEntries()) {
    // 检查这是否是用户交互导致的布局偏移
    if (!entry.hadRecentInput) {
      // 累加布局偏移分数
      // 分数计算：影响分数 × 距离分数
      clsValue += entry.value;

      console.log("检测到布局偏移:", entry.value);
      console.log("受影响元素:", entry.sources);
    }
  }

  console.log("当前 CLS:", clsValue);

  // CLS 理想值应该小于 0.1
  if (clsValue > 0.1) {
    console.warn("CLS 过高，页面视觉不稳定");
  }
});

clsObserver.observe({ type: "layout-shift", buffered: true });

// 5. 观察资源加载
const resourceObserver = new PerformanceObserver((list) => {
  list.getEntries().forEach((entry) => {
    // 只关注慢资源（加载超过1秒）
    if (entry.duration > 1000) {
      console.warn("慢资源检测:", entry.name, "耗时:", entry.duration, "ms");

      // 可以进一步分析
      if (entry.transferSize > 500 * 1024) {
        console.warn("大资源提醒:", (entry.transferSize / 1024).toFixed(2), "KB");
      }
    }
  });
});

resourceObserver.observe({ entryTypes: ["resource"] });

// 页面卸载前上报最终数据
window.addEventListener("visibilitychange", () => {
  if (document.visibilityState === "hidden") {
    // 页面隐藏时，上报最终的性能数据
    reportFinalPerformance({
      lcp: lcpValue,
      cls: clsValue,
      longTasks: [], // 可以记录长任务次数
      timestamp: Date.now(),
    });
  }
});
```

### 2.5 真实用户监控 (RUM) 实践

真实用户监控（Real User Monitoring）是从真实用户浏览器收集性能数据：

```javascript
// 真实用户监控 (RUM) SDK 示例

class RealUserMonitor {
  constructor(options = {}) {
    // 监控服务器地址
    this.endpoint = options.endpoint || "/api/monitor";

    // 应用标识
    this.appId = options.appId || "my-app";

    // 用户标识（可选）
    this.userId = options.getUserId?.() || null;

    // 采样率（0-1），用于减少数据量
    this.sampleRate = options.sampleRate || 1;

    // 性能数据队列
    this.performanceQueue = [];

    // 错误数据队列
    this.errorQueue = [];

    // 初始化
    this.init();
  }

  // 初始化监控
  init() {
    // 1. 监听页面加载性能
    this.monitorPerformance();

    // 2. 监听JS错误
    this.monitorErrors();

    // 3. 监听资源加载错误
    this.monitorResourceErrors();

    // 4. 监听用户行为
    this.monitorUserBehavior();

    // 5. 监听页面卸载，确保数据发送
    this.monitorPageUnload();

    // 6. 定期上报数据（每30秒）
    setInterval(() => this.flush(), 30000);
  }

  // 性能监控
  monitorPerformance() {
    // 使用 PerformanceObserver 监控关键指标
    const observer = new PerformanceObserver((list) => {
      list.getEntries().forEach((entry) => {
        // 构建性能数据
        const perfData = {
          type: entry.entryType,
          name: entry.name,
          value: entry.value || entry.duration || 0,
          timestamp: Date.now(),
          url: window.location.href,
          referrer: document.referrer,
          userId: this.userId,
        };

        // 添加到队列
        this.performanceQueue.push(perfData);
      });
    });

    // 观察所有支持的指标
    observer.observe({
      entryTypes: [
        "navigation",      // 页面导航
        "resource",        // 资源加载
        "paint",           // 绘制时间
        "longtask",        // 长任务
        "largest-contentful-paint", // LCP
        "layout-shift",    // CLS
        "first-input",     // FID
      ],
    });
  }

  // 错误监控
  monitorErrors() {
    // 监听未处理的Promise拒绝
    window.addEventListener("unhandledrejection", (event) => {
      const error = {
        type: "unhandledrejection",
        message: event.reason?.message || String(event.reason),
        stack: event.reason?.stack || "",
        timestamp: Date.now(),
        url: window.location.href,
        userId: this.userId,
      };

      this.errorQueue.push(error);
    });

    // 监听JavaScript错误
    window.addEventListener("error", (event) => {
      const error = {
        type: "javascript-error",
        message: event.message,
        filename: event.filename,
        lineno: event.lineno,
        colno: event.colno,
        stack: event.error?.stack || "",
        timestamp: Date.now(),
        url: window.location.href,
        userId: this.userId,
      };

      this.errorQueue.push(error);
    });
  }

  // 资源加载错误监控
  monitorResourceErrors() {
    // 监听资源加载失败（图片、脚本、样式表等）
    window.addEventListener("error", (event) => {
      // 只处理资源错误（event.target !== window）
      if (event.target !== window) {
        const target = event.target;

        const error = {
          type: "resource-error",
          // 资源URL
          url: target.src || target.href,
          // 资源类型
          resourceType: target.tagName.toLowerCase(),
          // 错误消息
          message: event.message,
          timestamp: Date.now(),
          url: window.location.href,
          userId: this.userId,
        };

        this.errorQueue.push(error);
      }
    }, true); // 使用捕获阶段
  }

  // 用户行为监控
  monitorUserBehavior() {
    // 点击事件
    document.addEventListener("click", (event) => {
      // 获取点击的元素
      const target = event.target;

      // 只记录有data-monitor属性的元素
      if (target.dataset?.monitor) {
        this.reportUserAction({
          action: "click",
          element: target.dataset.monitor,
          text: target.textContent?.slice(0, 50),
          timestamp: Date.now(),
        });
      }
    });

    // 页面可见性变化
    document.addEventListener("visibilitychange", () => {
      this.reportUserAction({
        action: document.hidden ? "page-hidden" : "page-visible",
        timestamp: Date.now(),
        duration: Date.now() - this.pageStartTime,
      });
    });
  }

  // 页面卸载监控
  monitorPageUnload() {
    // 记录页面开始时间
    this.pageStartTime = Date.now();

    // 页面卸载前发送数据
    window.addEventListener("beforeunload", () => {
      this.flush();
    });

    // 使用 visibilitychange 作为保底
    document.addEventListener("visibilitychange", () => {
      if (document.visibilityState === "hidden") {
        this.flush();
      }
    });
  }

  // 上报数据
  async reportUserAction(data) {
    // 可以用于实时监控用户操作
    console.log("用户行为:", data);
  }

  // 批量发送数据
  flush() {
    if (this.performanceQueue.length === 0 && this.errorQueue.length === 0) {
      return;
    }

    const payload = {
      appId: this.appId,
      performance: this.performanceQueue.splice(0), // 取走并清空
      errors: this.errorQueue.splice(0),
      timestamp: Date.now(),
    };

    // 使用 sendBeacon 确保数据发送
    if (navigator.sendBeacon) {
      navigator.sendBeacon(
        this.endpoint,
        JSON.stringify(payload)
      );
    } else {
      // 降级方案：使用同步 XMLHttpRequest
      const xhr = new XMLHttpRequest();
      xhr.open("POST", this.endpoint, false); // false = 同步
      xhr.setRequestHeader("Content-Type", "application/json");
      xhr.send(JSON.stringify(payload));
    }
  }
}

// 使用示例
const monitor = new RealUserMonitor({
  endpoint: "/api/monitor",
  appId: "my-web-app",
  sampleRate: 0.1, // 10%的用户
  getUserId: () => localStorage.getItem("userId"),
});
```

## 三、错误监控（Error Monitoring）

### 3.1 为什么错误监控至关重要？

错误监控是前端监控中最关键的部分。用户的每一次错误，都可能意味着：
- 用户无法完成操作
- 用户可能离开你的网站
- 用户可能会向客服抱怨
- 最坏的情况：用户流失

### 3.2 Sentry：最流行的错误监控平台

Sentry是目前最流行的前端错误监控平台，它提供了：
- 实时错误收集
- 错误聚合和去重
- Source Map 支持
- 性能监控
- 问题追踪
- 告警通知

```javascript
// Sentry SDK 集成示例

// 1. 安装 @sentry/browser
// npm install @sentry/browser

// 2. 初始化 Sentry
import * as Sentry from "@sentry/browser";

// 初始化配置
Sentry.init({
  // Sentry DSN (Data Source Name) - 在Sentry官网获取
  dsn: "https://xxxxx@sentry.io/xxxxx",

  // 环境
  environment: process.env.NODE_ENV,

  // 发布版本
  release: "my-app@1.0.0",

  // 采样率（0-1）
  tracesSampleRate: 0.1,

  // 采样率用于错误（0-1）
  sampleRate: 1,

  // 默认标签
  defaultIntegrations: true,

  // 过滤不需要收集的错误
  beforeSend(event, hint) {
    // 过滤某些自定义错误
    const error = hint?.originalException;

    // 忽略某些类型的错误
    if (error instanceof CustomIgnoreError) {
      return null; // 不上报
    }

    // 过滤某些环境的错误
    if (process.env.NODE_ENV === "development") {
      return null; // 开发环境不上报
    }

    return event; // 其他错误上报
  },

  // 收集用户信息
  setUser({
    id: "user-id",
    email: "user@example.com",
```

### 3.4 错误聚合与去重

生产环境中会产生大量相似错误，需要聚合去重：

```javascript
// 错误去重和聚合策略

class ErrorAggregator {
  constructor() {
    // 错误缓存（用于去重）
    this.errorCache = new Map();

    // 错误计数器
    this.errorCounts = new Map();

    // 去重的时间窗口（毫秒）
    this.dedupWindow = 60000; // 1分钟内相同错误只计算一次
  }

  // 生成错误的指纹（用于去重）
  generateFingerprint(error) {
    // 基于错误类型、消息和堆栈生成指纹
    const msg = error.message || "";
    const stack = error.stack || "";

    // 简化堆栈：只保留文件路径和行号
    const simplifiedStack = stack
      .split("\n")
      .slice(0, 3) // 只取前3行
      .map((line) => {
        // 提取文件名和行号
        const match = line.match(/\((.*):(\d+):(\d+)\)/);
        if (match) {
          return `${match[1]}:${match[2]}`;
        }
        return line;
      })
      .join("|");

    // 生成指纹：类型 + 消息 + 堆栈指纹
    return `${error.type}:${msg.slice(0, 100)}:${simplifiedStack}`;
  }

  // 检查是否是重复错误
  isDuplicate(fingerprint) {
    const now = Date.now();
    const lastSeen = this.errorCache.get(fingerprint);

    if (lastSeen && now - lastSeen < this.dedupWindow) {
      // 在时间窗口内，是重复错误
      return true;
    }

    // 更新最后见到的时间
    this.errorCache.set(fingerprint, now);
    return false;
  }

  // 记录错误
  recordError(error) {
    const fingerprint = this.generateFingerprint(error);

    if (this.isDuplicate(fingerprint)) {
      // 重复错误，只增加计数
      const count = this.errorCounts.get(fingerprint) || 0;
      this.errorCounts.set(fingerprint, count + 1);

      // 定期更新（避免存储太多）
      if ((count + 1) % 10 === 0) {
        this.reportErrorCount(fingerprint, count + 1);
      }

      return false; // 不重复上报
    }

    // 新错误，重置计数
    this.errorCounts.set(fingerprint, 1);

    // 立即上报
    return true;
  }

  // 上报错误计数
  reportErrorCount(fingerprint, count) {
    console.log(`错误 "${fingerprint}" 发生了 ${count} 次`);
    // 发送到服务器
  }
}
```

```javascript
  generateSessionId() {
    // 优先使用sessionStorage，如果没有则生成新ID
    let sessionId = sessionStorage.getItem("monitor_session_id");

    if (!sessionId) {
      sessionId = "sess_" + Date.now() + "_" + Math.random().toString(36).slice(2);
      sessionStorage.setItem("monitor_session_id", sessionId);
    }

    return sessionId;
  }

  // 获取用户ID
  getUserId() {
    // 优先使用登录用户的ID
    const userId = localStorage.getItem("userId");
    if (userId) return userId;

    // 降级为设备ID（生成并缓存）
    let deviceId = localStorage.getItem("monitor_device_id");
    if (!deviceId) {
      deviceId = "dev_" + Date.now() + "_" + Math.random().toString(36).slice(2);
      localStorage.setItem("monitor_device_id", deviceId);
    }

    return deviceId;
  }

  // 监听页面可见性变化
  setupVisibilityTracking() {
    let pageVisibleTime = 0;
    let lastVisibleTime = Date.now();

    document.addEventListener("visibilitychange", () => {
      if (document.hidden) {
        // 页面隐藏，累加停留时间
        pageVisibleTime += Date.now() - lastVisibleTime;
      } else {
        // 页面可见，更新最后可见时间
        lastVisibleTime = Date.now();
      }
    });

    // 页面卸载时上报
    window.addEventListener("beforeunload", () => {
      this.trackPageView({
        duration: pageVisibleTime + (Date.now() - lastVisibleTime),
        type: "unload",
      });
    });
  }

  // 监听浏览器历史变化（单页应用需要）
  setupHistoryTracking() {
    // 监听 popstate 事件（浏览器前进后退）
    window.addEventListener("popstate", () => {
      this.trackPageView();
    });

    // 拦截 pushState 和 replaceState（HTML5路由）
    const originalPushState = history.pushState;
    const originalReplaceState = history.replaceState;

    history.pushState = (...args) => {
      originalPushState.apply(history, args);
      this.trackPageView();
    };

    history.replaceState = (...args) => {
      originalReplaceState.apply(history, args);
      this.trackPageView();
    };
  }

  // 上报页面访问
  trackPageView(data = {}) {
    const pageView = {
      // 页面标识
      appId: this.appId,
      sessionId: this.sessionId,
      userId: this.userId,

      // 页面信息
      url: window.location.href,
      referrer: document.referrer,
      title: document.title,

      // 时间
      timestamp: new Date().toISOString(),
      timestampTs: Date.now(),

      // 访问时长
      duration: data.duration || 0,

      // 访问类型
      type: data.type || "page_view", // page_view, unload,history_change

      // 设备信息
      screenWidth: window.innerWidth,
      screenHeight: window.innerHeight,

      // 用户代理
      userAgent: navigator.userAgent,
    };

    // 存储到历史
    this.pageViews.push(pageView);

    // 发送数据
    this.report(pageView);
  }

  // 发送数据
```

```javascript
// 用户交互追踪

class InteractionTracker {
  constructor(options = {}) {

          text: target.textContent?.slice(0, 50) || null,
          // 元素的唯一标识
          monitorId: monitorId,
          // 元素路径（用于识别相同位置的不同元素）
          path: this.getElementPath(target),
        },

        // 页面信息
        url: window.location.href,
        timestamp: Date.now(),
      };

      this.report(clickData);
    });
  }

  // 输入追踪（用于分析表单填写）
  setupInputTracking() {
    // 追踪表单输入
    document.addEventListener("input", (event) => {
      const target = event.target;

      // 只追踪有监控标识的输入框
      if (!target.dataset?.monitorId) {
        return;
      }

      const inputData = {
        sessionId: this.sessionId,
        userId: localStorage.getItem("userId"),

        // 输入信息
        element: {
          tag: target.tagName.toLowerCase(),
          type: target.type,
          monitorId: target.dataset.monitorId,
          // 输入值的长度（不记录实际内容，保护隐私）
          valueLength: target.value?.length || 0,
        },

        url: window.location.href,
        timestamp: Date.now(),
      };

      this.report(inputData);
    });
  }

  // 获取元素路径（用于唯一标识点击位置）
  getElementPath(element) {
    const path = [];

    // 向上遍历父元素
    let current = element;
    while (current && current !== document.body) {
      // 获取元素的唯一标识
      let identifier = current.tagName.toLowerCase();

      if (current.id) {
        identifier += "#" + current.id;
      } else if (current.className) {
        identifier += "." + current.className.split(" ")[0];
      }

      // 添加索引（如果 siblings 有重复标签）
      const siblings = current.parentElement?.children || [];
      const sameTagSiblings = Array.from(siblings).filter(
        (s) => s.tagName === current.tagName
      );
      if (sameTagSiblings.length > 1) {
        const index = sameTagSiblings.indexOf(current);
        identifier += `:nth-child(${index + 1})`;
      }

      path.unshift(identifier);
      current = current.parentElement;
    }

    return path.join(" > ");
  }

  // 判断是否是重要元素
  isImportantElement(element) {
    const importantTags = ["a", "button", "input", "select", "textarea"];
    return importantTags.includes(element.tagName.toLowerCase());
  }

  // 上报数据
  report(data) {
    navigator.sendBeacon?.("/api/analytics/interaction", JSON.stringify(data));
  }
}
```

```javascript
// 漏斗分析追踪

class FunnelTracker {
  constructor(appId) {
    this.appId = appId;
    this.funnels = this.loadFunnels();
  }

  // 加载漏斗配置
  loadFunnels() {

    return {
      // 注册漏斗
      register: [
        { name: "进入注册页", event: "page_view", condition: (data) => data.url.includes("/register") },
        { name: "填写表单", event: "interaction", condition: (data) => data.element?.monitorId === "register-form" },

      purchase: [
        { name: "进入商品页", event: "page_view", condition: (data) => data.url.includes("/product/") },
        { name: "加入购物车", event: "interaction", condition: (data) => data.element?.monitorId === "add-to-cart" },
        { name: "进入结算页", event: "page_view", condition: (data) => data.url.includes("/checkout") },
        { name: "点击支付", event: "interaction", condition: (data) => data.element?.monitorId === "payment-submit" },
        { name: "支付成功", event: "custom", condition: (data) => data.event === "payment-success" },
      ],
    };
  }

  // 追踪漏斗步骤
  trackFunnel(funnelName, eventData) {
    const funnel = this.funnels[funnelName];
    if (!funnel) return;

    // 获取或初始化漏斗状态
    const funnelState = this.getFunnelState(funnelName);

    // 找到当前应该完成的步骤
    const currentStepIndex = funnelState.currentStep;

    for (let i = currentStepIndex; i < funnel.length; i++) {
      const step = funnel[i];

      // 检查是否满足步骤条件
      if (this.matchesStep(eventData, step)) {
        // 完成该步骤
        funnelState.steps.push({
          name: step.name,
          timestamp: Date.now(),
          data: eventData,
        });

        funnelState.currentStep = i + 1;

        // 如果是最后一步，完成漏斗
        if (i === funnel.length - 1) {
          funnelState.completed = true;
          funnelState.completedAt = Date.now();
        }

        break;
      }
    }

    // 保存状态
    this.saveFunnelState(funnelName, funnelState);

    // 检查流失点（某一步骤后没有继续）
    this.checkDropOff(funnelName, funnelState);
  }

  // 检查漏斗数据是否匹配步骤
  matchesStep(eventData, step) {
    // 检查事件类型
    if (step.event !== "custom" && eventData.event !== step.event) {
      return false;
    }

    // 检查条件函数
    if (step.condition && !step.condition(eventData)) {
      return false;
    }

    return true;
  }

  // 获取漏斗状态
  getFunnelState(funnelName) {
    const stored = localStorage.getItem(`funnel_${funnelName}`);
    if (stored) {
      return JSON.parse(stored);
    }

    // 初始化新漏斗
    return {
      funnelName,
      currentStep: 0,
      steps: [],
      completed: false,
      startedAt: Date.now(),
    };
  }

  // 保存漏斗状态
  saveFunnelState(funnelName, state) {
    localStorage.setItem(`funnel_${funnelName}`, JSON.stringify(state));
  }

  // 检查流失点
  checkDropOff(funnelName, state) {
    // 如果漏斗未完成，且超过30分钟没有新进展，认为是流失
    if (!state.completed) {
      const lastStepTime = state.steps[state.steps.length - 1]?.timestamp;
      const now = Date.now();

      if (lastStepTime && now - lastStepTime > 30 * 60 * 1000) {
        console.warn(`漏斗 [${funnelName}] 用户流失，停留在步骤 ${state.currentStep}`);

        // 上报流失数据
        this.reportDropOff(funnelName, state);
      }
    }
  }

  // 上报流失数据
  reportDropOff(funnelName, state) {
    const dropOffData = {
      funnelName,
      reachedStep: state.currentStep,
      reachedStepName: state.steps[state.currentStep - 1]?.name,
      droppedAt: Date.now(),
      timeSpent: Date.now() - state.startedAt,
    };

    navigator.sendBeacon?.(
      "/api/analytics/funnel-dropoff",
      JSON.stringify(dropOffData)
    );
  }
}
```

```javascript
class RealtimeDashboard {

      };

      // 连接错误
      this.ws.onerror = (error) => {
        console.error("实时大屏：连接错误", error);

      return;
    }

    this.reconnectAttempts++;

    console.log(`实时大屏：${this.reconnectInterval}ms 后重连...`);

    setTimeout(() => {
      this.connect();
    }, this.reconnectInterval);
  }

  // 认证
  authenticate() {
    this.send({
      type: "auth",
      token: this.getAuthToken(),
    });
  }

  // 获取认证令牌
  getAuthToken() {
    return localStorage.getItem("authToken");
  }

  // 发送消息
  send(data) {
    if (this.ws && this.ws.readyState === WebSocket.OPEN) {
      this.ws.send(JSON.stringify(data));
    }
  }

  // 处理接收到的消息
  handleMessage(data) {
    switch (data.type) {
      case "error":
        console.error("服务器错误:", data.message);
        break;

      case "metrics":
        // 更新指标数据
        this.updateMetrics(data.payload);
        break;

      case "alert":
        // 处理告警
        this.handleAlert(data.payload);
        break;

      case "notification":
        // 处理通知
        this.showNotification(data.payload);
        break;
    }
  }

  // 更新指标数据
  updateMetrics(metrics) {
    // 触发自定义事件，让UI组件可以订阅
    window.dispatchEvent(
      new CustomEvent("dashboard-metrics-update", {
        detail: metrics,
      })
    );
  }

  // 处理告警
  handleAlert(alert) {
    console.warn("收到告警:", alert);

    // 可以触发声音提醒
    if (alert.level === "critical") {
      this.playAlertSound();
    }

    // 可以发送浏览器通知
    if (Notification.permission === "granted") {
      new Notification("系统告警", {
        body: alert.message,
        icon: "/logo.png",
      });
    }
  }

  // 播放告警声音
  playAlertSound() {
    const audio = new Audio("/alert.mp3");
    audio.play().catch(() => {});
  }

  // 显示通知
  showNotification(notification) {
    window.dispatchEvent(
      new CustomEvent("dashboard-notification", {
        detail: notification,
      })
    );
  }

  // 断开连接
  disconnect() {
    if (this.ws) {
      this.ws.close();
      this.ws = null;
    }
  }

  // 订阅特定指标
  subscribe(metrics) {
    this.send({
      type: "subscribe",
      metrics,
    });
  }

  // 取消订阅
  unsubscribe(metrics) {
    this.send({
      type: "unsubscribe",
      metrics,
    });
  }
}
```

### 6.1 自建 vs 使用第三方服务

| 维度 | 自建监控 | 第三方服务（如Sentry） |
|------|----------|------------------------|
| **成本** | 服务器、运维人员成本 | 按量付费或订阅制 |
| **灵活性** | 完全可控，可定制 | 受限于服务商功能 |
| **隐私** | 数据完全在自己服务器 | 数据上传到第三方 |
| **维护** | 需要专人维护 | 服务商维护 |
| **功能** | 需要从零开发 | 功能完善，开箱即用 |
| **接入复杂度** | 高 | 低 |

### 6.2 推荐方案

#### 小型项目（个人/小团队）
- 使用Sentry免费版（5000次错误/月）

- ClickHouse + Grafana（时序数据）

### 6.3 监控数据隐私合规

监控数据可能包含用户隐私，需要注意：

```javascript
// 隐私合规配置

class PrivacyCompliantMonitor {
  constructor() {
    // 需要脱敏的字段
    this.sensitiveFields = [
      "password",
      "token",
      "secret",
      "creditCard",
      "ssn", // 身份证号
      "phone",
      "email",
    ];
  }

  // 脱敏处理
  sanitize(data) {
    const result = { ...data };

    // 递归遍历对象
    const sanitizeObject = (obj) => {
      for (const key in obj) {
        if (typeof obj[key] === "object") {
          sanitizeObject(obj[key]);
        } else if (this.isSensitive(key)) {
          // 将敏感字段替换为星号
          obj[key] = "***";
        }
      }
    };

    sanitizeObject(result);

    // 移除可能包含隐私的URL参数
    if (result.url) {
      result.url = this.removeQueryParams(result.url);
    }

    return result;
  }

  // 判断是否是敏感字段
  isSensitive(key) {
    const lowerKey = key.toLowerCase();
    return this.sensitiveFields.some((field) => lowerKey.includes(field));
  }

  // 移除URL中的敏感查询参数
  removeQueryParams(url) {
    try {
      const urlObj = new URL(url);
      const sensitiveParams = ["token", "id", "phone", "email"];

      sensitiveParams.forEach((param) => {
        urlObj.searchParams.delete(param);
      });

      return urlObj.toString();
    } catch {
      return url;
    }
  }
}
```
