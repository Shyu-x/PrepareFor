## 一、标题层级规范

### 1.1 整体结构

本书采用 **卷-章-节-小节** 四级标题结构：

| 层级 | 格式 | 示例 | 说明 |
|:---:|:---|:---|:---|
| 一级 | `# 第X卷-卷名` | `# 第1卷-基础核心` | 卷标题 |
| 二级 | `## 第Y章 章名` | `## 第1章 HTML与CSS` | 章标题 |
| 三级 | `### Z.Z 节名` | `### 1.1 HTML核心原理` | 节标题 |
| 四级 | `#### Z.Z.Z 小节名` | `#### 1.1.1 HTML5新增特性` | 小节标题 |

### 1.2 编号规则

```
卷号：第1卷、第2卷 ... 第7卷
章号：第1章、第2章 ... 第N章
节号：X.Y （X为章号，Y为节序号）
小节号：X.Y.Z （在节号后加序号）
```

### 1.3 标题命名规范

- **卷名**：简洁明确，不超过12个字
- **章名**：使用中文描述，如"HTML与CSS"
- **节名**：使用阿拉伯数字编号，如"1.1 HTML核心原理"
- **小节名**：使用阿拉伯数字编号，如"1.1.1 HTML5新增特性"

## 二、内容格式规范

### 2.1 代码块

```markdown
​```javascript
// 代码示例
function hello() {
  console.log('Hello World');
}
​```
```

- 代码语言标记必须明确：`javascript`、`typescript`、`html`、`css`、`bash` 等
- 代码过长时使用滚动条
- 关键代码添加中文注释

### 2.2 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|:---|:---:|:---|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

- 表头使用**加粗**
- 列对齐：`:---` 左对齐，`:---:` 居中，`---:` 右对齐

### 2.3 列表

- 无序列表使用 `-` 或 `*`
- 有序列表使用 `1.`、`2.`
- 嵌套列表缩进2个空格

### 2.4 引用

```markdown
> 引用内容
```

- 使用 `>` 标记引用
- 多行引用使用多个 `>`

## 四、内容排版规范

### 4.1 段落

- 段落之间空一行
- 不使用手动换行，保持自然流
- 行长控制在80-120字符

### 4.2 强调

- **加粗**使用 `**内容**`
- *斜体*使用 `*内容*`
- `代码`使用反引号

### 4.3 分割线

使用 `---` 作为章节分割线

## 五、面试题格式规范

### 5.1 题目样式

```markdown
### 1.1.1 HTML5 新增了哪些新特性？

**参考要点：**
1. 语义化标签
2. 表单增强
3. 多媒体支持
4. Canvas/WebGL
5. 本地存储
6. Web Worker
7. WebSocket
```

### 5.2 参考答案

- 使用 **参考答案：** 或 **参考要点：** 标记
- 关键点使用列表清晰展示
- 重要概念加粗强调
