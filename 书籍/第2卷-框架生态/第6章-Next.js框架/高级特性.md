## 6.11 高级特性

### 6.11.1 ISR（增量静态再生）

**参考答案：**

ISR 允许静态页面在后台定期重新生成。

**基础 ISR：**

```tsx
// src/app/blog/[slug]/page.tsx
export const revalidate = 60  // 每60秒重新验证

export async function generateStaticParams() {
  const posts = await getPosts()
  return posts.map((post) => ({ slug: post.slug }))
}

export default async function BlogPost({ params }: Props) {
  const post = await getPost(params.slug)

  return <article>{post.content}</article>
}
```

**按需 ISR：**

```tsx
// src/app/api/revalidate/route.ts
import { revalidatePath, revalidateTag } from 'next/cache'
import { NextRequest, NextResponse } from 'next/server'

export async function POST(request: NextRequest) {
  const { tag, path } = await request.json()

  if (tag) {
    revalidateTag(tag)
  }

  if (path) {
    revalidatePath(path)
  }

  return NextResponse.json({
    revalidated: true,
    now: Date.now(),
  })
}
```

### 6.11.2 Draft Mode（预览模式）

**参考答案：**

```tsx
// src/app/api/draft/route.ts
import { draftMode } from 'next/headers'
import { redirect } from 'next/navigation'
import { getPost } from '@/lib/posts'

export async function GET(request: Request) {
  const { searchParams } = new URL(request.url)
  const secret = searchParams.get('secret')
  const slug = searchParams.get('slug')

  // 验证密钥
  if (secret !== process.env.DRUPAL_PREVIEW_SECRET || !slug) {
    return new Response('Invalid token', { status: 401 })
  }

  // 获取文章
  const post = await getPost(slug)

  if (!post) {
    return new Response('Invalid slug', { status: 401 })
  }

  // 启用 Draft Mode
  draftMode().enable()

  // 跳转
  redirect(`/posts/${post.slug}`)
}
```

**关闭 Draft Mode：**

```tsx
// src/app/api/disable-draft/route.ts
import { draftMode } from 'next/headers'
import { redirect } from 'next/navigation'

export async function GET() {
  draftMode().disable()
  redirect('/')
}
```

### 6.11.3 Internationalization（国际化）

**参考答案：**

**next-intl 集成：**

```tsx
// src/i18n.ts
import { getRequestConfig } from 'next-intl/server'

export default getRequestConfig(async ({ locale }) => ({
  messages: (await import(`./messages/${locale}.json`)).default
}))
```

**根布局配置：**

```tsx
// src/app/[locale]/layout.tsx
import { NextIntlClientProvider, useMessages } from 'next-intl'

export default function LocaleLayout({
  children,
  params,
}: {
  children: React.ReactNode
  params: Promise<{ locale: string }>
}) {
  const { locale } = await params
  const messages = useMessages()

  return (
    <html lang={locale}>
      <body>
        <NextIntlClientProvider locale={locale} messages={messages}>
          {children}
        </NextIntlClientProvider>
      </body>
    </html>
  )
}
```

**Middleware：**

```tsx
// src/middleware.ts
import createMiddleware from 'next-intl/middleware'
import { routing } from './i18n/routing'

export default createMiddleware(routing)

export const config = {
  matcher: ['/', '/(de|en|es)/:path*']
}
```

### 6.11.4 OpenTelemetry（可观测性）

**参考答案：**

**instrumentation.ts：**

```tsx
// src/instrumentation.ts
import { NodeSDK } from '@opentelemetry/sdk-node'
import { JaegerExporter } from '@opentelemetry/exporter-jaeger'
import { HttpTraceContextLayer } from '@opentelemetry/context-zone-node'
import { registerInstrumentations } from '@opentelemetry/instrumentation'

const sdk = new NodeSDK({
  traceExporter: new JaegerExporter(),
  serviceName: 'nextjs-app',
  contextManager: new HttpTraceContextLayer(),
})

sdk.start()

registerInstrumentations({
  instrumentations: [],
})

process.on('SIGTERM', () => {
  sdk.shutdown().catch(console.error)
})
```

### 6.11.5 流式渲染

**参考答案：**

**使用 Suspense：**

```tsx
// src/app/blog/page.tsx
import { Suspense } from 'react'
import { PostList, PostListSkeleton } from '@/components'

export default function BlogPage() {
  return (
    <div>
      <h1>Blog</h1>
      <Suspense fallback={<PostListSkeleton />}>
        <PostList />
      </Suspense>
    </div>
  )
}
```

**streaming 实现：**

```tsx
// src/app/dashboard/page.tsx
import { Suspense } from 'react'
import { RevenueChart, RecentSales, SalesSkeleton } from '@/components'

export default function DashboardPage() {
  return (
    <main>
      <h1>Dashboard</h1>
      <div className="grid grid-cols-2 gap-4">
        <RevenueChart />
        <Suspense fallback={<SalesSkeleton />}>
          <RecentSales />
        </Suspense>
      </div>
    </main>
  )
}
```

### 6.11.6 懒加载

**参考答案：**

**动态导入：**

```tsx
import dynamic from 'next/dynamic'

// 懒加载组件
const DynamicComponent = dynamic(
  () => import('@/components/HeavyComponent'),
  {
    loading: () => <p>Loading...</p>,
    ssr: false,  // 禁用服务器端渲染
  }
)

// 懒加载第三方库
const HeavyLib = dynamic(() => import('heavy-lib'), {
  ssr: false,
})
```

### 6.11.7 面试常见问题

**Q1: ISR vs SSR vs SSG 区别？**

| 渲染方式 | 构建时 | 请求时 | 重新生成 |
|:---|:---|:---|:---|
| SSG | ✅ | ❌ | ❌ |
| ISR | ✅ | ❌ | ✅ 定时 |
| SSR | ❌ | ✅ | ❌ |
| CSR | ❌ | ✅ | ❌ |

**Q2: Draft Mode 适用场景？**

- CMS 内容预览
- 预发布版本查看
- 个性化页面预览

**Q3: next/image 的 sizes 属性作用？**

告诉浏览器在不同断点下使用的图片尺寸，用于生成响应式图片。
