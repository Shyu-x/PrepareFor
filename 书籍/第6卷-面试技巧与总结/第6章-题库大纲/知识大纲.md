### 6.5 面试流程全解析

```
面试流程：
简历投递 → 简历筛选 → 笔试初筛 → 技术一面 → 技术二面 → 主管面 → HR面 → 薪资谈判 → 体检 → 入职

各环节考察重点：
- 简历筛选：教育背景、技术栈匹配度、项目经验
- 笔试初筛：基础知识、算法能力
- 技术一面：深度原理、解决问题能力
- 技术二面：系统设计、项目经验
- 主管面：业务理解、团队协作
- HR面：价值观匹配、薪资期望
```

### 6.7 学习复习路线图

#### 6.7.1 零基础入门路线

```
第一阶段：HTML/CSS基础（1-2周）
┣━━ HTML语义化标签
┣━━ CSS基础选择器
┣━━ 盒模型与浮动
┣━━ Flex布局入门
┗━━ 常见布局实现

第二阶段：JavaScript基础（2-3周）
┣━━ 数据类型与转换
┣━━ 函数与作用域
┣━━ DOM操作基础
┣━━ 事件处理
┗━━ 异步编程入门

第三阶段：框架入门（2-3周）
┣━━ React或Vue基础
┣━━ 组件化开发
┣━━ 状态管理入门
┗━━ 路由基础

第四阶段：工程化（1-2周）
┣━━ Git版本控制
┣━━ npm/yarn使用
┣━━ 构建工具入门
┗━━ 代码规范基础
```
