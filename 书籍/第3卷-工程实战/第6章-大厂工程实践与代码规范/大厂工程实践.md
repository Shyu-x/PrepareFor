### 6.5.4 Git 提交规范

```
<type>(<scope>): <subject>

<body>

<footer>
```

| 类型 | 说明 |
|:---|:---|

```
feat(user): add login API

- add token refresh logic
- add logout functionality

Closes #123
```

4. **文化层**：代码评审 + 培训

**Q3: 如何选择代码规范？**

| 场景 | 推荐规范 |
|:---|:---|
| React 项目 | Airbnb + TypeScript |
| Vue 项目 | 阿里规范 + TypeScript |
| 简单项目 | Google 规范简化版 |
